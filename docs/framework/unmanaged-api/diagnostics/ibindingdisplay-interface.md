# <a name="ibindingdisplay-interface"></a>IBindingDisplay 인터페이스

## <a name="methods"></a>메서드  
  
|메서드|설명|  
|------------|-----------------|  
|[GetCurrentDisplay 메서드](ibindingdisplay-getcurrentdisplay-method.md)|현재 바인딩 표시 정보를 반환 합니다.|
