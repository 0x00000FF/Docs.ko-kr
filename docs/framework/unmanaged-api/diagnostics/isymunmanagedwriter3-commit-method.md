# <a name="isymunmanagedwriter3commit-method"></a>ISymUnmanagedWriter3::Commit 메서드
지금까지 작성 된 변경 내용을 스트림에 커밋합니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
HRESULT Commit();  
```  
  
## <a name="return-value"></a>Return Value  
 메서드가 성공 하면이 고, 그렇지 않으면 S_OK입니다. 그렇지 않으면 E_FAIL 또는 일부 다른 오류 코드입니다.
