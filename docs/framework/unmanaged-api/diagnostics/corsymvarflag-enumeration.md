# <a name="corsymvarflag-enumeration"></a>CorSymVarFlag 열거형
컴파일러에서 생성 된 변수 인지를 나타냅니다.  
  
## <a name="syntax"></a>구문  
  
```  
typedef enum CorSymVarFlag   
{  
    VAR_IS_COMP_GEN = 1  
} CorSymVarFlag;  
```  
  
## <a name="members"></a>멤버  
  
|멤버|설명|  
|------------|-----------------|  
|`VAR_IS_COMP_GEN`|지정한 변수로 컴파일러에서 생성 된 임을 나타냅니다.|  
  
## <a name="requirements"></a>요구 사항
