|메서드|설명|  
|------------|-----------------|  
|[AutoAttach 메서드](../../../../docs/framework/unmanaged-api/diagnostics/idebugautoattach-autoattach-method.md)|서버에서 호출한 디버거 자동 수행 연결 합니다.|
