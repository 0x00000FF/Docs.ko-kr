`IAssemblyName` 개체의 배열에 대 한 열거자를 나타냅니다.  
  
## <a name="methods"></a>메서드  
  
|메서드|설명|  
|------------|-----------------|  
|[Clone 메서드](iassemblyenum-clone-method.md)|이 `IAssemblyEnum` 개체의 단순 복사본을 만듭니다.|
