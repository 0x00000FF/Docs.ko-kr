# <a name="iassemblycacheitem-interface"></a>IAssemblyCacheItem 인터페이스
전역 어셈블리 캐시에 단일 어셈블리를 나타냅니다.  
  
## <a name="methods"></a>메서드  
  
|메서드|설명|  
|------------|-----------------|  
|[AbortItem 메서드](../../../../docs/framework/unmanaged-api/fusion/iassemblycacheitem-abortitem-method.md)|전역 어셈블리 캐시에 어셈블리를 해제 되기 전에 정리 작업을 수행할 수 있습니다.|  
|[Commit 메서드](../../../../docs/framework/unmanaged-api/fusion/iassemblycacheitem-commit-method.md)|메모리에 캐시 된 어셈블리 참조를 커밋합니다.|  
|[CreateStream 메서드](../../../../docs/framework/unmanaged-api/fusion/iassemblycacheitem-createstream-method.md)|지정 된 이름 및 형식 스트림을 만듭니다.|
