# <a name="closeenum-method"></a>CloseEnum 메서드
표시 된 열거형을 닫고 연결 된 리소스를 해제 합니다.  
  
## <a name="syntax"></a>구문  
  
```  
HRESULT CloseEnum(  
    HALINKENUM hEnum  
) PURE;  
```  
  
## <a name="parameters"></a>매개 변수  
 `hEnum`  
 핸들을 닫아야 하는 열거형입니다.  
  
## <a name="return-value"></a>반환 값  
 메서드가 성공 하면 S_OK를 반환 합니다.  
  
## <a name="requirements"></a>요구 사항  
 Alink.h 필요
