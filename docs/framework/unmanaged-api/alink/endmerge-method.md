# <a name="endmerge-method"></a>EndMerge 메서드
내보내기 범위에 모든 사용자 지정 특성 집합이 병합 된 나타냅니다.  
  
## <a name="syntax"></a>구문  
  
```  
HRESULT EndMerge(  
    mdAssembly AssemblyID  
) PURE;  
```  
  
## <a name="parameters"></a>매개 변수  
 `AssemblyID`  
 어셈블리의 ID입니다.  
  
## <a name="return-value"></a>반환 값  
 메서드가 성공 하면 S_OK를 반환 합니다.  
  
## <a name="requirements"></a>요구 사항  
 Alink.h 필요
