# <a name="ihostfilter-interface"></a>IHostFilter 인터페이스
Provides a method to indicate that a specified token will be processed.  
  
> [!NOTE]
> The `IHostFilter` interface must be implemented by the client.  
  
## <a name="methods"></a>메서드  
  
|메서드|설명|  
|------------|-----------------|  
|[MarkToken 메서드](../../../../docs/framework/unmanaged-api/metadata/ihostfilter-marktoken-method.md)|Indicates that the specified metadata token will be processed.|
