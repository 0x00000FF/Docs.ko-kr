# <a name="imetadatatablesgetguidheapsize-method"></a>IMetaDataTables::GetGuidHeapSize 메서드
GUID 힙 바이트의 크기를 가져옵니다.  
  
## <a name="syntax"></a>구문  
  
```  
HRESULT GetGuidHeapSize (  
    [out] ULONG   *pcbGuids  
);  
```  
  
## <a name="parameters"></a>매개 변수  
 `pcbGuids`  
 [out] GUID 힙 바이트의 크기에 대 한 포인터입니다.
