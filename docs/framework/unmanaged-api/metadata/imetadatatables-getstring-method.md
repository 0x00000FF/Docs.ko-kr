# <a name="imetadatatablesgetstring-method"></a>IMetaDataTables::GetString 메서드
현재 참조 범위의 테이블 열에서 지정된 인덱스에 문자열을 가져옵니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
HRESULT GetString (
    [in]  ULONG       ixString,  
    [out] const char  **ppString  
);  
```  
  
## <a name="parameters"></a>매개 변수  
 `ixString`  
 【인】 다음 값을 검색하기 시작할 인덱스입니다.  
  
 `ppString`  
 【아웃】 반환 된 문자열 값에 대 한 포인터입니다.
