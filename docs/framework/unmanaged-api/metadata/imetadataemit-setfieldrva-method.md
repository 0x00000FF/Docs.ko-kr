# <a name="imetadataemitsetfieldrva-method"></a>IMetaDataEmit::SetFieldRVA 메서드
지정된 토큰에서 참조하는 필드의 상대 가상 주소에 대한 전역 변수 값을 설정합니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
HRESULT SetFieldRVA (
    [in]  mdFieldDef  fd,
    [in]  ULONG       ulRVA
);  
```  
  
## <a name="parameters"></a>매개 변수  
 `fd`  
 【인】 대상 필드에 대한 토큰입니다.  
  
 `ulRVA`  
 【인】 코드 또는 데이터 영역의 주소입니다.
