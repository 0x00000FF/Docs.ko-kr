# <a name="imaptoken-interface"></a>IMapToken 인터페이스
어셈블리 간의 매핑 기능을 제공합니다.  
  
> [!NOTE]
>  `IMapToken` 클라이언트에서 인터페이스를 구현 해야 합니다.  
  
## <a name="methods"></a>메서드  
  
|메서드|설명|  
|------------|-----------------|  
|[Map 메서드](../../../../docs/framework/unmanaged-api/metadata/imaptoken-map-method.md)|메타 데이터 서명을 사용 하 여 두 어셈블리 사이의 관계를 매핑합니다.|
