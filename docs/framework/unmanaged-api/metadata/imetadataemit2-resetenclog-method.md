# <a name="imetadataemit2resetenclog-method"></a>IMetaDataEmit2::ResetENCLog 메서드
Resets the edit-and-continue log and starts a new session.  
  
## <a name="syntax"></a>구문  
  
```cpp  
HRESULT ResetENCLog ();  
```  
  
## <a name="parameters"></a>매개 변수  
 `ResetENCLog` takes no parameters.
