# <a name="imetadataemitsavetomemory-method"></a>IMetaDataEmit::SaveToMemory 메서드
현재 범위의 모든 메타데이터를 지정된 메모리 영역에 저장합니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
HRESULT SaveToMemory (
    [out]  void        *pbData,
    [in]   ULONG       cbData
);  
```  
  
## <a name="parameters"></a>매개 변수  
 `pbData`  
 【아웃】 메타데이터 작성을 시작할 주소입니다.  
  
 `cbData`  
 【인】 할당된 메모리의 크기(바이트)입니다.
