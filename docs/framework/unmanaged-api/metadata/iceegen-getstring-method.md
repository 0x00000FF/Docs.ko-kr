# <a name="iceegengetstring-method"></a>ICeeGen::GetString 메서드
지정된 된 상대 가상 주소에 저장 된 문자열을 가져옵니다.  
  
 이 메서드는 사용 되지 않으며 사용할 수 없습니다.  
  
## <a name="syntax"></a>구문  
  
```  
HRESULT GetString (  
    [in]  ULONG      RVA,   
    [out] LPWSTR     *lpString  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `RVA`  
 [in] 반환할 문자열의 상대 가상 주소입니다.  
  
 `lpString`  
 [out] 반환 된 문자열입니다.
