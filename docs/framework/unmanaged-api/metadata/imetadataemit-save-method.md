# <a name="imetadataemitsave-method"></a>IMetaDataEmit::Save 메서드
현재 범위의 모든 메타데이터를 지정된 주소의 파일에 저장합니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
HRESULT Save (
    [in]  LPCWSTR     szFile,
    [in]  DWORD       dwSaveFlags  
);  
```  
  
## <a name="parameters"></a>매개 변수  
 `wzFile`  
 【인】 저장할 파일의 이름입니다. 이 값이 null이면 메모리 내 복사본이 사용된 마지막 위치에 저장됩니다.  
  
 `dwSaveFlags`  
 [in] 예약되어 있습니다. 0이어야 합니다.
