# <a name="imetadataimportresetenum-method"></a>IMetaDataImport::ResetEnum 메서드
지정한 열거자를 지정한 위치로 다시 설정합니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
HRESULT ResetEnum (  
   [in] HCORENUM    hEnum,   
   [in] ULONG       ulPos  
);  
```  
  
## <a name="parameters"></a>매개 변수  
 `hEnum`  
 [in] The enumerator to reset.  
  
 `ulPos`  
 [in] The new position at which to place the enumerator.
