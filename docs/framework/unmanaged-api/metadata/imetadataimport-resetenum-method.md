# <a name="imetadataimportresetenum-method"></a>IMetaDataImport::ResetEnum 메서드
지정한 열거자를 지정한 위치로 다시 설정합니다.  
  
## <a name="syntax"></a>구문  
  
```  
HRESULT ResetEnum (  
   [in] HCORENUM    hEnum,   
   [in] ULONG       ulPos  
);  
```  
  
## <a name="parameters"></a>매개 변수  
 `hEnum`  
 [in] 다시 설정 하는 열거자입니다.  
  
 `ulPos`  
 [in] 열거자를 배치할 새 위치입니다.
