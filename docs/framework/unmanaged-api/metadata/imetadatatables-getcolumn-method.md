# <a name="imetadatatablesgetcolumn-method"></a>IMetaDataTables::GetColumn 메서드
지정 된 열 및 지정된 된 테이블에서 행의 셀에 포함 된 값에 대 한 포인터를 가져옵니다.  
  
## <a name="syntax"></a>구문  
  
```  
HRESULT GetColumn (   
    [in]  ULONG   ixTbl,  
    [in]  ULONG   ixCol,  
    [in]  ULONG   rid,  
    [out] ULONG   *pVal  
);  
```  
  
## <a name="parameters"></a>매개 변수  
 `ixTbl`  
 [in] 테이블의 인덱스입니다.  
  
 `ixCol`  
 [in] 테이블의 열 인덱스입니다.  
  
 `rid`  
 [in] 테이블에서 행의 인덱스입니다.  
  
 `pVal`  
 [out] 셀의 값에 대 한 포인터입니다.
