# <a name="imetadatainfo-interface"></a>IMetaDataInfo 인터페이스

## <a name="methods"></a>메서드  
  
|메서드|설명|  
|------------|-----------------|  
|[GetFileMapping 메서드](../../../../docs/framework/unmanaged-api/metadata/imetadatainfo-getfilemapping-method.md)|매핑된 파일과 매핑 종류의 메모리 영역을 가져옵니다.|
