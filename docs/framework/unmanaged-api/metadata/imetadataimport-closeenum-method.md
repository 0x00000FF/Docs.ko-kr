# <a name="imetadataimportcloseenum-method"></a>IMetaDataImport::CloseEnum 메서드

## <a name="syntax"></a>구문  
  
```  
void CloseEnum (  
   [in] HCORENUM hEnum  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `hEnum`  
 [in] 닫습니다 열거자에 대 한 핸들입니다.
