# <a name="imetadatatablesgetnextguid-method"></a>IMetaDataTables::GetNextGuid 메서드
Gets the index of the next GUID value in the current table column.  
  
## <a name="syntax"></a>구문  
  
```cpp  
HRESULT GetNextGuid (  
    [in]  ULONG   ixGuid,  
    [out] ULONG   *pNext  
);  
```  
  
## <a name="parameters"></a>매개 변수  
 `ixGuid`  
 [in] The index value from a GUID table column.  
  
 `pNext`  
 [out] A pointer to the index of the next GUID value.  
  
## <a name="remarks"></a>주의  
 We do not recommend the use of this method, because it does not return consistent results.
