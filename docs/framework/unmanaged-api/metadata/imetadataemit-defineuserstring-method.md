# <a name="imetadataemitdefineuserstring-method"></a>IMetaDataEmit::DefineUserString 메서드
지정된 리터럴 문자열에 대한 메타데이터 토큰을 가져옵니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
HRESULT DefineUserString (
    [in]  LPCWSTR     szString,
    [in]  ULONG       cchString,
    [out] mdString    *pstk
);  
```  
  
## <a name="parameters"></a>매개 변수  
 `szString`  
 【인】 저장할 사용자 문자열입니다.  
  
 `cchString`  
 【인】 에서 와이드 문자의 `szString`수입니다.  
  
 `pstk`  
 【아웃】 할당된 문자열 토큰입니다.
