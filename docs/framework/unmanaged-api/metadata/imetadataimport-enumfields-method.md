# <a name="imetadataimportenumfields-method"></a>IMetaDataImport::EnumFields 메서드
지정한 TypeDef 토큰이 참조하는 형식에 대한 FieldDef 토큰을 열거합니다.  
  
## <a name="syntax"></a>구문  
  
```  
HRESULT EnumFields (   
   [in, out] HCORENUM    *phEnum,   
   [in]      mdTypeDef   cl,   
   [out]     mdFieldDef  rFields[],   
   [in]      ULONG       cMax,   
   [out]     ULONG       *pcTokens  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `phEnum`  
 [out에서] 열거자에 대 한 포인터입니다.  
  
 `cl`  
 [in] 해당 필드를 열거할 수 클래스의 TypeDef 토큰입니다.  
  
 `rFields`  
 [out] 목록 FieldDef 토큰입니다.  
  
 `cMax`  
 [in] `rFields` 배열의 최대 크기입니다.  
  
 `pcTokens`  
 [out] 반환 된 FieldDef 토큰의 실제 수 `rFields`합니다.  
  
## <a name="return-value"></a>반환 값  
  
|HRESULT|설명|  
|-------------|-----------------|  
|`S_OK`|`EnumFields`성공적으로 반환 합니다.|  
|`S_FALSE`|열거 하는 필드가 있습니다. 이 경우 `pcTokens` 은 0입니다.|
