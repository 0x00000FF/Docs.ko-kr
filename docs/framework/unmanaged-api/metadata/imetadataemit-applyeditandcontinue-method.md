# <a name="imetadataemitapplyeditandcontinue-method"></a>IMetaDataEmit::ApplyEditAndContinue 메서드
현재 어셈블리 범위 지정된 된 메타 데이터에서 변경으로 업데이트 합니다.  
  
## <a name="syntax"></a>구문  
  
```  
HRESULT ApplyEditAndContinue (   
    [in]  IUnknown    *pImport  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `pImport`  
 [in] 에 대 한 포인터는 <<!--zzxref:IUnknown --> `IUnknown`> pe (이식 가능) 파일에서 델타 메타 데이터를 나타내는 개체입니다.  
  
 델타 메타 데이터는 모듈의 실제 메타 데이터의 복사본에 대 한 변경 내용을 포함 하는 메타 데이터 블록입니다.
