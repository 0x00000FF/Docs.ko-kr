# <a name="iceegengetsectiondatalen-method"></a>ICeeGen::GetSectionDataLen 메서드
지정된 된 섹션의 길이 가져옵니다.  
  
 이 메서드는 사용 되지 않으며 사용할 수 없습니다.  
  
## <a name="syntax"></a>구문  
  
```  
HRESULT GetSectionDataLen (  
    [in]  HCEESECTION  section,  
    [out] ULONG        *dataLen  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `section`  
 [in] 데이터 섹션 길이가 검색 됩니다.  
  
 `dataLen`  
 [out] 지정된 된 섹션의 반환 된 길이입니다.  
  
## <a name="remarks"></a>설명  
 호출 `GetSectionDataLen` 다른 방법으로 처리 되지 않는 섹션 특별 한 요구 사항이 있는 경우에 합니다.
