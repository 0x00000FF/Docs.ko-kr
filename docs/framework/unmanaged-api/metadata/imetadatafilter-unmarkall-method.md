# <a name="imetadatafilterunmarkall-method"></a>IMetaDataFilter::UnmarkAll 메서드
현재 메타 데이터 범위에서 모든 토큰에서 처리 표시를 제거합니다.  
  
## <a name="syntax"></a>구문  
  
```  
HRESULT UnmarkAll ();  
```  
  
## <a name="parameters"></a>매개 변수  
 `UnmarkAll` 매개 변수를 사용 합니다.
