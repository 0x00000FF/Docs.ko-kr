# <a name="imetadataemitsetrva-method"></a>IMetaDataEmit::SetRVA 메서드
지정된 메서드의 상대 가상 주소를 설정합니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
HRESULT SetRVA (  
    [in]  mdMethodDef  md,
    [in]  ULONG        ulRVA
);  
```  
  
## <a name="parameters"></a>매개 변수  
 `md`  
 【인】 대상 메서드 또는 메서드 구현에 대 한 토큰입니다.  
  
 `ulRVA`  
 【인】 코드 또는 데이터 영역의 주소입니다.
