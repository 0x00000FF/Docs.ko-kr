# <a name="imetadatatables2-interface"></a>IMetaDataTables2 인터페이스
확장 [IMetaDataTables](../../../../docs/framework/unmanaged-api/metadata/imetadatatables-interface.md) 메타 데이터 스트림 작업 하기 위한 메서드를 포함 하도록 합니다.  
  
## <a name="methods"></a>메서드  
  
|메서드|설명|  
|------------|-----------------|  
|[GetMetaDataStorage 메서드](../../../../docs/framework/unmanaged-api/metadata/imetadatatables2-getmetadatastorage-method.md)|크기와 지정된 된 섹션에 저장 된 메타 데이터의 내용을 가져옵니다.|  
|[GetMetaDataStreamInfo 메서드](../../../../docs/framework/unmanaged-api/metadata/imetadatatables2-getmetadatastreaminfo-method.md)|이름, 크기 및 지정된 된 인덱스에서 메타 데이터 스트림의 내용을 가져옵니다.|
