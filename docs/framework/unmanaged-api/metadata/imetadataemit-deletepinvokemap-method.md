# <a name="imetadataemitdeletepinvokemap-method"></a>IMetaDataEmit::DeletePinvokeMap 메서드
지정된 토큰에서 참조하는 개체에 대한 PInvoke 매핑 메타데이터를 삭제합니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
HRESULT DeletePinvokeMap (
    [in]  mdToken     tk
);  
```  
  
## <a name="parameters"></a>매개 변수  
 `tk`  
 【인】 PInvoke 매핑 메타데이터를 삭제할 개체를 나타내는 `mdFieldDef` 또는 `mdMethodDef` 토큰입니다.
