# <a name="imetadatafiltermarktoken-method"></a>IMetaDataFilter::MarkToken 메서드
지정한 메타 데이터 토큰이 처리 되었는지 나타내는 값을 설정 합니다.  
  
## <a name="syntax"></a>구문  
  
```  
HRESULT MarkToken (  
    [in] mdToken   tk  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `tk`  
 [in] 처리 된 것으로 표시 하려면 토큰입니다.
