# <a name="imetadataassemblyimportcloseenum-method"></a>IMetaDataAssemblyImport::CloseEnum 메서드

## <a name="syntax"></a>구문  
  
```  
void CloseEnum (  
    [in] HCORENUM     hEnum  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `hEnum`  
 [in] 닫을 수 있게 열거형 인스턴스입니다.
