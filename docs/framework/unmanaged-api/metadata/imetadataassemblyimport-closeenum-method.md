# <a name="imetadataassemblyimportcloseenum-method"></a>IMetaDataAssemblyImport::CloseEnum 메서드
지정 된 열거형 인스턴스에 대 한 참조를 해제합니다.  
  
## <a name="syntax"></a>구문  
  
```  
void CloseEnum (  
    [in] HCORENUM     hEnum  
);  
```  
  
## <a name="parameters"></a>매개 변수  
 `hEnum`  
 [in] 닫아야 열거형 인스턴스입니다.
