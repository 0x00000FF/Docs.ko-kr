# <a name="imetadataimportgetnestedclassprops-method"></a>IMetaDataImport::GetNestedClassProps 메서드
부모 TypeDef 토큰을 가져옵니다 <xref:System.Type> 지정 된 중첩 형식입니다.  
  
## <a name="syntax"></a>구문  
  
```  
HRESULT GetNestedClassProps (  
   [in]   mdTypeDef      tdNestedClass,  
   [out]  mdTypeDef      *ptdEnclosingClass  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `tdNestedClass`  
 [in] TypeDef 토큰을 나타내는 <xref:System.Type> 부모 클래스를 반환할에 대 한 토큰입니다.  
  
 `ptdEnclosingClass`  
 [out] 에 대 한 TypeDef 토큰에 대 한 포인터는 <xref:System.Type> 하 `tdNestedClass` 에 중첩 되어 있습니다.
