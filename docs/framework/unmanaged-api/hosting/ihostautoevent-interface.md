# <a name="ihostautoevent-interface"></a>IHostAutoEvent 인터페이스

## <a name="methods"></a>메서드  
  
|메서드|설명|  
|------------|-----------------|  
|[Set 메서드](../../../../docs/framework/unmanaged-api/hosting/ihostautoevent-set-method.md)|에서는 현재 `IHostAutoEvent` 인스턴스 신호를 받은 상태로 있습니다.|  
|[Wait 메서드](../../../../docs/framework/unmanaged-api/hosting/ihostautoevent-wait-method.md)|현재 `IHostAutoEvent` 이벤트가 소유 될 때까지 대기 하는 인스턴스 또는 지정된 된 시간 간격이 경과 시간입니다.|
