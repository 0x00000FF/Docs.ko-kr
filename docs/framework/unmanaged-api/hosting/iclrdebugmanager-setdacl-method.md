# <a name="iclrdebugmanagersetdacl-method"></a>ICLRDebugManager::SetDacl 메서드
이 메서드가 구현되지 않았습니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
HRESULT SetDacl (  
    [in] PACL pacl  
);  
```  
  
## <a name="parameters"></a>매개 변수  
 `pacl`  
 진행 ACL (Access Control 목록)에 대 한 포인터입니다.  
  
## <a name="return-value"></a>Return Value  
  
|HRESULT|설명|  
|-------------|-----------------|  
|E_NOTIMPL|메서드가 구현되지 않았습니다.|
