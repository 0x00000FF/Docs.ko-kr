# <a name="iclrdebugmanagersetdacl-method"></a>ICLRDebugManager::SetDacl 메서드
이 메서드가 구현되지 않았습니다.  
  
## <a name="syntax"></a>구문  
  
```  
HRESULT SetDacl (  
    [in] PACL pacl  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `pacl`  
 [in] 액세스 제어 목록 (ACL)에 포인터입니다.  
  
## <a name="return-value"></a>반환 값  
  
|HRESULT|설명|  
|-------------|-----------------|  
|E_NOTIMPL|메서드가 구현 되지 않았습니다.|
