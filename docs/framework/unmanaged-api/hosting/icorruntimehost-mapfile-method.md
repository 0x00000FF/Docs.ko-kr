# <a name="icorruntimehostmapfile-method"></a>ICorRuntimeHost::MapFile 메서드
지정 된 파일을 메모리에 매핑합니다. 이 메서드는 사용되지 않습니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
HRESULT MapFile(  
    [in]  HANDLE    hFile,  
    [out] HMODULE*  hMapAddress  
);  
```  
  
## <a name="parameters"></a>매개 변수  
 `hFile`  
 진행 매핑할 파일의 핸들입니다.  
  
 `hMapAddress`  
 제한이 파일 매핑을 시작할 시작 메모리 주소입니다.
