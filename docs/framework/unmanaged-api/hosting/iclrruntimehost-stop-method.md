# <a name="iclrruntimehoststop-method"></a>ICLRRuntimeHost::Stop 메서드
CLR (공용 언어 런타임)에서 코드의 실행을 중지합니다.  
  
> [!IMPORTANT]
>  이 메서드는 없는 호스트 리소스를 해제, 응용 프로그램 도메인 언로드 또는 스레드를 삭제 합니다. 이러한 리소스를 해제 하는 프로세스를 종료 해야 합니다.  
  
## <a name="syntax"></a>구문  
  
```  
HRESULT Stop();  
```  
  
## <a name="return-value"></a>반환 값  
  
|HRESULT|설명|  
|-------------|-----------------|  
|S_OK|`Stop` 성공적으로 반환 합니다.|  
|HOST_E_CLRNOTAVAILABLE|CLR이 로드 된 프로세스에 또는 CLR 상태인는 관리 코드를 실행 하거나 호출을 처리할 수 없습니다.|  
|HOST_E_TIMEOUT|호출 시간이 초과 되었습니다.|  
|HOST_E_NOT_OWNER|호출자가 잠금을 소유 하지 않습니다.|  
|HOST_E_ABANDONED|이벤트가 차단 된 스레드가 취소 된 또는 파이버를 대기 하 고 있습니다.|  
|E_FAIL|알 수 없는 치명적인 오류가 발생 했습니다. 메서드가 E_FAIL을 반환 하는 경우 CLR은 프로세스 내에서 사용할 수 없습니다. 메서드를 호스트 하는 데 대 한 후속 호출 HOST_E_CLRNOTAVAILABLE를 반환 합니다.|
