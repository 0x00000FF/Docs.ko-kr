# <a name="ihosttaskmanagergetstackguarantee-method"></a>IHostTaskManager::GetStackGuarantee 메서드

## <a name="syntax"></a>구문  
  
```  
HRESULT GetStackGuarantee(  
    [out] ULONG *pGuarantee
```  

 `pGuarantee`
