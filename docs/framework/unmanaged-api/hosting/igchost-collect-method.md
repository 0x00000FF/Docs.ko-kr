# <a name="igchostcollect-method"></a>IGCHost::Collect 메서드
현재 가비지 컬렉션의 상태에 관계 없이 지정된 된 세대에 대 한 컬렉션을 수행 합니다.  
  
## <a name="syntax"></a>구문  
  
```  
HRESULT Collect (  
    [in] LONG Generation  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `Generation`  
 [in] 가비지 수집을 수행 하기를 생성 합니다. 값이-1은 모든 세대 가비지 수집을 될 예정 나타냅니다.
