# <a name="corprfgcreason-enumeration"></a>COR_PRF_GC_REASON 열거형
가비지 컬렉션이 수행되는 이유를 나타냅니다.  
  
## <a name="syntax"></a>구문  
  
```  
typedef enum {  
    COR_PRF_GC_INDUCED = 1,  
    COR_PRF_GC_OTHER = 0  
} COR_PRF_GC_REASON;  
```  
  
## <a name="members"></a>멤버  
  
|멤버|설명|  
|------------|-----------------|  
|`COR_PRF_GC_INDUCED`|가비지 수집이 발생 하 여 한 <xref:System.GC.Collect%2A> 메서드.|  
|`COR_PRF_GC_OTHER`|이유 지정 되지 않았습니다.|
