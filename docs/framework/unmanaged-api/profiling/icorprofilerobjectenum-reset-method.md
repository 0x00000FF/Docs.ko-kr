# <a name="icorprofilerobjectenumreset-method"></a>ICorProfilerObjectEnum::Reset 메서드
Moves this enumerator's cursor to the starting position of the sequence.  
  
## <a name="syntax"></a>구문  
  
```cpp  
HRESULT Reset ();  
```
