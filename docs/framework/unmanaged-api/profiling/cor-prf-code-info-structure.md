# <a name="corprfcodeinfo-structure"></a>COR_PRF_CODE_INFO 구조체
메모리 내에 저장된 연속하는 네이티브 코드 블록 하나를 나타냅니다.  
  
## <a name="syntax"></a>구문  
  
```  
typedef struct _COR_PRF_CODE_INFO {  
    UINT_PTR startAddress;  
    SIZE_T size;  
} COR_PRF_CODE_INFO;  
```  
  
## <a name="members"></a>멤버  
  
|멤버|설명|  
|------------|-----------------|  
|`startAddress`|코드의 연속 블록의 시작 주소입니다.|  
|`size`|블록의 크기입니다.|
