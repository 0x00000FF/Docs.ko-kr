# <a name="icordebugchainenumnext-method"></a>ICorDebugChainEnum::Next 메서드
현재 위치부터 시작 하는 열거형에서 지정 된 ICorDebugChain 인스턴스 수를 가져옵니다.  
  
## <a name="syntax"></a>구문  
  
```  
HRESULT Next (  
    [in] ULONG  celt,  
    [out, size_is(celt), length_is(*pceltFetched)]  
        ICorDebugChain *chains[],  
    [out] ULONG *pceltFetched  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `celt`  
 [in] 수가 `ICorDebugChain` 인스턴스를 검색할 수 있습니다.  
  
 `chains`  
 [out] 각각 가리키는 포인터의 배열은 `ICorDebugChain` 체인을 나타내는 개체입니다.  
  
 `pceltFetched`  
 [out] 수에 대 한 포인터 `ICorDebugChain` 실제로 반환 된 인스턴스. 이 값은 null 일 수 있으면 `celt` 하나입니다.
