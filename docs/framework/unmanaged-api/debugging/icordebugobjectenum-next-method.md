# <a name="icordebugobjectenumnext-method"></a>ICorDebugObjectEnum::Next 메서드
현재 위치에서 시작하여 열거형에서 지정된 수의 개체 수의 상대 가상 주소(RAS)를 가져옵니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
HRESULT Next (  
    [in] ULONG celt,  
    [out, size_is(celt), length_is(*pceltFetched)]
        CORDB_ADDRESS objects[],  
    [out] ULONG *pceltFetched  
);  
```  
  
## <a name="parameters"></a>매개 변수  
 `celt`  
 [in] 검색할 개체 수입니다.  
  
 `objects`  
 【아웃】 CORDB_ADDRESS 개체를 가리키는 각각의 포인터 배열입니다.  
  
 `pceltFetched`  
 【아웃】 실제로 반환된 개체 수에 대한 포인터입니다. 이 `celt` 값은 null일 수 있습니다.
