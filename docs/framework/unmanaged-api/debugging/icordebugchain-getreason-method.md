# <a name="icordebugchaingetreason-method"></a>ICorDebugChain::GetReason 메서드
이 호출 체인이 발생 한 이유를 가져옵니다.  
  
## <a name="syntax"></a>구문  
  
```  
HRESULT GetReason (  
    [out] CorDebugChainReason *pReason  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `pReason`  
 [out] 이 호출 체인이 발생 한 이유를 나타내는 CorDebugChainReason 열거형의 값 (비트 조합)에 대 한 포인터입니다.
