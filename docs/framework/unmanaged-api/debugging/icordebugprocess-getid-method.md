# <a name="icordebugprocessgetid-method"></a>ICorDebugProcess::GetID 메서드
프로세스의 운영 체제 (OS) ID를 가져옵니다.  
  
## <a name="syntax"></a>구문  
  
```  
HRESULT GetID([out] DWORD *pdwProcessId);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `pdwProcessId`  
 [out] 프로세스의 고유 ID입니다.
