# <a name="debugging-unmanaged-api-reference"></a>디버깅(관리되지 않는 API 참조)
디버깅 API를 통해 디버거가 CLR(공용 언어 런타임) 환경에서 실행되는 코드를 디버그할 수 있습니다. 디버그할 코드는 CLR이 지원하는 코드 형식일 수 있습니다.
