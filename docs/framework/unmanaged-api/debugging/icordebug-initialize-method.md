# <a name="icordebuginitialize-method"></a>ICorDebug::Initialize 메서드
초기화는 `ICorDebug` 개체입니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
HRESULT Initialize ();  
```  
  
## <a name="remarks"></a>주의  
 디버거는 만든 시간에 `Initialize`를 호출 하 여 디버깅 서비스를 초기화 해야 합니다. `ICorDebug`에서 다른 메서드를 호출 하기 전에이 메서드를 호출 해야 합니다.
