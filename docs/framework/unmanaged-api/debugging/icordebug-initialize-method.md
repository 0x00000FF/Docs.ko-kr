# <a name="icordebuginitialize-method"></a>ICorDebug::Initialize 메서드
초기화는 `ICorDebug` 개체입니다.  
  
## <a name="syntax"></a>구문  
  
```  
HRESULT Initialize ();  
```  
  
## <a name="remarks"></a>설명  
 디버거를 호출 해야 `Initialize` 디버깅 초기화에 시간이 서비스를 만들 때. 이 메서드를 다른 메서드 전에 호출 해야 `ICorDebug` 호출 됩니다.
