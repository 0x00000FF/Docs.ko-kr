# <a name="icordebugvalue2-interface"></a>ICorDebugValue2 인터페이스
"ICorDebugType" 개체에 대 한 지원을 제공 하는 "ICorDebugValue" 인터페이스를 확장 합니다.  
  
## <a name="methods"></a>메서드  
  
|메서드|설명|  
|------------|-----------------|  
|[GetExactType 메서드](../../../../docs/framework/unmanaged-api/debugging/icordebugvalue2-getexacttype-method.md)|한 인터페이스 포인터를 가져옵니다는 `ICorDebugType` 나타내는 개체를 <xref:System.Type> 이 값의 합니다.|  
  
## <a name="remarks"></a>설명  
  
> [!NOTE]
>  이 인터페이스는 크로스 시스템 또는 크로스 프로세스 원격 호출을 지원하지 않습니다.
