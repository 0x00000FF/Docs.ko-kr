# <a name="cordebuggenerationtypes-enumeration"></a>CorDebugGenerationTypes 열거형
관리되는 힙에 대한 메모리 영역 생성을 지정합니다.  
  
## <a name="syntax"></a>구문  
  
```  
typedef enum CorDebugGenerationTypes {  
    CorDebug_Gen0 = 0,  
    CorDebug_Gen1 = 1,  
    CorDebug_Gen2 = 2,  
    CorDebug_LOH  = 3,  
} CorDebugRegionTypes;  
```

|`CorDebug_LOH`|대형 개체 힙입니다.|
