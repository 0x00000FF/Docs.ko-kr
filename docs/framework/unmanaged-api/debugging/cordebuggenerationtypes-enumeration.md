# <a name="cordebuggenerationtypes-enumeration"></a>CorDebugGenerationTypes 열거형
관리되는 힙에 대한 메모리 영역 생성을 지정합니다.  
  
## <a name="syntax"></a>구문  
  
```  
typedef enum CorDebugGenerationTypes {  
    CorDebug_Gen0 = 0,  
    CorDebug_Gen1 = 1,  
    CorDebug_Gen2 = 2,  
    CorDebug_LOH  = 3,  
} CorDebugRegionTypes;  
```  
  
## <a name="members"></a>멤버  
  
|멤버 이름|설명|  
|-----------------|-----------------|  
|`CorDebug_Gen0`|0세대.|  
|`CorDebug_Gen1`|1세대.|  
|`CorDebug_Gen2`|2세대.|  
|`CorDebug_LOH`|큰 개체 힙입니다.|
