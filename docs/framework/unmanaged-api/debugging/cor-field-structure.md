# <a name="corfield-structure"></a>COR_FIELD 구조체
개체의 필드에 대한 정보를 포함합니다.  
  
## <a name="syntax"></a>구문  
  
```  
typedef struct COR_FIELD{  
    mdFieldDef token;  
    ULONG32 offset;  
    COR_TYPEID id;  
    CorElementType fieldType;  
} COR_FIELD;  
```  
  
## <a name="members"></a>멤버  
  
|멤버|설명|  
|------------|-----------------|  
|`token`|`mdFieldDef` 필드 정보를 가져오는 데 사용할 수 있는 토큰입니다.|  
|`offset`|개체의 필드 데이터의 바이트 오프셋입니다.|  
|`id`|A [COR_TYPEID](../../../../docs/framework/unmanaged-api/debugging/cor-typeid-structure.md) 이 필드의 유형을 식별 하는 값입니다.|  
|`fieldType`|필드의 유형을 나타내는 CorElementType 열거형 값입니다.|
