# <a name="icordebugappdomainattach-method"></a>ICorDebugAppDomain::Attach 메서드
응용 프로그램 도메인에 디버거를 연결 합니다.  
  
## <a name="syntax"></a>구문  
  
```  
HRESULT Attach ();  
```  
  
## <a name="remarks"></a>설명  
 디버거는 이벤트를 수신 하는 응용 프로그램 도메인의 디버깅을 사용 하도록 응용 프로그램 도메인에 연결할 수 있어야 합니다.
