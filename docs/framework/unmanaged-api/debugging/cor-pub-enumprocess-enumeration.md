# <a name="corpubenumprocess-enumeration"></a>COR_PUB_ENUMPROCESS 열거형
열거할 프로세스 형식을 식별합니다.  
  
## <a name="syntax"></a>구문  
  
```  
typedef enum {  
    COR_PUB_MANAGEDONLY    = 0x00000001  
} COR_PUB_ENUMPROCESS;  
```  
  
## <a name="members"></a>멤버  
  
|멤버 이름|설명|  
|-----------------|-----------------|  
|`COR_PUB_MANAGEDONLY`|관리 되는 프로세스입니다.|  
  
## <a name="remarks"></a>설명  
 관리 되지 않는 디버깅 API의 현재 버전 관리 프로세스를 열거 합니다.
