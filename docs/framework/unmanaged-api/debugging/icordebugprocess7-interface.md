# <a name="icordebugprocess7-interface"></a>ICorDebugProcess7 인터페이스
[.NET Framework 4.5.2 이상 버전에서 지원됨]  
  
 대상 프로세스에서 디버거가 메모리 내 메타데이터 업데이트를 처리하도록 구성하는 메서드를 제공합니다.  
  
## <a name="methods"></a>메서드  
  
|메서드|설명|  
|------------|-----------------|  
|[SetWriteableMetadataUpdateMode 메서드](../../../../docs/framework/unmanaged-api/debugging/icordebugprocess7-setwriteablemetadataupdatemode-method.md)|대상 프로세스 내에서 디버거가 메타데이터에 대한 메모리 내 업데이트 처리 방법을 결정하는 값을 설정합니다.|
