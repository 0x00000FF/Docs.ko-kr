# <a name="icordebugcode4-interface"></a>ICorDebugCode4 인터페이스
디버거를 지역 변수 및 함수에 인수를 열거 하는 메서드를 제공 합니다.  
  
## <a name="methods"></a>메서드  
  
|메서드|설명|  
|------------|-----------------|  
|[EnumerateVariableHomes 메서드](../../../../docs/framework/unmanaged-api/debugging/icordebugcode4-enumeratevariablehomes-method.md)|함수에서 지역 변수 및 인수에는 열거자를 가져옵니다.|
