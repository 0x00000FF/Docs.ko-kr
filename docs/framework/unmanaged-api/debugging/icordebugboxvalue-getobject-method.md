# <a name="icordebugboxvaluegetobject-method"></a>ICorDebugBoxValue::GetObject 메서드
Boxed 값을 가져옵니다.  
  
## <a name="syntax"></a>구문  
  
```  
HRESULT GetObject (  
    [out] ICorDebugObjectValue **ppObject  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `ppObject`  
 [out] Boxed 값을 나타내는 ICorDebugObjectValue 개체의 주소에 대 한 포인터입니다.
