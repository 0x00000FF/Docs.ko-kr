# <a name="icordebughandlevaluedispose-method"></a>ICorDebugHandleValue::Dispose 메서드
명시적으로 인터페이스 포인터를 해제 하지 않고이 ICorDebugHandleValue 개체에서 참조 하는 핸들을 해제 합니다.  
  
## <a name="syntax"></a>구문  
  
```  
HRESULT Dispose ();  
```
