# <a name="icordebugenumgetcount-method"></a>ICorDebugEnum::GetCount 메서드
열거형에서 항목의 수를 가져옵니다.  
  
## <a name="syntax"></a>구문  
  
```  
HRESULT GetCount (  
    [out] ULONG *pcelt  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `pcelt`  
 [out] 열거형에는 항목 수에 대 한 포인터입니다.
