# <a name="icordebugobjectvaluegetcontext-method"></a>ICorDebugObjectValue::GetContext 메서드
이 버전의 .NET Framework에서는 `GetContext` 구현 되어 있지 않습니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
HRESULT GetContext (  
    [out] ICorDebugContext   **ppContext  
);  
```
