# <a name="icordebugarrayvaluegetcount-method"></a>ICorDebugArrayValue::GetCount 메서드
배열에 있는 요소의 총 수를 가져옵니다.  
  
## <a name="syntax"></a>구문  
  
```  
HRESULT GetCount (  
    [out] ULONG32 *pnCount  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `pnCount`  
 [out] 배열에 있는 요소의 총 수에 대 한 포인터입니다.
