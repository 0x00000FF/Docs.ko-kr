# <a name="icorpublishenumgetcount-method"></a>ICorPublishEnum::GetCount 메서드
열거형의 항목 수를 가져옵니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
HRESULT GetCount (  
    [out] ULONG   *pcelt  
);  
```  
  
## <a name="parameters"></a>매개 변수  
 `pcelt`  
 제한이 열거형의 항목 수에 대 한 포인터입니다.
