# <a name="icordebugevalnewstring-method"></a>ICorDebugEval::NewString 메서드
지정 된 내용으로 새 문자열 인스턴스를 할당합니다.  
  
## <a name="syntax"></a>구문  
  
```  
HRESULT NewString (  
    [in] LPCWSTR   string  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `string`  
 [in] 문자열에 대 한 내용에 대 한 포인터입니다.  
  
## <a name="remarks"></a>설명  
 문자열은 항상 스레드가 현재 실행 중인 응용 프로그램 도메인에 만들어집니다.
