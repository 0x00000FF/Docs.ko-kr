# <a name="icordebugheapvalue2-interface"></a>ICorDebugHeapValue2 인터페이스

CLR (공용 언어 런타임) 처리에 대 한 지원을 제공 하는 ICorDebugHeapValue의 확장입니다.  
  
## <a name="methods"></a>메서드  
  
|메서드|설명|  
|------------|-----------------|  
|[CreateHandle 메서드](../../../../docs/framework/unmanaged-api/debugging/icordebugheapvalue2-createhandle-method.md)|이 대 한 지정 된 형식의 핸들을 만들고 `ICorDebugHeapValue2` 개체입니다.|  
  
## <a name="remarks"></a>설명
