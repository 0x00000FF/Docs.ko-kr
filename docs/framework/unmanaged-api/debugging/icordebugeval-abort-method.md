# <a name="icordebugevalabort-method"></a>ICorDebugEval::Abort 메서드
이 ICorDebugEval 개체에서 현재 수행 중인 계산을 중단 합니다.  
  
## <a name="syntax"></a>구문  
  
```  
HRESULT Abort ();  
```  
  
## <a name="remarks"></a>설명  
 평가 중첩 되어 있고 한 최신 없는 경우는 `Abort` 메서드가 실패할 수 있습니다.
