# <a name="icordebugmanagedcallbackloadassembly-method"></a>ICorDebugManagedCallback::LoadAssembly 메서드
공용 언어 런타임 (CLR) 어셈블리 성공적으로 로드 된 디버거에 알립니다.  
  
## <a name="syntax"></a>구문  
  
```  
HRESULT LoadAssembly (  
    [in] ICorDebugAppDomain *pAppDomain,  
    [in] ICorDebugAssembly  *pAssembly  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `pAppDomain`  
 [in] 어셈블리가 로드 된 응용 프로그램 도메인을 나타내는 ICorDebugAppDomain 개체에 대 한 포인터입니다.  
  
 `pAssembly`  
 [in] 어셈블리를 나타내는 ICorDebugAssembly 개체에 대 한 포인터입니다.
