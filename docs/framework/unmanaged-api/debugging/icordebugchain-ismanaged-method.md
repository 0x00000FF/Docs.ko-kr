# <a name="icordebugchainismanaged-method"></a>ICorDebugChain::IsManaged 메서드
이 체인 관리 되는 코드를 실행 하 고 있는지 여부를 나타내는 값을 가져옵니다.  
  
## <a name="syntax"></a>구문  
  
```  
HRESULT IsManaged (  
    [out] BOOL               *pManaged  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `pManaged`  
 [out] `true` 이 체인 관리 되는 코드를 실행 하는 경우 이렇게 하지 않으면 `false`합니다.
