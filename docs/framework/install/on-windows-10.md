# <a name="install-the-net-framework-on-windows-10-and-windows-server-2016"></a>Windows 10 및 Windows Server 2016에서.NET Framework를 설치 합니다.

.NET Framework가 Windows에서 많은 응용 프로그램을 실행 하려면 필요 합니다. 이 문서의 지침에서는 필요한.NET Framework 버전을 설치 하면 해결 됩니다. [.NET Framework 4.7.1](https://www.microsoft.com/download/details.aspx?id=56115&desc=dotnet47) 은 사용 가능한 최신 버전입니다.

응용 프로그램을 실행 하 고 다음 예와 비슷한 컴퓨터에는 대화 상자를 표시 한 후이 페이지에 도달한 메시지가 수 있습니다.

![이 응용 프로그램을 시작할 수 없습니다.](./media/this-application-could-not-be-started.png)

## <a name="net-framework-471"></a>.NET framework 4.7.1

.NET Framework 4.7.1으로 포함 되어 있습니다.

* [Windows 10 년 작성자 업데이트 (버전 1709)](https://www.microsoft.com/software-download/windows10)
* [Windows Server 2016 (버전 1709)](https://docs.microsoft.com/windows-server/get-started/get-started-with-1709)

> [!div class="button"]
[.NET Framework 4.7.1 다운로드](https://www.microsoft.com/net/download/thank-you/net471?utm_source=ms-docs&utm_medium=referral)

[.NET Framework 4.7.1](https://www.microsoft.com/download/details.aspx?id=56115&desc=dotnet47) 는 4.7.1 통해.NET Framework 4.0 용으로 작성 된 응용 프로그램을 실행 하는 데 사용할 수 있습니다.

설치할 수 있습니다는 [.NET Framework 4.7.1](https://www.microsoft.com/en-us/download/details.aspx?id=56115&desc=dotnet47) 에:

* Windows 10 작성자 업데이트 (버전 1703)
* Windows 10 Anniversary 업데이트 (버전 1607)
* Windows Server 2016

.NET Framework 4.7.1에서 지원 되지 않습니다.

* Windows 10 1507
* Windows 10 1511

Windows 10 1507 또는 1511을 사용 하는 경우.NET Framework 4.7.1를 설치 하려면 먼저 Windows 10 이후 버전으로 업그레이드 해야 합니다.

## <a name="net-framework-461"></a>.NET Framework 4.6.1

[.NET Framework 4.6.1](https://www.microsoft.com/download/details.aspx?id=49981) 최신 지원 되는.NET Framework 버전에서 Windows 10 1507 및 1511 됩니다.

.NET Framework 4.6.1 4.6.1 통해.NET Framework 4.0 용으로 작성 된 앱을 지원 합니다.

## <a name="net-framework-35"></a>.NET Framework 3.5

지침에 따라 [.NET Framework 3.5를 Windows 10에](dotnet-35-windows-10.md) 설치합니다.

.NET Framework 3.5 3.5 통해.NET Framework 1.0 용으로 작성 된 앱을 지원 합니다.

## <a name="additional-information"></a>추가 정보

.NET framework 4.x 버전은 이전 버전의 내부 업데이트입니다. 다음은 의미합니다.

- .NET Framework의 버전을 하나만 지정할 수 있습니다 4.x 컴퓨터에 설치 합니다.

- 이후 버전이 이미 설치 되어 있으면 컴퓨터에는 이전 버전의.NET Framework를 설치할 수 없습니다.

- 해당 버전을 통해.NET Framework 4.0 용으로 작성 된 응용 프로그램을 실행할 4.x 버전의.NET Framework를 사용할 수 있습니다. 예를 들어.NET Framework 4.7 4.7 통해.NET Framework 4.0 용으로 작성 된 응용 프로그램을 실행 데 사용할 수 있습니다. 작성 된 응용 프로그램 실행 4.0과 함께 시작 하는.NET Framework의 모든 버전 최신 버전 (.NET Framework 4.7.1)를 사용할 수 있습니다.

목록이 다운로드 하기 위해.NET Framework의 모든 버전에 대 한 참조는 [.NET 다운로드](https://www.microsoft.com/net/download?utm_source=ms-docs&utm_medium=referral) 페이지.
