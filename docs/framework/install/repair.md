# <a name="repair-the-net-framework"></a>.NET Framework 복구

경우에 따라 .NET Framework 설치가 손상될 수 있고 복구가 필요합니다. 앱을 시작한 직후 앱이 충돌하거나 최신 .NET Framework 버전을 설치할 수 없는 경우가 여기에 해당합니다.

[.NET Framework 복구 도구](http://download.microsoft.com/download/2/B/D/2BDE5459-2225-48B8-830C-AE19CAF038F1/NetFxRepairTool.exe)를 사용하여 .NET Framework 설치를 복구할 수 있습니다.

.NET Framework를 복구한 후에도 앱이 시작되지 않으면 앱에 문제가 있는 것일 수 있습니다. 이 경우 앱 게시자에게 문의해야 합니다.
