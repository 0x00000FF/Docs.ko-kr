# <a name="install-the-net-framework-on-windows-8"></a>Windows 8에 .NET Framework 설치

Windows 8에서 앱을 실행하려면 .NET Framework가 필요할 수 있습니다. 다음 지침이 도움이 될 수 있습니다. 지침은 필요한 .NET Framework 버전에 따라 다릅니다.

## <a name="net-framework-47"></a>.NET Framework 4.7

Windows 8에는 [.NET Framework 4.7](https://www.microsoft.com/net/download/framework)을 설치할 수 있습니다.

.NET Framework 4.7은 [Windows 10 크리에이터스 업데이트](https://www.microsoft.com/software-download/windows10)에 포함됩니다.

.NET Framework 4.7은 .NET Framework 4.0 이상용으로 빌드된 앱을 지원합니다. 다른 버전의.NET Framework는 [Download .NET Framework](https://www.microsoft.com/net/download/framework)(.NET Framework 다운로드) 웹 페이지에서 사용할 수 있습니다.

## <a name="net-framework-35"></a>.NET Framework 3.5

지침에 따라 [.NET Framework 3.5를 Windows 10에](dotnet-35-windows-10.md) 설치합니다.

.NET Framework 3.5는 .NET Framework 1.0~3.5용으로 빌드된 앱을 지원합니다.

## <a name="help"></a>도움말

설치된 .NET Framework의 정확한 버전을 확인할 수 없는 경우 [Microsoft에 지원을 문의](mailto:dotnet-install-help@microsoft.com?subject=Help installing .NET Framework)할 수 있습니다.
