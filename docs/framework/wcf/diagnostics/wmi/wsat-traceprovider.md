# WSAT_TraceProvider
WSAT\_TraceProvider  
  
## 구문  
  
```  
class WSAT_TraceProvider : EventTrace  
{  
};  
```  
  
## 메서드  
 WSAT\_TraceProvider 클래스는 EventTrace에서 상속되지만 추가 메서드나 속성을 정의하지 않습니다.  
  
## 요구 사항  
  
|MOF|Servicemodel.mof에 선언되어 있습니다.|  
|---------|----------------------------------|  
|네임스페이스|root\\ServiceModel에 정의되어 있습니다.|
