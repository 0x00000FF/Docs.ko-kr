# TraceListenerArgument
TraceListenerArgument  
  
## 구문  
  
```  
class TraceListenerArgument  
{  
  string Name;  
  string Value;  
};  
```  
  
## 메서드  
 TraceListenerArgument 클래스는 메서드를 정의하지 않습니다.  
  
## 속성  
 TraceListenerArgument 클래스에는 다음 속성이 있습니다.  
  
### 이름  
 데이터 형식: string  
  
 액세스 형식: 읽기 전용  
  
 인수의 이름입니다.  
  
### 값  
 데이터 형식: string  
  
 액세스 형식: 읽기 전용  
  
 인수 값입니다.
