# Behavior 클래스
## 구문  
  
```  
class Behavior  
{  
  string Type;  
};  
```  
  
## 메서드  
 Behavior 클래스는 메서드를 정의하지 않습니다.  
  
## 속성  
 Behavior 클래스에는 다음과 같은 속성이 있습니다.  
  
### 형식  
 데이터 형식: string  
  
 액세스 형식: 읽기 전용  
  
 동작의 형식입니다.  
  
## 요구 사항  
  
|MOF|Servicemodel.mof에 선언되어 있습니다.|  
|---------|----------------------------------|  
|네임스페이스|root\\ServiceModel에 정의되어 있습니다.|
