# TraceListener
TraceListener  
  
## 구문  
  
```  
class TraceListener  
{  
  string Name;  
  TraceListenerArgument TraceListenerArguments[];  
};  
```  
  
## 메서드  
 TraceListener 클래스는 메서드를 정의하지 않습니다.  
  
## 속성  
 TraceListener 클래스에는 다음 속성이 있습니다.  
  
### 이름  
 데이터 형식: string  
  
 액세스 형식: 읽기 전용  
  
 추적 수신기의 이름입니다.  
  
### TraceListenerArguments  
 데이터 형식: TraceListenerArgument 배열  
  
 액세스 형식: 읽기 전용  
  
 추적 수신기의 인수입니다.
