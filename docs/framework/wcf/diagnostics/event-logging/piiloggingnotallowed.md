# PiiLoggingNotAllowed
ID: 108  
  
 심각도: 오류  
  
 범주: 추적  
  
## 설명  
 이 이벤트는 기록할 알려진 PII가 없음을 나타냅니다.  알려진 PII 로깅이 허용되지 않습니다.  알려진 PII 로깅을 허용하려면 Machine.config에서 "enableLoggingKnownPii"를 `true`로 설정합니다.  이 이벤트는 프로세스 이름과 프로세스 ID를 표시합니다.
