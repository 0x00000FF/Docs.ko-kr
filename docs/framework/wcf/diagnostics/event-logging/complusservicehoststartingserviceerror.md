# <a name="complusservicehoststartingserviceerror"></a>ComPlusServiceHostStartingServiceError
ID: 130  
  
 심각도: Error  
  
 범주: ServiceModel  
  
## <a name="description"></a>설명  
 이 이벤트는 COM+ 서비스를 시작하는 동안 오류가 발생했음을 나타냅니다. 이벤트는 AppID, ClsID, 서로게이트, 예외, 프로세스 이름 및 프로세스 ID를 나열합니다.
