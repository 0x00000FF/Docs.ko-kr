# PerformanceCounterInitializationFailure
ID: 149  
  
 심각도: 오류  
  
 범주: TransactionBridge  
  
## 설명  
 이 이벤트는 성능 카운터를 초기화하지 못했음을 나타냅니다.이 이벤트는 카운터 이름, 예외, 프로세스 이름 및 프로세스 ID를 나열합니다.
