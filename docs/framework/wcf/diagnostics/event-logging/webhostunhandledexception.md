# WebHostUnhandledException
ID: 109  
  
 심각도: 오류  
  
 범주: ServiceModel  
  
## 설명  
 이 이벤트는 WebHost에서 처리되지 않은 예외가 발생했음을 나타냅니다.이벤트는 발신자 정보, 예외, 프로세스 이름 및 프로세스 ID를 표시합니다.
