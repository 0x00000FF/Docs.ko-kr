# ThumbPrintNotValidated
ID: 153  
  
 심각도: 오류  
  
 범주: TransactionBridge  
  
## 설명  
 이 이벤트는 지정한 지문의 ID 인증서를 확인할 수 없음을 나타냅니다.  이 이벤트는 프로세스 이름과 프로세스 ID를 표시합니다.
