# <a name="webhostfailedtoprocessrequest"></a>WebHostFailedToProcessRequest
ID: 111  
  
 심각도: Error  
  
 범주: ServiceModel  
  
## <a name="description"></a>설명  
 이 이벤트는 WebHost가 요청을 처리하지 못했음을 나타냅니다. 이벤트는 발신자 정보, 예외, 프로세스 이름 및 프로세스 ID를 표시합니다.
