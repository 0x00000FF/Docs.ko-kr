# <a name="webhostfailedtolisten"></a>WebHostFailedToListen
Id: 112  
  
 심각도: 오류  
  
 범주: ServiceModel  
  
## <a name="description"></a>설명  
 이 이벤트는 특정 URL을 수신 대기하는 동안 발생한 오류를 나타냅니다. 이 작업자 프로세스가 종료됩니다. 이 이벤트는 예외, 프로세스 이름 및 프로세스 ID를 표시합니다.
