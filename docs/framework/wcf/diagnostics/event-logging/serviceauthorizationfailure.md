# <a name="serviceauthorizationfailure"></a>ServiceAuthorizationFailure
ID: 168  
  
 심각도: Error  
  
 범주: SecurityAudit  
  
## <a name="description"></a>설명  
 이 이벤트는 서비스 인증 실패를 나타냅니다. 이벤트에서는 서비스, 동작, 클라이언트 ID, AuthorizationContext, 동작 ID 및 ServiceAuthorizationManager를 나열합니다.
