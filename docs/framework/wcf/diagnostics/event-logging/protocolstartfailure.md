# <a name="protocolstartfailure"></a>ProtocolStartFailure
ID: 143  
  
 심각도: 오류  
  
 범주: TransactionBridge  
  
## <a name="description"></a>설명  
 이 이벤트는 WS-Atomic Transaction 프로토콜 서비스가 시작되지 않았음을 나타냅니다. 따라서 WS-Atomic Transaction 기능을 사용하도록 설정할 수 없습니다. 이 이벤트는 프로토콜 ID, 프로토콜 이름, 예외, 프로세스 이름 및 프로세스 ID를 나열합니다. 이어야 합니다.
