# <a name="protocolstopfailure"></a>ProtocolStopFailure
Id: 147  
  
 심각도: 오류  
  
 범주: TransactionBridge  
  
## <a name="description"></a>설명  
 이 이벤트는 WS-Atomic Transaction 프로토콜 서비스가 중지되지 않았음을 나타냅니다. 이 이벤트는 프로토콜 ID, 프로토콜 이름, 예외, 프로세스 이름 및 프로세스 ID를 나열합니다.
