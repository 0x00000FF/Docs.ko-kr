# <a name="wasdisconnected"></a>WasDisconnected
ID: 162  
  
 심각도: Error  
  
 범주: SMSvcHost  
  
## <a name="description"></a>설명  
 이 이벤트는 WAS(Windows Process Activation)의 연결이 끊어졌음을 나타냅니다. 이 이벤트는 HRESULT, 프로세스 이름 및 프로세스 ID를 나열합니다.
