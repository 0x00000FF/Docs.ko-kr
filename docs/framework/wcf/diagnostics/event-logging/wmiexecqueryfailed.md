# <a name="wmiexecqueryfailed"></a>WmiExecQueryFailed
ID: 124  
  
 심각도: Error  
  
 범주: ServiceModel  
  
## <a name="description"></a>설명  
 이 이벤트는 WMI ExecQuery 요청이 처리되지 않았음을 나타냅니다. 이 이벤트는 예외, 프로세스 이름 및 프로세스 ID를 나열합니다.
