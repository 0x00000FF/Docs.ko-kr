# <a name="messagequeueduplicatedsocketleak"></a>MessageQueueDuplicatedSocketLeak
ID: 165  
  
 심각도: Error  
  
 범주: SMSvcHost  
  
## <a name="description"></a>설명  
 이 이벤트는 중복 소켓을 발송하는 동안 발생한 오류를 나타냅니다. 현재 이 핸들은 프로세스에서 누락됩니다. 이 이벤트는 소스, 예외, 프로세스 이름 및 프로세스 ID를 표시합니다.
