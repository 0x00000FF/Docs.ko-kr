# UnknownListenerAdapterError
ID: 161  
  
 심각도: 오류  
  
 범주: SMSvcHost  
  
## 설명  
 이 이벤트는 WAS\(Windows Process Activation Service\) 알림을 처리하는 동안 수신기 어댑터에서 예상치 못한 오류가 발생했음을 나타냅니다.  프로세스가 종료됩니다.  이 이벤트는 프로토콜, 예외, 프로세스 이름 및 프로세스 ID를 표시합니다.
