# WmiRegistrationFailed
Id: 126  
  
 심각도: 오류  
  
 범주: ServiceModel  
  
## 설명  
 이 이벤트는 WMI 공급자가 등록되지 않았음을 나타냅니다.또한 이 이벤트는 WMI 개체, 오류, 프로세스 이름 및 프로세스 ID를 표시합니다.
