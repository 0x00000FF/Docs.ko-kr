# <a name="wmiregistrationfailed"></a>WmiRegistrationFailed
ID: 126  
  
 심각도: Error  
  
 범주: ServiceModel  
  
## <a name="description"></a>설명  
 이 이벤트는 WMI 공급자가 등록되지 않았음을 나타냅니다. 또한 이 이벤트는 WMI 개체, 오류, 프로세스 이름 및 프로세스 ID를 표시합니다.
