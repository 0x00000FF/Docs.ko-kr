# <a name="securitynegotiationfailure"></a>SecurityNegotiationFailure
ID: 172  
  
 심각도: Error  
  
 범주: SecurityAudit  
  
## <a name="description"></a>설명  
 이 이벤트는 보안 협상이 실패했음을 나타냅니다. 해당 이벤트는 서비스, 동작, 클라이언트 ID, 동작 ID 및 협상을 나열합니다.
