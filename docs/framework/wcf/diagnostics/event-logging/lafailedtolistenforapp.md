# <a name="lafailedtolistenforapp"></a>LAFailedToListenForApp
ID: 160  
  
 심각도: 오류  
  
 범주 SMSvcHost  
  
## <a name="description"></a>Description  
 이 이벤트는 지정된 사이트를 수신 대기하는 동안 지정된 프로토콜의 지정된 Activation Service에서 오류가 발생했으므로 사이트에 대한 해당 프로토콜을 임시로 사용할 수 없음을 나타냅니다. 이 이벤트는 URL, 상태, 예외, 프로세스 이름 및 프로세스 ID를 나열합니다.
