# <a name="fatalunexpectedstatemachineevent"></a>FatalUnexpectedStateMachineEvent
Id: 137  
  
 심각도: 오류  
  
 범주: TransactionBridge  
  
## <a name="description"></a>설명  
 이 이벤트는 상태 시스템이 예기치 않은 이벤트를 처리하려고 했음을 나타냅니다. 이벤트가 치명적인 것으로 인식되었습니다. 이 이벤트는 트랜잭션 ID, 상태 시스템, 현재 상태, 기록, 이벤트 이름, 이벤트 상세 정보, 프로세스 이름 및 프로세스 ID를 표시합니다.
