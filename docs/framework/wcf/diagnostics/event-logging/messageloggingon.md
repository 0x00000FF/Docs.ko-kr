# <a name="messageloggingon"></a>MessageLoggingOn
ID: 116  
  
 심각도: Error  
  
 범주: ServiceModel  
  
## <a name="description"></a>설명  
 메시지 로깅이 켜졌습니다. 와이어에서 암호화되는 경우라도 중요한 정보가 기록되었을 수 있습니다. 예를 들면 메시지 본문이 있을 수 있습니다. 이 이벤트는 AppID, 예외, 프로세스 이름 및 프로세스 ID를 표시합니다.
