# UnhandledStateMachineExceptionRecordDescription
ID: 136  
  
 심각도: 오류  
  
 범주: TransactionBridge  
  
## 설명  
 이 이벤트는 상태 시스템이 이벤트를 처리하는 동안 처리되지 않은 예외가 throw되었음을 나타냅니다.  이 이벤트는 트랜잭션 ID, 상태 시스템 이름, 현재 상태, 기록, 인리스트먼트 ID, 예외, 프로세스 이름 및 프로세스 ID를 표시합니다.
