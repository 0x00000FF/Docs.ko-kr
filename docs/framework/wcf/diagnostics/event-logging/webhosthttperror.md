# <a name="webhosthttperror"></a>WebHostHttpError
ID: 110  
  
 심각도: 오류  
  
 범주: ServiceModel  
  
## <a name="description"></a>설명  
 이 이벤트는 WebHost HTTP 예외가 발생했음을 나타냅니다. 이벤트는 발신자 정보, HtmlErrorMessage, 예외, 프로세스 이름 및 프로세스 ID를 표시합니다.
