# <a name="sslnoprivatekey"></a>SslNoPrivateKey
Id: 154  
  
 심각도: 오류  
  
 범주: TransactionBridge  
  
## <a name="description"></a>설명  
 이 이벤트는 특정 주체 이름 및 지문의 ID 인증서에 개인 키가 없음을 나타냅니다. 이 이벤트는 프로세스 이름과 프로세스 ID를 표시합니다.
