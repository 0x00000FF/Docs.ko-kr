# <a name="failedtoremoveperformancecounter"></a>FailedToRemovePerformanceCounter
ID: 119  
  
 심각도: 오류  
  
 범주: ServiceModel  
  
## <a name="description"></a>설명  
 이 이벤트는 성능 카운터가 제거되지 않았음을 나타냅니다. 이벤트는 범주 이름, 카운터 이름, 인스턴스 이름, 예외, 프로세스 이름 및 프로세스 ID를 표시합니다.
