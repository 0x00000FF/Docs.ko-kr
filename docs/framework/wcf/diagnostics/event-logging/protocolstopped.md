# ProtocolStopped
ID: 151  
  
 심각도: 오류  
  
 범주: TransactionBridge  
  
## 설명  
 이 이벤트는 WS\-AT 프로토콜 서비스가 중지되었음을 나타냅니다.  이벤트는 프로토콜 ID, 프로토콜 이름, 프로세스 이름 및 프로세스 ID를 표시합니다.
