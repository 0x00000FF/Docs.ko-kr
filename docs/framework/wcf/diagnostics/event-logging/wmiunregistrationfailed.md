# <a name="wmiunregistrationfailed"></a>WmiUnregistrationFailed
Id: 127  
  
 심각도: 오류  
  
 범주: ServiceModel  
  
## <a name="description"></a>설명  
 이 이벤트는 WMI 공급자의 등록이 취소되지 않았음을 나타냅니다. 또한 이 이벤트는 WMI 개체, 오류, 프로세스 이름 및 프로세스 ID를 표시합니다.
