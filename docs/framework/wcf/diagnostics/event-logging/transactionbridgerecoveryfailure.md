# TransactionBridgeRecoveryFailure
ID: 146  
  
 심각도: 오류  
  
 범주: TransactionBridge  
  
## 설명  
 이 이벤트는 복구 중에 MSDTC TransactionBridge가 실패한 것을 나타냅니다.치명적인 상태이기 때문에 MSDTC 서비스가 종료되었습니다.이 이벤트는 예외, 프로세스 이름 및 프로세스 ID를 표시합니다.
