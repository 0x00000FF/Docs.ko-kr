# CoordinatorRecoveryLogEntryCreationFailure
ID: 140  
  
 심각도: 오류  
  
 범주: TransactionBridge  
  
## 설명  
 코디네이터 인리스트먼트를 위해 복구 로그 항목을 생성할 수 없습니다.트랜잭션이 중단됩니다.이벤트는 트랜잭션 ID, 이유, 예외, 프로세스 이름 및 프로세스 ID를 나열합니다.
