# 3825 - RoutingServiceAbandoningReceiveContext
## 속성

## 설명  
 이 이벤트는 라우팅 서비스가 지정된 ID의 메시지에 대해 ReceiveContext.Abandon을 호출할 때 내보내집니다.  
  
## 메시지  
 라우팅 서비스에서 ID가 '%1'인 메시지에 대해 ReceiveContext.Abandon을 호출하는 중입니다.
