# 3819 - RoutingServiceMovedToBackup

## 설명  
 이 이벤트는 라우팅 서비스가 백업 끝점으로 이동되었을 때 내보내집니다.  
  
## 메시지  
 ID가 '%1'인 라우팅 서비스 메시지\[작업 %2\]를 백업 끝점 '%3'\(으\)로 이동했습니다.
