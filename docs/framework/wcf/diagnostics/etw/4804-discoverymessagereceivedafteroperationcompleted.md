## 설명  
 이 이벤트는 작업이 성공한 후 검색 메시지가 수신되었을 때 내보내집니다.  
  
## 메시지  
 해당 %3 작업이 완료되어 messageId\='%2'인 %1 메시지가 DiscoveryClient에서 삭제되었습니다.
