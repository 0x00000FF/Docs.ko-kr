# 4806 - DiscoveryMessageWithInvalidRelatesToOrOperationCompleted
## 속성  
  
|||

## 설명  
 이 이벤트는 해당 작업이 완료되었거나 relatesTo 값이 잘못되어 검색 메시지가 DiscoveryClient에서 삭제되었을 때 내보내집니다.  
  
## 메시지  
 해당 %4 작업이 완료되었거나 relatesTo 값이 잘못되어 messageId\='%2'이고 relatesTo\='%3'인 %1 메시지가 DiscoveryClient에서 삭제되었습니다.
