## 설명  
 이 이벤트는 클라이언트 WebSocket 팩터리에서 WebSocketVersion을 사용할 경우 내보내집니다.  
  
## 메시지  
 클라이언트 WebSocket 팩터리 형식 '%1'에서 WebSocketVersion을 사용하는 중입니다.
