## <a name="description"></a>설명  
 이 이벤트는 ReplyTo 주소가 없어 검색 메시지가 DiscoveryClient에서 삭제될 때 내보내집니다.  
  
## <a name="message"></a>메시지  
 ReplyTo 주소가 없어 messageId='%1'인 검색 요청 메시지가 삭제되었습니다.
