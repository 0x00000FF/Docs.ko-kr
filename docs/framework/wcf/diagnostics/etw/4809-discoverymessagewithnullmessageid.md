## <a name="description"></a>설명  
 이 이벤트는 메시지 머리글에 필요한 MessageId 속성이 없어 검색 메시지가 삭제될 때 내보내집니다.  
  
## <a name="message"></a>메시지  
 메시지 머리글에 필요한 MessageId 속성이 없어 %1 메시지가 삭제되었습니다.
