# 3803 - RoutingServiceProcessingFailure
## 속성  
  
|||  
|-|-|  
|ID|3803|  
|키워드가|RoutingServices|  
|수준|오류|  
|채널|Microsoft\-Windows\-응용 프로그램 서버\-응용 프로그램\/디버그|  
  
## 설명  
 이 이벤트는 끝점에서 메시지를 처리하는 동안 라우팅 서비스가 실패할 때 내보내집니다.  
  
## 메시지  
 주소가 '%1'인 끝점에서 메시지를 처리하는 동안 라우팅 서비스에 오류가 발생했습니다.
