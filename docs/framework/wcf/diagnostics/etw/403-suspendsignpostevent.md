|||  
|-|-|  
|ID|403|  
|키워드|문제 해결|

## <a name="description"></a>설명  
 이 이벤트는 종단 간 동작의 일시 중단을 표시합니다. 여기에는 동작의 이름이 포함됩니다.

 동작 경계입니다.
