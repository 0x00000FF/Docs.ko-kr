## <a name="description"></a>설명  
 이 이벤트는 DiscoveryClientChannel에서 SynchronizationContext가 원래 값으로 다시 설정되었을 때 내보내집니다.  
  
## <a name="message"></a>메시지  
 DiscoveryClientChannel에서 SynchronizationContext가 원래 값 %1(으)로 다시 설정되었습니다.
