# 3821 - RoutingServiceCloseFailed
## 속성  
  
|||  
|-|-|  
|ID|3821|  
|키워드가|RoutingServices|  
|수준|경고|

## 설명  
 이 이벤트는 라우팅 서비스가 아웃바운드 클라이언트를 닫을 때 오류가 발생한 경우 내보내집니다.  
  
## 메시지  
 아웃바운드 클라이언트 '%1'을\(를\) 닫는 동안 라우팅 서비스에 오류가 발생했습니다.
