## 설명  
 이 이벤트는 찾기 작업을 시작하기 전 DiscoveryClientChannel에서 SynchronizationContext가 Null로 설정되었을 때 내보내집니다.  
  
## 메시지  
 찾기 작업을 시작하기 전에 DiscoveryClientChannel에서 SynchronizationContext가 Null로 설정되었습니다.
