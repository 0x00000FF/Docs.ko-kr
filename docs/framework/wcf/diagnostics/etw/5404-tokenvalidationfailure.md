# 5404 - TokenValidationFailure

## 설명  
 이 이벤트는 SecurityToken 유효성 검사가 실패할 때 내보내집니다.

 SecurityToken\('%1' 형식, ID '%2'\) 유효성 검사에 실패했습니다.  %3
