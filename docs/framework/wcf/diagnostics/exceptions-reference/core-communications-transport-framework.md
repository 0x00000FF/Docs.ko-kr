# 핵심 통신: 전송 프레임워크
이 항목에서는 전송 프레임워크에 의해 생성된 모든 예외를 보여 줍니다.  
  
## 예외 목록  
  
|리소스 코드|리소스 문자열|  
|------------|-------------|  
|Hosting\_TransportBindingNotFound|지정한 주소와 일치하는 프로토콜 바인딩이 없습니다.프로토콜 바인딩이 IIS\(인터넷 정보 서비스\) 또는 Windows Process Activation Service 구성에서 사이트 수준에 구성되었습니다.|
