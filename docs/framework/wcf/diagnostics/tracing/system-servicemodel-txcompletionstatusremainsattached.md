# System.ServiceModel.TxCompletionStatusRemainsAttached
지정된 작업에 대해 지정된 트랜잭션이 계속 InstanceContext에 연결되어 있습니다.  
  
## 설명  
 ServiceOperation이 끝났지만 트랜잭션이 complete로 설정되어 있지 않은 경우 추적됩니다.
