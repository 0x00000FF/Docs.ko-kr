# System.ServiceModel.ServiceOperationMissingReplyContext
System.ServiceModel.ServiceOperationMissingReplyContext  
  
## 설명  
 ReplyContext는 양방향 작업에 대해 null입니다.추적 데이터에서 "Reply 작업에 회신에 사용할 IRequestContext가 없습니다."라는 메시지를 표시합니다.  
  
 이 추적은 개발 작업 중 디버깅하는 데 유용합니다.
