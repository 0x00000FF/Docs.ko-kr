# System.ServiceModel.Channels.PnrpResolveException
PnrpResolverService가 확인을 수행하려는 동안 예외가 발생했습니다.  
  
## 설명  
 이 추적은 PNRP 확인자가 확인 작업을 수행하는 동안 발생합니다.
