# System.ServiceModel.GetChannelEndpointElement
System.ServiceModel.GetChannelEndpointElement  
  
## 설명  
 이 추적은 정보 수준에서 채널 팩터리를 생성할 때 내보냅니다.  클라이언트가 통신하는 끝점에 대한 설명을 제공합니다\(예: 원격 주소, 바인딩, 계약 이름\).
