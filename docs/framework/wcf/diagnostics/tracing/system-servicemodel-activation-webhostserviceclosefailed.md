# System.ServiceModel.Activation.WebHostServiceCloseFailed
서비스를 정상적으로 닫을 수 없고 중단된 경우 발생합니다.  
  
## 설명  
 이 오류 코드는 로그 파일에서만 나타납니다.예를 들어 중단을 이미 호출한 후 서비스를 닫으려는 경우와 같이 일반적으로 프로그래밍 오류를 표시합니다.  
  
## 문제 해결  
 응용 프로그램 소스 코드를 확인합니다.
