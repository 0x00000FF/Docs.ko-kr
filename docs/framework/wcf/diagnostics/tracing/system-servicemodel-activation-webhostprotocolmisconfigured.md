# System.ServiceModel.Activation.WebHostProtocolMisconfigured
WAS\(Windows Process Activation Service\)에서 프로토콜이 잘못 구성되었습니다.  
  
## 설명  
 이 오류는 전송 프로토콜이 잘못 구성되었을 때 발생합니다.  
  
## 문제 해결  
 web.config에서 전송 구성 매개 변수를 확인하여 수정하세요.
