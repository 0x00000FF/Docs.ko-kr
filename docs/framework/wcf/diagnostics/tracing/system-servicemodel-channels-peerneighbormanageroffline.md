# System.ServiceModel.Channels.PeerNeighborManagerOffline
PeerNode가 현재 연결되어 있는 환경이 없음을 나타내는 오프라인 상태입니다.  
  
## 설명  
 이 추적은 로컬 노드의 환경 목록에서 환경을 제거할 때 발생합니다.
