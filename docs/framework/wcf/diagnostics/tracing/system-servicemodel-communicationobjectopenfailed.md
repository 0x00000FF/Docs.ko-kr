# System.ServiceModel.CommunicationObjectOpenFailed
통신 개체를 열지 못했습니다.  
  
## 설명  
 Open 메서드를 실행하는 동안 예외가 throw되었기 때문에 통신 개체를 열지 못했습니다.근접한 다른 추적에 예외에 대한 세부 정보가 포함되어 있을 수 있습니다.
