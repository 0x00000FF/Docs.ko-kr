# System.ServiceModel.ServiceHostTimeoutOnClose
System.ServiceModel.ServiceHostTimeoutOnClose  
  
## 설명  
 ServiceHost의 닫기 작업이 시간 초과되었습니다.이것은 클라이언트가 일부 세션 채널을 제대로 종료하지 않아 중단되었음을 나타냅니다.기본 시간을 수정하여 닫기 제한 시간을 조정합니다.
