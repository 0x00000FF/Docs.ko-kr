# System.ServiceModel.TxReleaseServiceInstanceOnCompletion
ReleaseServiceInstanceOnTransactionComplete ServiceBehaviorAttribute가 true로 설정되었으므로 '{0}' 트랜잭션이 완료될 때 서비스 인스턴스가 릴리스되었습니다.  
  
## 설명  
 현재 서비스 인스턴스가 릴리스될 때 추적되거나 현재 활성 트랜잭션이 완료되고 ReleaseServiceInstanceOnTransactionComplete가 `true`로 설정되어 삭제됩니다.
