# System.Runtime.Serialization.FactoryTypeNotFound
System.Runtime.Serialization.FactoryTypeNotFound  
  
## 설명  
 deserializer가 deserialization 도중에 사용할 형식을 알지 못합니다.  XML에서 팩터리 형식으로 지정된 형식을 알려진 형식 목록에 추가해 보세요. 예를 들어 KnownTypeAttribute 특성을 사용합니다.
