# System.ServiceModel.TxSourceTxScopeRequiredIsAttachedTransaction
InstanceContext 트랜잭션에서 지정된 작업에 대해 지정된 트랜잭션이 수신되었습니다.  
  
## 설명  
 사용자 코드에서 사용 가능한 트랜잭션이 이 호출 이전에 호출된 AutoComplete\=false 메서드에 의해 남겨진 경우 추적됩니다.
