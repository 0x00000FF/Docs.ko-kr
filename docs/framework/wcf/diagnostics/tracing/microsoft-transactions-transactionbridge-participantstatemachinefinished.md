# Microsoft.Transactions.TransactionBridge.ParticipantStateMachineFinished
참가자 인리스트먼트에 대한 상태 시스템이 완료 상태가 되었습니다.  
  
## 설명  
 하위 참여 인리스트먼트에서 2PC\(2단계 커밋\) 처리를 완료한 경우 추적됩니다.인리스트먼트 결과는 커밋됨 또는 중단됨입니다.참가자가 준비 과정에서 ReadOnly를 선택하는 경우에도 추적됩니다.
