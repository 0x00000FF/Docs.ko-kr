# System.ServiceModel.TxSourceTxScopeRequiredIsTransactionFlow
지정된 트랜잭션이 지정된 작업으로 이동했습니다.  
  
## 설명  
 사용자 코드에 대해 사용 가능한 트랜잭션이 외부 소스에서 이동한 경우 추적됩니다.
