# Microsoft.Transactions.TransactionBridge.RegisterParticipant
WS\-Atomic Transaction 프로토콜 서비스가 제어 프로토콜에 참가자를 등록했습니다.  
  
## 설명  
 로컬 트랜잭션 관리자가 2PC\(2단계 커밋\) 프로토콜에 참가자 등록 요청을 승인했을 때 추적됩니다.
