# Microsoft.Transactions.TransactionBridge.ProtocolInitialized
WS\-Atomic Transaction 프로토콜 서비스가 초기화되었습니다.

 성공적으로 초기화된 프로토콜을 나타내기 위해 트랜잭션 관리자를 시작할 때 추적됩니다.
