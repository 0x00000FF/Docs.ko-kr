# System.ServiceModel.Channels.PnrpUnregisteredAddresses
Pnrp Resolver에서 로컬 노드 주소의 등록을 취소했습니다.  
  
## 설명  
 이 추적은 PNRP 서비스에서 로컬 노드 주소의 등록을 성공적으로 취소하면 발생합니다.
