# System.ServiceModel.TxCompletionStatusCompletedForSetComplete
지정한 작업에 대해 지정된 트랜잭션이 SetTransactionComplete에 대한 호출로 인해 완료되었습니다.  
  
## 설명  
 사용자가 OperationContext를 통해 현재 트랜잭션을 완료하도록 명시적으로 결정한 경우 추적됩니다.
