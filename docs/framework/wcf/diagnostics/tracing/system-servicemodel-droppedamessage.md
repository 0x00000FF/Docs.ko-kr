# <a name="systemservicemodeldroppedamessage"></a>System.ServiceModel.DroppedAMessage
System.ServiceModel.DroppedAMessage  
  
## <a name="description"></a>설명  
 응답 메시지를 받았지만 이 메시지를 기다리고 있는 처리되지 않은 요청이 없습니다. Windows Communication Foundation (WCF) 런타임에서 메시지가 삭제 됩니다.  
  
 이 추적은 손실된 메시지의 디버깅에 사용할 수 있습니다.
