# System.ServiceModel.DroppedAMessage
System.ServiceModel.DroppedAMessage  
  
## 설명  
 응답 메시지를 받았지만 이 메시지를 기다리고 있는 처리되지 않은 요청이 없습니다.[!INCLUDE[indigo1](../../../../../includes/indigo1-md.md)] 런타임에서 메시지가 손실됩니다.  
  
 이 추적은 손실된 메시지의 디버깅에 사용할 수 있습니다.
