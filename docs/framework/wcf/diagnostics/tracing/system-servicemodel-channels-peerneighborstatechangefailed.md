# System.ServiceModel.Channels.PeerNeighborStateChangeFailed
환경 상태가 손상되었습니다.  
  
## 설명  
 이 추적은 환경 상태 변경하는 데 실패했을 때 발생합니다.이것은 예기치 않은 내부 오류 또는 손상된 메모리를 나타냅니다.
