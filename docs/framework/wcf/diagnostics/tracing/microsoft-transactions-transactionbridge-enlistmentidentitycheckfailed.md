# Microsoft.Transactions.TransactionBridge.EnlistmentIdentityCheckFailed
들어오는 메시지 발신자의 ID가 인리스먼트의 끝점 참조에서 파생된 ID와 일치하지 않습니다.  
  
## 설명  
 메시지 보안 유효성 검사에 실패한 경우 추적됩니다.  
  
## 문제 해결  
 트랜잭션에 참여하는 모든 시스템에 대해 트랜잭션 보안이 올바르게 구성되어 있는지 확인합니다.
