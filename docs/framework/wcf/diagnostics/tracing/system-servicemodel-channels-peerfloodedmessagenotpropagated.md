# System.ServiceModel.Channels.PeerFloodedMessageNotPropagated
PeerNode에서 메시지를 받았지만 환경에 전파되지 않았습니다.  
  
## 설명  
 이 추적은 들어오는 서비스 장애 유발 메시지를 처리하는 동안 발생합니다.  
  
## 문제 해결  
 메시지 전파 필터가 있는지 확인합니다.
