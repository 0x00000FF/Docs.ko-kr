# Microsoft.Transactions.TransactionBridge.RegisterCoordinator
WS\-Atomic Transaction 프로토콜 서비스가 제어 프로토콜의 자체 코디네이터에 등록되었습니다.  
  
## 설명  
 로컬 TransactionManager가 상위 TransactionManage에 등록된 경우 추적됩니다.
