# System.ServiceModel.TxSourceTxScopeRequiredIsTransactedTransport
MSMQ와 같은 트랜잭션 전송에서 지정된 작업에 대한 지정된 트랜잭션을 받았습니다.  
  
## 설명  
 사용자 코드에 대해 사용 가능한 트랜잭션이 전송에서 작성된 경우 추적됩니다.
