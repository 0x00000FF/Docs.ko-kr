# Microsoft.Transactions.TransactionBridge.CreateTransactionFailure
트랜잭션을 만들 수 없습니다.  
  
## 설명  
 이 추적은 MSDTC에서 트랜잭션을 만들 수 없는 경우에 생성됩니다.  리소스 부족, 로그 공간 부족 또는 기타 오류로 인해 발생할 수 있습니다.  
  
## 문제 해결  
 추적 메시지 내의 상태 문자열을 검사하여 실행 가능한 항목이 있는지 확인하세요.
