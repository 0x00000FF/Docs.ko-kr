# <a name="wcf-and-net-framework-client-profile"></a>WCF 및 .NET Framework 클라이언트 프로필
[!INCLUDE[arrowhead](../../../includes/arrowhead-md.md)]은 전체 프레임워크가 필요하지 않은 클라이언트를 위해 디자인된 전체 .Net Framework의 약식 버전입니다. 클라이언트 프레임워크에서는 일부 Windows Communication Foundation이 지원되지 않습니다.  
  
## <a name="wcf-features-supported-by-the-net-framework-client-profile"></a>.Net Framework 클라이언트 프로필에서 지원되는 WCF 기능  
 [!INCLUDE[arrowhead](../../../includes/arrowhead-md.md)]에서 지원하는 Windows Communication Foundation 기능은 다음과 같습니다.  
  
-   Cardspace와 웹 호스팅을 제외하고 모든 WCF가 지원됩니다.  
  
-   원격 TCP/IP 채널이 지원됩니다.  
  
-   Asmx(웹 서비스)는 지원되지 않습니다.
