## <a name="using-the-wcf-client"></a><span data-ttu-id="098e0-135">WCF 클라이언트 사용</span><span class="sxs-lookup"><span data-stu-id="098e0-135">Using the WCF Client</span></span>  
 <span data-ttu-id="098e0-136">[!INCLUDE[indigo2](../../../includes/indigo2-md.md)] 클라이언트를 사용하려면 다음 코드와 같이 [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] 클라이언트 인스턴스를 만든 다음 해당 메서드를 호출합니다.</span><span class="sxs-lookup"><span data-stu-id="098e0-136">To use the [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] client, create an instance of the [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] client, and then call its methods, as shown in the following code.</span></span>
