## <a name="the-wsstreamedhttpbinding-sample-service"></a><span data-ttu-id="2a093-132">WSStreamedHttpBinding 샘플 서비스</span><span class="sxs-lookup"><span data-stu-id="2a093-132">The WSStreamedHttpBinding Sample Service</span></span>  
 <span data-ttu-id="2a093-133">`WSStreamedHttpBinding`을 사용하는 샘플 서비스는 서비스 하위 디렉터리에 있습니다.</span><span class="sxs-lookup"><span data-stu-id="2a093-133">The sample service that uses `WSStreamedHttpBinding` is located in the service subdirectory.</span></span> <span data-ttu-id="2a093-134">`OperationContract` 구현에서는 `MemoryStream`을 반환하기 전에 `MemoryStream`을 사용하여 들어오는 스트림에서 모든 데이터를 검색합니다.</span><span class="sxs-lookup"><span data-stu-id="2a093-134">The implementation of `OperationContract` uses a `MemoryStream` to first retrieve all data from the incoming stream before returning the `MemoryStream`.</span></span> <span data-ttu-id="2a093-135">샘플 서비스는 IIS(인터넷 정보 서비스)에 의해 호스팅됩니다.</span><span class="sxs-lookup"><span data-stu-id="2a093-135">The sample service is hosted by Internet Information Services (IIS).</span></span>  
  
```  
[ServiceContract]  
public interface IStreamedEchoService  
{  
    [OperationContract]  
    Stream Echo(Stream data);  
}  
  
public class StreamedEchoService : IStreamedEchoService  
{  
    public Stream Echo(Stream data)  
    {  
        MemoryStream dataStorage = new MemoryStream();  
        byte[] byteArray = new byte[8192];  
        int bytesRead = data.Read(byteArray, 0, 8192);  
        while (bytesRead > 0)  
        {  
            dataStorage.Write(byteArray, 0, bytesRead);  
            bytesRead = data.Read(byteArray, 0, 8192);  
        }  
        data.Close();  
        dataStorage.Seek(0, SeekOrigin.Begin);  
  
        return dataStorage;  
    }  
}  
```
