# 호스팅
이 단원에는 [!INCLUDE[indigo1](../../../../includes/indigo1-md.md)] 서비스의 호스팅 방법을 보여 주는 샘플이 포함되어 있습니다.  
  
## 단원 내용  
 [Windows 프로세스 활성화](../../../../docs/framework/wcf/samples/windows-process-activation.md)  
 네트워크 프로토콜을 통한 서비스 활성화를 보여 줍니다.  
  
 [구성 기반 활성화](../../../../docs/framework/wcf/samples/configuration-based-activation.md)  
 .svc 파일 없이 [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] 서비스를 활성화하는 방법을 보여 줍니다.  
  
 [SystemWebRouting Integration 샘플](../../../../docs/framework/wcf/samples/systemwebrouting-integration-sample.md)  
 호스팅 계층과 <xref:System.Web.Routing> 네임스페이스에 있는 클래스의 통합을 보여 줍니다.  
  
 [ASP.NET 호환성](../../../../docs/framework/wcf/samples/aspnet-compatibility.md)  
 [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)]에서 [!INCLUDE[vstecasp](../../../../includes/vstecasp-md.md)] 호환 모드를 사용하도록 설정하는 방법을 보여 줍니다.  
  
 [인라인 코드를 사용한 IIS 호스팅](../../../../docs/framework/wcf/samples/iis-hosting-using-inline-code.md)  
 IIS\(인터넷 정보 서비스\)에서 호스팅하는 서비스를 구현하는 방법을 보여 줍니다. 여기서 서비스 코드는 .svc 파일에 인라인으로 포함되어 있으며 필요할 때 컴파일됩니다.  
  
 [Windows Service Host](../../../../docs/framework/wcf/samples/windows-service-host.md)  
 관리되는 Windows 서비스에 호스팅된 [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] 서비스를 보여 줍니다.  
  
 [자체 호스팅](../../../../docs/framework/wcf/samples/self-host.md)  
 콘솔 응용 프로그램에서 자체 호스팅 서비스를 구현하는 방법을 보여 줍니다.
