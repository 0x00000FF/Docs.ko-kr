 [!INCLUDE[crabout](../../../../includes/crabout-md.md)]<span data-ttu-id="3a837-109">코드에서 메타 데이터 게시 참조 [하는 방법: 서비스를 사용 하 여 코드에 대 한 메타 데이터 게시](../../../../docs/framework/wcf/feature-details/how-to-publish-metadata-for-a-service-using-code.md)합니다.</span><span class="sxs-lookup"><span data-stu-id="3a837-109"> publishing metadata in code, see [How to: Publish Metadata for a Service Using Code](../../../../docs/framework/wcf/feature-details/how-to-publish-metadata-for-a-service-using-code.md).</span></span> <span data-ttu-id="3a837-110">메타데이터를 게시하면 클라이언트에서 WS-Transfer GET 요청을 사용하는 메타데이터 또는 `?wsdl` 쿼리 문자열을 사용하는 HTTP/GET 요청을 검색할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="3a837-110">Publishing metadata allows clients to retrieve the metadata using a WS-Transfer GET request or an HTTP/GET request using the `?wsdl` query string.</span></span> <span data-ttu-id="3a837-111">코드가 작동 중인지 확인하려면 기본 [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] 서비스를 만듭니다.</span><span class="sxs-lookup"><span data-stu-id="3a837-111">To be sure that the code is working, create a basic [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] service.</span></span> <span data-ttu-id="3a837-112">편의상 다음 코드로 된 기본 자체 호스팅 서비스가 제공됩니다.</span><span class="sxs-lookup"><span data-stu-id="3a837-112">For simplicity, a basic self-hosted service is provided in the following code.</span></span>  
  
```csharp  
using System;  
using System.Runtime.Serialization;  
using System.ServiceModel;  
using System.ServiceModel.Description;  
  
namespace Metadata.Samples  
{  
    [ServiceContract]  
    public interface ISimpleService  
    {  
        [OperationContract]  
        string SimpleMethod(string msg);  
    }  
  
    class SimpleService : ISimpleService  
    {  
        public string SimpleMethod(string msg)  
        {  
            Console.WriteLine("The caller passed in " + msg);  
            return "Hello " + msg;  
        }  
    }  
  
    class Program  
    {  
        static void Main(string[] args)  
        {  
            ServiceHost host = new ServiceHost(typeof(SimpleService),  
                new Uri("http://localhost:8001/MetadataSample"));   
            try  
            {  
                // Open the service host to accept incoming calls  
                host.Open();  
  
                // The service can now be accessed.  
                Console.WriteLine("The service is ready.");  
                Console.WriteLine("Press <ENTER> to terminate service.");  
                Console.WriteLine();  
                Console.ReadLine();  
  
                // Close the ServiceHostBase to shutdown the service.  
                host.Close();  
            }  
            catch (CommunicationException commProblem)  
            {  
                Console.WriteLine("There was a communication problem. " + commProblem.Message);  
                Console.Read();  
            }  
        }  
    }  
}  
```  
  
 <span data-ttu-id="3a837-113">이 서비스는 구성 파일을 사용하여 구성된 자체 호스팅 서비스입니다.</span><span class="sxs-lookup"><span data-stu-id="3a837-113">This service is a self-hosted service, which is configured using a configuration file.</span></span> <span data-ttu-id="3a837-114">다음 구성 파일을 사용해 작업을 시작합니다.</span><span class="sxs-lookup"><span data-stu-id="3a837-114">The following configuration file serves as a starting point.</span></span>

```xml  
  <system.serviceModel>

      <service name="Metadata.Example.SimpleService">  
        <endpoint address=""  
                  binding="basicHttpBinding"  
                  contract="Metadata.Example.ISimpleService" />  
      </service>  
    </services>  
    <behaviors>  
  
    </behaviors>  
  </system.serviceModel>  
</configuration>  
```  
  
### <a name="to-publish-metadata-for-a-wcf-service-using-an-application-configuration-file"></a><span data-ttu-id="3a837-115">응용 프로그램 구성 파일을 사용하여 WCF 서비스의 메타데이터를 게시하려면</span><span class="sxs-lookup"><span data-stu-id="3a837-115">To publish metadata for a WCF service using an application configuration file</span></span>  
  
1.  <span data-ttu-id="3a837-116">App.config 파일 내에서 `</services>` 요소를 닫은 후 `<behaviors>` 요소를 만듭니다.</span><span class="sxs-lookup"><span data-stu-id="3a837-116">Within the App.config file, after the closing `</services>` element, create a `<behaviors>` element.</span></span>  
  
  
  
2.  <span data-ttu-id="3a837-117">`<behaviors>` 요소 내에서 `<serviceBehaviors>` 요소를 추가합니다.</span><span class="sxs-lookup"><span data-stu-id="3a837-117">Within the `<behaviors>` element, add a `<serviceBehaviors>` element.</span></span>  
  
  
  
3.  <span data-ttu-id="3a837-118">`<behavior>``name` 요소를 `<serviceBehaviors>``<behavior>` 요소에 추가하고  요소의  특성에 값을 지정합니다.</span><span class="sxs-lookup"><span data-stu-id="3a837-118">Add a `<behavior>` element to the `<serviceBehaviors>` element and specify a value for the `name` attribute of the `<behavior>` element.</span></span>  
  
  
  
4.  <span data-ttu-id="3a837-119">`<serviceMetadata>` 요소를 `<behavior>` 요소에 추가합니다.</span><span class="sxs-lookup"><span data-stu-id="3a837-119">Add a `<serviceMetadata>` element to the `<behavior>` element.</span></span> <span data-ttu-id="3a837-120">`httpGetEnabled` 특성을 `true`로 설정하고 `policyVersion` 특성을 Policy15로 설정합니다.</span><span class="sxs-lookup"><span data-stu-id="3a837-120">Set the `httpGetEnabled` attribute to `true` and the `policyVersion` attribute to Policy15.</span></span> <span data-ttu-id="3a837-121">`httpGetEnabled`를 사용하면 서비스가 HTTP GET 요청으로 수행된 메타데이터 요청에 응답할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="3a837-121">`httpGetEnabled` allows the service to respond to metadata requests made by an HTTP GET request.</span></span> <span data-ttu-id="3a837-122">`policyVersion`에 따라 서비스는 메타데이터를 생성할 때 WS-Policy 1.5를 준수합니다.</span><span class="sxs-lookup"><span data-stu-id="3a837-122">`policyVersion` tells the service to conform to WS-Policy 1.5 when generating metadata.</span></span>  
  
  
  
5.  <span data-ttu-id="3a837-123">다음 코드 예제와 같이 `behaviorConfiguration` 특성을 `<service>` 요소에 추가하고 1단계에 추가된 `name` 요소의 `<behavior>` 특성을 지정합니다.</span><span class="sxs-lookup"><span data-stu-id="3a837-123">Add a `behaviorConfiguration` attribute to the `<service>` element and specify the `name` attribute of the `<behavior>` element added in step 1, as shown in the following code example.</span></span>

    ```xml  
    <services>  
      <service  
          name="Metadata.Example.SimpleService"  
          behaviorConfiguration="SimpleServiceBehavior">  
        ...  
      </service>  
    </services>  
    <behaviors>  
      <serviceBehaviors>  
        <behavior name="SimpleServiceBehavior">  
          <serviceMetadata httpGetEnabled="True" policyVersion="Policy15" />  
        </behavior>  
      </serviceBehaviors>  
    </behaviors>  
    ```  
  
6.  <span data-ttu-id="3a837-124">다음 코드 예제와 같이 계약이 `<endpoint>`로 설정된 하나 이상의 `IMetadataExchange` 요소를 추가합니다.</span><span class="sxs-lookup"><span data-stu-id="3a837-124">Add one or more `<endpoint>` elements with the contract set to `IMetadataExchange`, as shown in the following code example.</span></span>

    ```xml  
          name="Metadata.Example.SimpleService"  
          behaviorConfiguration="SimpleServiceBehavior">  
  
        <endpoint address=""  
                  binding="wsHttpBinding"  
                  contract="Metadata.Example.ISimpleService" />  
  
        <endpoint address="mex"  
                  binding="mexHttpBinding"  
                  contract="IMetadataExchange" />  
      </service>  
    </services>  
    ```

    -   <span data-ttu-id="3a837-127">HTTPS 게시의 경우 `mexHttpsBinding`</span><span class="sxs-lookup"><span data-stu-id="3a837-127">`mexHttpsBinding` for HTTPS publication.</span></span>  
  
    -   <span data-ttu-id="3a837-128">명명된 파이프 게시의 경우 `mexNamedPipeBinding`</span><span class="sxs-lookup"><span data-stu-id="3a837-128">`mexNamedPipeBinding` for named pipe publication.</span></span>  
  
    -   <span data-ttu-id="3a837-129">TCP 게시의 경우 `mexTcpBinding`</span><span class="sxs-lookup"><span data-stu-id="3a837-129">`mexTcpBinding` for TCP publication.</span></span>  
  
8.  <span data-ttu-id="3a837-130">이전 단계에 추가된 메타데이터 끝점에 대해 주소를 다음과 같이 설정합니다.</span><span class="sxs-lookup"><span data-stu-id="3a837-130">For the metadata endpoints added in a previous step, set the address equal to:</span></span>  
  
    -   <span data-ttu-id="3a837-131">기본 주소가 메타데이터 바인딩과 동일한 경우 호스트 응용 프로그램의 기본 주소를 게시 지점으로 사용하는 빈 문자열</span><span class="sxs-lookup"><span data-stu-id="3a837-131">An empty string to use the host application's base address as the publication point if the base address is the same as the metadata binding.</span></span>  
  
    -   <span data-ttu-id="3a837-132">호스트 응용 프로그램에 기본 주소가 있는 경우 상대 주소</span><span class="sxs-lookup"><span data-stu-id="3a837-132">A relative address if the host application has a base address.</span></span>  
  
    -   <span data-ttu-id="3a837-133">절대 경로</span><span class="sxs-lookup"><span data-stu-id="3a837-133">An absolute address.</span></span>  
  
9. <span data-ttu-id="3a837-134">콘솔 응용 프로그램을 빌드하고 실행합니다.</span><span class="sxs-lookup"><span data-stu-id="3a837-134">Build and run the console application.</span></span>

## <a name="example"></a><span data-ttu-id="3a837-142">예제</span><span class="sxs-lookup"><span data-stu-id="3a837-142">Example</span></span>  
 <span data-ttu-id="3a837-143">다음 코드 예제에서는 기본 [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] 서비스의 구현 및 서비스의 메타데이터를 게시하는 구성 파일을 보여 줍니다.</span><span class="sxs-lookup"><span data-stu-id="3a837-143">The following code example shows the implementation of a basic [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] service and the configuration file that publishes metadata for the service.</span></span>  
  
```csharp  
using System;  
using System.Runtime.Serialization;  
using System.ServiceModel;  
using System.ServiceModel.Description;  
  
namespace Metadata.Samples  
{  
    [ServiceContract]  
    public interface ISimpleService  
    {  
        [OperationContract]  
        string SimpleMethod(string msg);  
    }  
  
    class SimpleService : ISimpleService  
    {  
        public string SimpleMethod(string msg)  
        {  
            Console.WriteLine("The caller passed in " + msg);  
            return "Hello " + msg;  
        }  
    }  
  
    class Program  
    {  
        static void Main(string[] args)  
        {  
            ServiceHost host = new ServiceHost(typeof(SimpleService),  
                new Uri("http://localhost:8001/MetadataSample"));   
            try  
            {  
                // Open the service host to accept incoming calls  
                host.Open();  
  
                // The service can now be accessed.  
                Console.WriteLine("The service is ready.");  
                Console.WriteLine("Press <ENTER> to terminate service.");  
                Console.WriteLine();  
                Console.ReadLine();  
  
                // Close the ServiceHostBase to shutdown the service.  
                host.Close();  
            }  
            catch (CommunicationException commProblem)  
            {  
                Console.WriteLine("There was a communication problem. " + commProblem.Message);  
                Console.Read();  
            }  
        }  
    }  
}  
```
