## <a name="configure-the-wcf-service"></a><span data-ttu-id="5ba7b-162">WCF 서비스 구성</span><span class="sxs-lookup"><span data-stu-id="5ba7b-162">Configure the WCF Service</span></span>

### <a name="endpoint-addresses-for-iis-hosted-services"></a><span data-ttu-id="5ba7b-169">IIS에서 호스팅되는 서비스의 엔드포인트 주소</span><span class="sxs-lookup"><span data-stu-id="5ba7b-169">Endpoint Addresses for IIS-Hosted Services</span></span>
