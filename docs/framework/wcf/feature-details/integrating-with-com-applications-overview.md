 <span data-ttu-id="a248b-112">또한 다른 [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] 클라이언트와 공통적으로, 모니커를 사용하면 응용 프로그램에서 서비스와 통신할 주소, 바인딩 및 계약을 지정합니다.</span><span class="sxs-lookup"><span data-stu-id="a248b-112">In common with other [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] clients, when using the moniker, applications specify the address, binding and contract to communicate with a service.</span></span> <span data-ttu-id="a248b-113">계약은 다음 방법 중 하나로 지정할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="a248b-113">The contract can be specified in one of the following ways:</span></span>
