1.  <span data-ttu-id="ce9ff-112">인스턴스를 만들고는 <<!--zz xref:System.ServiceModel.WsHttpBinding --> `xref:System.ServiceModel.WsHttpBinding`>, 바인딩의 보안 모드 설정 `SecurityMode.Message`설정는 `ClientCredentialType` 에 대 한 바인딩 `MessageCredentialType.UserName`, 표시 된 것 처럼 서비스 호스트에 구성 된 바인딩을 사용 하 여 서비스 끝점을 추가 하 고 다음 코드:</span><span class="sxs-lookup"><span data-stu-id="ce9ff-112">Create an instance of the <<!--zz xref:System.ServiceModel.WsHttpBinding --> `xref:System.ServiceModel.WsHttpBinding`>, set the security mode of the binding to `SecurityMode.Message`, set the `ClientCredentialType` of the binding to `MessageCredentialType.UserName`, and add a service endpoint using the configured binding to the service host as shown in the following code:</span></span>  
  
    ```  
    // ...

    // ...  
    ```

### <a name="to-call-the-service-passing-username-and-password"></a><span data-ttu-id="ce9ff-126">사용자 이름과 암호를 전달하는 서비스를 호출하려면</span><span class="sxs-lookup"><span data-stu-id="ce9ff-126">To call the service passing username and password</span></span>  
  
1.  <span data-ttu-id="ce9ff-127">클라이언트 응용 프로그램은 사용자에게 사용자 이름과 암호를 물어야 합니다.</span><span class="sxs-lookup"><span data-stu-id="ce9ff-127">The client application must prompt the user for their username and password.</span></span> <span data-ttu-id="ce9ff-128">다음 코드는 사용자에게 사용자 이름과 암호를 묻습니다.</span><span class="sxs-lookup"><span data-stu-id="ce9ff-128">The following code asks the user for username and password.</span></span>  
  
    > [!WARNING]
    >  <span data-ttu-id="ce9ff-129">암호는 입력하는 동안 표시되므로 이 코드는 프로덕션에서 사용하지 말아야 합니다.</span><span class="sxs-lookup"><span data-stu-id="ce9ff-129">This code should not be used in production as the password is displayed while being entered.</span></span>  
  
    ```  
    public static void GetPassword(out string username, out string password)  
            {  
                Console.WriteLine("Provide a valid machine or domain account. [domain\\user]");  
                Console.WriteLine("   Enter username:");  
                username = Console.ReadLine();  
                Console.WriteLine("   Enter password:");  
                password = Console.ReadLine();             
                return;  
            }  
    ```  
  
2.  <span data-ttu-id="ce9ff-130">다음 코드와 같이 클라이언트의 자격 증명을 지정하는 클라이언트 프록시 인스턴스를 만듭니다.</span><span class="sxs-lookup"><span data-stu-id="ce9ff-130">Create an instance of the client proxy specifying the client’s credentials as shown in the following code:</span></span>  
  
    ```  
    string username;  
    string password;  
  
    // Instantiate the proxy  
    Service1Client proxy = new Service1Client();  
  
    // Prompt the user for username & password  
    GetPassword(out username, out password);  
  
    // Set the user’s credentials on the proxy  
    proxy.ClientCredentials.UserName.UserName = username;  
    proxy.ClientCredentials.UserName.Password = password;  
  
    // Treat the test certificate as trusted  
    proxy.ClientCredentials.ServiceCertificate.Authentication.CertificateValidationMode = System.ServiceModel.Security.X509CertificateValidationMode.PeerOrChainTrust;  
    // Call the service operation using the proxy  
    ```
