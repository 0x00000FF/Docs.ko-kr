 <span data-ttu-id="19b8a-111">프로토콜 선택에 따라 다음 사항이 결정됩니다.</span><span class="sxs-lookup"><span data-stu-id="19b8a-111">The choice of protocol influences:</span></span>  
  
-   <span data-ttu-id="19b8a-112">클라이언트에서 서버로 트랜잭션을 하는 데 사용되는 메시지 헤더의 형식</span><span class="sxs-lookup"><span data-stu-id="19b8a-112">The format of the message headers used to flow the transaction from client to server.</span></span>
