# <a name="data-contract-serializer"></a>데이터 계약 Serializer
[!INCLUDE[indigo1](../../../../includes/indigo1-md.md)] 인프라는 <xref:System.Runtime.Serialization.DataContractSerializer> 클래스를 기본 serializer로 사용합니다. 이 단원의 항목에서는 클래스를 독립 실행형 serializer로 사용하기 위한 자세한 정보를 제공합니다.  
  
## <a name="in-this-section"></a>단원 내용  
 [Serialization 및 Deserialization](../../../../docs/framework/wcf/feature-details/serialization-and-deserialization.md)  
 <xref:System.Runtime.Serialization.DataContractSerializer> 클래스를 사용하여 개체 그래프를 serialize하고 XML 문서를 개체에 deserialize하는 방법에 대해 자세하게 설명합니다.  
  
 [스키마 가져오기 및 내보내기](../../../../docs/framework/wcf/feature-details/schema-import-and-export.md)  
 [!INCLUDE[dnprdnshort](../../../../includes/dnprdnshort-md.md)] 클래스에서 XSD(XML 스키마 정의 언어) 스키마를 생성하거나 XSD 스키마 문서에서 코드를 생성하는 방법에 대해 설명합니다.
