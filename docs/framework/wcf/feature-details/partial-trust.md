# <a name="partial-trust"></a>부분 신뢰

.NET Framework 3.5부터 부분적으로 신뢰할 수 있는 호출자는 <xref:System.ServiceModel>, <xref:System.Runtime.Serialization>및 <xref:System.ServiceModel.Web>에서 구현 된 공용 형식 및 메서드에 액세스할 수 있습니다. 이 섹션에서는 부분적으로 신뢰할 수 있는 응용 프로그램 내에서 WCF (Windows Communication Foundation)를 사용 하기 위한 지원 되는 시나리오와 축소 된 CAS (코드 액세스 보안)로 실행 되는 응용 프로그램에서 사용할 수 있는 WCF 기능의 제한 된 하위 집합 권한에.
