### <a name="moving-a-com-hosted-integration-app"></a><span data-ttu-id="9cb29-105">COM+ 호스팅 통합 응용 프로그램 이동</span><span class="sxs-lookup"><span data-stu-id="9cb29-105">Moving a COM+ hosted Integration App</span></span>

3.  <span data-ttu-id="9cb29-108">컴퓨터 B로 응용 프로그램을 가져옵니다.</span><span class="sxs-lookup"><span data-stu-id="9cb29-108">Import the application on machine B.</span></span>
