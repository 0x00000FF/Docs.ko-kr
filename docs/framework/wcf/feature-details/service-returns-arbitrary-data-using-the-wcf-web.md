### <a name="to-implement-the-service-contract"></a><span data-ttu-id="b4ca5-107">서비스 계약을 구현하려면</span><span class="sxs-lookup"><span data-stu-id="b4ca5-107">To implement the service contract</span></span>

2.  <span data-ttu-id="b4ca5-111">서비스 계약을 구현합니다.</span><span class="sxs-lookup"><span data-stu-id="b4ca5-111">Implement the service contract.</span></span> <span data-ttu-id="b4ca5-112">이 계약에는 하나의 작업(`GetImage`)만 있습니다.</span><span class="sxs-lookup"><span data-stu-id="b4ca5-112">The contract has only one operation (`GetImage`).</span></span> <span data-ttu-id="b4ca5-113">이 메서드는 비트맵을 생성하여 <xref:System.IO.MemoryStream>에 .jpg 형식으로 저장합니다.</span><span class="sxs-lookup"><span data-stu-id="b4ca5-113">This method generates a bitmap and then save it to a <xref:System.IO.MemoryStream> in .jpg format.</span></span> <span data-ttu-id="b4ca5-114">그런 다음 이 작업은 해당 스트림을 호출자에 반환합니다.</span><span class="sxs-lookup"><span data-stu-id="b4ca5-114">The operation then returns that stream to the caller.</span></span>

           {  
               Bitmap bitmap = new Bitmap(width, height);  
               for (int i = 0; i < bitmap.Width; i++)  
               {  
                   for (int j = 0; j < bitmap.Height; j++)

               bitmap.Save(ms, System.Drawing.Imaging.ImageFormat.Jpeg);

## <a name="example"></a><span data-ttu-id="b4ca5-129">예제</span><span class="sxs-lookup"><span data-stu-id="b4ca5-129">Example</span></span>  
 <span data-ttu-id="b4ca5-130">다음은 이 항목에 해당되는 전체 코드 목록입니다.</span><span class="sxs-lookup"><span data-stu-id="b4ca5-130">The following is a complete listing of the code for this topic.</span></span>  
  
```  
using System;  
using System.Collections.Generic;  
using System.Text;  
using System.ServiceModel;  
using System.ServiceModel.Web;  
using System.ServiceModel.Description;  
using System.IO;  
using System.Drawing;  
  
namespace RawImageService  
{  
    // Define the service contract  
    [ServiceContract]  
    public interface IImageServer  
    {  
        [WebGet]  
        Stream GetImage(int width, int height);  
    }  
  
    // implement the service contract  
    public class Service : IImageServer  
    {  
        public Stream GetImage(int width, int height)  
        {  
            // Although this method returns a jpeg, it can be  
            // modified to return any data you want within the stream  
            Bitmap bitmap = new Bitmap(width, height);  
            for (int i = 0; i < bitmap.Width; i++)  
            {  
                for (int j = 0; j < bitmap.Height; j++)  
                {  
                    bitmap.SetPixel(i, j, (Math.Abs(i - j) < 2) ? Color.Blue : Color.Yellow);  
                }  
            }  
            MemoryStream ms = new MemoryStream();  
            bitmap.Save(ms, System.Drawing.Imaging.ImageFormat.Jpeg);  
            ms.Position = 0;  
            WebOperationContext.Current.OutgoingResponse.ContentType = "image/jpeg";  
            return ms;  
        }  
    }  
  
    class Program  
    {  
        static void Main(string[] args)  
        {  
            string baseAddress = "http://" + Environment.MachineName + ":8000/Service";  
            ServiceHost host = new ServiceHost(typeof(Service), new Uri(baseAddress));  
            host.AddServiceEndpoint(typeof(IImageServer), new WebHttpBinding(), "").Behaviors.Add(new WebHttpBehavior());  
            host.Open();  
            Console.WriteLine("Service is running");  
            Console.Write("Press ENTER to close the host");  
            Console.ReadLine();  
            host.Close();  
  
        }  
    }  
}  
```
