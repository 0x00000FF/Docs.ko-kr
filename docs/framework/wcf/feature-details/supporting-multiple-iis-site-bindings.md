# <a name="supporting-multiple-iis-site-bindings"></a><span data-ttu-id="047d5-102">여러 IIS 사이트 바인딩 지원</span><span class="sxs-lookup"><span data-stu-id="047d5-102">Supporting Multiple IIS Site Bindings</span></span>

## <a name="multiple-base-addresses"></a><span data-ttu-id="047d5-109">여러 기본 주소</span><span class="sxs-lookup"><span data-stu-id="047d5-109">Multiple Base Addresses</span></span>  
 <span data-ttu-id="047d5-110">이 기능은 IIS에서 호스팅되는 WCF 서비스에 사용할 수만 있습니다.</span><span class="sxs-lookup"><span data-stu-id="047d5-110">This feature is only available to WCF services that are hosted under IIS.</span></span> <span data-ttu-id="047d5-111">이 기능은 기본적으로 사용하지 않도록 설정되어 있습니다.</span><span class="sxs-lookup"><span data-stu-id="047d5-111">This feature is not enabled by default.</span></span> <span data-ttu-id="047d5-112">추가 사용 하도록 설정 해야 합니다는 `multipleSiteBindingsEnabled` 특성을 <`serviceHostingEnvironment`> Web.config 파일에 요소 파일 및 설정 `true`다음 예제에서와 같이 합니다.</span><span class="sxs-lookup"><span data-stu-id="047d5-112">To enable it you must add the `multipleSiteBindingsEnabled` attribute to the <`serviceHostingEnvironment`> element in your Web.config file and set it to `true`, as shown in the following example.</span></span>  
  
```xml  
<serviceHostingEnvironment multipleSiteBindingsEnabled="true"/>  
```  
  
 <span data-ttu-id="047d5-113">IIS에서 WCF 서비스 호스팅, IIS 응용 프로그램을 포함 하는 가상 디렉터리 URI에 따라 하나의 기본 주소를 만듭니다.</span><span class="sxs-lookup"><span data-stu-id="047d5-113">When hosting a WCF service under IIS, IIS creates one base address for you based on the URI to the virtual directory that contains the application.</span></span> <span data-ttu-id="047d5-114">인터넷 정보 서비스 관리자를 사용하여 웹 사이트에 하나 이상의 바인딩을 추가하여 동일한 프로토콜을 사용하는 기본 주소를 추가할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="047d5-114">You can add additional base addresses that use the same protocol by using Internet Information Services Manager to add one or more bindings to your Web site.</span></span> <span data-ttu-id="047d5-115">각 바인딩에 대해 프로토콜(HTTP 또는 HTTPS), IP 주소, 포트 및 호스트 이름을 지정합니다.</span><span class="sxs-lookup"><span data-stu-id="047d5-115">For each binding specify a protocol (HTTP or HTTPS), an IP address, a port, and a host name.</span></span> <span data-ttu-id="047d5-116">인터넷 정보 서비스 관리자를 사용 하는 방법에 대 한 자세한 내용은 참조 하십시오 [IIS 관리자 (IIS 7)](https://go.microsoft.com/fwlink/?LinkId=164057)합니다.</span><span class="sxs-lookup"><span data-stu-id="047d5-116">For more information about using Internet Information Services Manager, see [IIS Manager (IIS 7)](https://go.microsoft.com/fwlink/?LinkId=164057).</span></span> <span data-ttu-id="047d5-117">사이트에 바인딩을 추가 하는 방법에 대 한 자세한 내용은 참조 하세요. [(IIS 7) 웹 사이트 만들기](https://go.microsoft.com/fwlink/?LinkId=164060)</span><span class="sxs-lookup"><span data-stu-id="047d5-117">For more information about adding bindings to a site, see [Create a Web Site (IIS 7)](https://go.microsoft.com/fwlink/?LinkId=164060)</span></span>  
  
 <span data-ttu-id="047d5-118">동일한 사이트에 대 한 여러 기본 주소 지정 스키마 가져오기 및 서비스에 의해 생성 된 WSDL/MEX 정보에 WCF 도움말 페이지의 내용을 영향을 줍니다.</span><span class="sxs-lookup"><span data-stu-id="047d5-118">Specifying multiple base addresses for the same site affects the content of the WCF Help page, importing schema, and the WSDL/MEX information generated by the service.</span></span> <span data-ttu-id="047d5-119">WCF 도움말 페이지를 사용 하는 서비스와 통신할 수 있는 WCF 클라이언트 생성 명령줄이 표시 됩니다.</span><span class="sxs-lookup"><span data-stu-id="047d5-119">The WCF Help page displays the command line to use to generate a WCF client that can communicate with the service.</span></span> <span data-ttu-id="047d5-120">이 명령줄에는 웹 사이트에 대한 IIS 바인딩에 지정된 첫 번째 주소만 포함됩니다.</span><span class="sxs-lookup"><span data-stu-id="047d5-120">This command line contains only the first address specified in the IIS binding for the Web site.</span></span> <span data-ttu-id="047d5-121">마찬가지로 스키마를 가져올 때도 IIS 바인딩에 지정된 첫 번째 기본 주소만 사용됩니다.</span><span class="sxs-lookup"><span data-stu-id="047d5-121">Similarly when importing schema, only the first base address specified in the IIS binding is used.</span></span> <span data-ttu-id="047d5-122">WSDL 및 MEX 데이터에는 IIS 바인딩에 지정된 모든 기본 주소가 포함됩니다.</span><span class="sxs-lookup"><span data-stu-id="047d5-122">WSDL and MEX data contain all the base addresses specified in the IIS bindings.</span></span>
