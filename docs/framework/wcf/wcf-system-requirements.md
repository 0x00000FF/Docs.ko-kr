# <a name="wcf-system-requirements"></a>WCF 시스템 요구 사항

WCF (Windows Communication Foundation)는 분산 응용 프로그램을 만드는 데 사용 되는 통신 인프라입니다. 다음은 WCF를 실행할 수 있도록 하는 요구 사항 목록입니다.

## <a name="system-requirements"></a>시스템 요구 사항

WCF는 기본적으로에 [!INCLUDE[wv](../../../includes/wv-md.md)]설치 됩니다.

WCF는, [!INCLUDE[wxpsp2](../../../includes/wxpsp2-md.md)] [!INCLUDE[ws2003r2](../../../includes/ws2003r2-md.md)]또는 [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)]에 설치할 수도 있습니다.

> [!NOTE]
> WCF의 MSMQ (메시지 큐 [!INCLUDE[wv](../../../includes/wv-md.md)]) 기능은 [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)], [!INCLUDE[ws2003r2](../../../includes/ws2003r2-md.md)], 및 [!INCLUDE[wxppro](../../../includes/wxppro-md.md)]에서만 지원 됩니다.
