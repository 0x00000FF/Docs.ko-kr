# WCF 시스템 요구 사항
[!INCLUDE[indigo1](../../../includes/indigo1-md.md)]는 분산 응용 프로그램을 만드는 데 사용되는 통신 인프라입니다.  다음 목록에서는 [!INCLUDE[indigo2](../../../includes/indigo2-md.md)]를 실행하는 데 필요한 요구 사항을 보여 줍니다.  
  
## 시스템 요구 사항  
 [!INCLUDE[indigo2](../../../includes/indigo2-md.md)]는 기본적으로 [!INCLUDE[wv](../../../includes/wv-md.md)]에 설치됩니다.  
  
 [!INCLUDE[indigo2](../../../includes/indigo2-md.md)]를 [!INCLUDE[wxpsp2](../../../includes/wxpsp2-md.md)], [!INCLUDE[ws2003r2](../../../includes/ws2003r2-md.md)] 또는 [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)]에 설치할 수도 있습니다.  
  
 **참고** [!INCLUDE[indigo2](../../../includes/indigo2-md.md)]의 MSMQ\(메시지 큐\) 기능은 [!INCLUDE[wv](../../../includes/wv-md.md)], [!INCLUDE[ws2003r2](../../../includes/ws2003r2-md.md)], [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)] 및 [!INCLUDE[wxppro](../../../includes/wxppro-md.md)]에서만 지원됩니다.
