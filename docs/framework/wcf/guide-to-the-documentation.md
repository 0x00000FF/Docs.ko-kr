# 설명서에 대한 안내
여기에서는 [!INCLUDE[indigo1](../../../includes/indigo1-md.md)] 설명서에 대한 참고 자료를 제공합니다.구체적인 관심사와 지식 수준에 따라 그룹화된 링크 문서부터 보시는 것이 좋습니다.  
  
 설명서와 도움말 뷰어의 독립 실행형 버전을 설치하려면 [Microsoft Windows SDK v 7.1](http://go.microsoft.com/fwlink/?LinkID=194146&clcid=0x412)을 다운로드합니다.  
  
## Windows Communication Foundation 프로그래밍을 처음 사용하는 경우  
  
-   [!INCLUDE[indigo2](../../../includes/indigo2-md.md)]를 사용하여 처음으로 프로그래밍을 하려는 경우, 작동하는 샘플 응용 프로그램을 보기만 하려면 [Windows Communication Foundation 샘플](../../../docs/framework/wcf/samples/index.md)에 나열된 항목을 참조하십시오.  
  
-   [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] 서비스 및 클라이언트를 만드는 기본 단계를 설명하는 자습서는 [초보자를 위한 자습서](../../../docs/framework/wcf/getting-started-tutorial.md)를 참조하십시오.  
  
-   [!INCLUDE[indigo2](../../../includes/indigo2-md.md)]의 개념에 관심이 있는 경우에는 [개념적 개요](../../../docs/framework/wcf/conceptual-overview.md) 단원의 항목을 참조하십시오.  
  
-   클라이언트\/서버 보안 구성의 그래픽 예제를 보려면 [일반적인 보안 시나리오](../../../docs/framework/wcf/feature-details/common-security-scenarios.md)를 참조하십시오.  
  
## 세부 프로그래밍  
  
-   응용 프로그램 개발을 시작하려면 [기본 WCF 프로그래밍](../../../docs/framework/wcf/basic-wcf-programming.md)을 참조하십시오.  
  
-   [!INCLUDE[indigo2](../../../includes/indigo2-md.md)]의 특정 기능에 대한 참고 자료를 찾는 경우에는 [WCF 기능 정보](../../../docs/framework/wcf/feature-details/index.md)의 항목들을 참조하십시오.  
  
-   사용자 요구 사항을 충족하도록 [!INCLUDE[indigo2](../../../includes/indigo2-md.md)]를 확장하거나 사용자 지정하려면 [WCF 확장](../../../docs/framework/wcf/extending/extending-wcf.md)을 참조하십시오.  
  
-   [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] 응용 프로그램을 만들고 디버깅하는 데 도움이 되는 도구에 대한 자세한 내용은 [Windows Communication Foundation 도구](../../../docs/framework/wcf/tools.md)를 참조하십시오.  
  
-   XML 파일을 사용하여 구성하는 것이 [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] 서비스 및 클라이언트를 프로그래밍하는 기본적인 방법입니다.구성 파일에서 사용하는 XML 요소에 대한 참조 문서는 [WCF 구성 스키마](../../../docs/framework/configure-apps/file-schema/wcf/index.md)를 참조하십시오.  
  
## 문제 해결  
 일반적인 [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] 문제를 해결하는 방법은 [WCF 문제 해결 퀵 스타트](../../../docs/framework/wcf/wcf-troubleshooting-quickstart.md)를 참조하십시오.  
  
## 다른 기술과 함께 Windows Communication Foundation 사용  
  
-   ASP.NET 클라이언트와 통신하는 서비스를 만들려면 [방법: ASP.NET 웹 서비스 클라이언트와 상호 운용하도록 WCF 서비스 구성](../../../docs/framework/wcf/feature-details/config-wcf-service-with-aspnet-web-service.md)을 참조하십시오.  
  
-   [!INCLUDE[dnprdnshort](../../../includes/dnprdnshort-md.md)] 원격과의 통합에 대해서는 [.NET Remoting 응용 프로그램을 WCF로 마이그레이션](../../../docs/framework/wcf/feature-details/migrating-net-remoting-applications-to-wcf.md)에서 설명합니다.  
  
-   기존 COM\+ 응용 프로그램을 [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] 서비스 또는 클라이언트와 통합하려면 [COM\+ 응용 프로그램과 통합 개요](../../../docs/framework/wcf/feature-details/integrating-with-com-plus-applications-overview.md)를 참조하십시오.  
  
-   기존 COM 응용 프로그램을 [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] 서비스 또는 클라이언트와 통합하려면 [COM 응용 프로그램과 통합](../../../docs/framework/wcf/feature-details/integrating-with-com-applications.md)을 참조하십시오.  
  
-   기존 MSMQ 응용 프로그램을 [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] 서비스 또는 클라이언트와 통합하려면 [방법: 대기 중인 메시지와 WCF 끝점 교환](../../../docs/framework/wcf/feature-details/how-to-exchange-queued-messages-with-wcf-endpoints.md) 및 [방법: WCF 끝점 및 메시지 큐 응용 프로그램과 메시지 교환](../../../docs/framework/wcf/feature-details/how-to-exchange-messages-with-wcf-endpoints-and-message-queuing-applications.md)을 참조하십시오.  
  
-   서비스를 호스팅하기 위해 IIS\(인터넷 정보 서비스\)를 사용하려면 [서비스 호스팅](../../../docs/framework/wcf/hosting-services.md)을 참조하십시오.  
  
-   [!INCLUDE[indigo2](../../../includes/indigo2-md.md)]를 사용하여 WSE\(Web Services Extensions\) 3.0 서비스를 사용하려면 [방법: WSE 3.0 서비스 액세스](../../../docs/framework/wcf/feature-details/how-to-access-a-wse-3-0-service-with-a-wcf-client.md)을 참조하십시오.  
  
## Windows Communication Foundation에서 지원되는 WS\-\* 프로토콜  
 시스템 제공 바인딩에서 지원되는 프로토콜의 목록을 보려면 [시스템 제공 상호 운용성 바인딩에서 지원하는 웹 서비스 프로토콜](../../../docs/framework/wcf/feature-details/web-services-protocols-supported-by-system-provided-interoperability-bindings.md)을 참조하십시오.시스템 제공 바인딩 목록을 보려면 [시스템 제공 바인딩](../../../docs/framework/wcf/system-provided-bindings.md)을 참조하십시오.
