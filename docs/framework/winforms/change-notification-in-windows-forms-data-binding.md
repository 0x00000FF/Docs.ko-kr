 <span data-ttu-id="32cb6-106">데이터 바인딩의 종류에 따라 다음과 같은 다양 한 종류의 변경 알림이 있습니다.</span><span class="sxs-lookup"><span data-stu-id="32cb6-106">There are different kinds of change notification, depending on the kind of data binding:</span></span>
