# <a name="helpprovider-component-windows-forms"></a>HelpProvider 구성 요소(Windows Forms)
Windows Forms `HelpProvider` 구성 요소는 HTML 도움말 1.x 도움말 파일(HTML Help Workshop으로 생성된 .chm 파일 또는 .htm 파일)을 Windows 기반 애플리케이션에 연결하는 데 사용됩니다.  
  
## <a name="in-this-section"></a>섹션 내용  
 [HelpProvider 구성 요소 개요](helpprovider-component-overview-windows-forms.md)  
 HTML 도움말 파일을 Windows 기반 애플리케이션에 연결할 수 있게 해주는 `HelpProvider` 구성 요소의 일반적인 개념을 소개합니다.
