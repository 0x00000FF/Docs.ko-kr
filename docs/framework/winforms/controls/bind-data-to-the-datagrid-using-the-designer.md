### <a name="to-bind-the-control-to-a-data-source"></a><span data-ttu-id="4bd70-117">데이터 소스에 컨트롤을 바인딩하려면</span><span class="sxs-lookup"><span data-stu-id="4bd70-117">To bind the control to a data source</span></span>  
  
1.  <span data-ttu-id="4bd70-118">스마트 태그 문자 모양을 클릭 (![스마트 태그 문자 모양](../../../../docs/framework/winforms/controls/media/vs-winformsmttagglyph.gif "VS_WinFormSmtTagGlyph"))의 오른쪽 위 모서리에는 <xref:System.Windows.Forms.DataGridView> 제어 합니다.</span><span class="sxs-lookup"><span data-stu-id="4bd70-118">Click the smart tag glyph (![Smart Tag Glyph](../../../../docs/framework/winforms/controls/media/vs-winformsmttagglyph.gif "VS_WinFormSmtTagGlyph")) on the upper-right corner of the <xref:System.Windows.Forms.DataGridView> control.</span></span>  
  
2.  <span data-ttu-id="4bd70-119">**데이터 소스 선택** 옵션에서 드롭다운 화살표를 클릭합니다.</span><span class="sxs-lookup"><span data-stu-id="4bd70-119">Click the drop-down arrow for the **Choose Data Source** option.</span></span>  
  
3.  <span data-ttu-id="4bd70-120">프로젝트에 데이터 소스가 아직 없는 경우 **프로젝트 데이터 소스 추가**를 클릭하고 마법사에 의해 표시된 단계를 따릅니다.</span><span class="sxs-lookup"><span data-stu-id="4bd70-120">If your project does not already have a data source, click **Add Project Data Source** and follow the steps indicated by the wizard.</span></span>  
  
     <span data-ttu-id="4bd70-121">자세한 내용은 [데이터 소스 구성 마법사](https://msdn.microsoft.com/library/c4df7de5-5da0-4064-940c-761dd6d9e28f)를 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="4bd70-121">For more information, see [Data Source Configuration Wizard](https://msdn.microsoft.com/library/c4df7de5-5da0-4064-940c-761dd6d9e28f).</span></span> <span data-ttu-id="4bd70-122">새 데이터 소스은 **데이터 소스 선택** 드롭다운 창에 표시됩니다.</span><span class="sxs-lookup"><span data-stu-id="4bd70-122">Your new data source will appear in the **Choose Data Source** drop-down window.</span></span> <span data-ttu-id="4bd70-123">새 데이터 소스에 단일 데이터베이스 테이블과 같은 하나의 구성원만이 포함되는 경우 컨트롤은 해당 구성원에 자동으로 바인딩됩니다.</span><span class="sxs-lookup"><span data-stu-id="4bd70-123">If your new data source contains only one member, such as a single database table, the control will automatically bind to that member.</span></span> <span data-ttu-id="4bd70-124">그렇지 않으면 다음 단계를 계속합니다.</span><span class="sxs-lookup"><span data-stu-id="4bd70-124">Otherwise, continue to the next step.</span></span>  
  
4.  <span data-ttu-id="4bd70-125">**기타 데이터 소스** 및 **프로젝트 데이터 소스** 노드가 아직 확장되지 않은 경우 확장한 다음 컨트롤을 바인딩할 데이터 소스를 선택합니다.</span><span class="sxs-lookup"><span data-stu-id="4bd70-125">Expand the **Other Data Sources** and **Project Data Sources** nodes if they are not already expanded, and then select the data source to bind the control to.</span></span>  
  
5.  <span data-ttu-id="4bd70-126">데이터 원본에 둘 이상의 멤버가 포함 되어 경우와 같이 만든 경우는 <xref:System.Data.DataSet?displayProperty=nameWithType> 여러 테이블을 포함 하는 데이터 소스를 확장 하 고 바인딩할 특정 멤버를 선택 합니다.</span><span class="sxs-lookup"><span data-stu-id="4bd70-126">If your data source contains more than one member, such as if you have created a <xref:System.Data.DataSet?displayProperty=nameWithType> that contains multiple tables, expand the data source, and then select the specific member to bind to.</span></span>  
  
6.  <span data-ttu-id="4bd70-127">마스터/세부 관계를 만들려면 합니다 **데이터 소스 선택** 잠시 드롭다운 창 <xref:System.Windows.Forms.DataGridView> 컨트롤을 확장 하 고는 <xref:System.Windows.Forms.BindingSource> 부모 테이블에 대해 생성 한 다음 목록에서 관련된 자식 테이블을 선택 표시 합니다.</span><span class="sxs-lookup"><span data-stu-id="4bd70-127">To create a master/detail relationship, in the **Choose Data Source** drop-down window for a second <xref:System.Windows.Forms.DataGridView> control, expand the <xref:System.Windows.Forms.BindingSource> created for the parent table, and then select the related child table from the list shown.</span></span>  
  
    > [!NOTE]
    >  <span data-ttu-id="4bd70-128">프로젝트에 데이터 소스가 이미 있으면 **데이터 소스** 창을 사용하여 데이터 양식을 만들 수도 있습니다.</span><span class="sxs-lookup"><span data-stu-id="4bd70-128">If your project already has a data source, you can also use the **Data Sources** window to create a data form.</span></span> <span data-ttu-id="4bd70-129">자세한 내용은 [데이터 소스 창](https://msdn.microsoft.com/library/0d20f699-cc95-45b3-8ecb-c7edf1f67992)을 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="4bd70-129">For more information, see [Data Sources Window](https://msdn.microsoft.com/library/0d20f699-cc95-45b3-8ecb-c7edf1f67992).</span></span>
