# Windows Forms 컨트롤
Windows Forms 응용 프로그램의 사용자 인터페이스를 디자인 및 수정할 때 컨트롤의 추가, 맞춤, 위치 지정 등이 필요합니다.  컨트롤은 폼 개체에 포함된 개체입니다.  각 컨트롤 형식에는 특정 용도에 맞게 사용할 수 있는 자체의 속성, 메서드 및 이벤트 집합이 있습니다.  컨트롤은 디자이너에서 조작할 수 있고 코드를 작성하여 런타임에 동적으로 추가할 수 있습니다.
