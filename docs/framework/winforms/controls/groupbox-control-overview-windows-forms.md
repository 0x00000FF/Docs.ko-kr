# GroupBox 컨트롤 개요(Windows Forms)
Windows Forms <xref:System.Windows.Forms.GroupBox> 컨트롤은 다른 컨트롤을 구별할 수 있도록 그룹화하는 데 사용됩니다.  일반적으로 그룹 상자를 사용하여 폼을 기능별로 세분합니다.  예를 들면 사용할 야간 운송 수단과 같은 우편물 옵션을 지정하는 주문 양식을 만들 수 있습니다.  모든 옵션을 하나의 그룹 상자에 그룹화하면 사용자에게 논리적이고 시각적인 신호가 제공될 뿐만 아니라 <xref:System.Windows.Forms.GroupBox> 컨트롤을 이동할 경우 포함된 모든 컨트롤이 함께 이동되므로 디자인 타임에 컨트롤을 쉽게 이동할 수 있습니다.  
  
 그룹 상자의 캡션은 <xref:System.Windows.Forms.GroupBox.Text%2A> 속성을 사용하여 정의합니다.  자세한 내용은 [방법: Windows Forms 컨트롤에서 표시하는 텍스트 설정](../../../../docs/framework/winforms/controls/how-to-set-the-text-displayed-by-a-windows-forms-control.md)를 참조하십시오.  
  
## GroupBox 및 Panel  
 <xref:System.Windows.Forms.GroupBox> 컨트롤은 <xref:System.Windows.Forms.Panel> 컨트롤과 비슷하지만 <xref:System.Windows.Forms.GroupBox> 컨트롤에만 캡션을 표시할 수 있고 <xref:System.Windows.Forms.Panel> 컨트롤에만 스크롤 막대가 있다는 점이 다릅니다.
