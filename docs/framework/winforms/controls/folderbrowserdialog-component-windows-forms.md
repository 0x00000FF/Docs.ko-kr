# <a name="folderbrowserdialog-component-windows-forms"></a>FolderBrowserDialog 구성 요소(Windows Forms)
Windows Forms `FolderBrowserDialog` 구성 요소는 사용자가 폴더를 찾아보거나 선택 하거나 새 폴더를 만들 수 있는 인터페이스를 표시 합니다. 이는 파일을 검색 하 고 선택 하는 데 사용 되는 [OpenFileDialog 구성](openfiledialog-component-windows-forms.md) 요소 구성 요소를 보완 합니다.
