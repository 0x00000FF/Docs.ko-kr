# FolderBrowserDialog 구성 요소(Windows Forms)
Windows Forms `FolderBrowserDialog` 구성 요소는 사용자가 폴더를 찾아 선택하거나 새 폴더를 만들 수 있는 인터페이스를 표시합니다.  이 구성 요소는 파일을 찾아 선택하는 데 사용되는 [OpenFileDialog 구성 요소](../../../../docs/framework/winforms/controls/openfiledialog-component-windows-forms.md) 구성 요소를 보완하는 역할을 합니다.

 이 컨트롤의 정의, 주요 기능 및 속성을 설명합니다.  
  
 [방법: Windows Forms FolderBrowserDialog 구성 요소를 사용하여 폴더 선택](../../../../docs/framework/winforms/controls/how-to-choose-folders-with-the-windows-forms-folderbrowserdialog-component.md)  
 대화 상자에서 선택한 폴더를 프로그래밍 방식으로 추출하는 방법과 구성 요소의 기타 선택적인 속성을 사용하여 작업하는 방법을 설명합니다.
