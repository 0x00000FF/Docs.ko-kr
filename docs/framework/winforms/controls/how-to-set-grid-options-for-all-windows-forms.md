# 방법: 모든 Windows Forms에 대한 모눈 옵션 설정
[!INCLUDE[vsprvs](../../../../includes/vsprvs-md.md)]의 개발 환경에서 작업하는 데 익숙해지면 Windows Forms 디자이너에서 사용하는 모든 폼과 프로젝트에 대한 기본 설정을 지정할 수 있습니다.  
  
> [!NOTE]
>  표시되는 대화 상자와 메뉴 명령은 활성 설정이나 버전에 따라 도움말에서 설명하는 것과 다를 수 있습니다.  설정을 변경하려면 **도구** 메뉴에서 **설정 가져오기 및 내보내기**를 선택합니다.  자세한 내용은 [Customizing Development Settings in Visual Studio](http://msdn.microsoft.com/ko-kr/22c4debb-4e31-47a8-8f19-16f328d7dcd3)을 참조하십시오.  
  
### 전역 Windows Forms 옵션을 설정하려면  
  
1.  **도구** 메뉴에서 **옵션**을 선택합니다.  
  
2.  **옵션** 대화 상자의 왼쪽 창에서 **Windows Forms 디자이너**를 클릭합니다.  
  
     오른쪽 창의 **레이아웃 설정** 제목 아래에서 새로 만드는 모든 폼의 기본 모눈 설정을 지정할 수 있습니다.  이러한 설정에는 모눈 크기, 컨트롤을 모눈에 맞춤 여부 및 기본값으로 모눈을 표시할지 여부가 포함됩니다.  또한 **SnapToGrid** 및 **SnapLines** 레이아웃 모드 중에서 선택할 수 있습니다.  맞춤선에 대한 자세한 내용은 [연습: 맞춤선을 사용하여 Windows Forms에서 컨트롤 정렬](../../../../docs/framework/winforms/controls/walkthrough-arranging-controls-on-windows-forms-using-snaplines.md)을 참조하십시오.
