# ToolStripStatusLabel 컨트롤 개요
<xref:System.Windows.Forms.ToolStripStatusLabel>은 <xref:System.Windows.Forms.StatusStrip>에 대한 레이블입니다.  <xref:System.Windows.Forms.Label> 또는 <xref:System.Windows.Forms.ToolStripLabel>과 같이 <xref:System.Windows.Forms.ToolStripStatusLabel>은 텍스트나 이미지 또는 둘 다에 대한 클릭할 수 없는 표시 영역을 제공합니다.  <xref:System.Windows.Forms.ToolStripStatusLabel>은 <xref:System.Windows.Forms.StatusStrip>에 의해 호스팅됩니다.  
  
### 중요한 ToolStripStatusLabel 멤버  
  
|Name|설명|  
|----------|--------|  
|<xref:System.Windows.Forms.ToolStripStatusLabel.Spring%2A>|폼의 크기가 조정될 때 <xref:System.Windows.Forms.ToolStripStatusLabel>이 <xref:System.Windows.Forms.StatusStrip>의 사용할 수 있는 공간을 자동으로 채울지 여부를 나타내는 값을 가져오거나 설정합니다.|  
|<xref:System.Windows.Forms.ToolStripStatusLabel.BorderSides%2A>|<xref:System.Windows.Forms.ToolStripStatusLabel>에 테두리를 표시할 방향을 나타내는 값을 가져오거나 설정합니다.|  
|<xref:System.Windows.Forms.ToolStripStatusLabel.BorderStyle%2A>|<xref:System.Windows.Forms.ToolStripStatusLabel>의 테두리 스타일을 가져오거나 설정합니다.|
