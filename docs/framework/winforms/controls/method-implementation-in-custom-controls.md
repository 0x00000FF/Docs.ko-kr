## <a name="overloaded-methods"></a><span data-ttu-id="341d8-117">오버로드된 메서드</span><span class="sxs-lookup"><span data-stu-id="341d8-117">Overloaded Methods</span></span>  
 <span data-ttu-id="341d8-118">컨트롤 사용자가 메서드에 서로 다른 매개 변수 조합을 제공할 수 있도록 하려면 명시적 데이터 형식을 사용하여 메서드의 여러 오버로드를 제공합니다.</span><span class="sxs-lookup"><span data-stu-id="341d8-118">If you want to allow users of your control to supply different combinations of parameters to a method, provide multiple overloads of the method, using explicit data types.</span></span> <span data-ttu-id="341d8-119">모든 데이터 형식을 포함할 수 있는 `As Object`로 선언된 매개 변수는 만들지 않아야 합니다. 이러한 매개 변수를 만들면 오류가 발생해도 테스트에서 확인되지 않을 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="341d8-119">Avoid creating parameters declared `As Object` that can contain any data type, as this can lead to errors that might not be caught in testing.</span></span>
