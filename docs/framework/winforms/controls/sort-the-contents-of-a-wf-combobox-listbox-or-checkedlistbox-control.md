# <a name="how-to-sort-the-contents-of-a-windows-forms-combobox-listbox-or-checkedlistbox-control"></a>방법: Windows Forms ComboBox, ListBox 또는 CheckedListBox 컨트롤의 내용 정렬
Windows Forms 컨트롤은 데이터 바인딩된 경우에는 정렬 되지 않습니다. 정렬 된 데이터를 표시 하려면 정렬을 지 원하는 데이터 원본을 사용 하 고 데이터 원본에 정렬 합니다. 정렬을 지 원하는 데이터 원본은 데이터 뷰, 데이터 뷰 관리자 및 정렬 된 배열입니다.  
  
 컨트롤이 데이터 바인딩되지 않은 경우 정렬할 수 있습니다.  
  
### <a name="to-sort-the-list"></a>목록을 정렬 하려면  
  
1. `Sorted` 속성을 `true`로 설정합니다.  
  
     이 설정은 모든 기존 목록 항목의 위치를 정렬 된 순서로 설정 합니다.
