# 방법: Windows Forms 간에 컨트롤 복사
컨트롤을 같은 폼에 복사하거나 프로젝트 내의 다른 폼에 복사할 수 있습니다. 또한 컨트롤을 클립보드에 복사하여 다른 솔루션에서 사용할 수도 있습니다.  
  
### 컨트롤을 복사하려면  
  
1.  컨트롤을 선택한 다음 **편집** 메뉴에서 **복사**를 선택합니다.  
  
 이렇게 하면 이 형식의 컨트롤을 수용하는 모든 폼에 이 컨트롤을 붙여넣을 수 있을 뿐만 아니라  클립보드에도 컨트롤이 추가됩니다.
