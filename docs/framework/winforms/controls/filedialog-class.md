# FileDialog 클래스
Windows Forms <xref:System.Windows.Forms.FileDialog> 클래스는 <xref:System.Windows.Forms.OpenFileDialog> 및 <xref:System.Windows.Forms.SaveFileDialog> 구성 요소에 대한 공통 기본 클래스입니다.  <xref:System.Windows.Forms.FileDialog> 클래스를 변경하면 응용 프로그램이 실행되는 Windows의 버전에 따라 이러한 대화 상자의 모양과 동작에 영향을 줄 수 있습니다.
