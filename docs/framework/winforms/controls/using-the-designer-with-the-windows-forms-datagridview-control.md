# Windows Forms DataGridView 컨트롤에 디자이너 사용
Visual Studio에서는 코드를 작성하지 않고도 여러 가지 설정 작업을 수행할 수 있도록 `DataGridView` 컨트롤에 대한 디자이너 지원 기능을 제공합니다.  이러한 작업에는 컨트롤을 데이터 소스에 바인딩하거나, 데이터를 표시하는 데 사용되는 열을 수정하거나, 컨트롤의 모양과 기본 동작을 조정하는 등의 작업이 포함됩니다.
