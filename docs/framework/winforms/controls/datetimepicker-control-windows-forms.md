# DateTimePicker 컨트롤(Windows Forms)
Windows Forms `DateTimePicker` 컨트롤을 사용하면 날짜 또는 시간 목록에서 하나의 항목을 선택할 수 있습니다.  이 컨트롤은 데이터를 표시할 때 두 부분으로 나타납니다. 텍스트로 날짜를 표시하는 드롭다운 목록 부분과 목록 옆에 있는 아래쪽 화살표를 클릭했을 때 나타나는 모눈 부분이 있습니다.
