# Windows Forms DataGridView 컨트롤의 데이터 입력
`DataGridView` 컨트롤은 사용자가 컨트롤의 데이터를 추가하거나 수정하는 방법을 변경할 수 있는 여러 기능을 제공합니다.  예를 들어, 새 행에 대한 기본값을 제공하고 오류 발생 시 사용자에게 알림으로써 데이터 입력을 보다 효율적으로 수행할 수 있습니다.
