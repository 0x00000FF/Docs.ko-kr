## <a name="to-set-the-tab-order-of-a-control"></a><span data-ttu-id="e4bd1-107">컨트롤의 탭 순서를 설정 하려면</span><span class="sxs-lookup"><span data-stu-id="e4bd1-107">To set the tab order of a control</span></span>

Z 순서에 대 한 자세한 내용은 [Windows Forms에서 개체 계층화](how-to-layer-objects-on-windows-forms.md)를 참조 하세요.</span><span class="sxs-lookup"><span data-stu-id="e4bd1-115">The z-order determines which controls are in front of other controls.)
