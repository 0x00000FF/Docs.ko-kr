# <a name="errorprovider-component-windows-forms"></a>ErrorProvider 구성 요소(Windows Forms)
Windows Forms `ErrorProvider` 구성 요소를 사용 하 여 사용자가 잘못 된 기능을 방해 하지 않는 방식으로 표시 합니다. 일반적으로 폼에서 사용자 입력의 유효성을 검사 하거나 데이터 집합 내에 오류를 표시 하는 것과 함께 사용 됩니다.
