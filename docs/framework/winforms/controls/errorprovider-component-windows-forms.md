# ErrorProvider 구성 요소(Windows Forms)
Windows Forms `ErrorProvider` 구성 요소를 사용하면 오류 발생 사실을 사용자에게 방해가 되지 않는 방법으로 알려줄 수 있습니다.  일반적으로 이 구성 요소는 폼에 대한 사용자 입력의 유효성을 검사하거나 데이터 집합의 오류를 표시하는 데 사용됩니다.
