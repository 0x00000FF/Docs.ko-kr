# PrintDocument 구성 요소(Windows Forms)
Windows Forms `PrintDocument` 구성 요소는 인쇄 대상을 설명하는 속성을 설정한 다음 Windows 기반 응용 프로그램 내에서 문서를 인쇄하는 데 사용됩니다.  모든 문서 인쇄 측면의 명령에 포함되도록 <xref:System.Windows.Forms.PrintDialog> 구성 요소와 함께 사용할 수 있습니다.  
  
## 단원 내용  
 [PrintDocument 구성 요소 개요](../../../../docs/framework/winforms/controls/printdocument-component-overview-windows-forms.md)  
 인쇄 대상을 설명하는 속성을 설정할 수 있게 해주고 Windows 기반 응용 프로그램에서 인쇄를 시작하는 `PrintDocument` 구성 요소의 일반적인 개념을 소개합니다.  
  
## 참조  
 <xref:System.Drawing.Printing.PrintDocument>  
 클래스 및 해당 멤버에 대한 참조 정보를 제공합니다.  
  
## 관련 단원  
 [Windows Forms 인쇄 지원](../../../../docs/framework/winforms/advanced/windows-forms-print-support.md)  
 Windows Forms와 관련된 인쇄 항목의 목록을 제공합니다.  
  
 [PrintDialog 구성 요소](../../../../docs/framework/winforms/controls/printdialog-component-windows-forms.md)  
 사용자가 프린터 및 인쇄할 페이지를 선택하고 인쇄 관련 설정을 결정하는 데 사용할 수 있는 미리 구성된 대화 상자를 표시할 수 있는 <xref:System.Windows.Forms.PrintDialog> 구성 요소의 일반적인 개념을 소개합니다.  
  
 [PrintPreviewControl 컨트롤](../../../../docs/framework/winforms/controls/printpreviewcontrol-control-windows-forms.md)  
 고유한 인쇄 미리 보기 대화 상자 또는 구성 요소를 디자인하는 데 사용할 수 있는 <xref:System.Windows.Forms.PrintPreviewControl>의 일반적인 개념을 소개합니다.  
  
 [PrintPreviewDialog 컨트롤](../../../../docs/framework/winforms/controls/printpreviewdialog-control-windows-forms.md)  
 사용자가 인쇄 시 표시되는 모양대로 특정 버전의 문서를 보는 데 사용할 수 있는 미리 구성된 대화 상자를 표시할 수 있게 해주는 <xref:System.Windows.Forms.PrintPreviewDialog> 컨트롤의 일반적인 개념을 소개합니다.
