# 방법: 데이터 바인딩된 Windows Forms DataGridView 컨트롤에서 열 자동 생성
다음 코드 예제에서는 <xref:System.Windows.Forms.DataGridView> 컨트롤의 바인딩된 데이터 소스에서 열을 표시하는 방법을 보여 줍니다.  <xref:System.Windows.Forms.DataGridView.AutoGenerateColumns%2A> 속성 값이 `true`\(기본값\)이면 데이터 소스 테이블의 각 열에 대해 <xref:System.Windows.Forms.DataGridViewColumn>이 만들어집니다.  
  
 <xref:System.Windows.Forms.DataGridView> 컨트롤에 이미 열이 있는 경우 <xref:System.Windows.Forms.DataGridViewComboBoxColumn.DataSource%2A> 속성을 설정하면 기존의 바인딩된 열과 데이터 소스의 열이 비교되어 일치하는 내용이 유지됩니다.  바인딩되지 않은 열은 항상 유지됩니다.  데이터 소스의 열과 일치하지 않는 바인딩된 열은 제거됩니다.  데이터 소스의 열에 컨트롤의 열과 일치하는 내용이 없는 경우 새 <xref:System.Windows.Forms.DataGridViewColumn> 개체가 생성되고 <xref:System.Windows.Forms.DataGridView.Columns%2A> 컬렉션의 끝에 이 개체가 추가됩니다.
