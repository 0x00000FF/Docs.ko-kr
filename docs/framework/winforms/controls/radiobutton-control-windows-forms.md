# <a name="radiobutton-control-windows-forms"></a>RadioButton 컨트롤(Windows Forms)
Windows Forms `RadioButton` 컨트롤은 둘 이상의 상호 배타적인 선택 항목 집합을 사용자에 게 제공 합니다. 라디오 단추와 확인란이 비슷하게 작동 하는 것 처럼 보일 수 있지만 중요 한 차이점이 있습니다. 사용자가 라디오 단추를 선택 하면 동일한 그룹의 다른 라디오 단추도 선택할 수 없습니다.
