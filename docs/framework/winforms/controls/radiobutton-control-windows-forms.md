# RadioButton 컨트롤(Windows Forms)
Windows Forms `RadioButton` 컨트롤을 사용하면 두 개 이상의 항목으로 구성된 집합에서 하나의 항목을 상호 배타적으로 선택하도록 할 수 있습니다.  라디오 단추와 확인란은 유사한 기능을 가지고 있지만 중요한 차이가 있습니다. 라디오 단추는 하나의 그룹에서 한 개만 선택할 수 있습니다.
