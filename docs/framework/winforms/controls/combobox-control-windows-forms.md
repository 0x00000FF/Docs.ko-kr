# <a name="combobox-control-windows-forms"></a>ComboBox 컨트롤(Windows Forms)
Windows Forms `ComboBox` 컨트롤은 드롭다운 콤보 상자에 데이터를 표시 하는 데 사용 됩니다. 기본적으로 `ComboBox` 컨트롤은 두 부분으로 표시 됩니다. 맨 위 부분은 사용자가 목록 항목을 입력할 수 있는 텍스트 상자입니다. 두 번째 부분은 사용자가 하나를 선택할 수 있는 항목 목록을 표시 하는 목록 상자입니다.
