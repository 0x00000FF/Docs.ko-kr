# <a name="picturebox-control-overview-windows-forms"></a>PictureBox 컨트롤 개요(Windows Forms)
Windows Forms <xref:System.Windows.Forms.PictureBox> 컨트롤은 비트맵, GIF, JPEG, 메타 파일 또는 아이콘 형식으로 그래픽을 표시 하는 데 사용 됩니다.  
  
## <a name="key-properties-and-methods"></a>키 속성 및 메서드  
 표시 되는 그림은 런타임에 설정 하거나 디자인 타임에 설정할 수 있는 <xref:System.Windows.Forms.PictureBox.Image%2A> 속성에 의해 결정 됩니다. 또는 <xref:System.Windows.Forms.PictureBox.ImageLocation%2A> 속성을 설정 하 여 이미지를 지정한 다음 <xref:System.Windows.Forms.PictureBox.Load%2A> 메서드를 사용 하 여 동기적으로 이미지를 로드 하거나 <xref:System.Windows.Forms.PictureBox.LoadAsync%2A> 메서드를 사용 하 여 비동기적으로 이미지를 로드할 수 있습니다. <xref:System.Windows.Forms.PictureBox.SizeMode%2A> 속성은 이미지와 컨트롤이 서로 어떻게 일치 하는 정도를 제어 합니다. 자세한 내용은 [방법: 런타임에 그림의 크기 또는 위치 수정](how-to-modify-the-size-or-placement-of-a-picture-at-run-time-windows-forms.md)을 참조 하세요.
