# PictureBox 컨트롤 개요(Windows Forms)
Windows Forms <xref:System.Windows.Forms.PictureBox> 컨트롤은 비트맵, GIF, JPEG, 메타파일 또는 아이콘 형식의 그래픽을 표시하는 데 사용됩니다.  
  
## 주요 속성 및 메서드  
 표시되는 그림은 <xref:System.Windows.Forms.PictureBox.Image%2A> 속성으로 지정합니다. 이 속성은 런타임에 또는 디자인 타임에 설정할 수 있습니다.  또는 <xref:System.Windows.Forms.PictureBox.ImageLocation%2A> 속성을 설정한 다음 <xref:System.Windows.Forms.PictureBox.Load%2A> 메서드를 사용하여 동기적으로 이미지를 로드하거나 <xref:System.Windows.Forms.PictureBox.LoadAsync%2A> 메서드를 사용하여 비동기적으로 이미지를 로드하여 이미지를 지정할 수 있습니다.  <xref:System.Windows.Forms.PictureBox.SizeMode%2A> 속성은 이미지와 컨트롤의 크기가 서로 일치하도록 조정합니다.  자세한 내용은 [방법: 런타임에 그림의 크기 또는 위치 수정](../../../../docs/framework/winforms/controls/how-to-modify-the-size-or-placement-of-a-picture-at-run-time-windows-forms.md)를 참조하십시오.
