### <a name="to-determine-checked-items-in-a-checkedlistbox-control"></a><span data-ttu-id="16a78-106">CheckedListBox 컨트롤에서 선택한 항목을 확인 하려면</span><span class="sxs-lookup"><span data-stu-id="16a78-106">To determine checked items in a CheckedListBox control</span></span>  
  
1.  <span data-ttu-id="16a78-107">반복은 <xref:System.Windows.Forms.CheckedListBox.CheckedItems%2A> 컬렉션, 컬렉션은 0부터 시작 하므로 0부터 시작 합니다.</span><span class="sxs-lookup"><span data-stu-id="16a78-107">Iterate through the <xref:System.Windows.Forms.CheckedListBox.CheckedItems%2A> collection, starting at 0 since the collection is zero-based.</span></span> <span data-ttu-id="16a78-108">이 메서드는 전체 목록이 아니라 선택 된 항목의 목록에서 항목 번호를 알려 참고 합니다.</span><span class="sxs-lookup"><span data-stu-id="16a78-108">Note that this method will give you the item number in the list of checked items, not the overall list.</span></span> <span data-ttu-id="16a78-109">아래 코드 같은 텍스트에 표시 되는 목록에서 첫 번째 항목 체크 하지 하는 경우 두 번째 항목이 선택 되는 "선택한 항목 1 = MyListItem2"입니다.</span><span class="sxs-lookup"><span data-stu-id="16a78-109">So if the first item in the list is not checked and the second item is checked, the code below will display text like "Checked Item 1 = MyListItem2".</span></span>  
  
    ```vb  
    ' Determine if there are any items checked.  
    If CheckedListBox1.CheckedItems.Count <> 0 Then  
       ' If so, loop through all checked items and print results.  
       Dim x As Integer  
       Dim s As String = ""  
       For x = 0 To CheckedListBox1.CheckedItems.Count - 1  
          s = s & "Checked Item " & (x + 1).ToString & " = " & CheckedListBox1.CheckedItems(x).ToString & ControlChars.CrLf  
       Next x  
       MessageBox.Show(s)  
    End If  
    ```
