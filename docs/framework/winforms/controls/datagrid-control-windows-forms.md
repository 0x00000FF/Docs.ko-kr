# DataGrid 컨트롤(Windows Forms)
> [!NOTE]
>  <xref:System.Windows.Forms.DataGridView> 컨트롤은 `DataGrid` 컨트롤을 대체하고 여기에 다른 기능을 추가하여 새로 도입된 컨트롤이지만 이전 버전과의 호환성 및 이후 사용 가능성을 고려하여 `DataGrid` 컨트롤을 계속 유지하도록 선택할 수 있습니다.  자세한 내용은 [Windows Forms DataGridView 컨트롤과 DataGrid 컨트롤의 차이점](../../../../docs/framework/winforms/controls/differences-between-the-windows-forms-datagridview-and-datagrid-controls.md)을 참조하십시오.  
  
 Windows Forms `DataGrid` 컨트롤은 [!INCLUDE[vstecado](../../../../includes/vstecado-md.md)] 데이터 집합에 대한 사용자 인터페이스를 제공하며 표 형식 데이터를 표시하고 데이터 소스에 대한 업데이트를 허용합니다.  
  
 `DataGrid` 컨트롤을 올바른 데이터 소스로 설정하면 컨트롤에 자동으로 데이터가 채워지고 데이터의 모양에 기반하여 열과 행이 만들어집니다.  `DataGrid` 컨트롤은 단일 테이블이나 테이블 집합 간의 계층 관계를 표시하는 데 사용할 수 있습니다.
