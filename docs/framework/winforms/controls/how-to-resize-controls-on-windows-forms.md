# <a name="how-to-resize-controls-on-windows-forms"></a>방법: Windows Forms에서 컨트롤의 크기 조정
개별 컨트롤의 크기를 조정할 수 있고와 같은 동일한 또는 다른 종류의 여러 컨트롤을 크기 조정할 수 <xref:System.Windows.Forms.Button> 및 <xref:System.Windows.Forms.GroupBox> 컨트롤입니다.  
  
### <a name="to-resize-a-control"></a>컨트롤의 크기를 조정 하려면  
  
1.  크기를 조정할 수 및 8 개의 크기 조정 핸들 중 하나를 끕니다. 컨트롤을 클릭 합니다.  
  
    > [!NOTE]
    >  컨트롤을 선택 하 고 한 번에 한 픽셀씩 컨트롤 크기를 조정 하려면 SHIFT 키를 누른 채 화살표 키를 누릅니다. 큰에서 컨트롤의 크기를 조정 하려면 SHIFT 키나 CTRL 키를 누른 채 아래쪽 또는 오른쪽 화살표 키를 누릅니다.  
  
### <a name="to-resize-multiple-controls-on-a-form"></a>폼에 여러 컨트롤의 크기를 조정 하려면  
  
1.  CTRL 또는 SHIFT 키를 누른 크기를 조정 하려면 컨트롤을 선택 합니다. 선택한 첫 번째 컨트롤의 크기는 다른 컨트롤에 사용 됩니다.  
  
2.  에 **형식** 메뉴 선택 **같은 크기로**, 네 가지 옵션 중 하나를 선택 합니다. 처음 세 명령은 선택한 첫 번째 컨트롤에 맞게 컨트롤의 크기를 변경 합니다.
