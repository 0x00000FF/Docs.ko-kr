### <a name="to-resize-multiple-controls-on-a-form"></a><span data-ttu-id="dbee7-108">폼에서 여러 컨트롤의 크기를 조정 하려면</span><span class="sxs-lookup"><span data-stu-id="dbee7-108">To resize multiple controls on a form</span></span>
