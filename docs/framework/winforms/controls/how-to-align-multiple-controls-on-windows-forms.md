# 방법: Windows Forms에서 여러 컨트롤 맞춤
하나의 명령을 사용하여 컨트롤 그룹 위치를 지정하면 Windows 기반 응용 프로그램의 UI\(사용자 인터페이스\) 레이아웃을 표준화할 수 있습니다.  
  
> [!NOTE]
>  표시되는 대화 상자와 메뉴 명령은 활성 설정이나 버전에 따라 도움말에서 설명하는 것과 다를 수 있습니다.  설정을 변경하려면 **도구** 메뉴에서 **설정 가져오기 및 내보내기**를 선택합니다.  자세한 내용은 [Customizing Development Settings in Visual Studio](http://msdn.microsoft.com/ko-kr/22c4debb-4e31-47a8-8f19-16f328d7dcd3)을 참조하십시오.  
  
### 폼에서 여러 컨트롤의 위치를 맞추려면  
  
1.  **Windows Forms 디자이너**에서 위치를 지정할 컨트롤이 포함된 폼을 엽니다.  
  
2.  위치를 맞출 컨트롤을 선택합니다. 첫 번째 선택하는 컨트롤이 다른 컨트롤과 위치를 맞추는 데 사용할 주 컨트롤이 됩니다.  
  
3.  **서식**메뉴에서 **맞춤**을 선택한 다음 사용 가능한 7개의 명령 중에서 하나를 클릭합니다.
