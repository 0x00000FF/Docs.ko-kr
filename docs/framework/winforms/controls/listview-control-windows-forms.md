# ListView 컨트롤(Windows Forms)
Windows Forms `ListView` 컨트롤은 아이콘이 포함된 항목 목록을 표시합니다.  목록 뷰를 사용하여 Windows 탐색기의 오른쪽 창과 같은 사용자 인터페이스를 만들 수 있습니다.
