# 방법: Windows Forms에서 맞춤선과 모눈을 사용하여 컨트롤 정렬
[!INCLUDE[vsprvs](../../../../includes/vsprvs-md.md)]의 레이아웃 기능을 사용하여 폼에 컨트롤이 배치될 위치를 정확하게 지정할 수 있습니다.  폼에서 컨트롤을 추가하거나 이동할 때 컨트롤이 자동으로 Windows Forms 디자이너에 있는 모눈의 행과 열에 맞춰지거나 맞춤선 기능을 사용하여 컨트롤을 맞출 수 있습니다.  
  
> [!NOTE]
>  표시되는 대화 상자와 메뉴 명령은 활성 설정이나 버전에 따라 도움말에서 설명하는 것과 다를 수 있습니다.  설정을 변경하려면 **도구** 메뉴에서 **설정 가져오기 및 내보내기**를 선택합니다.  자세한 내용은 [Customizing Development Settings in Visual Studio](http://msdn.microsoft.com/ko-kr/22c4debb-4e31-47a8-8f19-16f328d7dcd3)을 참조하십시오.  
  
### 모든 컨트롤을 모눈에 맞추려면  
  
-   Windows Forms 디자이너의 **옵션** 대화 상자에서 **SnapToGrid** 레이아웃 모드를 선택합니다.  
  
     자세한 내용은 [General, Windows Forms Designer, Options Dialog Box](http://msdn.microsoft.com/ko-kr/8dd170af-72f0-4212-b04b-034ceee92834)을 참조하십시오.  모든 컨트롤이 모눈의 점에 따라 맞춰집니다.  
  
     개별 컨트롤을 해당 위치에 잠가서 모눈에 맞출 수도 있습니다.  그러나 컨트롤을 잠그면 이동하거나 크기를 조정할 수 없습니다.  컨트롤 잠금에 대한 자세한 내용은 [방법: Windows Forms에 컨트롤 고정](../../../../docs/framework/winforms/controls/how-to-lock-controls-to-windows-forms.md)을 참조하십시오.  
  
### 맞춤선을 사용하여 컨트롤을 맞추려면  
  
-   Windows Forms 디자이너의 **옵션** 대화 상자에서 **SnapLines** 레이아웃 모드를 선택합니다.  
  
     자세한 내용은 [연습: 맞춤선을 사용하여 Windows Forms에서 컨트롤 정렬](../../../../docs/framework/winforms/controls/walkthrough-arranging-controls-on-windows-forms-using-snaplines.md)을 참조하십시오.  이제 맞춤선을 사용하여 폼에서 컨트롤을 맞추고 정렬할 수 있습니다.
