# 방법: 디자이너를 사용하여 표준 항목을 포함하는 기본 Windows Forms ToolStrip 만들기
다음 절차에서는 <xref:System.Windows.Forms.ToolStrip>을 만들고 일반적인 작업을 나타내는 7개의 <xref:System.Windows.Forms.ToolStripButton> 컨트롤을 추가하는 방법을 보여 줍니다.  
  
> [!NOTE]
>  표시되는 대화 상자와 메뉴 명령은 활성 설정이나 버전에 따라 도움말에서 설명하는 것과 다를 수 있습니다.  설정을 변경하려면 **도구** 메뉴에서 **가져오기 및 내보내기 설정**을 선택합니다.  자세한 내용은 [Customizing Development Settings in Visual Studio](http://msdn.microsoft.com/ko-kr/22c4debb-4e31-47a8-8f19-16f328d7dcd3)을 참조하십시오.  
  
### 디자이너에서 표준 항목을 추가하려면  
  
1.  <xref:System.Windows.Forms.ToolStrip> 컨트롤을 만듭니다.  
  
2.  <xref:System.Windows.Forms.ToolStrip>의 오른쪽 위 모퉁이에서 스마트 작업 화살표를 클릭하여 **ToolStrip 작업** 창을 표시합니다.  
  
3.  **ToolStrip 작업** 창에서 **표준 항목 삽입**을 선택합니다.
