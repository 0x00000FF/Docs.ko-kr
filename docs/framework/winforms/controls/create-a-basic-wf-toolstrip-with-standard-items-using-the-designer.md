# <a name="how-to-create-a-basic-windows-forms-toolstrip-with-standard-items-using-the-designer"></a>방법: 디자이너를 사용하여 표준 항목을 포함하는 기본 Windows Forms ToolStrip 만들기
다음 절차에서는 <xref:System.Windows.Forms.ToolStrip>을 만들고 일반적인 작업을 나타내는 일곱 개의 <xref:System.Windows.Forms.ToolStripButton> 컨트롤을 추가 하는 방법을 보여 줍니다.

## <a name="to-add-standard-items-in-the-designer"></a>디자이너에서 표준 항목을 추가 하려면

1. <xref:System.Windows.Forms.ToolStrip> 컨트롤을 만듭니다.

2. <xref:System.Windows.Forms.ToolStrip>의 오른쪽 위 모서리에서 스마트 작업 화살표를 클릭 하 여 **ToolStrip 작업** 창을 표시 합니다.

3. **ToolStrip 작업** 창에서 **표준 항목 삽입**을 선택 합니다.
