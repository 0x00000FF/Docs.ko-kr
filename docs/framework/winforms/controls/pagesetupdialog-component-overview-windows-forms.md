# PageSetupDialog 구성 요소 개요(Windows Forms)
Windows Forms <xref:System.Windows.Forms.PageSetupDialog> 구성 요소는 Windows 기반 응용 프로그램에서 인쇄 작업을 위해 페이지 세부 정보를 설정하는 데 사용하는 미리 구성된 대화 상자입니다.  Windows 기반 응용 프로그램에서 직접 대화 상자를 구성하는 대신 이 구성 요소를 간단한 솔루션으로 사용하여 사용자가 페이지를 기본 설정할 수 있도록 합니다.  테두리 및 여백, 머리글 및 바닥글, 세로 또는 가로 방향 등을 설정할 수 있습니다.  표준 Windows 대화 상자를 사용하여 사용자가 기본 기능을 쉽게 이해할 수 있는 응용 프로그램을 만들 수 있습니다.  
  
## 주요 속성 및 메서드  
 <xref:System.Windows.Forms.CommonDialog.ShowDialog%2A> 메서드를 사용하여 런타임에 대화 상자를 표시합니다.  이 구성 요소에는 단일 페이지\(<xref:System.Drawing.Printing.PrintDocument> 클래스\) 또는 문서\(<xref:System.Drawing.Printing.PageSettings> 클래스\)와 관련하여 설정할 수 있는 속성이 포함되어 있습니다.  <xref:System.Windows.Forms.PageSetupDialog> 구성 요소를 사용하여 <xref:System.Drawing.Printing.PrinterSettings> 클래스에 저장되어 있는 특정 프린터 설정을 확인할 수도 있습니다.  
  
 <xref:System.Windows.Forms.PageSetupDialog> 구성 요소가 폼에 추가되면 Windows Forms 디자이너의 아래쪽에 있는 트레이에 나타납니다.
