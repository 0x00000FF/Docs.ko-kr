# NotifyIcon 구성 요소(Windows Forms)
Windows Forms `NotifyIcon` 구성 요소는 백그라운드에서 실행되는 프로세스에 대한 아이콘을 작업 표시줄의 상태 알림 영역에 표시하며 이렇게 하지 않으면 사용자 인터페이스가 나타나지 않습니다.  작업 표시줄의 상태 알림 영역에 있는 아이콘을 클릭하여 액세스할 수 있는 바이러스 예방 프로그램이 이러한 예가 될 수 있습니다.
