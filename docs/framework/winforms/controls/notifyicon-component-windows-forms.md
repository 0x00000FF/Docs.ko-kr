# <a name="notifyicon-component-windows-forms"></a>NotifyIcon 구성 요소(Windows Forms)
Windows Forms `NotifyIcon` 구성 요소에는 백그라운드에서 실행 되는 프로세스에 대 한 작업 표시줄의 상태 알림 영역에 아이콘이 표시 되 고 그렇지 않으면 사용자 인터페이스가 표시 되지 않습니다. 예를 들어 작업 표시줄의 상태 알림 영역에서 아이콘을 클릭하여 액세스할 수 있는 바이러스 방지 프로그램이 있습니다.
