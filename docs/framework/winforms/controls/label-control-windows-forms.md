# <a name="label-control-windows-forms"></a>Label 컨트롤(Windows Forms)
> [!IMPORTANT]
> <xref:System.Windows.Forms.ToolStripLabel> 컨트롤은 기능을 대체 하 고 <xref:System.Windows.Forms.Label> 컨트롤에 기능을 추가 합니다. <xref:System.Windows.Forms.ToolStripDropDown>와 같은 다른 새 컨트롤에 <xref:System.Windows.Forms.ToolStripLabel>를 사용할 수 있습니다. 그러나 <xref:System.Windows.Forms.Label> 컨트롤은 이전 버전과의 호환성 및 향후 사용을 위해 유지 됩니다 (선택 하는 경우).  
  
 Windows Forms <xref:System.Windows.Forms.Label> 컨트롤은 사용자가 편집할 수 없는 텍스트 또는 이미지를 표시 하는 데 사용 됩니다. 이러한 항목은 폼의 개체를 식별 하는 데 사용 됩니다. 예를 들어 클릭 하면 특정 컨트롤이 수행할 작업에 대 한 설명을 제공 하거나 응용 프로그램의 런타임 이벤트 또는 프로세스에 대 한 응답으로 정보를 표시 하는 데 사용 됩니다. <xref:System.Windows.Forms.Label> 컨트롤은 포커스를 받을 수 없기 때문에 다른 컨트롤에 대 한 액세스 키를 만드는 데 사용할 수도 있습니다.
