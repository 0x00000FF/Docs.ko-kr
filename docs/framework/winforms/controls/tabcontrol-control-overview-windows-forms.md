# <a name="tabcontrol-control-overview-windows-forms"></a>TabControl 컨트롤 개요(Windows Forms)
Windows Forms <xref:System.Windows.Forms.TabControl>은 파일 캐비닛의 폴더 집합에 노트북 또는 레이블의 구분선과 같은 여러 탭을 표시합니다. 탭에는 사진 및 기타 컨트롤이 포함될 수 있습니다. 탭 컨트롤을 사용 하 여 Windows 운영 체제에 표시 되는 여러 페이지 (예: 제어판 표시 속성)의 여러 대화 상자를 만들 수 있습니다. 또한 <xref:System.Windows.Forms.TabControl>를 사용 하 여 관련 속성 그룹을 설정 하는 데 사용 되는 속성 페이지를 만들 수 있습니다.  
  
## <a name="working-with-tabcontrol"></a>TabControl 사용  
 <xref:System.Windows.Forms.TabControl>의 가장 중요 한 속성은 개별 탭이 포함 된 <xref:System.Windows.Forms.TabControl.TabPages%2A>입니다. 각 개별 탭은 <xref:System.Windows.Forms.TabPage> 개체입니다. 탭을 클릭 하면 해당 <xref:System.Windows.Forms.TabPage> 개체에 대 한 <xref:System.Windows.Forms.Control.Click> 이벤트가 발생 합니다.
