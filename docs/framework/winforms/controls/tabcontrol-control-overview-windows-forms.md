# TabControl 컨트롤 개요(Windows Forms)
Windows Forms <xref:System.Windows.Forms.TabControl>은 수첩의 구분표나 파일 캐비닛의 폴더 모음에 있는 레이블처럼 여러 개의 탭을 표시합니다.  탭은 그림이나 다른 컨트롤을 포함할 수 있습니다.  탭 컨트롤을 사용하면 Windows 운영 체제에서 자주 볼 수 있는 여러 페이지로 구성된 대화 상자\(예: 제어판 표시 속성\)를 만들 수 있습니다.  또한 <xref:System.Windows.Forms.TabControl>을 사용하면 관련 속성 그룹을 설정하는 데 사용되는 속성 페이지를 만들 수 있습니다.  
  
## TabControl 작업  
 <xref:System.Windows.Forms.TabControl>의 가장 중요한 속성은 개별 탭을 포함하는 <xref:System.Windows.Forms.TabControl.TabPages%2A>입니다.  각 개별 탭은 <xref:System.Windows.Forms.TabPage> 개체입니다.  탭을 클릭하면 해당 <xref:System.Windows.Forms.TabPage> 개체에 대한 <xref:System.Windows.Forms.Control.Click> 이벤트가 발생합니다.
