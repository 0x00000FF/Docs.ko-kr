# FontDialog 구성 요소(Windows Forms)
Windows Forms <xref:System.Windows.Forms.FontDialog> 구성 요소는 미리 구성된 대화 상자입니다.  이것은 Windows 운영 체제에서 볼 수 있는 **글꼴** 대화 상자와 같으며  <xref:System.Windows.Forms.CommonDialog> 클래스에서 구성 요소가 상속됩니다.  
  
## 단원 내용  
 [FontDialog 구성 요소 개요](../../../../docs/framework/winforms/controls/fontdialog-component-overview-windows-forms.md)  
 미리 구성된 대화 상자를 표시하는 데 사용되는 <xref:System.Windows.Forms.FontDialog> 구성 요소의 일반적인 개념을 소개합니다.  사용자는 이 대화 상자를 통해 글꼴과 해당 설정을 조작할 수 있습니다.  
  
 [방법: FontDialog 구성 요소를 사용하여 글꼴 목록 표시](../../../../docs/framework/winforms/controls/how-to-show-a-font-list-with-the-fontdialog-component.md)  
 런타임에 <xref:System.Windows.Forms.FontDialog> 구성 요소의 인스턴스를 통해 글꼴을 선택하는 방법을 설명합니다.
