# <a name="fontdialog-component-windows-forms"></a>FontDialog 구성 요소(Windows Forms)
Windows Forms <xref:System.Windows.Forms.FontDialog> 구성 요소는 미리 구성된 대화 상자입니다. Windows 운영 체제에서 제공 하는 것과 동일한 **글꼴** 대화 상자입니다. 구성 요소는 <xref:System.Windows.Forms.CommonDialog> 클래스에서 상속 합니다.  
  
## <a name="in-this-section"></a>섹션 내용  
 [FontDialog 구성 요소 개요](fontdialog-component-overview-windows-forms.md)  
 미리 구성 된 대화 상자를 표시 하는 데 사용 하는 <xref:System.Windows.Forms.FontDialog> 구성 요소의 일반적인 개념을 소개 합니다. 사용자는 대화 상자를 사용 하 여 글꼴 및 해당 설정을 조작할 수 있습니다.  
  
 [방법: FontDialog 구성 요소를 사용하여 글꼴 목록 표시](how-to-show-a-font-list-with-the-fontdialog-component.md)  
 <xref:System.Windows.Forms.FontDialog> 구성 요소의 인스턴스를 통해 런타임에 글꼴을 선택 하는 방법에 대해 설명 합니다.
