# 옵션 목록 표시에 사용하는 Windows Forms 컨트롤
선택 가능한 옵션 목록을 사용자에게 제공하려면 다양한 컨트롤을 Windows Form에 추가하면 됩니다.  원하는 사용자 입력 제한 수준에 따라 <xref:System.Windows.Forms.ListBox> 컨트롤, <xref:System.Windows.Forms.ComboBox> 컨트롤 또는 <xref:System.Windows.Forms.CheckedListBox> 컨트롤을 추가할 수 있습니다.  다음 링크를 사용하여 가장 적합한 컨트롤을 결정합니다.
