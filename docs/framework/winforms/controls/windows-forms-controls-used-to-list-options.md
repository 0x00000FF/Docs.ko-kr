# <a name="windows-forms-controls-used-to-list-options"></a>옵션 목록 표시에 사용하는 Windows Forms 컨트롤
선택할 수 있는 옵션 목록을 사용자에 게 제공 하려는 경우에는 다양 한 컨트롤을 Windows Form에 추가할 수 있습니다. 사용자의 입력을 제한 하려는 양에 따라 <xref:System.Windows.Forms.ListBox> 컨트롤, <xref:System.Windows.Forms.ComboBox> 컨트롤 또는 <xref:System.Windows.Forms.CheckedListBox> 컨트롤을 추가할 수 있습니다. 다음 링크를 사용 하 여 요구 사항에 가장 적합 한 컨트롤을 결정 합니다.

 컨트롤의 항목 목록에서 항목을 추가 하거나 제거 하는 방법에 대 한 지침을 제공 합니다.
