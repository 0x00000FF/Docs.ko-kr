# <a name="how-to-position-a-toolstripitem-on-a-toolstrip"></a>방법: ToolStrip에 ToolStripItem 배치
이동 하거나 추가할 수는 <xref:System.Windows.Forms.ToolStripItem> 의 왼쪽 또는 오른쪽에는 <xref:System.Windows.Forms.ToolStrip>합니다.  
  
### <a name="to-move-or-add-a-toolstripitem-to-the-left-side-of-a-toolstrip"></a>이동 하거나 ToolStripItem ToolStrip의 왼쪽에 추가 하려면  
  
1.  설정의 <xref:System.Windows.Forms.ToolStripItemAlignment> 의 속성은 <xref:System.Windows.Forms.ToolStripItem> 를 <xref:System.Windows.Forms.ToolStripItemAlignment.Left>합니다.  
  
### <a name="to-move-or-add-a-toolstripitem-to-the-right-side-of-a-toolstrip"></a>이동 하거나 ToolStripItem ToolStrip의 오른쪽에 추가 하려면  
  
1.  설정의 <xref:System.Windows.Forms.ToolStripItemAlignment> 의 속성은 <xref:System.Windows.Forms.ToolStripItem> 를 <xref:System.Windows.Forms.ToolStripItemAlignment.Right>합니다.
