# 방법: ToolStrip에 ToolStripItem 배치
<xref:System.Windows.Forms.ToolStripItem>을 <xref:System.Windows.Forms.ToolStrip>의 왼쪽 또는 오른쪽으로 이동하거나 추가할 수 있습니다.  
  
### ToolStripItem을 ToolStrip의 왼쪽으로 이동하거나 추가하려면  
  
1.  <xref:System.Windows.Forms.ToolStripItem>의 <xref:System.Windows.Forms.ToolStripItemAlignment> 속성을 <xref:System.Windows.Forms.ToolStripItemAlignment>로 설정합니다.  
  
### ToolStripItem을 ToolStrip의 오른쪽으로 이동하거나 추가하려면  
  
1.  <xref:System.Windows.Forms.ToolStripItem>의 <xref:System.Windows.Forms.ToolStripItemAlignment> 속성을 <xref:System.Windows.Forms.ToolStripItemAlignment>로 설정합니다.
