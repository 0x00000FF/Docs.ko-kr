# <a name="monthcalendar-control-windows-forms"></a>MonthCalendar 컨트롤(Windows Forms)
Windows Forms `MonthCalendar` 컨트롤은 사용자가 날짜 정보를 보고 설정 하기 위한 직관적인 그래픽 인터페이스를 제공 합니다. 컨트롤은 요일 아래의 열에 정렬 된 해당 월의 번호가 매겨진 날짜를 포함 하는 표를 표시 합니다. 월 캡션의 한쪽에 있는 화살표 단추를 클릭 하 여 다른 달을 선택할 수 있습니다. 비슷한 <xref:System.Windows.Forms.DateTimePicker> 컨트롤과 달리이 컨트롤을 사용 하 여 날짜 범위를 선택할 수 있습니다. 그러나 <xref:System.Windows.Forms.DateTimePicker> 컨트롤을 사용 하면 날짜 및 날짜를 설정할 수 있습니다.
