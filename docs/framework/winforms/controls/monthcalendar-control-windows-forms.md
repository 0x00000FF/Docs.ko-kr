# MonthCalendar 컨트롤(Windows Forms)
Windows Forms `MonthCalendar` 컨트롤은 날짜 정보를 보고 설정할 수 있는 직관적인 그래픽 인터페이스를 제공합니다.  이 컨트롤에서는 현재 달의 날짜가 요일별로 나열된 표 형태의 모눈을 표시합니다.  월 캡션의 양쪽에 있는 화살표 단추를 클릭하여 다른 월을 선택할 수 있습니다.  이 컨트롤에서는 <xref:System.Windows.Forms.DateTimePicker> 컨트롤과 다르게 날짜 범위만을 선택할 수 있습니다. 그러나 <xref:System.Windows.Forms.DateTimePicker> 컨트롤은 날짜뿐 아니라 시간도 설정할 수 있습니다.
