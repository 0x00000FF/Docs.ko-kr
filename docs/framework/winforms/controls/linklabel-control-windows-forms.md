# <a name="linklabel-control-windows-forms"></a>LinkLabel 컨트롤(Windows Forms)
Windows 양식 `LinkLabel` 컨트롤을 사용하면 Windows Forms 응용 프로그램에 웹 스타일 링크를 추가할 수 있습니다. 컨트롤을 `LinkLabel` 사용할 수 있는 모든 것에 <xref:System.Windows.Forms.Label> 대 한 컨트롤을 사용할 수 있습니다. 텍스트의 일부를 개체 또는 웹 페이지에 대한 링크로 설정할 수도 있습니다.
