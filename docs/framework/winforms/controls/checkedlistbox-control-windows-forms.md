# CheckedListBox 컨트롤(Windows Forms)
Windows Forms `CheckedListBox` 컨트롤은 <xref:System.Windows.Forms.ListBox> 컨트롤과 같이 항목의 목록을 표시할 뿐만 아니라 목록의 항목 옆에 확인 표시를 나타낼 수도 있습니다.  
  
## 단원 내용  
 [CheckedListBox 컨트롤 개요](../../../../docs/framework/winforms/controls/checkedlistbox-control-overview-windows-forms.md)  
 이 컨트롤의 정의, 주요 기능 및 속성을 설명합니다.  
  
 [방법: Windows Forms CheckedListBox 컨트롤에서 선택된 항목 확인](../../../../docs/framework/winforms/controls/how-to-determine-checked-items-in-the-windows-forms-checkedlistbox-control.md)  
 목록을 단계적으로 검사하여 확인 표시된 항목을 확인하는 방법을 설명합니다.  
  
## 참조  
 <xref:System.Windows.Forms.CheckedListBox> 클래스  
 이 클래스를 설명하며 이 클래스의 모든 멤버에 대한 링크가 포함되어 있습니다.
