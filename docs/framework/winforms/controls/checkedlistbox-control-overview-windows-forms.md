# CheckedListBox 컨트롤 개요(Windows Forms)
Windows Forms <xref:System.Windows.Forms.CheckedListBox> 컨트롤은 <xref:System.Windows.Forms.ListBox> 컨트롤을 확장합니다.  이 컨트롤은 목록 상자의 거의 모든 기능을 가지고 있을 뿐만 아니라 목록의 항목 옆에 확인 표시를 할 수도 있습니다.  두 컨트롤의 또 다른 차이점은 확인 목록 상자만 <xref:System.Windows.Forms.DrawMode?displayProperty=fullName>을 지원한다는 것과 확인 목록 상자에서는 한 항목만 선택하거나 아무 항목도 선택하지 않을 수 있다는 것입니다.  선택된 항목은 폼에서 강조 표시되며 이 항목은 확인 표시된 항목과는 다릅니다.  
  
 확인 목록 상자에서 **문자열 컬렉션 편집기**를 사용하여 디자인 타임에 항목을 추가하거나 <xref:System.Windows.Forms.CheckedListBox.Items%2A> 속성을 사용하여 런타임에 컬렉션의 항목을 동적으로 추가할 수 있습니다.  자세한 내용은 [방법: Windows Forms ComboBox, ListBox 또는 CheckedListBox 컨트롤에서 항목 추가 및 제거](../../../../docs/framework/winforms/controls/add-and-remove-items-from-a-wf-combobox.md)를 참조하십시오.
