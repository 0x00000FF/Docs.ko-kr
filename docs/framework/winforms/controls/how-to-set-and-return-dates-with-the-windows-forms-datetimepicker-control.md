### <a name="to-set-the-date-and-time-value-of-the-control"></a><span data-ttu-id="1c937-114">컨트롤의 날짜 및 시간 값을 설정하려면</span><span class="sxs-lookup"><span data-stu-id="1c937-114">To set the date and time value of the control</span></span>  
  
-   <span data-ttu-id="1c937-115"><xref:System.Windows.Forms.DateTimePicker.Value%2A> 속성을 날짜 또는 시간 값으로 설정합니다.</span><span class="sxs-lookup"><span data-stu-id="1c937-115">Set the <xref:System.Windows.Forms.DateTimePicker.Value%2A> property to a date or time value.</span></span>

  
    ```csharp  
    dateTimePicker1.Value = new DateTime(2001, 10, 20);  
    ```  
  
    ```cpp  
    dateTimePicker1->Value = DateTime(2001, 10, 20);  
    ```
