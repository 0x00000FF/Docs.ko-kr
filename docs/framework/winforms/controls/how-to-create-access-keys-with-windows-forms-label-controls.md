1.  <span data-ttu-id="aee9c-108">먼저, 레이블을 그린 하 한 다음 다른 컨트롤을 그립니다.</span><span class="sxs-lookup"><span data-stu-id="aee9c-108">Draw the label first, and then draw the other control.</span></span>  
  
     <span data-ttu-id="aee9c-109">또는</span><span class="sxs-lookup"><span data-stu-id="aee9c-109">-or-</span></span>
