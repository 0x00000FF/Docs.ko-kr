# <a name="contextmenu-component-windows-forms"></a>ContextMenu 구성 요소(Windows Forms)
> [!NOTE]
>  있지만 `MenuStrip` 하 고 `ContextMenuStrip` 바꾸고 기능을 추가 합니다 <xref:System.Windows.Forms.MainMenu> 및 <xref:System.Windows.Forms.ContextMenu> 이전 버전의 컨트롤 <xref:System.Windows.Forms.MainMenu> 및 <xref:System.Windows.Forms.ContextMenu> 선택 하면 이전 버전과 호환성 및 향후 사용을 위해 유지 됩니다.  
  
 Windows Forms `ContextMenu` 선택한 개체와 연관 된 자주 사용 되는 명령의 쉽게 액세스할 수 있는 바로 가기 메뉴를 사용 하 여 사용자에 게 제공 구성 요소를 사용 합니다. 바로 가기 메뉴의 항목은 주로 응용 프로그램에서 다른 곳에 나타나는 기본 메뉴에서 항목의 하위 항목입니다. 마우스를 마우스 오른쪽 단추로 클릭 하 여 바로 가기 메뉴는 일반적으로 사용할 수 있습니다. Windows Forms에서 다른 컨트롤과 연결 않습니다.
