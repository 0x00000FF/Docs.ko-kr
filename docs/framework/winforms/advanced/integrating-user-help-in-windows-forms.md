# <a name="integrating-user-help-in-windows-forms"></a>Windows Forms에 사용자 도움말 통합
Windows 기반 응용 프로그램을 구축 하는 데 있어 중요 한 점은 간과 될 때 사용자가 도움을 받을 수 있기 때문입니다. Windows Forms는 [HelpProvider 구성 요소](../controls/helpprovider-component-windows-forms.md)에서 제공 하는 두 가지 유형의 도움말을 지원 합니다. 첫 번째 방법은 사용자가 HTML 또는 HTML 도움말 1의 도움말 파일을 가리키는 것입니다. *x* 또는 큰 형식입니다. 두 번째는 개별 컨트롤에 대 한 간략 한 "What 's" 형식 도움말을 표시할 수 있습니다. 이 기능은 대화 상자에서 특히 유용 합니다. 동일한 폼에서 두 가지 유형의 도움말을 사용할 수 있습니다.  
  
 또한 [도구 설명 구성 요소](../controls/tooltip-component-windows-forms.md) 를 사용 하 여 Windows Forms 컨트롤에 대 한 개별 도움말을 제공할 수 있습니다.
