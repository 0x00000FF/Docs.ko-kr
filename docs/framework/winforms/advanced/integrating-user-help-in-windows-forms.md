# Windows Forms에 사용자 도움말 통합
도움말 시스템은 작업 중 의문이 생길 때 지원을 받을 수 있는 기능으로, 자주 간과되기는 하지만 Windows 응용 프로그램 작성의 필수적인 부분입니다.  Windows Forms에서는 [HelpProvider 구성 요소](../../../../docs/framework/winforms/controls/helpprovider-component-windows-forms.md)를 통해 각각 제공되는 두 가지 형식의 도움말을 지원합니다.  첫 번째는 HTML 또는 HTML 도움말 1.*x* 이상 형식으로 사용자에게 제공되는 도움말 파일입니다.  두 번째는 각 컨트롤에 대해 간단한 "설명" 형식으로 표시되는 도움말로서, 대화 상자에 특히 유용합니다.  두 가지 형식의 도움말을 모두 같은 폼에서 사용할 수 있습니다.  
  
 또한 [ToolTip 구성 요소](../../../../docs/framework/winforms/controls/tooltip-component-windows-forms.md)는 Windows Forms의 컨트롤에 개별 도움말을 제공할 때 사용할 수 있습니다.
