### <a name="to-copy-and-paste-an-elementhost-control-at-design-time"></a><span data-ttu-id="8a720-107">복사 하 여 디자인 타임에 ElementHost 컨트롤을 붙여 넣습니다.</span><span class="sxs-lookup"><span data-stu-id="8a720-107">To copy and paste an ElementHost control at design time</span></span>  
  
1.  <span data-ttu-id="8a720-108">새 WPF 추가 <xref:System.Windows.Controls.UserControl> Windows Forms 프로젝트.</span><span class="sxs-lookup"><span data-stu-id="8a720-108">Add a new WPF <xref:System.Windows.Controls.UserControl> to your Windows Forms project.</span></span> <span data-ttu-id="8a720-109">컨트롤 형식의 기본 이름인 `UserControl1.xaml`을 사용합니다.</span><span class="sxs-lookup"><span data-stu-id="8a720-109">Use the default name for the control type, `UserControl1.xaml`.</span></span> <span data-ttu-id="8a720-110">자세한 내용은 참조 [연습: 새 WPF 콘텐츠 만들기 디자인 타임에 Windows Forms](../../../../docs/framework/winforms/advanced/walkthrough-creating-new-wpf-content-on-windows-forms-at-design-time.md)합니다.</span><span class="sxs-lookup"><span data-stu-id="8a720-110">For more information, see [Walkthrough: Creating New WPF Content on Windows Forms at Design Time](../../../../docs/framework/winforms/advanced/walkthrough-creating-new-wpf-content-on-windows-forms-at-design-time.md).</span></span>  
  
2.  <span data-ttu-id="8a720-111">에 **속성** 창의 설정의 값은 <xref:System.Windows.FrameworkElement.Width%2A> 및 <xref:System.Windows.FrameworkElement.Height%2A> 의 속성 `UserControl1` 를 `200`합니다.</span><span class="sxs-lookup"><span data-stu-id="8a720-111">In the **Properties** window, set the value of the <xref:System.Windows.FrameworkElement.Width%2A> and <xref:System.Windows.FrameworkElement.Height%2A> properties of `UserControl1` to `200`.</span></span>  
  
3.  <span data-ttu-id="8a720-112"><xref:System.Windows.Controls.Control.Background%2A> 속성 값을 `Blue`로 설정합니다.</span><span class="sxs-lookup"><span data-stu-id="8a720-112">Set the value of the <xref:System.Windows.Controls.Control.Background%2A> property to `Blue`.</span></span>  
  
4.  <span data-ttu-id="8a720-113">프로젝트를 빌드합니다.</span><span class="sxs-lookup"><span data-stu-id="8a720-113">Build the project.</span></span>  
  
5.  <span data-ttu-id="8a720-114">Windows Forms 디자이너에서 `Form1`을 엽니다.</span><span class="sxs-lookup"><span data-stu-id="8a720-114">Open `Form1` in the Windows Forms Designer.</span></span>  
  
6.  <span data-ttu-id="8a720-115">**도구 상자**의 인스턴스를 끌어 `UserControl1` 폼입니다.</span><span class="sxs-lookup"><span data-stu-id="8a720-115">From the **Toolbox**, drag an instance of `UserControl1` onto the form.</span></span>  
  
     <span data-ttu-id="8a720-116">`UserControl1` 인스턴스가 `elementHost1`이라는 새 <xref:System.Windows.Forms.Integration.ElementHost> 컨트롤에서 호스트됩니다.</span><span class="sxs-lookup"><span data-stu-id="8a720-116">An instance of `UserControl1` is hosted in a new <xref:System.Windows.Forms.Integration.ElementHost> control named `elementHost1`.</span></span>  
  
7.  <span data-ttu-id="8a720-117">`elementHost1`을 선택하고 Ctrl+C를 눌러 클립보드에 복사합니다.</span><span class="sxs-lookup"><span data-stu-id="8a720-117">With `elementHost1` selected, press CTRL+C to copy it to the clipboard.</span></span>  
  
8.  <span data-ttu-id="8a720-118">CTRL + V를 붙여 복사한 컨트롤을 폼으로 키를 누릅니다.</span><span class="sxs-lookup"><span data-stu-id="8a720-118">Press CTRL+V to paste the copied control onto the form.</span></span>  
  
     <span data-ttu-id="8a720-119">새 <xref:System.Windows.Forms.Integration.ElementHost> 라는 컨트롤 `elementHost2` 폼에 만들어집니다.</span><span class="sxs-lookup"><span data-stu-id="8a720-119">A new <xref:System.Windows.Forms.Integration.ElementHost> control named `elementHost2` is created on the form.</span></span>
