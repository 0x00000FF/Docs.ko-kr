# 응용 프로그램 설정 및 사용자 설정 사용
.NET Framework 2.0부터는 응용 프로그램 실행 세션 간에 유지되는 값을 만들고 액세스할 수 있습니다.  이러한 값을 *설정*이라고 합니다.  설정은 응용 프로그램에서 사용해야 하는 사용자 기본 설정이나 중요한 정보를 나타낼 수 있습니다.  예를 들어 응용 프로그램의 색 구성표에 대한 사용자 기본 설정이 저장되는 일련의 설정을 만들 수 있습니다.  또는 응용 프로그램에서 사용할 데이터베이스를 지정하는 연결 문자열을 저장할 수 있습니다.  설정을 사용하여 응용 프로그램에 중요한 정보를 코드 외부에 저장하고, 개별 사용자의 기본 설정을 저장하는 프로필을 만들 수 있습니다.  
  
 이 단원의 항목에서는 디자인 타임 및 런타임에 설정을 사용하는 방법에 대해 설명합니다.
