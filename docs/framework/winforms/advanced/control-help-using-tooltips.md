# <a name="control-help-using-tooltips"></a>ToolTip을 사용한 컨트롤 도움말
사용할 수는 <xref:System.Windows.Forms.ToolTip> 간단 하면서도 표시할 구성 요소를 특수 한 Windows Forms에 있는 개별 컨트롤에 대 한 도움말 메시지입니다. <xref:System.Windows.Forms.ToolTip> 양식의 각 컨트롤에 대해 표시 되는 텍스트를 지정 하는 속성을 제공 하는 구성 요소입니다. 작업에 대 한 자세한 내용은 <xref:System.Windows.Forms.ToolTip> 이러한 방식으로 구성 요소 참조 [하는 방법: 디자인 타임에 Windows Form의 컨트롤에 대 한 도구 설명을 설정](../../../../docs/framework/winforms/controls/how-to-set-tooltips-for-controls-on-a-windows-form-at-design-time.md)합니다. 또한 구성할 수는 <xref:System.Windows.Forms.ToolTip> 표시 되기 전에 지연이 인수가 있도록 구성 요소입니다. 자세한 내용은 참조 [하는 방법: Windows Forms ToolTip 구성 요소의 지연 변경](../../../../docs/framework/winforms/controls/how-to-change-the-delay-of-the-windows-forms-tooltip-component.md)합니다.
