# ToolTip을 사용한 컨트롤 도움말
<xref:System.Windows.Forms.ToolTip> 구성 요소를 사용하면 Windows Forms에 있는 개별 컨트롤에 대해 간단하면서도 특수한 도움말 메시지를 표시할 수 있습니다.  <xref:System.Windows.Forms.ToolTip> 구성 요소는 폼에 있는 각 컨트롤에 표시될 텍스트를 지정하는 속성을 제공합니다.  이런 방식의 <xref:System.Windows.Forms.ToolTip> 구성 요소 작업에 대한 자세한 내용은 [방법: 디자인 타임에 Windows Form의 컨트롤에 도구 설명 설정](../../../../docs/framework/winforms/controls/how-to-set-tooltips-for-controls-on-a-windows-form-at-design-time.md)을 참조하십시오.  또한 지정된 시간이 지난 다음 <xref:System.Windows.Forms.ToolTip> 구성 요소가 표시되도록 구성할 수 있습니다.  자세한 내용은 [방법: Windows Forms ToolTip 구성 요소의 지연 변경](../../../../docs/framework/winforms/controls/how-to-change-the-delay-of-the-windows-forms-tooltip-component.md)을 참조하십시오.
