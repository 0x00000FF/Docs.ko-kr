## 참조  
 <xref:System.Drawing>  
 [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)]의 기본 그래픽 기능에 액세스할 수 있도록 합니다.  
  
 <xref:System.Drawing.Drawing2D>  
 고급 2차원 및 벡터 그래픽 기능을 제공합니다.  
  
 <xref:System.Drawing.Imaging>  
 고급 [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)] 이미징 기능을 제공합니다.  
  
 <xref:System.Drawing.Text>  
 고급 [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)] 입력 체계 기능을 제공합니다.  
  
 <xref:System.Drawing.Printing>  
 인쇄 관련 서비스를 제공합니다.  
  
 <xref:System.Windows.Forms.TextRenderer>  
 [!INCLUDE[ndptecgdi](../../../../includes/ndptecgdi-md.md)] 텍스트 그리기 및 길이 맞춤 기능을 제공합니다.
