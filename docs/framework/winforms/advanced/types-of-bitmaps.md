 <span data-ttu-id="0fcec-167">JPEG 압축은 선 그리기, 단색 블록 및 선명한 경계의 경우 제대로 작동 하지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="0fcec-167">JPEG compression does not work well for line drawings, blocks of solid color, and sharp boundaries.</span></span> <span data-ttu-id="0fcec-168">다음 그림에서는 두 개의 Jpeg와 GIF를 함께 사용 하는 BMP를 보여 줍니다.</span><span class="sxs-lookup"><span data-stu-id="0fcec-168">The following illustration shows a BMP along with two JPEGs and a GIF.</span></span> <span data-ttu-id="0fcec-169">Jpeg와 GIF는 BMP에서 압축 되었습니다.</span><span class="sxs-lookup"><span data-stu-id="0fcec-169">The JPEGs and the GIF were compressed from the BMP.</span></span> <span data-ttu-id="0fcec-170">압축 비율은 GIF의 경우 4:1, 작은 JPEG의 경우 4:1, 큰 JPEG의 경우 8:3입니다.</span><span class="sxs-lookup"><span data-stu-id="0fcec-170">The compression ratio is 4:1 for the GIF, 4:1 for the smaller JPEG, and 8:3 for the larger JPEG.</span></span> <span data-ttu-id="0fcec-171">GIF는 선을 따라 선명한 경계를 유지 하지만 Jpeg는 경계를 흐리게 하는 경향이 있습니다.</span><span class="sxs-lookup"><span data-stu-id="0fcec-171">Note that the GIF maintains the sharp boundaries along the lines, but the JPEGs tend to blur the boundaries.</span></span>

 <span data-ttu-id="0fcec-173">JPEG는 파일 형식이 아니라 압축 스키마입니다.</span><span class="sxs-lookup"><span data-stu-id="0fcec-173">JPEG is a compression scheme, not a file format.</span></span> <span data-ttu-id="0fcec-174">JPEG 파일 교환 형식 (.JFIF)은 JPEG 체계에 따라 압축 된 이미지를 저장 하 고 전송 하는 데 일반적으로 사용 되는 파일 형식입니다.</span><span class="sxs-lookup"><span data-stu-id="0fcec-174">JPEG File Interchange Format (JFIF) is a file format commonly used for storing and transferring images that have been compressed according to the JPEG scheme.</span></span> <span data-ttu-id="0fcec-175">웹 브라우저에서 표시 되는 .JFIF 파일은 .jpg 확장명을 사용 합니다.</span><span class="sxs-lookup"><span data-stu-id="0fcec-175">JFIF files displayed by Web browsers use the .jpg extension.</span></span>  
  
### <a name="exchangeable-image-file-exif"></a><span data-ttu-id="0fcec-176">EXIF (교환 이미지 파일)</span><span class="sxs-lookup"><span data-stu-id="0fcec-176">Exchangeable Image File (EXIF)</span></span>  
 <span data-ttu-id="0fcec-177">EXIF는 디지털 카메라에서 캡처한 사진에 사용 되는 파일 형식입니다.</span><span class="sxs-lookup"><span data-stu-id="0fcec-177">EXIF is a file format used for photographs captured by digital cameras.</span></span> <span data-ttu-id="0fcec-178">EXIF 파일은 JPEG 사양에 따라 압축 된 이미지를 포함 합니다.</span><span class="sxs-lookup"><span data-stu-id="0fcec-178">An EXIF file contains an image that is compressed according to the JPEG specification.</span></span> <span data-ttu-id="0fcec-179">또한 EXIF 파일에는 사진 (찍은 날짜, 셔터 속도, 노출 시간 등) 및 카메라에 대 한 정보 (제조업체, 모델 등)에 대 한 정보가 포함 되어 있습니다.</span><span class="sxs-lookup"><span data-stu-id="0fcec-179">An EXIF file also contains information about the photograph (date taken, shutter speed, exposure time, and so on) and information about the camera (manufacturer, model, and so on).</span></span>  
  
### <a name="portable-network-graphics-png"></a><span data-ttu-id="0fcec-180">PNG(이동식 네트워크 그래픽)</span><span class="sxs-lookup"><span data-stu-id="0fcec-180">Portable Network Graphics (PNG)</span></span>  
 <span data-ttu-id="0fcec-181">PNG 형식은 GIF 형식의 많은 이점을 제공 하지만 GIF 이외의 기능도 제공 합니다.</span><span class="sxs-lookup"><span data-stu-id="0fcec-181">The PNG format retains many of the advantages of the GIF format but also provides capabilities beyond those of GIF.</span></span> <span data-ttu-id="0fcec-182">GIF 파일과 마찬가지로 PNG 파일은 정보의 손실 없이 압축 됩니다.</span><span class="sxs-lookup"><span data-stu-id="0fcec-182">Like GIF files, PNG files are compressed with no loss of information.</span></span> <span data-ttu-id="0fcec-183">PNG 파일은 픽셀당 8, 24 또는 48 비트 및 회색조 1, 2, 4, 8 또는 16 비트의 색을 저장할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="0fcec-183">PNG files can store colors with 8, 24, or 48 bits per pixel and grayscales with 1, 2, 4, 8, or 16 bits per pixel.</span></span> <span data-ttu-id="0fcec-184">이와 대조적으로, GIF 파일은 픽셀당 1, 2, 4 또는 8 비트만 사용할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="0fcec-184">In contrast, GIF files can use only 1, 2, 4, or 8 bits per pixel.</span></span> <span data-ttu-id="0fcec-185">PNG 파일에는 픽셀의 색이 배경색과 혼합 되는 정도를 지정 하는 알파 값을 각 픽셀에 저장할 수도 있습니다.</span><span class="sxs-lookup"><span data-stu-id="0fcec-185">A PNG file can also store an alpha value for each pixel, which specifies the degree to which the color of that pixel is blended with the background color.</span></span>

### <a name="tag-image-file-format-tiff"></a><span data-ttu-id="0fcec-188">TIFF (태그 이미지 파일 형식)</span><span class="sxs-lookup"><span data-stu-id="0fcec-188">Tag Image File Format (TIFF)</span></span>
