# Windows Forms 시각적 상속
프로젝트를 만들 때 이전 프로젝트에서 만든 것과 유사한 폼을 호출하는 경우가 있습니다 .  또는 워터마크나 특정 컨트롤 레이아웃과 같은 설정이 포함된 기본 폼을 만든 다음 원본 폼 템플릿을 반복 수정하여 프로젝트에서 다시 사용하는 경우가 있습니다.  폼 상속을 사용하면 기본 폼을 만든 다음 필요한 원본 설정을 유지한 상태로 이 폼을 상속하여 수정할 수 있습니다.  
  
 시각적 상속 선택 대화 상자를 사용하거나 프로그래밍 방식으로 파생 클래스 폼을 만들 수 있습니다.
