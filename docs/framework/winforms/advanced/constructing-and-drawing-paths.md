# 경로 구성 및 그리기
경로는 하나의 단위로서 조작하고 그릴 수 있는 선, 사각형, 곡선, 텍스트 등과 같은 일련의 그래픽 기본 요소입니다.  경로는 여러 개의 열린 *그림*이나 닫힌 그림으로 분리될 수 있고,  그림은 여러 개의 기본 요소로 분리될 수 있습니다.  
  
 <xref:System.Drawing.Graphics> 클래스의 <xref:System.Drawing.Graphics.DrawPath%2A> 메서드를 호출하여 경로를 그릴 수 있고 <xref:System.Drawing.Graphics> 클래스의 <xref:System.Drawing.Graphics.FillPath%2A> 메서드를 호출하여 경로를 채울 수 있습니다.
