# 영역 사용
[!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)] <xref:System.Drawing.Region> 클래스를 사용하면 사용자 지정 도형을 정의할 수 있습니다.  도형은 선, 다각형 및 곡선으로 이루어질 수 있습니다.  
  
 영역을 사용하는 일반적인 예로 적중 테스트와 클리핑이 있습니다.  적중 테스트에서는 화면의 특정 영역에서 마우스를 클릭했는지 확인합니다.  클리핑에서는 그리기를 특정 영역으로 제한합니다.  
  
## 단원 내용  
 [방법: 영역을 사용하여 적중 테스트](../../../../docs/framework/winforms/advanced/how-to-use-hit-testing-with-a-region.md)  
 <xref:System.Drawing.Region>을 사용하여 적중 테스트를 수행하는 방법에 대해 설명합니다.  
  
 [방법: 영역을 사용하여 클리핑](../../../../docs/framework/winforms/advanced/how-to-use-clipping-with-a-region.md)  
 <xref:System.Drawing.Graphics> 개체의 클리핑 영역을 설정하는 방법에 대해 설명합니다.

 이 클래스를 설명하며 이 클래스의 모든 멤버에 대한 링크를 포함합니다.  
  
 <xref:System.Drawing.Graphics>  
 이 클래스를 설명하며 이 클래스의 모든 멤버에 대한 링크를 포함합니다.
