# Windows Forms 응용 프로그램의 네트워킹
.NET Framework는 웹 페이지 표시, 웹 콘텐츠 다운로드, FTP 사이트와의 상호 작용 및 Web Services 사용을 위한 클래스를 제공합니다. 이러한 클래스를 사용하면 응용 프로그램에 네트워크 기능을 쉽게 제공할 수 있습니다.  다음 리소스는 .NET Framework의 네트워킹 기술과 이 기술을 Windows Forms에 통합하는 방법을 이해하는 데 도움을 줍니다.  
  
## 참조  
 <xref:System.Net>  
 네트워크 연결을 처리하는 .NET Framework의 클래스에 대한 루트 네임스페이스입니다.  
  
 <xref:System.Net.WebClient>  
 웹 또는 HTTP 기반 콘텐츠를 프로그래밍 방식으로 편리하게 검색하는 데 사용되는 클래스입니다.  
  
 <xref:System.Net.FtpWebRequest>  
 FTP를 통해 파일을 보내고 받는 데 사용되는 클래스입니다.  
  
 <xref:System.Windows.Forms.WebBrowser>  
 Windows에 포함된 `WebBrowser` 컨트롤에 대한 관리되는 래퍼 클래스입니다.
