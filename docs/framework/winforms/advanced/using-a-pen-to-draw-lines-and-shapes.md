# 펜을 사용하여 선과 도형 그리기
[!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)] `Pen` 개체를 사용하여 선분, 곡선 및 도형의 윤곽선을 그릴 수 있습니다.  이 단원에서 *선*은 짧은 선만을 지칭하기 위해 별도로 지정한 경우가 아니면 일반적으로 이러한 모든 선을 의미합니다.  펜으로 그리는 선의 색, 굵기, 정렬 및 스타일을 제어하는 펜의 속성을 설정합니다.
