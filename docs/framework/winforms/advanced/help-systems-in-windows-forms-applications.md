# Windows Forms 응용 프로그램의 도움말 시스템
유용한 도움말 시스템을 만드는 일은 응용 프로그램 개발자가 사용자를 위해 할 수 있는 가장 중요한 작업 중 하나입니다.  사용자들은 응용 프로그램의 기능에 대해 의문 사항이 있거나 혼동스러울 때 도움말 시스템에 의존합니다.  [HelpProvider 구성 요소](../../../../docs/framework/winforms/controls/helpprovider-component-windows-forms.md)를 사용하여 Window 기반 응용 프로그램의 도움말 시스템을 쉽게 제공할 수 있습니다.  
  
## 여러 형식의 도움말  
 Windows Forms <xref:System.Windows.Forms.HelpProvider> 구성 요소를 사용하여 HTML Help 1.x 도움말 파일\(HTML Help Workshop으로 만든 .chm 또는 .htm 파일\)을 Windows 기반 응용 프로그램에 연결할 수 있습니다.  <xref:System.Windows.Forms.HelpProvider> 구성 요소를 사용하여 Windows Forms의 컨트롤이나 특정 컨트롤에 대해 상황에 맞는 도움말을 제공할 수 있습니다.  또한 <xref:System.Windows.Forms.HelpProvider> 구성 요소는 목차의 주 페이지, 색인, 검색 함수 등의 특정 영역에서 도움말 파일을 엽니다.  <xref:System.Windows.Forms.HelpProvider> 구성 요소에 대한 자세한 내용은 [HelpProvider 구성 요소 개요](../../../../docs/framework/winforms/controls/helpprovider-component-overview-windows-forms.md)를 참조하십시오.  <xref:System.Windows.Forms.HelpProvider> 구성 요소를 사용하여 Windows Forms에 팝업 도움말을 표시하는 방법은 [방법: 팝업 도움말 표시](../../../../docs/framework/winforms/advanced/how-to-display-pop-up-help.md)를 참조하십시오.  <xref:System.Windows.Forms.ToolTip> 구성 요소를 사용하여 컨트롤 관련 도움말을 표시하는 방법은 [ToolTip을 사용한 컨트롤 도움말](../../../../docs/framework/winforms/advanced/control-help-using-tooltips.md)을 참조하십시오.  
  
 HTML Help Workshop을 사용하여 HTML Help 1.x 파일을 만들 수 있습니다.  HTML 도움말에 대한 자세한 내용은 MSDN에서 "HTML Help Workshop"이나 다른 "HTML Help" 항목을 참조하십시오.
