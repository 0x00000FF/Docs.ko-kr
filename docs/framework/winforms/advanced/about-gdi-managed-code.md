# <a name="about-gdi-managed-code"></a>GDI+ 관리 코드 정보

GDI +는 2 차원 벡터 그래픽, 이미징 및 입력 체계를 제공 하는 Windows 운영 체제의 일부입니다. Gdi +는 새 기능을 추가 하 고 기존 기능을 최적화 하 여 GDI (이전 버전의 Windows에 포함 된 그래픽 장치 인터페이스)를 개선 합니다.

GDI + 관리 되는 클래스 인터페이스 (래퍼 집합)는 XML Web services 및 기타 응용 프로그램을 빌드, 배포 및 실행 하기 위한 환경인 .NET Framework의 일부입니다.

이 섹션에서는 관리 코드를 사용 하는 프로그래머를 위한 GDI + API에 대 한 정보를 제공 합니다.
