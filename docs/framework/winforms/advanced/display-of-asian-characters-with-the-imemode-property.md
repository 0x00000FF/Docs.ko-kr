# <a name="display-of-asian-characters-with-the-imemode-property"></a>ImeMode 속성을 사용하여 전자 단어 표시
<xref:System.Windows.Forms.Control.ImeMode%2A> 속성 폼과 컨트롤에서 입력된 방법 편집기 (입력기)에 대 한 특정 모드를 강제 적용을 사용 합니다. IME는 일반 키보드로 인코딩할 수 있는 것보다 더 많은 문자를 포함하고 있으므로 중국어, 일본어 및 한국어 스크립트를 작성하는 데 필수적인 구성 요소입니다. 예를 들어 특정 텍스트 상자에 ASCII 문자만 허용하도록 할 수 있습니다. 이러한 경우에 설정할 수 있습니다는 <xref:System.Windows.Forms.Control.ImeMode%2A> 속성을 <xref:System.Windows.Forms.ImeMode> 해당 특정 텍스트 상자에 대 한 ASCII 문자를 입력할 수만 있습니다. 기본값은 <xref:System.Windows.Forms.Control.ImeMode%2A> 속성은 <xref:System.Windows.Forms.ImeMode>이므로 양식의 모든 컨트롤 폼에 대 한 속성을 설정 하는 경우 해당 설정을 상속 합니다. 자세한 내용은 참조 <xref:System.Windows.Forms.Control.ImeMode%2A> ) 및 <xref:System.Windows.Forms.ImeMode>합니다.
