# 이미지 다시 칠하기
다시 칠하기는 이미지의 색을 조절하는 작업입니다.  다시 칠하기의 예로는 색을 다른 색으로 바꾸기, 다른 색을 기준으로 색의 농도 조절, 모든 색의 밝기 또는 대비 조절 및 색을 회색 음영으로 변환 등이 있습니다.
