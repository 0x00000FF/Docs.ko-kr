# 응용 프로그램 프로토콜 사용
.NET Framework 일반적으로 사용 되는 인터넷 응용 프로그램 프로토콜을 지원합니다.  이 섹션을 사용 하 여 정보가  [HTTP](../../../docs/framework/network-programming/http.md),  [TCP](../../../docs/framework/network-programming/tcp-udp.md), 및  [UDP](../../../docs/framework/network-programming/tcp-udp.md) 프로토콜 뿐만 아니라 정보를 사용 하는  [Windows 소켓](../../../docs/framework/network-programming/sockets.md) 사용자 지정 프로토콜을 구현 하는 인터페이스.
