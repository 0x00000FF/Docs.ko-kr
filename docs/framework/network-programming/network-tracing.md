 <span data-ttu-id="e754f-110">추적 기능이 사용될 때, **System.Net** 클래스에 의해 출력되는 추적 정보를 캡처할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="e754f-110">When tracing is enabled, you can capture trace information that is output by **System.Net** classes.</span></span> <span data-ttu-id="e754f-111">추적 정보를 생성하는 네트워킹 클래스 멤버에는 해당 .NET Framework 클래스 라이브러리 설명서의 설명 섹션에 다음 내용이 포함되어 있습니다.</span><span class="sxs-lookup"><span data-stu-id="e754f-111">Networking class members that generate tracing information include the following note in the Remarks section of their NET Framework class library documentation:</span></span>
