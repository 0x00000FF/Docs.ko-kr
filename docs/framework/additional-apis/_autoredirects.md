# <a name="httpwebrequestautoredirects-field"></a>HttpWebRequest 합니다. \_AutoRedirects 필드

`HttpWebRequest._AutoRedirects`이 <xref:System.Int32> 이 대 한 리디렉션 수를 반영 하는 <xref:System.Net.HttpWebRequest>합니다.

## <a name="syntax"></a>구문  
  
```csharp  
private int _AutoRedirects
```

> [!WARNING]
> `HttpWebRequest._AutoRedirects` 필드는 내부 전용 이며 코드에서 직접 사용할 업그레이드용은 아닙니다.
> 
> Microsoft은 프로덕션 응용 프로그램의 어떤 상황에서이 필드의 사용을 지원 하지 않습니다.
