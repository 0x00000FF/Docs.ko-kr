# <a name="connection-class"></a>Connection 클래스

`Connection` 클래스 구문 분석 하 여 서버 응답, 큐 요청과 파이프라인 요청 합니다.

## <a name="syntax"></a>구문
  
```csharp  
internal class Connection : PooledStream
```

> [!WARNING]
> `Connection` 클래스는 내부 전용 이며 코드에서 직접 사용할 업그레이드용은 아닙니다.
> 
> Microsoft은 프로덕션 응용 프로그램의 어떤 상황에서이 클래스의 사용을 지원 하지 않습니다.

## <a name="requirements"></a>요구 사항

**Namespace:**<xref:System.Net>

**어셈블리:** 시스템 (System.dll)

**.NET framework 버전:** 2.0부터 사용 가능 합니다.
