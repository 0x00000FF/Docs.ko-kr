# <a name="sqlstreamchars-constructor"></a>SqlStreamChars 생성자

`SqlStreamChars` 클래스의 새 인스턴스를 초기화합니다. 이 생성자를 포함 하는 어셈블리에는 SQLAccess .dll과의 friend 관계가 있습니다. SQL Server에서 사용 하기 위한 것입니다. 다른 데이터베이스의 경우 해당 데이터베이스에서 제공 하는 호스팅 메커니즘을 사용 합니다.

