# <a name="datamemberlisteditor-class"></a>데이터 멤버리스트편집기 클래스

데이터 바인딩된 개체(null이 `DataSource` 아닌 속성이 있는 개체)의 속성을 편집하기 위한 드롭다운 `DataSource` 사용자 인터페이스를 제공하여 개체의 모든 속성을 나열하여 값 원본을 선택합니다.  
  
## <a name="syntax"></a>구문
  
```csharp  
internal class DataMemberListEditor : UITypeEditor
```

> [!WARNING]
> 클래스는 `DataMemberListEditor` 내부이며 코드에서 직접 사용할 수 없습니다.
>
> Microsoft는 어떠한 상황에서도 프로덕션 응용 프로그램에서 이 클래스의 사용을 지원하지 않습니다.
