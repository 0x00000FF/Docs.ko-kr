# <a name="datamemberfieldeditor-class"></a>데이터멤버필드에디터 클래스

개체의 모든 `DataSource` 속성을 나열하여 데이터 바인딩된 개체의 속성을 편집하여 값 원본을 선택할 수 있는 사용자 인터페이스를 제공합니다.  
  
## <a name="syntax"></a>구문
  
```csharp
internal class DataMemberFieldEditor : UITypeEditor
```

> [!WARNING]
> 클래스는 `DataMemberFieldEditor` 내부이며 코드에서 직접 사용할 수 없습니다.
>
> Microsoft는 어떠한 상황에서도 프로덕션 응용 프로그램에서 이 클래스의 사용을 지원하지 않습니다.
