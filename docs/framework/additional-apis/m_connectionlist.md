# <a name="connectiongroupmconnectionlist-field"></a>ConnectionGroup.m\_ConnectionList 필드

`ConnectionGroup.m_ConnectionList` 이 <xref:System.Collections.ArrayList> 동일한 URI 및 공유를 사용 하는 연결 개체의 일부 다른 속성에 대 한 동일한 값 만료 및 인증을 선택 합니다.

## <a name="syntax"></a>구문
  
```csharp  
private ArrayList m_ConnectionList
```

> [!WARNING]
> `ConnectionGroup.m_ConnectionList` 필드는 전용 이며 코드에서 직접 사용할 업그레이드용은 아닙니다.
> 
> Microsoft은 프로덕션 응용 프로그램의 어떤 상황에서이 필드의 사용을 지원 하지 않습니다.
