## <a name="description"></a>설명  
 Catch 또는 Finally 작업에서 예외가 throw되었음을 나타냅니다.  
  
## <a name="message"></a>메시지  
 TryCatch 작업 '%1'에 연결된 Catch 또는 Finally 작업에서 예외가 throw되었습니다.
