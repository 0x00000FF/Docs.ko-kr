## <a name="properties"></a>속성  
  
|||  
|-|-|  
|ID|1007|  
|키워드|WFRuntime|  
|수준|정보|  
|채널|Microsoft-Windows-응용 프로그램 서버-응용 프로그램/디버그|

 WorkflowApplication Id: '%1'이(가) 지속되었습니다.
