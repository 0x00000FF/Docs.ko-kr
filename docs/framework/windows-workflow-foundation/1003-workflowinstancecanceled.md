## <a name="description"></a>설명  
 워크플로 인스턴스가 Closed 상태에서 완료되었음을 나타냅니다.  
  
## <a name="message"></a>메시지  
 WorkflowInstance Id: '%1'이(가) Canceled 상태에서 완료되었습니다.
