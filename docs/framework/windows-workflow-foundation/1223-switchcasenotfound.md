# 1223 - SwitchCaseNotFound
## 속성  
  
|||  
|-|-|  
|ID|1223|  
|키워드|WFActivities|  
|수준|정보|  
|채널|Microsoft\-Windows\-응용 프로그램 서버\-응용 프로그램\/디버그|  
  
## 설명  
 Expression 결과와 일치하는 Case 작업을 Switch 작업에서 찾을 수 없음을 나타냅니다.  
  
## 메시지  
 Switch 작업 '%1'에서 Expression 결과와 일치하는 Case 작업을 찾을 수 없습니다.
