# 연결 문자열 및 연결 문자열 이름
**Connection String** 속성은 SQL 워크플로 인스턴스 저장소에서 지속성 데이터베이스에 연결하기 위해 사용해야 하는 연결 문자열을 지정합니다.이 매개 변수는 선택적 매개 변수입니다.**Connection String Name** 속성은 SQL 워크플로 인스턴스 저장소에서 지속성 데이터베이스에 연결하기 위해 사용해야 하는 명명된 연결 문자열의 이름을 지정합니다.이 매개 변수는 선택적 매개 변수입니다.SQL 워크플로 인스턴스 저장소에서 명명된 기본 연결 문자열 **DefaultSqlWorkflowInstanceStoreConnectionString**을 사용하지 않도록 하려면 Connection String Name 속성 또는 Connection String 속성 값을 지정해야 합니다.
