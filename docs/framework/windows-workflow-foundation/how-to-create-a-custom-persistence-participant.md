     <span data-ttu-id="54fc3-112">ReadWriteValues 사전의 값으로 패키지 되 **InstanceValue** 개체입니다.</span><span class="sxs-lookup"><span data-stu-id="54fc3-112">The values in the readWriteValues dictionary are packaged as **InstanceValue** objects.</span></span> <span data-ttu-id="54fc3-113">쓰기 전용 사전의 값으로 패키지 되 **InstanceValue** 은 InstanceValueOptions.Optional 및 InstanceValueOption.WriteOnly가 설정 된 개체입니다.</span><span class="sxs-lookup"><span data-stu-id="54fc3-113">The values in the write-only dictionary are packaged as **InstanceValue** objects with InstanceValueOptions.Optional and InstanceValueOption.WriteOnly set.</span></span> <span data-ttu-id="54fc3-114">각 **InstanceValue** 에서 제공 합니다 **CollectValues** 모든 지 속성 참석자를 통해 구현에는 고유한 이름이 있어야 합니다.</span><span class="sxs-lookup"><span data-stu-id="54fc3-114">Each **InstanceValue** provided by the **CollectValues** implementations across all persistence participants must have a unique name.</span></span>
