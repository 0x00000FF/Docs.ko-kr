# 4207 - MaximumRetriesExceededForSqlCommand
## 속성  
  
|||  
|-|-|  
|ID|4207|  
|키워드|할당량, WFInstanceStore|  
|수준|정보|  
|채널|Microsoft\-Windows\-응용 프로그램 서버\-응용 프로그램\/디버그|  
  
## 설명  
 SQL 공급자가 최대 재시도 횟수를 초과하여 SQL 명령 재시도를 중단했음을 나타냅니다.  
  
## 메시지  
 최대 재시도 횟수를 초과하여 SQL 명령 재시도를 중단합니다.
