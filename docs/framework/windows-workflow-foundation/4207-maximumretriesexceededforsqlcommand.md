## <a name="description"></a>설명  
 SQL 공급자가 최대 재시도 횟수를 초과하여 SQL 명령 재시도를 중단했음을 나타냅니다.  
  
## <a name="message"></a>메시지  
 최대 재시도 횟수를 초과하여 SQL 명령 재시도를 중단합니다.
