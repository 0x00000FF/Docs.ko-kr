# .NET Framework 4.5 기본 제공 활동 라이브러리
[!INCLUDE[netfx_current_long](../../../includes/netfx-current-long-md.md)]에는 확장된 기능이 있는 새로운 활동 라이브러리가 포함됩니다.활동 유형은 다음을 포함합니다.  
  
-   [흐름 제어](../../../docs/framework/windows-workflow-foundation//control-flow-activities-in-wf.md)  
  
-   [순서도](../../../docs/framework/windows-workflow-foundation//flowchart-activities-in-wf.md)  
  
-   [상태 시스템](../../../docs/framework/windows-workflow-foundation//state-machine-activities-in-wf.md)  
  
-   [메시징 활동](../../../docs/framework/wcf/feature-details/messaging-activities.md)  
  
-   [런타임](../../../docs/framework/windows-workflow-foundation//runtime-activities-in-wf.md)  
  
-   [기본 형식](../../../docs/framework/windows-workflow-foundation//primitives-activities-in-wf.md)  
  
-   [트랜잭션](../../../docs/framework/windows-workflow-foundation//transaction-activities-in-wf.md)  
  
-   [컬렉션](../../../docs/framework/windows-workflow-foundation//collection-activities-in-wf.md)  
  
-   [오류 처리](../../../docs/framework/windows-workflow-foundation//error-handling-activities-in-wf.md)  
  
-   [마이그레이션](../../../docs/framework/windows-workflow-foundation//migration-activity-in-wf.md)  
  
 기본 제공 활동 라이브러리의 활동은 워크플로에 구성하거나 사용자 지정 활동과 함께 사용할 수 있습니다.기본 제공 활동 라이브러리의 활동은 봉인됩니다. 상속을 통해 새로운 기능을 만드는 데 사용되지 않습니다.
