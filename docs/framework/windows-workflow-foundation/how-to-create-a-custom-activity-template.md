## <a name="example"></a><span data-ttu-id="f8399-138">예</span><span class="sxs-lookup"><span data-stu-id="f8399-138">Example</span></span>  
 <span data-ttu-id="f8399-139">MyDelayActivity.cs 파일에 다음 코드가 들어 있어야 합니다.</span><span class="sxs-lookup"><span data-stu-id="f8399-139">The MyDelayActivity.cs file should contain the following code.</span></span>
