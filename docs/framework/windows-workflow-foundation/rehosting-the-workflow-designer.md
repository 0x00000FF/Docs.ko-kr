# Workflow Designer 재호스팅
워크플로 만들기, 수정, 모니터링을 위해 [!INCLUDE[vs_current_long](../../../includes/vs-current-long-md.md)] 외부 환경에서 [!INCLUDE[wfd1](../../../includes/wfd1-md.md)]를 다시 호스팅할 수 있습니다.  
  
 <xref:System.Activities.Presentation.WorkflowDesigner> 형식은 캔버스, 속성표 및 기타 요소의 래퍼이며 기본 프로그래밍 모델을 노출하여 대부분의 디자이너 다시 호스팅 시나리오를 처리합니다.[!INCLUDE[avalon1](../../../includes/avalon1-md.md)] 응용 프로그램 내부에 <xref:System.Activities.Presentation.WorkflowDesigner>를 호스팅하는 것은 [!INCLUDE[wfd2](../../../includes/wfd2-md.md)]에 대한 일반적인 재호스팅 시나리오입니다.
