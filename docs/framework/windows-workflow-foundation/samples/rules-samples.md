# 규칙 샘플
이 단원에는 [!INCLUDE[netfx35_long](../../../../includes/netfx35-long-md.md)]의 워크플로 규칙 엔진을 사용하는 방법을 보여 주는 샘플이 포함되어 있습니다.  
  
## 단원 내용  
 [Advanced Policy](../../../../docs/framework/windows-workflow-foundation/samples/advanced-policy.md)  
 Simple Policy 샘플을 확장하는 방법을 보여 줍니다.Simple Policy 예제의 가계 할인 규칙과 기업 할인 규칙 이외에 새로운 규칙이 몇 가지 추가되었습니다.  
  
 [Simple Policy](../../../../docs/framework/windows-workflow-foundation/samples/simple-policy.md)  
 워크플로에 <xref:System.Workflow.Activities.PolicyActivity> 활동을 사용하는 방법을 보여 줍니다.  
  
 [IfElse With Rules](../../../../docs/framework/windows-workflow-foundation/samples/ifelse-with-rules.md)  
 <xref:System.Workflow.Activities.IfElseActivity> 활동에 규칙 조건을 사용하는 방법을 보여 줍니다.  
  
 [조건화된 활동 그룹](../../../../docs/framework/windows-workflow-foundation/samples/conditioned-activity-group.md)  
 여행 예약 응용 프로그램을 보여 줍니다.  
  
 [정책을 사용한 주문 처리](../../../../docs/framework/windows-workflow-foundation/samples/order-processing-with-policy.md)  
 [!INCLUDE[wf](../../../../includes/wf-md.md)]의 [!INCLUDE[netfx35_long](../../../../includes/netfx35-long-md.md)]에 도입된 몇 가지 주요 기능을 보여 줍니다.
