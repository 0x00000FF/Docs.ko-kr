# <a name="persistence"></a>지속성
이 섹션에는 워크플로 영구적인 매체에 Windows WF (Workflow Foundation)에 유지를 보여 주는 샘플이 포함 되어 있습니다.  
  
## <a name="in-this-section"></a>섹션 내용  
 [워크플로 응용 프로그램 유지](../../../../docs/framework/windows-workflow-foundation/samples/persisting-a-workflow-application.md)  
 <xref:System.Activities.WorkflowApplication>을 실행하고, 유휴 상태가 되면 언로드한 다음, 다시 로드하여 실행을 계속하는 방법을 보여 줍니다.  
  
 [기본 제공 구성](../../../../docs/framework/windows-workflow-foundation/samples/built-in-configuration.md)  
 SQL 워크플로 인스턴스 저장소를 사용하고 구성하는 방법을 보여 줍니다.  
  
 [SQLStoreExtensibility](../../../../docs/framework/windows-workflow-foundation/samples/sqlstoreextensibility.md)  
 SQL 워크플로 인스턴스 저장소에서 승격된 속성을 사용하고 구성하는 방법을 보여 줍니다.  
  
 [속성 승격 활동](../../../../docs/framework/windows-workflow-foundation/samples/property-promotion-activity.md)  
 이 샘플에서는 <xref:System.Activities.DurableInstancing.SqlWorkflowInstanceStore> 승격 기능을 워크플로 작성 환경에 직접 통합하는 종단 간 솔루션을 제공합니다.
