 <span data-ttu-id="a5098-107">활동 작성자는 (활동 내에서) 다음을 수행해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="a5098-107">The activity author (inside the activity) must:</span></span>

-   <span data-ttu-id="a5098-110">`System.Activities.CodeActivity.OnGetConstraints()`를 재정의하고 `PriceGreaterThanCost` 컬렉션에 제약 조건(<xref:System.Collections.IList>)을 추가합니다.</span><span class="sxs-lookup"><span data-stu-id="a5098-110">Override `System.Activities.CodeActivity.OnGetConstraints()` and add the constraint (`PriceGreaterThanCost`) to the constraints <xref:System.Collections.IList>.</span></span>
