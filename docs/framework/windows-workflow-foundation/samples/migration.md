# 마이그레이션
이 단원에는 [!INCLUDE[netfx35_short](../../../../includes/netfx35-short-md.md)]를 사용하여 작성된 [!INCLUDE[wf](../../../../includes/wf-md.md)] 콘텐츠를 [!INCLUDE[netfx40_short](../../../../includes/netfx40-short-md.md)]로 마이그레이션하는 방법을 보여 주는 샘플이 포함되어 있습니다.  
  
## 단원 내용  
 [.NET Framework 4.5 워크플로에서 .NET Framework 3.0 또는 .NET Framework 3.5 활동 사용](../../../../docs/framework/windows-workflow-foundation/samples/using-a-net-3-0-or-net-3-5-activity-in-a-net-4-5-workflow.md)  
 <xref:System.Activities.Statements.Interop> 활동을 사용하여 문자열을 사용자 지정 [!INCLUDE[vstecwinfx](../../../../includes/vstecwinfx-md.md)] 활동에 인수로 전달하는 방법을 보여 줍니다.  
  
 [외부 데이터 교환과 함께 Interop 사용](../../../../docs/framework/windows-workflow-foundation/samples/using-interop-with-external-data-exchange.md)  
 WF4 워크플로 서비스의 <xref:System.Activities.Statements.Interop> 활동을 사용하여 <xref:System.Workflow.Activities.ExternalDataExchangeService> 및 메서드 호출과 이벤트 처리를 위한 해당 사용자 지정 활동을 사용하는 WF3 워크플로를 구성하고 실행하는 방법을 보여 줍니다.
