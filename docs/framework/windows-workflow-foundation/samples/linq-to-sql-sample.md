## <a name="code-sample-that-uses-the-custom-activity"></a><span data-ttu-id="3d402-122">사용자 지정 활동을 사용하는 코드 샘플</span><span class="sxs-lookup"><span data-stu-id="3d402-122">Code Sample that uses the Custom Activity</span></span>  
 <span data-ttu-id="3d402-123">다음 코드 예제에서는 `FindInSqlTable` 사용자 지정 활동을 사용하여 `Employee`라는 SQL Server 테이블에서 `Role` 열이 `SDE`인 모든 행을 찾습니다.</span><span class="sxs-lookup"><span data-stu-id="3d402-123">The following code example uses the `FindInSqlTable` custom activity to find all rows in a SQL Server table named `Employee` where the `Role` column is equal to `SDE`.</span></span>  
  
```csharp  
new FindInSqlTable<Employee>   
{  
    ConnectionString = @"Data Source=.\SQLExpress;Initial Catalog=LinqToSqlSample;Integrated Security=True",                          
    Predicate = new LambdaValue<Func<Employee, bool>>(c => new Func<Employee, bool>(emp => emp.Role.Equals("SDE"))),  
    Result = new OutArgument<IList<Employee>>(employees)  
},  
```
