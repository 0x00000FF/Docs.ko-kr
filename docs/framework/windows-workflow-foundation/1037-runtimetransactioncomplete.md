# 1037 - RuntimeTransactionComplete
## 속성  
  
|||  
|-|-|  
|ID|1037|  
|키워드|WFRuntime|  
|수준|Verbose|  
|채널|Microsoft\-Windows\-응용 프로그램 서버\-응용 프로그램\/디버그|  
  
## 설명  
 런타임 트랜잭션이 완료되었음을 나타냅니다.  
  
## 메시지  
 런타임 트랜잭션이 '%1' 상태로 완료되었습니다.
