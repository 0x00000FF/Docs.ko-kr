# Manipulations and Inertia
조작 및 관성 프로세서 클래스는 응용 프로그램이 화면에서 사용자 인터페이스\(UI\) 요소를 자연스럽고 직관적인 방법으로 이동할 수 있도록 도와줍니다.  조작은 사용자가 마우스 장치, 스타일러스, 손가락\(직접 터치하여 조작이 가능한 경우\) 등의 조작자를 사용하여 구성 요소를 이동하고 회전하고 크기를 조정할 수 있도록 해줍니다.  관성은 동작 중인 UI 요소에 대한 마찰력을 시뮬레이션하여 해당 UI 요소가 현실에서와 같이 움직이도록 합니다.  그러면 해당 요소의 선형 및 각도 움직임이 모두 점차적으로 느려진 후에 멈추게 됩니다.  
  
 조작 및 관성 처리는 Microsoft WPF\(Windows Presentation Foundation\) 또는 Microsoft XNA 같은 다양한 UI 프레임워크에서 사용할 수 있습니다.
