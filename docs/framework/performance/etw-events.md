# .NET Framework의 ETW 이벤트
ETW\(Windows용 이벤트 추적\)는 Windows 운영 체제에서 제공하는 오버헤드가 낮고, 확장 가능한 고성능 시스템입니다.  .NET Framework에서 제공하는 프로파일링 및 디버깅 지원을 보완하며 다양한 시나리오의 문제를 해결하는 데 사용될 수 있습니다.  ETW에 대 한 일반적인 정보를 보려면 MSDN Library에서 [Improve Debugging and Performance Tuning with ETW](http://go.microsoft.com/fwlink/?LinkID=161142) 를 참조하십시오.  
  
 .NET Framework에서 ETW 이벤트 추적은 CLR\(공용 언어 런타임\), [작업 병렬 라이브러리](../../../docs/standard/parallel-programming/task-parallel-library-tpl.md) 및 [Parallel LINQ \(PLINQ\)](../../../docs/standard/parallel-programming/parallel-linq-plinq.md)에 대해 사용할 수 있습니다.
