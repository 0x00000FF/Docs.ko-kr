# .NET Framework 4.6 및 4.5
> [!NOTE]
>  이 .NET Framework 콘텐츠 집합에는 .NET Framework 버전 4.5, 4.5.1, 4.5.2, 4.6 및 4.6.1에 대한 정보가 포함되어 있습니다. .NET Framework를 다운로드하려면 [.NET Framework 설치](../../docs/framework/install/guide-for-developers.md)를 참조하세요. .NET Framework 4.5,  [!INCLUDE[net_v46](../../includes/net-v46-md.md)] 및 해당 포인트 릴리스의 새로운 기능과 변경 사항 목록은 [새로운 기능](../../docs/framework/whats-new/index.md)을 참조하세요. 지원되는 플랫폼 목록은 [.NET Framework 시스템 요구 사항](../../docs/framework/get-started/system-requirements.md)을 참조하십시오.  
  
 .NET Framework는 Windows, Windows Phone, Windows Server 및 Microsoft Azure용 앱을 빌드하기 위한 개발 플랫폼으로, CLR\(공용 언어 런타임\)과 광범위한 기술을 지원하는 클래스, 인터페이스 및 값 형식이 포함된 .NET Framework 클래스 라이브러리로 구성됩니다. .NET Framework에서는 관리되는 실행 환경, 간단한 개발 및 배포, Visual Basic 및 Visual C\#을 포함한 다양한 프로그래밍 언어와의 통합 기능을 제공합니다.  
  
> [!NOTE]
>  .NET Framework를 다운로드하려면 [.NET Framework 설치](../../docs/framework/install/guide-for-developers.md)를 참조하세요.  
  
 사용자와 개발자 모두를 위한 .NET Framework에 대한 일반적인 소개는 [시작](../../docs/framework/get-started/index.md)을 참조하십시오. .NET Framework의 아키텍처 및 주요 기능에 대한 소개는 [개요](../../docs/framework/get-started/overview.md)를 참조하십시오.  
  
 .NET Framework에 대한 설명서에는 포괄적인 클래스 라이브러리 참조, 개념적 개요, 단계별 절차와 샘플, 컴파일러 및 명령줄 도구에 대한 정보가 포함되어 있습니다. 관심있는 정보를 찾으려면 다음 주요 항목 목록을 참조하십시오.
