# <a name="net-framework"></a>.NET Framework

.NET Framework는 Windows, Windows Phone, Windows Server 및 Microsoft Azure용 응용 프로그램을 빌드하기 위한 개발 플랫폼으로, CLR(공용 언어 런타임)과 광범위한 기술을 지원하는 클래스, 인터페이스 및 값 형식이 포함된 .NET Framework 클래스 라이브러리로 구성됩니다. .NET Framework에서는 관리되는 실행 환경, 간단한 개발 및 배포, Visual Basic, Visual C# 및 F#을 포함한 다양한 프로그래밍 언어와의 통합 기능을 제공합니다.

[Windows 컨테이너](https://msdn.microsoft.com/en-us/virtualization/windowscontainers/about/about_overview)에서 기존 응용 프로그램을 실행하는 방법을 알아보려면 [.NET Framework의 Docker](docker/index.md)를 확인해 보세요.
