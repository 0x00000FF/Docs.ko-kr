# <a name="developing-web-applications-with-aspnet"></a>ASP.NET을 사용하여 웹 응용 프로그램 개발
ASP.NET은 웹 응용 프로그램을 만들기 위한 .NET Framework 기술입니다. ASP.NET 설명서는 MSDN 라이브러리의 [웹 개발](http://go.microsoft.com/fwlink/p/?LinkID=235794) 섹션과 [www.asp.net 웹 사이트](http://go.microsoft.com/fwlink/p/?LinkId=251042)에 있습니다. 사용자 편의를 위해 제공되는 다음 링크를 참조하십시오.  
  
-   [ASP.NET 및 웹용 Visual Studio](http://msdn.microsoft.com/library/c3e0fd2f-7a94-47c4-afc0-ecfb7233e15a)  
  
-   [ASP.NET MVC](http://go.microsoft.com/fwlink/p/?LinkID=227227)  
  
-   [ASP.NET 웹 페이지 2](http://go.microsoft.com/fwlink/p/?LinkId=251040)  
  
-   [ASP.NET Web API](http://go.microsoft.com/fwlink/p/?LinkId=251041)
