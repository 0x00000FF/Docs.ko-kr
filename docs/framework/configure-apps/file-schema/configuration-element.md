# <a name="configuration-element"></a>\<구성 > 요소

공용 언어 런타임 및 .NET Framework 응용 프로그램에서 사용하는 모든 구성 파일의 루트 요소입니다.

**\<구성>**

## <a name="syntax"></a>구문

```xml
<configuration>
  <!-- Configuration settings -->
</configuration>
```

## <a name="attributes"></a>특성

없음

## <a name="parent-element"></a>부모 요소

없음
