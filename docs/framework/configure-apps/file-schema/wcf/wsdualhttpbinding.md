## <a name="remarks"></a><span data-ttu-id="62306-200">설명</span><span class="sxs-lookup"><span data-stu-id="62306-200">Remarks</span></span>  
 <span data-ttu-id="62306-201">`WSDualHttpBinding`은 `WSHttpBinding`과 동일하게 웹 서비스 프로토콜을 지원하지만 이중 계약에만 사용할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="62306-201">The `WSDualHttpBinding` provides the same support for Web Service protocols as the `WSHttpBinding`, but for use with duplex contracts.</span></span> <span data-ttu-id="62306-202">`WSDualHttpBinding`은 SOAP 보안만 지원하며 신뢰할 수 있는 메시징이 필요합니다.</span><span class="sxs-lookup"><span data-stu-id="62306-202">`WSDualHttpBinding` only supports SOAP security and requires reliable messaging.</span></span> <span data-ttu-id="62306-203">이 바인딩에서는 서비스의 콜백 끝점을 제공하는 공용 URI가 클라이언트에 있어야 합니다.</span><span class="sxs-lookup"><span data-stu-id="62306-203">This binding requires that the client has a public URI that provides a callback endpoint for the service.</span></span> <span data-ttu-id="62306-204">이 공용 URI는 `clientBaseAddress` 특성에서 제공됩니다.</span><span class="sxs-lookup"><span data-stu-id="62306-204">This is provided by the `clientBaseAddress` attribute.</span></span> <span data-ttu-id="62306-205">이중 바인딩은 클라이언트의 IP 주소를 서비스에 노출합니다.</span><span class="sxs-lookup"><span data-stu-id="62306-205">A dual binding exposes the IP address of the client to the service.</span></span> <span data-ttu-id="62306-206">클라이언트는 보안을 사용하여 신뢰하는 서비스에만 연결해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="62306-206">The client should use security to ensure that it only connects to services it trusts.</span></span>  
  
 <span data-ttu-id="62306-207">이 바인딩을 사용하여 하나 이상의 SOAP 중간 매개자를 통해 안정적으로 통신할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="62306-207">This binding can be used to communicate reliably through one or more SOAP intermediaries.</span></span>  
  
 <span data-ttu-id="62306-208">이 바인딩은 기본적으로 안정성을 위한 WS-ReliableMessaging, 메시지 보안 및 인증을 위한 WS-Security, 메시지 배달을 위한 HTTP 및 텍스트/XML 메시지 인코딩을 지원하는 런타임 스택을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="62306-208">By default, this binding generates a runtime stack with WS-ReliableMessaging for reliability, WS-Security for message security and authentication, HTTP for message delivery, and a Text/XML message encoding.</span></span>
