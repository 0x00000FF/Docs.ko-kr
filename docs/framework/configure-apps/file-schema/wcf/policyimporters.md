# <a name="policyimporters"></a>\<policyImporters>
이 구성 요소는 바인딩에 대한 사용자 지정 정책 어설션의 가져오기를 제어하는 모든 정책 가져오기 지정합니다. 각 자식 요소는 <`policyImporter`> 어설션에서 요구 하는 기능을 구현 하는 사용자 지정 바인딩 요소를 연결할 수 있을 뿐만 아니라 바인딩 기능에 대 한 사용자 지정 정책 어설션을 검색 하는 데 사용 되는 합니다.
