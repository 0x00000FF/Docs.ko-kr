# 전송
전송 계층은 채널 스택의 최하위 수준에 있습니다.  [!INCLUDE[indigo1](../../../../../includes/indigo1-md.md)]에 사용되는 기본 전송은 HTTP, HTTPS, TCP 및 명명된 파이프입니다.  이 섹션은 다양한 전송 형식을 구성할 수 있게 해 주는 모든 구성 요소를 나열합니다.
