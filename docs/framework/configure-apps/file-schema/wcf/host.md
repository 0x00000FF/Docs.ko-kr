# <a name="lthostgt"></a>&lt;호스트&gt;
서비스 호스트의 설정을 지정합니다.  
  
 \<시스템입니다. ServiceModel >  
\<서비스 >  
\<서비스 >  
\<호스트 >  
  
## <a name="syntax"></a>구문  
  
```xml  
<host>  
      <baseAddresses>  
         <baseAddress baseAddress="string" />  
      </baseAddresses>  
      <timeOuts closeTimeout="TimeSpan"  
         openTimeout="TimeSpan" >  
</host>  
```  
  
## <a name="type"></a>형식  
 `Type`
