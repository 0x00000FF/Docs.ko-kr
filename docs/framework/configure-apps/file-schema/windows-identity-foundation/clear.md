# <a name="ltcleargt"></a>&lt;clear&gt;

 \<system.identityModel >  
\<identityConfiguration >  
\<securityTokenHandlers >  
\<지우기 >  
  
## <a name="syntax"></a>구문  
  
```xml  
<system.identityModel>  
  <identityConfiguration>  
    <securityTokenHandlers>  
      <clear>  
      </clear>  
    </securityTokenHandlers>  
  </identityConfiguration>  
</system.identityModel>  
```
