# <a name="deploying-net-framework-applications-with-docker"></a>Docker를 통해 .NET Framework 애플리케이션 배포

Windows 컨테이너를 사용하여 Docker를 통해 .NET Framework 애플리케이션을 배포할 수 있습니다. [Windows 컨테이너](/virtualization/windowscontainers/about/) 사용의 요구 사항 및 [Windows용 Docker 시작](https://docs.docker.com/docker-for-windows/) 방법을 알아볼 수 있습니다. 

먼저 [Docker로 콘솔 애플리케이션을 실행](console.md)할 수 있습니다.
Docker에서 웹 애플리케이션을 실행하기 위해 [ASP.NET MVC applications in Docker](/aspnet/mvc/overview/deployment/docker-aspnetmvc)(Docker의 ASP.NET MVC 애플리케이션) 내용을 읽어볼 수 있습니다.
