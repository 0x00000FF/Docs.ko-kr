     <span data-ttu-id="f35b6-123">표시 유형을 같은 어셈블리로 제한해야 하는 형식이 포함된 모듈을 그룹화합니다.</span><span class="sxs-lookup"><span data-stu-id="f35b6-123">Group modules containing types whose visibility should be restricted to the same assembly.</span></span>
