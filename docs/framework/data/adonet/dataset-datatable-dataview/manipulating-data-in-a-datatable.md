# DataTable에서 데이터 조작
<xref:System.Data.DataSet>에서 <xref:System.Data.DataTable>을 만든 후에는 데이터베이스에서 테이블을 사용할 때와 동일하게 작업을 수행할 수 있습니다.  테이블에서 데이터를 추가, 확인, 편집 및 삭제하고, 오류 및 이벤트를 모니터링하며, 테이블에서 데이터를 쿼리할 수 있습니다.  **DataTable**에서 데이터를 수정하는 경우에는 수정된 내용이 정확한지 확인하고, 수정된 내용을 프로그래밍 방식으로 승인하거나 거부할지를 결정할 수도 있습니다.
