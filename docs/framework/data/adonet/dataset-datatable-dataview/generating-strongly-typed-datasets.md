 <span data-ttu-id="44dde-104">(데이터베이스 테이블에서 xsd를 만들려면 <xref:System.Data.DataSet.WriteXmlSchema%2A> 또는 [Visual Studio에서 데이터 집합 작업](http://msdn.microsoft.com/library/8bw9ksd6.aspx)).</span><span class="sxs-lookup"><span data-stu-id="44dde-104">(To create an xsd from database tables, see <xref:System.Data.DataSet.WriteXmlSchema%2A> or [Working with Datasets in Visual Studio](http://msdn.microsoft.com/library/8bw9ksd6.aspx)).</span></span>  
  
 <span data-ttu-id="44dde-105">다음 코드를 생성 하기 위한 구문을 보여 줍니다.는 **DataSet** 이 도구를 사용 합니다.</span><span class="sxs-lookup"><span data-stu-id="44dde-105">The following code shows the syntax for generating a **DataSet** using this tool.</span></span>

**DataSet.**</span></span> <span data-ttu-id="44dde-108">이 옵션은 `/d` 옵션도 함께 지정한 경우에 사용합니다.</span><span class="sxs-lookup"><span data-stu-id="44dde-108">This option is used when the `/d` option is also data-stu-id="44dde-109">For more information, see [Querying Typed DataSets](../../../../../docs/framework/data/adonet/querying-typed-datasets.md).</span></span> <span data-ttu-id="44dde-110">선택적 `/n:` 지시문 알려도 대 한 네임 스페이스를 생성 하는 도구는 **데이터 집합** 호출 **XSDSchema.Namespace**합니다.</span><span class="sxs-lookup"><span data-stu-id="44dde-110">The optional `/n:` directive tells the tool to also generate a ```csharp  
using XSDSchema.Namespace;

 <span data-ttu-id="44dde-117">다음 코드 예제에서는 형식화 된 **DataSet** 라는 **CustomerDataSet** 의 고객 목록을 로드 하는 **Northwind** 데이터베이스입니다.</span><span class="sxs-lookup"><span data-stu-id="44dde-117">The following code example uses a typed **DataSet** named **CustomerDataSet** to load a list of customers from the **Northwind** database.</span></span> <span data-ttu-id="44dde-118">데이터를 사용 하 여 로드 되 면는 **채우기** 메서드, 반복 실행에 있는 각 고객의 **고객** 형식화 된를 사용 하 여 **CustomersRow** ( **DataRow**) 개체입니다.</span><span class="sxs-lookup"><span data-stu-id="44dde-118">Once the data is loaded using the **Fill** method, the example loops through each customer in the **Customers** table using the typed **CustomersRow** (**DataRow**) object.</span></span> <span data-ttu-id="44dde-119">이 제공에 대 한 직접 액세스는 **CustomerID** 열을 통하지는 **DataColumnCollection**합니다.</span><span class="sxs-lookup"><span data-stu-id="44dde-119">This provides direct access to the **CustomerID** column, as opposed to through the **DataColumnCollection**.</span></span>  
  
```vb  
Dim customers As CustomerDataSet= New CustomerDataSet()  
Dim adapter As SqlDataAdapter New SqlDataAdapter( _  
  "SELECT * FROM dbo.Customers;", _  
  "Data Source=(local);Integrated " & _  
  "Security=SSPI;Initial Catalog=Northwind")  
  
adapter.Fill(customers, "Customers")  
  
Dim customerRow As CustomerDataSet.CustomersRow  
For Each customerRow In customers.Customers  
  Console.WriteLine(customerRow.CustomerID)  
Next  
```  
  
```csharp  
CustomerDataSet customers = new CustomerDataSet();  
SqlDataAdapter adapter = new SqlDataAdapter(  
  "SELECT * FROM dbo.Customers;",  
  "Data Source=(local);Integrated " +  
  "Security=SSPI;Initial Catalog=Northwind");  
  
adapter.Fill(customers, "Customers");  
  
foreach(CustomerDataSet.CustomersRow customerRow in customers.Customers)  
  Console.WriteLine(customerRow.CustomerID);  
```
