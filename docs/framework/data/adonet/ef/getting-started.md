# <a name="getting-started-entity-framework"></a>시작(Entity Framework)
ADO.NET [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] 는 데이터 중심 응용 프로그램 및 서비스를 지원 하 고, 논리 관계형 수준에서 개념 수준으로 추상화 수준을 발생 시키는 데이터를 프로그래밍 하기 위한 플랫폼을 제공 합니다. 개발자가 좀 더 높은 추상화 수준에서 데이터 작업을 할 수 있게 됨으로써, [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)]에서는 특정 데이터 스토리지 엔진이나 관계형 스키마에 대해 독립적인 코드가 지원됩니다. 자세한 내용은 [Entity Framework 개요](overview.md)를 참조 하세요.  
  
 최신 버전의 Entity Framework 사용을 신속 하 게 시작 하려면 [시작](https://go.microsoft.com/fwlink/?LinkId=235280) 을 참조 하세요.
