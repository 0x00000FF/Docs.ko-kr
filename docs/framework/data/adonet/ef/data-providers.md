# Entity Framework 데이터 공급자
이 단원에서는 [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)]를 지원하는 데이터 공급자에 대한 정보를 제공합니다.  
  
## 단원 내용  
 [Entity Framework용 EntityClient 공급자](../../../../../docs/framework/data/adonet/ef/entityclient-provider-for-the-entity-framework.md)  
 EntityClient 데이터 공급자에 대해 설명합니다.  이 공급자는 데이터 모델에 대한 쿼리를 정식 명령 트리로 변환합니다.  그런 다음 [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)]용 [!INCLUDE[dnprdnshort](../../../../../includes/dnprdnshort-md.md)] 데이터 공급자에서 이 명령 트리를 사용할 수 있습니다.  
  
 [Entity Framework용 SqlClient](../../../../../docs/framework/data/adonet/ef/sqlclient-for-the-entity-framework.md)  
 [!INCLUDE[ssNoVersion](../../../../../includes/ssnoversion-md.md)] 데이터베이스에 사용할 [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)]를 지원하는 [!INCLUDE[dnprdnshort](../../../../../includes/dnprdnshort-md.md)] 데이터 공급자에 대해 설명합니다.  
  
## 관련 단원  
 [Entity Framework\(SQL Server Compact\)](http://go.microsoft.com/fwlink/?LinkId=135638)\(영문\)  
 공급자 제한 사항과 SQL Server Compact 데이터베이스에 [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)]를 사용하는 방법에 대해 설명합니다.  
  
 [Entity Framework용 타사 공급자](http://go.microsoft.com/fwlink/?LinkId=143699)\(영문\)  
 현재 Entity Framework에 사용할 수 있는 타사 데이터 공급자 목록입니다.
