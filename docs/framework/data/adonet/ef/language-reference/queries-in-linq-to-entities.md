 <span data-ttu-id="cfbc9-108">LINQ 쿼리 작업은 데이터 소스 가져오기, 쿼리 만들기 및 쿼리 실행으로 구성됩니다.</span><span class="sxs-lookup"><span data-stu-id="cfbc9-108">A LINQ query operation consists of three actions: obtain the data source or sources, create the query, and execute the query.</span></span>
