# <a name="-not-entity-sql"></a>! (NOT)(Entity SQL)
`Boolean` 식을 부정합니다.  
  
## <a name="syntax"></a>구문  
  
```  
NOT boolean_expression  
or  
! boolean_expression  
```  
  
## <a name="arguments"></a>인수  
 `boolean_expression`  
 부울을 반환하는 모든 유효한 식입니다.  
  
## <a name="remarks"></a>설명  
 느낌표(!)는 NOT 연산자와 같은 기능을 합니다.
