# Entity Framework 함수용 SqlClient
Entity Framework용 .NET Framework Data Provider for SQL Server\(SqlClient\)에서는 수치 및 집계 계산을 수행할 함수 집합과 `System.DateTime` 및 `string` 연산을 수행할 함수를 제공합니다.  이러한 함수는 `SQLServer` 네임스페이스에 있습니다.  
  
 모든 공급자에서 작동하는 함수 목록은 [정식 함수](../../../../../docs/framework/data/adonet/ef/language-reference/canonical-functions.md)를 참조하세요.  
  
 정식 함수가 SQL Server 함수에 매핑되는 방법에 대한 자세한 내용은 [개념적 모델 정식에서 SQL Server 함수로의 매핑](../../../../../docs/framework/data/adonet/ef/conceptual-model-canonical-to-sql-server-functions-mapping.md)을 참조하세요.
