# 참조
이 단원에서는 [!INCLUDE[vbtecdlinq](../../../../../../includes/vbtecdlinq-md.md)] 개발자를 위한 참조 정보를 제공합니다.  
  
 또한 MSDN Library에서 특정 문제를 검색할 수 있으며, 특히 전문가와 함께 보다 복잡한 항목을 자세히 논의할 수 있는 [LINQ 포럼](http://go.microsoft.com/fwlink/?LinkId=76488)\(영문\)에 참여할 수 있습니다.  또한 Visual Basic 및 C\# 코드 예제를 통해 [!INCLUDE[vbtecdlinq](../../../../../../includes/vbtecdlinq-md.md)] 기술에 대해 자세히 설명하는 백서를 학습할 수 있습니다.  자세한 내용은 [LINQ to SQL: 관계형 데이터에 대한 .NET Language\-Integrated Query](http://go.microsoft.com/fwlink/?LinkId=93205)\(영문\)를 참조하세요.
