# 샘플 데이터베이스 다운로드
[!INCLUDE[vbtecdlinq](../../../../../../includes/vbtecdlinq-md.md)] 설명서의 여러 가지 예제와 연습에서는 Northwind 샘플 데이터베이스와 SQL Server Express Edition을 사용합니다.  Microsoft 다운로드 사이트에서 해당 제품을 무료로 다운로드할 수 있습니다.  
  
## Northwind 데이터베이스 다운로드  
  
#### SQL Server용 Northwind 샘플 데이터베이스를 다운로드하여 설치하려면  
  
1.  Internet Explorer를 시작합니다.  
  
2.  [Northwind 및 Pubs 샘플 데이터베이스](http://go.microsoft.com/fwlink?linkid=64296) 웹 사이트로 이동합니다.  
  
3.  **다운로드**를 클릭합니다.  
  
4.  **파일 다운로드** 대화 상자에서 **저장**을 선택합니다.  
  
5.  파일을 다운로드한 후에는 **Nwind.exe** 파일을 두 번 클릭하여 데이터베이스를 설치합니다.  
  
     기본적으로 데이터베이스는 *drive*:\\SQL Server 2000 Sample Databases에 설치됩니다.  
  
## SQL Server Express Edition 다운로드  
 SQL Server Express Edition은 무료로 제공되며 응용 프로그램과 함께 재배포할 수 있습니다.  [!INCLUDE[vsprvs](../../../../../../includes/vsprvs-md.md)]를 사용하는 경우에 SQL Server Express Edition은 Pro and greater editions에 포함되어 있습니다.  
  
#### SQL Server Express Edition을 다운로드하여 설치하려면  
  
1.  Internet Explorer를 시작합니다.  
  
2.  [Microsoft 다운로드 센터](http://go.microsoft.com/fwlink?linkid=74602) 웹 사이트로 이동합니다.  
  
3.  키워드 상자에 SQL Server Express를 입력합니다.  
  
4.  **찾기**를 클릭합니다.  
  
5.  결과 페이지에서 **Microsoft SQL Server 2005 Express Edition** 다운로드 페이지로 연결되는 링크를 클릭합니다.  
  
6.  웹 사이트의 설치 지침을 따릅니다.  
  
## Management Studio Express 다운로드  
 다운로드한 데이터베이스를 수정해야 하는 경우 [!INCLUDE[vs_current_short](../../../../../../includes/vs-current-short-md.md)] IDE\(통합 개발 환경\)의 **서버 탐색기**에서 데이터에 액세스하거나 SSMSE\(Microsoft SQL Server Management Studio Express\)를 사용합니다.  
  
#### Management Studio Express 다운로드하려면  
  
-   [SSMSE 사이트](http://go.microsoft.com/fwlink/?LinkId=95933)의 지침에 따르세요.
