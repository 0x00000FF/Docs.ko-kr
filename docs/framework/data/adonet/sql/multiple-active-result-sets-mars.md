# MARS(Multiple Active Result Sets)
MARS\(Multiple Active Result Sets\)는 단일 연결에서 여러 배치를 실행할 수 있도록 하는 기능입니다.  이전 버전에서는 단일 연결에 대해 한 번에 하나의 배치만 실행할 수 있었습니다.  MARS로 여러 배치를 실행한다고 해서 여러 작업을 동시에 실행할 수 있는 것은 아닙니다.
