```csharp  
workAdapter.TableMappings.Add("AuthorsMapping", "Authors");
```  

 <span data-ttu-id="19988-107">A **DataTableMapping** 의 열 이름을 사용할 수 있습니다는 **DataTable** 데이터베이스의 다른 합니다.</span><span class="sxs-lookup"><span data-stu-id="19988-107">A **DataTableMapping** enables you to use column names in a **DataTable** that are different from those in the database.</span></span> <span data-ttu-id="19988-108">**DataAdapter** 는 테이블이 업데이트 될 때 열을 일치 하는 매핑을 사용 합니다.</span><span class="sxs-lookup"><span data-stu-id="19988-108">The **DataAdapter** uses the mapping to match the columns when the table is updated.</span></span>  
  
 <span data-ttu-id="19988-109">지정 하지 않는 경우는 **TableName** 또는 **DataTableMapping** 이름을 호출 하는 경우는 **채우기** 또는 **업데이트** 의 메서드는  **DataAdapter**, **DataAdapter** 찾습니다는 **DataTableMapping** "Table" 이라는 합니다.</span><span class="sxs-lookup"><span data-stu-id="19988-109">If you do not specify a **TableName** or a **DataTableMapping** name when calling the **Fill** or **Update** method of the **DataAdapter**, the **DataAdapter** looks for a **DataTableMapping** named "Table".</span></span> <span data-ttu-id="19988-110">해당 경우 **DataTableMapping** 존재 하지 않는 **TableName** 의 **DataTable** 은 "Table"입니다.</span><span class="sxs-lookup"><span data-stu-id="19988-110">If that **DataTableMapping** does not exist, the **TableName** of the **DataTable** is "Table".</span></span> <span data-ttu-id="19988-111">기본값을 지정할 수 **DataTableMapping** 만들어는 **DataTableMapping** "Table"의 이름으로 합니다.</span><span class="sxs-lookup"><span data-stu-id="19988-111">You can specify a default **DataTableMapping** by creating a **DataTableMapping** with the name of "Table".</span></span>


 <span data-ttu-id="19988-117">때는 **채우기** 메서드의 인스턴스를 전달 되는 **데이터 집합** 및 **DataTableMapping** 이름, 해당 이름의 매핑이 있으면이 고, 그렇지 않으면 사용 되는 한  **DataTable** 해당 이름이 사용 됩니다.</span><span class="sxs-lookup"><span data-stu-id="19988-117">When the **Fill** method is passed an instance of a **DataSet** and a **DataTableMapping** name, if a mapping with that name exists it is used; otherwise, a **DataTable** with that name is used.</span></span>  
  
 <span data-ttu-id="19988-118">다음 예제에서는 한 **DataTableMapping** 의 이름으로 **고객** 및 **DataTable** 이름 **BizTalkSchema**합니다.</span><span class="sxs-lookup"><span data-stu-id="19988-118">The following examples create a **DataTableMapping** with a name of **Customers** and a **DataTable** name of **BizTalkSchema**.</span></span> <span data-ttu-id="19988-119">예제에서는 다음를 SELECT 문에서 반환한 행을 매핑하는 **BizTalkSchema** **DataTable**합니다.</span><span class="sxs-lookup"><span data-stu-id="19988-119">The example then maps the rows returned by the SELECT statement to the **BizTalkSchema** **DataTable**.</span></span>

 <span data-ttu-id="19988-126">경우에 **SelectCommand** 여러 테이블을 반환 **채우기** 테이블에 대 한 증분 값을 갖는 테이블 이름을 자동으로 생성는 **데이터 집합**부터 테이블 이름 사이에서 형식 지정 **TableName** *N*부터 **TableName1**합니다.</span><span class="sxs-lookup"><span data-stu-id="19988-126">If your **SelectCommand** returns multiple tables, **Fill** automatically generates table names with incremental values for the tables in the **DataSet**, starting with the specified table name and continuing on in the form **TableName** *N*, starting with **TableName1**.</span></span> <span data-ttu-id="19988-127">테이블 매핑을 사용 하 여 원하는 테이블에 대해 지정 된 이름에 자동으로 생성 된 테이블 이름을 매핑하는 **DataSet**합니다.</span><span class="sxs-lookup"><span data-stu-id="19988-127">You can use table mappings to map the automatically generated table name to a name you want specified for the table in the **DataSet**.</span></span> <span data-ttu-id="19988-128">예는 **SelectCommand** 두 테이블을 반환 하는 **고객** 및 **Orders**, 다음 호출을 실행 **채우기**합니다.</span><span class="sxs-lookup"><span data-stu-id="19988-128">For example, for a **SelectCommand** that returns two tables, **Customers** and **Orders**, issue the following call to **Fill**.</span></span>
