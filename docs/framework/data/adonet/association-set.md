<span data-ttu-id="a907a-103">*연결 집합* 는 대 한 논리적 컨테이너 [연결](../../../../docs/framework/data/adonet/association-type.md) 동일한 형식의 인스턴스.</span><span class="sxs-lookup"><span data-stu-id="a907a-103">An *association set* is a logical container for [association](../../../../docs/framework/data/adonet/association-type.md) instances of the same type.</span></span> <span data-ttu-id="a907a-104">연결 집합은 데이터 모델링 구문이 아니므로 데이터 또는 관계의 구조를 설명하지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="a907a-104">An association set is not a data modeling construct; that is, it does not describe the structure of data or relationships.</span></span> <span data-ttu-id="a907a-105">대신 연결 집합은 연결 인스턴스를 그룹화하여 데이터 저장소에 매핑할 수 있도록 호스팅 또는 저장소 환경(예: 공용 언어 런타임 또는 SQL Server 데이터베이스)에 대한 구문을 제공합니다.</span><span class="sxs-lookup"><span data-stu-id="a907a-105">Instead, an association set provides a construct for a hosting or storage environment (such as the common language runtime or a SQL Server database) to group association instances so that they can be mapped to a data store.</span></span>  
  
 <span data-ttu-id="a907a-106">연결 집합 내에 정의 되어 있는 [엔터티 컨테이너](../../../../docs/framework/data/adonet/entity-container.md)의 논리적 그룹인 [엔터티 집합](../../../../docs/framework/data/adonet/entity-set.md) 및 연결 집합입니다.</span><span class="sxs-lookup"><span data-stu-id="a907a-106">An association set is defined within an [entity container](../../../../docs/framework/data/adonet/entity-container.md), which is a logical grouping of [entity sets](../../../../docs/framework/data/adonet/entity-set.md) and association sets.</span></span>  
  
 <span data-ttu-id="a907a-107">연결 집합 정의에는 다음 정보가 들어 있습니다.</span><span class="sxs-lookup"><span data-stu-id="a907a-107">A definition for an association set contains the following information:</span></span>  
  
-   <span data-ttu-id="a907a-108">연결 집합 이름</span><span class="sxs-lookup"><span data-stu-id="a907a-108">The association set name.</span></span> <span data-ttu-id="a907a-109">(필수)</span><span class="sxs-lookup"><span data-stu-id="a907a-109">(Required)</span></span>  
  
-   <span data-ttu-id="a907a-110">연결 집합에 인스턴스가 포함될 연결</span><span class="sxs-lookup"><span data-stu-id="a907a-110">The association of which it will contain instances.</span></span> <span data-ttu-id="a907a-111">(필수)</span><span class="sxs-lookup"><span data-stu-id="a907a-111">(Required)</span></span>  
  
-   <span data-ttu-id="a907a-112">두 개의 [연결 집합 end](../../../../docs/framework/data/adonet/association-set-end.md)합니다.</span><span class="sxs-lookup"><span data-stu-id="a907a-112">Two [association set ends](../../../../docs/framework/data/adonet/association-set-end.md).</span></span>
