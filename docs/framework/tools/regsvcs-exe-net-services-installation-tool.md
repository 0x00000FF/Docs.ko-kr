## <a name="examples"></a><span data-ttu-id="3ac2a-166">예제</span><span class="sxs-lookup"><span data-stu-id="3ac2a-166">Examples</span></span>  
 <span data-ttu-id="3ac2a-167">다음 명령을 사용하여 `myTest.dll`에 들어 있는 모든 공용 클래스를 `myTargetApp`(기존의 COM+ 응용 프로그램)에 추가하고 `myTest.tlb` 형식 라이브러리를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="3ac2a-167">The following command adds all public classes contained in `myTest.dll` to `myTargetApp` (an existing COM+ application) and produces the `myTest.tlb` type library.</span></span>

