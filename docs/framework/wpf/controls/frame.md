# 프레임
<xref:System.Windows.Controls.Frame> 컨트롤은 콘텐츠 내에서의 콘텐츠 탐색을 지원합니다.  <xref:System.Windows.Controls.Frame>은 <xref:System.Windows.Window>, <xref:System.Windows.Navigation.NavigationWindow>, <xref:System.Windows.Controls.Page>, <xref:System.Windows.Controls.UserControl>, <xref:System.Windows.Documents.FlowDocument> 등과 같은 루트 요소에 의해 호스팅되거나 루트 요소에 속하는 콘텐츠 트리 내에서 아일랜드로 호스팅될 수 있습니다.
