# 스타일 및 템플릿
WPF\(Windows Presentation Foundation\) 스타일 및 템플릿 지정은 응용 프로그램, 문서 또는 UI\(사용자 인터페이스\) 디자이너가 보기 좋은 응용 프로그램을 만들고 제품 특유의 모양을 표준화하는 데 사용할 수 있는 기능 집합\(스타일, 템플릿, 트리거 및 스토리보드\)과 관련됩니다.
