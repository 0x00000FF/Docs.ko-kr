# Expander 스타일 및 템플릿
이 항목에서는 <xref:System.Windows.Controls.Expander> 컨트롤의 스타일 및 템플릿에 대해 설명합니다.  기본 <xref:System.Windows.Controls.ControlTemplate>을 수정하여 컨트롤에 고유한 모양을 지정할 수 있습니다.  자세한 내용은 [ControlTemplate을 만들어 기존 컨트롤의 모양 사용자 지정](../../../../docs/framework/wpf/controls/customizing-the-appearance-of-an-existing-control.md)을 참조하십시오.  
  
## Expander 요소  
 <xref:System.Windows.Controls.Expander> 컨트롤에는 명명된 요소가 없습니다.  
  
## Expander 상태  
 다음 표에서는 <xref:System.Windows.Controls.Expander> 컨트롤의 시각적 상태를 보여 줍니다.  
  
||||  
|-|-|-|  
|VisualState 이름|VisualStateGroup 이름|설명|  
|보통|CommonStates|기본 상태입니다.|  
|MouseOver|CommonStates|마우스 포인터가 컨트롤 위에 있습니다.|  
|Disabled|CommonStates|컨트롤이 사용하지 않도록 설정되어 있습니다.|  
|Focused|FocusStates|컨트롤에 포커스가 있습니다.|  
|Unfocused|FocusStates|컨트롤에 포커스가 없습니다.|  
|전체 확장|ExpansionStates|컨트롤이 확장된 상태입니다.|  
|Collapsed|ExpansionStates|컨트롤이 확장되지 않은 상태입니다.|  
|ExpandDown|ExpandDirectionStates|컨트롤이 아래로 확장됩니다.|  
|ExpandUp|ExpandDirectionStates|컨트롤이 위로 확장됩니다.|  
|ExpandLeft|ExpandDirectionStates|컨트롤이 왼쪽으로 확장됩니다.|  
|ExpandRight|ExpandDirectionStates|컨트롤이 오른쪽으로 확장됩니다.|  
|Valid|ValidationStates|이 컨트롤은 <xref:System.Windows.Controls.Validation> 클래스를 사용하며 연결된 속성 <xref:System.Windows.Controls.Validation.HasError%2A?displayProperty=fullName>는 `false`입니다.|  
|InvalidFocused|ValidationStates|컨트롤에 포커스가 있는 경우 연결된 속성 <xref:System.Windows.Controls.Validation.HasError%2A?displayProperty=fullName>는 `true`입니다.|  
|InvalidUnfocused|ValidationStates|컨트롤에 포커스가 없는 경우 연결된 속성 <xref:System.Windows.Controls.Validation.HasError%2A?displayProperty=fullName>는 `true`입니다.|
