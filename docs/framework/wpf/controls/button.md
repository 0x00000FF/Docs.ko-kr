# 단추
<xref:System.Windows.Controls.Button> 컨트롤은 마우스, 키보드, 스타일러스 또는 기타 입력 장치의 사용자 입력에 반응하여 <xref:System.Windows.Controls.Primitives.ButtonBase.Click> 이벤트를 발생시킵니다.  <xref:System.Windows.Controls.Button>은 텍스트와 같은 간단한 콘텐츠를 포함할 수 있는 기본 [!INCLUDE[TLA#tla_ui](../../../../includes/tlasharptla-ui-md.md)] 구성 요소이며 이미지 및 <xref:System.Windows.Controls.Panel> 컨트롤과 같은 복잡한 콘텐츠도 포함할 수 있습니다.
