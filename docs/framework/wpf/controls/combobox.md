# ComboBox
<xref:System.Windows.Controls.ComboBox> 컨트롤은 사용자에게 옵션 목록을 표시합니다.  이 목록은 컨트롤을 확장하면 표시되고 축소하면 숨겨집니다.  축소되어 한 가지 옵션만 표시되는 상태가 기본 상태입니다.  사용자는 단추를 클릭하여 전체 옵션 목록을 볼 수 있습니다.
