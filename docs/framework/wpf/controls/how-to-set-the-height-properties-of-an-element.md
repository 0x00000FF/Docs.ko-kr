# <a name="how-to-set-the-height-properties-of-an-element"></a>방법: 요소의 높이 속성 설정

 이 예제에서는 시각적으로 렌더링에 4 개의 높이 관련 속성 중에서 동작의 차이점을 설명 [!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)]합니다.
