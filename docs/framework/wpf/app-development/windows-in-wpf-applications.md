# WPF 응용 프로그램의 창
사용자는 창을 통해 응용 프로그램과 상호 작용합니다.  창의 기본 용도는 콘텐츠를 호스팅하고 표시하는 것입니다.  창에서 호스팅하는 콘텐츠의 형식은 응용 프로그램에서 사용하는 데이터 형식에 따라 달라집니다. 여기에는 미디어, XAML\(Extensible Application Markup Language\) 페이지, 웹 페이지, 문서, 데이터베이스 테이블과 레코드 및 시스템 정보가 포함될 수 있습니다.
