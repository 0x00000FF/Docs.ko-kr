# WPF 응용 프로그램 호스팅
WPF XBAP\(XAML 브라우저 응용 프로그램\)는 웹 서버에 배포하고 브라우저에서 시작할 수 있는 다양한 기능의 클라이언트 응용 프로그램입니다.  WPF 호스트\(PresentationHost.exe\)는 XBAP 파일과 XAML 파일의 셸 및 MIME 처리기로 등록됩니다.  따라서 Internet Explorer에서는 XBAP가 시작되면 이를 인식하고 WPF 호스트를 시작합니다.  Firefox 사용자는 Firefox에서 XBAP를 정상적으로 호스팅하는 데 필요한 Firefox 추가 기능을 설치할 수 있습니다.  WPF에서 제공하는 네이티브 브라우저 호스팅 API를 사용하여 다른 브라우저나 독립 실행형 응용 프로그램에서 XBAP를 호스팅할 수 있습니다.
