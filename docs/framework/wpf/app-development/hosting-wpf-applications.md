# <a name="hosting-wpf-applications"></a>WPF 애플리케이션 호스팅
WPF Xbap (XAML 브라우저 응용 프로그램)는 웹 서버에 배포 하 고 브라우저에서 시작할 수 있는 리치 클라이언트 응용 프로그램입니다. WPF 호스트 (Presentationhost.exe)는 XBAP 및 XAML 파일에 대 한 셸 및 MIME 처리기로 등록 됩니다. 따라서 Internet Explorer는 XBAP를 시작할 때 WPF 호스트를 시작 하는 것을 알고 있습니다. Firefox 사용자는 firefox를 사용 하 여 Xbap를 호스트할 수 있는 Firefox 추가 기능을 설치할 수 있습니다. XBAP는 WPF에서 제공 하는 기본 브라우저 호스팅 Api를 사용 하 여 다른 브라우저나 독립 실행형 응용 프로그램에서 호스팅될 수 있습니다.
