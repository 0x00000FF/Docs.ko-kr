# WPF 응용 프로그램 빌드 및 배포
빌드 및 배포 모델에서는 다음을 비롯하여 로컬 및 원격으로 응용 프로그램을 빌드하고 배포하는 기능을 제공합니다.  
  
-   MSBuild: Microsoft.Build.Tasks.Windows 네임스페이스에 있는 .NET 빌드 시스템  
  
-   리소스: UI 리소스 작업
