## <a name="example"></a><span data-ttu-id="98d81-104">예제</span><span class="sxs-lookup"><span data-stu-id="98d81-104">Example</span></span>  
 <span data-ttu-id="98d81-105">다음 예제에서는 <xref:System.Windows.Media.Animation.ColorAnimationUsingKeyFrames> 애니메이션 효과를 줄 클래스는 <xref:System.Windows.Media.SolidColorBrush.Color%2A> 속성의는 <xref:System.Windows.Media.SolidColorBrush>합니다.</span><span class="sxs-lookup"><span data-stu-id="98d81-105">The following example uses the <xref:System.Windows.Media.Animation.ColorAnimationUsingKeyFrames> class to animate the <xref:System.Windows.Media.SolidColorBrush.Color%2A> property of a <xref:System.Windows.Media.SolidColorBrush>.</span></span> <span data-ttu-id="98d81-106">이 애니메이션은 다음과 같은 방식으로 세 가지 키 프레임을 사용합니다.</span><span class="sxs-lookup"><span data-stu-id="98d81-106">This animation uses three key frames in the following manner:</span></span>  
  
1.  <span data-ttu-id="98d81-107">인스턴스를 사용 하 여 첫 번째 2 초 동안는 <xref:System.Windows.Media.Animation.LinearColorKeyFrame> 점진적으로 녹색에서 빨간색으로 색을 변경 하는 클래스입니다.</span><span class="sxs-lookup"><span data-stu-id="98d81-107">During the first two seconds, uses an instance of the <xref:System.Windows.Media.Animation.LinearColorKeyFrame> class to gradually change the color from green to red.</span></span> <span data-ttu-id="98d81-108">같은 키 프레임 선형 <xref:System.Windows.Media.Animation.LinearColorKeyFrame> 값 사이의 부드러운 선형 전환을 만듭니다.</span><span class="sxs-lookup"><span data-stu-id="98d81-108">Linear key frames like <xref:System.Windows.Media.Animation.LinearColorKeyFrame> create a smooth linear transition between values.</span></span>  
  
2.  <span data-ttu-id="98d81-109">다음의 끝 중 0.5 초에서 사용 하는 <xref:System.Windows.Media.Animation.DiscreteColorKeyFrame> 클래스를 신속 하 게 노란색으로 빨강에서 색을 변경 합니다.</span><span class="sxs-lookup"><span data-stu-id="98d81-109">During the end of the next half second, uses an instance of the <xref:System.Windows.Media.Animation.DiscreteColorKeyFrame> class to quickly change the color from red to yellow.</span></span> <span data-ttu-id="98d81-110">과 같은 불연속 키 프레임 <xref:System.Windows.Media.Animation.DiscreteColorKeyFrame> 급격 한 변경과 값 간에 즉, 만들고 애니메이션의이 부분의 색 변경 신속 하 게 발생 효과입니다.</span><span class="sxs-lookup"><span data-stu-id="98d81-110">Discrete key frames like <xref:System.Windows.Media.Animation.DiscreteColorKeyFrame> create sudden changes between values, that is, the color change in this part of the animation occurs quickly and is not subtle.</span></span>  
  
3.  <span data-ttu-id="98d81-111">마지막 2 초 동안의 인스턴스를 사용 하 여는 <xref:System.Windows.Media.Animation.SplineColorKeyFrame> 다시 색을 변경 하려면 클래스-녹색에서 노랑 다시이 이번 합니다.</span><span class="sxs-lookup"><span data-stu-id="98d81-111">During the final two seconds, uses an instance of the <xref:System.Windows.Media.Animation.SplineColorKeyFrame> class to change the color again—this time from yellow back to green.</span></span> <span data-ttu-id="98d81-112">같은 스플라인 키 프레임 <xref:System.Windows.Media.Animation.SplineColorKeyFrame> 변수 값의 값을 따르는 사이의 전환을 만들려면는 <xref:System.Windows.Media.Animation.SplineColorKeyFrame.KeySpline%2A> 속성입니다.</span><span class="sxs-lookup"><span data-stu-id="98d81-112">Spline key frames like <xref:System.Windows.Media.Animation.SplineColorKeyFrame> create a variable transition between values according to the values of the <xref:System.Windows.Media.Animation.SplineColorKeyFrame.KeySpline%2A> property.</span></span> <span data-ttu-id="98d81-113">이 예제에서 색 변화는 느리게 시작하다가 시간 세그먼트의 끝에 다가가면 기하급수적으로 빨라집니다.</span><span class="sxs-lookup"><span data-stu-id="98d81-113">In this example, the change in color begins slowly and speeds up exponentially toward the end of the time segment.</span></span>
