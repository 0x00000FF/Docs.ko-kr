# 시각적 계층 프로그래밍
<xref:System.Windows.Media.Visual> 개체는 핵심 [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] 개체이며 렌더링 지원을 제공하는 기본적인 역할을 수행합니다.  <xref:System.Windows.Controls.Button> 및 <xref:System.Windows.Controls.TextBox>와 같은 사용자 인터페이스 컨트롤은 <xref:System.Windows.Media.Visual> 클래스에서 파생되며 렌더링 데이터를 유지하는 데 사용됩니다.
