# 리소스(WPF)
리소스 응용 프로그램에서 서로 다른 위치에서 다시 사용할 수 있는 개체입니다.  WPF 다양 한 종류의 리소스를 지원합니다.  이 리소스는 주로 두 가지 종류의 리소스입니다: XAML 리소스와 리소스 데이터 파일입니다.  XAML 리소스 예제 브러쉬 및 스타일이 포함 됩니다.  리소스 데이터 파일은 응용 프로그램에 필요한 비실행 데이터 파일입니다.
