# 고급 잉크 처리
[!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)]에는 응용 프로그램에 추가하여 잉크를 즉시 수집하고 표시할 수 있는 요소인 <xref:System.Windows.Controls.InkCanvas>가 있습니다.  하지만 <xref:System.Windows.Controls.InkCanvas> 컨트롤의 제어 수준은 정밀하지 못하기 때문에 <xref:System.Windows.Input.StylusPlugIns>를 사용하여 고유한 잉크 컬렉션과 잉크 렌더링 클래스를 사용자 지정함으로써 더 높은 수준의 컨트롤 정밀도의 유지할 수 있습니다.  
  
 <xref:System.Windows.Input.StylusPlugIns> 클래스는 <xref:System.Windows.Input.Stylus> 입력에 대한 저수준 제어를 구현하고 잉크를 동적으로 렌더링하는 메커니즘을 제공합니다.  <xref:System.Windows.Input.StylusPlugIns.StylusPlugIn> 클래스에서는 사용자 지정 동작을 구현하고 최적의 성능을 위해 스타일러스 장치를 통해 입력되는 데이터 스트림에 해당 동작을 적용하는 메커니즘을 제공합니다.  특수한 <xref:System.Windows.Input.StylusPlugIns.StylusPlugIn>인 <xref:System.Windows.Input.StylusPlugIns.DynamicRenderer>를 사용하면 잉크 데이터의 실시간 동적 렌더링을 사용자 지정할 수 있습니다. 즉, <xref:System.Windows.Input.StylusPoint> 데이터가 생성되면 <xref:System.Windows.Input.StylusPlugIns.DynamicRenderer>에서 디지털 잉크를 즉시 그리므로 스타일러스 장치에서 디지털 잉크가 "흘러나오는" 것처럼 표시됩니다.
