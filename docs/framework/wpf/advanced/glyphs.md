# 문자 모양
문자 모양은 화면에 그릴 문자에 대한 하위 수준 표현입니다.  [!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)]는 사용자가 텍스트에 서식을 설정한 후 이를 가로채 유지할 수 있도록 문자 모양에 대한 직접 액세스를 지원합니다.
