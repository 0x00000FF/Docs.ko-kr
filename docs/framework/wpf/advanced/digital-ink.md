# 디지털 잉크
이 단원에서는 [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)]에서의 디지털 잉크 사용에 대해 설명합니다.  일반적으로 Tablet PC SDK에서만 제공되었던 디지털 잉크를 이제는 핵심 Windows Presentation Foundation에서 사용할 수 있습니다.  이제 강력한 Windows Presentation Foundation을 활용하여 완전한 기능을 갖춘 Tablet PC 응용 프로그램을 개발할 수 있게 된 것입니다.
