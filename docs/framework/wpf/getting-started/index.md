# 시작(WPF)
WPF\(Windows Presentation Foundation\)는 풍부한 대화형 클라이언트 응용 프로그램을 만드는 UI 프레임워크입니다.  WPF 개발 플랫폼은 응용 프로그램 모델, 리소스, 컨트롤, 그래픽, 레이아웃, 데이터 바인딩, 문서 및 보안을 포함하여 다양한 응용 프로그램 개발 기능 집합을 지원합니다.  .NET Framework의 하위 집합이므로 이전에 ASP.NET 또는 Windows Forms를 사용하여 .NET Framework로 응용 프로그램을 빌드한 경우 프로그래밍 환경이 비슷합니다.  WPF는 XAML\(Extensible Application Markup Language\)을 사용하여 응용 프로그램 프로그래밍을 위한 선언적 모델을 제공합니다.  이 섹션의 항목에서는 WPF를 소개하고 WPF를 시작하는 데 유용한 정보를 제공합니다.
