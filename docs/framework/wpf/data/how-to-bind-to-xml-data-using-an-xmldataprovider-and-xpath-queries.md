 <span data-ttu-id="3e6ce-104"><xref:System.Windows.Data.XmlDataProvider>를 사용 하면 응용 프로그램에서 데이터 바인딩을 통해 액세스할 수 있는 기본 데이터가 모든 XML 노드의 트리가 될 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="3e6ce-104">With an <xref:System.Windows.Data.XmlDataProvider>, the underlying data that can be accessed through data binding in your application can be any tree of XML nodes.</span></span> <span data-ttu-id="3e6ce-105">즉, <xref:System.Windows.Data.XmlDataProvider>은 모든 XML 노드 트리를 바인딩 소스로 사용 하는 편리한 방법을 제공 합니다.</span><span class="sxs-lookup"><span data-stu-id="3e6ce-105">In other words, an <xref:System.Windows.Data.XmlDataProvider> provides a convenient way to use any tree of XML nodes as a binding source.</span></span>
