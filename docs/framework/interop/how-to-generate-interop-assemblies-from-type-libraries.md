1. <span data-ttu-id="96547-108">다음 명령을 사용합니다.</span><span class="sxs-lookup"><span data-stu-id="96547-108">Use the following command:</span></span>  
  
     <span data-ttu-id="96547-109">**tlbimp** \<*type-library-file*></span><span class="sxs-lookup"><span data-stu-id="96547-109">**tlbimp** \<*type-library-file*></span></span>  
  
     <span data-ttu-id="96547-110">**/out:** 스위치를 추가하면 변경된 이름(예: LOANLib.dll)을 가진 interop 어셈블리가 생성됩니다.</span><span class="sxs-lookup"><span data-stu-id="96547-110">Adding the **/out:** switch produces an interop assembly with an altered name, such as LOANLib.dll.</span></span> <span data-ttu-id="96547-111">Interop 어셈블리 이름을 변경하면 원래 COM DLL과 구별하고 중복 이름으로 인해 발생할 수 있는 문제를 방지할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="96547-111">Altering the interop assembly name can help distinguish it from the original COM DLL and prevent problems that can occur from having duplicate names.</span></span>
