- <span data-ttu-id="5c4ff-104">컨테이너 기반 솔루션은 궁극적으로 비용 절감 혜택을 제공합니다.</span><span class="sxs-lookup"><span data-stu-id="5c4ff-104">Container-based solutions ultimately provide cost savings benefits.</span></span> <span data-ttu-id="5c4ff-105">컨테이너는 프로덕션 환경에서 종속성이 없는 경우 발생하는 충돌을 제거하기 때문에 배포 문제에 대한 솔루션입니다.</span><span class="sxs-lookup"><span data-stu-id="5c4ff-105">Containers are a solution to deployment problems because they remove the friction caused by an absence of dependencies in production environments.</span></span> <span data-ttu-id="5c4ff-106">이러한 문제를 제거하여 개발/테스트, DevOps 및 프로덕션 작업을 크게 향상시킬 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="5c4ff-106">By removing those issues, it improves Dev/Test, DevOps, and production operations significantly.</span></span>

- <span data-ttu-id="5c4ff-107">Docker 컨테이너는 서버 기반 애플리케이션 또는 서비스 배포의 표준 단위가 되고 있습니다.</span><span class="sxs-lookup"><span data-stu-id="5c4ff-107">A Docker container is becoming the standard unit of deployment for any server-based application or service.</span></span>

- <span data-ttu-id="5c4ff-109">컨테이너를 호스트하는 Azure VM은 클라우드에서 소규모 개발/테스트 환경을 만드는 빠르고 간단한 방법입니다.</span><span class="sxs-lookup"><span data-stu-id="5c4ff-109">Azure VMs hosting containers are a fast and simple way to create small Dev/Test environments in the cloud.</span></span>
