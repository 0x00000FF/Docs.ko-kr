# <a name="when-to-deploy-windows-containers-in-your-on-premises-iaas-vm-infrastructure"></a>온-프레미스 IaaS VM 인프라에서 Windows 컨테이너를 배포해야 하는 경우

- 조직에서 클라우드로 이동할 준비가 되지 않았거나 비즈니스 이유로 클라우드로 이동하지 못할 수 있습니다. 그러나 자체 데이터 센터에서 Windows 컨테이너를 사용하는 이점은 여전히 얻을 수 있습니다.

- 온-프레미스에서 사용되는 다른 아티팩트가 있을 수 있으며, 이로 인해 클라우드로 이동하는 과정이 느려질 수 있습니다. 예를 들어 온-프레미스 Windows Server Active Directory 또는 다른 온-프레미스 자산에 대한 보안 또는 인증 종속성이 있습니다.

- 이제부터 Windows 컨테이너를 사용하기 시작한다면 나중에 훨씬 더 유리한 입장에서 클라우드로 단계적 마이그레이션을 추진할 수 있습니다. Windows 컨테이너는 잠금 없이 모든 클라우드의 배포 단위가 되고 있습니다.
