<span data-ttu-id="e1b2e-120">다음 코드는 [Azure Table Storage](https://docs.microsoft.com/azure/cosmos-db/table-storage-overview) 인스턴스에 새 행을 삽입 하는 데 걸리는 시간을 측정 합니다.</span><span class="sxs-lookup"><span data-stu-id="e1b2e-120">The following code measures how long it takes to insert a new row into an [Azure Table Storage](https://docs.microsoft.com/azure/cosmos-db/table-storage-overview) instance:</span></span>

```csharp
var operation = TableOperation.Insert(entry);

```
