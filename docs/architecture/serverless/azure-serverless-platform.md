# <a name="azure-serverless-platform"></a>Azure 서버리스 플랫폼

Azure 서버리스 플랫폼에는 Azure Functions, Logic Apps 및 Event Grid가 포함됩니다. 이러한 서비스는 함께 작동하며 수많은 다른 리소스와 연결됩니다. 서버리스 플랫폼은 데이터베이스 및 저장소에서 분석 및 기계 학습/인공 지능에 이르기까지 모든 작업이 가능합니다.

진단 추적 및 원격 분석을 캡처하기 위한 서버리스 플랫폼인 Application Insights를 사용할 수도 있습니다. Application Insights는 서버리스 구현뿐만 아니라 모든 유형의 애플리케이션(데스크톱, 모바일 또는 웹)에서 사용할 수 있습니다. 다음 다이어그램에는 플랫폼이 시각화되어 있습니다.
