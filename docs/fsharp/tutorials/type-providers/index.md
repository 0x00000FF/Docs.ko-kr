# <a name="type-providers"></a><span data-ttu-id="e357f-103">형식 공급자</span><span class="sxs-lookup"><span data-stu-id="e357f-103">Type Providers</span></span>

<span data-ttu-id="e357f-104">F# 형식 공급자는 사용자 프로그램에서 사용할 형식, 속성 및 메서드를 제공하는 구성 요소입니다.</span><span class="sxs-lookup"><span data-stu-id="e357f-104">An F# type provider is a component that provides types, properties, and methods for use in your program.</span></span> <span data-ttu-id="e357f-105">형식 공급자 라는 항목을 생성 **제공 된 형식**, F # 컴파일러에서 생성 하 고 외부 데이터 원본을 기반으로 합니다.</span><span class="sxs-lookup"><span data-stu-id="e357f-105">Type Providers generate what are known as **Provided Types**, which are generated by the F# compiler and are based on an external data source.</span></span>

<span data-ttu-id="e357f-106">예를 들어, SQL 용 F # 형식 공급자는 관계형 데이터베이스의 테이블 및 열을 나타내는 형식을 생성할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="e357f-106">For example, an F# Type Provider for SQL can generate types representing tables and columns in a relational database.</span></span> <span data-ttu-id="e357f-107">실제로 [SQLProvider](https://fsprojects.github.io/SQLProvider/) 형식 공급자가 있습니다.</span><span class="sxs-lookup"><span data-stu-id="e357f-107">In fact, this is what the [SQLProvider](https://fsprojects.github.io/SQLProvider/) Type Provider does.</span></span>

<span data-ttu-id="e357f-108">제공 유형은 입력된 매개 변수를 형식 공급자에 따라 다릅니다.</span><span class="sxs-lookup"><span data-stu-id="e357f-108">Provided Types depend on input parameters to a Type Provider.</span></span> <span data-ttu-id="e357f-109">이러한 입력 (예: JSON 스키마 파일), 샘플 데이터 소스 수 직접 외부 서비스 또는 데이터 원본에 연결 문자열을 가리키는 URL입니다.</span><span class="sxs-lookup"><span data-stu-id="e357f-109">Such input can be a sample data source (such as a JSON schema file), a URL pointing directly to an external service, or a connection string to a data source.</span></span> <span data-ttu-id="e357f-110">형식 공급자는 형식의 그룹이 필요할 때;만 확장 되도록 할 수도 있습니다. 즉, 형식을 프로그램에서 참조 하는 실제로 경우 확장 됩니다.</span><span class="sxs-lookup"><span data-stu-id="e357f-110">A Type Provider can also ensure that groups of types are only expanded on demand; that is, they are expanded if the types are actually referenced by your program.</span></span> <span data-ttu-id="e357f-111">따라서 강력한 형식으로 온라인 데이터 시장과 같은 대규모 정보 공간의 직접적인 주문형 통합을 허용합니다.</span><span class="sxs-lookup"><span data-stu-id="e357f-111">This allows for the direct, on-demand integration of large-scale information spaces such as online data markets in a strongly typed way.</span></span>

## <a name="generative-and-erased-type-providers"></a><span data-ttu-id="e357f-112">생성 및 지워진 형식 공급자</span><span class="sxs-lookup"><span data-stu-id="e357f-112">Generative and Erased Type Providers</span></span>

<span data-ttu-id="e357f-113">형식 공급자는 두 가지 형태로 제공: 생성 및 삭제 합니다.</span><span class="sxs-lookup"><span data-stu-id="e357f-113">Type Providers come in two forms: Generative and Erased.</span></span>

<span data-ttu-id="e357f-114">생성 형식 공급자는.NET 형식으로 생성 된 어셈블리에 쓸 수 있는 형식을 생성 합니다.</span><span class="sxs-lookup"><span data-stu-id="e357f-114">Generative Type Providers produce types that can be written as .NET types into the assembly in which they are produced.</span></span> <span data-ttu-id="e357f-115">이렇게 하면 다른 어셈블리의 코드에서 사용할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="e357f-115">This allows them to be consumed from code in other assemblies.</span></span> <span data-ttu-id="e357f-116">즉, 데이터 원본의 입력 된 표현을 나타내는.NET 형식에 하나 이어야 일반적으로 합니다.</span><span class="sxs-lookup"><span data-stu-id="e357f-116">This means that the typed representation of the data source must generally be one that is feasible to represent with .NET types.</span></span>

<span data-ttu-id="e357f-117">지우기 형식 공급자는 어셈블리 또는 프로젝트에서 생성 된만 사용할 수 있는 형식을 생성 합니다.</span><span class="sxs-lookup"><span data-stu-id="e357f-117">Erasing Type Providers produce types that can only be consumed in the assembly or project they are generated from.</span></span> <span data-ttu-id="e357f-118">유형은 임시입니다. 즉, 어셈블리에 기록 되지 않고 하며 다른 어셈블리의 코드에서 사용 될 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="e357f-118">The types are ephemeral; that is, they are not written into an assembly and cannot be consumed by code in other assemblies.</span></span> <span data-ttu-id="e357f-119">포함 될 수 있습니다 *지연* 멤버, 잠재적으로 무한 정보 공간에서 제공 하는 사용 유형을 수 있게 해줍니다.</span><span class="sxs-lookup"><span data-stu-id="e357f-119">They can contain *delayed* members, allowing you to use provided types from a potentially infinite information space.</span></span> <span data-ttu-id="e357f-120">광범위 하 고 상호 연결 된 데이터 원본의 작은 하위 집합을 사용 하는 데 유용 합니다.</span><span class="sxs-lookup"><span data-stu-id="e357f-120">They are useful for using a small subset of a large and interconnected data source.</span></span>
