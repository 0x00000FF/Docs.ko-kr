# <a name="getting-started-with-f"></a>F# 시작# #

F#으로 시작하는 기본 방법에는 다음과 같이 세 가지가 있습니다.

Windows를 사용하는 경우 [Getting Started with F# in Visual Studio](getting-started-visual-studio.md)(Visual Studio에서 F# 시작)를 확인하세요.

Windows, Linux 또는 macOS에서 간단한 IDE로 코드를 작성하는 데 관심이 있다면 [Getting Started with F# in Visual Studio Code with Ionide](getting-started-vscode.md)(Ionide를 사용하여 Visual Studio 코드에서 F# 시작)를 확인하세요.

임의의 운영 체제에서 명령줄을 사용하려는 경우 [Getting Started with F# with Command-line Tools](getting-started-command-line.md)(명령줄 도구로 F# 시작)를 확인하세요.
