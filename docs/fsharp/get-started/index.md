# <a name="getting-started-with-f"></a>시작 하기F# #

여러 가지 방법으로 시작 하려면 F#입니다.  여러 문서 각 주요 방법에 대 한 지침을 제공 했습니다.  다음 표에서 의사 결정을 내리는 데 사용할 수 있습니다.

| OS | Visual Studio를 선호 합니다. | Visual Studio Code를 선호 합니다. | 명령줄을 선호 |
| -- |------------------------|--------------------------|-----------------------------|-------------------------|
| Windows | [Visual Studio 시작](get-started-visual-studio.md) | [VSCode 및 Ionide 시작](get-started-vscode.md) | [.NET Core CLI를 사용 하 여 시작](get-started-command-line.md) |
| macOS | [Mac 용 VS를 사용 하 여 시작](get-started-with-visual-studio-for-mac.md) | [VSCode 및 Ionide 시작](get-started-vscode.md) | [.NET Core CLI를 사용 하 여 시작](get-started-command-line.md) |
| Linux | N/A | [VSCode 및 Ionide 시작](get-started-vscode.md) | [.NET Core CLI를 사용 하 여 시작](get-started-command-line.md) |

일반적으로 시작 하려면 나머지 보다는 없는 특정 방법이 있습니다.  사용 하는 모든 방법을 시도 하는 것이 좋습니다 F# 원하는 작업을 확인 하려면 컴퓨터에 가장!
