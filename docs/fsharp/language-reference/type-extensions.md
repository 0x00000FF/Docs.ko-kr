# <a name="type-extensions"></a><span data-ttu-id="4aa43-104">형식 확장명</span><span class="sxs-lookup"><span data-stu-id="4aa43-104">Type Extensions</span></span>

<span data-ttu-id="4aa43-105">형식 확장을 사용 하면 이전에 정의 된 개체 형식에 새 멤버를 추가할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="4aa43-105">Type extensions let you add new members to a previously defined object type.</span></span>

## <a name="syntax"></a><span data-ttu-id="4aa43-106">구문</span><span class="sxs-lookup"><span data-stu-id="4aa43-106">Syntax</span></span>

```fsharp
// Intrinsic extension.
type typename with
    member self-identifier.member-name =
        body
    ...
[ end ]

// Optional extension.
type typename with
    member self-identifier.member-name =
        body
    ...
[ end ]
```
