# <a name="do-bindings"></a>do 바인딩

A `do` 코드를 실행 하는 함수 또는 값을 정의 하지 않고 바인딩이 사용 됩니다. 수행 바인딩 수도 있습니다 참조 클래스에 사용 [ `do` 클래스에서](../members/do-bindings-in-classes.md)합니다.


## <a name="syntax"></a>구문

```fsharp
[ attributes ]
[ do ]expression
```
