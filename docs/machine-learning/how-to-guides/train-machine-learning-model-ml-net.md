<span data-ttu-id="a2122-125">다음 코드 조각은 `Size` 및 `HistoricalPrices` 열을 새 열 `Features`에 출력되는 단일 기능 벡터로 결합합니다.</span><span class="sxs-lookup"><span data-stu-id="a2122-125">The following snippet combines the `Size` and `HistoricalPrices` columns into a single feature vector that is output to a new column called `Features`.</span></span> <span data-ttu-id="a2122-126">배율에 차이가 있기 때문에 [`NormalizeMinMax`](xref:Microsoft.ML.NormalizationCatalog.NormalizeMinMax*)는 데이터 표준화를 위해 `Features` 열에 적용됩니다.</span><span class="sxs-lookup"><span data-stu-id="a2122-126">Because there is a difference in scales, [`NormalizeMinMax`](xref:Microsoft.ML.NormalizationCatalog.NormalizeMinMax*) is applied to the `Features` column to normalize the data.</span></span>

```csharp
// 1. Concatenate Size and Historical into a single feature vector output to a new column called Features
// 2. Normalize Features vector

    mlContext.Transforms.Concatenate("Features", "Size", "HistoricalPrices")
        .Append(mlContext.Transforms.NormalizeMinMax("Features"));

// Create data prep transformer

// Apply transforms to training data

```
