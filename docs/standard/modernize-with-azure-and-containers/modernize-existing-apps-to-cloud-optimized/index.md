# <a name="modernize-existing-net-apps-to-cloud-optimized-applications"></a>클라우드 액세스에 최적화 된 응용 프로그램에 기존.NET 앱을 최신식

> 비전: 응용 프로그램을 더 빠르게 제공 하 고 응용 프로그램의 배달 비용을 줄일 수 있도록 배포 민첩성, 상당히 개선에 클라우드 액세스에 최적화 된 기존.NET Framework 응용 프로그램을 최신식 합니다.

컨테이너와 같은 새로운 기술과 클라우드의 이점을 활용하려면 기존.NET 응용 프로그램을 부분적으로나마 현대화해야 합니다. 궁극적으로, 엔터프라이즈 응용 프로그램을 현대화하면 총 소유 비용이 감소합니다.

응용 프로그램을 부분적으로 현대화 반드시 의미는 전체 마이그레이션 및 rearchitecture 합니다. 중요 하지만 쉽게 현대화 할와 기존 응용 프로그램 처음 현대화 할 수 있습니다. 모든 Windows 및 IIS 종속성을 사용하여 기존 .NET Framework 버전으로 작성된 현재 코드베이스를 유지 관리할 수 있습니다. 그림 4-1 어떻게 클라우드 액세스에 최적화 된 응용 프로그램을 강조 표시 현대화 maturity 모델 Azure 응용 프로그램에에서 배치 합니다.
