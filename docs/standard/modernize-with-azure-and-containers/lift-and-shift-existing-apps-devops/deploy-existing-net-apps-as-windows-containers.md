<span data-ttu-id="d0da0-221">또는 Hyper-v 격리 하지 않고도 windows 컨테이너를 실행 하는 것은 런타임에 결정 합니다.</span><span class="sxs-lookup"><span data-stu-id="d0da0-221">Running a container on Windows with or without Hyper-V isolation is a run-time decision.</span></span> <span data-ttu-id="d0da0-222">처음에 Hyper-v 격리 컨테이너를 만들도록 선택할 수도 있으며 런타임 시 Windows Server 컨테이너로 대신 실행 하도록 선택할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="d0da0-222">You might choose to create the container with Hyper-V isolation initially, and at run time, choose to run it as a Windows Server container instead.</span></span>
