# <a name="inserting-nodes-into-an-xml-document"></a>XML 문서에 노드 삽입
새 노드를 삽입하여 문서를 수정할 수 있습니다. 새로 만든 노드 또는 같은 문서에 있는 기존 노드나 다른 문서에서 가져온 노드를 삽입할 수 있습니다. 새 노드를 만드는 방법에 대 한 자세한 내용은 참조 [DOM에서 새 노드 만들기](../../../../docs/standard/data/xml/create-new-nodes-in-the-dom.md)합니다. 기존 노드 복사에 대 한 자세한 내용은 참조 하십시오. [기존 노드 복사](../../../../docs/standard/data/xml/copy-existing-nodes.md), [기존 노드 복사 한 문서에서 다른](../../../../docs/standard/data/xml/copying-existing-nodes-from-one-document-to-another.md), 또는 [문서 조각 복사](../../../../docs/standard/data/xml/copying-document-fragments.md)합니다.
