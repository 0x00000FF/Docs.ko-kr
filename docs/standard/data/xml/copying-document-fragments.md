# 문서 조각 복사
**XmlDocumentFragment** 노드를 만든 다음 그 아래에 노드를 추가할 수 있습니다.  **InsertNode** 메서드를 사용하여 **XmlDocumentFragment**를 삽입하면 XML DOM\(문서 개체 모델\)에 **XmlDocumentFragment** 노드가 복사되지 않고 해당 노드의 자식 노드가 삽입됩니다.
