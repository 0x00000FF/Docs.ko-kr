# <a name="azure-serverless-platform"></a>Azure 서버 리스 플랫폼

Azure 서버 리스 플랫폼 Azure Functions, Logic Apps 및 Event Grid를 포함합니다. 이러한 서비스 함께 연결 하 고 있는 수많은 기타 리소스입니다. 서버 리스 플랫폼의 작동 방식 모두를 사용 하 여 데이터베이스 및 저장소에서 분석 및 기계 학습/인공 지능을 합니다.

Application Insights에서 진단 추적 및 원격 분석 캡처를 위해 서버 리스 플랫폼을 사용할 수도 있습니다. Application Insights는 모든 종류의 응용 프로그램에 사용할 수 있습니다 (데스크톱, 모바일, 또는 웹) 및 서버 리스 구현 합니다. 플랫폼은 다음 다이어그램에 시각화 됩니다.
