# <a name="regular-expression-examples"></a>정규식 예제
이 섹션은 일반적인 응용 프로그램에서 정규식 사용을 보여 주는 코드 예제를 포함합니다.  
  
> [!NOTE]
>  <xref:System.Web.RegularExpressions> 네임 스페이스에는 HTML, XML 및 ASP.NET 문서에서 문자열 구문 분석 하기 위한 미리 정의 된 정규식 패턴을 구현 하는 정규식 개체 수가 포함 되어 있습니다. 예를 들어는 <xref:System.Web.RegularExpressions.TagRegex> 문자열의 시작 태그를 식별 하는 클래스 및 <xref:System.Web.RegularExpressions.CommentRegex> 클래스는 문자열의 ASP.NET 주석을 식별 합니다.
