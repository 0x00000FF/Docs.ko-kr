# .NET Framework의 기본적인 문자열 작업
응용 프로그램에서는 대개 사용자가 입력한 내용에 따라 메시지를 만들어 표시하는 방법으로 응답합니다.  예를 들어, 웹 사이트에 새로운 사용자가 로그온하면 해당 사용자의 이름을 넣은 특별한 인사말이 표시되는 경우를 흔히 볼 수 있습니다.  <xref:System.String?displayProperty=fullName> 및 <xref:System.Text.StringBuilder?displayProperty=fullName> 클래스에는 동적으로 사용자 인터페이스에 표시할 사용자 지정 문자열을 만드는 데 사용할 수 있는 몇 가지 메서드가 있습니다.  이러한 메서드는 바이트의 배열에서 새 문자열을 만들고, 문자열 값을 비교하고, 기존 문자열을 수정하는 등의 기본적인 문자열 작업을 수행하는 데도 유용합니다.
