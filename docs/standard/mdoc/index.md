# <a name="building-documentation-with-mdoc"></a>mdoc를 사용하여 설명서 작성

mdoc는 관리되는 참조 설명서에 대한 설명서 작성 시스템입니다. mdoc는 어셈블리를 통해 반영하는 방식으로 관리되는 참조 설명서의 기본 형식을 만들고 작성된 내용으로 보완합니다.
