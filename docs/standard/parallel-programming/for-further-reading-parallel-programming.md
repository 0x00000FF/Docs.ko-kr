# <a name="for-further-reading-parallel-programming"></a>추가 정보(병렬 프로그래밍)
.NET Framework의 병렬 프로그래밍에 대 한 추가 정보를 포함 하는 다음 리소스:  
  
-   [병렬 컴퓨팅 개발자 센터](http://go.microsoft.com/fwlink/?LinkID=160570) 의 MSDN 웹 사이트에 대 한 링크가 최신 콘텐츠 및 네이티브 c + +에서 및.NET Framework의 병렬 프로그래밍에 대 한 포럼 게시물.  
  
-   [.NET 블로그에서 병렬 프로그래밍에](http://go.microsoft.com/fwlink/?LinkID=169627) 블로그 사이트는 MSDN에서.NET Framework의 병렬 프로그래밍에 대 한 많은 심층 분석 문서를 포함 합니다.  
  
-   [동시성 시각화 도우미 블로그](http://go.microsoft.com/fwlink/?LinkID=169630) MSDN 블로그 사이트에서는 새 성능 프로 파일링의 일부 버전에 포함 된 도구 [!INCLUDE[vs_dev10_long](../../../includes/vs-dev10-long-md.md)]합니다.  
  
-   [병렬 확장 포럼](http://go.microsoft.com/fwlink/?LinkID=169628) msdn 포럼 사이트는 위치를 요청 하 고 병렬 프로그래밍에 대 한 질문에 답변 합니다.  
  
-   [병렬 확장명 샘플](http://go.microsoft.com/fwlink/?LinkID=165717) MSDN 코드 갤러리 웹 사이트의 페이지에는 중급 및 고급 병렬 프로그래밍 기법을 보여 주는 많은 샘플이 포함 되어 있습니다.
