# Parallel Diagnostic Tools
[!INCLUDE[vs_dev10_ext](../../../includes/vs-dev10-ext-md.md)]에서는 다중 스레드 응용 프로그램의 디버깅 및 프로파일링을 광범위하게 지원합니다.  
  
## 디버깅  
 Visual Studio 디버거에 병렬 응용 프로그램 디버깅을 위한 창이 새로 추가되었습니다.  자세한 내용은 다음 항목을 참조하십시오.  
  
-   [병렬 스택 창 사용](../Topic/Using%20the%20Parallel%20Stacks%20Window.md)  
  
-   [작업 창 사용](../Topic/Using%20the%20Tasks%20Window.md)  
  
-   [연습: 병렬 응용 프로그램 디버깅](../Topic/Walkthrough:%20Debugging%20a%20Parallel%20Application.md).  
  
## 프로파일링  
 동시성 시각화 보고서 뷰를 사용하면 병렬 프로그램의 스레드가 서로 어떻게 상호 작용하는지, 시스템에서 진행되는 다른 프로세스의 스레드와는 어떻게 상호 작용하는지 시각화하여 볼 수 있습니다.  자세한 내용은 [동시성 시각화 도우미](../Topic/Concurrency%20Visualizer.md)을 참조하십시오.
