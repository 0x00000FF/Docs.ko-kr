# <a name="notinheritable-visual-basic"></a>NotInheritable(Visual Basic)
클래스를 기본 클래스로 사용할 수 없도록 지정 합니다.  
  
## <a name="remarks"></a>설명  
 **대체 용어**. 상속할 수 없는 클래스를 *봉인* 된 클래스 라고도 합니다.  
  
 이 컨텍스트에서는 `NotInheritable` 한정자를 사용할 수 있습니다.
