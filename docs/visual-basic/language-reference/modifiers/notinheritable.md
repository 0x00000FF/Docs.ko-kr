# NotInheritable (Visual Basic)
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

클래스를 기본 클래스로 사용할 수 없도록 지정합니다.  
  
## 설명  
 **대체 조건**.  상속할 수 없는 클래스를 *봉인* 클래스라고도 합니다.
