# <a name="module-keyword-visual-basic"></a>Module \<키워드 > (Visual Basic)
소스 파일의 시작 부분에 있는 특성이 현재 어셈블리 모듈에 적용 되도록 지정 합니다.  
  
## <a name="remarks"></a>주의  
 많은 특성은 클래스 또는 속성과 같은 개별 프로그래밍 요소와 관련이 있습니다. 꺾쇠 괄호 (`< >`) 내에서 선언 문에 직접 특성 블록을 연결 하 여 이러한 특성을 적용 합니다.  
  
 특성이 다음 요소 뿐만 아니라 현재 어셈블리 모듈에만 관련 되는 경우 특성 블록을 소스 파일의 시작 부분에 놓고 `Module` 키워드를 사용 하 여 특성을 식별 합니다. 전체 어셈블리에 적용 되는 경우 [assembly](../../../visual-basic/language-reference/modifiers/assembly.md) 키워드를 사용 합니다.  
  
 `Module` 한정자는 [Module 문과](../../../visual-basic/language-reference/statements/module-statement.md)다릅니다.
