# <a name="module-ltkeywordgt-visual-basic"></a>모듈 &lt;키워드&gt; (Visual Basic)
소스 파일의 시작 부분에 있는 특성이 현재 어셈블리 모듈에 적용 되도록 지정 합니다.  
  
## <a name="remarks"></a>설명  
 다 수의 특성이 클래스나 속성 같은 개별 프로그래밍 요소에 적용 됩니다. 꺾쇠 괄호 안에 특성 블록을 연결 하 여 특성을 적용 (`< >`), 선언문에 직접 합니다.  
  
 특성에 다음 요소 뿐 아니라 현재 어셈블리 모듈에 관련 된 경우 소스 파일의 시작 부분에 특성 블록을 저장 한 특성으로 식별 한는 `Module` 키워드입니다. 사용 하면 전체 어셈블리에 적용 되는 경우는 [어셈블리](../../../visual-basic/language-reference/modifiers/assembly.md) 키워드입니다.  
  
 `Module` 한정자는 동일 하지는 [Module 문](../../../visual-basic/language-reference/statements/module-statement.md)합니다.
