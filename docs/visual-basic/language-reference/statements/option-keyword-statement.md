# <a name="option-ltkeywordgt-statement"></a>옵션 &lt;키워드&gt; 문
전체 소스 파일에 적용 되는 컴파일러 옵션을 지정 하는 문을 소개 합니다.  
  
## <a name="remarks"></a>설명  
 컴파일러 옵션 여부 변수를 모두 명시적으로 선언 해야, 형식 변환에 축소 변환을 명시적 이어야 하는지 여부 또는 텍스트 또는 이진 값으로 문자열을 비교 하는 여부를 제어할 수 있습니다.  
  
 `Option` 키워드는 다음 컨텍스트에서 사용할 수 있습니다.  
  
 [Option Compare 문](../../../visual-basic/language-reference/statements/option-compare-statement.md)  
  
 [Option Explicit 문](../../../visual-basic/language-reference/statements/option-explicit-statement.md)  
  
 [Option Infer 문](../../../visual-basic/language-reference/statements/option-infer-statement.md)
