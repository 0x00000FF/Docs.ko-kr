## <a name="remarks"></a><span data-ttu-id="b01e7-116">설명</span><span class="sxs-lookup"><span data-stu-id="b01e7-116">Remarks</span></span>

 <span data-ttu-id="b01e7-120">`Binary` 비교에서는 각 문자열 내 각 문자의 숫자 유니코드 값을 비교합니다.</span><span class="sxs-lookup"><span data-stu-id="b01e7-120">A `Binary` comparison compares the numeric Unicode value of each character in each string.</span></span> <span data-ttu-id="b01e7-121">`Text` 비교에서는 현재 문화권의 어휘 의미에 따라 각 유니코드 문자를 비교합니다.</span><span class="sxs-lookup"><span data-stu-id="b01e7-121">A `Text` comparison compares each Unicode character based on its lexical meaning in the current culture.</span></span>  
  
 <span data-ttu-id="b01e7-122">Microsoft windows에서 정렬 순서는 코드 페이지에 의해 결정됩니다.</span><span class="sxs-lookup"><span data-stu-id="b01e7-122">In Microsoft Windows, sort order is determined by the code page.</span></span> <span data-ttu-id="b01e7-123">자세한 내용은 [코드 페이지](/cpp/c-runtime-library/code-pages) 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="b01e7-123">For more information, see [Code Pages](/cpp/c-runtime-library/code-pages).</span></span>

 <span data-ttu-id="b01e7-125">`Option Compare Text`를 사용하여 같은 코드 페이지의 같은 문자를 정렬하면 다음 텍스트 정렬 순서가 생성됩니다.</span><span class="sxs-lookup"><span data-stu-id="b01e7-125">When the same characters in the same code page are sorted by using `Option Compare Text`, the following text sort order is produced.</span></span>
