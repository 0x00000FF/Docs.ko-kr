# <a name="mycomputerfilesystemspecialdirectories-object"></a>My.Computer.FileSystem.SpecialDirectories 개체
일반적으로 참조되는 디렉터리에 액세스하기 위한 속성을 제공합니다.  
  
## <a name="remarks"></a>설명  
 메서드 및 속성에 대 한 내용은 `My.Computer.FileSystem.SpecialDirectories` 개체, 참조 <xref:Microsoft.VisualBasic.FileIO.SpecialDirectories>합니다.  
  
 자세한 내용은 참조 [하는 방법: 내 문서 디렉터리의 내용을 검색](../../../visual-basic/developing-apps/programming/drives-directories-files/how-to-retrieve-the-contents-of-the-my-documents-directory.md)합니다.

 **Namespace:**<xref:Microsoft.VisualBasic.MyServices>  
  
 **클래스:** <xref:Microsoft.VisualBasic.MyServices.SpecialDirectoriesProxy> (에 대 한 액세스를 제공 <xref:Microsoft.VisualBasic.FileIO.SpecialDirectories>)
