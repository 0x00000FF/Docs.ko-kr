# My.Computer.FileSystem.SpecialDirectories Object
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

주로 참조되는 디렉터리에 액세스하기 위한 속성을 제공합니다.  
  
## 설명  
 메서드 및 속성에 대 한 자세한 내용은 `My.Computer.FileSystem.SpecialDirectories` 개체을 참조 하십시오 <xref:Microsoft.VisualBasic.FileIO.SpecialDirectories>.  
  
 자세한 내용은 [How to: Retrieve the Contents of the My Documents Directory](../../../visual-basic/developing-apps/programming/drives-directories-files/how-to-retrieve-the-contents-of-the-my-documents-directory.md)를 참조하십시오.
