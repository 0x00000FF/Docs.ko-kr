# <a name="mycomputerports-object"></a>My.Computer.Ports 개체
컴퓨터의 직렬 포트에 액세스 하기 위한 메서드와 속성을 제공 합니다.  
  
## <a name="remarks"></a>설명  
 메서드 및 속성에 대 한 내용은 `My.Computer.Ports` 개체를 참조 하십시오 <xref:Microsoft.VisualBasic.Devices.Ports>.</xref:Microsoft.VisualBasic.Devices.Ports>  
  
 자세한 내용은 참조 [컴퓨터의 포트에 액세스](../../../visual-basic/developing-apps/programming/computer-resources/accessing-the-computer-s-ports.md)합니다.  
  
> [!NOTE]
>  메서드와 속성을 사용할 수도 있습니다는 <xref:System.IO.Ports.SerialPort?displayProperty=fullName>컴퓨터의 직렬 포트에 액세스 하는 클래스입니다.</xref:System.IO.Ports.SerialPort?displayProperty=fullName>
