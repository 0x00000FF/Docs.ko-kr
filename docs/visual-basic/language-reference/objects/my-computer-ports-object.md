# My.Computer.Ports Object
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

컴퓨터의 직렬 포트에 액세스하기 위한 속성과 메서드를 제공합니다.  
  
## 설명  
 메서드 및 속성에 대 한 자세한 내용은 `My.Computer.Ports` 개체을 참조 하십시오 <xref:Microsoft.VisualBasic.Devices.Ports>.  
  
 자세한 내용은 [Accessing the Computer's Ports](../../../visual-basic/developing-apps/programming/computer-resources/accessing-the-computer-s-ports.md)를 참조하십시오.  
  
> [!NOTE]
>  속성 및 메서드를 사용할 수도 있습니다의 <xref:System.IO.Ports.SerialPort?displayProperty=fullName> 컴퓨터의 직렬 포트에 액세스 하는 클래스입니다.
