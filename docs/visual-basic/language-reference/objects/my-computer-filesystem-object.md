# My.Computer.FileSystem Object
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

드라이브, 파일 및 디렉터리 관련 작업을 수행하기 위한 속성과 메서드를 제공합니다.  
  
## 설명  
 메서드 및 속성에 대 한 자세한 내용은 `My.Computer.FileSystem` 개체을 참조 하십시오 <xref:Microsoft.VisualBasic.FileIO.FileSystem>.  
  
 자세한 내용은 [File Access with Visual Basic](../../../visual-basic/developing-apps/programming/drives-directories-files/file-access.md)를 참조하십시오.  
  
> [!NOTE]
>  클래스를 사용할 수도 있습니다의 <xref:System.IO?displayProperty=fullName> 드라이브, 파일 및 디렉터리를 작동 하는 네임 스페이스입니다.
