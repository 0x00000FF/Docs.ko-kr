# <a name="mycomputermouse-object"></a>My.Computer.Mouse 개체
형식 및 로컬 컴퓨터에 설치 된 마우스의 구성에 대 한 정보를 가져오기 위한 속성을 제공 합니다.  
  
## <a name="remarks"></a>설명  
 메서드 및 속성에 대 한 내용은 `My.Computer.Mouse` 개체, 참조 <xref:Microsoft.VisualBasic.Devices.Mouse>합니다.  
  
 자세한 내용은 참조 [마우스에 액세스](../../../visual-basic/developing-apps/programming/computer-resources/accessing-the-mouse.md)합니다.
