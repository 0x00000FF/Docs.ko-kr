# <a name="myuser-object"></a>My.User 개체
현재 사용자에 대한 정보에 액세스할 수 있습니다.  
  
## <a name="remarks"></a>설명  
 메서드 및 속성에 대 한 내용은 `My.User` 개체, 참조 <xref:Microsoft.VisualBasic.ApplicationServices.User?displayProperty=nameWithType>합니다.  
  
 자세한 내용은 참조 [사용자 데이터에 액세스](../../../visual-basic/developing-apps/programming/accessing-user-data.md)합니다.  
  
## <a name="requirements"></a>요구 사항  
 **어셈블리:** Visual Basic 런타임 라이브러리 (Microsoft.VisualBasic.dll)
