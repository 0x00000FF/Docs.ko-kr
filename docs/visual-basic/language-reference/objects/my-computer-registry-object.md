# <a name="mycomputerregistry-object"></a>My.Computer.Registry 개체
레지스트리를 조작하기 위한 속성 및 메서드를 제공합니다.  
  
## <a name="remarks"></a>설명  
 메서드 및 속성에 대 한 내용은 `My.Computer.Registry` 개체, 참조 <xref:Microsoft.VisualBasic.MyServices.RegistryProxy>합니다.  
  
 자세한 내용은 참조 [읽고 레지스트리에 쓰는](../../../visual-basic/developing-apps/programming/computer-resources/reading-from-and-writing-to-the-registry.md)합니다.  
  
> [!NOTE]
>  메서드를 사용 하 여 레지스트리를 조작할 수도 있습니다는 <xref:Microsoft.Win32.Registry?displayProperty=nameWithType> 클래스입니다.
