# My.Computer.Registry Object
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

레지스트리 조작을 위한 속성과 메서드를 제공합니다.  
  
## 설명  
 메서드 및 속성에 대 한 자세한 내용은 `My.Computer.Registry` 개체을 참조 하십시오 <xref:Microsoft.VisualBasic.MyServices.RegistryProxy>.  
  
 자세한 내용은 [Reading from and Writing to the Registry](../../../visual-basic/developing-apps/programming/computer-resources/reading-from-and-writing-to-the-registry.md)를 참조하십시오.  
  
> [!NOTE]
>  메서드를 사용 하 여 레지스트리를 조작할 수 있는 <xref:Microsoft.Win32.Registry?displayProperty=fullName> 클래스입니다.
