# <a name="mycomputernetwork-object"></a>My.Computer.Network 개체
속성, 이벤트 및 컴퓨터를 연결할 네트워크와 상호 작용 하기 위한 메서드를 제공 합니다.  
  
## <a name="remarks"></a>설명  
 메서드 및 속성에 대 한 내용은 `My.Computer.Network` 개체, 참조 <xref:Microsoft.VisualBasic.Devices.Network>합니다.  
  
 자세한 내용은 참조 [네트워크 작업을 수행](../../../visual-basic/developing-apps/programming/computer-resources/performing-network-operations.md)합니다.
