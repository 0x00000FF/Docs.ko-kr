# <a name="mycomputernetwork-object"></a>My.Computer.Network 개체
Provides a property, event, and methods for interacting with the network to which the computer is connected.  
  
## <a name="remarks"></a>주의  
 `My.Computer.Network` 개체의 메서드 및 속성에 대한 자세한 내용은 <xref:Microsoft.VisualBasic.Devices.Network>를 참조하세요.  
  
 For more information, see [Performing Network Operations](../../../visual-basic/developing-apps/programming/computer-resources/performing-network-operations.md).  
  
## <a name="requirements"></a>요구 사항  
 **네임스페이스:** <xref:Microsoft.VisualBasic.Devices?displayProperty=nameWithType>  
  
 **클래스:** <xref:Microsoft.VisualBasic.Devices.Network>  
  
 **Assembly:** Visual Basic Runtime Library (in Microsoft.VisualBasic.dll)
