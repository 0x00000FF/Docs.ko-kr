# My.Log Object
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

이벤트와 예외 정보를 응용 프로그램의 로그 수신기에 쓰기 위한 속성과 메서드를 제공합니다.  
  
## 설명  
 메서드 및 속성에 대 한 자세한 내용은 `My.Log` 개체을 참조 하십시오 <xref:Microsoft.VisualBasic.Logging.AspLog>.  
  
 `My.Log` 개체는 ASP를 사용할 수 있습니다.NET 응용 프로그램에만 있습니다.  클라이언트 응용 프로그램의 경우 [My.Application.Log 개체](../../../visual-basic/language-reference/objects/my-application-log-object.md)를 사용합니다.
