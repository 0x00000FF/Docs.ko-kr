# <a name="mylog-object"></a>My.Log 개체
속성 및 응용 프로그램의 로그 수신기에 이벤트 및 예외 정보를 쓰기 위한 메서드를 제공 합니다.  
  
## <a name="remarks"></a>설명  
 메서드 및 속성에 대 한 내용은 `My.Log` 개체, 참조 <xref:Microsoft.VisualBasic.Logging.AspLog>합니다.  
  
 `My.Log` 개체는 ASP.NET 응용 프로그램의 경우에 사용할 수 있습니다. 클라이언트 응용 프로그램에 사용 하 여 [My.Application.Log 개체](../../../visual-basic/language-reference/objects/my-application-log-object.md)합니다.  
  
## <a name="requirements"></a>요구 사항  
 **네임스페이스:** <xref:Microsoft.VisualBasic.Logging>  
  
 **클래스:** <xref:Microsoft.VisualBasic.Logging.AspLog>
