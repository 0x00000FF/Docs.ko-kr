# My.Application.Info Object
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

버전 번호, 설명, 로드된 어셈블리 등의 응용 프로그램에 대한 정보를 가져오기 위한 속성을 제공합니다.  
  
## 설명  
 메서드 및 속성에 대 한 자세한 내용은 `My.Application.Info` 개체을 참조 하십시오 <xref:Microsoft.VisualBasic.ApplicationServices.AssemblyInfo>.  
  
> [!NOTE]
>  속성을 사용할 수 있는 <xref:System.Diagnostics.FileVersionInfo?displayProperty=fullName> 디스크에 있는 파일에 대 한 정보를 얻을 수 있는 클래스.
