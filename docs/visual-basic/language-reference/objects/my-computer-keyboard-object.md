# My.Computer.Keyboard Object
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

현재 눌러져 있는 키와 같은 키보드의 현재 상태에 액세스하기 위한 속성을 제공하고 활성 창에 키 입력을 보내기 위한 메서드를 제공합니다.  
  
## 설명  
 메서드 및 속성에 대 한 자세한 내용은 `My.Computer.Keyword` 개체을 참조 하십시오 <xref:Microsoft.VisualBasic.Devices.Keyboard>.  
  
 자세한 내용은 [Accessing the Keyboard](../../../visual-basic/developing-apps/programming/computer-resources/accessing-the-keyboard.md)를 참조하십시오.
