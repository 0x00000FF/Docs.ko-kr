# My.Computer Object
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

오디오, 시계, 키보드, 파일 시스템 등과 같은 컴퓨터 구성 요소를 조작하기 위한 속성을 제공합니다.  
  
## 설명  
 메서드 및 속성에 대 한 자세한 내용은 `My.Computer` 개체을 참조 하십시오 <xref:Microsoft.VisualBasic.Devices.Computer>.  기본 클래스 <xref:Microsoft.VisualBasic.Devices.ServerComputer> 의 모든 프로젝트에 사용할 수 있는 멤버를 제공 합니다.  
  
## 요구 사항  
 **네임스페이스:** <xref:Microsoft.VisualBasic.Devices>  
  
 **클래스:** <xref:Microsoft.VisualBasic.Devices.Computer> \(기본 클래스 <xref:Microsoft.VisualBasic.Devices.ServerComputer> 의 모든 프로젝트에 사용할 수 있는 멤버를 제공\).  
  
 **어셈블리:** Visual Basic 런타임 라이브러리\(Microsoft.VisualBasic.dll\)
