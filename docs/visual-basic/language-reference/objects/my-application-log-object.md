# <a name="myapplicationlog-object"></a>My.Application.Log 개체
응용 프로그램의 로그 수신기에 이벤트 및 예외 정보를 쓸 수 있는 속성 및 메서드를 제공합니다.  
  
## <a name="remarks"></a>설명  
 메서드 및 속성에 대 한 내용은 `My.Application.Log` 개체를 참조 하십시오 <xref:Microsoft.VisualBasic.Logging.Log>.</xref:Microsoft.VisualBasic.Logging.Log>  
  
 자세한 내용은 참조 [응용 프로그램에서 정보 로깅](../../../visual-basic/developing-apps/programming/log-info/logging-information-from-the-application.md)합니다.  
  
> [!NOTE]
>  .NET Framework의 클래스를 사용하여 응용 프로그램의 정보를 기록할 수도 있습니다. 자세한 내용은 참조 [추적 및 응용 프로그램을 계측](http://msdn.microsoft.com/library/773b6fc4-9013-4322-b728-5dec7a72e743)합니다.
