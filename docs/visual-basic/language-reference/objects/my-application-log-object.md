# My.Application.Log 개체
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

응용 프로그램의 로그 수신기에 이벤트 및 예외 정보를 쓸 수 있는 속성 및 메서드를 제공합니다.  
  
## 설명  
 `My.Application.Log` 개체의 메서드 및 속성에 대한 자세한 내용은 <xref:Microsoft.VisualBasic.Logging.Log>를 참조하세요.  
  
 자세한 내용은 [Logging Information from the Application](../../../visual-basic/developing-apps/programming/log-info/logging-information-from-the-application.md)을 참조하세요.  
  
> [!NOTE]
>  .NET Framework의 클래스를 사용하여 응용 프로그램의 정보를 기록할 수도 있습니다. 자세한 내용은 [Tracing and Instrumenting Applications](../Topic/Tracing%20and%20Instrumenting%20Applications.md)을 참조하세요.
