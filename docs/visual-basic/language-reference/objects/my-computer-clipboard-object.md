# <a name="mycomputerclipboard-object"></a>My.Computer.Clipboard 개체
클립보드를 조작하기 위한 메서드를 제공합니다.  
  
## <a name="remarks"></a>설명  
 메서드 및 속성에 대 한 내용은 `My.Computer.Clipboard` 개체, 참조 <xref:Microsoft.VisualBasic.MyServices.ClipboardProxy>합니다.  
  
 자세한 내용은 참조 [데이터를 저장 하 고 클립보드에서 읽기](../../../visual-basic/developing-apps/programming/computer-resources/storing-data-to-and-reading-from-the-clipboard.md)합니다.  
  
> [!NOTE]
>  메서드를 사용할 수도 있습니다는 <xref:System.Windows.Forms.Clipboard?displayProperty=nameWithType> 클립보드를 조작 하는 클래스입니다.
