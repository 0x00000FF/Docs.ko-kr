# Can&#39;t create necessary temporary file
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

TEMP 환경 변수에서 지정한 디렉터리를 포함한 드라이브가 꽉 찼거나, TEMP 환경 변수에서 잘못되거나 읽기 전용인 드라이브 또는 디렉터리를 지정했습니다.  
  
### 이 오류를 해결하려면  
  
1.  꽉 찬 드라이브에서 파일을 제거합니다.  
  
2.  TEMP 환경 변수에서 다른 드라이브를 지정합니다.  
  
3.  TEMP 환경 변수에 올바른 드라이브를 지정합니다.  
  
4.  현재 지정된 드라이브 또는 디렉터리에서 읽기 전용 제한을 제거합니다.
