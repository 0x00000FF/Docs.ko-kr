# Unable to load information for class &#39;&lt;classname&gt;&#39;
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

클래스를 참조할 수 없습니다.  
  
 **오류 ID:** BC30712  
  
### 이 오류를 해결하려면  
  
1.  클래스가 정의되었는지 그리고 클래스 이름의 철자가 올바른지 확인합니다.  
  
2.  모듈에 선언된 멤버 중 하나에 액세스해 보십시오.  멤버가 선언된 모듈이 로드되지 않아 디버깅 환경에서 멤버를 찾을 수 없는 경우도 있습니다.
