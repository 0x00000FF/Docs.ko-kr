# <a name="unable-to-load-information-for-class-39ltclassnamegt39"></a>클래스에 대 한 정보를 로드할 수 없습니다&lt;classname&gt;'
사용할 수 있는 클래스에 대 한 참조가.  
  
 **오류 ID:** BC30712  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  클래스를 정의 하 고 이름을 제대로 입력 했는지 확인 합니다.  
  
2.  모듈에 선언된 멤버 중 하나에 액세스해 봅니다. 선언된 모듈이 아직 로드되지 않아 디버깅 환경에서 멤버를 찾을 수 없는 경우도 있습니다.
