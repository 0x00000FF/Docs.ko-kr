# Identifier expected
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

요소 이름이 필요한 곳에 인식할 수 있는 선언된 요소 이름이 아닌 프로그래밍 요소를 사용했습니다.  예를 들어, 문의 시작 부분이 아닌 곳에 특성이 지정된 경우 이 오류가 발생합니다.  
  
 **오류 ID:** BC30203  
  
### 이 오류를 해결하려면  
  
-   특성이 모두 문의 시작 부분에 있는지 확인합니다.  
  
-   문에 있는 요소 이름의 철자가 모두 올바른지 확인합니다.
