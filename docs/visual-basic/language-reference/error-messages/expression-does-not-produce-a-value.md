# <a name="expression-does-not-produce-a-value"></a>식이 값을 산출하지 않습니다.
컨텍스트에서 값을 산출 호출 등을 생성 하지 않는 식을 사용 하려는 `Sub` 컨텍스트에서 여기서는 `Function` 예상 됩니다.  
  
 **오류 ID:** BC30491  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   값을 생성 하는 식을 변경 합니다.
