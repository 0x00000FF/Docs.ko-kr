# Expression does not produce a value
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

`Function`이 필요한 곳에서 `Sub`를 호출하는 것과 같이 값을 산출하는 컨텍스트에서 값을 산출하지 않는 식을 사용하려고 했습니다.  
  
 **오류 ID:** BC30491  
  
### 이 오류를 해결하려면  
  
-   값을 산출하는 식으로 변경합니다.
