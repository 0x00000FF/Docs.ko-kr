# <a name="variable-uses-an-automation-type-not-supported-in-visual-basic"></a>변수가 Visual Basic에서 지원되지 않는 자동화 형식 사용
형식 라이브러리 또는에서 지원 되지 않습니다 데이터 형식이 있는 개체 라이브러리에서 정의 된 변수를 사용 하 려 [!INCLUDE[vbprvb](../../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]합니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   인식 하는 형식의 변수를 사용 하 여 [!INCLUDE[vbprvb](../../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]합니다.  
  
     또는  
  
-   사용 하는 동안이 오류가 발생 하는 경우 `FileGet` 또는 `FileGetOBject`를 해야 사용 하려는 파일으로 쓴와 `FilePut` 또는 `FilePutObject`합니다.
