# File already open
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

일부 경우 다른 `FileOpen` 또는 다른 작업이 발생하기 전에 파일을 닫아야 합니다.  이 오류가 발생하는 몇 가지 원인은 다음과 같습니다.  
  
-   순차적 출력 모드 `FileOpen` 작업이 이미 열린 파일에 대해 실행되었습니다.  
  
-   문에서 열린 파일을 참조했습니다.  
  
### 이 오류를 해결하려면  
  
1.  문을 실행하기 전에 파일을 닫습니다.
