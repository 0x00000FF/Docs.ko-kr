# &#39;&lt;name&gt;&#39; is ambiguous in the namespace &#39;&lt;namespacename&gt;&#39;
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

모호한 이름을 지정했으므로 다른 이름과 충돌합니다.  [!INCLUDE[vbprvb](../../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)] 컴파일러에는 충돌을 해결하는 규칙이 없으므로 사용자가 이름을 명확히 지정해야 합니다.  
  
 **오류 ID:** BC30560  
  
### 이 오류를 해결하려면  
  
-   이름을 정규화합니다.
