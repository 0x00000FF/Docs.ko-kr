# Property not found
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

이 개체는 지정된 속성을 지원하지 않습니다.  
  
### 이 오류를 해결하려면  
  
1.  속성 이름의 철자를 확인합니다.  
  
2.  개체의 설명서를 확인하여 개체가 실제로 "caption" 또는 비슷한 이름의 속성을 지원할 때 "text" 속성과 같은 것에 액세스하려고 시도하지 않았는지 확인합니다.
