# Device I/O error
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

현재 프로그램에서 프린터 또는 디스크 드라이브와 같은 장치를 사용하는 동안 입\/출력 오류가 발생했습니다.  
  
### 이 오류를 해결하려면  
  
-   장치가 제대로 작동하고 있는지 확인한 다음 다시 작업합니다.
