# &#39;&lt;name1&gt;&#39; is ambiguous, imported from the namespaces or types &#39;&lt;name2&gt;&#39;
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

모호한 이름을 지정했으므로 다른 이름과 충돌합니다.  [!INCLUDE[vbprvb](../../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] 컴파일러에는 충돌을 해결하는 규칙이 없으므로 사용자가 이름을 명확히 지정해야 합니다.  
  
 **오류 ID:** BC30561  
  
### 이 오류를 해결하려면  
  
1.  네임스페이스 가져오기를 제거하여 이름을 명확하게 합니다.  
  
2.  이름을 정규화합니다.
