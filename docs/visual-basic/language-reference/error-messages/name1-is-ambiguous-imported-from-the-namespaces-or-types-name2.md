# <a name="39ltname1gt39-is-ambiguous-imported-from-the-namespaces-or-types-39ltname2gt39"></a>'&lt;name1&gt;'입니다. 모호한, 가져온 네임 스페이스 또는 형식에서 '&lt;name2&gt;'
모호한 이름을 제공했으므로 다른 이름과 충돌합니다. [!INCLUDE[vbprvb](../../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] 컴파일러에는 충돌 해결 규칙이 없으므로 직접 이름을 명확하게 해야 합니다.  
  
 **오류 ID:** BC30561  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  네임 스페이스 가져오기를 제거 하 여 이름을 명확 하 게 합니다.  
  
2.  이름을 정규화합니다.
