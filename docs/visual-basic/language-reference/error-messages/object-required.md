# Object required (Visual Basic)
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

속성 및 메서드에 대한 참조에는 종종 명시적인 개체 한정자가 필요합니다.  지금이 이러한 경우입니다.  
  
### 이 오류를 해결하려면  
  
1.  메서드 또는 개체 속성에 대한 참조의 개체 한정자가 올바른지 확인합니다.  지정하지 않았으면 개체 한정자를 지정합니다.  
  
2.  개체 한정자의 철자를 확인한 다음 참조하고 있는 프로그램에서 개체가 표시되는지 확인합니다.  
  
3.  호스트 응용 프로그램의 **파일 열기** 명령에 경로를 지정한 경우 포함된 인수가 올바른지 확인합니다.  
  
4.  개체의 설명서를 확인하여 해당 작업이 올바른지 확인합니다.
