# Array subscript expression missing
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

배열 초기화에서 배열 범위를 정의하는 첨자가 하나 이상 생략되었습니다.  예를 들어, 문에 세 번째 첨자가 생략된 `myArray (5,5,,10)` 식이 있습니다.  
  
 **오류 ID:** BC30306  
  
### 이 오류를 해결하려면  
  
-   누락된 첨자를 지정합니다.
