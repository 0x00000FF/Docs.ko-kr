# <a name="array-subscript-expression-missing"></a>배열 첨자식이 없습니다.
하나 이상의 배열 범위를 정의 하는 아래 첨자를 배제 하는 배열 초기화 합니다. 예를 들어, 문이 식이 있을 `myArray (5,5,,10)`는 세 번째 첨자 배제 합니다.  
  
 **오류 ID:** BC30306  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   누락 된 아래 첨자를 제공 합니다.
