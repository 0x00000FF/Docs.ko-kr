# <a name="can39t-open-39ltfilenamegt39-for-writing"></a>수 &#39; t 열기 &#39; &lt;filename&gt;&#39; 쓰기를 위한

 **오류 ID:** BC2012

2.  파일의 권한을 확인 합니다.
