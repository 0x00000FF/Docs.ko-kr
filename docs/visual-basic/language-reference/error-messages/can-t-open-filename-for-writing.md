# <a name="can39t-open-39ltfilenamegt39-for-writing"></a>수&#39;t 열기 &#39; &lt;filename&gt; &#39; 쓰기용
지정된 된 파일을 쓰기용 이미 열려 있으므로 열 수 없습니다.  
  
 **오류 ID:** BC2012  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  파일을 닫고 다시 엽니다.  
  
2.  파일의 권한을 확인 합니다.
