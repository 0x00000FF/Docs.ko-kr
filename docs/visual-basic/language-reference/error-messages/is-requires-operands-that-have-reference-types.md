# &#39;Is&#39; requires operands that have reference types, but this operand has the value type &#39;&lt;typename&gt;&#39;
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

`Is` 비교 연산자가 두 개체 변수가 같은 인스턴스를 참조하는지 여부를 확인합니다.  이 비교는 값 형식에 대해서는 정의되어 있지 않습니다.  
  
 **오류 ID:** BC30020  
  
### 이 오류를 해결하려면  
  
-   적합한 산술 비교 연산자 또는 `Like` 연산자를 사용하여 두 값 형식을 비교합니다.
