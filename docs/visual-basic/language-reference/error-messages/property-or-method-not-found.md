# Property or method not found
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

참조된 개체 메서드나 개체 속성이 정의되지 않았습니다.  
  
### 이 오류를 해결하려면  
  
-   개체 이름의 철자가 잘못되었을 수 있습니다.  개체에 정의된 속성과 메서드를 보려면 개체 브라우저를 표시합니다.  적당한 개체 라이브러리를 선택하여 사용할 수 있는 속성과 메서드의 목록을 확인합니다.
