# Line is too long
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

소스 텍스트 줄은 65535자를 초과할 수 없습니다.  
  
 **오류 ID:** BC30494  
  
### 이 오류를 해결하려면  
  
-   줄의 길이를 65535자 이하로 줄입니다.
