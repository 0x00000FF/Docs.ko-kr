# <a name="line-is-too-long"></a>줄이 너무 깁니다.
소스 텍스트 줄에는 65,535 자를 초과할 수 없습니다.  
  
 **오류 ID:** BC30494  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 65535 자 이하의 줄 길이 줄이십시오.
