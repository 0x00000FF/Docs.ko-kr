지정된 된 키 파일에서 강력한 이름의 어셈블리를 만들 수 없습니다.  
  
 **오류 ID:** BC31026

1.  올바른 키 파일 지정 되어 있는지 그리고 다른 응용 프로그램에 의해 잠겨 있지 않은지 확인 하십시오.
