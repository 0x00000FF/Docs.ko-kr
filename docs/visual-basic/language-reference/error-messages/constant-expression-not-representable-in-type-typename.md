# Constant expression not representable in type &#39;&lt;typename&gt;&#39;
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

대상 형식에 맞지 않는 상수를 계산하려고 했습니다. 이 오류가 발생하는 이유는 대개 상수 범위가 오버플로되기 때문입니다.  
  
 **오류 ID:** BC30439  
  
### 이 오류를 해결하려면  
  
1.  대상 형식을 상수를 처리할 수 있는 형식으로 변경합니다.
