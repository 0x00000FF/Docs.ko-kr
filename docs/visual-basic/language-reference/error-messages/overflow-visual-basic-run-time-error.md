# <a name="overflow-visual-basic-run-time-error"></a>오버플로가 발생했습니다(Visual Basic 런타임 오류).
할당 대상의 한계를 초과 하 여 할당 하려고 할 때 오버플로가 발생 했습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  결과가 너무 커서 변수 값의 해당 형식에 대 한 범위에서 나타낼 수 없는 변환과 형식 변수에 값 할당 할당, 계산 및 데이터 형식 값의 더 큰 범위를 수용할 수 있는지 확인 필요한 경우.  
  
2.  속성에 할당 맞게 수행 되는 속성의 범위에 있는지 확인 합니다.  
  
3.  정수도 강제 변환 하는 계산에 사용 되는 번호 정수 보다 큰 결과 포함 하지 않는지 확인 합니다.
