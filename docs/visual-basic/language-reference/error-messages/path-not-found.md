# Path not found
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

파일 액세스 또는 디스크 액세스 작업을 수행하는 동안 운영 시스템에서 지정된 경로를 찾을 수 없습니다.  파일에 대한 경로는 파일의 위치에 이르기까지 거치는 드라이브 사양, 디렉터리 및 하위 디렉터리를 포함합니다.  경로는 상대적이거나 절대적일 수 있습니다.  
  
### 이 오류를 해결하려면  
  
-   경로를 확인한 다음 다시 지정합니다.
