# <a name="procedure-call-or-argument-is-not-valid-visual-basic"></a>프로시저 호출 또는 인수가 잘못되었습니다(Visual Basic).
호출의 일부를 완료할 수 없습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   허용 되는 값 넘지 않도록 확인에 대 한 인수에 대 한 허용된 범위를 확인 합니다.
