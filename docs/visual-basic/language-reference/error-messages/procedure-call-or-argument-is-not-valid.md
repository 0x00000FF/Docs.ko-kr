# <a name="procedure-call-or-argument-is-not-valid-visual-basic"></a>프로시저 호출 또는 인수가 잘못되었습니다(Visual Basic).
호출의 일부를 완료할 수 없습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 허용 되는 값을 초과 하는 배열이 없는지 확인 하려면 인수에 대해 허용 되는 범위를 확인 합니다.
