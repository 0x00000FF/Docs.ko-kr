# Events cannot be declared with a delegate type that has a return type
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

대리자를 함수 프로시저에 지정했습니다.  
  
 **오류 ID:** BC31084  
  
### 이 오류를 해결하려면  
  
-   `Sub` 프로시저에 대리자를 지정합니다.
