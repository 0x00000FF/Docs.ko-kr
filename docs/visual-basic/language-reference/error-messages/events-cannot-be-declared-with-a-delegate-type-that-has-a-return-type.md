대리자는 function 프로시저에 지정 되었습니다.  
  
 **오류 ID:** BC31084  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   에 대리자를 지정 된 `Sub` 프로시저입니다.
