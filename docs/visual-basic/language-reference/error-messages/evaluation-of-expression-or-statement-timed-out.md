# Evaluation of expression or statement timed out
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

지정된 시간 안에 식을 계산하지 못했습니다.  
  
 **오류 ID:** BC30722  
  
### 이 오류를 해결하려면  
  
1.  입력한 코드가 올바른지 확인합니다.  
  
2.  식을 단순화하여 실행 시간을 줄입니다.
