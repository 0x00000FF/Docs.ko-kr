# <a name="evaluation-of-expression-or-statement-timed-out"></a>식 또는 문 계산 시간이 초과되었습니다.
식의 계산 시기 적절 하 게에서 완료 되지 않았습니다.  
  
 **오류 ID:** BC30722  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  입력 한 코드가 올바른지 확인 합니다.  
  
2.  실행 하는 시간이 짧으므로 식을 단순화 합니다.
