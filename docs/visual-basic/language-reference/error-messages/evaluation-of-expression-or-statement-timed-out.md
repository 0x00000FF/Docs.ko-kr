# <a name="evaluation-of-expression-or-statement-timed-out"></a>식 또는 문 계산 시간이 초과되었습니다.
식의 평가 적시에 완료 되지 않았습니다.  
  
 **오류 ID:** BC30722  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1. 입력 한 코드가 올바른지 확인 합니다.  
  
2. 실행 시간이 적게 소요 되는 식을 간소화 합니다.
