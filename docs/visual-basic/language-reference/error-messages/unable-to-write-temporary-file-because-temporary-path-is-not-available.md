# Unable to write temporary file because temporary path is not available
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

[!INCLUDE[vbprvb](../../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)]에서 임시 파일을 저장할 경로를 확인할 수 없습니다.  
  
 **오류 ID:** BC30698  
  
### 이 오류를 해결하려면  
  
1.  [!INCLUDE[vsprvs](../../../csharp/includes/vsprvs-md.md)]를 다시 시작합니다.  
  
2.  문제가 계속되면 [!INCLUDE[vsprvs](../../../csharp/includes/vsprvs-md.md)]를 다시 설치합니다.
