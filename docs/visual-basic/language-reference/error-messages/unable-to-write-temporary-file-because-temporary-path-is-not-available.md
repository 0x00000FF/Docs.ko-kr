# <a name="unable-to-write-temporary-file-because-temporary-path-is-not-available"></a>임시 경로를 사용할 수 없으므로 임시 파일에 쓸 수 없습니다.
[!INCLUDE[vbprvb](../../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]에서 임시 파일을 저장할 경로를 확인할 수 없습니다.  
  
 **오류 ID:** BC30698  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  [!INCLUDE[vsprvs](../../../csharp/includes/vsprvs_md.md)]를 다시 시작합니다.  
  
2.  문제가 계속되면 [!INCLUDE[vsprvs](../../../csharp/includes/vsprvs_md.md)]를 다시 설치합니다.
