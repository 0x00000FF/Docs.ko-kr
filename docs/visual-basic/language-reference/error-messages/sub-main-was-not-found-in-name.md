# <a name="39sub-main39-was-not-found-in-39ltnamegt39"></a>&#39; Sub Main &#39; 찾을 수 없습니다 &#39; &lt;이름&gt;&#39;
`Sub Main`이 누락되었거나 잘못된 위치가 지정되었습니다.  
  
 **오류 ID:** BC30420  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  누락된 `Sub Main` 문을 제공하거나, 해당 문이 있으면 이 문을 코드의 적합한 위치로 이동합니다. 대 한 자세한 내용은 `Sub Main`, 참조 [Visual Basic의 Main 프로시저](../../../visual-basic/programming-guide/program-structure/main-procedure.md)합니다.  
  
2.  프로젝트의 시작 개체의 위치를 지정 된 **시작 폼** 상자는 **프로젝트 디자이너**합니다.
