# &#39;Sub Main&#39; was not found in &#39;&lt;name&gt;&#39;
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

`Sub Main`이 누락되었거나 잘못된 위치가 지정되었습니다.  
  
 **오류 ID:** BC30420  
  
### 이 오류를 해결하려면  
  
1.  누락된 `Sub Main` 문을 제공하거나, 해당 문이 있으면 이 문을 코드의 적합한 위치로 이동합니다.  `Sub Main`에 대한 자세한 내용은 [Main Procedure in Visual Basic](../../../visual-basic/programming-guide/program-structure/main-procedure.md)를 참조하세요.  
  
2.  **프로젝트 디자이너**의 **시작 폼**에서 프로젝트 시작 개체의 위치를 지정합니다.
