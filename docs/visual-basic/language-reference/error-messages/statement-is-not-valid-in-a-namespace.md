# <a name="statement-is-not-valid-in-a-namespace"></a>네임스페이스에는 문을 사용할 수 없습니다.
문을 네임 스페이스의 수준에 나타날 수 없습니다. 네임 스페이스 수준에서 선언할 모듈, 인터페이스, 클래스, 대리자, 열거형 및 구조체 선언 됩니다.  
  
 **오류 ID:** BC30001  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   문을 모듈, 클래스, 인터페이스, 구조체, 열거형 또는 대리자 정의 내의 위치로 이동 합니다.
