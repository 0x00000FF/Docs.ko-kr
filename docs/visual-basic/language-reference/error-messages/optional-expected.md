# <a name="39optional39-expected"></a>&#39; 선택 사항 &#39; 필요합니다.
프로시저 선언에서 선택적 인수는 필수 인수에 나옵니다. 모든 인수는 선택적 인수 뒤에 선택적 이어야 합니다.  
  
 **오류 ID:** BC30202  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  인수가 필요할 수 하려는 경우에 인수 목록에 첫 번째 선택적 인수를 앞으로 이동 합니다.  
  
2.  선택적 인수는 의도 한 경우 사용 하 여는 `Optional` 키워드입니다.
