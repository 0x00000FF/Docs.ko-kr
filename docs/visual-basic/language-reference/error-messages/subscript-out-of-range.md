# <a name="subscript-out-of-range-visual-basic"></a>첨자가 범위를 벗어났습니다. (Visual Basic)
배열 첨자가 허용 범위를 벗어났기 때문에 올바르지 않습니다. 차원에 대 한 최저 첨자 값은 항상 0이 고 최고 첨자 값 반환한는 `GetUpperBound` 해당 차원에 대 한 메서드.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 첨자를 변경 하 여 유효한 범위 내에 속합니다.
