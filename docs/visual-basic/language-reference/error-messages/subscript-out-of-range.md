# <a name="subscript-out-of-range-visual-basic"></a>첨자가 범위를 벗어났습니다. (Visual Basic)
배열 첨자가 허용 범위를 벗어나므로 잘못 되었습니다. 차원에 대 한 가장 낮은 첨자 값은 항상 0이 고 가장 높은 첨자 값은 해당 차원에 대 한 `GetUpperBound` 메서드에서 반환 됩니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 올바른 범위에 속하도록 첨자를 변경 합니다.
