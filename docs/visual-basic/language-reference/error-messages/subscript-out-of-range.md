# <a name="subscript-out-of-range-visual-basic"></a>첨자가 범위를 벗어났습니다. (Visual Basic)
배열 첨자 허용 가능한 범위를 벗어났기 때문에 올바르지 않습니다. 차원에 대 한 첨자 가장 낮은 값은 항상 0이 고, 최대 첨자 값에서 반환 되는 `GetUpperBound` 해당 차원에 대 한 메서드.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   첨자를 변경 하 여 유효한 범위 내에 속합니다.
