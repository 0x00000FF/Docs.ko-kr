# Property let procedure not defined and property get procedure did not return an object
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

특정 속성, 메서드 및 작업만 `Collection` 개체에 적용할 수 있습니다.  컬렉션에 대해 실행할 수 있는 작업 또는 속성을 지정했지만 개체가 컬렉션이 아닙니다.  
  
### 이 오류를 해결하려면  
  
1.  개체 또는 속성 이름의 철자를 확인하거나 개체가 `Collection` 개체인지 확인합니다.  
  
2.  개체를 컬렉션에 추가하는 데 사용한 `Add` 메서드를 확인하여 구문 및 식별자의 철자가 올바른지 확인합니다.
