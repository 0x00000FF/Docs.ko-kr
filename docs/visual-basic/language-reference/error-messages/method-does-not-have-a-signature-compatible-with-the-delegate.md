# <a name="method-does-not-have-a-signature-compatible-with-the-delegate"></a>메서드에 대리자와 호환되는 시그니처가 없습니다.
메서드 서명와 사용 하려고 하는 대리자 간의 비 호환성을는 없습니다. `Delegate` 문에서 대리자 클래스의 매개 변수 형식 및 반환 형식을 정의합니다. 호환 가능한 형식 매개 변수에 일치 하는 및 형식을이 대리자 형식의 인스턴스를 만드는 데 사용할 수를 반환 하는 모든 프로시저입니다.  
  
 **오류 ID**: BC36563
