# <a name="too-many-files"></a>파일이 너무 많습니다.
운영 체제에서 허용 더 많은 파일이 루트 디렉터리에 생성 하거나 더 많은 파일에 지정 된 숫자 보다 열린는 **파일 =** config입니다. SYS 파일입니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  프로그램 열기, 닫기 또는 루트 디렉터리에 파일을 저장 하는 경우에 하위 디렉터리를 사용 하도록 프로그램을 변경 합니다.  
  
2.  에 지정 된 파일 수를 늘리려면 프로그램 **파일 =** config입니다. SYS 파일을 선택한 컴퓨터를 다시 시작 합니다.
