# <a name="too-many-files"></a>파일이 너무 많습니다.
자세한 파일이 만들어져 루트 디렉터리에 운영 체제에서 허용 또는 더 많은 파일에서 지정한 숫자 보다 열린 합니다 **파일 =** config에서 설정 합니다. SYS 파일입니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  프로그램 열기, 닫기 또는 루트 디렉터리에 파일을 저장 하는 경우 하위 디렉터리를 사용할 수 있도록 프로그램을 변경 합니다.  
  
2.  에 지정 된 파일 수를 늘릴 하 **파일 =** config에서 설정 합니다. SYS 파일과 컴퓨터 다시 시작 합니다.
