# Overflow (Visual Basic Error)
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

리터럴이 대입될 데이터 형식의 범위를 벗어나는 값을 나타냅니다.  
  
 **오류 ID:** BC30036  
  
### 이 오류를 해결하려면  
  
-   대상 데이터 형식의 값 범위를 확인하여 해당 범위에 맞게 리터럴을 다시 작성합니다.
