# <a name="property-array-index-is-not-valid"></a>속성 배열 인덱스가 잘못되었습니다.
제공 된 값에 대 한 속성 배열 인덱스가 올바르지 않습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   인덱스가 지정된 된 속성에 대 한 유효한 범위 내에 있는지 확인 하려면 해당 구성 요소의 설명서를 확인 합니다.
