에 대 한 초기 호출에서 `Dir` 함수 포함 되지 않습니다는 `PathName` 인수입니다. 첫 번째 호출 `Dir` 포함 되어야 합니다는 `PathName`, 하지만 후속 호출을 `Dir` 다음 항목을 검색 하는 매개 변수를 포함할 필요가 없습니다.
