# String constants must end with a double quote
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

문자열 상수는 큰따옴표로 시작하고 끝나야 합니다.  
  
 **오류 ID:** BC30648  
  
### 이 오류를 해결하려면  
  
-   문자열 리터럴이 따옴표\("\)로 끝나도록 해야 합니다.  다른 텍스트 편집기에서 값을 붙여넣은 경우 붙여넣은 문자가 따옴표와 비슷한 "둥근 따옴표"\(""\) 또는 두 개의 "작은따옴표"\(''\)로 잘못 끝난 것은 아닌지 확인합니다.
