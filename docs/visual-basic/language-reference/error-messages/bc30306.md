# <a name="array-subscript-expression-missing"></a>배열 첨자식이 없습니다.
배열 초기화는 배열 범위를 정의 하는 첨자가 하나 이상 생략 됩니다. 예를 들어 문은 세 번째 첨자를 유지 하는-0 @no__t 식을 포함할 수 있습니다.  
  
 **오류 ID:** BC30306  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 누락 된 첨자를 제공 합니다.
