# <a name="xml-namespace-uri-39httpwwww3orgxml1998namespace39-can-be-bound-only-to-39xmlns39"></a>XML 네임 스페이스 URI &#39; http://www.w3.org/XML/1998/namespace &#39; 에 바인딩될 수 &#39; xmlns &#39;
URI http://www.w3.org/XML/1998/namespace XML 네임 스페이스 선언에 사용 됩니다. 이 URI는 예약 된 네임 스페이스와 XML 네임 스페이스 선언에 포함 될 수 없습니다.  
  
 **오류 ID:** BC31183  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   XML 네임 스페이스 선언을 제거 하거나 유효한 네임 스페이스 URI와은 URI http://www.w3.org/XML/1998/namespace를 바꿉니다.
