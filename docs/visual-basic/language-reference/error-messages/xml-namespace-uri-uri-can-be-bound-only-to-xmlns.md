# XML namespace URI &#39;http://www.w3.org/XML/1998/namespace&#39; can be bound only to &#39;xmlns&#39;
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

XML 네임스페이스 선언에 http:\/\/www.w3.org\/XML\/1998\/namespace URI를 사용했습니다.  이 URI는 예약된 네임스페이스이므로 XML 네임스페이스 선언에 포함할 수 없습니다.  
  
 **오류 ID:** BC31183  
  
### 이 오류를 해결하려면  
  
-   XML 네임스페이스 선언을 제거하거나 http:\/\/www.w3.org\/XML\/1998\/namespace URI를 올바른 네임스페이스 URI로 바꿉니다.
