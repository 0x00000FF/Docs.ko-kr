# Labels that are numbers must be followed by colons
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

줄 번호는 다른 종류의 레이블과 같은 규칙을 따르며 콜론이 포함되어야 합니다.  
  
 **오류 ID:** BC30801  
  
### 이 오류를 해결하려면  
  
-   코드 줄 시작 부분에서 줄 번호 뒤에 콜론을 붙입니다. 예를 들면 다음과 같습니다.  
  
    ```  
    400:    X += 1  
    ```
