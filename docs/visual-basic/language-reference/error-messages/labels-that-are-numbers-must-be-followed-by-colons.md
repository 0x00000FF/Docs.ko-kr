# <a name="labels-that-are-numbers-must-be-followed-by-colons"></a>숫자 레이블 뒤에는 콜론이 와야 합니다.
줄 번호 레이블, 다른 종류와 동일한 규칙을 따르고 콜론을 포함 해야 합니다.  
  
 **오류 ID:** BC30801  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 코드 줄의 시작 부분에 콜론 수를 배치 합니다. 예를 들어:  
  
    ```  
    400:    X += 1  
    ```
