# Type mismatch (Visual Basic)
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

잘못된 방식으로 한 형식을 다른 형식으로 변환하려고 했습니다.  
  
### 이 오류를 해결하려면  
  
1.  할당이 올바른지 확인합니다.  
  
2.  단일 속성이나 값이 필요한 프로시저에 개체를 전달하지 않았는지 확인합니다.  
  
3.  식이 필요한 곳에 모듈 또는 프로젝트 이름을 사용하지 않았는지 확인합니다.
