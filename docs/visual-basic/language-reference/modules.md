# Modules (Visual Basic)
[!INCLUDE[vs2017banner](../../csharp/includes/vs2017banner.md)]

[!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]에서는 문자열 조작, 수학 계산 수행, 시스템 정보 가져오기, 파일 및 디렉터리 작업 수행 등의 일반적인 작업을 코드에서 간단하게 수행할 수 있도록 여러 모듈을 제공합니다.  다음 표에서는 [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]에서 제공되는 모듈을 나열합니다.  
  
|||  
|-|-|  
|<xref:Microsoft.VisualBasic.Constants>|기타 상수를 포함합니다.  이 상수는 코드의 모든 위치에서 사용할 수 있습니다.|  
|<xref:Microsoft.VisualBasic.ControlChars>|텍스트를 인쇄 및 표시하기 위한 상수 제어 문자를 포함합니다.|  
|<xref:Microsoft.VisualBasic.Conversion>|10진수를 다른 진수로, 숫자를 문자열로, 문자열을 숫자로, 데이터 형식을 다른 데이터 형식으로 변환하는 멤버를 포함합니다.|  
|<xref:Microsoft.VisualBasic.DateAndTime>|현재 날짜 또는 시간을 가져오거나, 날짜를 계산하거나, 날짜 또는 시간을 반환하거나, 날짜 또는 시간을 설정하거나, 프로세스 진행 기간을 계산하는 멤버를 포함합니다.|  
|<xref:Microsoft.VisualBasic.ErrObject>|런타임 오류 및 오류를 발생시키거나 지우는 메서드에 대한 정보를 포함합니다.|  
|<xref:Microsoft.VisualBasic.FileSystem>|파일, 디렉터리 또는 폴더 및 시스템 작업을 수행하는 멤버를 포함합니다.|  
|<xref:Microsoft.VisualBasic.Financial>|재무 관련 계산을 수행하는 데 사용되는 프로시저를 포함합니다.|  
|<xref:Microsoft.VisualBasic.Globals>|현재 스크립팅 엔진 버전에 대한 정보를 포함합니다.|  
|<xref:Microsoft.VisualBasic.Information>|배열 크기, 형식 이름 등의 정보를 반환, 테스트 또는 확인하는 멤버를 포함합니다.|  
|<xref:Microsoft.VisualBasic.Interaction>|개체, 응용 프로그램 및 시스템과 상호 작용하는 멤버를 포함합니다.|  
|<xref:Microsoft.VisualBasic.Strings>|문자열 서식 다시 지정, 문자열 검색, 문자열의 길이 가져오기 등의 문자열 작업을 수행하는 멤버를 포함합니다.|  
|<xref:Microsoft.VisualBasic.VBMath>|수학 연산을 수행하는 멤버를 포함합니다.|
