# <a name="modules-visual-basic"></a>모듈(Visual Basic)
[!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]문자열 조작, 시스템 정보 가져오기, 파일 및 디렉터리 작업을 수행 수학 계산 수행을 비롯 하 여 코드의 일반적인 작업을 간소화할 수 있도록 하는 몇 가지 모듈을 제공 합니다. 제공 하는 모듈을 나열 하는 다음 표에서 [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]합니다.  
  
|||  
|---|---|  
|<xref:Microsoft.VisualBasic.Constants></xref:Microsoft.VisualBasic.Constants>|기타 상수를 포함합니다. 코드에서 이러한 상수를 어디서 나 사용할 수 있습니다.|  
|<xref:Microsoft.VisualBasic.ControlChars></xref:Microsoft.VisualBasic.ControlChars>|인쇄 및 표시 텍스트에 대 한 상수 제어 문자를 포함 합니다.|  
|<xref:Microsoft.VisualBasic.Conversion></xref:Microsoft.VisualBasic.Conversion>|숫자를 문자열로, 문자열을 숫자로, 한 데이터 형식을 다른 형식으로 숫자를 다른 진수로 변환 하는 멤버를 포함 합니다.|  
|<xref:Microsoft.VisualBasic.DateAndTime></xref:Microsoft.VisualBasic.DateAndTime>|현재 날짜 또는 시간, 날짜 계산을 수행, 날짜 또는 시간 반환, 날짜 또는 시간을 설정 하거나 프로세스의 진행 기간 멤버가 포함 되어 있습니다.|  
|<xref:Microsoft.VisualBasic.ErrObject></xref:Microsoft.VisualBasic.ErrObject>|런타임 오류 및 메서드를 발생 시키거나 오류를 해결 하는 방법에 대 한 정보가 들어 있습니다.|  
|<xref:Microsoft.VisualBasic.FileSystem></xref:Microsoft.VisualBasic.FileSystem>|파일, 디렉터리 또는 폴더 및 시스템 작업을 수행 하는 멤버를 포함 합니다.|  
|<xref:Microsoft.VisualBasic.Financial></xref:Microsoft.VisualBasic.Financial>|재무 계산을 수행 하는 데 사용 되는 절차가 포함 됩니다.|  
|<xref:Microsoft.VisualBasic.Globals></xref:Microsoft.VisualBasic.Globals>|현재 스크립팅 엔진 버전에 대 한 정보를 포함합니다.|  
|<xref:Microsoft.VisualBasic.Information></xref:Microsoft.VisualBasic.Information>|반환, 테스트 또는 배열 크기, 형식 이름 등의 정보를 확인 하는 멤버를 포함 합니다.|  
|<xref:Microsoft.VisualBasic.Interaction></xref:Microsoft.VisualBasic.Interaction>|포함 개체, 응용 프로그램 및 시스템과 상호 작용 하는 멤버입니다.|  
|<xref:Microsoft.VisualBasic.Strings></xref:Microsoft.VisualBasic.Strings>|문자열은 문자열의 길이 가져오는, 문자열 검색 문자열을 다시 포맷 하는 등의 작업을 수행 하는 멤버를 포함 합니다.|  
|<xref:Microsoft.VisualBasic.VBMath></xref:Microsoft.VisualBasic.VBMath>|포함 된 멤버에는 수학 연산을 수행 합니다.|
