# <a name="xml-axis-properties-visual-basic"></a>XML 축 속성(Visual Basic)
이 섹션의 항목에 XML 축 속성의 구문은 문서화 [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)]합니다. XML 축 속성 쉽게 XML 코드에서 직접 액세스할 수 있습니다.  
  
## <a name="in-this-section"></a>단원 내용  
  
|항목|설명|  
|-----------|-----------------|  
|[XML Attribute 축 속성](../../../visual-basic/language-reference/xml-axis/xml-attribute-axis-property.md)|속성을 액세스 하는 방법을 설명는 <xref:System.Xml.Linq.XElement> 개체입니다.|  
|[XML Child 축 속성](../../../visual-basic/language-reference/xml-axis/xml-child-axis-property.md)|자식에 액세스 하는 방법을 설명는 <xref:System.Xml.Linq.XElement> 개체입니다.|  
|[XML Descendant 축 속성](../../../visual-basic/language-reference/xml-axis/xml-descendant-axis-property.md)|하위 항목에 액세스 하는 방법을 설명는 <xref:System.Xml.Linq.XElement> 개체입니다.|  
|[확장명 인덱서 속성](../../../visual-basic/language-reference/xml-axis/extension-indexer-property.md)|컬렉션의 개별 요소에 액세스 하는 방법을 설명 <xref:System.Xml.Linq.XElement> 또는 <xref:System.Xml.Linq.XAttribute> 개체입니다.|  
|[XML 값 속성](../../../visual-basic/language-reference/xml-axis/xml-value-property.md)|컬렉션의 첫 번째 요소 값에 액세스 하는 방법에 설명 <xref:System.Xml.Linq.XElement> 또는 <xref:System.Xml.Linq.XAttribute> 개체입니다.|
