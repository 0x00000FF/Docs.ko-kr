# XML Axis Properties (Visual Basic)
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

이 단원의 항목에서는 [!INCLUDE[vbprvb](../../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)] XML 축 속성의 구문을 보여 줍니다.  XML 축 속성을 사용하여 코드에서 XML에 직접 액세스할 수 있습니다.  
  
## 단원 내용  
  
|항목|설명|  
|--------|--------|  
|[XML Attribute Axis Property](../../../visual-basic/language-reference/xml-axis/xml-attribute-axis-property.md)|<xref:System.Xml.Linq.XElement> 개체의 특성에 액세스하는 방법을 설명합니다.|  
|[XML Child Axis Property](../../../visual-basic/language-reference/xml-axis/xml-child-axis-property.md)|<xref:System.Xml.Linq.XElement> 개체의 자식에 액세스하는 방법을 설명합니다.|  
|[XML Descendant Axis Property](../../../visual-basic/language-reference/xml-axis/xml-descendant-axis-property.md)|<xref:System.Xml.Linq.XElement> 개체의 하위 항목에 액세스하는 방법을 설명합니다.|  
|[Extension Indexer Property](../../../visual-basic/language-reference/xml-axis/extension-indexer-property.md)|<xref:System.Xml.Linq.XElement> 또는 <xref:System.Xml.Linq.XAttribute> 개체의 컬렉션에 있는 개별 요소에 액세스하는 방법을 설명합니다.|  
|[XML Value Property](../../../visual-basic/language-reference/xml-axis/xml-value-property.md)|<xref:System.Xml.Linq.XElement> 또는 <xref:System.Xml.Linq.XAttribute> 개체의 컬렉션에 있는 첫 번째 요소의 값에 액세스하는 방법을 설명합니다.|
