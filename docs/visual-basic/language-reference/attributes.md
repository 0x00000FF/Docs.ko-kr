# Attributes (Visual Basic)
[!INCLUDE[vs2017banner](../../visual-basic/developing-apps/includes/vs2017banner.md)]

[!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)]에서는 개체가 비관리 코드와 상호 운용될 수 있도록 하는 여러 특성을 제공하고 모듈 이름을 사용하지 않고 모듈 멤버에 액세스할 수 있는 특성 하나를 제공합니다.  다음 표에는 [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)]에 사용되는 특성이 나와 있습니다.  
  
|||  
|-|-|  
|<xref:Microsoft.VisualBasic.ComClassAttribute>|메타데이터를 추가하라는 명령을 컴파일러에 내려 클래스를 COM 개체로 노출할 수 있도록 합니다.|  
|<xref:Microsoft.VisualBasic.HideModuleNameAttribute>|모듈에 필요한 한정만 사용하여 모듈 멤버에 액세스할 수 있도록 합니다.|  
|<xref:Microsoft.VisualBasic.VBFixedArrayAttribute>|구조체 또는 지역 변수가 아닌 변수의 배열을 고정 길이 배열로 처리해야 함을 나타냅니다.|  
|<xref:Microsoft.VisualBasic.VBFixedStringAttribute>|문자열을 고정 길이인 것처럼 처리해야 함을 나타냅니다.|
