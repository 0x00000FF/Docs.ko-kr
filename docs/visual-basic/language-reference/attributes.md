# <a name="attributes-visual-basic"></a>특성(Visual Basic)
Visual Basic 개체 비관리 코드와 상호 운용 될 수 있도록 하는 여러 특성과 모듈 멤버에는 모듈 이름 없이 액세스할 수 있도록 하나의 특성을 제공 합니다. 다음 표에서 Visual Basic에서 사용 된 특성을 나열 합니다.  
  
|||  
|---|---|  
|<xref:Microsoft.VisualBasic.ComClassAttribute>|COM 개체로 노출 해야 하는 클래스를 허용 하는 메타 데이터를 추가 하도록 컴파일러에 지시 합니다.|  
|<xref:Microsoft.VisualBasic.HideModuleNameAttribute>|모듈 멤버에 모듈에 필요한 자격만을 사용 하 여 액세스할 수 있도록 합니다.|  
|<xref:Microsoft.VisualBasic.VBFixedArrayAttribute>|구조 또는 로컬이 아닌 변수의 배열을 고정 길이 배열로 처리 해야 함을 나타냅니다.|  
|<xref:Microsoft.VisualBasic.VBFixedStringAttribute>|문자열을 고정 길이인 것 처럼 처리 해야 함을 나타냅니다.|
