# Queries (Visual Basic)
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

[!INCLUDE[vbprvb](../../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)]을 사용하면 코드에서 [!INCLUDE[vbteclinqext](../../../csharp/getting-started/includes/vbteclinqext-md.md)] 식을 만들 수 있습니다.  
  
## 단원 내용  
 [Aggregate Clause](../../../visual-basic/language-reference/queries/aggregate-clause.md)  
 컬렉션에 하나 이상의 집계 함수를 적용하는 `Aggregate` 절에 대해 설명합니다.  
  
 [Distinct Clause](../../../visual-basic/language-reference/queries/distinct-clause.md)  
 현재 범위 변수의 값을 제한하여 쿼리 결과에서 중복 값을 제거하는 `Distinct` 절에 대해 설명합니다.  
  
 [From Clause](../../../visual-basic/language-reference/queries/from-clause.md)  
 쿼리의 컬렉션과 범위 변수를 지정하는 `From` 절에 대해 설명합니다.  
  
 [Group By 절](../../../visual-basic/language-reference/queries/group-by-clause.md)  
 쿼리 결과의 요소를 그룹화하며, 각 그룹에 집계 함수를 적용하는 데 사용할 수 있는 `Group By` 절에 대해 설명합니다.  
  
 [Group Join Clause](../../../visual-basic/language-reference/queries/group-join-clause.md)  
 두 개의 컬렉션을 단일 계층적 컬렉션으로 결합하는 `Group Join` 절에 대해 설명합니다.  
  
 [Join Clause](../../../visual-basic/language-reference/queries/join-clause.md)  
 두 개의 컬렉션을 단일 컬렉션으로 결합하는 `Join` 절에 대해 설명합니다.  
  
 [Let Clause](../../../visual-basic/language-reference/queries/let-clause.md)  
 값을 계산하고 이 값을 쿼리의 새 변수에 할당하는 `Let` 절에 대해 설명합니다.  
  
 [Order By Clause](../../../visual-basic/language-reference/queries/order-by-clause.md)  
 쿼리의 열 정렬 순서를 지정하는 `Order By` 절에 대해 설명합니다.  
  
 [Select Clause](../../../visual-basic/language-reference/queries/select-clause.md)  
 쿼리의 범위 변수 집합을 선언하는 `Select` 절에 대해 설명합니다.  
  
 [Skip Clause](../../../visual-basic/language-reference/queries/skip-clause.md)  
 컬렉션에서 지정된 수의 요소를 건너뛴 다음 나머지 요소를 반환하는 `Skip` 절에 대해 설명합니다.  
  
 [Skip While Clause](../../../visual-basic/language-reference/queries/skip-while-clause.md)  
 지정된 조건이 `true`이면 컬렉션에 있는 요소를 무시하고 나머지 요소를 반환하는 `Skip While` 절에 대해 설명합니다.  
  
 [Take Clause](../../../visual-basic/language-reference/queries/take-clause.md)  
 컬렉션 시작 위치에서 지정된 수의 연속 요소를 반환하는 `Take` 절에 대해 설명합니다.  
  
 [Take While Clause](../../../visual-basic/language-reference/queries/take-while-clause.md)  
 지정된 조건이 `true`이면 컬렉션에 있는 요소를 포함하고 나머지 요소를 무시하는 `Take While` 절에 대해 설명합니다.  
  
 [Where Clause](../../../visual-basic/language-reference/queries/where-clause.md)  
 쿼리의 필터링 조건을 지정하는 `Where` 절에 대해 설명합니다.
