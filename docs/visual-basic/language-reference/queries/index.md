# <a name="queries-visual-basic"></a>쿼리(Visual Basic)
Visual Basic를 사용 하 여 코드에서 [!INCLUDE[vbteclinqext](~/includes/vbteclinqext-md.md)] 식을 만들 수 있습니다.  
  
## <a name="in-this-section"></a>섹션 내용  
 [Aggregate 절](../../../visual-basic/language-reference/queries/aggregate-clause.md)  
 하나 이상의 집계 함수를 컬렉션에 적용 하는 `Aggregate` 절을 설명 합니다.  
  
 [Distinct 절](../../../visual-basic/language-reference/queries/distinct-clause.md)  
 현재 범위 변수의 값을 제한 하 여 쿼리 결과에서 중복 값을 제거 하는 `Distinct` 절에 대해 설명 합니다.  
  
 [From 절](../../../visual-basic/language-reference/queries/from-clause.md)  
 쿼리의 범위 변수와 컬렉션을 지정 하는 `From` 절에 대해 설명 합니다.  
  
 [Group By 절](../../../visual-basic/language-reference/queries/group-by-clause.md)  
 쿼리 결과의 요소를 그룹화 하 고 각 그룹에 집계 함수를 적용 하는 데 사용할 수 있는 `Group By` 절에 대해 설명 합니다.  
  
 [Group Join 절](../../../visual-basic/language-reference/queries/group-join-clause.md)  
 두 컬렉션을 단일 계층 구조 컬렉션으로 결합 하는 `Group Join` 절에 대해 설명 합니다.  
  
 [Join 절](../../../visual-basic/language-reference/queries/join-clause.md)  
 두 컬렉션을 단일 컬렉션으로 결합 하는 `Join` 절에 대해 설명 합니다.  
  
 [Let 절](../../../visual-basic/language-reference/queries/let-clause.md)  
 값을 계산 하 여 쿼리의 새 변수에 할당 하는 `Let` 절을 설명 합니다.  
  
 [Order By 절](../../../visual-basic/language-reference/queries/order-by-clause.md)  
 쿼리에서 열의 정렬 순서를 지정 하는 `Order By` 절에 대해 설명 합니다.  
  
 [Select 절](../../../visual-basic/language-reference/queries/select-clause.md)  
 쿼리의 범위 변수 집합을 선언 하는 `Select` 절에 대해 설명 합니다.  
  
 [Skip 절](../../../visual-basic/language-reference/queries/skip-clause.md)  
 컬렉션에서 지정 된 수의 요소를 무시 하 고 나머지 요소를 반환 하는 `Skip` 절을 설명 합니다.  
  
 [Skip While 절](../../../visual-basic/language-reference/queries/skip-while-clause.md)  
 지정 된 조건이 `true` 되는 동안 컬렉션의 요소를 무시 하 고 나머지 요소를 반환 하는 `Skip While` 절을 설명 합니다.  
  
 [Take 절](../../../visual-basic/language-reference/queries/take-clause.md)  
 컬렉션의 시작 부분에서 지정 된 개수의 연속 요소를 반환 하는 `Take` 절을 설명 합니다.  
  
 [Take While 절](../../../visual-basic/language-reference/queries/take-while-clause.md)  
 지정 된 조건이 `true` 되 고 나머지 요소를 무시 하는 한 컬렉션의 요소를 포함 하는 `Take While` 절에 대해 설명 합니다.  
  
 [Where 절](../../../visual-basic/language-reference/queries/where-clause.md)  
 쿼리의 필터링 조건을 지정 하는 `Where` 절에 대해 설명 합니다.
