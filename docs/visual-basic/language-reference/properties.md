# <a name="properties-visual-basic"></a>속성(Visual Basic)
이 페이지는 Visual Basic 모듈의 멤버는 속성을 나열 합니다. 특정 Visual Basic 개체의 구성원 인 다른 속성은 나열 [개체](../../visual-basic/language-reference/objects/index.md)합니다.  
  
## <a name="visual-basic-properties"></a>Visual Basic 속성  
  
|||  
|---|---|  
|<xref:Microsoft.VisualBasic.DateAndTime.DateString%2A>|사용 중인 시스템에 따라 현재 날짜를 나타내는 `String`을 반환하거나 설정합니다.|  
|<xref:Microsoft.VisualBasic.DateAndTime.Now%2A>|시스템 설정에 따라 현재 날짜와 시간이 들어 있는 `Date` 값을 반환합니다.|  
|<xref:Microsoft.VisualBasic.Globals.ScriptEngine%2A>|현재 사용 중인 런타임을 나타내는 `String`을 반환합니다.|  
|<xref:Microsoft.VisualBasic.Globals.ScriptEngineBuildVersion%2A>|현재 사용 중인 런타임의 빌드 버전 번호가 포함된 `Integer`를 반환합니다.|  
|<xref:Microsoft.VisualBasic.Globals.ScriptEngineMajorVersion%2A>|현재 사용 중인 런타임의 주 버전 번호가 포함된 `Integer`를 반환합니다.|  
|<xref:Microsoft.VisualBasic.Globals.ScriptEngineMinorVersion%2A>|현재 사용 중인 런타임의 부 버전 번호가 포함된 `Integer`를 반환합니다.|  
|<xref:Microsoft.VisualBasic.DateAndTime.TimeOfDay%2A>|시스템 설정에 따라 현재 시간이 들어 있는 `Date` 값을 반환하거나 설정합니다.|  
|<xref:Microsoft.VisualBasic.DateAndTime.Timer%2A>|자정 이후 경과된 초 수를 나타내는 `Double` 값을 반환합니다.|  
|<xref:Microsoft.VisualBasic.DateAndTime.TimeString%2A>|시스템 설정에 따라 현재 시간을 나타내는 `String` 값을 반환하거나 설정합니다.|  
|<xref:Microsoft.VisualBasic.DateAndTime.Today%2A>|사용 중인 시스템에 따라 현재 날짜가 들어 있는 `Date` 값을 반환하거나 설정합니다.|
