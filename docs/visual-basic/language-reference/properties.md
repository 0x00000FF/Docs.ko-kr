# <a name="properties-visual-basic"></a>속성(Visual Basic)

이 페이지에는 Visual Basic 모듈의 멤버인 속성이 나열 됩니다. 특정 Visual Basic 개체의 멤버인 다른 속성은 [개체](../../visual-basic/language-reference/objects/index.md)에 나열 됩니다.  
  
## <a name="visual-basic-properties"></a>Visual Basic 속성  
  
|||  
|---|---|  
|<xref:Microsoft.VisualBasic.DateAndTime.DateString%2A>|시스템에 따라 현재 날짜를 나타내는 `String` 값을 반환 하거나 설정 합니다.|  
|<xref:Microsoft.VisualBasic.DateAndTime.Now%2A>|시스템에 따라 현재 날짜 및 시간을 포함 하는 `Date` 값을 반환 합니다.|  
|<xref:Microsoft.VisualBasic.Globals.ScriptEngine%2A>|현재 사용 중인 런타임을 나타내는 `String`를 반환 합니다.|  
|<xref:Microsoft.VisualBasic.Globals.ScriptEngineBuildVersion%2A>|현재 사용 중인 런타임의 빌드 버전 번호를 포함 하는 `Integer`를 반환 합니다.|  
|<xref:Microsoft.VisualBasic.Globals.ScriptEngineMajorVersion%2A>|현재 사용 중인 런타임의 주 버전 번호를 포함 하는 `Integer`를 반환 합니다.|  
|<xref:Microsoft.VisualBasic.Globals.ScriptEngineMinorVersion%2A>|현재 사용 중인 런타임의 부 버전 번호를 포함 하는 `Integer`를 반환 합니다.|  
|<xref:Microsoft.VisualBasic.DateAndTime.TimeOfDay%2A>|시스템에 따라 현재 시간을 포함 하는 `Date` 값을 반환 하거나 설정 합니다.|  
|<xref:Microsoft.VisualBasic.DateAndTime.Timer%2A>|자정 이후 경과 된 시간 (초)을 나타내는 `Double` 값을 반환 합니다.|  
|<xref:Microsoft.VisualBasic.DateAndTime.TimeString%2A>|시스템에 따라 현재 시간을 나타내는 `String` 값을 반환 하거나 설정 합니다.|  
|<xref:Microsoft.VisualBasic.DateAndTime.Today%2A>|시스템에 따라 현재 날짜를 포함 하는 `Date` 값을 반환 하거나 설정 합니다.|
