# <a name="special-characters-visual-basic-reference"></a>특수 문자 (Visual Basic 참조)

특수 문자는 앞는 프로그램 요소를 수정 하는 미리 정의 된, 상황에 맞는 문자입니다. Visual Basic에서는 다음과 같은 특수 문자를 지원합니다. 

- [$](interpolated.md), 보간된 문자열 문자입니다.
