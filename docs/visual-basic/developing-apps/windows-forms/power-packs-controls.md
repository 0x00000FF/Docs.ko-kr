# Visual Basic Power Packs Controls
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

Visual Basic Power Packs 컨트롤은 추가 Windows Forms 컨트롤이며  Visual Studio에는 포함되지 않습니다.  이러한 컨트롤을 [다운로드](http://go.microsoft.com/fwlink/?LinkId=321343)할 수는 있지만 다운로드 시에는 지원 없이 있는 그대로 제공됩니다.  
  
## 단원 내용  
 [Line and Shape Controls](../../../visual-basic/developing-apps/windows-forms/line-and-shape-controls-visual-studio.md)  
 Line 및 Shape 컨트롤을 소개하고 자세한 정보를 확인할 수 있는 링크를 제공합니다.  
  
 [DataRepeater Control](../../../visual-basic/developing-apps/windows-forms/datarepeater-control-visual-studio.md)  
 DataRepeater 컨트롤을 소개하고 추가 정보를 확인할 수 있는 링크를 제공합니다.
