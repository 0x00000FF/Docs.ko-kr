# Accessing the Keyboard (Visual Basic)
[!INCLUDE[vs2017banner](../../../../visual-basic/developing-apps/includes/vs2017banner.md)]

`My.Computer.Keyboard` 개체는 현재 어떤 키를 눌렀는지와 같은 키보드의 현재 상태에 액세스하기 위한 속성을 제공하며 활성 창에 키 입력을 보내기 위한 메서드를 제공합니다.  
  
## 작업  
 다음 표에서는 `My.Computer.Keyboard` 개체와 관련된 작업을 보여 주고 각 작업을 수행하는 방법을 보여 주는 항목에 대한 링크를 제공합니다.  
  
|To|참조|  
|--------|--------|  
|Caps Lock이 켜져 있는지 여부를 확인합니다.|<xref:Microsoft.VisualBasic.Devices.Keyboard.CapsLock%2A>|  
|Shift 키를 눌렀는지 여부를 확인합니다.|<xref:Microsoft.VisualBasic.Devices.Keyboard.ShiftKeyDown%2A>|  
|Alt 키를 눌렀는지 여부를 확인합니다.|<xref:Microsoft.VisualBasic.Devices.Keyboard.AltKeyDown%2A>|  
|Ctrl 키를 눌렀는지 여부를 확인합니다.|<xref:Microsoft.VisualBasic.Devices.Keyboard.CtrlKeyDown%2A>|  
|Num Lock이 켜져 있는지 여부를 확인합니다.|<xref:Microsoft.VisualBasic.Devices.Keyboard.NumLock%2A>|  
|Scroll Lock이 켜져 있는지 여부를 확인합니다.|<xref:Microsoft.VisualBasic.Devices.Keyboard.ScrollLock%2A>|  
|응용 프로그램을 시작하고 이 응용 프로그램에 키 입력을 보냅니다.|[How to: Start an Application and Send it Keystrokes \(Visual Basic\)](../../../../visual-basic/developing-apps/programming/computer-resources/how-to-start-an-application-and-send-it-keystrokes.md)|
