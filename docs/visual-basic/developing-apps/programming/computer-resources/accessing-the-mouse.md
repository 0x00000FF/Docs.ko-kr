# Accessing the Mouse (Visual Basic)
[!INCLUDE[vs2017banner](../../../../visual-basic/developing-apps/includes/vs2017banner.md)]

`My.Computer.Mouse` 개체를 사용하면 컴퓨터 마우스에 대한 정보를 볼 수 있습니다. 예를 들어, 마우스가 있는 경우 마우스 단추 수, 마우스 휠에 대한 세부 사항 등을 찾아볼 수 있습니다.  
  
## 설명  
 이 표에서는 `My.Computer.Mouse` 개체와 관련된 작업과 각 작업의 수행 방법을 설명하는 항목을 보여 줍니다.  
  
|To|참조|  
|--------|--------|  
|마우스에 스크롤 휠이 있는지 여부를 확인합니다.|<xref:Microsoft.VisualBasic.Devices.Mouse.WheelExists%2A>|  
|마우스 왼쪽 단추의 기능과 마우스 오른쪽 단추의 기능이 서로 교환되었는지 여부를 확인합니다.|<xref:Microsoft.VisualBasic.Devices.Mouse.ButtonsSwapped%2A>|  
|마우스 휠을 한 번 회전할 때 스크롤되는 범위를 설정합니다.|<xref:Microsoft.VisualBasic.Devices.Mouse.WheelScrollLines%2A>|
