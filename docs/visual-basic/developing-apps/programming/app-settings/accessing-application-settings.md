# Accessing Application Settings (Visual Basic)
[!INCLUDE[vs2017banner](../../../../csharp/includes/vs2017banner.md)]

이 단원에는 `My.Settings` 개체와 이 개체를 사용하여 수행할 수 있는 작업에 대해 설명하는 항목이 포함되어 있습니다.  
  
## My.Settings  
 `My.Settings` 개체의 속성을 사용하면 응용 프로그램의 설정에 액세스할 수 있습니다.  설정을 추가 하거나 제거 하려면 사용 하는  **설정** 의 창의  **프로젝트 디자이너**.  
  
 `My.Settings` 개체의 메서드를 사용하면 현재 사용자 설정을 저장하거나 사용자 설정을 마지막으로 저장된 값으로 되돌릴 수 있습니다.
