# How to: Compile and Run a Project in Visual Basic
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

[!INCLUDE[vsprvs](../../../csharp/includes/vsprvs-md.md)] IDE\(통합 개발 환경\)에서는 쉽게 프로젝트를 컴파일하고 그 결과로 생성된 응용 프로그램을 실행할 수 있습니다.  
  
 IDE에서 기본 제공되는 디버거를 사용해 프로젝트를 실행하면서 디버그할 수 있습니다.  디버깅에 대한 자세한 내용은 [Debugging Your Visual Basic Application](../../../visual-basic/developing-apps/debugging.md) 및 [편집하며 계속하기\(Visual Basic\)](/visual-studio/debugger/edit-and-continue-visual-basic)를 참조하세요.  
  
 [!INCLUDE[note_settings_general](../../../csharp/language-reference/compiler-messages/includes/note-settings-general-md.md)]  
  
### 현재 프로젝트를 컴파일하고 실행하려면  
  
-   [!INCLUDE[vsprvs](../../../csharp/includes/vsprvs-md.md)] IDE\(통합 개발 환경\) 내에서 F5 키를 누릅니다.  
  
     IDE가 프로젝트를 컴파일하고 [!INCLUDE[vsprvs](../../../csharp/includes/vsprvs-md.md)] 디버거 내에서 응용 프로그램을 실행합니다.  
  
### 메뉴에서 현재 프로젝트를 컴파일하고 실행하려면  
  
-   [!INCLUDE[vsprvs](../../../csharp/includes/vsprvs-md.md)] IDE의 **디버그** 메뉴에서 **디버깅 시작**을 선택합니다.  
  
     IDE가 프로젝트를 컴파일하고 [!INCLUDE[vsprvs](../../../csharp/includes/vsprvs-md.md)] 디버거 내에서 응용 프로그램을 실행합니다.  
  
### 디버그하지 않고 현재 프로젝트를 컴파일하고 실행하려면  
  
-   [!INCLUDE[vsprvs](../../../csharp/includes/vsprvs-md.md)] IDE에서 Ctrl\+F5를 누릅니다.  
  
     IDE가 프로젝트를 컴파일하고 응용 프로그램을 실행합니다.  
  
 명령줄 컴파일러를 사용하여 [!INCLUDE[vbprvb](../../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)] 코드를 컴파일 및 실행하는 방법에 대한 자세한 내용은 [명령줄에서 빌드](../../../visual-basic/reference/command-line-compiler/building-from-the-command-line.md)를 참조하세요.
