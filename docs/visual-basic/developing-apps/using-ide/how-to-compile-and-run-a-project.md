# <a name="how-to-compile-and-run-a-project-in-visual-basic"></a>방법: Visual Basic에서 프로젝트 컴파일 및 실행
[!INCLUDE[vsprvs](../../../csharp/includes/vsprvs_md.md)] IDE(통합 개발 환경)에서는 쉽게 프로젝트를 컴파일하고 그 결과로 생성된 응용 프로그램을 실행할 수 있습니다.  
  
 IDE에서 기본 제공되는 디버거를 사용해 프로젝트를 실행하면서 디버그할 수 있습니다. 디버깅 하는 방법에 대 한 자세한 내용은 참조 [Visual Basic 응용 프로그램 디버깅](../../../visual-basic/developing-apps/debugging.md) 및 [편집 하며 계속 하기 (Visual Basic)](https://docs.microsoft.com/visualstudio/debugger/edit-and-continue-visual-basic)합니다.  

[!INCLUDE[note_settings_general](../../../csharp/language-reference/compiler-messages/includes/note_settings_general_md.md)]  
  
### <a name="to-compile-and-run-the-current-project"></a>현재 프로젝트를 컴파일하고 실행하려면  
  
-   [!INCLUDE[vsprvs](../../../csharp/includes/vsprvs_md.md)] IDE(통합 개발 환경) 내에서 F5 키를 누릅니다.  
  
     IDE가 프로젝트를 컴파일하고 [!INCLUDE[vsprvs](../../../csharp/includes/vsprvs_md.md)] 디버거 내에서 응용 프로그램을 실행합니다.  
  
### <a name="to-compile-and-run-the-current-project-from-the-menu"></a>메뉴에서 현재 프로젝트를 컴파일하고 실행하려면  
  
-   **디버그** 의 메뉴는 [!INCLUDE[vsprvs](../../../csharp/includes/vsprvs_md.md)] IDE 선택 **디버깅 시작**합니다.  
  
     IDE가 프로젝트를 컴파일하고 [!INCLUDE[vsprvs](../../../csharp/includes/vsprvs_md.md)] 디버거 내에서 응용 프로그램을 실행합니다.  
  
### <a name="to-compile-and-run-the-current-project-without-debugging"></a>디버그하지 않고 현재 프로젝트를 컴파일하고 실행하려면  
  
-   [!INCLUDE[vsprvs](../../../csharp/includes/vsprvs_md.md)] IDE에서 Ctrl+F5를 누릅니다.  
  
     IDE가 프로젝트를 컴파일하고 응용 프로그램을 실행합니다.  
  
 컴파일 및 실행에 대 한 내용은 [!INCLUDE[vbprvb](../../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] 명령줄 컴파일러를 사용 하는 코드를 참조 하십시오 [명령줄에서 빌드](../../../visual-basic/reference/command-line-compiler/building-from-the-command-line.md)합니다.
