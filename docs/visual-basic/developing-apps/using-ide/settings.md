# <a name="visual-basic-settings"></a>Visual Basic 설정
Visual Basic 설정은 사용자 환경을 최적화 하 고 Visual Basic 개발자의 생산성을 최대화 하기 위해 설계 되었습니다. 일부인는 [Visual Studio에서 개발 설정 사용자 지정](http://msdn.microsoft.com/en-us/22c4debb-4e31-47a8-8f19-16f328d7dcd3) 기능을 사용자 지정 하 고에 Visual Studio 통합 개발 환경 (IDE) 설정을 저장할 수 있습니다.  
  
 포트 또는 다른 컴퓨터의 IDE 설정에 다시 로드할 수도 있습니다. 자세한 내용은 참조 [하는 방법: 컴퓨터 간에 공유 설정을](http://msdn.microsoft.com/en-us/1131fb10-35c1-42da-9cd8-91aa3235b882)합니다.  
  
## <a name="windows-and-views"></a>창 및 뷰  
  
|기능|기본적으로 표시?|노트|  
|---|---|---|    
|[클래스 뷰](https://docs.microsoft.com/visualstudio/ide/viewing-the-structure-of-code)|아니요|숨깁니다 **숨겨진 형식 및 멤버**|  
|[명령 창](https://docs.microsoft.com/visualstudio/ide/reference/command-window)|아니요||  
|**동적 도움말** 창|아니요|F1 키를 누르면 표시 되지 않습니다.|  
|[직접 실행 창](https://docs.microsoft.com/visualstudio/ide/reference/immediate-window)|예, 디버깅을 시작할 때.||  
|[개체 브라우저](https://docs.microsoft.com/visualstudio/ide/viewing-the-structure-of-code)|아니요|기본적으로 표시합니다.<br /><br /> 네임 스페이스<br />-공용 멤버<br />-상속 된 멤버|  
|[출력 창](https://docs.microsoft.com/visualstudio/ide/reference/output-window)|예, 빌드를 시작할 때입니다.||  
|[속성 창](https://docs.microsoft.com/visualstudio/ide/reference/properties-window)|예||  
|**솔루션 탐색기**|예||  
|**서버 탐색기**|아니요|데이터 연결 및 사용 가능한 시스템 서비스를 표시합니다.|  
|[시작 페이지](https://docs.microsoft.com/visualstudio/ide/customizing-the-start-page-for-visual-studio)|예|처음에 IDE를 시작할 때가 나타납니다. 사용자 지정할 수 있습니다.|  
|**작업 목록**|예 때 컴파일 오류 또는 경고가 존재 합니다.||  
|[팀 탐색기](http://msdn.microsoft.com/library/1372e56c-b34f-42c2-b72c-94b57620c75c)|예|팀 프로젝트에 액세스 하 고 Team Foundation의 응용 프로그램 수명 주기 관리 기능을 사용 하려면 팀 탐색기를 사용할 수 있습니다. 여기에 버전 제어, 빌드, 프로젝트 관리, 통합 된 보고서 및 대시보드가 포함 됩니다.|  
|[도구 상자](https://docs.microsoft.com/visualstudio/ide/reference/toolbox)|예|알파벳 순서로 컨트롤 및 구성 요소를 표시합니다.|  
  
 또한 다음 요소는 특정 동작을 선택할 경우 활성화 **Visual Basic 개발 설정을**합니다.  
  
## <a name="dialog-boxes"></a>대화 상자  
  
|기능|동작|
|---|---|  
|**새 프로젝트** 대화 상자|컴퓨터에 다른 플랫폼을 설치 하면 IDE의 대상된 플랫폼을 지정할 수 있는 콤보 상자가 있습니다.|  
|[옵션 대화 상자](https://docs.microsoft.com/visualstudio/ide/reference/options-dialog-box-visual-studio)|Visual Basic 수 있도록 간소화 된 **옵션** 페이지입니다. 클릭는 **모든 설정 표시** 확인란 옵션의 전체 목록을 볼 수 있습니다.|  
  
## <a name="keyboard"></a>키보드
