# Visual Basic IntelliSense Code Snippets
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

[!INCLUDE[vbprvb](../../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)]에는 응용 프로그램에 삽입할 수 있는 IntelliSense 코드 조각이라는 수백 개의 코드 조각으로 구성된 코드 라이브러리가 포함되어 있습니다.  각 조각에서는 사용자 지정 예외 만들기, 전자 메일 메시지 보내기 또는 원 그리기와 같은 프로그래밍 작업을 완벽하게 수행합니다.  이러한 조각은 마우스 클릭 몇 번만으로도 소스 코드에 삽입할 수 있습니다.  또한 비즈니스 요구 사항에 따라 사용자 고유의 조각을 만들어 라이브러리에 추가한 다음 필요한 경우에 사용할 수 있습니다.  자세한 내용은 [코드 조각](/visual-studio/ide/code-snippets)을 참조하십시오.  
  
## Visual Basic 사용자를 위한 생산성 도구  
 이러한 조각을 사용하면 코드 샘플을 찾는 데 소모되는 시간을 줄이고, 익숙하지 않은 기능의 사용 방법을 익히는 데 필요한 시간을 줄이며, 코드의 재사용을 지원함으로써 생상성을 향상시킬 수 있습니다.  
  
 코드 라이브러리에서는 다음과 같은 기능을 지원합니다.  
  
-   조각 중 하나를 **코드 편집기**에 삽입할 수 있습니다.  
  
-   프로젝트에서 다시 사용할 수 있는 새 작업을 만들 수 있습니다.  
  
-   작업 그룹 및 동료와 공유할 새 작업을 만들 수 있습니다.  
  
-   이러한 작업을 편집할 수 있습니다.  
  
-   타사에서 더 많은 작업을 다운로드할 수 있습니다.  
  
 이러한 코드 블록은 [!INCLUDE[vsprvs](../../../csharp/includes/vsprvs-md.md)] 전체에서 사용할 수 있습니다.  
  
-   **코드 편집기**의 바로 가기 메뉴를 통해 이러한 코드 블록을 추가할 수 있습니다.  
  
-   XML 코드 파일을 끌 수 있습니다  **파일 탐색기** 소스 코드 파일을 끌어다 놓으십시오.
