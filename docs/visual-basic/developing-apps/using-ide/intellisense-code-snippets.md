# <a name="visual-basic-intellisense-code-snippets"></a>Visual Basic IntelliSense 코드 조각
[!INCLUDE[vbprvb](../../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]응용 프로그램에 삽입할 수 있는 IntelliSense 코드 조각 라고 하는 코드의 몇 가지 백 조각으로 구성 된 코드 라이브러리가 포함 되어 있습니다. 각 코드 조각은 사용자 지정 예외 만들기, 보내는 전자 메일 메시지 또는 원 그리기와 같은 완전 한 프로그래밍 작업을 수행 합니다. 몇 번의 마우스 클릭 하면 소스 코드에 코드 조각을 삽입할 수 있습니다. 또한 비즈니스 요구에 맞게를 라이브러리에 추가 하 고 필요할 때 사용 하 여 사용자 고유의 조각을 만들 수 있습니다. 자세한 내용은 [코드 조각](https://docs.microsoft.com/visualstudio/ide/code-snippets)을 참조하세요.  
  
## <a name="a-productivity-tool-for-visual-basic-users"></a>Visual Basic 사용자 생산성 도구  
 이러한 코드 조각은 코드 샘플을 찾고을 소비 되는 시간을 줄여, 알 수 없는 기능을 사용 하는 방법을 설명 하는 데 필요한 시간을 줄여 및 코드의 재사용을 지원 함으로써 생산성을 높여 줍니다.  
  
 코드 라이브러리는 다음과 같은 기능을 지원합니다.  
  
-   삽입 중에 코드 조각을 하나는 **코드 편집기**합니다.  
  
-   프로젝트에서 다시 사용할 수 있는 새 작업을 만듭니다.  
  
-   작업 그룹 및 동료와 공유 하는 새 작업을 만듭니다.  
  
-   작업을 편집 합니다.  
  
-   제&3; 자에서 더 많은 작업을 다운로드 합니다.  
  
 이러한 코드 블록 전체에서 사용할 수 있는 [!INCLUDE[vsprvs](../../../csharp/includes/vsprvs_md.md)]:  
  
-   바로 가기 메뉴를 통해 추가할 수는 **코드 편집기**합니다.  
  
-   XML 코드 파일을 끌어 놓을 수 **파일 탐색기** 소스 코드 파일에 있습니다.
