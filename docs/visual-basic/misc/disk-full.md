# 디스크가 꽉 찼습니다(Visual Basic).
디스크에 `Print`, `Write` 또는 `FileClose` 작업을 완료하거나 필요한 파일을 만들기에 충분한 공간이 없습니다.  
  
### 이 오류를 해결하려면  
  
-   일부 파일을 다른 디스크로 이동하거나 일부 파일을 삭제하여 디스크에 더 많은 공간을 만듭니다.
