# <a name="throw-statements-are-not-valid-in-the-immediate-window"></a>직접 실행 창에서는 'Throw' 문을 사용할 수 없습니다.
`Throw` 문은 소스 코드에서만 사용할 수 있습니다.  
  
 **오류 ID:** BC30725  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   디버깅 코드에서 `Throw` 문을 제거합니다.
