# <a name="method-cannot-contain-both-a-39try39-statement-and-an-39on-error39-or-39resume39-statement"></a>메서드는 모두 포함할 수 없습니다는 &#39;시도&#39; 문 및 &#39;On Error&#39; 또는 &#39;Resume&#39; 문
`Try` 문을 `On Error` 또는 `Resume`과 결합했습니다.  
  
 **오류 ID:** BC30544  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  `On Error` 또는 `Resume`을 제거합니다.
