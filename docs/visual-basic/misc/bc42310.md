# XML 주석 태그 &#39;include&#39;에는 &#39;&lt;attribute&gt;&#39; 특성이 있어야 합니다.
XML 주석 태그 'include'에는 \<attribute\> 특성이 있어야 합니다. XML 주석이 무시됩니다.  
  
 `<`include`>` 태그로 전달되는 `file` 또는 `path` 특성이 누락되었거나 불완전합니다.  
  
 **오류 ID:** BC42310  
  
### 이 오류를 해결하려면  
  
-   필요한 특성을 \<include\> 태그에 추가하거나 완성합니다.
