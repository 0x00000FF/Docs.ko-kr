# <a name="39ltmathfunction1gt39-is-not-declared"></a>&#39; &lt;mathfunction1&gt;&#39; 선언 되지 않았습니다
'\<mathfunction1 >'가 선언 되지 않았습니다. System.Math 클래스로 이동 되어 함수와 이제 이라는 '\<mathfunction2 >'입니다.  
  
 내장 된 여러 함수가 [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] 으로 이전 버전에서 이동 될는 <xref:System.Math?displayProperty=nameWithType> 네임 스페이스입니다. 그 결과 모든 프로그래밍 언어에서 더 일반적으로 기능을 사용할 수 있게 되었습니다.  
  
 **오류 ID:** BC30819  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   에 선언 된 메서드를 사용 하 여 <xref:System.Math?displayProperty=nameWithType>합니다.
