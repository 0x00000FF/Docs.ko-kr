# <a name="mathfunction1-is-not-declared"></a>'\<mathfunction1 >'가 선언 되지 않았습니다
'\<mathfunction1 >'가 선언 되지 않았습니다. System.Math 클래스 움직인 함수와 라고 '\<mathfunction2 >'입니다.  
  
 이전 버전의 Visual Basic에 내장 된 여러 함수가 옮겨지면는 <xref:System.Math?displayProperty=nameWithType> 네임 스페이스입니다. 그 결과 모든 프로그래밍 언어에서 더 일반적으로 기능을 사용할 수 있게 되었습니다.  
  
 **오류 ID:** BC30819  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   <xref:System.Math?displayProperty=nameWithType>에 선언된 메서드를 사용합니다.
