# <a name="39ltmathfunction1gt39-is-not-declared"></a>'&lt;mathfunction1&gt;' 선언 되지 않았습니다
'\<mathfunction1 > ' 선언 되지 않았습니다. 함수 System.Math 클래스 위치로 이동 되었습니다. 이제 라고 하 고 '\<mathfunction2 > '입니다.  
  
 여러 함수를 내장 된 [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] 이전 버전에서 이동 되었습니다는 <xref:System.Math?displayProperty=fullName>네임 스페이스.</xref:System.Math?displayProperty=fullName> 그 결과 모든 프로그래밍 언어에서 더 일반적으로 기능을 사용할 수 있게 되었습니다.  
  
 **오류 ID:** BC30819
