# <a name="39goto39-statements-are-not-valid-in-the-immediate-window"></a>&#39;GoTo&#39; 문은 직접 실행 창에 유효 하지 않습니다
`GoTo` 문은 분기를 수행하며 디버깅 컨텍스트에서는 허용되지 않습니다.  
  
 **오류 ID:** BC30133  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `GoTo` 직접 실행 **창에서** 문을 실행하지 않아야 합니다.
