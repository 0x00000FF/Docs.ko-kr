# <a name="label-ltlabelnamegt-is-not-defined"></a>레이블 '&lt;labelname&gt;' 정의 되지 않았습니다
줄 레이블 또는 줄 번호가 참조되었지만 참조 범위 내에서 정의되어 있지 않습니다. 레이블은 참조를 포함하는 프로시저 내에 있어야 합니다.  
  
 **오류 ID:** BC30132  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   줄 레이블과 참조가 동일한 프로시저에 있도록 코드를 재구성합니다.
