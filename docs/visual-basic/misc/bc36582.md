# <a name="too-many-arguments-to-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39"></a>확장 메서드 &#39; 인수가 너무 많습니다. &lt;methodname&gt;&#39;에서 정의 &#39;&lt; typename&gt;&#39;
확장 메서드 호출에서 제공된 인수 개수가 메서드에 대해 정의된 인수 개수를 초과합니다.  
  
 **오류 ID:** BC36582  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   확장 메서드의 정의를 참조하고 호출에서 불필요한 인수를 제거합니다.
