# <a name="try-must-have-at-least-one-39catch39-or-a-39finally39"></a>Try 하나 이상 &#39; 있어야 합니다. Catch &#39; 또는 &#39; 마지막으로 &#39;
`Try` 블록에서는 `Finally` 문 앞에 `Catch` 블록 또는 하나 이상의 `End Try` 블록이 있어야 합니다.  
  
 **오류 ID:** BC30030  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `Catch` 블록 내에 `Finally` 또는 `Try` 블록을 추가합니다.
