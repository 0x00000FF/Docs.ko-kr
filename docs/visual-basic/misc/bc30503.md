# <a name="properties-in-a-module-cannot-be-declared-specifier"></a>모듈의 속성을 선언할 수 없습니다 '\<지정자 >'
모듈 내의 속성에서 `MustOverride`, `NotOverridable`또는 `Overridable`등의 지정자를 사용하려고 했습니다.  
  
 **오류 ID:** BC30503  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 지정자를 제거합니다.
