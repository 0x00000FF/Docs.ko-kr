# <a name="39end-removehandler39-must-be-preceded-by-a-matching-39removehandler39-declaration"></a>&#39;RemoveHandler 종료&#39; 짝이 되는 뒤에 야 &#39;RemoveHandler&#39; 선언
`End RemoveHandler` 문이 해당 `RemoveHandler` 문 없이 발생합니다. `End RemoveHandler` 는 해당 `RemoveHandler` 문 뒤에 와야 합니다.  
  
 **오류 ID:** BC31125  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   앞의 `RemoveHandler` 문이 유효하고 철자가 올바른지 확인합니다.
