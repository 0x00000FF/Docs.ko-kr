# <a name="unable-to-parse-xml-lterrorgt"></a>XML 구문 분석 하지 못했습니다: &lt;오류&gt;
사용되는 XML 파일에 오류가 있습니다.  
  
 **오류 ID:** BC31023  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   XML이 잘 구성되었으며 파일이 손상되지 않았는지 확인합니다.
