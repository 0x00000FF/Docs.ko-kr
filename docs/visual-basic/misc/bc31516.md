# &#39;&lt;typename&gt;&#39; 형식은 &#39;Public&#39;으로 선언되지 않았으므로 특성으로 사용할 수 없습니다.
특성은 public이어야 합니다.  
  
 **오류 ID:** BC31516  
  
### 이 오류를 해결하려면  
  
1.  특성 클래스에 `Public` 한정자를 추가합니다.
