# <a name="type-lttypenamegt-cannot-be-used-as-an-attribute-because-it-is-not-declared-public"></a>형식 '&lt;typename&gt;' 'Public' 선언 되지 않았으므로 특성으로 사용할 수 없습니다.
특성은 public이어야 합니다.  
  
 **오류 ID:** BC31516  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  특성 클래스에 `Public` 한정자를 추가합니다.
