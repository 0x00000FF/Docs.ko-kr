# <a name="parameter-already-declared-with-name-39ltargumentnamegt39"></a>매개 변수 이름으로 이미 선언 &#39; &lt;argumentname&gt;&#39;
프로시저 선언에서 이름이 같은 두 개의 인수를 정의합니다. 프로시저 선언의 모든 인수 이름은 고유해야 합니다.  
  
 **오류 ID:** BC30237  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   인수 중 하나의 이름을 변경합니다.
