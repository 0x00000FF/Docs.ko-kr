# &#39;EndIf&#39; 문은 더 이상 지원되지 않습니다. 대신 &#39;End If&#39; 문을 사용합니다.
`EndIf` 문이 더 이상 지원되지 않습니다.  
  
 **오류 ID:** BC30826  
  
### 이 오류를 해결하려면  
  
-   `End If`를 사용하여 `If` 문을 종료합니다.
