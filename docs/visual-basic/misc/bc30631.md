# <a name="39get39-statement-must-end-with-a-matching-39end-get39"></a>&#39;가져오기&#39; 문은 짝이 되는 끝나야 &#39;End g&#39;
`Get` 속성 프로시저는 `End Get` 문으로 끝나야 합니다.  
  
 **오류 ID:** BC30631  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `Get` 속성 프로시저가 `End Get` 구문으로 끝나는지 확인합니다.
