# <a name="qualifiedmethod-is-not-accessible-in-this-context-because-it-is-modifier"></a>'\<qualifiedmethod >' 이므로이 컨텍스트에서 액세스할 수 없는 '\<한정자 >'
적용된 한정자 때문에 이 컨텍스트에서 액세스할 수 없는 메서드에 액세스하려고 했습니다.  
  
 **오류 ID:** BC30390  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 한정자를 제거합니다.
