# <a name="39addhandler39-declaration-must-end-with-a-matching-39end-addhandler39"></a>&#39;AddHandler&#39; 선언은 짝이 되는로 끝나야 &#39;끝 AddHandler&#39;
`AddHandler` 선언은 `End AddHandler` 문으로 끝나야 합니다.  
  
 **오류 ID:** BC31115  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `AddHandler` 선언이 `End AddHandler` 문으로 끝나는지 확인합니다.
