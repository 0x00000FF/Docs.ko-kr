# Const 선언에는 배열 이니셜라이저를 사용할 수 없습니다.
`Const` 선언에서 배열 이니셜라이저를 사용하려고 했습니다.  
  
 **오류 ID:** BC30445  
  
### 이 오류를 해결하려면  
  
-   묶는 중괄호를 제거하여 이니셜라이저를 유효한 식으로 변경합니다.
