# <a name="process-ltprocessnamegt-was-not-found"></a>프로세스&lt;processname&gt;' 찾을 수 없습니다
지정한 프로세스를 찾을 수 없습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  올바른 프로세스 이름을 지정했는지 확인합니다.
