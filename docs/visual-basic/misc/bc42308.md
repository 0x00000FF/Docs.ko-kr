# <a name="xml-comment-parameter-must-have-a-39name39-attribute"></a>XML 주석 매개 변수가 있어야는 &#39;이름&#39; 특성
XML 주석 매개 변수에는 'name' 특성이 있어야 합니다. XML 주석이 무시됩니다.  
  
 XML 태그에서 필요한 `name` 특성이 누락되었습니다.  
  
 **오류 ID:** BC42308  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   태그에 필요한 `name` 특성을 제공합니다.
