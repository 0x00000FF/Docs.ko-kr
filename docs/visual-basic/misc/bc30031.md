# <a name="fully-qualified-names-for-types-in-namespaces-cannot-be-longer-than-ltnumbergt-characters"></a>네임 스페이스의 형식에 대 한 정규화 된 이름 보다 길 수 없습니다 &lt;번호&gt; 문자
형식의 정규화된 이름은 지정한 문자 수를 초과할 수 없습니다.  
  
 **오류 ID:** BC30031  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   형식의 이름을 줄입니다.
