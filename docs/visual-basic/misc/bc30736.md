# <a name="expression-cannot-be-evaluated-at-this-time"></a>지금은 식을 계산할 수 없습니다.
현재 컨텍스트에서는 지정된 식을 계산할 수 없습니다.  
  
 **오류 ID:** BC30736  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   식의 구문을 변경합니다.
