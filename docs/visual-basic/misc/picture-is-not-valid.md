# <a name="picture-is-not-valid"></a>그림이 잘못되었습니다.
폼 또는 컨트롤의 `Picture` 속성에 비트맵, 아이콘 또는 Windows 메타파일 이외의 그래픽 형식을 할당하려고 했습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  로드 하려는 파일이 있는지 확인 하는 `Picture` 속성은에서 지원 되는 그래픽 파일 [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]합니다.
