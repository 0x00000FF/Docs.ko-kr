# XML 특성 값에 짝이 되는 닫는 큰따옴표가 필요합니다.
큰따옴표로 구분된 특성 값에 닫는 따옴표 문자가 없습니다.  
  
 **오류 ID:** BC31164  
  
### 이 오류를 해결하려면  
  
-   닫는 큰따옴표 문자를 추가하여 특성 값을 구분합니다.
