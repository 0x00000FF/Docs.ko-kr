# <a name="option-ltspecifiergt-statement-can-only-appear-once-per-file"></a>' 옵션 &lt;지정자&gt;' 문을 하나만 사용할 수 있습니다 파일당
`Option Compare`, `Option Explicit`및 `Option Strict` 문은 같은 소스 파일에서 각각 한 번만 사용할 수 있습니다.  
  
 **오류 ID:** BC30225  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   중복된 `Option` 문을 제거합니다.
