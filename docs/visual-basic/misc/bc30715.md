# <a name="39catch39-statements-are-not-valid-in-the-immediate-window"></a>&#39; Catch &#39; 문은 직접 실행 창에 유효 하지 않습니다.
`Catch` 문은 소스 코드에서만 사용할 수 있습니다.  
  
 **오류 ID:** BC30715  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   디버깅 코드에서 `Catch` 문을 제거합니다.
