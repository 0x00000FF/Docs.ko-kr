# XML 리터럴과 XML 축 속성은 이 어셈블리에서 사용되지 않으므로 이 디버깅 세션 중에 사용할 수 없습니다.
XML 리터럴 또는 XML 축 속성이 [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)] 기능의 XML을 사용할 수 없는 디버깅 세션 중에 **조사식** 또는 **즉시 실행** 창에서 참조되었습니다. 이는 [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)] 기능의 XML을 사용하지 않거나 릴리스 빌드인 어셈블리의 경우입니다.  
  
 **오류 ID:** BC31196  
  
### 이 오류를 해결하려면  
  
-   어셈블리의 디버그 빌드를 사용하여 새 디버깅 세션을 시작합니다.
