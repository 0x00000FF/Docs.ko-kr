# <a name="xml-literals-and-xml-axis-properties-are-not-available-during-this-debugging-session-because-they-are-not-used-in-this-assembly"></a>XML 리터럴과 XML 축 속성은 이 어셈블리에서 사용되지 않으므로 이 디버깅 세션 중에 사용할 수 없습니다.
XML 리터럴 또는 XML 축 속성에서 참조 되었습니다는 **조사식** 또는 **직접 실행** Visual Basic의 xml에서 기능 사용할 수 없는 디버깅 세션 중 창. 이 경우 Visual Basic 기능에서 XML을 사용 하지 않도록 하거나 릴리스 빌드는 어셈블리에 대 한 합니다.  
  
 **오류 ID:** BC31196  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   어셈블리의 디버그 빌드를 사용하여 새 디버깅 세션을 시작합니다.
