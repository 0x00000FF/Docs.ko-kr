# <a name="39ltmembergt39-implicitly-defined-for-39lteventnamegt39-cannot-shadow-a-39mustoverride39-method-in-the-base-ltclassgt-39ltclassnamegt39"></a>'&lt;멤버&gt;', 암시적으로 정의 대 한'&lt;eventname&gt;', 'MustOverride' 메서드는 기본을 숨길 수 없습니다 &lt;클래스&gt; '&lt;classname&gt;'
지정된 이벤트는 `MustOverride` 한정자로 선언된 메서드와 동일한 이름을 가진 멤버를 암시적으로 선언합니다.  
  
 **오류 ID:** BC31413  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   기본 클래스의 메서드에서 `MustOverride` 한정자를 제거하거나 속성 또는 메서드에 고유한 이름을 지정합니다.
