# <a name="ltmembergt-implicitly-defined-for-lteventnamegt-cannot-shadow-a-mustoverride-method-in-the-base-ltclassgt-ltclassnamegt"></a>'&lt;멤버&gt;'에 대 한 암시적으로 정의 된'&lt;eventname&gt;', 'MustOverride' 메서드를 자료에서 숨길 수 없습니다 &lt;클래스&gt; '&lt;classname&gt;'
지정된 이벤트는 `MustOverride` 한정자로 선언된 메서드와 동일한 이름을 가진 멤버를 암시적으로 선언합니다.  
  
 **오류 ID:** BC31413  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   기본 클래스의 메서드에서 `MustOverride` 한정자를 제거하거나 속성 또는 메서드에 고유한 이름을 지정합니다.
