# &#39;&lt;typename&gt;&#39;이 예약된 이름이므로 &#39;&lt;implementsclause&gt;&#39;에서 &#39;&lt;typename&gt;&#39;을 구현할 수 없습니다.
`Implements` 문이 예약된 이름을 지정했습니다.  
  
 **오류 ID:** BC31415  
  
### 이 오류를 해결하려면  
  
-   구현할 수 없는 인터페이스 멤버의 이름을 바꿉니다.
