# <a name="project-ltprojectnamegt-cannot-reference-project-ltprojectnamegt-because-ltprojectnamegt-directly-or-indirectly-references-ltprojectnamegt"></a>프로젝트 '&lt;projectname&gt;'프로젝트를 참조할 수 없습니다'&lt;projectname&gt;'하므로'&lt;projectname&gt;'직접 또는 간접적으로 참조'&lt;projectname &gt;'
프로젝트에 두 번째 프로젝트에 대한 참조가 포함되어 있습니다. 또한 두 번째 프로젝트에는 해당 프로젝트를 참조하는 프로젝트에 대한 참조가 포함되어 있습니다.  
  
 **오류 ID:** BC30761  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   참조 중 하나를 제거합니다.
