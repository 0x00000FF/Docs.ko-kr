# <a name="structures-cannot-have-39inherits39-statements"></a>구조체 가질 수 없으며 &#39;Inherits&#39; 문
클래스와 달리 구조체는 상속을 지원하지 않습니다.  
  
 **오류 ID:** BC30628  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   구조체에서 `Inherits` 문을 제거합니다.  
  
-   대신 클래스를 사용하도록 응용 프로그램을 다시 설계합니다.
