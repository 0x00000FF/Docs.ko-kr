# COM 클래스에서 파생된 관리되는 클래스는 런타임에 바인딩된 호출을 할 수 없습니다.
COM 클래스에서 파생된 관리되는 클래스에 런타임에 바인딩된 호출을 하려고 했습니다. 이러한 호출은 지원되지 않습니다.  
  
### 문제를 해결하려면  
  
-   초기 바인딩된 호출을 합니다.
