# <a name="unable-to-create-temp-file-in-path-ltfilenamegt-lterror-messagegt"></a>경로에 임시 파일을 만들 수 없습니다. '&lt;filename&gt;': &lt;오류 메시지&gt;
Visual Basic 컴파일러는 매니페스트를 사용 하 여 어셈블리를 생성 하려면 어셈블리 링커 (Al.exe, Alink 라고도)를 호출 합니다. 링커가 파일을 만들거나 메모리 내 리소스를 쓰는 동안 오류를 보고했습니다. 이는 구성 문제 때문일 수 있습니다.  
  
 **오류 ID:** BC30138  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  따옴표 붙은 오류 메시지를 확인 하 고 항목을 참조 하십시오 [Al.exe (어셈블리 링커)](../../framework/tools/al-exe-assembly-linker.md) 추가 설명과 권장 사항을 대 한 합니다.  
  
2.  오류가 계속 발생하면 해당 상황에 대한 정보를 수집하여 Microsoft 기술 지원 서비스에 알립니다.
