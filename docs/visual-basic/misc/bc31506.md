# <a name="lttypenamegt-cannot-be-used-as-an-attribute-because-it-is-declared-mustinherit"></a>'&lt;typename&gt;' 'm'으로 선언 되어 해당 특성으로 사용할 수 없습니다.
사용자 지정 특성 클래스는 `MustInherit`로 선언될 수 없습니다.  
  
 **오류 ID:** BC31506  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   사용자 지정 특성에서 `MustInherit` 한정자를 제거합니다.
