# /moduleassemblyname 옵션은 &#39;module&#39; 형식의 대상을 빌드할 때만 지정할 수 있습니다.
`/target` 옵션이 `module` 이외의 값으로 설정되었을 때 Visual Basic 컴파일러에 `/moduleassemblyname` 컴파일러 옵션이 전달되었습니다.  
  
 **오류 ID:** BC2030  
  
### 이 오류를 해결하려면  
  
1.  `/moduleassemblyname` 컴파일러 옵션을 제거하거나 `/target` 옵션을 `module`로 설정합니다.
