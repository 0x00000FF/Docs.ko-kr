# &#39;Structure&#39; 문은 짝이 되는 &#39;End Structure&#39;로 끝나야 합니다.
구조체는 `Structure` 문으로 시작하여 `End Structure` 구문으로 끝납니다.  
  
 **오류 ID:** BC30624  
  
### 이 오류를 해결하려면  
  
-   구조체 블록이 `End Structure` 문으로 끝나는지 확인합니다.
