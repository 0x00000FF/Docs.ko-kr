# <a name="39option-explicit39-can-be-followed-only-by-39on39-or-39off39"></a>&#39; 명시적 옵션 &#39; 수만 올 수 &#39; &#39; 또는 &#39; 해제 &#39;
`Option Explicit` 문은 `On` 또는 `Off`를 지정해야 합니다. 다른 모든 값이 올바르지 않습니다.  
  
 **오류 ID:** BC30640  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `Option Explicit On` 또는 `Option Explicit Off`를 지정합니다.
