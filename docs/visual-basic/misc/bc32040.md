# <a name="the-39ltkeywordgt39-keyword-is-used-to-overload-inherited-members-do-not-use-the-39ltkeywordgt39-keyword-when-overloading-39sub-new39"></a>&#39; &lt;키워드&gt;&#39; 키워드는 상속 된 멤버를 사용 하지 않는 &#39;&lt; 키워드&gt;&#39; 오버 로드할 때 키워드 &#39; 새 하위 &#39;
생성자가 `Overloads` 키워드를 사용하여 선언되었습니다.  
  
 Visual Basic은 생성자 상속 또는 오버로드를 지원하지 않습니다.  
  
 **오류 ID:** BC32040  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   모든 생성자 선언에서 `Overloads` 키워드를 제거합니다.
