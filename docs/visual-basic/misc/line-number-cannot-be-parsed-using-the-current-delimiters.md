# <a name="line-number-cannot-be-parsed-using-the-current-delimiters"></a>줄 \<수 > 현재 구분 기호를 사용 하 여 구문 분석할 수 없습니다
지정된 것과 다른 구분 기호를 사용하기 때문에 지정한 줄을 구문 분석할 수 없습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 줄을 올바르게 구문 분석할 수 있도록 `Delimiters` 를 조정하거나 줄을 처리하도록 예외 처리 코드를 삽입합니다.
