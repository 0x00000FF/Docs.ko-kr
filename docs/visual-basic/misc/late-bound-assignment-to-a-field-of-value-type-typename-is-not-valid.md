# 런타임에 바인딩된 식의 결과가 &#39;&lt;name&gt;&#39;일 경우 &#39;&lt;typename&gt;&#39; 값 형식의 필드에 대한 런타임에 바인딩된 할당을 사용할 수 없습니다.
시도한 런타임에 바인딩된 할당이 잘못되었습니다.  
  
### 이 오류를 해결하려면  
  
-   할당을 초기 바인딩으로 설정합니다.
