# <a name="ltpropertygt-parameters-cannot-be-declared-39byref39"></a>&lt;속성&gt; 매개 변수를 선언할 수 없습니다 &#39;ByRef&#39;
속성에 대한 매개 변수는 `ByVal` 한정자로 선언해야 합니다.  
  
 **오류 ID:** BC30651  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   속성 매개 변수를 `ByVal`로 정의합니다.
