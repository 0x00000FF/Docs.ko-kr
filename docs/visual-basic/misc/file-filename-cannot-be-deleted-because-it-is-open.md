# &#39;&lt;filename&gt;&#39; 파일은 열려 있으므로 삭제할 수 없습니다.
열려 있는 파일에서 삭제를 시도하고 있습니다.  
  
### 이 오류를 해결하려면  
  
-   삭제하기 전에 파일을 닫습니다.
