# <a name="expected-3939-for-xml-end-tag"></a>예상 &#39;/&#39; XML 끝 태그에
XML 리터럴의 XML 끝 태그에 필수 '/' 문자가 없습니다.  
  
 **오류 ID:** BC31167  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   XML 리터럴의 XML 끝 태그 시작에 '/' 문자를 추가합니다.
