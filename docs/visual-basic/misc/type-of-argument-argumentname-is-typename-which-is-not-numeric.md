# &#39;&lt;argumentname&gt;&#39;인수의 형식은 숫자가 아닌 &#39;&lt;typename&gt;&#39;입니다.
숫자 인수가 필요한 상황에서 숫자가 아닌 인수가 발생합니다.  
  
### 이 오류를 해결하려면  
  
-   인수가 숫자인지 확인합니다.
