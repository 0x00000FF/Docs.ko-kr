# <a name="39default39-cannot-be-combined-with-39ltspecifiergt39"></a>&#39; 기본 &#39; 함께 사용할 수 없는 &#39; &lt;지정자&gt;&#39;
이 컨텍스트에서 유효하지 않은 `Default` 또는 `Shared` 과 같은 지정자와 `Private`키워드를 결합하려고 했습니다.  
  
 **오류 ID:** BC30490  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `Default`를 제거합니다.  
  
     또는  
  
-   충돌하는 지정자를 제거합니다.
