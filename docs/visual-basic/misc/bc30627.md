# <a name="39option39-statements-must-precede-any-declarations-or-39imports39-statements"></a>&#39;옵션&#39; 문을 모든 선언 앞에 야 또는 &#39;Imports&#39; 문
`Option` 문이 소스 코드의 첫 번째 비주석 문이어야 합니다.  
  
 **오류 ID:** BC30627  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `Option` 문을 `Imports` 및 `Namespace` 문 바로 앞에 소스 코드의 맨 위로 이동합니다.
