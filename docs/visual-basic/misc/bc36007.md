# <a name="39end-using39-must-be-preceded-by-a-matching-39using39"></a>&#39;사용 하 여 종료&#39; 짝이 되는 뒤에 야 &#39;사용&#39;
`End Using` 문이 앞에서 일치하는 `Using` 선언 없이 나타납니다.  
  
 **오류 ID:** BC36007  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `End Using` 문이 중복되는 경우 제거합니다.  
  
-   누락 된 제공 [문을 사용 하 여](../../visual-basic/language-reference/statements/using-statement.md) 없는 경우.  
  
-   `End Using` 문을 코드의 적절한 위치로 이동합니다.
