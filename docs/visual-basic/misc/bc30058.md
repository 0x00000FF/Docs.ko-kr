# <a name="statements-and-labels-are-not-valid-between-39select-case39-and-first-39case39"></a>문 및 레이블을 사이의 올바르지 않습니다. &#39; 대/소문자 &#39;를 선택 합니다. 첫 번째 &#39; 및 대/소문자 &#39;
`Select` 또는 `Select Case` 문과 첫 번째 `Case` 문 사이에 주석이 아닌 문을 사용했습니다.  
  
 **오류 ID:** BC30058  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   간섭 문이 주석인 경우 앞에 주석 구분 기호(`'` 또는 `REM`)를 추가합니다. 그렇지 않으면 문을 이동하거나 삭제합니다.
