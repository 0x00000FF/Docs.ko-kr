# <a name="operator-is-not-valid-for-ltname1gt-and-ltname2gt"></a>연산자에 대해 올바르지 않습니다. '&lt;name1&gt;'및'&lt;name2&gt;'
연산자가 작업 중인 형식에 적합하지 않습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  연산자가 작업 중인 형식에 적합한지 확인합니다.
