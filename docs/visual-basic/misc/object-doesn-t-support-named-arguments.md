# 개체가 지정된 인수를 지원하지 않습니다.
메서드가 명명된 인수를 지원하지 않는 개체에 액세스하려고 했습니다.  
  
### 이 오류를 해결하려면  
  
-   이 개체의 메서드에 액세스할 때 인수를 올바른 위치에 지정합니다. 인수의 위치와 형식에 대한 자세한 내용은 개체 설명서를 참조하세요.
