# <a name="conversion-from-lttype1gt-to-lttype2gt-cannot-occur-in-a-constant-expression"></a>변환에서 '&lt;type1&gt;'to'&lt;type2&gt;'는 상수 식에 발생할 수 없습니다.
`Const` 문의 초기화 식은 상수의 선언된 형식으로 변환할 수 없는 데이터 형식으로 평가됩니다.  
  
 **오류 ID:** BC30060  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  상수를 상수에 대해 선언된 형식으로 변환할 수 있는 데이터 형식의 식을 사용하여 초기화합니다.
