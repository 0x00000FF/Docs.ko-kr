# <a name="39new39-cannot-be-used-on-class-39ltclassnamegt39-because-it-contains-a-39mustoverride39-member-that-has-not-been-overridden"></a>&#39; 새로 만들기 &#39; 클래스 &#39;에서 사용할 수 없습니다. &lt;classname&gt;&#39; 포함 하기 때문에 &#39; MustOverride &#39; 재정의 되지 않은 멤버
재정의되지 않은 `New` 멤버가 있는 클래스에서 `MustOverride` 를 사용하려고 했습니다.  
  
 **오류 ID:** BC30376  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `New` 문을 제거합니다.
