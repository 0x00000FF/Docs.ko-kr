# <a name="new-cannot-be-used-on-class-ltclassnamegt-because-it-contains-a-mustoverride-member-that-has-not-been-overridden"></a>'New' 클래스에서 사용할 수 없습니다&lt;classname&gt;' 재정의 되지 않은 'MustOverride' 멤버인 있기 때문에
재정의되지 않은 `New` 멤버가 있는 클래스에서 `MustOverride` 를 사용하려고 했습니다.  
  
 **오류 ID:** BC30376  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `New` 문을 제거합니다.
