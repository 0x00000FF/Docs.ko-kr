# &#39;&lt;classname&gt;&#39; 클래스에 재정의되지 않은 &#39;MustOverride&#39; 멤버가 있으므로 &#39;New&#39;는 이 클래스에 사용할 수 없습니다.
재정의되지 않은 `MustOverride` 멤버가 있는 클래스에서 `New`를 사용하려고 했습니다.  
  
 **오류 ID:** BC30376  
  
### 이 오류를 해결하려면  
  
-   `New` 문을 제거합니다.
