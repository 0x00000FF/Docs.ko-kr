# <a name="registry-key-ltkeynamegt-could-not-be-found"></a>레지스트리 키 '&lt;keyname&gt;' 찾을 수 없습니다
레지스트리 키를 찾을 수 없습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  키 이름의 철자를 확인합니다.  
  
2.  키에 액세스할 수 있는 보안 권한이 있는지 확인합니다.
