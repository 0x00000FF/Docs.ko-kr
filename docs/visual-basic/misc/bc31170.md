# <a name="character-39character39-0xlthexadecimalgt-is-not-allowed-in-an-xml-name"></a>문자 &#39;문자&#39; (0 x&lt;16 진수&gt;) XML 이름에 허용 되지 않습니다
지원되지 않는 문자가 XML 리터럴 또는 XML 축 속성의 이름에 포함되었습니다.  
  
 **오류 ID:** BC31170  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   XML 리터럴 또는 XML 축 속성의 이름에서 지원되지 않는 문자를 제거합니다.
