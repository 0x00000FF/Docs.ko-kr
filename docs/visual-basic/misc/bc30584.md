# <a name="lttypegt-cannot-be-inherited-more-than-once"></a>'&lt;형식&gt;' 한 번 이상 상속할 수 없습니다.
`Inherits` 문이 중복되었습니다.  
  
 **오류 ID:** BC30584  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   중복 `Inherits` 문을 제거합니다.
