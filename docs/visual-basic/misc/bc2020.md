# <a name="the-maximum-number-of-errors-has-been-exceeded"></a>최대 오류 개수를 초과했습니다.
컴파일러 최대 오류 개수인 100개에 도달했습니다.  
  
 **오류 ID:** BC2020  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   각 오류를 차례로 디버그하고 해결하여 생성된 오류 수를 줄입니다.
