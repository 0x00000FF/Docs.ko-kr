# <a name="39notoverridable39-cannot-be-specified-on-methods-that-do-not-override-another-method"></a>&#39; NotOverridable &#39; 다른 메서드를 재정의 하지 않는 메서드에 지정할 수 없습니다.
속성 및 메서드는 기본적으로 `NotOverridable` 입니다. `NotOverridable` 한정자는 다른 속성이나 메서드를 재정의하는 메서드에만 사용할 수 있습니다.  
  
 **오류 ID:** BC31088  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   다른 멤버를 재정의하지 않는 속성과 메서드에서 `NotOverridable` 한정자를 제거합니다.
