# <a name="the-path-has-not-been-set"></a>경로가 설정되지 않았습니다.
필요한 경로를 제공하지 않고 파일을 이동 및 복사하는 등 파일 작업이 시도되었습니다. 이는 경로를 잘못 구문 분석하면 발생할 수 있습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   파일 경로가 제공되었고 올바른지 확인합니다.
