# <a name="this-operation-can-only-be-done-when-the-file-is-closed"></a>이 작업은 파일이 닫혀 있을 때만 수행할 수 있습니다.
파일이 열려 있는데 닫혀 있는 파일에서만 가능한 작업을 수행하려고 했습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   파일을 닫고 작업을 다시 시도합니다.
