# XML 주석 형식 매개 변수에는 &#39;name&#39; 특성이 있어야 함
XML 주석 형식 매개 변수에는 'name' 특성이 있어야 합니다. XML 주석이 무시됩니다.  
  
 XML 주석 태그에 없거나 완전하지 않은 `name` 특성이 필요합니다.  
  
 **오류 ID:** BC42318  
  
### 이 오류를 해결하려면  
  
-   필요한 `name` 특성을 태그에 추가하거나 완성합니다.
