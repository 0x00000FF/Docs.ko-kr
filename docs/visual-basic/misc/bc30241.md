# <a name="named-argument-expected"></a>명명된 인수가 필요합니다.
프로시저 호출의 인수 목록이 이름으로 인수를 전달한 다음 위치로 인수를 전달합니다. 모든 위치 인수는 프로시저 호출에서 이름으로 전달된 인수 앞에 와야 합니다.  
  
 **오류 ID:** BC30241  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   위치로 전달된 인수가 이름으로 전달된 인수 앞에 오도록 인수 목록을 다시 작성합니다.
