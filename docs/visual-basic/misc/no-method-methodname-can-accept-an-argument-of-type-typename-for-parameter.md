# <a name="no-method-methodname-can-accept-an-argument-of-type-typename-for-parameter-parametername"></a>메서드가 없는 '\<methodname >' 형식의 인수를 사용할 수 '\<typename >' 매개 변수에 대 한 '\<parametername >'
메서드 호출에 유효하지 않은 형식의 인수가 포함되어 있습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  인수의 형식을 확인합니다.  
  
2.  메서드의 매개 변수 형식과 인수 형식을 일치시켜 계약 내에 있는지 확인합니다. 필요에 따라 형식을 변경합니다.
