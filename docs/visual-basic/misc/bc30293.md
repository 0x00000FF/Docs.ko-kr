# <a name="lterrorgt-ltstructurename1gt-contains-ltstructurename2gt"></a>&lt;오류&gt;: '&lt;structurename1&gt;'포함'&lt;structurename2&gt;'
구조체가 자신을 해당 멤버 중 하나로 지정하는 경우 순환 구조 정의가 발생합니다.  
  
 **오류 ID:** BC30293  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   구조체 멤버의 이름을 변경합니다.
