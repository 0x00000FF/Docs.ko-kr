# <a name="39module39-statement-must-end-with-a-matching-39end-module39"></a>&#39;모듈&#39; 문은 일치 하는 끝나야 합니다. &#39;End&#39;
모듈 블록 끝나야는 `End Module` 문입니다.  
  
 **오류 ID:** BC30625  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `End Module` 블록 끝에 `Module` 문을 추가합니다.
