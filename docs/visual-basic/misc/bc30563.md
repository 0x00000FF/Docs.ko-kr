# <a name="39ltnamegt39-is-ambiguous-in-the-application-objects-39ltlistgt39"></a>'&lt;이름&gt;'모호 합니다. 응용 프로그램 개체에 '&lt;목록&gt;'
모호한 이름을 제공했으므로 다른 이름과 충돌합니다. [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] 컴파일러에는 충돌 해결 규칙이 없으므로 직접 이름을 명확하게 해야 합니다. 오류 메시지에 제공된 목록에서 이름이 모호한 응용 프로그램 개체를 자세히 설명합니다.  
  
 **오류 ID:** BC30563  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   이름을 정규화합니다.
