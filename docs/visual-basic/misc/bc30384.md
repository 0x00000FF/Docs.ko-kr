# <a name="39try39-must-end-with-a-matching-39end-try39"></a>&#39; Try &#39; 일치 하는 &#39;로 끝나야 합니다. End Try &#39;
`Try`시작 하는 데 사용 되는 `Try` 차단; 되므로 일치 하는 블록의 시작 부분에만 나타날 수 `End Try` 문 블록을 종료 합니다. 중복된 `Try`가 있거나, `Try` 를 사용하여 `Finally`블록을 종료하지 않았습니다.  
  
 **오류 ID:** BC30384  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  잘못 사용된 `Try`를 찾아 제거하거나, 짝이 되는 `End Try`를 사용하여 블록을 종료합니다.
