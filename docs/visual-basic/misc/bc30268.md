# <a name="39ltdeclaration1gt39-cannot-override-39ltdeclaration2gt39-because-it-is-declared-39shared39"></a>&#39;&lt;declaration1&gt; &#39; 재정의할 수 없습니다. &#39; &lt;declaration2&gt; &#39; 선언 되어 있어 &#39;공유&#39;
프로시저나 속성 선언이 동일한 이름의 상속된 요소를 재정의하려고 하지만 상속된 요소가 `Shared`로 지정되었습니다. 공유 요소는 클래스 인스턴스와 연결되어 있지 않으므로 재정의할 수 없습니다.  
  
 **오류 ID:** BC30268  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   상속된 요소에서 `Shared` 키워드를 제거하거나 재정의 선언을 제거합니다.
