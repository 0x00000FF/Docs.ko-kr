# <a name="39inherits39-can-appear-only-once-within-a-39class39-statement-and-can-only-specify-one-class"></a>&#39; 상속 &#39; 내에서 한 번만 나타날 수는 &#39; 클래스 &#39; 문을 있으며 하나의 클래스만 지정할 수 있습니다
둘 이상의 `Inherits` 문이 같은 클래스에 나타나거나 `Inherits` 문에서 둘 이상의 클래스를 지정합니다. 클래스는 하나의 기본 클래스에서만 상속할 수 있습니다.  
  
 **오류 ID:** BC30121  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   불필요한 `Inherits` 문을 제거하고 나머지 `Inherits` 문이 하나의 기본 클래스만 지정하는지 확인합니다.
