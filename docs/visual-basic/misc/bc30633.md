# <a name="39set39-statement-must-end-with-a-matching-39end-set39"></a>&#39; 설정 &#39; 문은 짝이 되는 &#39;로 끝나야 합니다. 최종 설정 &#39;
`Set` 속성 프로시저는 `End Set` 문으로 끝나야 합니다.  
  
 **오류 ID:** BC30633  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `Set` 속성 프로시저가 `End Set` 구문으로 끝나는지 확인합니다.
