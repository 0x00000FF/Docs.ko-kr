# <a name="39end-synclock39-must-be-preceded-by-a-matching-39synclock39"></a>&#39; 최종 SyncLock &#39; 뒤에 야 일치 하는 &#39; SyncLock &#39;
`SyncLock` 블록이 `SyncLock` 키워드로 시작하고 `End``SyncLock` 구문으로 끝납니다.  
  
 **오류 ID:** BC30674  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `SyncLock` 블록은 `SyncLock` 문으로 시작해야 합니다.
