# &#39;End SyncLock&#39;은 짝이 되는 &#39;SyncLock&#39; 뒤에 와야 합니다.
`SyncLock` 블록이 `SyncLock` 키워드로 시작하고 `End` `SyncLock` 구문으로 끝납니다.  
  
 **오류 ID:** BC30674  
  
### 이 오류를 해결하려면  
  
-   `SyncLock` 블록은 `SyncLock` 문으로 시작해야 합니다.
