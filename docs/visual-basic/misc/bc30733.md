# <a name="ltmembernamegt-is-already-declared-by-ltprocedurenamegt-which-was-generated-for-this-ltcontainernamegt"></a>'&lt;membername&gt;'가 이미 선언 에서'&lt;procedurename&gt;',이 대 한 생성 된 &lt;containername&gt;
공용 언어 런타임에 의해 선언된 멤버와 동일한 이름을 가진 멤버가 선언되었습니다.  
  
 **오류 ID:** BC30733  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   멤버의 이름을 변경하여 충돌을 제거합니다.
