# <a name="shared-cannot-be-combined-with-specifier-on-a-method-declaration"></a>'Shared'와 함께 사용할 수 없습니다 '\<지정자 >'는 메서드 선언에서
`Shared` 한정자를 메서드 선언에서 이러한 조합으로 사용할 수 없는 `Overridable`, `NotOverridable`또는 `MustOverride`와 같은 지정자와 결합하려고 했습니다.  
  
 **오류 ID:** BC30501  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 지정자를 제거합니다.
