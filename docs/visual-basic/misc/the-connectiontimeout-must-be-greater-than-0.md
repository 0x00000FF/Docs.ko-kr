# ConnectionTimeout은 0보다 커야 합니다.
[My.Computer.Network Object](../../visual-basic/language-reference/objects/my-computer-network-object.md)를 사용하여 파일을 업로드 및 다운로드할 때 `0`보다 큰 `connectionTimeout`을 지정해야 합니다.  
  
### 이 오류를 해결하려면  
  
-   `0`보다 큰 `connectionTimeout`을 제공합니다.
