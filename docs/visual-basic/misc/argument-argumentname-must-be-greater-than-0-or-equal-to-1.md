# <a name="argument-ltargumentnamegt-must-be-greater-than-0-or-equal-to--1"></a>인수 '&lt;argumentname&gt;' 0 보다 크거나-1 이어야 합니다
인수의 값이 0보다 작지만 -1과 같지 않습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  인수 값이 유효한 범위 내에 있는지 확인합니다.  
  
2.  인수를 계산에서 제공하는 경우 계산에서 처리하는 값을 확인합니다.
