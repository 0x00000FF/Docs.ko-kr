# <a name="expression-39ltnamegt39-is-not-a-procedure-but-occurs-as-the-target-of-a-procedure-call"></a>식 &#39; &lt;이름&gt;&#39;은 (는) 프로시저가 아니지만 프로시저 호출의 대상 처럼 나타납니다
프로시저가 아닌 식에 대한 프로시저 호출을 만들었습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  프로시저 이름의 철자가 틀리지 않았는지 확인합니다.  
  
2.  프로시저는 <`name`> 호출 합니다.
