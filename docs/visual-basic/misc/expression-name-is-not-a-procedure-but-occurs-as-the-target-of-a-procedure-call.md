# <a name="expression-ltnamegt-is-not-a-procedure-but-occurs-as-the-target-of-a-procedure-call"></a>식을 '&lt;이름을&gt;'은 (는) 프로시저가 있지만 프로시저 호출의 대상으로 발생 합니다.
프로시저가 아닌 식에 대한 프로시저 호출을 만들었습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  프로시저 이름의 철자가 틀리지 않았는지 확인합니다.  
  
2.  <`name`>이 호출하는 프로시저를 제공합니다.
