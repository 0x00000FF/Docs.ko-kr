# &#39;ByVal&#39;과 &#39;ByRef&#39;는 함께 사용할 수 없습니다.
`ByVal` 및 `ByRef` 키워드는 함께 사용할 수 없습니다.  
  
 **오류 ID:** BC30641  
  
### 이 오류를 해결하려면  
  
-   `ByVal` 또는 `ByRef` 중 하나만 지정합니다.
