# <a name="xml-documentation-parse-error-start-tag-39lttaggt39-doesn39t-have-a-matching-end-tag"></a>XML 문서 구문 분석 오류: 시작 태그 &#39; &lt;태그&gt; &#39; 대상이&#39;t는 짝이 되는 끝 태그
XML 문서 구문 분석 오류: 시작 태그 \<태그 >는 짝이 되는 끝 태그가 있습니다. XML 주석이 무시됩니다.  
  
 XML 주석이 시작 태그를 포함하지만 일치하는 끝 태그를 포함하지 않습니다.  
  
 **오류 ID:** BC42316  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   시작 태그와 일치하는 끝 태그를 추가합니다.  
  
     — 또는 —  
  
-   태그와 같은 내부 텍스트가 없습니다을 포함 하는 경우 [ \<seealso >](../../visual-basic/language-reference/xmldoc/seealso.md), 닫는 꺾쇠 괄호 앞에 슬래시를 지정 합니다.
