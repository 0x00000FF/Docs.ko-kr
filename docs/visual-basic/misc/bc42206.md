# <a name="maximum-number-of-warnings-has-been-exceeded"></a>최대 경고 개수를 초과했습니다.
컴파일 경고의 최대 개수는 100개입니다. 컴파일러는 오류 및 경고 보고서의 크기를 줄이기 위해 이 제한을 적용합니다.  
  
 이 메시지는 기본적으로 경고입니다. 경고를 숨기 거 나 경고를 오류로 처리에 대 한 자세한 내용은 참조 하십시오. [Visual Basic에서 경고 구성](https://docs.microsoft.com/visualstudio/ide/configuring-warnings-in-visual-basic)합니다.  
  
 **오류 ID:** BC42206  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   이 컴파일에서 생성된 다른 경고를 검토하고 가능한 한 많이 해결합니다. 소스 코드에 100개가 넘는 경고가 있을 경우 다음 컴파일 시 더 많은 경고가 표시될 수도 있습니다.
