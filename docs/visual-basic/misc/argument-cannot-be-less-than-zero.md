# <a name="argument-cannot-be-less-than-zero"></a>인수는 0보다 작을 수 없습니다.
인수는 0보다 작을 수 없는데 0보다 작은 값을 제공했습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 해당 값이 계산된 방식을 검사하여 올바른지 확인합니다.
