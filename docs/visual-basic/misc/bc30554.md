# <a name="39ltnamespacenamegt39-is-ambiguous"></a>'&lt;namespacename&gt;' 모호 합니다.
모호한 이름을 제공했으므로 다른 이름과 충돌합니다. [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] 컴파일러에서는 모든 충돌 해결 규칙 없으므로; 이름을 직접 명확 해야 합니다.  
  
 **오류 ID:** BC30554  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   이름을 정규화합니다.
