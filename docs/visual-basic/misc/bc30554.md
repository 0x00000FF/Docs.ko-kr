# &#39;&lt;namespacename&gt;&#39;이 모호합니다.
모호한 이름을 제공했으므로 다른 이름과 충돌합니다.[!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)] 컴파일러에 충돌 해결 규칙이 없으므로 이름을 직접 명확하게 해야 합니다.  
  
 **오류 ID:** BC30554  
  
### 이 오류를 해결하려면  
  
-   이름을 정규화합니다.
