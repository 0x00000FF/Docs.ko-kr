# &#39;&lt;modifier&gt;&#39;는 지역 변수 선언에서 사용할 수 없습니다.
프로시저 내의 `Dim` 문에 `Overloads`와 같은 잘못된 키워드가 포함되어 있습니다.  
  
 **오류 ID:** BC30247  
  
### 이 오류를 해결하려면  
  
-   `Dim` 문에서 잘못된 키워드를 제거합니다.
