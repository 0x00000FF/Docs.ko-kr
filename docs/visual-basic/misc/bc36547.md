# <a name="anonymous-type-member-or-property-propertyname-is-already-declared"></a><span data-ttu-id="23342-102">무명 형식 멤버 또는 속성 '\<propertyname >'가 이미 선언 되었습니다</span><span class="sxs-lookup"><span data-stu-id="23342-102">Anonymous type member or property '\<propertyname>' is already declared</span></span>
<span data-ttu-id="23342-103">속성 이름은 무명 형식 선언에서 한 번만 설정할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="23342-103">A property name can be established only once in the declaration of an anonymous type.</span></span> <span data-ttu-id="23342-104">예를 들어 다음 선언은 잘못된 것입니다.</span><span class="sxs-lookup"><span data-stu-id="23342-104">For example, the following declarations are not valid:</span></span>  
  
```vb  
'' Not valid, because the Label property is assigned to twice.  
' Dim anonType1 = New With {.Label = "Debra Garcia", .Label = .Label & ", President"}  
'' Not valid, because the property name inferred for both properties is  
'' Name.  
' Dim anonType2 = New With {Key product.Name, Key car1.Name}  
```  
  
 <span data-ttu-id="23342-105">**오류 ID:** BC36547</span><span class="sxs-lookup"><span data-stu-id="23342-105">**Error ID:** BC36547</span></span>

  
- <span data-ttu-id="23342-108">이름과 값을 유추하는 변수 또는 속성 이름에 대해 새 이름을 제공합니다.</span><span class="sxs-lookup"><span data-stu-id="23342-108">Provide new names for the variables or property names from which you are inferring names and values.</span></span>  
  
```vb

```
