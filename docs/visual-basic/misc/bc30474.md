# <a name="39ltmethod1gt39-cannot-override-39ltmethod2gt39-because-it-is-a-39declare39-statement"></a>&#39; &lt;method1&gt;&#39; 없습니다 재정의 &#39;&lt; method2&gt;&#39; 이기 때문에 &#39; Declare &#39; 문
`Declare` 문으로 선언된 기본 클래스 이름의 대리자를 재정의하려고 했습니다.  
  
 **오류 ID:** BC30474  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  재정의된 멤버를 `Declare` 문이 아니도록 변경합니다.  
  
2.  이 메서드를 재정의하지 마세요.
