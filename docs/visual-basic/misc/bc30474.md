# &#39;Declare&#39; 문이므로 &#39;&lt;method1&gt;&#39;은 &#39;&lt;method2&gt;&#39;를 재정의할 수 없습니다.
`Declare` 문으로 선언된 기본 클래스 이름의 대리자를 재정의하려고 했습니다.  
  
 **오류 ID:** BC30474  
  
### 이 오류를 해결하려면  
  
1.  재정의된 멤버를 `Declare` 문이 아니도록 변경합니다.  
  
2.  이 메서드를 재정의하지 마세요.
