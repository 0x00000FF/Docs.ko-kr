# <a name="39addhandler39-and-39removehandler39-method-parameters-cannot-be-declared-39byref39"></a>'AddHandler' 및 'RemoveHandler' 메서드 매개 변수를 선언할 수 없습니다 'ByRef'
`AddHandler` 및 `RemoveHandler` 메서드의 매개 변수는 `ByRef` 한정자로 선언할 수 없습니다.  
  
 **오류 ID:** BC31134  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   매개 변수에서 `ByRef` 키워드를 제거합니다.
