# <a name="xml-comment-tag-lttaggt-is-not-permitted-on-a-ltelementgt-language-element"></a>XML 주석 태그 '&lt;태그&gt;'에서 허용 되지 않습니다는 '&lt;요소&gt;' 언어 요소
XML 주석 태그 \<태그 >에서 허용 되지 않습니다는 '\<요소 >' 요소입니다. XML 주석이 무시됩니다.  
  
 XML 주석 태그가 적용되는 언어 요소에서 지원되지 않습니다.  
  
 **오류 ID:** BC42306  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   지원되지 않는 XML 주석 태그를 제거합니다.
