# <a name="type-39lttypenamegt39-is-not-defined-or-the-module-containing-it-is-not-loaded-in-the-debugging-session"></a>형식 &#39; &lt;typename&gt; &#39; 정의 되지 않았거나 포함 하는 모듈이 디버깅 세션에서 로드 되지 않았습니다
사용할 수 없거나 정의되지 않은 형식을 참조했습니다.  
  
 **오류 ID:** BC30743  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  형식 이름의 철자가 올바른지 확인합니다.  
  
2.  현재 클래스 또는 모듈 내에서 형식을 정의합니다.
