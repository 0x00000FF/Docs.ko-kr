# <a name="element-is-missing-an-end-tag"></a>요소에 끝 태그가 없습니다.
XML 요소 리터럴에 XML 끝 태그 또는 빈 태그 식별자 '/'(예: `<tagName />`)가 없습니다.  
  
 **오류 ID:** BC31151  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   XML 태그에 대한 끝 태그를 추가하거나 XML 태그에 빈 태그 식별자 '/'를 추가합니다.
