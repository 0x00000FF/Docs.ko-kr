# &#39;&lt;specifier&gt;&#39;는 WithEvents 선언에서 사용할 수 없습니다.
`Dim` 문에 `ReadOnly`와 같이 `WithEvents` 키워드와 함께 사용할 수 없는 키워드가 포함되어 있습니다.  
  
 **오류 ID:** BC30234  
  
### 이 오류를 해결하려면  
  
1.  `Dim` 문에서 잘못된 키워드를 제거합니다.
