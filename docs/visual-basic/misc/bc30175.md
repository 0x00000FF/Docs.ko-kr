# <a name="lttype1gt-lttypenamegt-and-lttype2gt-lttypenamegt-declared-in-ltnamegt-conflict-in-ltnamespacegt-ltnamespacenamegt"></a>&lt;type1&gt; '&lt;typename&gt;' 및 &lt;type2&gt; '&lt;typename&gt;', 선언 된'&lt;이름을&gt;'에서 충돌 &lt;네임 스페이스 &gt; '&lt;namespacename&gt;'
클래스 및 모듈 등 선언된 두 요소에 동일한 네임스페이스의 동일한 이름이 있습니다.  
  
 **오류 ID:** BC30175  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   한 요소의 선언을 다른 네임스페이스로 이동합니다.  
  
-   한 요소의 이름을 변경합니다.
