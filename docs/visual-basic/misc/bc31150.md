# 끝 태그 &lt;/tagName&gt;이 필요합니다.
XML 요소 리터럴의 끝 태그가 해당 시작 태그와 일치하지 않습니다.  
  
 **오류 ID:** BC31150  
  
### 이 오류를 해결하려면  
  
-   끝 XML 태그를 수정하여 XML 요소 리터럴의 시작 XML 태그와 일치시킵니다.
