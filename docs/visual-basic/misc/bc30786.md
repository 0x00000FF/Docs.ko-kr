# <a name="declare-statements-in-a-module-cannot-be-declared-39ltspecifiergt39"></a>모듈의 문을 선언할 수 없습니다 선언 &#39; &lt;지정자&gt;&#39;
`Declare` 선언에 `Module` 선언에서 유효하지 않은 지정자가 있습니다. 모듈은 인스턴스화될 수 없고 상속을 지원하지 않으며 인터페이스를 구현할 수 없습니다.  
  
 **오류 ID:** BC30786  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   지정자를 제거합니다.
