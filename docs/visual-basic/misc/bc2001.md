# <a name="file-filename-could-not-be-found"></a>파일 \<파일 이름 > 찾을 수 없습니다
지정된 파일을 찾을 수 없습니다. 파일 경로를 잘못 제공했기 때문일 수 있습니다.  
  
 **오류 ID:** BC2001  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   파일 이름 및 경로가 올바른지 확인합니다.
