# <a name="source-file-ltfilenamegt-specified-multiple-times"></a>소스 파일 &lt;filename&gt; 여러 번 지정 했습니다.
파일을 여러 번 지정했습니다.  
  
 **오류 ID:** BC2002  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   중복된 파일 지정자를 제거합니다.
