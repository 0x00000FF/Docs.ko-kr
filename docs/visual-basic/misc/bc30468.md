# <a name="type-declaration-characters-are-not-valid-in-this-context"></a>이 컨텍스트에서는 형식 선언 문자를 사용할 수 없습니다.
$와 같이 이 컨텍스트에서 유효하지 않은 문자를 사용하려고 했습니다.  
  
 **오류 ID:** BC30468  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1. 이 문자를 제거합니다.
