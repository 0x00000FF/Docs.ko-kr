# <a name="39ltfilenamegt39-cannot-be-referenced-because-it-is-not-an-assembly"></a>'&lt;filename&gt;' 어셈블리 아니기 때문에 참조할 수 없습니다
.NET 어셈블리가 아닌 파일에 참조가 만들어졌습니다.  
  
 **오류 ID:** BC31519  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  어셈블리의 이름 및 위치를 확인합니다.  
  
2.  어셈블리가 손상되지 않았는지 확인합니다.
