# <a name="reference-required-to-assembly-39ltassemblynamegt39-containing-the-type-39ltmembernamegt39"></a>참조 어셈블리에 필요한 '&lt;assemblyname&gt;'유형을 포함 하 는'&lt;membername&gt;'
참조 어셈블리에 필요한 '\<assemblyname > ' 형식이 포함 된 '\<membername > '입니다. 하나를 프로젝트에 추가합니다.  
  
 프로젝트에는 프로젝트 참조에 포함되지 않은 어셈블리와 다른 형식에 종속되는 형식이 있습니다.  
  
 **오류 ID:** BC30652  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   프로젝트의 종속 형식에 대한 프로젝트 참조를 추가합니다.
