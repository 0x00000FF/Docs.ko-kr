# <a name="39systemruntimeinteropservicesdllimportattribute39-cannot-be-applied-to-a-39get39-or-39set39"></a>&#39; System.Runtime.InteropServices.DllImportAttribute &#39; 에 적용할 수 없습니다는 &#39; Get &#39; 또는 &#39; 설정 &#39;
`DllImportAttribute` 특성을 `Get` 또는 `Set` 속성 프로시저에 적용했습니다.  
  
 **오류 ID:** BC31524  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  `DllImportAttribute` 및 `Get` 속성 프로시저에서 `Set` 를 제거합니다.
