# <a name="39let39-and-39set39-assignment-statements-are-no-longer-supported"></a>&#39;Let&#39; 및 &#39;설정&#39; 대입문은 더 이상 지원 되지
`Let` 및 `Set` 은 할당에서 사용할 수 없습니다.  
  
 **오류 ID:** BC30807  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   대입문에서 `Let` 및 `Set` 을 생략합니다. 예를 들면 다음과 같습니다.  
  
     `X = 2`
