# <a name="array-exceeds-the-limit-of-32-dimensions"></a>배열이 한계인 32차원을 초과합니다.
배열은 32차원을 초과할 수 없습니다.  
  
 **오류 ID:** BC30052  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   배열 선언에서 차원 수를 줄입니다.
