# <a name="conflicts-with-name1-which-is-implicitly-created-for-name2-in-type-name3"></a>충돌 '\<name1 > '에 대 한 암시적으로 만들어진'\<name2 >'에 \<유형 > '\<name3 >'
형식 멤버의 이름이 암시적으로 만들어진 다른 멤버와 충돌합니다.  
  
 **오류 ID:** BC31058  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   이름 충돌을 제거하려면 명시적으로 선언된 멤버 이름을 바꿉니다.
