# <a name="conflicts-with-39ltname1gt39-which-is-implicitly-created-for-39ltname2gt39-in-lttypegt-39ltname3gt39"></a>충돌 &#39; &lt;name1&gt;&#39;에 대 한 암시적으로 만들어진 &#39; &lt;name2&gt; &#39; 에 &lt;형식&gt; &#39; &lt;name3&gt;&#39;
형식 멤버의 이름이 암시적으로 만들어진 다른 멤버와 충돌합니다.  
  
 **오류 ID:** BC31058  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   이름 충돌을 제거하려면 명시적으로 선언된 멤버 이름을 바꿉니다.
