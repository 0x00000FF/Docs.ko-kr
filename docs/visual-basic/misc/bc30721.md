# <a name="evaluation-of-expression-or-statement-stopped"></a>식 또는 문 계산이 중단되었습니다.
**계산 중지** 메뉴 항목이 선택되어 계산이 중지되었습니다.  
  
 **오류 ID:** BC30721  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   **계산 중지**를 선택하지 않고 계속 실행합니다.
