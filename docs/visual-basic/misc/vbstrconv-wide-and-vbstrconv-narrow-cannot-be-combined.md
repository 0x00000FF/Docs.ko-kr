# <a name="vbstrconvwide-and-vbstrconvnarrow-cannot-be-combined"></a>VbStrConv.Wide와 VbStrConv.Narrow는 함께 사용할 수 없습니다.
애플리케이션에서 함께 사용할 수 없는 `VbStrConv` 열거형 멤버 `Wide` 및 `Narrow`를 결합하려고 합니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  `VbStrConv.Wide` 또는 `VbStrConv.Narrow`를 제거합니다.
