# <a name="reference-required-to-module-39ltmodulenamegt39-containing-the-definition-for-event-39lteventnamegt39"></a>모듈 &#39;에 필요한 참조 &lt;modulename&gt;&#39; 이벤트 &#39;에 대 한 정의가 포함 된&lt; eventname&gt;&#39;
모듈에 필요한 참조 ' <`modulename`>' 이벤트에 대 한 정의가 포함 된 ' <`eventname`>'입니다. 하나를 프로젝트에 추가합니다.  
  
 이벤트는 프로젝트에서 직접 참조되지 않은 모듈에서 정의됩니다. 이벤트가 둘 이상의 모듈에서 정의되는 경우 [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] 컴파일러가 참조하는 데 모호성이 없어야 합니다.  
  
 **오류 ID:** BC30006  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   프로젝트 참조에 참조되지 않은 모듈의 이름을 포함합니다.
