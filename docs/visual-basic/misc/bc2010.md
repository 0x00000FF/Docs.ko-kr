# <a name="compilation-failed-message"></a>컴파일 실패: \<메시지 >
오류 메시지에 지정된 상황으로 인해 컴파일하지 못했습니다.  
  
 **오류 ID:** BC2010  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   오류 메시지를 확인하여 오류의 정확한 원인을 파악합니다.
