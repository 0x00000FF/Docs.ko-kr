# <a name="leading-3939-or-3939-cannot-appear-in-a-constant-expression"></a>선행 &#39;입니다. &#39; 또는 &#39;! &#39; 는 상수 식에 표시할 수 없습니다.
멤버 액세스 (.) 및 사전 멤버 액세스 (!) 대부분의 경우 상수 식을 포함 하 여 멤버를 포함 하는 요소를 지정 하는 식이 필요 합니다. 다음 선언은 올바르지 않습니다.  
  
```  
' Not valid.  
Const c As String = .name  
```  
  
 **오류 ID:** BC30995  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   액세스하려는 멤버를 포함하는 인스턴스를 지정합니다.
