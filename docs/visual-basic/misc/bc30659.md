# <a name="property-or-field-39ltnamegt39-does-not-have-a-valid-attribute-type"></a>속성 또는 필드 &#39; &lt;이름&gt;&#39; 유효한 특성 유형이 없는
특성에 다른 적절한 생성자가 있을 수 있어도 유효한 특성 유형을 허용하지 않는 특성 생성자를 사용하고 있을 수 있습니다.  
  
 **오류 ID:** BC30659  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  이 특성의 사용 현황을 확인합니다.
