# <a name="property-or-field-ltnamegt-does-not-have-a-valid-attribute-type"></a>속성 또는 필드 '&lt;이름을&gt;' 특성 형식이 잘못 되지 않은
있습니다 특성은 다른 적절 한 생성자를 사용할 수 있지만 유효한 특성 형식에 사용할 수 없는 특성 생성자를 사용할 수 있습니다.  
  
 **오류 ID:** BC30659  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  이 특성의 사용 현황을 확인합니다.
