# <a name="3939-unexpected"></a>'(' unexpected
예기치 않은 '('입니다. 인스턴스화되지 않은 제네릭 형식의 배열은 사용할 수 없습니다.  
  
 [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]특정 데이터 형식의 배열을 컴파일할 수 있습니다. 제네릭 형식의 데이터 형식 매개 변수를 배열 데이터 형식으로 사용할 수 없습니다.  
  
 **오류 ID:** BC32095  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   배열을 사용해야 하는 경우 특정 데이터 형식으로 선언해야 합니다. 데이터 형식 매개 변수를 사용할 수 없습니다.  
  
-   데이터 형식 매개 변수에 대해 제공해야 하는 데이터 형식의 요소 그룹이 필요한 경우 배열 대신 컬렉션을 사용해야 합니다.
