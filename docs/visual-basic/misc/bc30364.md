# <a name="39sub-new39-cannot-be-declared-39ltmodifiergt39"></a>&#39;Sub&#39; 선언할 수 없습니다 &#39; &lt;한정자&gt;&#39;
`Sub New` 를 `Overrides`와 같은 한정자로 선언하려고 했으며, 유효하지 않습니다.  
  
 **오류 ID:** BC30364  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   한정자를 제거합니다.
