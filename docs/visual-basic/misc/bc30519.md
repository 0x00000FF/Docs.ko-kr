# <a name="overload-resolution-failed-because-no-accessible-39ltmethodgt39-can-be-called-without-a-narrowing-conversion-lterrorgt"></a>오버 로드 확인에 실패 했습니다 액세스 가능한 &#39; &lt;메서드&gt; &#39; 축소 변환 없이 호출할 수 있습니다: &lt;오류&gt;
오버로드된 메서드를 호출했지만 컴파일러에서 축소 변환 없이 호출할 수 있는 메서드를 찾을 수 없습니다. 축소 변환은 가능한 값의 일부를 정확하게 저장하지 못할 수도 있는 데이터 형식으로 값을 변경합니다.  
  
 **오류 ID:** BC30519  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `Option Strict Off`을 지정합니다.
