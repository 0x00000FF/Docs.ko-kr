# <a name="web-methods-cannot-be-evaluated-in-debug-windows"></a>디버그 창에서는 웹 메서드를 실행할 수 없습니다.
디버그 창 내에서 XML Web services 메서드를 테스트할 수 없습니다.  
  
 **오류 ID:** BC30732  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 코드 편집기에서 XML Web services 메서드를 테스트합니다.
