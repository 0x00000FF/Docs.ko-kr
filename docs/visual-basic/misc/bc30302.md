# <a name="type-character-39ltcharactergt39-cannot-be-used-in-a-declaration-with-an-explicit-type"></a>문자 형식 &#39; &lt;문자&gt;&#39; 명시적 형식이 있는 선언에 사용할 수 없습니다
명시적으로 형식을 지정할 때 `$`와 같은 형식 문자를 사용했습니다.  
  
 **오류 ID:** BC30302  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   형식 문자를 제거하거나 선언에서 `As <Type>` 절을 제거합니다.
