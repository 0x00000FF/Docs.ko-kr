# &#39;operator&#39; 연산자가 &#39;type1&#39; 및 &#39;type2&#39; 형식에 대해 정의되지 않았습니다.
'operator' 연산자가 'type1' 및 'type2' 형식에 대해 정의되지 않았습니다. 'Value' 속성을 사용하여 '\<parentElement\>'의 첫 번째 요소의 문자열 값을 가져올 수 있습니다.  
  
 XML 리터럴이 이진 또는 `Boolean` 비교에서 사용되었습니다.  
  
 **오류 ID:** BC31195  
  
### 이 오류를 해결하려면  
  
-   이진 또는 `Boolean` 비교에 대해 XML 리터럴의 `Value` 속성을 사용합니다.
