# <a name="ltnamegt-necessary-for-compiling-this-construct-cannot-be-found"></a>'&lt;이름을&gt;' 필요한이 구문을 컴파일하 찾을 수 없습니다
컴파일에 필요한 요소를 찾을 수 없습니다.  
  
 **오류 ID:** BC30930  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  필요한 도우미 요소를 제공합니다.  
  
2.  코드가 올바른 관리 코드인지 확인합니다.
