# <a name="expression-cannot-be-converted-into-an-expression-tree"></a>식을 식 트리로 변환할 수 없습니다.
대부분의 식은 식 트리로 변환할 수 있지만 일부는 변환할 수 없습니다. 이 오류는 변환할 수 없는 식을 변환하려는 시도를 나타냅니다.  
  
 **오류 ID:** BC36534
