# <a name="comma-3939-or-a-valid-expression-continuation-expected"></a>쉼표, &#39;) &#39; 또는 이어지는 올바른 식이 필요 합니다.
프로시저 정의 또는 프로시저 호출의 인수 목록과 같은 목록에서 해당 멤버가 제대로 구분되거나 종료되지 않았습니다. 두 목록 멤버가 공백으로만 구분되었기 때문일 수 있습니다.  
  
 **오류 ID:** BC32017  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   쉼표(,)를 사용하여 목록 멤버를 구분하고, 닫는 괄호())를 사용하여 목록을 종료합니다.
