# <a name="39on-error39-statements-are-not-valid-within-39using39-statements"></a>&#39;오류 발생 시&#39; 문은 내에서 유효 하지 않습니다. &#39;Using&#39; 문
`On Error` 문이 `Using` 문 내에 나타나지만 해당 컨텍스트에서 사용할 수 없습니다.  
  
 **오류 ID:** BC36013  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `Try…Catch` 문 대신 `On Error` 블록과 같은 구조적 오류 처리를 사용합니다.
