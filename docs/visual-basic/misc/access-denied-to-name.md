# &#39;&lt;name&gt;&#39;에 대한 액세스가 거부되었습니다.
컴퓨터의 보안 권한이 충분하지 않으므로 장치, 파일 또는 디스크에 액세스할 수 없습니다.  
  
### 이 오류를 해결하려면  
  
1.  필요한 권한이 있는지도 확인해야 합니다.
