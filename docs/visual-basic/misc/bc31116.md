# &#39;RemoveHandler&#39; 선언은 짝이 되는 &#39;End RemoveHandler&#39;로 끝나야 합니다.
`RemoveHandler` 선언은 `End RemoveHandler` 문으로 끝나야 합니다.  
  
 **오류 ID:** BC31116  
  
### 이 오류를 해결하려면  
  
-   `RemoveHandler` 선언이 `End RemoveHandler` 문으로 끝나는지 확인합니다.
