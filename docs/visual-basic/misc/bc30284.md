# <a name="type1-typename-cannot-be-declared-overrides-because-it-does-not-override-a-type1-in-a-base-type2"></a>\<type1 > '\<typename >' 재정의 하지 않으므로 'o v e로 선언할 수 없습니다는 \<type1 > 자료의 \<type2 >
기본 클래스에 동일한 이름을 가진 형식이 없을 때 `Sub`, `Function`또는 `Property` 문에서 `Overrides` 를 지정합니다.  
  
 **오류 ID:** BC30284  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  형식 이름 철자가 올바른지 확인합니다.  
  
2.  불필요한 `Overrides` 키워드를 제거합니다.
