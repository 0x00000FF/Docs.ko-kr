# <a name="baudrate-must-be-greater-than-0"></a>BaudRate는 0보다 커야 합니다.
`BaudRate` 메서드에 지정된 `My.Computer.Ports.OpenSerialPort` 인수는 0보다 커야 합니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `BaudRate` 인수 값을 양수로 변경합니다.
