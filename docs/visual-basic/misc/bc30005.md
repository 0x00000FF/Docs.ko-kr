# <a name="reference-required-to-assembly-39ltassemblynamegt39-containing-the-definition-for-event-39lteventnamegt39"></a>어셈블리 &#39;에 필요한 참조 &lt;assemblyname&gt;&#39; 이벤트 &#39;에 대 한 정의가 포함 된&lt; eventname&gt;&#39;
어셈블리에 필요한 참조 ' <`assemblyname`>' 이벤트에 대 한 정의가 포함 된 ' <`eventname`>'입니다. 참조를 프로젝트에 추가합니다.  
  
 이벤트는 프로젝트에서 직접 참조하지 않는 어셈블리 또는 DLL(동적 연결 라이브러리)에서 정의됩니다. 둘 이상이 DLL 또는 어셈블리에서 이벤트가 정의된 경우 모호성을 방지하기 위해 [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] 컴파일러에 참조가 필요합니다.  
  
 **오류 ID:** BC30005  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   참조되지 않은 DLL 또는 어셈블리 이름을 프로젝트 참조에 포함합니다.
