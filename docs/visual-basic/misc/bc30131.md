# <a name="security-attribute-attributename-cannot-be-applied-to-a-module"></a>보안 특성 '\<attributename >' 모듈에 적용할 수 없습니다.
공용 언어 런타임은 모듈에 대한 보안 특성을 사용할 수 없습니다.

**오류 ID:** BC30131

## <a name="to-correct-this-error"></a>이 오류를 해결하려면

- 선언에서 특성을 제거합니다.
