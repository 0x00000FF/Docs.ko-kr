# <a name="label-ltlabelnamegt-is-already-defined-in-the-current-methodmultiline-lambda"></a>레이블 '&lt;labelname&gt;' 현재 메서드/여러 줄 람다에 이미 정의 되어
레이블 이름이 프로시저 내에서 중복됩니다. 모든 이름이 프로시저 내에서 고유해야 합니다.  
  
 **오류 ID:** BC30094  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  중복된 레이블의 이름을 바꿉니다.
