# &#39;End Sub&#39;가 필요합니다.
`End` 문은 종료하는 블록과 일치해야 합니다.`Sub` 프로시저 정의가 제대로 종료되지 않았습니다.  
  
 **오류 ID:** BC30026  
  
### 이 오류를 해결하려면  
  
-   프로시저의 마지막에 `End Sub` 문을 추가합니다.
