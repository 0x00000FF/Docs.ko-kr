# <a name="overload-resolution-failed-because-no-accessible-ltmethodgt-accepts-this-number-of-arguments"></a>오버 로드 확인에 실패 했습니다 액세스 가능한 '&lt;메서드&gt;'이 개수의 인수를 허용 합니다.
오버로드된 메서드를 호출했지만, 컴파일러에서 일치하는 인수 목록이 있는 메서드를 찾을 수 없습니다.  
  
 **오류 ID:** BC30516  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   제공한 인수 목록을 확인합니다.
