# <a name="value-valuename1-cannot-be-converted-to-valuename2"></a>값 '\<valuename1 >'로 변환할 수 없습니다 '\<valuename2 >'

 **오류 ID:** BC30742

-   캐스트 중인 형식을 확인합니다.
