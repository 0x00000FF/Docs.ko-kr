# <a name="ltmethodgt-ltmethodname1gt-must-be-declared-overloads-because-another-ltmethodname2gt-is-declared-overloads"></a>&lt;메서드&gt; '&lt;methodname1&gt;'로 선언 되어야 합니다 '' 때문에 다른 '&lt;methodname2&gt;'는 '로 선언
동일한 이름의 다른 속성 또는 메서드가 `Overloads` 한정자로 선언되었습니다.  
  
 **오류 ID:** BC31409  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  `Overloads` 한정자를 속성 또는 메서드에 추가합니다.  
  
2.  `Overloads` 한정자를 동일한 이름의 모든 속성 또는 메서드에서 제거합니다.
