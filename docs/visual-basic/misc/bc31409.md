# <a name="ltmethodgt-39ltmethodname1gt39-must-be-declared-39overloads39-because-another-39ltmethodname2gt39-is-declared-39overloads39"></a>&lt;메서드&gt; &#39;&lt; methodname1&gt;&#39; 선언 해야 합니다. &#39; 오버 로드 &#39; 때문에 다른 &#39;&lt; methodname2&gt;&#39; 선언 &#39; 오버 로드 &#39;
동일한 이름의 다른 속성 또는 메서드가 `Overloads` 한정자로 선언되었습니다.  
  
 **오류 ID:** BC31409  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  `Overloads` 한정자를 속성 또는 메서드에 추가합니다.  
  
2.  `Overloads` 한정자를 동일한 이름의 모든 속성 또는 메서드에서 제거합니다.
