# <a name="unable-to-open-module-file-ltfilenamegt-lterrorgt"></a>모듈 파일을 열 수 없습니다. '&lt;filename&gt;': &lt;오류&gt;
이 프로젝트에 대한 모듈을 포함하는 파일을 열 수 없습니다.  
  
 **오류 ID:** BC31007  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  파일이 지정된 위치에 있고 다른 응용 프로그램에 의해 잠기지 않았는지 확인합니다.
