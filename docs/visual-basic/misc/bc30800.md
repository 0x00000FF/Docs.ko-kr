# <a name="method-arguments-must-be-enclosed-in-parentheses"></a>메서드 인수는 괄호로 묶어야 합니다.
프로시저 호출을 제어 하는 규칙은 최신 버전의 [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]합니다. 모든 인수는 괄호로 묶어야 합니다.  
  
 **오류 ID:** BC30800  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   인수 목록을 괄호로 묶습니다. 예:  
  
    ```  
    MySub(ArrayIndex, NewValue)  
    ```
