# <a name="39ltmodifiergt39-is-not-valid-on-a-local-constant-declaration"></a>&#39;&lt;한정자&gt; &#39; 지역 상수 선언에서 유효 하지 않습니다
프로시저 내의 `Const` 문에 `Overloads`와 같은 잘못된 키워드가 포함되어 있습니다.  
  
 **오류 ID:** BC30246  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  `Const` 문에서 잘못된 키워드를 제거합니다.
