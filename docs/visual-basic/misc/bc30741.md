# <a name="lttypename1gt-cannot-be-converted-to-lttypename2gt"></a>'&lt;typename1&gt;'로 변환할 수 없습니다'&lt;typename2&gt;'
호환되지 않는 두 형식 간에 형식을 변환하려고 했습니다.  
  
 **오류 ID:** BC30741  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   캐스트 중인 형식을 확인합니다.
