# &lt;variable&gt; 범위 변수가 이미 선언되어 있습니다.
`Select` 절이나 `Aggregate`, `Group By` 또는 `Group Join` 절의 `Into` 부분에 지정된 범위 변수 이름이 쿼리 절에 이미 지정된 범위 변수 이름과 중복됩니다.  
  
 **오류 ID:** BC36600  
  
### 이 오류를 해결하려면  
  
1.  쿼리 절에 있는 모든 범위 변수 이름이 고유한지 확인합니다.
