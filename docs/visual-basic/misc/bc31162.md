# <a name="expected-closing-gt-for-xml-cdata-section"></a>필요한 닫는 ']]&gt;' XML CDATA 섹션
XML CDATA 리터럴에 필요한 닫는 ']]>' 문자가 없습니다.  
  
 **오류 ID:** BC31162  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   XML CDATA 리터럴의 끝에 필요한 닫는 ']]>' 문자를 추가합니다.
