# <a name="unable-to-load-dll-ltfilenamegt-lterrorgt"></a>DLL을 로드할 수 없습니다. '&lt;filename&gt;': &lt;오류&gt;
이 프로젝트에서 참조하는 DLL을 로드할 수 없습니다.  
  
 **오류 ID:** BC31013  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  파일이 지정된 위치에 있고 다른 애플리케이션에 의해 잠기지 않았는지 확인합니다.
