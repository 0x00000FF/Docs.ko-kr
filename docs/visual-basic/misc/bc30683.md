# <a name="39inherits39-statement-must-precede-all-declarations-in-a-class"></a>&#39; 상속 &#39; 문 앞에 야 하는 모든 선언이 클래스에서
`Inherits` 문은 클래스 정의의 시작 부분에 있어야 합니다.  
  
 **오류 ID:** BC30683  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `Inherits` 문을 클래스의 맨 위로 이동합니다.
