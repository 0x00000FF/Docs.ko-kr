# <a name="39ltmethod1gt39-cannot-override-39ltmethod2gt39-because-they-differ-by-their-return-types"></a>&#39; &lt;method1&gt;&#39; 없습니다 재정의 &#39;&lt; method2&gt;&#39; 반환 형식이 다르기 때문에
반환 형식이 다른 메서드로 메서드를 재정의하려고 했습니다. 형식은 동일한 이름 및 서명을 갖는 메서드를 선언하고 이 선언을 `Overrides` 한정자로 표시하여 재정의 가능한 상속된 메서드를 재정의할 수 있습니다. 두 메서드의 서명이 일치해야 합니다.  
  
 **오류 ID:** BC30437  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   두 메서드의 반환 형식을 확인하고 필요에 따라 변경하여 일치시킵니다.
