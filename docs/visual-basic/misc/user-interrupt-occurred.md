# <a name="user-interrupt-occurred"></a>사용자가 중단했습니다.
사용자는 Ctrl+Break 또는 다른 인터럽트 키를 눌렀습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   작업을 다시 시도합니다.
