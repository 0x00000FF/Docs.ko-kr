# <a name="nullable-modifier-cannot-be-specified-in-variable-declarations-with-39as-new39"></a>Null 허용 한정자를 사용 하는 변수 선언에 지정할 수 없습니다 &#39;As n e w&#39;
nullable 형식 한정자(?)가 `As New` 가 지정된 변수 선언에 포함되어 있습니다. 다음 예제는 이 오류를 생성합니다.  
  
```vb  
Dim num? As New ExampleStructure  
```  
  
 **오류 ID:** BC33109  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  다음 예제와 같이 null 허용 변수 선언에서 `New` 키워드를 제거합니다.  
  
```vb  
Dim num? As ExampleStructure  
```
