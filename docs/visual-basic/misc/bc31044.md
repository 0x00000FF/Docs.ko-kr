# <a name="events-declared-with-an-39as39-clause-must-have-a-delegate-type"></a>사용 하 여 이벤트 선언는 &#39; 으로 &#39; 절에는 대리자 형식이 있어야 합니다.
대리자가 아닌 형식을 지정했습니다.  
  
 **오류 ID:** BC31044  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  `As` 절을 제거합니다.  
  
2.  `As` 절을 사용하여 대리자를 지정합니다.
