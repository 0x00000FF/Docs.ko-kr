# <a name="method-methodname-cannot-handle-event-eventname-because-they-do-not-have-the-same-signature"></a>메서드 '\<methodname >' 이벤트를 처리할 수 없습니다 '\<eventname >' 동일한 서명이 없기 때문에
지정된 이벤트 처리기의 매개 변수 또는 매개 변수 형식이 지정된 이벤트에 대해 정의된 매개 변수 또는 매개 변수 형식과 일치하지 않습니다.  
  
 **오류 ID:** BC31029
