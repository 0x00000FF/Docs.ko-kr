# <a name="xml-dtds-are-not-supported"></a>XML DTD는 지원되지 않습니다.
DTD(문서 종류 정의)가 XML 리터럴로 지정되었습니다.  
  
 **오류 ID:** BC31175  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   DTD를 제거합니다. XSD(XML 스키마 정의)를 사용하여 XML 문서에 대한 스키마를 지정합니다.
