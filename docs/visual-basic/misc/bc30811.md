# <a name="39redim39-statements-can-no-longer-be-used-to-declare-array-variables"></a>&#39; ReDim &#39; 배열 변수를 선언 하 문은 더 이상 사용할 수 없습니다.
`ReDim` 는 기존 배열 크기를 변경하는 데에만 사용할 수 있습니다.  
  
 **오류 ID:** BC30811  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   선언된 경우 배열 크기를 지정합니다. 예를 들면 다음과 같습니다.  
  
    ```  
    Dim X(20) As Integer  
    ```
