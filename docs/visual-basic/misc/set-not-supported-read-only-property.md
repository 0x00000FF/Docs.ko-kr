# <a name="set-not-supported-read-only-property"></a>Set은 지원되지 않습니다(읽기 전용 속성).
읽기 전용 속성을 설정하거나 변경하려고 했습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  사용자 코드에서 속성에 대한 참조를 제거합니다.  
  
2.  런타임에만 속성의 값을 반환하도록 참조를 변경합니다.
