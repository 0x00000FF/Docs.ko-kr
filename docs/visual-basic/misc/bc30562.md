모호한 이름을 제공했으므로 다른 이름과 충돌합니다.[!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)] 컴파일러에는 충돌 해결 규칙이 없으므로 직접 이름을 명확하게 해야 합니다.

-   이름을 정규화합니다.
