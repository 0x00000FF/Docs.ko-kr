# <a name="ltnamegt-is-ambiguous-between-declarations-in-modules-ltmodulename1gt-and-ltmodulename2gt"></a>'&lt;이름을&gt;'입니다. 모듈의 선언 사이 모호한'&lt;modulename1&gt;'및'&lt;modulename2&gt;'
모호한 이름을 제공했으므로 다른 이름과 충돌합니다. Visual Basic 컴파일러에는 충돌 해결 규칙이; 없습니다 직접 이름을 구분 해야 합니다.  
  
 **오류 ID:** BC30562  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   이름을 정규화합니다.
