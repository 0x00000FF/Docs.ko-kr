# <a name="property-accessors-cannot-be-declared-39ltkeywordgt39"></a>속성 접근자를 선언할 수 없습니다 &#39; &lt;키워드&gt;&#39;
`Get` 또는 `Set` 프로시저 선언에 속성 프로시저에 적합하지 않은 키워드가 포함되어 있습니다.  
  
 [Get 문을](../../visual-basic/language-reference/statements/get-statement.md) 및 [Set 문을](../../visual-basic/language-reference/statements/set-statement.md) 허용 된 액세스 한정자 (`Public`, `Protected`, `Friend`, `Protected Friend`, `Private`).  
  
 **오류 ID:** BC31099  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `Get` 또는 `Set` 문에서 잘못된 키워드를 제거합니다.
