# <a name="no-accessible-overloaded-ltmethodnamegt-can-be-called-with-these-arguments-without-a-widening-conversion-ltlistgt"></a>액세스 가능한 오버 로드 된 '&lt;methodname&gt;' 확대 변환 없이이 인수로 호출할 수 있습니다: &lt;목록&gt;
오버로드된 메서드를 호출했지만 확대 변환 없이 제공된 인수 목록과 일치하는 메서드가 없습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `Option Strict Off`을 지정합니다.  
  
-   오버로드된 메서드 서명 중 하나와 일치하도록 인수를 변경합니다.
