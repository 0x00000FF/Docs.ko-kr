# &#39;&lt;typename&gt;&#39;이 &#39;&lt;containername&gt;&#39;의 형식이므로 식으로 사용할 수 없습니다.
사용하고 있는 이름 중 하나가 이미 형식의 이름으로 선언되므로 식으로 사용할 수 없습니다.  
  
 **오류 ID:** BC30691  
  
### 이 오류를 해결하려면  
  
-   고유하게 멤버의 이름을 지정합니다.  
  
-   멤버 이름을 대괄호로 둘러쌉니다.
