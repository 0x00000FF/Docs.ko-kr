 <span data-ttu-id="831b1-104">**오류 ID:** BC36631</span><span class="sxs-lookup"><span data-stu-id="831b1-104">**Error ID:** BC36631</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="831b1-105">이 오류를 해결하려면</span><span class="sxs-lookup"><span data-stu-id="831b1-105">To correct this error</span></span>  
  
1. <span data-ttu-id="831b1-106">다음 예제에서 표시된 대로 `Join` 키워드를 `Group Join` 절에 추가합니다.</span><span class="sxs-lookup"><span data-stu-id="831b1-106">Add the `Join` keyword to the `Group Join` clause, as shown in the following example:</span></span>
