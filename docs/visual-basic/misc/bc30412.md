# <a name="39withevents39-variables-must-have-an-39as39-clause"></a>&#39; WithEvents &#39; 변수의 해야는 &#39; 으로 &#39; 절
`As` 절에 `WithEvents`키워드를 제공하지 않았습니다. 이벤트를 발생시킬 수 있는 특정 클래스로 변수를 선언합니다.
