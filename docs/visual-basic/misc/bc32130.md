# <a name="nullable-types-are-not-supported-because-the-current-version-of-mscorlibdll-does-not-define-the-type-nullableof-t"></a>현재 버전의 'mscorlib.dll'에서 'Nullable(of T)' 형식을 정의하지 않으므로 nullable 형식은 지원되지 않습니다.
사용 중인 프레임워크 버전은 nullable 값 형식을 지원하지 않습니다.  
  
 **오류 ID:** BC32130
