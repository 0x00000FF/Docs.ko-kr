# <a name="myapplicationlog-cannot-determine-the-amount-of-free-disk-space"></a>My.Application.Log에서 사용 가능한 디스크 공간의 크기를 확인할 수 없습니다.
<xref:Microsoft.VisualBasic.Logging.FileLogTraceListener> 클래스에서 사용 가능한 디스크 공간의 크기를 확인할 수 없습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   오류가 발생한 상황을 파악하여 Microsoft 기술 지원 서비스에 문의합니다.
