# <a name="the-specified-path-does-not-exist"></a>지정한 경로가 없습니다.
아마도 문자열을 결합하는 오류로 인해 지정된 경로가 없습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   경로가 정확한지 확인합니다.
