# &#39;&lt;filename&gt;&#39; 파일이 쓰기 금지되어 있습니다.
쓰려는 파일이 쓰기 금지되어 있으므로 변경할 수 없습니다.  
  
### 이 오류를 해결하려면  
  
1.  파일의 사용 권한을 읽기\/쓰기로 변경하거나 다른 파일에 씁니다.
