# <a name="as-expected"></a>'As'가 필요합니다.
선언될 하나 이상의 요소에 `As` 절을 제공하지 않는 선언이 발생합니다.  
  
 **오류 ID:** BC30197  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 필요에 따라 `As` 절을 제공하거나 `Option Strict Off`를 설정합니다.
