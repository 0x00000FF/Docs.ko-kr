# <a name="39currency39-is-no-longer-a-supported-type-use-the-39decimal39-type-instead"></a>&#39;통화&#39; 더 이상 지원 되는 형식이; 사용 하 여는 &#39;10 진수&#39; 대신 입력
`Currency` 데이터 형식이 `Decimal` 데이터 형식으로 대체되었습니다.  
  
 **오류 ID:** BC30803  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `Decimal` 대신 `Currency`데이터 형식을 사용합니다.
