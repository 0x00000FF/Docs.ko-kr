# <a name="character-39character39-0xlthexadecimalgt-is-not-allowed-in-xml-literal"></a>문자 &#39;문자&#39; (0 x&lt;16 진수&gt;)에서 허용 되지 않는 XML 리터럴
XML 리터럴에 지원되지 않는 문자를 사용했습니다.  
  
 **오류 ID:** BC31176  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   지원되지 않는 문자를 제거합니다.
