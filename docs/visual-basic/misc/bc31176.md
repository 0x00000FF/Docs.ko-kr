# &#39;character&#39;(0x&lt;hexadecimal&gt;) 문자는 XML 리터럴에 사용할 수 없습니다.
XML 리터럴에 지원되지 않는 문자를 사용했습니다.  
  
 **오류 ID:** BC31176  
  
### 이 오류를 해결하려면  
  
-   지원되지 않는 문자를 제거합니다.
