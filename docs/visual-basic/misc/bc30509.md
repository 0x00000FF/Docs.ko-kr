# <a name="specifier1-type-cannot-inherit-from-a-specifier2-type-because-it-expands-the-access-of-the-base-type"></a>\<specifier1 > \<형식 >에서 상속할 수 없습니다는 \<specifier2 > \<유형 > 기본에 대 한 액세스를 확장 하므로 \<형식 >
`Private` 또는 `Friend`로 표시된 기본 클래스에서 공용 클래스를 상속하려고 했습니다.  
  
 **오류 ID:** BC30509  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   기본 클래스 `Public` 을 선언하거나 상속하는 클래스 `Private` 또는 `Friend`를 선언합니다.
