# <a name="comment-statements-cannot-be-evaluated-not-in-toc"></a>주석 문은 계산할 수 없습니다 (없습니다 목차)
디버깅 세션 동안 직접 실행 창에 주석 문이 입력되었습니다. 직접 실행 창의 주석 문이 평가되지 않았습니다.  
  
 **오류 ID:** BC30974
