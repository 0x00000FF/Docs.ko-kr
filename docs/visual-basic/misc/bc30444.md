# <a name="39set39-is-already-declared"></a>&#39;설정&#39; 이미 선언
`Set` 문이 코드 블록에 두 번 이상 나타납니다. `Set` 문은 속성에 값을 할당하는 데 사용되는 `Set` 속성 프로시저를 선언합니다.  
  
 **오류 ID:** BC30444  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   중복 `Set` 문을 제거합니다.
