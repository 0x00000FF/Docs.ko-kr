path로 지정된 레지스트리 키에 둘 이상의 연속된 백슬래시가 포함되어 있습니다. 여러 문자열을 결합하여 경로를 구성하고 실수로 백슬래시를 너무 많이 포함한 결과일 수 있습니다.
