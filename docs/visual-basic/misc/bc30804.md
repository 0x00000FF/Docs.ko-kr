# <a name="39variant39-is-no-longer-a-supported-type-use-the-39object39-type-instead"></a>&#39;Variant&#39; 더 이상 지원 되는 형식이; 사용 하 여는 &#39;개체&#39; 대신 입력
`Object` 데이터 형식은 `Variant`를 대체합니다.  
  
 **오류 ID:** BC30804  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   항목을 `Object`로 선언합니다.
