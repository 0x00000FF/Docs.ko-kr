# <a name="classes-can-inherit-only-from-other-classes"></a>클래스는 다른 클래스에서만 상속할 수 있습니다.
`Inherits` 문에서 클래스가 아닌 형식을 기본 형식으로 지정합니다.  
  
 **오류 ID:** BC30258  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   기본 형식을 이미 정의되어 있는 클래스로 변경합니다.
