# <a name="39as39-comma-or-3939-expected"></a>&#39;마찬가지로&#39;, 쉼표 또는 &#39;)&#39; 필요 합니다.
제공된 형식 매개 변수에 `As`, 쉼표(`,`) 또는 닫는 괄호( `)` )가 없습니다.  
  
 **오류 ID:** BC32100  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   누락된 `As`, 쉼표 또는 닫는 괄호를 제공합니다.
