# <a name="as-comma-or--expected"></a>'As', 쉼표 또는 ')'가 필요합니다.
제공된 형식 매개 변수에 `As`, 쉼표(`,`) 또는 닫는 괄호( `)` )가 없습니다.  
  
 **오류 ID:** BC32100  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   누락된 `As`, 쉼표 또는 닫는 괄호를 제공합니다.
