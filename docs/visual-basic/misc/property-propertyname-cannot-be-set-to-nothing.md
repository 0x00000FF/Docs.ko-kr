# <a name="property-propertyname-cannot-be-set-to-nothing"></a>속성 \<propertyname > Nothing으로 설정할 수 없습니다.
속성에 `Nothing`이 아닌 값이 필요합니다. 이는 값을 잘못 계산한 결과일 수 있습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   올바른 값을 제공합니다.
