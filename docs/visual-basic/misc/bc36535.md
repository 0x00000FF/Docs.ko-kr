1.  <span data-ttu-id="1b944-107">다음 예제와 같이 `Me` 에 대한 참조에서 반환된 값 또는 인스턴스 멤버의 복사본을 만들고 쿼리 식에서 복사본을 사용합니다.</span><span class="sxs-lookup"><span data-stu-id="1b944-107">Create a copy of the instance member or the value returned by the reference to `Me` and use the copy in the query expression, as shown in the following example.</span></span>
