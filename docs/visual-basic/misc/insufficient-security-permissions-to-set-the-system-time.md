# <a name="insufficient-security-permissions-to-set-the-system-time"></a>시스템 시간을 설정할 수 있는 보안 권한이 없습니다.
시스템 시간을 설정할 수 있는 보안 권한이 없습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  보안 권한을 변경합니다.
