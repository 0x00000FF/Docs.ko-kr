# &#39;Get&#39; 문은 더 이상 지원되지 않습니다.
`Get` 문은 더 이상 지원되지 않습니다. 파일 I\/O 기능은 `Microsoft.VisualBasic` 네임스페이스에서 사용할 수 있습니다.  
  
 `Get`은 파일 작업에 지원되지 않으며 속성 프로시저 구문에만 사용할 수 있습니다.  
  
 **오류 ID:** BC30829  
  
### 이 오류를 해결하려면  
  
1.  `System.IO`, `FileSystemObject` 및 [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)] 런타임 함수의 멤버를 사용하여 파일 작업을 수행합니다.
