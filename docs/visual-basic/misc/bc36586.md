# <a name="argument-not-specified-for-parameter-ltparameternamegt-of-extension-method-ltmethodnamegt-defined-in-lttypenamegt"></a>매개 변수에 대해 지정 되지 않은 인수 '&lt;parametername&gt;'확장 메서드의'&lt;methodname&gt;'에 정의 된'&lt;typename&gt;'
확장 메서드에서 필요한 매개 변수에 대해 인수를 제공하지 않았습니다.  
  
 **오류 ID:** BC36586  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   지정된 매개 변수에 대한 인수를 제공합니다.
