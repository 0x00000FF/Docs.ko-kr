# <a name="argument-not-specified-for-parameter-39ltparameternamegt39-of-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39"></a>매개 변수 &#39;에 대해 지정 되지 않은 인수 &lt;parametername&gt;&#39; 확장 메서드 &#39;&lt; methodname&gt;&#39;에서 정의 &#39;&lt; typename&gt;&#39;
확장 메서드에서 필요한 매개 변수에 대해 인수를 제공하지 않았습니다.  
  
 **오류 ID:** BC36586

-   지정된 매개 변수에 대한 인수를 제공합니다.
