# <a name="run-time-exception-thrown"></a>런타임 예외가 발생했습니다.
디버그 창에서 식 중 하나에서 런타임 오류가 발생했습니다.  
  
 **오류 ID:** BC30704  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1. 코드를 확인합니다.
