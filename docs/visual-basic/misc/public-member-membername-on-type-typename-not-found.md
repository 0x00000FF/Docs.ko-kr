# &#39;&lt;typename&gt;&#39; 형식에 Public 멤버 &#39;&lt;membername&gt;&#39;이 없습니다.
액세스하려는 멤버를 찾을 수 없습니다. 이 오류는 런타임에 바인딩된 개체의 public이 아닌 멤버에 액세스할 때 발생할 수 있습니다.  
  
### 이 오류를 해결하려면  
  
1.  멤버가 있는 경우 `Public`으로 선언되었는지 확인합니다.
