# <a name="assembly-39ltassemblynamegt39-cannot-be-created-because-its-path-is-longer-than-259-characters"></a>어셈블리 &#39; &lt;assemblyname&gt; &#39; 해당 경로가 259 자를 초과 하므로 만들 수 없습니다
이 어셈블리에 대한 경로가 최대 길이인 259자를 초과합니다.  
  
 **오류 ID:** BC31518  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   더 짧은 이름을 사용하거나 깊이가 더 작은 파일 계층 구조를 선택하여 더 짧은 경로를 만듭니다.
