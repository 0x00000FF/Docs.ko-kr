# <a name="xml-processing-instruction-name-39name39-is-not-valid"></a>XML 처리 명령 이름 &#39; 이름 &#39; 사용할 수 없습니다.
XML 처리 명령에 지정한 이름이 잘못되었습니다.  
  
 **오류 ID:** BC31174  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   XML 처리 명령 이름의 모든 오류를 수정하거나 제거합니다.
