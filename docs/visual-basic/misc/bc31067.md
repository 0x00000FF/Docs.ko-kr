# 구조체의 메서드는 &#39;Protected&#39; 또는 &#39;Protected Friend&#39;로 선언할 수 없습니다.
구조체의 메서드가 `Protected` 또는 `Protected Friend`로 선언되었습니다.  
  
 **오류 ID:** BC31067  
  
### 이 오류를 해결하려면  
  
-   구조체 내의 메서드를 `Public` 또는 `Private`으로 선언합니다.
