# <a name="option-optionname-requires-argument"></a>옵션 \<optionname > 필요한 \<인수 >
필수 인수가 생략되었습니다.  
  
 **오류 ID:** BC2006  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 누락된 인수를 제공합니다.
