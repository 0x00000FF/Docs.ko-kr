# ParamArray 매개 변수는 &#39;ByVal&#39;로 선언해야 합니다.
`ParamArray` 매개 변수는 `ByRef` 한정자를 사용할 수 없습니다.  
  
 **오류 ID:** BC30667  
  
### 이 오류를 해결하려면  
  
-   `ByVal` 한정자를 사용하여 `ParamArray` 매개 변수를 선언합니다.
