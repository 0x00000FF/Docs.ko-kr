# <a name="name-cannot-expose-type-type1-in-specifier-type2-typename-through-name"></a>'\<이름 >' 형식을 노출할 수 없습니다 \<type1 >에서 \<지정자 > \<type2 > '\<typename >'를 통해 \<이름 >
공용 클래스 `MyClass`외부에 전용 형식을 노출하려고 했습니다.  
  
 **오류 ID:** BC30508  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `Public`형식을 선언합니다.
