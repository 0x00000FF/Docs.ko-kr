# <a name="39ltelementnamegt39-is-not-a-parameter-of-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39"></a>&#39;&lt;elementname&gt; &#39; 확장 메서드의 매개 변수가 아닙니다. &#39; &lt;methodname&gt; &#39; 에 정의 된 &#39; &lt;typename&gt;&#39;
프로시저 호출이 확장 메서드에서 정의되지 않은 추가 인수를 제공합니다.  
  
 **오류 ID:** BC36585  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   인수 목록에서 불필요한 인수를 제거합니다.
