# <a name="overload-resolution-failed-because-no-accessible-ltmethodgt-can-be-called-with-these-argumentslterrorgt"></a>오버 로드 확인에 실패 했습니다 액세스 가능한 '&lt;메서드&gt;'이 인수로 호출할 수 있습니다:&lt;오류&gt;
오버로드된 메서드를 호출했지만, 컴파일러에서 일치하는 인수 목록이 있는 메서드를 찾을 수 없습니다.  
  
 **오류 ID:** BC30518  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   제공한 인수 목록을 확인합니다.
