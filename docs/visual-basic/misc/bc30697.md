# <a name="ltmembername1gt-cannot-override-ltmembername2gt-because-they-differ-by-the-types-of-optional-parameters"></a>'&lt;membername1&gt;'를 재정의할 수 없습니다'&lt;membername2&gt;' 선택적 매개 변수의 형식이 서로 다르므로
오버로드된 멤버는 하나 이상의 필수 매개 변수에 서로 다른 데이터 형식이 있어야 합니다.  
  
 **오류 ID:** BC30697  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   필수 매개 변수 중 하나 이상을 다른 데이터 형식으로 정의합니다.
