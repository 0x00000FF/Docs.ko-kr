# <a name="referenced-39ltmembernamegt39-has-a-value-of-39nothing39"></a>참조 된 &#39; &lt;membername&gt; &#39; 값 &#39;Nothing&#39;
참조되는 항목에 값이 할당되지 않았거나 `Nothing`값이 할당되었습니다.  
  
 **오류 ID:** BC30759  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   항목 이름의 철자가 올바르고 제대로 설정되었는지 확인합니다.
