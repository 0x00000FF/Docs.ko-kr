# <a name="39ltargumentnamegt39-is-not-a-parameter-of-39ltmethoddeclarationgt39"></a>&#39;&lt;argumentname&gt; &#39; 의 매개 변수가 아닙니다. &#39; &lt;methoddeclaration&gt;&#39;
프로시저 호출이 프로시저에서 정의되지 않은 추가 인수를 제공합니다.  
  
 **오류 ID:** BC30272  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   인수 목록에서 불필요한 인수를 제거합니다.
