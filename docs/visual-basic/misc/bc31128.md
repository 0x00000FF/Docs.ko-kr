# &#39;RemoveHandler&#39;가 이미 선언되었습니다.
둘 이상의 `RemoveHandler` 선언이 사용자 지정 이벤트 선언에 표시되었습니다.`RemoveHandler` 선언에서 이벤트 처리기를 제거하는 데 사용하는 프로시저를 선언합니다.  
  
 **오류 ID:** BC31128  
  
### 이 오류를 해결하려면  
  
-   중복 `RemoveHandler` 문을 제거합니다.
