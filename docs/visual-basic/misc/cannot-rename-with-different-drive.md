# <a name="cannot-rename-with-different-drive"></a>다른 드라이브로 이름을 바꿀 수 없습니다.
`Rename` 문에서 파일 경로의 드라이브 문자를 지정할 수 없습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 파일 경로에서 드라이브 문자를 제거합니다.
