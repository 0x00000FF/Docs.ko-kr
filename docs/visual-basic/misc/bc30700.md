# 이 컨텍스트에서 식을 계산하는 동안 부작용을 사용할 수 없습니다.
이 컨텍스트에서 개체의 상태를 변경하는 문을 사용할 수 없습니다.  
  
 **오류 ID:** BC30700  
  
### 이 오류를 해결하려면  
  
-   다른 항목의 상태를 변경하지 않고 문이 계산되도록 코드를 변경합니다.
