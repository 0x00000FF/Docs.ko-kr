# <a name="name-is-declared-in-project-projectname1-which-is-not-referenced-by-project-projectname2"></a>' \<name > '은 (는) ' ' \<projectname2 > ' 프로젝트에서 참조 하지 않는 ' \<projectname1 > ' 프로젝트에서 선언 되었습니다.
현재 프로젝트에 참조가 없는 프로그래밍 요소가 프로젝트에 정의되어 있습니다.  
  
 **오류 ID:** BC32004  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 현재 프로젝트에서 원하는 요소를 포함하는 프로젝트에 대한 참조를 추가합니다.
