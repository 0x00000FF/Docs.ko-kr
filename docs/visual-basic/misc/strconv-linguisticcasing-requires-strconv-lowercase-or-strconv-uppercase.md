# <a name="39strconvlinguisticcasing39-requires-39strconvlowercase39-or-39strconvuppercase39"></a>&#39; StrConv.LinguisticCasing &#39; 필요한 &#39; StrConv.LowerCase &#39; 또는 &#39; StrConv.UpperCase &#39;
`StrConv.LinguisticCasing`또는 `StrConv.LowerCase` 과 함께 사용하는 경우에만 유효한 `StrConv.UpperCase`를 사용하려고 했습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  `StrConv.LowerCase` 과 함께 `StrConv.UpperCase` 또는 `StrConv.LinguisticCasing`를 사용합니다.
