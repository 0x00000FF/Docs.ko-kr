# <a name="39ltelementnamegt39-is-not-a-method-parameter"></a>&#39;&lt;elementname&gt; &#39; 메서드 매개 변수가 아닙니다.
프로시저 호출이 메서드에서 정의되지 않은 추가 인수를 제공합니다.  
  
 **오류 ID:** BC36568  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   인수 목록에서 불필요한 인수를 제거합니다.
