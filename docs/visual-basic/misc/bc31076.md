# <a name="modulename-is-a-module-and-cannot-be-referenced-as-an-assembly"></a>'\<modulename >'은 모듈이 며 어셈블리로 참조할 수 없습니다.
모듈을 어셈블리로 사용하려고 했습니다.  
  
 **오류 ID:** BC31076  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 어셈블리에 대한 참조를 변경합니다.
