# <a name="implicit-variable-39ltvariablenamegt39-is-invalid-because-of-39ltmessagegt39"></a>암시적 변수 '&lt;variablename&gt;'올바르지 때문 에'&lt;메시지&gt;'
디버그하는 동안 잘못된 암시적 변수가 생성되었습니다.  
  
 **오류 ID:** BC36016
