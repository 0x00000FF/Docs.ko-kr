# <a name="39ltnamegt39-is-not-a-member-of-39ltclassnamegt39"></a>&#39; &lt;이름&gt;&#39;는의 구성원 &#39;&lt; classname&gt;&#39;
제공한 멤버가 클래스의 멤버가 아닙니다.  
  
 **오류 ID:** BC30456  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  멤버의 이름이 정확한지 확인합니다.  
  
2.  클래스의 실제 멤버를 사용합니다.
