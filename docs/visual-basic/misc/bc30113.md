# <a name="39ltnamespacegt39-is-not-a-valid-name-and-cannot-be-used-as-the-root-namespace-name"></a>&#39; &lt;네임 스페이스&gt;&#39; 올바른 이름이 아닙니다 및 루트 네임 스페이스 이름으로 사용할 수 없습니다
정식 네임스페이스 이름이 올바른 요소 이름이 아닙니다. 이름은 영문자, 10진수 숫자 및 밑줄만 포함해야 하며 알파벳 문자 또는 밑줄(`_`)로 시작해야 합니다.  
  
 **오류 ID:** BC30113  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   네임스페이스가 중첩된 네임스페이스 집합에 속하는 경우 외부 이름에 잘못된 문자가 포함되었는지 확인합니다.
