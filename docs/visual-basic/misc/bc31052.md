# <a name="modules-cannot-be-declared-39ltspecifiergt39"></a>모듈을 선언할 수 없습니다 &#39; &lt;지정자&gt;&#39;
모듈이 잘못된 액세스 한정자로 선언되었습니다.  
  
 **오류 ID:** BC31052  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   액세스 한정자를 `Public` 또는 `Friend`와 같은 유효한 액세스 한정자로 바꿉니다.
