# 연산자는 &#39;&lt;typename&gt;&#39; 형식에 사용할 수 없습니다.
연산자가 형식에 적합하지 않습니다.  
  
### 이 오류를 해결하려면  
  
-   연산자가 작업 중인 형식에 적합한지 확인합니다.
