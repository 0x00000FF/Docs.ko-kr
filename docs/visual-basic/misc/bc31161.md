# <a name="expected-closing----for-xml-comment"></a>XML 주석에 닫는 '-->'가 필요합니다.
XML 주석 리터럴에 필요한 닫는 '-->' 문자가 없습니다.  
  
 **오류 ID:** BC31161  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   XML 주석 리터럴의 끝에 필요한 닫는 '-->' 문자를 추가합니다.
