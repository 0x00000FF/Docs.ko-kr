# <a name="byref-parameter-ltparameternamegt-cannot-be-used-in-a-lambda-expression"></a>'ByRef' 매개 변수 '&lt;parametername&gt;'는 람다 식에 사용할 수 없습니다.
`Sub` 또는 함수에서 선언된 람다 식이 해당 `ByRef` 또는 함수의 `Sub` 매개 변수를 사용할 수 없습니다. 예를 들어 다음 코드는 `ByRef` 매개 변수 `n` 이 람다 식에 사용되기 때문에 이 오류가 발생합니다.  
  
```  
'' Not valid.   
'Sub ExampleSub(ByRef n As Integer)  
  
'    Dim lambda = Function(p As Integer) p + n  
  
'End Sub  
```  
  
 **오류 ID:** BC36639  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   다음 코드와 같이 `ByRef` 매개 변수를 지역 변수에 할당하고 람다 식에서 지역 변수를 사용합니다.  
  
    ```  
    Sub ExampleSub(ByRef n As Integer)  
  
        Dim temp = n  
        Dim lambda = Function(p As Integer) p + temp  
  
    End Sub  
    ```
