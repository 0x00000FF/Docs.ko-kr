# <a name="namespace-or-type-ltnamegt-has-already-been-imported"></a>Namespace 또는 형식 '&lt;이름을&gt;' 이미 가져온
네임스페이스를 두 번 이상 가져오려고 했습니다.  
  
 **오류 ID:** BC31051  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   중복 `Imports` 문을 제거합니다.
