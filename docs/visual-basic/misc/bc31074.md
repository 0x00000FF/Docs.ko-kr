# <a name="delegate-constructor-for-type-39lttypenamegt39-necessary-for-the-implementation-of-delegates-cannot-be-found"></a>형식에 대 한 대리자 생성자 &#39; &lt;typename&gt;&#39;, 필요한을 대리자의 구현을 찾을 수 없습니다
형식을 포함하는 어셈블리가 손상되었거나 잘못되었을 수 있습니다.  
  
 **오류 ID:** BC31074  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  이 형식을 포함하는 어셈블리가 유효한지 확인합니다.  
  
2.  프로그램을 다시 컴파일하여 오류가 다시 발생하는지 확인합니다.  
  
3.  오류가 계속 발생 하는 경우 작업 내용을 저장 하 고 Visual Studio를 다시 시작 합니다.
