# <a name="ltpropertyname1gt-cannot-override-ltpropertyname2gt-because-they-differ-by-readonly-or-writeonly"></a>'&lt;propertyname1&gt;'를 재정의할 수 없습니다'&lt;propertyname2&gt;' 'ReadOnly' 또는 'WriteOnly'가 다르므로
첫 번째 속성과 `ReadOnly` 또는 `WriteOnly`만 다른 두 번째 속성으로 속성을 재정의하려고 했습니다.  
  
 **오류 ID:** BC30362  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `ReadOnly` 및 `WriteOnly`이외에도 다른 점이 있는 속성을 사용합니다.
