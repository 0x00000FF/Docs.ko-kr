# <a name="39ltpropertyname1gt39-cannot-override-39ltpropertyname2gt39-because-they-differ-by-39readonly39-or-39writeonly39"></a>&#39; &lt;propertyname1&gt;&#39; 없습니다 재정의 &#39;&lt; propertyname2&gt;&#39; 다르기 때문에 &#39; 읽기 전용 &#39; 또는 &#39; WriteOnly &#39;
첫 번째 속성과 `ReadOnly` 또는 `WriteOnly`만 다른 두 번째 속성으로 속성을 재정의하려고 했습니다.
