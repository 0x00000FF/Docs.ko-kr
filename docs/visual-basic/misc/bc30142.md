# <a name="unable-to-generate-a-reference-to-file-39ltfilenamegt39-use-tlbimp-utility-to-reference-com-dlls-lterror-messagegt"></a>파일에 대 한 참조를 생성할 수 없습니다. '&lt;filename&gt;' (COM Dll 참조를 TLBIMP 유틸리티를 사용 하 여): &lt;오류 메시지&gt;
[!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] 컴파일러가 어셈블리 링커(Al.exe, Alink라고도 함)를 호출하여 매니페스트를 사용해 어셈블리를 생성합니다. 링커가 COM+ DLL 파일의 유효성 검사 또는 오류 발생을 보고했습니다.  
  
 **오류 ID:** BC30142  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  따옴표로 묶인된 오류 메시지를 검사 하는 항목을 검토 [Al.exe 도구 오류 및 경고](http://msdn.microsoft.com/en-us/7f125d49-0a03-47a6-9ba9-d61a679a7d4b) 추가 설명과 권장 사항을 대 한 합니다.  
  
2.  원하는 참조는 COM + DLL 보다 COM DLL을 사용 하 여는 [Tlbimp.exe (형식 라이브러리 가져오기)](http://msdn.microsoft.com/library/ec0a8d63-11b3-4acd-b398-da1e37e97382) 에 대 한 참조를 생성 합니다.  
  
3.  오류가 계속 발생하면 해당 상황에 대한 정보를 수집하여 Microsoft 기술 지원 서비스에 알립니다.
