# <a name="39option-infer39-can-be-followed-only-by-39on39-or-39off39"></a>&#39;Infer 옵션&#39; 만 올 수 있습니다 &#39;에&#39; 또는 &#39;해제&#39;
`Option Infer` 문은 `On` 또는 `Off`를 지정해야 합니다. 다른 값이 잘못되었습니다.  
  
 **오류 ID:** BC30979  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `Option Infer On` 또는 `Option Infer Off`를 지정합니다.
