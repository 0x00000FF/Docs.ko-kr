# <a name="specifier-is-not-valid-on-a-structure-declaration"></a>'\<지정자 >' 구조체 선언에서 유효 하지 않은
`Dim` 선언에서 유효하지 않은 `Override`또는 `Structure` 와 같은 지정자를 사용했습니다.  
  
 **오류 ID:** BC30395  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 지정자를 제거합니다.
