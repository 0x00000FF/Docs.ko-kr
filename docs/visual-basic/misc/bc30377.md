# <a name="method-ltmethodnamegt-is-already-declared-in-interface-ltinterfacenamegt"></a>메서드 '&lt;methodname&gt;'에 이미 선언 되었습니다 인터페이스'&lt;interfacename&gt;'
인터페이스 메서드를 두 번 선언하여 중복되었습니다.  
  
 **오류 ID:** BC30377  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   중복된 선언을 제거합니다.
