# &#39;&lt;specifier1&gt;&#39;과 &#39;&lt;specifier2&gt;&#39;는 함께 사용할 수 없습니다.
호환되지 않는 두 지정자\(예: `Shadows` 및 `Overrides`\)를 함께 사용했습니다.  
  
 **오류 ID:** BC31408  
  
### 이 오류를 해결하려면  
  
-   지정자 중 하나를 제거합니다.
