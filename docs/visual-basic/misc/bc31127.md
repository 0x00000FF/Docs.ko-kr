# &#39;AddHandler&#39;가 이미 선언되었습니다.
둘 이상의 `AddHandler` 선언이 사용자 지정 이벤트 선언에 표시되었습니다.`AddHandler` 선언은 이벤트 처리기를 추가하는 데 사용되는 프로시저를 선언합니다.  
  
 **오류 ID:** BC31127  
  
### 이 오류를 해결하려면  
  
-   중복 `AddHandler` 문을 제거합니다.
