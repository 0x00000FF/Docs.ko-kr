# <a name="39finally39-must-end-with-a-matching-39end-try39"></a>&#39; 마지막으로 &#39; 일치 하는 &#39;로 끝나야 합니다. End Try &#39;
A `Finally` 문이 코드에서 짝이 되는 없이 `End Try` 문. `Finally` 문은 `End Try` 문으로 완료되어야 합니다.  
  
 **오류 ID:** BC30442  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  `Finally` 문을 제거합니다.  
  
2.  `End Try` 문을 추가하여 블록을 완료합니다.
