# <a name="unable-to-get-type-information-for-39lttypenamegt39"></a>에 대 한 형식 정보 &#39; 가져올 수 없습니다. &lt;typename&gt;&#39;

 **오류 ID:** BC30751  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   현재 범위 내에서 형식이 정의되도록 코드를 재구성합니다.
