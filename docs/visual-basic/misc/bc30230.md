# <a name="39inherits39-not-valid-in-modules"></a>&#39; 상속 &#39; 모듈에서 유효 하지 않습니다
`Inherits` 문이 모듈 내에서 발생합니다. 모듈이 클래스에서 상속할 수 없습니다.  
  
 **오류 ID:** BC30230  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `Inherits` 문을 제거하거나 모듈을 클래스로 다시 입력합니다.
