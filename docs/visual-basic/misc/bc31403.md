# <a name="imports-alias-ltnamegt-conflicts-with-ltname2gt-declared-in-the-root-namespace"></a>Imports 별칭&lt;이름을&gt;'충돌'&lt;name2&gt;' 루트 네임 스페이스에서 선언
`Imports` 문에서 사용되는 별칭 이름이 루트 네임스페이스에서 선언된 다름 이름과 충돌합니다.  
  
 **오류 ID:** BC31403  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `Imports` 문에서 사용하는 별칭에 대해 고유한 이름을 선택합니다.
