# Imports 별칭 &#39;&lt;name&gt;&#39;이 루트 네임스페이스에 선언된 &#39;&lt;name2&gt;&#39;와 충돌합니다.
`Imports` 문에서 사용되는 별칭 이름이 루트 네임스페이스에서 선언된 다름 이름과 충돌합니다.  
  
 **오류 ID:** BC31403  
  
### 이 오류를 해결하려면  
  
-   `Imports` 문에서 사용하는 별칭에 대해 고유한 이름을 선택합니다.
