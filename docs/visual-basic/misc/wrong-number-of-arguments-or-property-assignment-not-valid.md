# <a name="wrong-number-of-arguments-or-property-assignment-not-valid"></a>인수 개수가 잘못되었거나 잘못된 속성이 할당되었습니다.
잘못된 할당을 시도했습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  지정한 인수의 수가 대상에 필요한 수와 일치하는지 확인합니다.  
  
2.  속성 할당을 확인합니다.
