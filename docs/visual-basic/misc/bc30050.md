# ParamArray 매개 변수는 배열이어야 합니다.
`ParamArray` 인수는 1차원 배열 형식으로 선언되어야 합니다. 선언에서 `ParamArray` 인수 이름 뒤에 괄호가 없습니다.  
  
 **오류 ID:** BC30050  
  
### 이 오류를 해결하려면  
  
-   `ParamArray` 인수 이름 뒤에 괄호 쌍을 추가합니다.
