# <a name="39get39-is-already-declared"></a>&#39;가져오기&#39; 이미 선언
`Get` 문이 코드 블록에 두 번 이상 나타납니다. `Get` 문은 속성 값을 검색하는 데 사용되는 `Get` 속성 프로시저를 선언합니다.  
  
 **오류 ID:** BC30443  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   중복 `Get` 문을 제거합니다.
