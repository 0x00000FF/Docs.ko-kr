# <a name="automation-object-does-not-have-a-default-value"></a>자동화 개체에 기본값이 없음

### <a name="to-correct-the-error"></a>오류를 수정하려면  
  
-   개체의 설명서를 확인하고 메서드나 속성을 명시적으로 지정합니다.
