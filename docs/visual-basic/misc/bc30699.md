# <a name="ltmembernamegt-is-not-declared-or-the-module-containing-it-is-not-loaded-in-the-debugging-session"></a>'&lt;membername&gt;'으로 선언 되지 않았거나이 포함 하는 모듈이 디버깅 세션에서 로드 되지 않았습니다
지정된 멤버를 찾을 수 없습니다.  
  
 **오류 ID:** BC30699  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  멤버가 정의되었으며 철자가 맞는지 확인합니다.  
  
2.  모듈에 선언된 멤버 중 하나에 액세스합니다. 선언된 모듈이 아직 로드되지 않아 디버깅 환경에서 멤버를 찾을 수 없는 경우도 있습니다.
