# &#39;Overrides&#39;로 선언된 메서드는 암시적으로 재정의할 수 있으므로 &#39;Overridable&#39;로 선언할 수 없습니다.
대부분의 메서드와 달리 `Overrides` 한정자로 표시된 메서드는 기본적으로 재정의할 수 있습니다.  
  
 **오류 ID:** BC30730  
  
### 이 오류를 해결하려면  
  
-   `Overrides` 한정자로 표시된 메서드에서 `Overridable` 한정자를 생략합니다.
