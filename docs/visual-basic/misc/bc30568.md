# <a name="array-initializer-has-ltnumbergt-too-many-elements"></a>배열 이니셜라이저의 &lt;번호&gt; 요소가 너무 많습니다.
배열 이니셜라이저에 요소가 너무 많습니다.  
  
 **오류 ID:** BC30568  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `ReDim` 을 사용하여 배열 크기를 변경합니다.
