# 배열 이니셜라이저의 요소가 &lt;number&gt;개 많습니다.
배열 이니셜라이저에 요소가 너무 많습니다.  
  
 **오류 ID:** BC30568  
  
### 이 오류를 해결하려면  
  
-   `ReDim`을 사용하여 배열 크기를 변경합니다.
