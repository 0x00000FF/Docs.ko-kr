# <a name="shared-sub-new-cannot-have-any-parameters"></a>공유 'Sub New'에는 매개 변수를 사용할 수 없습니다.
공유 `Sub New` 문에 매개 변수를 포함했는데, 이는 잘못된 것입니다.  
  
 **오류 ID:** BC30479  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   매개 변수를 제거합니다.
