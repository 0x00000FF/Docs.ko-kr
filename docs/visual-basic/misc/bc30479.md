# <a name="shared-39sub-new39-cannot-have-any-parameters"></a>공유 &#39;Sub New&#39; 매개 변수를 가질 수 없습니다
공유 `Sub New` 문에 매개 변수를 포함했는데, 이는 잘못된 것입니다.  
  
 **오류 ID:** BC30479  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   매개 변수를 제거합니다.
