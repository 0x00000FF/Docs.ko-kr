# <a name="39withevents39-variable-does-not-raise-any-events"></a>&#39;WithEvents&#39; 변수 모든 이벤트를 발생 시 키 지 않습니다
`WithEvents` 키워드로 선언된 개체 변수는 이벤트를 발생시킬 수 있는 클래스 인스턴스를 참조하지 않습니다.  
  
 **오류 ID:** BC30591  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  개체 변수가 이벤트를 발생시킬 수 있는 클래스 인스턴스를 참조하는지 확인합니다.
