# <a name="length-of-argument-39ltargumentnamegt39-must-be-greater-than-zero"></a>인수의 길이 &#39; &lt;argumentname&gt; &#39; 0 보다 커야 합니다
인수의 값이 0보다 작거나 같습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   0 보다 큰 인수를 제공합니다.
