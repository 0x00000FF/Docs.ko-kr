# &#39;End Namespace&#39;는 짝이 되는 &#39;Namespace&#39; 뒤에 와야 합니다.
네임스페이스 블록은 `Namespace` 문으로 시작하여 `End Namespace` 구문으로 끝납니다.  
  
 **오류 ID:** BC30623  
  
### 이 오류를 해결하려면  
  
-   네임스페이스 블록은 `Namespace` 문으로 시작해야 합니다.
