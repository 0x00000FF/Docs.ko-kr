# <a name="39end-namespace39-must-be-preceded-by-a-matching-39namespace39"></a>&#39; 최종 Namespace &#39; 뒤에 야 일치 하는 &#39; Namespace &#39;
네임스페이스 블록은 `Namespace` 문으로 시작하여 `End Namespace` 구문으로 끝납니다.  
  
 **오류 ID:** BC30623  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   네임스페이스 블록은 `Namespace` 문으로 시작해야 합니다.
