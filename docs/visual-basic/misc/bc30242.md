# <a name="ltmodifiergt-is-not-valid-on-a-method-declaration"></a>'&lt;한정자&gt;' 메서드 선언에서 유효 하지 않은
`Function` 또는 `Sub` 문에 잘못된 키워드가 포함되어 있습니다.  
  
 **오류 ID:** BC30242  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `Function` 또는 `Sub` 문에서 잘못된 키워드를 제거합니다.
