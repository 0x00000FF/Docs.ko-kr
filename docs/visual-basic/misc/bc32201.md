# 속성 매개 변수 이름에는 &#39;Value&#39;를 사용할 수 없습니다.
`Property` 문의 매개 변수는 이름이 `Value`입니다. 이 이름은 `Set` 프로시저의 속성에 할당되는 새 값에 대한 속성 정의 내에 예약되어 있습니다.  
  
 **오류 ID:** BC32201  
  
### 이 오류를 해결하려면  
  
-   `Property` 문의 매개 변수의 이름을 변경합니다.
