# <a name="39raiseevent39-declaration-must-end-with-a-matching-39end-raiseevent39"></a>&#39;RaiseEvent&#39; 선언은 짝이 되는로 끝나야 &#39;끝 RaiseEvent&#39;
`RaiseEvent` 선언은 `End RaiseEvent` 문으로 끝나야 합니다.  
  
 **오류 ID:** BC31117  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `RaiseEvent` 선언이 `End RaiseEvent` 문으로 끝나는지 확인합니다.
