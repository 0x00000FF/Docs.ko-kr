# 매개 변수에는 자신을 정의하는 함수와 같은 이름을 사용할 수 없습니다.
매개 변수 및 해당 정의 함수에 같은 이름을 사용했습니다.  
  
 **오류 ID:** BC30530  
  
### 이 오류를 해결하려면  
  
-   함수 내의 매개 변수 이름을 확인하고 충돌하는 매개 변수의 이름을 변경합니다.
