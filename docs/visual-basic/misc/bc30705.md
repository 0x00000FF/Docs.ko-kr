# <a name="referenced-object-39ltobjectnamegt39-has-a-value-of-39nothing39"></a>참조 된 개체 &#39; &lt;objectname&gt;&#39;에 값으로 &#39; 아무것도 &#39;
참조되는 개체를 설정하지 않았거나 `Nothing`값으로 설정했습니다.  
  
 **오류 ID:** BC30705  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   개체 이름의 철자가 맞는지, 개체가 설정되었는지 확인합니다.
