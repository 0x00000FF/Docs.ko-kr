# <a name="the-address-for-uploadfile-needs-to-include-a-filename"></a>UploadFile의 주소에는 파일 이름이 포함되어야 합니다.
`UploadFile` 메서드를 사용하여 파일을 업로드하는 경우 대상 경로에 파일 이름을 제공해야 합니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   파일 이름을 제공합니다.
