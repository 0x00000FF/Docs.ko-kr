 <span data-ttu-id="63729-104">다음 코드에서는 이 오류를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="63729-104">The following code generates this error.</span></span>  
  
```

```  
  
 <span data-ttu-id="63729-105">다음 코드에서는 오류를 방지합니다.</span><span class="sxs-lookup"><span data-stu-id="63729-105">The following code avoids the error.</span></span>

    intArray(0, i) = i

Next i
