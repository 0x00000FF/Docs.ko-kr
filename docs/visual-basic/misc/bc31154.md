# <a name="xml-declaration-does-not-allow-attribute-39attributename39"></a>XML 선언 특성을 사용 하지 못하도록 &#39;attributeName&#39;
XML 문서 리터럴에 대한 잘못된 특성이 지정되었습니다. XML 문서 리터럴에 대한 올바른 특성 값은 XML 1.0 사양을 기반으로 합니다.  
  
 **오류 ID:** BC31154  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   XML 문서 리터럴에서 잘못된 특성을 제거합니다.
