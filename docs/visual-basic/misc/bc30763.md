# .NET Compact Framework의 대상 버전에서 ANSI, Auto 또는 Unicode 한정자의 사용을 지원하지 않습니다.
사용 중인 .NET Compact Framework의 버전에서는 `Ansi`, `Auto` 및 `Unicode` 한정자를 지원하지 않습니다.  
  
 **오류 ID:** BC30763  
  
### 이 오류를 해결하려면  
  
-   한정자를 제거합니다.
