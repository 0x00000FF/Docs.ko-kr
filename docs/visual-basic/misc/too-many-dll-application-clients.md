# <a name="too-many-dll-application-clients"></a>DLL 애플리케이션 클라이언트가 너무 많습니다.
Visual Basic에 대 한 동적 연결 라이브러리 (DLL)은 제한 된 수의 호스트 응용 프로그램 액세스만 허용할 수 있습니다. 응용 프로그램 및 Visual Basic 호스트 (일부는 액세스할 수 있습니다 응용 프로그램)는 다른 응용 프로그램 모두에 액세스 하려는 Visual Basic DLL 동시에 합니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   Visual Basic에 액세스 하는 응용 프로그램의 수를 줄입니다.
