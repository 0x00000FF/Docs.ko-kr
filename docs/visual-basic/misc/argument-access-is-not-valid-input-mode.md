# <a name="argument-39access39-is-not-valid-valid-values-for-input-mode-are-39openaccessread39-and-39openaccessdefault39"></a>인수 &#39; 액세스 &#39; 잘못 되었습니다. 입력 모드에 대 한 유효한 값은 #39;OpenAccess.Read &#39; 및 &#39;OpenAccess.Default &#39;
액세스 값이 `Input` 모드에서 유효하지 않습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   값을 `OpenAccess.Read` 또는 `OpenAccess.Default`중 하나로 변경합니다.
