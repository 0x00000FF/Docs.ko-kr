# &#39;Access&#39; 인수가 잘못되었습니다. 입력 모드의 올바른 값은 &#39;OpenAccess.Read&#39;와 &#39;OpenAccess.Default&#39;입니다.
액세스 값이 `Input` 모드에서 유효하지 않습니다.  
  
### 이 오류를 해결하려면  
  
-   값을 `OpenAccess.Read` 또는 `OpenAccess.Default` 중 하나로 변경합니다.
