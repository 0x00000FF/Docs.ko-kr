# <a name="the-file-is-already-open"></a>파일이 이미 열려 있습니다.
이미 열려 있는 파일을 열려고 했습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   파일을 열려고 하기 전에 파일을 닫습니다.  
  
     -또는-  
  
-   코드에서 중복 `FileOpen` 문을 제거합니다.
