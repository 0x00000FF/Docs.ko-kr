# <a name="ltnamegt-cannot-be-named-as-a-parameter-in-an-attribute-specifier-because-it-is-not-a-field-or-property"></a>'&lt;이름을&gt;' 필드 또는 속성이 아니므로 특성 지정자에서 매개 변수로 명명할 수 없습니다.
특성 블록은 특성의 비가변 멤버에 대한 값을 설정합니다. 값은 필드 또는 속성과 같은 가변 멤버에만 할당할 수 있습니다.  
  
 **오류 ID:** BC32010  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  특성 및 멤버 이름의 철자가 정확한지 확인합니다.  
  
2.  멤버가 비가변인 경우 특성 블록에서 인수를 제거합니다.
