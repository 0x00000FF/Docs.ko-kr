# 다른 버전의 &#39;&lt;assemblyname&gt;&#39; 어셈블리에서 &#39;&lt;typename&gt;&#39; 형식을 가져옵니다.
다른 버전의 '\<assemblyname\>' 어셈블리에서 '\<typename\>' 형식을 가져옵니다. 동일한 프로젝트에서 동일한 어셈블리의 서로 다른 버전을 사용할 수 없습니다.  
  
 동일한 어셈블리의 다른 버전에서 형식을 가져와 모호성이 발생합니다.  
  
 **오류 ID:** BC31525  
  
### 이 오류를 해결하려면  
  
-   동일한 어셈블리의 여러 버전에 대한 참조를 제거합니다.
