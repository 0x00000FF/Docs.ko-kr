# <a name="type-39lttypenamegt39-is-imported-from-different-versions-of-assembly-39ltassemblynamegt39"></a>형식 &#39; &lt;typename&gt; &#39; 서로 다른 버전의 어셈블리에서 가져온 &#39; &lt;assemblyname&gt;&#39;
형식 '\<형식 이름 >' 어셈블리의 서로 다른 버전에서 가져온 '\<assemblyname >'입니다. 동일한 프로젝트에서 동일한 어셈블리의 서로 다른 버전을 사용할 수 없습니다.  
  
 동일한 어셈블리의 다른 버전에서 형식을 가져와 모호성이 발생합니다.  
  
 **오류 ID:** BC31525  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   동일한 어셈블리의 여러 버전에 대한 참조를 제거합니다.
