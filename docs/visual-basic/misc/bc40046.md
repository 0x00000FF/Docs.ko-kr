# <a name="type-39lttypenamegt39-and-partial-type-39lttypenamegt39-conflict-in-container-39ltcontainernamegt39-but-are-being-merged-because-one-of-them-is-declared-partial"></a>형식 &#39; &lt;typename&gt;&#39; 및 부분 형식 &#39;&lt; typename&gt;&#39; 컨테이너 &#39;에서 충돌&lt; containername&gt;&#39; 하지만 그 중 하나가 partial 선언 되었으므로 병합 됩니다.
클래스 또는 구조체가 동일한 컨테이너 형식에서 여러 정의에 표시되고 둘 이상의 정의가 `Partial`로 표시되지 않습니다.  
  
 클래스 또는 구조체의 여러 정의 중 하나 이상에서 [Partial](../../visual-basic/language-reference/modifiers/partial.md) 키워드를 사용해야 하지만 모든 부분 정의에서 사용하는 것이 좋습니다.  
  
 이 메시지는 기본적으로 경고입니다. 경고를 숨기거나 오류로 처리하는 방법에 대한 자세한 내용은 [Configuring Warnings in Visual Basic](/visualstudio/ide/configuring-warnings-in-visual-basic)을 참조하세요.  
  
 **오류 ID:** BC40046  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   클래스 또는 구조체의 모든 부분 정의에 [Partial](../../visual-basic/language-reference/modifiers/partial.md) 키워드를 사용합니다.
