# <a name="object-doesn39t-support-current-locale-setting-visual-basic"></a>개체 원본이 &#39; t 지원 현재 로캘 설정 (Visual Basic)
현재 프로젝트에 대한 로캘 설정을 지원하지 않는 개체에 액세스하려고 했습니다.

-   개체가 지원하는 로캘 설정을 확인합니다.
