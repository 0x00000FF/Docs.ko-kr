# <a name="for-loop-control-variable-ltvariablenamegt-already-in-use-by-an-enclosing-for-loop"></a>For 루프 제어 변수 '&lt;variablename&gt;'에 이미 바깥쪽 For 루프 사용
중첩된 `For` 루프는 모두 고유한 제어 변수를 사용해야 합니다.  
  
 **오류 ID:** BC30069  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   제어 변수 이름 중 하나를 변경합니다.
