# <a name="39compare39-expected"></a>&#39;비교&#39; 필요 합니다.
`Option` 문에서 `Text` 또는 `Binary`를 지정하지만 `Compare`를 지정하지 않습니다.  
  
 **오류 ID:** BC30208  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `Compare` 문에 `Option` 키워드를 추가합니다(예: `Option Compare Binary`).
