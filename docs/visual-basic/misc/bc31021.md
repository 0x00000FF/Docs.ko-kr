# <a name="visual-basic-compiler-is-unable-to-recover-from-the-following-error-lterrorgt"></a>Visual Basic 컴파일러에서 다음 오류를 복구할 수 없는: &lt;오류&gt;
Visual Basic 컴파일러에서 다음 오류를 복구할 수 없는: <`error`> 합니다.  
  
 일부 IDE 기능을 사용하지 않도록 설정했습니다.  
  
 Visual Basic 컴파일러에서 내부 오류가 발생 했습니다.  
  
 **오류 ID:** BC31021  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  작업 내용을 저장 하 고 Visual Studio를 다시 시작 합니다.  
  
2.  오류가 다시 발생 하는 경우 Visual Studio를 다시 설치 합니다.  
  
3.  다시 설치 후에도 오류가 계속 발생하면 Microsoft 기술 지원 서비스에 알립니다.
