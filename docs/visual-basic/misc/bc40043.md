# <a name="systemclscompliantattribute-cannot-be-applied-to-property-39get3939set39"></a>System.clscompliantattribute는 속성 &#39;에 적용할 수 없습니다. Get &#39; / &#39; 설정 &#39;
속성 정의는 <xref:System.CLSCompliantAttribute> 특성을 해당 `Get` 또는 `Set` 문에 적용합니다.  
  
 과 호환 되도록 속성에 대 한는 [언어 독립성 및 언어 독립적 구성 요소](../../standard/language-independence-and-language-independent-components.md) (CLS)는 전체 속성으로 표시 되어야 합니다 `<CLSCompliant(True)>`합니다. 적용 해야 <xref:System.CLSCompliantAttribute> 에 [Property 문](../../visual-basic/language-reference/statements/property-statement.md)아닌는 `Get` 또는 `Set` 문.  
  
 <xref:System.CLSCompliantAttribute> 를 프로그래밍 요소에 적용하는 경우 특성의 `isCompliant` 매개 변수를 `True` 또는 `False` 로 설정하여 준수 여부를 나타냅니다. 이 매개 변수에는 기본값이 없으며 값을 제공해야 합니다.  
  
 요소에 <xref:System.CLSCompliantAttribute> 를 적용하지 않으면 비규격인 것으로 간주됩니다.  
  
 이 메시지는 기본적으로 경고입니다. 경고를 숨기거나 오류로 처리하는 방법에 대한 자세한 내용은 [Configuring Warnings in Visual Basic](/visualstudio/ide/configuring-warnings-in-visual-basic)을 참조하세요.  
  
 **오류 ID:** BC40043  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   <xref:System.CLSCompliantAttribute> 또는 `Get` 문에서 `Set` 를 제거합니다.  
  
-   속성이 CLS 규격을 준수해야 하면 `Property` 문을 `<CLSCompliant(True)>`로 표시합니다.
