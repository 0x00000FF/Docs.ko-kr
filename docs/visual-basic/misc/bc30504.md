# &#39;&lt;propertyname&gt;&#39; 속성은 기본 &lt;type&gt; &#39;&lt;typename&gt;&#39;에서 기본값이 아닌 속성을 재정의하므로 &#39;기본값&#39;으로 선언될 수 없습니다.
`Default` 속성을 선언하려고 시도했지만 기본 클래스에서 `Default`로 선언되지 않은 속성을 재정의하므로 유효한 선언이 아닙니다.  
  
 **오류 ID:** BC30504  
  
### 이 오류를 해결하려면  
  
-   `Default` 선언을 제거합니다.
