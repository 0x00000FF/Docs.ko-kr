# <a name="too-many-arguments-to-procedurename"></a>너무 많은 인수를 '\<procedurename >'
프로시저 호출에 제공된 인수 개수가 프로시저에 대해 정의된 인수 개수를 초과합니다.  
  
 **오류 ID:** BC30057  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   프로시저의 정의를 확인하고 호출에서 추가 인수를 제거합니다.
