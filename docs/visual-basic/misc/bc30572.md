# <a name="alias-namespacename-is-already-declared"></a>별칭\<namespacename >'가 이미 선언 되었습니다
`Imports` 를 사용하여 이미 선언된 네임스페이스를 가져오려고 했습니다.  
  
 **오류 ID:** BC30572  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   가져오려는 네임스페이스를 확인합니다.
