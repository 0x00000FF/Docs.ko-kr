# &#39;ReDim&#39; Preserve 피연산자는 Nothing이 될 수 없습니다.
`ReDim` 문에서 `Preserve` 키워드를 사용하여 마지막 차원이 아닌 배열의 차원을 변경하려고 했지만, 해당 피연산자에 대한 올바른 값을 제공하지 않았습니다.  
  
### 이 오류를 해결하려면  
  
-   `Preserve` 피연산자를 올바른 값으로 변경합니다.
