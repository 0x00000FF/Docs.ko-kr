# <a name="39raiseevent39-statements-are-not-valid-in-the-immediate-window"></a>&#39;RaiseEvent&#39; 문은 직접 실행 창에 유효 하지 않습니다
`RaiseEvent` 직접 실행 **창에서는** 문을 사용할 수 없습니다.  
  
 **오류 ID:** BC30739  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `RaiseEvent` 코드 편집기 **의 코드에서**문을 테스트합니다.
