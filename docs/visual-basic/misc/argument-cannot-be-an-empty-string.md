# <a name="argument-cannot-be-an-empty-string"></a>인수는 빈 문자열이 아니어야 합니다.
인수가 빈 문자열("")로 정의되었습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   인수에 유효한 값을 제공합니다.
