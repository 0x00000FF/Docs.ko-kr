# <a name="argument-39ltargumentnamegt39-cannot-be-a-multidimensional-array"></a>인수 &#39; &lt;argumentname&gt; &#39; 다차원 배열일 수 없습니다.
배열 인수의 차수는 1보다 크지만 1차원 배열만 가능합니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   인수의 차수를 1로 변경합니다.
