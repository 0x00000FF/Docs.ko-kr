# <a name="39handles39-in-modules-must-specify-a-39withevents39-variable-qualified-with-a-single-identifier"></a>&#39; 핸들 &#39; 모듈에서 지정 해야 합니다는 &#39; WithEvents &#39; 단일 식별자로 한정 된 변수
이벤트 처리기를 지정하려면 `Handles` 문이 `WithEvents` 키워드를 사용하여 선언된 개체 변수를 지정해야 합니다.  
  
 **오류 ID:** BC31418  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `WithEvents` 한정자를 사용하여 `Handles` 문에서 사용할 변수를 선언합니다.
