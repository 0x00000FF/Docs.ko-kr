# 클래스는 &#39;&lt;specifier&gt;&#39;로 선언할 수 없습니다.
잘못된 지정자를 사용하여 `Class` 또는 `Module`을 선언하려고 했습니다.  
  
 **오류 ID:** BC30461  
  
### 이 오류를 해결하려면  
  
-   지정자를 제거합니다.
