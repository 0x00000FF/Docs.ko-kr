# <a name="conversions-from-39lttypename1gt39-to-39lttypename2gt39-must-be-explicit"></a>변환할 때 &#39; &lt;typename1&gt; &#39; 를 &#39; &lt;typename2&gt; &#39; 명시적 이어야 합니다
이 변환은 암시적으로 수행할 수 없습니다.  
  
 **오류 ID:** BC30664  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `CType`또는 `CInt` 나 `CDec`등의 특정 변환 함수 중 하나를 사용합니다.
