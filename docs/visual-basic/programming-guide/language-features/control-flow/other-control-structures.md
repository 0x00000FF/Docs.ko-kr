# <a name="other-control-structures-visual-basic"></a>기타 제어 구조(Visual Basic)
Visual Basic 리소스를 삭제 하거나 개체 참조를 반복 해야 할 횟수를 줄일 제어 구조 수를 제공 합니다.  
  
## <a name="usingend-using-construction"></a>사용 하 여... 생성을 사용 하는 끝  
 `Using...End Using` 구문은 문 블록을 확인 하는 SQL 연결 같은 리소스의 사용 합니다. 사용 하 여 리소스를 선택적으로 가져올 수는 `Using` 문. 종료 하면는 `Using` 블록, Visual Basic는 리소스의 사용 하기 위해 다른 코드에 사용할 수 있도록 자동으로 삭제 합니다. 리소스는 로컬 이며 삭제할 이어야 합니다. 자세한 내용은 [using 문](../../../../visual-basic/language-reference/statements/using-statement.md)을 참조하세요.  
  
## <a name="withend-with-construction"></a>사용... 생성과 함께 종료  
 `With...End With` 생성 한 개체 참조를 한 번만 지정할 수 있습니다 다음 일련의 해당 멤버에 액세스 하는 문 실행 합니다. 이 코드를 단순화 하 고 Visual Basic 멤버에 액세스 하는 각 문에 대 한 참조를 다시 설정할 필요가 없기 때문에 성능이 향상 수 있습니다. 자세한 내용은 참조 [와... 문을 사용 하 여 종료](../../../../visual-basic/language-reference/statements/with-end-with-statement.md)합니다.
