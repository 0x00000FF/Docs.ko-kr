# XML Literals and the XML 1.0 Specification (Visual Basic)
[!INCLUDE[vs2017banner](../../../../csharp/includes/vs2017banner.md)]

[!INCLUDE[vbprvb](../../../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]의 XML 리터럴 구문은 대부분의 XML\(Extensible Markup Language\) 1.0 사양을 지원합니다.  XML 1.0 사양에 대한 자세한 내용은 W3C 웹 사이트에서 [Extensible Markup Language \(XML\) 1.0](http://go.microsoft.com/fwlink/?LinkId=73927)을 참조하십시오.  
  
## Visual Basic에서 지원하지 않는 기능  
  
-   XML 리터럴은 DTD\(문서 종류 정의\)를 포함할 수 없습니다.  
  
-   XML 문서 리터럴은 XML 문서 선언으로 시작해야 합니다.  
  
-   XML 리터럴은 한 줄에 65,535자를 초과하여 포함할 수 없습니다.  
  
-   XML 네임스페이스 접두사, 요소 이름 및 특성 이름은 1,024자를 초과하여 포함할 수 없습니다.  
  
## Visual Basic에서 지원하는 추가 기능  
  
-   문서와 요소 리터럴에 허용된 포함 식 구문이 유효하지 않은 XML입니다.
