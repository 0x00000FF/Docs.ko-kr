# <a name="creating-xml-in-visual-basic"></a>Visual Basic에서 XML 만들기
[!INCLUDE[vbprvb](../../../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]사용할 수 있도록 *XML 리터럴을* 코드에서 직접. XML 리터럴 구문을 나타내는 [!INCLUDE[sqltecxlinq](../../../../csharp/programming-guide/concepts/linq/includes/sqltecxlinq_md.md)] 개체는 XML 1.0 구문을 비슷합니다. 이렇게 하면 보다 쉽게 코드에 동일한 최종 XML 구조 때문에 XML 요소, 문서 및 조각을 프로그래밍 방식으로 만들 수 있습니다.
