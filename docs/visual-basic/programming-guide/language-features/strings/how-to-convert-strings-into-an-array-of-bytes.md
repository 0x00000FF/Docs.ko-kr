-   <xref:System.Text.Encoding.UTF7%2A?displayProperty=fullName>: UTF\-7 형식에 대한 인코딩을 가져옵니다.  
  
-   <xref:System.Text.Encoding.UTF8%2A?displayProperty=fullName>: UTF\-8 형식에 대한 인코딩을 가져옵니다.
