## <a name="current-instance"></a><span data-ttu-id="5d59f-118">현재 인스턴스</span><span class="sxs-lookup"><span data-stu-id="5d59f-118">Current Instance</span></span>

 <span data-ttu-id="5d59f-121">`Me` 현재 인스턴스를 참조 하는 개체 변수로 키워드를 적용 합니다.</span><span class="sxs-lookup"><span data-stu-id="5d59f-121">The `Me` keyword acts as an object variable referring to the current instance.</span></span> <span data-ttu-id="5d59f-122">프로시저 없으면 [공유](../../../../visual-basic/language-reference/modifiers/shared.md), 사용할 수 있습니다는 `Me` 키워드 현재 인스턴스에 대 한 포인터를 가져올 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="5d59f-122">If a procedure is not [Shared](../../../../visual-basic/language-reference/modifiers/shared.md), it can use the `Me` keyword to obtain a pointer to the current instance.</span></span> <span data-ttu-id="5d59f-123">공유 프로시저 클래스의 특정 인스턴스에 연결할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="5d59f-123">Shared procedures cannot be associated with a specific instance of a class.</span></span>  
  
 <span data-ttu-id="5d59f-124">사용 하 여 `Me` 프로시저 다른 모듈에서 현재 인스턴스를 전달 하는 데 특히 유용 합니다.</span><span class="sxs-lookup"><span data-stu-id="5d59f-124">Using `Me` is particularly useful for passing the current instance to a procedure in another module.</span></span> <span data-ttu-id="5d59f-125">예를 들어 XML 문서 수가 및 일부 표준 텍스트를 모두 추가.</span><span class="sxs-lookup"><span data-stu-id="5d59f-125">For example, suppose you have a number of XML documents and wish to add some standard text to all of them.</span></span> <span data-ttu-id="5d59f-126">다음 예제에서는이 작업을 수행 하는 절차를 정의 합니다.</span><span class="sxs-lookup"><span data-stu-id="5d59f-126">The following example defines a procedure to do this.</span></span>

  
 <span data-ttu-id="5d59f-127">그런 다음 모든 XML 문서 개체 프로시저를 호출 하 고 현재 인스턴스를 인수로 전달할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="5d59f-127">Every XML document object could then call the procedure and pass its current instance as an argument.</span></span> <span data-ttu-id="5d59f-128">다음은 이에 대한 예입니다.</span><span class="sxs-lookup"><span data-stu-id="5d59f-128">The following example demonstrates this.</span></span>
