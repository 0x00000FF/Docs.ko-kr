# Elementary Data Types (Visual Basic)
[!INCLUDE[vs2017banner](../../../../visual-basic/developing-apps/includes/vs2017banner.md)]

[!INCLUDE[vbprvb](../../../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)]에서는 다양한 프로그래밍 요소에 사용할 수 있는 미리 정의된 여러 가지 데이터 형식을 제공합니다.  이 단원에서는 이러한 데이터 형식과 사용 방법에 대해 설명합니다.  
  
> [!NOTE]
>  Visual Basic 모든 기본 데이터 형식은 구조체나 클래스에서 지원 하지는 <xref:System> 네임 스페이스입니다.  컴파일러는 각 데이터 형식 키워드를 내부 구조체나 클래스의 별칭으로 사용합니다.  예를 들어, 예약어를 사용 하 여 변수를 선언 `Byte` 같은 정규화 된 구조체 이름을 사용 하 여 선언 된 <xref:System.Byte?displayProperty=fullName>.
