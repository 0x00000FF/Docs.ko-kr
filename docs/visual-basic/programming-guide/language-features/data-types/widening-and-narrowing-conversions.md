- <span data-ttu-id="7eed9-176"><xref:System.OverflowException> -(정수 계열 형식만) 변환 된 값이 너무 커서 대상 유형에 적합 하지</span><span class="sxs-lookup"><span data-stu-id="7eed9-176"><xref:System.OverflowException> — (integral types only) if the converted value is too large for the target type</span></span>  
  
 <span data-ttu-id="7eed9-177">클래스 또는 구조를 정의 하는 경우는 [CType Function](../../../../visual-basic/language-reference/functions/ctype-function.md) 변환 연산자와 해당 클래스 또는 구조체에서 역할을 하는 `CType` 적절 하다 고 판단 되는 모든 예외를 throw 할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="7eed9-177">If a class or structure defines a [CType Function](../../../../visual-basic/language-reference/functions/ctype-function.md) to serve as a conversion operator to or from that class or structure, that `CType` can throw any exception it deems appropriate.</span></span> <span data-ttu-id="7eed9-178">또한는 `CType` Visual Basic 함수 또는 다양 한 예외를 throw 할 수 있는.NET Framework 메서드를 호출할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="7eed9-178">In addition, that `CType` might call Visual Basic functions or .NET Framework methods, which in turn could throw a variety of exceptions.</span></span>  
  
## <a name="changes-during-reference-type-conversions"></a><span data-ttu-id="7eed9-179">참조 형식 변환 중에 변경</span><span class="sxs-lookup"><span data-stu-id="7eed9-179">Changes During Reference Type Conversions</span></span>  
 <span data-ttu-id="7eed9-180">변환을 *유형을 참조* 포인터만 값을 복사 합니다.</span><span class="sxs-lookup"><span data-stu-id="7eed9-180">A conversion from a *reference type* copies only the pointer to the value.</span></span> <span data-ttu-id="7eed9-181">값 자체 복사 아니고 어떤 방식으로 변경 합니다.</span><span class="sxs-lookup"><span data-stu-id="7eed9-181">The value itself is neither copied nor changed in any way.</span></span> <span data-ttu-id="7eed9-182">변경할 수 있는 유일한 항목은 포인터를가지고 있는 변수의 데이터 형식.</span><span class="sxs-lookup"><span data-stu-id="7eed9-182">The only thing that can change is the data type of the variable holding the pointer.</span></span> <span data-ttu-id="7eed9-183">다음 예제에서는 데이터 형식에서 파생된 클래스에서 클래스의 기본 클래스로 변환 됩니다 있지만 두 변수가 가리키는 개체는 변경 되지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="7eed9-183">In the following example, the data type is converted from the derived class to its base class, but the object that both variables now point to is unchanged.</span></span>  
  
```  
' Assume class cSquare inherits from class cShape.  
Dim shape As cShape  
Dim square As cSquare = New cSquare  
' The following statement performs a widening  
' conversion from a derived class to its base class.  
shape = square  
```
