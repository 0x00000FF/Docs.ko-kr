    End Property  
  
End Class

-   `Equals`반환 `True` 두 익명 형식 인스턴스는 동일한 인스턴스 또는 다음 조건을 충족 하는 경우:

    -   동일한 순서로 동일한 이름 가진 속성이 선언 되 고 유추 된 형식입니다. 이름을 비교할 때는 대/소문자 구분 하지 않습니다.
