# Constants and Enumerations in Visual Basic
[!INCLUDE[vs2017banner](../../../../csharp/includes/vs2017banner.md)]

상수는 값 대신, 변하지 않는 의미 있는 이름을 사용하는 방법입니다.  상수는 이름이 의미하는 것처럼 응용 프로그램을 실행하는 동안 일정하게 유지되는 값을 저장합니다.  상수를 사용하여 숫자 대신 의미 있는 이름을 지정하면 코드를 이해하기 쉽게 만들 수 있습니다.  
  
 열거형은 관련 상수 집합을 사용하고 상수 값을 이름과 연결하는 편리한 방법을 제공합니다.  예를 들어, 요일과 관련된 정수 상수 집합에 대한 열거형을 선언한 다음 해당 정수 값 대신 요일 이름을 코드에 사용할 수 있습니다.
