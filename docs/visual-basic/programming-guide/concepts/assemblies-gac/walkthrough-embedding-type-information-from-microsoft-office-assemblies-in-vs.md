4.  <span data-ttu-id="05e64-119">CreateExcelWorkbook 프로젝트에 대 한 바로 가기 메뉴를 연 다음 선택 **속성**합니다.</span><span class="sxs-lookup"><span data-stu-id="05e64-119">Open the shortcut menu for the CreateExcelWorkbook project and then choose **Properties**.</span></span> <span data-ttu-id="05e64-120">선택 된 **참조** 탭 합니다. **추가** 단추를 선택합니다.</span><span class="sxs-lookup"><span data-stu-id="05e64-120">Choose the **References** tab. Choose the **Add** button.</span></span>  
  
5.  <span data-ttu-id="05e64-121">**.NET** 탭에서 `Microsoft.Office.Interop.Excel`의 최신 버전을 선택합니다.</span><span class="sxs-lookup"><span data-stu-id="05e64-121">On the **.NET** tab, choose the most recent version of `Microsoft.Office.Interop.Excel`.</span></span> <span data-ttu-id="05e64-122">예를 들어 **Microsoft.Office.Interop.Excel 14.0.0.0**입니다.</span><span class="sxs-lookup"><span data-stu-id="05e64-122">For example, **Microsoft.Office.Interop.Excel 14.0.0.0**.</span></span> <span data-ttu-id="05e64-123">**확인** 단추를 선택합니다.</span><span class="sxs-lookup"><span data-stu-id="05e64-123">Choose the **OK** button.</span></span>  
  
6.  <span data-ttu-id="05e64-124">**CreateExcelWorkbook** 프로젝트에 대한 참조 목록에서, 이전 단계에서 추가한 `Microsoft.Office.Interop.Excel`에 대한 참조를 선택합니다.</span><span class="sxs-lookup"><span data-stu-id="05e64-124">In the list of references for the **CreateExcelWorkbook** project, select the reference for `Microsoft.Office.Interop.Excel` that you added in the previous step.</span></span> <span data-ttu-id="05e64-125">**속성** 창에서 `Embed Interop Types` 속성이 `True`로 설정되어 있는지 확인합니다.</span><span class="sxs-lookup"><span data-stu-id="05e64-125">In the **Properties** window, make sure that the `Embed Interop Types` property is set to `True`.</span></span>  
  
    > [!NOTE]
    >  <span data-ttu-id="05e64-126">이 연습에서 만든 응용 프로그램은 포함된 interop 형식 정보 때문에 서로 다른 버전의 Microsoft Office에서 실행됩니다.</span><span class="sxs-lookup"><span data-stu-id="05e64-126">The application created in this walkthrough runs with different versions of Microsoft Office because of the embedded interop type information.</span></span> <span data-ttu-id="05e64-127">`Embed Interop Types` 속성을 `False`로 설정한 경우 응용 프로그램을 실행할 각 Microsoft Office 버전용 PIA를 포함해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="05e64-127">If the `Embed Interop Types` property is set to `False`, you must include a PIA for each version of Microsoft Office that the application will run with.</span></span>  
  
7.  <span data-ttu-id="05e64-128">Module1.vb 파일을 엽니다.</span><span class="sxs-lookup"><span data-stu-id="05e64-128">Open the Module1.vb file.</span></span> <span data-ttu-id="05e64-129">파일의 코드를 다음 코드로 바꿉니다.</span><span class="sxs-lookup"><span data-stu-id="05e64-129">Replace the code in the file with the following code:</span></span>  
  
    ```vb  
    Imports Excel = Microsoft.Office.Interop.Excel  
  
    Module Module1  
  
        Sub Main()  
            Dim values = {4, 6, 18, 2, 1, 76, 0, 3, 11}  
  
            CreateWorkbook(values, "C:\SampleFolder\SampleWorkbook.xls")  
        End Sub  
  
        Sub CreateWorkbook(ByVal values As Integer(), ByVal filePath As String)  
            Dim excelApp As Excel.Application = Nothing  
            Dim wkbk As Excel.Workbook  
            Dim sheet As Excel.Worksheet  
  
            Try  
                ' Start Excel and create a workbook and worksheet.  
                excelApp = New Excel.Application  
                wkbk = excelApp.Workbooks.Add()  
                sheet = CType(wkbk.Sheets.Add(), Excel.Worksheet)  
                sheet.Name = "Sample Worksheet"  
  
                ' Write a column of values.  
                ' In the For loop, both the row index and array index start at 1.  
                ' Therefore the value of 4 at array index 0 is not included.  
                For i = 1 To values.Length - 1  
                    sheet.Cells(i, 1) = values(i)  
                Next  
  
                ' Suppress any alerts and save the file. Create the directory   
                ' if it does not exist. Overwrite the file if it exists.  
                excelApp.DisplayAlerts = False  
                Dim folderPath = My.Computer.FileSystem.GetParentPath(filePath)  
                If Not My.Computer.FileSystem.DirectoryExists(folderPath) Then  
                    My.Computer.FileSystem.CreateDirectory(folderPath)  
                End If  
                wkbk.SaveAs(filePath)  
        Catch  
  
            Finally  
                sheet = Nothing  
                wkbk = Nothing  
  
                ' Close Excel.  
                excelApp.Quit()  
                excelApp = Nothing  
            End Try  
  
        End Sub  
    End Module  
    ```  
  
8.  <span data-ttu-id="05e64-130">프로젝트를 저장합니다.</span><span class="sxs-lookup"><span data-stu-id="05e64-130">Save the project.</span></span>  
  
9. <span data-ttu-id="05e64-131">Ctrl+F5를 눌러 프로젝트를 빌드하고 실행합니다.</span><span class="sxs-lookup"><span data-stu-id="05e64-131">Press CTRL+F5 to build and run the project.</span></span> <span data-ttu-id="05e64-132">예제 코드에서 지정한 위치에 Excel 통합 문서가 만들어졌는지 확인합니다(C:\SampleFolder\SampleWorkbook.xls).</span><span class="sxs-lookup"><span data-stu-id="05e64-132">Verify that an Excel workbook has been created at the location specified in the example code: C:\SampleFolder\SampleWorkbook.xls.</span></span>
