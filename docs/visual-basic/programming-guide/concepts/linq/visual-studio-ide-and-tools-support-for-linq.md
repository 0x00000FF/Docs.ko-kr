# <a name="visual-studio-ide-and-tools-support-for-linq-visual-basic"></a>Visual Studio IDE 및 도구 지원에 대 한 LINQ (Visual Basic)
Visual Studio 통합된 개발 환경 (IDE) LINQ 응용 프로그램 개발을 지 원하는 다음과 같은 기능을 제공 합니다.  
  
## <a name="object-relational-designer"></a>Object Relational Designer  
 개체 관계형 디자이너는 시각적 디자인 도구에서 사용할 수 있는 [LINQ to SQL](https://msdn.microsoft.com/library/bb386976) 기본 데이터베이스의 관계형 데이터를 표시 하는 응용 프로그램을 Visual basic에서 클래스를 생성 합니다. 자세한 내용은 참조 [LINQ to SQL 도구 Visual Studio에서](https://docs.microsoft.com/visualstudio/data-tools/linq-to-sql-tools-in-visual-studio2)합니다.  
  
## <a name="sqlmetal-command-line-tool"></a>SQLMetal 명령줄 도구  
 SQLMetal은 클래스를 생성 하려면 LINQ에서 사용 하기 위해 기존 데이터베이스에서 SQL 응용 프로그램을 빌드 프로세스에서 사용할 수 있는 명령줄 도구입니다. 자세한 내용은 참조 [SqlMetal.exe (코드 생성 도구)](http://msdn.microsoft.com/library/819e5a96-7646-4fdb-b14b-fe31221b0614)합니다.  
  
## <a name="linq-aware-code-editor"></a>LINQ 인식 코드 편집기  
 Visual Basic 코드 편집기는 IntelliSense 및 서식 지정 기능으로 광범위 하 게 LINQ를 지원합니다.  
  
## <a name="visual-studio-debugger-support"></a>Visual Studio 디버거 지원  
 Visual Studio 디버거는 쿼리 식의 디버깅을 지원 합니다. 자세한 내용은 참조 [LINQ 디버깅](https://docs.microsoft.com/visualstudio/debugger/debugging-linq)합니다.
