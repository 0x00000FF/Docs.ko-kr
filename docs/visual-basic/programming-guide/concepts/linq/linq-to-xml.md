# <a name="linq-to-xml-visual-basic"></a>LINQ to XML (Visual Basic)
LINQ to XML LINQ (language-integrated Query) 프레임 워크를 활용 하는 메모리 내 XML 프로그래밍 인터페이스를 제공 합니다. LINQ to XML에는 최신.NET Framework 언어 기능을 사용 하 여 되며 업데이트 되 고 다시 디자인 된 문서 개체 모델 (DOM) XML 프로그래밍 인터페이스에 비교할 수 있습니다.  
  
 LINQ 제품군의 기술 (LINQ to Objects) 개체, 관계형 데이터베이스 (LINQ to SQL) 및 XML (LINQ to XML)에 대 한 일관성 있는 쿼리 환경을 제공합니다.
