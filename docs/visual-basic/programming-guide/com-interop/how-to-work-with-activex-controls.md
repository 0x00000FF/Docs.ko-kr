# How to: Work with ActiveX Controls (Visual Basic)
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

ActiveX 컨트롤은 다른 사용자가 프로그래밍한 패키지로 만들어진 기능을 다시 사용하기 위해 웹 페이지나 다른 응용 프로그램에 삽입할 수 있는 COM 구성 요소 또는 개체입니다.  Visual Basic 6.0 및 이전 버전용으로 개발된 ActiveX 컨트롤을 사용하여 [!INCLUDE[vsprvs](../../../csharp/includes/vsprvs_md.md)]의 **도구 상자**에 기능을 추가할 수 있습니다.  
  
### 도구 상자에 ActiveX 컨트롤을 추가하려면  
  
1.  **도구** 메뉴에서 **도구 상자 항목 선택**을 클릭합니다.  
  
     **도구 상자 선택** 대화 상자가 표시됩니다.  
  
2.  **COM 구성 요소** 탭을 클릭합니다.  
  
3.  사용할 ActiveX 컨트롤 옆에 있는 확인란을 선택한 다음 **확인**을 클릭합니다.  
  
     **도구 상자**에 다른 도구가 있는 새 컨트롤이 나타납니다.  
  
    > [!NOTE]
    >  Aximp 유틸리티를 사용하여 수동으로 ActiveX 컨트롤용 interop 어셈블리를 만들 수 있습니다.  자세한 내용은 [Aximp.exe \(Windows Forms ActiveX 컨트롤 가져오기\)](../Topic/Aximp.exe%20\(Windows%20Forms%20ActiveX%20Control%20Importer\).md)을 참조하십시오.
