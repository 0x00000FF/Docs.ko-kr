<span data-ttu-id="dc2bc-114">패키지가 복원되었는지 확인한 후 라이브러리가 있는 디렉터리로 이동할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="dc2bc-114">After ensuring packages are restored, you can navigate to the directory where a library lives:</span></span>
