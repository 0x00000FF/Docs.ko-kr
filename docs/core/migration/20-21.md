* <span data-ttu-id="ac820-109">[대상 프레임워크](../../standard/frameworks.md) 값을 `netcoreapp2.0`에서 `netcoreapp2.1`로 변경합니다.</span><span class="sxs-lookup"><span data-stu-id="ac820-109">Change the [target framework](../../standard/frameworks.md) value from `netcoreapp2.0` to `netcoreapp2.1`.</span></span> <span data-ttu-id="ac820-110">대상 프레임워크는 `<TargetFramework>` 또는 `<TargetFrameworks>` 요소로 정의됩니다.</span><span class="sxs-lookup"><span data-stu-id="ac820-110">The target framework is defined by the `<TargetFramework>` or `<TargetFrameworks>` element.</span></span>

  * <span data-ttu-id="ac820-115">[dotnet-user-secrets](https://github.com/aspnet/DotNetTools/blob/master/src/dotnet-user-secrets/README.md)(Microsoft.Extensions.SecretManager.Tools)</span><span class="sxs-lookup"><span data-stu-id="ac820-115">[dotnet-user-secrets](https://github.com/aspnet/DotNetTools/blob/master/src/dotnet-user-secrets/README.md) (Microsoft.Extensions.SecretManager.Tools)</span></span>
  * <span data-ttu-id="ac820-116">[dotnet-sql-cache](https://github.com/aspnet/DotNetTools/blob/master/src/dotnet-sql-cache/README.md)(Microsoft.Extensions.Caching.SqlConfig.Tools)</span><span class="sxs-lookup"><span data-stu-id="ac820-116">[dotnet-sql-cache](https://github.com/aspnet/DotNetTools/blob/master/src/dotnet-sql-cache/README.md) (Microsoft.Extensions.Caching.SqlConfig.Tools)</span></span>
  * <span data-ttu-id="ac820-117">[dotnet-ef](/ef/core/miscellaneous/cli/dotnet)(Microsoft.EntityFrameworkCore.Tools.DotNet)</span><span class="sxs-lookup"><span data-stu-id="ac820-117">[dotnet-ef](/ef/core/miscellaneous/cli/dotnet) (Microsoft.EntityFrameworkCore.Tools.DotNet)</span></span>
  
  <span data-ttu-id="ac820-118">이전 .NET Core SDK 버전에서는 프로젝트 파일에 있는 이러한 도구 중 하나에 대한 참조는 다음 예제와 유사합니다.</span><span class="sxs-lookup"><span data-stu-id="ac820-118">In previous .NET Core SDK versions, the reference to one of these tools in your project file looks similar to the following example:</span></span>


  <span data-ttu-id="ac820-119">이 항목은 .NET Core SDK에서 더 이상 사용되지 않으므로, 프로젝트에 이러한 번들 도구 중 하나에 대한 참조가 있는 경우 다음과 유사한 경고가 표시됩니다.</span><span class="sxs-lookup"><span data-stu-id="ac820-119">Since this entry isn't used by the the .NET Core SDK any longer, you'll see a warning similar to the following if you still have references to one of these bundled tools in your project:</span></span>

  <span data-ttu-id="ac820-120">프로젝트에서 해당 도구에 대한 `<DotNetCliToolReference>` 참조를 제거하면 이 문제가 해결됩니다.</span><span class="sxs-lookup"><span data-stu-id="ac820-120">Removing the `<DotNetCliToolReference>` references for those tools from your project file fixes this issue.</span></span>
