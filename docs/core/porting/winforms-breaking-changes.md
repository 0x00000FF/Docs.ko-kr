# <a name="breaking-changes-in-windows-forms-net-framework-to-net-core"></a>Windows Forms의 호환성이 손상되는 변경(.NET Framework에서 .NET Core로 변경)

WPF 및 Windows Forms 지원이 버전 3.0의 .NET Core에 추가되었습니다. Windows Forms 또는 Windows Presentation Foundation 앱을 .NET Framework에서 .NET Core로 마이그레이션하는 경우, 다음 문서에 표시된 호환성이 손상되는 변경이 앱에 영향을 줄 수 있습니다.

호환성이 손상되는 변경 사항이 도입된 .NET Core 버전에 따라 그룹화됩니다.
