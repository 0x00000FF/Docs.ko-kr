<span data-ttu-id="b2390-113">MSBuild에 익숙한 경우 이미 존재하는 다른 참조 형식에 익숙할 것입니다.</span><span class="sxs-lookup"><span data-stu-id="b2390-113">If you are familiar with MSBuild, it will look familiar to the other reference types that already exist.</span></span> <span data-ttu-id="b2390-114">키는 프로젝트에 추가할 패키지 ID를 지정하는 `Include` 문입니다.</span><span class="sxs-lookup"><span data-stu-id="b2390-114">The key is the `Include` statement which specifies the package id that you wish to add to the project.</span></span> <span data-ttu-id="b2390-115">`<Version>` 자식 요소는 가져올 버전을 지정합니다.</span><span class="sxs-lookup"><span data-stu-id="b2390-115">The `<Version>` child element specifies the version to get.</span></span> <span data-ttu-id="b2390-116">버전은 [NuGet 버전 규칙](/nuget/create-packages/dependency-versions#version-ranges)에 따라 지정됩니다.</span><span class="sxs-lookup"><span data-stu-id="b2390-116">The versions are specified as per [NuGet version rules](/nuget/create-packages/dependency-versions#version-ranges).</span></span>
