# <a name="breaking-changes-for-migration-from-version-30-preview-6-to-30-preview-7"></a>버전 3.0 미리 보기 6에서 3.0 미리 보기 7로 마이그레이션 시 호환성이 손상되는 변경 사항

> [!IMPORTANT]
> 이 문서는 작성 중입니다. 이것은 .NET Core 호환성이 손상되는 변경 사항의 완전한 목록이 아닙니다. .NET Core 호환성이 손상되는 변경 사항에 대한 자세한 내용은 GitHub의 dotnet/docs 리포지토리에 있는 개별 [호환성이 손상되는 변경 문제](https://github.com/dotnet/docs/issues?q=is%3Aissue+is%3Aopen+label%3Abreaking-change)에서 검토할 수 있습니다. 

.NET Core, ASP.NET Core 또는 EF Core 3.0 미리 보기 6에서 3.0 미리 보기 7로 마이그레이션하는 경우, 다음 항목에서 앱에 영향을 줄 수 있는 호환성이 손상되는 변경 사항을 검토하세요.
