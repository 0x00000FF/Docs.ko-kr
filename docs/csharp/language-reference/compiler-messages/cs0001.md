# Compiler Error CS0001
내부 컴파일러 오류입니다.  
  
 예기치 않은 구문을 구문 분석할 수 없기 때문에 컴파일러가 실패하는지 여부를 확인해 봅니다.  이 오류가 되풀이하여 발생하면 Microsoft에 문의하십시오.
