# <a name="compiler-error-cs1936"></a><span data-ttu-id="23118-102">컴파일러 오류 CS1936</span><span class="sxs-lookup"><span data-stu-id="23118-102">Compiler Error CS1936</span></span>

<span data-ttu-id="23118-103">소스 형식 'type'에 대해 구현된 쿼리 패턴을 찾을 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="23118-103">Could not find an implementation of the query pattern for source type 'type'.</span></span>  <span data-ttu-id="23118-104">'method'를 찾을 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="23118-104">'method' not found.</span></span>  
  
<span data-ttu-id="23118-105">소스 형식을 쿼리하려면 사용자가 쿼리에서 호출하는 표준 쿼리 연산자 메서드를 해당 형식에서 구현해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="23118-105">In order to query a source type, that type must implement the standard query operator methods that you are invoking in the query.</span></span> <span data-ttu-id="23118-106">적절한 `using` 지시문을 포함하는 범위에 표시되는 클래스 멤버 또는 확장 메서드 형식으로 구현할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="23118-106">The implementation can be either in the form of class members or extension methods that are brought into scope with the appropriate `using` directive.</span></span>  

## <a name="to-correct-this-error"></a><span data-ttu-id="23118-107">이 오류를 해결하려면</span><span class="sxs-lookup"><span data-stu-id="23118-107">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="23118-110">예제</span><span class="sxs-lookup"><span data-stu-id="23118-110">Example</span></span>

<span data-ttu-id="23118-111">다음 예제에서는 CS1936 오류가 발생하는 경우를 보여 줍니다.</span><span class="sxs-lookup"><span data-stu-id="23118-111">The following example produces CS1936:</span></span>  

```csharp
// cs1936.cs  
using System.Collections;  
using System.Linq;  
class Test  
{  
    static int Main()  
    {  
        object obj;  
        IEnumerable e = from x in obj // CS1936  
                        select x;  
        return 0;  
    }  
}  
```

<span data-ttu-id="23118-112">일반적으로 이 오류는 이러한 개체 컬렉션 대신 실수로 일부 형식의 개체를 쿼리하는 경우에 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="23118-112">This error typically occurs when you accidentally try to query an object of some type instead of a collection of those objects.</span></span>
