# Compiler Error CS0178
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

잘못된 차수 지정자: ',' 또는 '\]'가 필요합니다.  
  
 배열 초기화가 잘못되었습니다.  예를 들어 배열 차원을 지정할 때 다음과 같이 지정할 수 있습니다.  
  
-   대괄호에 숫자를 하나 넣습니다.  
  
-   대괄호를 비웁니다.  
  
-   쉼표를 대괄호로 묶습니다.  
  
 자세한 내용은 [배열](../../../csharp/programming-guide/arrays/index.md) 및 C\# 사양\([C\# 언어 사양](../../../visual-basic/reference/language-specification.md)\)에서 배열 이니셜라이저 단원을 참조하십시오.  
  
## 예제  
 다음 샘플에서는 CS0178 오류가 발생하는 경우를 보여 줍니다.  
  
```  
// CS0178.cs  
class MyClass  
{  
   public static void Main()  
   {  
      int a = new int[5][,][][5;   // CS0178  
      int[,] b = new int[3,2];   // OK  
  
      int[][] c = new int[10][];  
      c[0] = new int[5][5];   // CS0178  
      c[0] = new int[2];   // OK  
      c[1] = new int[2]{1,2};   // OK  
   }  
}  
```
