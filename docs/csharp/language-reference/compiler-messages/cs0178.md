## <a name="example"></a><span data-ttu-id="11fcc-110">예</span><span class="sxs-lookup"><span data-stu-id="11fcc-110">Example</span></span>

```csharp  
   {  
      int a = new int[5][,][][5;   // CS0178

      c[1] = new int[2]{1,2};   // OK  
   }  
}  
```
