## <a name="to-correct-this-error"></a><span data-ttu-id="54f17-109">이 오류를 해결하려면</span><span class="sxs-lookup"><span data-stu-id="54f17-109">To correct this error</span></span>

- <span data-ttu-id="54f17-111">클래스 `B`가 `A`로부터 상속하고 `MyClass`가 `B`로부터 상속하도록 합니다.</span><span class="sxs-lookup"><span data-stu-id="54f17-111">Make class `B` inherit from `A`, and `MyClass` inherit from `B`:</span></span>

    ```csharp
    public class MyClass : A, B {}
    ```
