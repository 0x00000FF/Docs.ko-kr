# <a name="compiler-error-cs0039"></a><span data-ttu-id="6a2cd-102">컴파일러 오류 CS0039</span><span class="sxs-lookup"><span data-stu-id="6a2cd-102">Compiler Error CS0039</span></span>
<span data-ttu-id="6a2cd-103">참조 변환, boxing 변환, unboxing 변환, 래핑 변환 또는 null 형식 변환을 통해 “type1” 형식을 “type2”(으)로 변환할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="6a2cd-103">Cannot convert type 'type1' to 'type2' via a reference conversion, boxing conversion, unboxing conversion, wrapping conversion, or null type conversion</span></span>  
  
 <span data-ttu-id="6a2cd-104">[as](../../../csharp/language-reference/keywords/as.md) 연산자를 사용한 변환은 상속, 참조 변환 및 boxing 변환에서 허용됩니다.</span><span class="sxs-lookup"><span data-stu-id="6a2cd-104">A conversion with the [as](../../../csharp/language-reference/keywords/as.md) operator is allowed by inheritance, reference conversions, and boxing conversions.</span></span> <span data-ttu-id="6a2cd-105">자세한 내용은 [변환 연산자](../../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md)를 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="6a2cd-105">For more information, see [Conversion Operators](../../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="6a2cd-106">예제</span><span class="sxs-lookup"><span data-stu-id="6a2cd-106">Example</span></span>  
 <span data-ttu-id="6a2cd-107">다음 예제에서는 CS0039를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="6a2cd-107">The following example generates CS0039.</span></span>  
  
```csharp  
// CS0039.cs  
using System;  
class A  
{  
}  
class B: A  
{  
}  
class C: A  
{  
}  
class M  
{  
    static void Main()  
    {  
        A a = new C();  
        B b = new B();  
        C c;  
  
        // This is valid; there is a built-in reference  
        // conversion from A to C.  
        c = a as C;    
  
        //The following generates CS0039; there is no  
        // built-in reference conversion from B to C.  
        c = b as C;  // CS0039  
    }  
}  
```
