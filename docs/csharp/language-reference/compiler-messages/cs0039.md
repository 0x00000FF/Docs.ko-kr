# <a name="compiler-error-cs0039"></a><span data-ttu-id="733a4-102">컴파일러 오류 CS0039</span><span class="sxs-lookup"><span data-stu-id="733a4-102">Compiler Error CS0039</span></span>
<span data-ttu-id="733a4-103">참조 변환, boxing 변환, unboxing 변환, 래핑 변환 또는 null 형식 변환을 통해 “type1” 형식을 “type2”(으)로 변환할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="733a4-103">Cannot convert type 'type1' to 'type2' via a reference conversion, boxing conversion, unboxing conversion, wrapping conversion, or null type conversion</span></span>

## <a name="example"></a><span data-ttu-id="733a4-105">예</span><span class="sxs-lookup"><span data-stu-id="733a4-105">Example</span></span>

<span data-ttu-id="733a4-106">다음 예제에서는 CS0039를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="733a4-106">The following example generates CS0039.</span></span>  
  
```csharp
using System;

class A { }
class B : A { }
class C : A { }

class Example
{
    static void Main()
    {
        C c;

        // This compiles, because
        // there is an explicit reference conversion from type A to type C.
        A a = new C();
        c = a as C;

        // This generates CS0039, because be there is no  
        // there is no implicit or explicit reference conversion between B and C types.
        B b = new B();
        c = b as C;  // CS0039
    }
}
```
