# <a name="compiler-error-cs0579"></a>컴파일러 오류 CS0579
'attribute' 특성이 중복되었습니다.  
  
 특성의 **AttributeUsage**에 [AllowMultiple=true](../../language-reference/attributes/general.md)가 지정되지 않은 경우 동일한 특성을 두 번 이상 지정할 수 없습니다.  
  
## <a name="example"></a>예제  
 다음 예제에서는 CS0579를 생성합니다.  
  
```csharp  
// CS0579.cs  
using System;  
public class MyAttribute : Attribute  
{  
}  
  
[AttributeUsage(AttributeTargets.All,AllowMultiple=true)]  
public class MyAttribute2 : Attribute  
{  
}  
  
public class z  
{  
    [MyAttribute, MyAttribute]     // CS0579  
    public void zz()  
    {  
    }  
  
    [MyAttribute2, MyAttribute2]   // OK  
    public void zzz()  
    {  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
