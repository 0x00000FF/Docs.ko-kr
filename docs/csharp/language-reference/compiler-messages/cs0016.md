# Compiler Error CS0016
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

'file' 출력 파일에 쓸 수 없습니다. 'reason'  
  
 컴파일러가 출력 파일에 쓰지 못했습니다.  파일의 경로를 검사하여 파일이 존재하는지 확인합니다.  그 위치에 이전에 빌드된 파일이 이미 있는 경우 파일이 쓰기 가능하고 파일에 잠금을 설정한 프로세스가 없는지 확인합니다.  예를 들어, 빌드할 때 실행 파일이 실행되고 있지 않아야 합니다.
