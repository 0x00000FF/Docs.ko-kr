 <span data-ttu-id="d7c17-104">이 오류는 코드의 식에 접근할 수 없는 부분이 있을 때마다 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="d7c17-104">This error occurs whenever part of an expression in your code is unreachable.</span></span> <span data-ttu-id="d7c17-105">다음 예제에서는 `&&` 연산의 왼쪽이 항상 false라는 사실 때문에 `myTest()` 메서드가 평가되지 않으므로 `false && myTest()` 조건이 이 기준에 부합합니다.</span><span class="sxs-lookup"><span data-stu-id="d7c17-105">In the following example, the condition `false && myTest()` meets this criteria because the `myTest()` method will never get evaluated due to the fact that the left side of the `&&` operation is always false.</span></span> <span data-ttu-id="d7c17-106">`&&` 연산자는 `false` 문을 false로 평가하는 즉시 평가를 중지하고 오른쪽을 평가하지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="d7c17-106">As soon as the `&&` operator evaluates the `false` statement as false, it stops the evaluation, and will never evaluate the right side.</span></span>
