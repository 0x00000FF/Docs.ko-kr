# <a name="compiler-error-cs0507"></a><span data-ttu-id="948c7-102">컴파일러 오류 CS0507</span><span class="sxs-lookup"><span data-stu-id="948c7-102">Compiler Error CS0507</span></span>
<span data-ttu-id="948c7-103">‘function1’: ‘access’ 상속된 ‘function2’ 멤버를 재정의할 때 액세스 한정자를 변경할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="948c7-103">'function1' : cannot change access modifiers when overriding 'access' inherited member 'function2'</span></span>  
  
 <span data-ttu-id="948c7-104">메서드 재정의 액세스 사양을 변경하려고 했습니다.</span><span class="sxs-lookup"><span data-stu-id="948c7-104">An attempt was made to change the access specification in a method override.</span></span>  
  
## <a name="example"></a><span data-ttu-id="948c7-105">예제</span><span class="sxs-lookup"><span data-stu-id="948c7-105">Example</span></span>  
 <span data-ttu-id="948c7-106">다음 샘플에서는 CS0507을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="948c7-106">The following sample generates CS0507.</span></span>  
  
```csharp  
// CS0507.cs  
abstract public class clx  
{  
   virtual protected void f() {}  
}  
  
public class cly : clx  
{  
   public override void f() {}   // CS0507  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="948c7-107">예제</span><span class="sxs-lookup"><span data-stu-id="948c7-107">Example</span></span>  
 <span data-ttu-id="948c7-108">클래스가 참조된 메타데이터에서 정의된, `protected internal`로 표시된 메서드를 재정의하려고 하는 경우에도 CS0507이 발생할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="948c7-108">CS0507 can also occur if a class attempts to override a method marked as `protected internal` defined in referenced metadata.</span></span> <span data-ttu-id="948c7-109">이 경우 재정의 메서드를 `protected`로 표시해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="948c7-109">In this situation, the overriding method should be marked as `protected`.</span></span>  
  
```csharp  
// CS0507_b.cs  
// compile with: /target:library  
abstract public class clx  
{  
   virtual protected internal void f() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="948c7-110">예제</span><span class="sxs-lookup"><span data-stu-id="948c7-110">Example</span></span>  
 <span data-ttu-id="948c7-111">다음 샘플에서는 CS0507을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="948c7-111">The following sample generates CS0507.</span></span>
