<span data-ttu-id="2123f-103">‘struct1’ 형식의 ‘struct2 field’ 구조체 멤버는 구조체 레이아웃에서 순환됩니다.</span><span class="sxs-lookup"><span data-stu-id="2123f-103">Struct member 'struct2 field' of type 'struct1' causes a cycle in the struct layout</span></span>

 <span data-ttu-id="2123f-108">다음 샘플에서는 CS0523을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="2123f-108">The following sample generates CS0523:</span></span>  
  
```csharp

{  
   public RecursiveLayoutStruct2 field;  
}

{  
   public RecursiveLayoutStruct1 field;   // CS0523  
}  
```
