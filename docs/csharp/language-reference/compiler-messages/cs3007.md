# <a name="compiler-warning-level-1-cs3007"></a><span data-ttu-id="3226b-102">컴파일러 경고(수준 1) CS3007</span><span class="sxs-lookup"><span data-stu-id="3226b-102">Compiler Warning (level 1) CS3007</span></span>
<span data-ttu-id="3226b-103">명명되지 않은 배열 형식만 다른 오버로드된 ‘method’ 메서드는 CLS 규격이 아닙니다.</span><span class="sxs-lookup"><span data-stu-id="3226b-103">Overloaded method 'method' differing only by unnamed array types is not CLS-compliant</span></span>  
  
 <span data-ttu-id="3226b-104">이 오류는 오버로드된 메서드에서 인수로 가변 배열을 사용하고 메서드 시그니처에서 배열의 요소 형식만 다른 경우에 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="3226b-104">This error occurs if you have an overloaded method that takes a jagged array and the only difference between the method signatures is the element type of the array.</span></span> <span data-ttu-id="3226b-105">이 오류를 해결하려면 가변 배열 대신 사각형 배열을 사용하고, 함수 호출의 모호성을 없애기 위해 추가적인 매개 변수를 사용하고, 오버로드된 메서드 중 하나 이상의 이름을 바꾸거나, CLS 규격이 필요하지 않은 경우에는 <xref:System.CLSCompliantAttribute> 속성을 제거하세요.</span><span class="sxs-lookup"><span data-stu-id="3226b-105">To avoid this error, consider using a rectangular array rather than a jagged array; use an additional parameter to disambiguate the function call; rename one or more of the overloaded methods; or, if CLS Compliance is not needed, remove the <xref:System.CLSCompliantAttribute> attribute.</span></span> <span data-ttu-id="3226b-106">CLS 규격에 대한 자세한 내용은 [언어 독립성 및 언어 독립적 구성 요소](https://msdn.microsoft.com/library/12a7a7h3)를 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="3226b-106">For more information on CLS Compliance, see [Language Independence and Language-Independent Components](https://msdn.microsoft.com/library/12a7a7h3).</span></span>  
  
## <a name="example"></a><span data-ttu-id="3226b-107">예제</span><span class="sxs-lookup"><span data-stu-id="3226b-107">Example</span></span>  
 <span data-ttu-id="3226b-108">다음 예제에서는 CS3007 경고가 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="3226b-108">The following example generates CS3007:</span></span>  
  
```csharp  
// CS3007.cs  
[assembly: System.CLSCompliant(true)]  
public struct S  
{  
    public void F(int[][] array) { }  
    public void F(byte[][] array) { }  // CS3007  
    // Try this instead:  
    // public void F1(int[][] array) {}  
    // public void F2(byte[][] array) {}  
    // or   
    // public void F(int[,] array) {}  
    // public void F(byte[,] array) {}  
}  
```
