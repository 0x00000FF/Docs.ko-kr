## <a name="example"></a><span data-ttu-id="eadbe-105">예제</span><span class="sxs-lookup"><span data-stu-id="eadbe-105">Example</span></span>  
 <span data-ttu-id="eadbe-106">다음 예제에서는 CS1018을 생성하고 오류를 해결하는 여러 가지 방법을 제안합니다.</span><span class="sxs-lookup"><span data-stu-id="eadbe-106">The following example generates CS1018, and suggests several ways to resolve the error:</span></span>  
  
```csharp  
// CS1018.cs  
public class C  
{  
}  
  
public class a : C  
{  
    public a(int i)  
    {  
    }  
  
    public a () :   // CS1018  
    // possible resolutions:  
    // public a () resolves by removing the colon  
    // public a () : base() calls C's default constructor  
    // public a () : this(1) calls the assignment constructor of class a  
    {  
    }  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
