## <a name="example"></a><span data-ttu-id="82e72-108">예</span><span class="sxs-lookup"><span data-stu-id="82e72-108">Example</span></span>  
 <span data-ttu-id="82e72-109">다음 샘플에서는 CS0122를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="82e72-109">The following sample generates CS0122:</span></span>  
  
```csharp  
// CS0122.cs  
public class MyClass  
{  
    // Make public to resolve CS0122  
    void MyMethod()  
    {  
    }  
}  
  
public class MyClass2  
{  
    public static int Main()  
    {  
        MyClass a = new MyClass();  
        // MyMethod is private  
        a.MyMethod();   // CS0122  
        return 0;  
   }  
}  
```
