## <a name="example"></a><span data-ttu-id="fd25f-108">예제</span><span class="sxs-lookup"><span data-stu-id="fd25f-108">Example</span></span>  
 <span data-ttu-id="fd25f-109">다음 샘플에서는 CS0122를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="fd25f-109">The following sample generates CS0122:</span></span>  
  
```csharp  
// CS0122.cs  
public class MyClass  
{  
    // Make public to resolve CS0122  
    void MyMethod()  
    {  
    }  
}  
  
public class MyClass2  
{  
    public static int Main()  
    {  
        MyClass a = new MyClass();  
        // MyMethod is private  
        a.MyMethod();   // CS0122  
        return 0;  
   }  
}  
```
