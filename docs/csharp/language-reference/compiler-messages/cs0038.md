# <a name="compiler-error-cs0038"></a><span data-ttu-id="c9076-102">컴파일러 오류 CS0038</span><span class="sxs-lookup"><span data-stu-id="c9076-102">Compiler Error CS0038</span></span>
<span data-ttu-id="c9076-103">중첩 형식 ‘type2’을(를) 통해 외부 형식 ‘type1’의 static이 아닌 멤버에 액세스할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="c9076-103">Cannot access a nonstatic member of outer type 'type1' via nested type 'type2'</span></span>  
  
 <span data-ttu-id="c9076-104">중첩된 클래스는 클래스의 필드를 자동으로 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="c9076-104">A field in a class is not automatically available to a nested class.</span></span> <span data-ttu-id="c9076-105">중첩된 클래스가 사용할 수 있으려면 필드가 [static](../../../csharp/language-reference/keywords/static.md)이어야 합니다.</span><span class="sxs-lookup"><span data-stu-id="c9076-105">To be available to a nested class, the field must be [static](../../../csharp/language-reference/keywords/static.md).</span></span> <span data-ttu-id="c9076-106">static이 아니면 외부 클래스의 인스턴스를 만들어야 합니다.</span><span class="sxs-lookup"><span data-stu-id="c9076-106">Otherwise, you must create an instance of the outer class.</span></span> <span data-ttu-id="c9076-107">자세한 내용은 [중첩 형식](../../../csharp/programming-guide/classes-and-structs/nested-types.md)을 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="c9076-107">For more information, see [Nested Types](../../../csharp/programming-guide/classes-and-structs/nested-types.md).</span></span>  
  
 <span data-ttu-id="c9076-108">다음 샘플에서는 CS0038을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="c9076-108">The following sample generates CS0038:</span></span>  
  
```csharp  
// CS0038.cs  
class OuterClass  
{  
   public int count;  
   // try the following line instead  
   // public static int count;  
  
   class InnerClass  
   {  
      void func()  
      {  
         // or, create an instance  
         // OuterClass class_inst = new OuterClass();  
         // int count2 = class_inst.count;  
         int count2 = count;   // CS0038  
      }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
