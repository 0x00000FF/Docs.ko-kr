## <a name="example"></a><span data-ttu-id="ac5ad-108">예제</span><span class="sxs-lookup"><span data-stu-id="ac5ad-108">Example</span></span>  
 <span data-ttu-id="ac5ad-109">이 코드 샘플은 .dll로 컴파일하는 .cpp 파일과 해당 .dll을 사용하는 .cs 파일로 구성됩니다.</span><span class="sxs-lookup"><span data-stu-id="ac5ad-109">This code sample consists of a .cpp file, which compiles to a .dll, and a .cs file, which uses that .dll.</span></span> <span data-ttu-id="ac5ad-110">다음 코드는 .dll 파일용이며 .cs 파일에 있는 코드에서 액세스할 속성을 정의합니다.</span><span class="sxs-lookup"><span data-stu-id="ac5ad-110">The following code is for the .dll file and defines a property to be accessed by the code in the .cs file.</span></span>  
  
```cpp  
// CPP1546.cpp  
// compile with: /clr /LD  
using namespace System;  
public ref class MCPP  
{  
public:  
    property int Prop [int,int]  
    {  
        int get( int i, int b )  
        {  
            return i;  
        }  
    }  
};  
```
