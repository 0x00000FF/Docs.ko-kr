# C# Compiler Errors
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

일부 C\# 컴파일러 오류에는 오류의 원인을 설명하는 항목이 있으며 오류 해결 방법을 설명하는 항목이 있는 경우도 있습니다.  특정 오류 메시지에 대한 도움말이 있는지 보려면 다음 단계 중 하나를 수행합니다.  
  
-   오류 번호 \(예: CS0029\)를 [출력 창](/visual-studio/ide/reference/output-window)에서 찾고, MSDN에서 검색합니다.  
  
-   오류 번호 \(예: CS0029\)를 [출력 창](/visual-studio/ide/reference/output-window)에서 선택하고, F1 키를 선택합니다.  
  
-   색인에서, **찾을 대상** 상자에 오류 번호를 입력합니다.  
  
 어떠한 단계도 해당 오류에 대한 정보를 이끌지 않는 다면, 이 페이지의 끝으로 이동하고, 오류의 번호 또는 텍스트를 포함하는 피드백을 보냅니다.  
  
 C\#의 오류 및 경고 옵션을 구성하는 방법에 자세한 내용은 [프로젝트 디자이너, 빌드 페이지\(C\#\)](/visual-studio/ide/reference/build-page-project-designer-csharp)를 참조하십시오.
