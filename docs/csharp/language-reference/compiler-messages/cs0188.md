# <a name="compiler-error-cs0188"></a><span data-ttu-id="e6389-102">컴파일러 오류 CS0188</span><span class="sxs-lookup"><span data-stu-id="e6389-102">Compiler Error CS0188</span></span>
<span data-ttu-id="e6389-103">모든 필드에 값을 할당하기 전에는 ‘this’ 개체를 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="e6389-103">The 'this' object cannot be used before all of its fields are assigned to</span></span>  
  
 <span data-ttu-id="e6389-104">생성자가 `struct`의 모든 필드를 할당해야 생성자가 `struct`의 메서드를 호출할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="e6389-104">All fields in a `struct` have to be assigned by a constructor before the constructor can call a method in the `struct`.</span></span>  
  
 <span data-ttu-id="e6389-105">구조체 생성자의 속성을 초기화하려고 할 때 이 오류가 나타날 경우 해결 방법은 생성자 매개 변수를 변경하여 속성 자체 대신 지원 필드를 지정하는 것입니다.</span><span class="sxs-lookup"><span data-stu-id="e6389-105">If you see this error when trying to initialize a property in a struct constructor, the solution is to change the constructor parameter to specify the backing field instead of the property itself.</span></span> <span data-ttu-id="e6389-106">자동 구현 속성은 지원 필드가 없기 때문에 생성자에서 어떤 방식으로든 초기화할 수 없으므로 구조체에 사용하지 마세요.</span><span class="sxs-lookup"><span data-stu-id="e6389-106">Auto-implemented properties should be avoided in structs because they have no backing field and therefore cannot be initialized in any way from the constructor.</span></span>  
  
 <span data-ttu-id="e6389-107">자세한 내용은 [구조체 사용](../../../csharp/programming-guide/classes-and-structs/using-structs.md)을 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="e6389-107">For more information, see [Using Structs](../../../csharp/programming-guide/classes-and-structs/using-structs.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="e6389-108">예제</span><span class="sxs-lookup"><span data-stu-id="e6389-108">Example</span></span>  
 <span data-ttu-id="e6389-109">다음 샘플에서는 CS0188을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="e6389-109">The following sample generates CS0188:</span></span>  
  
```csharp  
// CS0188.cs  
// compile with: /t:library  
namespace MyNamespace  
{  
    class MyClass  
    {  
        struct S  
        {  
            public int a;  
  
            void MyMethod()  
            {  
            }  
  
            S(int i)  
            {  
                // a = i;  
                MyMethod();  // CS0188  
            }  
        }  
        public static void Main()  
        { }  
  
    }  
}  
```
