# Compiler Error CS0006
'dll\_name' 메타데이터 파일을 찾을 수 없습니다.  
  
 프로그램을 컴파일했고 메타데이터를 포함하고 있는 파일의 이름을 명시적으로 전달했지만 .dll을 찾을 수 없습니다.  자세한 내용은 [\/reference \(Import Metadata\)](../../../csharp/language-reference/compiler-options/reference-compiler-option.md)을 참조하십시오.
