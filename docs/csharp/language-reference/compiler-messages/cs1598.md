# Compiler Warning (level 1) CS1598
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

다음 이유로 인해 XML 파서를 로드할 수 없습니다. 'reason'XML 문서 파일 'file'이\(가\) 생성되지 않습니다.  
  
 [\/doc](../../../csharp/language-reference/compiler-options/doc-compiler-option.md) 옵션을 지정했지만 컴파일러에서 msxml3.dll을 찾아서 로드할 수 없습니다.  msxml3.dll 파일을 설치하고 등록했는지 확인하십시오.
