# <a name="compiler-error-cs0233"></a><span data-ttu-id="566fa-102">컴파일러 오류 CS0233</span><span class="sxs-lookup"><span data-stu-id="566fa-102">Compiler Error CS0233</span></span>
<span data-ttu-id="566fa-103">‘identifier’에 미리 정의된 크기가 없으므로 sizeof는 안전하지 않은 컨텍스트에서만 사용할 수 있습니다. System.Runtime.InteropServices.Marshal.SizeOf를 사용하세요.</span><span class="sxs-lookup"><span data-stu-id="566fa-103">'identifier' does not have a predefined size, therefore sizeof can only be used in an unsafe context (consider using System.Runtime.InteropServices.Marshal.SizeOf)</span></span>  
  
 <span data-ttu-id="566fa-104">[sizeof](../../../csharp/language-reference/keywords/sizeof.md) 연산자는 컴파일 시간 상수인 형식에만 사용할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="566fa-104">The [sizeof](../../../csharp/language-reference/keywords/sizeof.md) operator can only be used for types that are compile-time constants.</span></span> <span data-ttu-id="566fa-105">이 오류가 발생하는 경우 컴파일 시간에 식별자의 크기를 확인할 수 있는지 확인합니다.</span><span class="sxs-lookup"><span data-stu-id="566fa-105">If you are getting this error, make sure that the size of the identifier can be determined at compile time.</span></span> <span data-ttu-id="566fa-106">확인할 수 없다면 `sizeof` 대신 <xref:System.Runtime.InteropServices.Marshal.SizeOf%2A>를 사용합니다.</span><span class="sxs-lookup"><span data-stu-id="566fa-106">If it cannot, then use <xref:System.Runtime.InteropServices.Marshal.SizeOf%2A> instead of `sizeof`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="566fa-107">예제</span><span class="sxs-lookup"><span data-stu-id="566fa-107">Example</span></span>  
 <span data-ttu-id="566fa-108">다음 예제에서는 CS0233을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="566fa-108">The following example generates CS0233:</span></span>  
  
```csharp  
// CS0233.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Sequential)]  
public struct S  
{  
    public int a;  
}  
  
public class MyClass  
{  
    public static void Main()  
    {  
        S myS = new S();  
        Console.WriteLine(sizeof(S));   // CS0233  
        // Try the following line instead:  
        // Console.WriteLine(Marshal.SizeOf(myS));  
   }  
}  
```
