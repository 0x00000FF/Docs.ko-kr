# <a name="compiler-error-cs1001"></a><span data-ttu-id="08250-102">컴파일러 오류 CS1001</span><span class="sxs-lookup"><span data-stu-id="08250-102">Compiler Error CS1001</span></span>
<span data-ttu-id="08250-103">식별자가 필요합니다.</span><span class="sxs-lookup"><span data-stu-id="08250-103">Identifier expected</span></span>  
  
 <span data-ttu-id="08250-104">식별자를 제공하지 않았습니다.</span><span class="sxs-lookup"><span data-stu-id="08250-104">You did not supply an identifier.</span></span> <span data-ttu-id="08250-105">식별자는 사용자가 제공하는 클래스, 구조체, 네임스페이스, 메서드, 변수 등의 이름입니다.</span><span class="sxs-lookup"><span data-stu-id="08250-105">An identifier is the name of a class, struct, namespace, method, variable and so on that you provide.</span></span>  
  
 <span data-ttu-id="08250-106">다음 예제에서는 간단한 클래스를 선언하지만 클래스에 이름을 지정하지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="08250-106">The following example declares a simple class but does not give the class a name:</span></span>  
  
```csharp  
//cs1001.cs  
public class              //CS1001  
    {  
        public int Num {get; set;}  
        void MethodA(){}  
    }  
```  
  
 <span data-ttu-id="08250-107">열거형을 선언할 때 멤버를 지정해야 하기 때문에 다음 샘플에서는 CS1001을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="08250-107">The following sample generates CS1001 because, when declaring an enum, you must specify members:</span></span>  
  
```csharp  
// CS1001.cs  
public class clx  
{  
   enum Colors : int  
   {  
      'a', 'b' // CS1001, 'a' is not a valid int identifier  
       // The following line shows examples of valid identifiers:  
       // Blue, Red, Orange  
   };  
  
   public static void Main()  
   {  
   }  
}  
```  
  
 <span data-ttu-id="08250-108">예를 들어 컴파일러가 인터페이스 정의에 사용하지 않는 경우에도 매개 변수 이름이 필요합니다.</span><span class="sxs-lookup"><span data-stu-id="08250-108">Parameter names are required even if the compiler doesn't use them, for example, in an interface definition.</span></span> <span data-ttu-id="08250-109">이러한 매개 변수는 인터페이스를 사용하는 프로그래머가 해당 매개 변수의 의미를 파악하기 위해 필요합니다.</span><span class="sxs-lookup"><span data-stu-id="08250-109">These parameters are required so that programmers who are consuming the interface know something about the what the parameters mean.</span></span>  
  
```csharp  
// CS1001-2.cs  
// compile with: /target:library  
interface IMyTest  
{  
   void TestFunc1(int, int);  // CS1001  
   // Use the following line instead:  
   // void TestFunc1(int a, int b);  
}  
  
class CMyTest : IMyTest  
{  
   void IMyTest.TestFunc1(int a, int b)  
   {  
   }  
}  
```
