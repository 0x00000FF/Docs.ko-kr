# <a name="compiler-error-cs0052"></a><span data-ttu-id="d9f9a-102">컴파일러 오류 CS0052</span><span class="sxs-lookup"><span data-stu-id="d9f9a-102">Compiler Error CS0052</span></span>

<span data-ttu-id="d9f9a-103">일관성 없는 액세스 가능성: 'type' 필드 형식이 'field' 필드보다 액세스하기 어렵습니다.</span><span class="sxs-lookup"><span data-stu-id="d9f9a-103">Inconsistent accessibility: field type 'type' is less accessible than field 'field'</span></span>

 <span data-ttu-id="d9f9a-104">모든 public 구문은 공개적으로 액세스 가능한 개체를 반환해야 하기 때문에 필드 형식이 필드 자체보다 액세스하기 어려울 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="d9f9a-104">The type of a field cannot be less accessible than the field itself because all public constructs must return a publicly accessible object.</span></span>

## <a name="example"></a><span data-ttu-id="d9f9a-105">예</span><span class="sxs-lookup"><span data-stu-id="d9f9a-105">Example</span></span>

 <span data-ttu-id="d9f9a-106">다음 샘플에서는 CS0052를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="d9f9a-106">The following sample generates CS0052:</span></span>

```csharp
// CS0052.cs
public class MyClass2
{
    // The following line causes an error because the field, M, is declared
    // as public, but the type, MyClass, is private. Therefore the type is
    // less accessible than the field.
    public MyClass M;   // CS0052

    private class MyClass
    {
    }
    // One way to resolve the error is to change the accessibility of the type
    // to public.
    //public class MyClass
    // Another solution is to change the accessibility of the field to private.
    // private MyClass M;
}

public class MainClass
{
    public static void Main()
    {
    }
}
```
