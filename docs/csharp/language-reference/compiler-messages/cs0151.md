<span data-ttu-id="efa07-104">정수 데이터 형식이 필요한 상황에서 변수가 사용되었습니다.</span><span class="sxs-lookup"><span data-stu-id="efa07-104">A variable was used in a situation where an integral data type was required.</span></span> <span data-ttu-id="efa07-105">자세한 내용은 [형식](../../programming-guide/types/index.md)을 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="efa07-105">For more information, see [Types](../../programming-guide/types/index.md).</span></span>

## <a name="example-of-ambiguous-conversion"></a><span data-ttu-id="efa07-106">모호한 변환의 예제</span><span class="sxs-lookup"><span data-stu-id="efa07-106">Example of ambiguous conversion</span></span>

<span data-ttu-id="efa07-107">이 오류는 변환이 없거나 사용 가능한 암시적 변환으로 인해 모호한 상황이 나타나는 경우에 발생할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="efa07-107">This error can occur when there is no conversion or if the available implicit conversions result in an ambiguous situation.</span></span> <span data-ttu-id="efa07-108">다음 예제에서는 CS0151을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="efa07-108">The following example generates CS0151:</span></span>

```csharp
public class MyClass
{
   public static implicit operator int (MyClass aa)
   {
      return 0;
   }

   public static implicit operator long (MyClass aa)
   {
      return 0;
   }

   public static void Main()
   {
      MyClass a = new MyClass();

      // Compiler cannot choose between int and long.
      switch (a)   // CS0151
      // try the following line instead
      // switch ((int)a)
      {
         case 1:
            break;
      }
   }
}
```

## <a name="example-of-void-method"></a><span data-ttu-id="efa07-109">void 메서드의 예제</span><span class="sxs-lookup"><span data-stu-id="efa07-109">Example of void method</span></span>

<span data-ttu-id="efa07-110">[switch](../keywords/switch.md) 일치 식에서 [void](../builtin-types/void.md) 메서드 호출은 CS0151을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="efa07-110">A [void](../builtin-types/void.md) method invocation in a [switch](../keywords/switch.md) match expression generates CS0151.</span></span> <span data-ttu-id="efa07-111">[int](../builtin-types/integral-numeric-types.md) 또는 [long](../builtin-types/integral-numeric-types.md)과 같은 정수 형식을 반환하는 메서드를 대신 호출하면 오류를 해결할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="efa07-111">You can fix the error by calling a method that returns an integral type such as [int](../builtin-types/integral-numeric-types.md) or [long](../builtin-types/integral-numeric-types.md) instead.</span></span>

```csharp

{
    static void Main()
    {
        switch (M()) // CS0151
        {
            default:
                break;
        }
    }

    static void M()
    {
    }
}
```
