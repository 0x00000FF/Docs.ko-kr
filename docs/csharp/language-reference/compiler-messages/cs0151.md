# <a name="compiler-error-cs0151"></a><span data-ttu-id="6d186-102">컴파일러 오류 CS0151</span><span class="sxs-lookup"><span data-stu-id="6d186-102">Compiler Error CS0151</span></span>
<span data-ttu-id="6d186-103">정수 형식 값이 필요합니다.</span><span class="sxs-lookup"><span data-stu-id="6d186-103">A value of an integral type expected</span></span>  
  
 <span data-ttu-id="6d186-104">정수 데이터 형식이 필요한 상황에서 변수가 사용되었습니다.</span><span class="sxs-lookup"><span data-stu-id="6d186-104">A variable was used in a situation where an integral data type was required.</span></span> <span data-ttu-id="6d186-105">자세한 내용은 [형식](../../../csharp/programming-guide/types/index.md)을 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="6d186-105">For more information, see [Types](../../../csharp/programming-guide/types/index.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="6d186-106">예제</span><span class="sxs-lookup"><span data-stu-id="6d186-106">Example</span></span>  
 <span data-ttu-id="6d186-107">이 오류는 변환이 없거나 사용 가능한 암시적 변환으로 인해 모호한 상황이 나타나는 경우에 발생할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="6d186-107">This error can occur when there is no conversion or if the available implicit conversions result in an ambiguous situation.</span></span> <span data-ttu-id="6d186-108">다음 샘플에서는 CS0151을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="6d186-108">The following sample generates CS0151.</span></span>  
  
```csharp  
// CS0151.cs  
public class MyClass  
{  
   public static implicit operator int (MyClass aa)  
   {  
      return 0;  
   }  
  
   public static implicit operator long (MyClass aa)  
   {  
      return 0;  
   }  
  
   public static void Main()  
   {  
      MyClass a = new MyClass();  
  
      // Compiler cannot choose between int and long  
      switch (a)   // CS0151  
      // try the following line instead  
      // switch ((int)a)  
      {  
         case 1:  
            break;  
      }  
   }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="6d186-109">예제</span><span class="sxs-lookup"><span data-stu-id="6d186-109">Example</span></span>  
 <span data-ttu-id="6d186-110">Visual Studio 2008 이상에서는 [void](../../../csharp/language-reference/keywords/void.md) 메서드 호출 시 CS0151이 생성됩니다.</span><span class="sxs-lookup"><span data-stu-id="6d186-110">In Visual Studio 2008 and later, a [void](../../../csharp/language-reference/keywords/void.md) method invocation generates CS0151.</span></span> <span data-ttu-id="6d186-111">[int](../../../csharp/language-reference/keywords/int.md) 또는 [long](../../../csharp/language-reference/keywords/long.md)과 같은 정수 형식을 반환하는 메서드를 호출하면 오류를 해결할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="6d186-111">You can fix the error by calling a method that returns an integral type such as [int](../../../csharp/language-reference/keywords/int.md) or [long](../../../csharp/language-reference/keywords/long.md).</span></span>

```csharp  
class C

    }

    {  
    }  
}  
```
