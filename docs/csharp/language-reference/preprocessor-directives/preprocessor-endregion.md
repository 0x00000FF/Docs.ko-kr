# #endregion(C# 참조)
`#endregion`은 [\#region](../../../csharp/language-reference/preprocessor-directives/preprocessor-region.md) 블록의 끝을 표시합니다.  예를 들면 다음과 같습니다.  
  
```  
  
      #region MyClass definition  
class MyClass   
{  
    static void Main()   
    {  
    }  
}  
#endregion  
```
