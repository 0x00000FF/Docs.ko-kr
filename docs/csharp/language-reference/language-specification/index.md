# <a name="c-60-draft-language-specification"></a>C# 6.0 초안 언어 사양
C# 언어 사양은 C# 구문 및 사용을 위한 확정된 소스입니다. 이 사양에는 Visual C# 설명서에 나와 있지 않은 여러 항목을 비롯하여 C# 언어의 모든 측면에 대한 세부 정보가 포함되어 있습니다.

[Microsoft 다운로드 센터](http://www.microsoft.com/download/details.aspx?id=7029)에서 이 사양의 버전 5.0을 다운로드할 수 있습니다. Visual Studio 2015를 설치한 경우 Program Files (x86)/Microsoft Visual Studio 14.0/VC#/Specifications/1033 폴더에서 컴퓨터 사양을 확인할 수도 있습니다. 다른 버전의 Visual Studio가 설치되어 있거나 영어 이외의 언어로 Visual Studio를 설치한 경우 경로를 적절하게 변경합니다.

버전 6.0 사양은 표준으로 승인되지 않았습니다. 이 사이트에는 [ *초안* C# 6.0 사양](../../../../_csharplang/spec/lexical-structure.md)이 들어 있습니다. 초안 C# 6.0 사양은 [dotnet/csharplang GitHub 리포지토리](https://github.com/dotnet/csharplang/blob/master/spec/README.md)에 포함된 markdown 파일에서 빌드됩니다.

초안 사양에 관한 문제는 [dotnet/csharplang](https://github.com/dotnet/csharplang/issues) 리포지토리에 작성해야 합니다. 또는, 발견한 오류를 수정하는 데 관심이 있는 경우 동일한 리포지토리에 [끌어오기 요청](https://github.com/dotnet/csharplang/pulls)을 제출할 수 있습니다.
