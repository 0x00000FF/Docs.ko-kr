# C# 언어 사양
C\# 언어 사양은 C\# 구문 및 사용을 위한 확정된 소스입니다.  이 사양에는 Visual C\# 설명서에는 나와 있지 않은 여러 항목을 비롯하여 C\# 언어의 모든 측면에 대한 상세 정보가 포함되어 있습니다.  
  
 [Microsoft 다운로드 센터](http://www.microsoft.com/download/details.aspx?id=7029)에서 이 사양을 다운로드할 수 있습니다.  Visual Studio 2013을 설치한 경우에는 Program Files \(x86\)\/Microsoft Visual Studio 12.0\/VC\#\/Specifications\/1033 폴더에서 컴퓨터 사양을 확인할 수도 있습니다.  그러나 Visual Studio Express 2013을 설치하는 경우에는 이 파일이 포함되지 않습니다.
