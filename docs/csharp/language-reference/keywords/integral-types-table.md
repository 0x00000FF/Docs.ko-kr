# <a name="integral-types-table-c-reference"></a>정수 계열 형식 표(C# 참조)
다음 표에서는 단순 형식의 하위 집합을 구성하는 정수 형식의 크기와 범위를 보여 줍니다.  
  
|형식|범위|크기|  
|----------|-----------|----------|  
|[sbyte](../../../csharp/language-reference/keywords/sbyte.md)|-128 ~ 127|부호 있는 8비트 정수|  
|[byte](../../../csharp/language-reference/keywords/byte.md)|0 ~ 255|부호 없는 8비트 정수|  
|[char](../../../csharp/language-reference/keywords/char.md)|U+0000 ~ U+ffff|유니코드 16비트 문자|  
|[short](../../../csharp/language-reference/keywords/short.md)|–32,768 ~ 32,767|부호 있는 16비트 정수|  
|[ushort](../../../csharp/language-reference/keywords/ushort.md)|0 ~ 65,535|부호 없는 16비트 정수|  
|[int](../../../csharp/language-reference/keywords/int.md)|–2,147,483,648 ~ 2,147,483,647|부호 있는 32비트 정수|  
|[uint](../../../csharp/language-reference/keywords/uint.md)|0 ~ 4,294,967,295|부호 없는 32비트 정수|  
|[long](../../../csharp/language-reference/keywords/long.md)|–9,223,372,036,854,775,808 ~ 9,223,372,036,854,775,807|부호 있는 64비트 정수|  
|[ulong](../../../csharp/language-reference/keywords/ulong.md)|0 ~ 18,446,744,073,709,551,615|부호 없는 64비트 정수|  
  
 정수 리터럴이 나타내는 값이 `ulong`의 범위를 초과하는 경우 컴파일 오류가 발생합니다.
