# <a name="method-parameters-c-reference"></a>메서드 매개 변수(C# 참조)
매개 변수가 [ref](../../../csharp/language-reference/keywords/ref.md) 또는 [out](../../../csharp/language-reference/keywords/out.md) 없이 메서드에 대해 선언된 경우 매개 변수에 값이 연결될 수 있습니다. 메서드에서 해당 값을 변경할 수 있지만, 제어가 호출하는 프로시저로 다시 전달되면 변경된 값이 보존되지 않습니다. 메서드 매개 변수 키워드를 사용하여 이 동작을 변경할 수 있습니다.  
  
 이 섹션에서는 메서드 매개 변수를 선언할 때 사용할 수 있는 키워드를 설명합니다.  
  
-   [params](../../../csharp/language-reference/keywords/params.md)  
  
-   [ref](../../../csharp/language-reference/keywords/ref.md)  
  
-   [out](../../../csharp/language-reference/keywords/out.md)
