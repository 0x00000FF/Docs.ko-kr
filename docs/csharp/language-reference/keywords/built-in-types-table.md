## <a name="remarks"></a><span data-ttu-id="081f4-122">설명</span><span class="sxs-lookup"><span data-stu-id="081f4-122">Remarks</span></span>

<span data-ttu-id="081f4-123">`object` 및 `string`을 제외하고 표에 있는 모든 형식을 단순 형식이라고 합니다.</span><span class="sxs-lookup"><span data-stu-id="081f4-123">All of the types in the table, except `object` and `string`, are referred to as simple types.</span></span>  
  
<span data-ttu-id="081f4-124">C# 형식 키워드와 해당 별칭은 서로 바꿔서 사용할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="081f4-124">The C# type keywords and their aliases are interchangeable.</span></span> <span data-ttu-id="081f4-125">예를 들어 다음 선언 중 하나를 사용하여 정수 변수를 선언할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="081f4-125">For example, you can declare an integer variable by using either of the following declarations:</span></span>  

```csharp
int x = 123;
System.Int32 y = 123;
```

<span data-ttu-id="081f4-126">[typeof](../operators/type-testing-and-conversion-operators.md#typeof-operator) 연산자를 사용하여 지정된 형식을 나타내는 <xref:System.Type?displayProperty=nameWithType> 인스턴스를 가져옵니다.</span><span class="sxs-lookup"><span data-stu-id="081f4-126">Use the [typeof](../operators/type-testing-and-conversion-operators.md#typeof-operator) operator to get the <xref:System.Type?displayProperty=nameWithType> instance that represents the specified type:</span></span>

```csharp
Type stringType = typeof(string);
Console.WriteLine(stringType.FullName);

Type doubleType = typeof(System.Double);
Console.WriteLine(doubleType.FullName);

// Output:
// System.String
// System.Double
```
