# 부동 소수점 형식 표(C# 참조)
다음 표에서는 부동 소수점 형식의 전체 자릿수와 대략적인 범위를 보여 줍니다.  
  
|형식|근사 범위|전체 자릿수|  
|--------|-----------|------------|  
|[float](../../../csharp/language-reference/keywords/float.md)|±1.5e−45 ~ ±3.4e38|7개의 자릿수|  
|[double](../../../csharp/language-reference/keywords/double.md)|±5.0e−324 ~ ±1.7e308|15\-16개의 자릿수|
