# 형식(C# 참조)
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

C\# 형식은 다음과 같은 범주로 구성됩니다.  
  
-   [값 형식](../../../csharp/language-reference/keywords/value-types.md)  
  
-   [참조 형식](../../../csharp/language-reference/keywords/reference-types.md)  
  
-   [포인터 형식](../../../csharp/programming-guide/unsafe-code-pointers/pointer-types.md)  
  
 값 형식의 변수에는 실제 데이터가 저장되지만 참조 형식의 변수에는 실제 데이터에 대한 참조가 저장됩니다.  참조 형식을 개체라고도 합니다.  포인터 형식은 [unsafe](../../../csharp/language-reference/keywords/unsafe.md) 모드에서만 사용할 수 있습니다.  
  
 [boxing 및 unboxing](../../../csharp/programming-guide/types/boxing-and-unboxing.md)을 사용하여 값 형식을 참조 형식으로 변환하고 다시 값 형식으로 변환할 수 있습니다.  boxed 값 형식만 제외하고는 참조 형식을 값 형식으로 변환할 수 없습니다.  
  
 이 단원에서는 [void](../../../csharp/language-reference/keywords/void.md)에 대해서도 설명합니다.  
  
 또한 값 형식은 null 허용 형식이므로 값이 아닌 상태를 추가로 저장할 수 있습니다.  자세한 내용은 [nullable 형식](../../../csharp/programming-guide/nullable-types/index.md)을 참조하십시오.
