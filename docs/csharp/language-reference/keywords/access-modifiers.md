# 액세스 한정자(C# 참조)
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

액세스 한정자는 멤버 또는 형식의 선언된 액세스 가능성을 지정하는 데 사용하는 키워드입니다.  이 단원에서는 다음과 같은 네 가지 액세스 한정자에 대해 설명합니다.  
  
-   [public](../../../csharp/language-reference/keywords/public.md)  
  
-   [protected](../../../csharp/language-reference/keywords/protected.md)  
  
-   [internal](../../../csharp/language-reference/keywords/internal.md)  
  
-   [private](../../../csharp/language-reference/keywords/private.md)  
  
 액세스 한정자를 사용하여 다음의 다섯 가지 액세스 가능성 수준을 지정할 수 있습니다.  
  
 `public`: 액세스가 제한되지 않습니다.  
  
 `protected`: 포함하는 클래스 또는 여기에서 파생된 형식으로 액세스가 제한됩니다.  
  
 `Internal`: 액세스가 현재 어셈블리로 제한됩니다.  
  
 [protected internal](../../../csharp/programming-guide/classes-and-structs/access-modifiers.md): 액세스가 현재 어셈블리 또는 포함 하는 클래스에서 파생 되는 형식으로 제한 됩니다.  
  
 `private`: 액세스가 포함하는 형식으로 제한됩니다.  
  
 이 단원에서는 다음 내용도 설명합니다.  
  
-   [액세스 수준](../../../csharp/language-reference/keywords/accessibility-levels.md): 네 개의 액세스 한정자를 사용하여 다섯 가지 수준의 액세스를 선언합니다.  
  
-   [액세스 가능 도메인](../../../csharp/language-reference/keywords/accessibility-domain.md): 프로그램 섹션에서 멤버가 참조될 수 있는 범위를 지정합니다.  
  
-   [액세스 수준에 대한 제한](../../../csharp/language-reference/keywords/restrictions-on-using-accessibility-levels.md): 선언된 액세스 수준 사용에 대한 제한 사항이 요약되어 있습니다.
