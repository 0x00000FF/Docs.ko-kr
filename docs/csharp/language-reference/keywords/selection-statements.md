# <a name="selection-statements-c-reference"></a>선택문(C# 참조)

선택문을 사용하면 특정 조건이 `true`인지 여부에 따라 프로그램 제어가 특정 흐름에 전송됩니다.

선택문에서 사용되는 키워드는 다음과 같습니다.

- [if](if-else.md)

- [else](if-else.md)

- [switch](switch.md)

- [case](switch.md)

- [default](switch.md)
