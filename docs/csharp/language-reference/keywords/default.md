# default(C# 참조)
`default` 키워드입니다.  
  
 `default` 키워드는 `switch` 문이나 제네릭 코드에 사용할 수 있습니다.  
  
-   [switch 문](../../../csharp/language-reference/keywords/switch.md): 기본 레이블을 지정합니다.  
  
-   [제네릭 코드](../../../csharp/programming-guide/generics/default-keyword-in-generic-code.md): 형식 매개 변수의 기본값을 지정합니다.  기본값은 참조 형식의 경우 null이고 값 형식의 경우 0입니다.
