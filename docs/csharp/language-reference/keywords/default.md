# <a name="default-c-reference"></a>default(C# 참조)

`default` 키워드는 다음 두 가지 방법으로 사용할 수 있습니다.
- [`switch` 문](switch.md) - 기본 레이블을 지정합니다.
- [기본값 식](../../programming-guide/statements-expressions-operators/default-value-expressions.md) - 형식의 기본값을 생성합니다.
