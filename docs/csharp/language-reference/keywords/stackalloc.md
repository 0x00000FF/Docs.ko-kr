# <a name="stackalloc-c-reference"></a><span data-ttu-id="1c6e1-102">stackalloc(C# 참조)</span><span class="sxs-lookup"><span data-stu-id="1c6e1-102">stackalloc (C# Reference)</span></span>

<span data-ttu-id="1c6e1-103">`stackalloc` 키워드는 스택에 메모리 블록을 할당하기 위해 안전하지 않은 코드 컨텍스트에서 사용됩니다.</span><span class="sxs-lookup"><span data-stu-id="1c6e1-103">The `stackalloc` keyword is used in an unsafe code context to allocate a block of memory on the stack.</span></span>

```csharp

```

## <a name="remarks"></a><span data-ttu-id="1c6e1-104">설명</span><span class="sxs-lookup"><span data-stu-id="1c6e1-104">Remarks</span></span>

<span data-ttu-id="1c6e1-105">이 키워드는 지역 변수 이니셜라이저에서만 유효합니다.</span><span class="sxs-lookup"><span data-stu-id="1c6e1-105">The keyword is valid only in local variable initializers.</span></span> <span data-ttu-id="1c6e1-106">다음 코드를 실행하면 컴파일러 오류가 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="1c6e1-106">The following code causes compiler errors.</span></span>

```csharp

// The following assignment statement causes compiler errors. You
// can use stackalloc only when declaring and initializing a local
// variable.
block = stackalloc int[100];
```

<span data-ttu-id="1c6e1-107">C# 7.3부터는 `stackalloc` 배열에 대한 배열 이니셜라이저 구문을 사용할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="1c6e1-107">Beginning with C# 7.3, you can use array initializer syntax for `stackalloc` arrays.</span></span> <span data-ttu-id="1c6e1-108">다음 선언은 모두 값이 정수 `1`, `2` 및 `3`인 세 개의 요소가 있는 배열을 선언합니다.</span><span class="sxs-lookup"><span data-stu-id="1c6e1-108">All the following declarations declare an array with three elements whose values are the integers `1`, `2`, and `3`:</span></span>

```csharp

int* second = stackalloc int[] { 1, 2, 3 };
int* third = stackalloc[] { 1, 2, 3 };
```

<span data-ttu-id="1c6e1-109">포인터 형식이 사용되므로 `stackalloc`에 [안전하지 않은](unsafe.md) 컨텍스트가 필요합니다.</span><span class="sxs-lookup"><span data-stu-id="1c6e1-109">Because pointer types are involved, `stackalloc` requires an [unsafe](unsafe.md) context.</span></span> <span data-ttu-id="1c6e1-110">자세한 내용은 [안전하지 않은 코드 및 포인터](../../programming-guide/unsafe-code-pointers/index.md)를 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="1c6e1-110">For more information, see [Unsafe Code and Pointers](../../programming-guide/unsafe-code-pointers/index.md).</span></span>

<span data-ttu-id="1c6e1-111">`stackalloc`는 C 런타임 라이브러리의 [_alloca](/cpp/c-runtime-library/reference/alloca)와 유사합니다.</span><span class="sxs-lookup"><span data-stu-id="1c6e1-111">`stackalloc` is like [_alloca](/cpp/c-runtime-library/reference/alloca) in the C run-time library.</span></span>
