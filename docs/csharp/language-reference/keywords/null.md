# null(C# 참조)
`null` 키워드는 아무 개체도 참조하지 않는 null 참조를 나타내는 리터럴입니다.  `null`은 참조 형식 변수의 기본값입니다.  일반 값 형식은 null일 수 없습니다.  하지만 C\# 2.0에는 nullable 값 형식이 도입되었습니다.  자세한 내용은 [nullable 형식](../../../csharp/programming-guide/nullable-types/index.md)를 참조하십시오.
