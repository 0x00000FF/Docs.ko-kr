# using(C# 참조)
`using` 키워드는 아래와 같은 두 가지 주요한 용도로 사용할 수 있습니다.  
  
-   네임스페이스의 별칭을 만들거나 다른 네임스페이스에 정의된 형식을 가져올 때 지시문으로 사용할 수 있습니다.  자세한 내용은 [using 지시문](../../../csharp/language-reference/keywords/using-directive.md)을 참조하십시오.  
  
-   개체의 범위를 정의할 때 문으로 사용할 수 있습니다. 개체는 이 범위가 끝날 때 해제됩니다.  자세한 내용은 [using 문](../../../csharp/language-reference/keywords/using-statement.md)을 참조하십시오.
