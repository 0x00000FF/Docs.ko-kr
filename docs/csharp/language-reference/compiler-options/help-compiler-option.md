# /help, /? (C# Compiler Options)
이 옵션은 컴파일러 옵션 목록과 각 옵션에 대한 간단한 설명을 stdout에 보냅니다.  
  
## 구문  
  
```  
/help  
/?  
```  
  
## 설명  
 이 옵션이 컴파일에 포함되면 출력 파일이 만들어지지 않으며 컴파일도 실행되지 않습니다.  
  
 이 컴파일러 옵션은 Visual Studio에서 사용할 수 없으며 프로그래밍 방식으로 변경할 수도 없습니다.
