 <span data-ttu-id="08a2c-110">컴파일러에서 표시할 경고 수준을 지정하려면 [-warn](../../../csharp/language-reference/compiler-options/warn-compiler-option.md)을 사용합니다.</span><span class="sxs-lookup"><span data-stu-id="08a2c-110">Use [-warn](../../../csharp/language-reference/compiler-options/warn-compiler-option.md) to specify the level of warnings that you want the compiler to display.</span></span> <span data-ttu-id="08a2c-111">특정 경고를 사용하지 않으려면 [-nowarn](../../../csharp/language-reference/compiler-options/nowarn-compiler-option.md)을 사용합니다.</span><span class="sxs-lookup"><span data-stu-id="08a2c-111">Use [-nowarn](../../../csharp/language-reference/compiler-options/nowarn-compiler-option.md) to disable certain warnings.</span></span>
