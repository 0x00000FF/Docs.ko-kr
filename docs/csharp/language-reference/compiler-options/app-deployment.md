# Deployment of C# Applications
C\# 응용 프로그램을 빌드하고 나서 할 일은 이를 배포하는 것입니다.  C\#은 .NET 언어이므로 C\# 실행 파일을 다른 컴퓨터에 배포하려면 각 실행 컴퓨터에 .NET Framework를 설치해야 할 뿐 아니라 응용 프로그램에 따라 추가로 종속 파일이 필요합니다.  여러 가지 방법으로 .NET Framework를 배포할 수 있습니다.  개요는 [개발자를 위한 배포 가이드](../Topic/.NET%20Framework%20Deployment%20Guide%20for%20Developers.md)를 참조하십시오.  
  
 완성된 응용 프로그램을 다른 컴퓨터로 옮기는 작업을 일반적으로 배포라고 합니다.  Microsoft 개발 환경에서는 배포를 위한 메커니즘을 제공합니다. 자세한 내용은 [응용 프로그램 및 구성 요소 배포](/visual-studio/deployment/deploying-applications-services-and-components)를 참조하십시오.  
  
 주로 명령줄에서 응용 프로그램을 빌드하고 배포하는 경우에는 종속 파일을 배포하거나 재배포하기 위한 다른 방법을 고려해야 할 수도 있습니다.
