# &lt;code&gt;(C# 프로그래밍 가이드)
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

## 구문  
  
```  
<code>content</code>  
```  
  
#### 매개 변수  
 `content`  
 코드로 표시할 텍스트입니다.  
  
## 설명  
 \<code\> 태그는 여러 줄을 코드로 표시하는 데 사용합니다.  [\<c\>](../../../csharp/programming-guide/xmldoc/code-inline.md) 태그는 설명에 있는 텍스트를 코드로 표시하는 데 사용합니다.  
  
 [\/doc](../../../csharp/language-reference/compiler-options/doc-compiler-option.md)로 컴파일하여 문서 주석을 파일로 저장합니다.  
  
## 예제  
 \<code\> 태그의 사용 방법을 보여 주는 예제는 [\<example\>](../../../csharp/programming-guide/xmldoc/example.md) 항목을 참조하십시오.
