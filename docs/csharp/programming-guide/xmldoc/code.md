# <a name="ltcodegt-c-programming-guide"></a>&lt;code&gt;(C# 프로그래밍 가이드)
## <a name="syntax"></a>구문  
  
```xml  
<code>content</code>  
```  
  
#### <a name="parameters"></a>매개 변수  
 `content`  
 코드로 표시할 텍스트입니다.  
  
## <a name="remarks"></a>설명  
 \<code> 태그를 사용하면 여러 줄을 코드로 지정할 수 있습니다. 설명 내의 텍스트가 코드로 표시되도록 지정하려면 [\<c>](../../../csharp/programming-guide/xmldoc/code-inline.md)를 사용합니다.  
  
 [/doc](../../../csharp/language-reference/compiler-options/doc-compiler-option.md)로 컴파일하여 문서 주석을 파일로 처리합니다.
