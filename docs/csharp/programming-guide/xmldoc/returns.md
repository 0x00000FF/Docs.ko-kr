## <a name="syntax"></a>구문  
  
```xml  
<returns>description</returns>  
```  
  
#### <a name="parameters"></a>매개 변수  
 `description`  
 반환 값에 대한 설명입니다.

 \<returns> 태그는 메서드 선언의 주석에서 반환 값을 설명하는 데 사용해야 합니다.
