# &lt;returns&gt;(C# 프로그래밍 가이드)
## 구문  
  
```  
<returns>description</returns>  
```  
  
#### 매개 변수  
 `description`  
 반환 값에 대한 설명입니다.  
  
## 설명  
 반환 값을 설명하려면 메서드 선언의 주석에서 \<returns\> 태그를 사용해야 합니다.  
  
 [\/doc](../../../csharp/language-reference/compiler-options/doc-compiler-option.md)로 컴파일하여 문서 주석을 파일로 저장합니다.
