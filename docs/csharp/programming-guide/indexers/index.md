## <a name="indexers-overview"></a><span data-ttu-id="bd173-117">인덱서 개요</span><span class="sxs-lookup"><span data-stu-id="bd173-117">Indexers Overview</span></span>  
  
-   <span data-ttu-id="bd173-118">인덱서를 사용하면 배열과 유사한 방식으로 개체를 인덱싱할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="bd173-118">Indexers enable objects to be indexed in a similar manner to arrays.</span></span>  
  
-   <span data-ttu-id="bd173-119">`get` 접근자는 값을 반환합니다.</span><span class="sxs-lookup"><span data-stu-id="bd173-119">A `get` accessor returns a value.</span></span> <span data-ttu-id="bd173-120">`set` 접근자는 값을 할당합니다.</span><span class="sxs-lookup"><span data-stu-id="bd173-120">A `set` accessor assigns a value.</span></span>  
  
-   <span data-ttu-id="bd173-121">[this](../../../csharp/language-reference/keywords/this.md) 키워드는 인덱서를 정의하는 데 사용됩니다.</span><span class="sxs-lookup"><span data-stu-id="bd173-121">The [this](../../../csharp/language-reference/keywords/this.md) keyword is used to define the indexer.</span></span>  
  
-   <span data-ttu-id="bd173-122">[value](../../../csharp/language-reference/keywords/value.md) 키워드는 `set` 인덱서에서 할당하는 값을 정의하는 데 사용됩니다.</span><span class="sxs-lookup"><span data-stu-id="bd173-122">The [value](../../../csharp/language-reference/keywords/value.md) keyword is used to define the value being assigned by the `set` indexer.</span></span>  
  
-   <span data-ttu-id="bd173-123">인덱서는 정수 값으로 인덱싱될 필요가 없으며, 특정 조회 메커니즘을 정의하는 방법을 결정해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="bd173-123">Indexers do not have to be indexed by an integer value; it is up to you how to define the specific look-up mechanism.</span></span>  
  
-   <span data-ttu-id="bd173-124">인덱서는 오버로드될 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="bd173-124">Indexers can be overloaded.</span></span>  
  
-   <span data-ttu-id="bd173-125">예를 들어 인덱서는 2차원 배열에 액세스하는 경우 둘 이상의 정식 매개 변수를 사용할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="bd173-125">Indexers can have more than one formal parameter, for example, when accessing a two-dimensional array.</span></span>
