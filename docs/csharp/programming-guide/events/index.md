## <a name="events-overview"></a><span data-ttu-id="6c56d-109">이벤트 개요</span><span class="sxs-lookup"><span data-stu-id="6c56d-109">Events Overview</span></span>

-   <span data-ttu-id="6c56d-111">게시자는 이벤트 발생 시기를 결정합니다. 구독자는 이벤트에 대한 응답으로 수행할 작업을 결정합니다.</span><span class="sxs-lookup"><span data-stu-id="6c56d-111">The publisher determines when an event is raised; the subscribers determine what action is taken in response to the event.</span></span>

-   <span data-ttu-id="6c56d-118">[!INCLUDE[dnprdnshort](~/includes/dnprdnshort-md.md)] 클래스 라이브러리에서 이벤트는 <xref:System.EventHandler> 대리자 및 <xref:System.EventArgs> 기본 클래스를 기반으로 합니다.</span><span class="sxs-lookup"><span data-stu-id="6c56d-118">In the [!INCLUDE[dnprdnshort](~/includes/dnprdnshort-md.md)] class library, events are based on the <xref:System.EventHandler> delegate and the <xref:System.EventArgs> base class.</span></span>
