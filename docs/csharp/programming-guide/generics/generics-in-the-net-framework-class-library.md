# <a name="generics-in-the-net-framework-class-library-c-programming-guide"></a>.NET Framework 클래스 라이브러리의 제네릭(C# 프로그래밍 가이드)
.NET Framework 클래스 라이브러리 2.0 버전은 바로 사용할 수 있는 여러 가지 제네릭 컬렉션 클래스 및 연결된 인터페이스가 포함된 새로운 네임스페이스 <xref:System.Collections.Generic>을 제공합니다. 또한 <xref:System> 같은 다른 네임스페이스는 <xref:System.IComparable%601> 같은 새로운 제네릭 인터페이스를 제공합니다. 이러한 클래스 및 인터페이스는 .NET Framework의 이전 릴리스에서 제공된 제네릭이 아닌 컬렉션 클래스에 비해 보다 효율적이며 형식이 안전합니다. 사용자 지정 컬렉션 클래스를 디자인하고 구현하려면 먼저 .NET Framework 클래스 라이브러리에서 제공하는 클래스 중 하나에서 클래스를 사용하거나 파생시킬 수 있는지를 고려해야 합니다.
