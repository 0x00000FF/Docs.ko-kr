## <a name="subtracting-pointers"></a><span data-ttu-id="2db6c-109">포인터 빼기</span><span class="sxs-lookup"><span data-stu-id="2db6c-109">Subtracting Pointers</span></span>  
 <span data-ttu-id="2db6c-110">같은 형식의 포인터를 뺄 수도 있습니다.</span><span class="sxs-lookup"><span data-stu-id="2db6c-110">You can also subtract pointers of the same type.</span></span> <span data-ttu-id="2db6c-111">결과는 항상 `long` 형식입니다.</span><span class="sxs-lookup"><span data-stu-id="2db6c-111">The result is always of the type `long`.</span></span> <span data-ttu-id="2db6c-112">예를 들어 `p1` 및 `p2`가 `pointer-type*` 형식의 포인터인 경우 `p1-p2` 식의 결과는 다음과 같습니다.</span><span class="sxs-lookup"><span data-stu-id="2db6c-112">For example, if `p1` and `p2` are pointers of the type `pointer-type*`, then the expression `p1-p2` results in:</span></span>  
  
 `((long)p1 - (long)p2)/sizeof(pointer_type)`  
  
 <span data-ttu-id="2db6c-113">산술 연산이 포인터의 도메인을 오버플로하는 경우 예외가 생성되지 않고 결과는 구현에 따라 다릅니다.</span><span class="sxs-lookup"><span data-stu-id="2db6c-113">No exceptions are generated when the arithmetic operation overflows the domain of the pointer, and the result depends on the implementation.</span></span>
