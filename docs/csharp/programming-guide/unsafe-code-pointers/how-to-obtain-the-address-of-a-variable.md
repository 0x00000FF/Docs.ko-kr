<span data-ttu-id="882fc-103">고정 변수로 계산되는 단항 식의 주소를 가져오려면 address-of 연산자 `&`를 사용합니다.</span><span class="sxs-lookup"><span data-stu-id="882fc-103">To obtain the address of a unary expression, which evaluates to a fixed variable, use the address-of operator `&`:</span></span>  
  
```csharp  
int number;  
int* p = &number; //address-of operator &  
```  
  
 <span data-ttu-id="882fc-104">address-of 연산자는 변수에만 적용할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="882fc-104">The address-of operator can only be applied to a variable.</span></span> <span data-ttu-id="882fc-105">변수가 이동 가능한 변수인 경우 해당 주소를 가져오기 전에 [fixed 문](../../../csharp/language-reference/keywords/fixed-statement.md)을 사용하여 일시적으로 변수를 고정할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="882fc-105">If the variable is a moveable variable, you can use the [fixed statement](../../../csharp/language-reference/keywords/fixed-statement.md) to temporarily fix the variable before obtaining its address.</span></span>  
  
 <span data-ttu-id="882fc-106">변수가 초기화되었는지 확인해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="882fc-106">It's your responsibility to ensure that the variable is initialized.</span></span> <span data-ttu-id="882fc-107">변수가 초기화되지 않은 경우 컴파일러가 오류 메시지를 실행하지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="882fc-107">The compiler doesn't issue an error message if the variable is not initialized.</span></span>  
  
 <span data-ttu-id="882fc-108">상수 또는 값의 주소를 가져올 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="882fc-108">You can't get the address of a constant or a value.</span></span>

 <span data-ttu-id="882fc-110">이 예제에서는 `int`에 대한 포인터인 `p`가 선언되고 정수 변수 `number`의 주소가 할당됩니다.</span><span class="sxs-lookup"><span data-stu-id="882fc-110">In this example, a pointer to `int`, `p`, is declared and assigned the address of an integer variable, `number`.</span></span> <span data-ttu-id="882fc-111">`number` 변수는 `*p`에 대한 할당 결과로 초기화됩니다.</span><span class="sxs-lookup"><span data-stu-id="882fc-111">The variable `number` is initialized as a result of the assignment to `*p`.</span></span> <span data-ttu-id="882fc-112">이 대입문을 주석으로 처리하면 `number` 변수의 초기화가 제거되지만 컴파일 시간 오류가 발생하지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="882fc-112">If you comment out this assignment statement, the initialization of the variable `number` is removed, but no compile-time error is issued.</span></span>
