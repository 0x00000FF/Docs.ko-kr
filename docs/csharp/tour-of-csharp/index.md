<span data-ttu-id="79278-127">최신 버전에서는 C#이 다른 프로그래밍 패러다임을 받아들였습니다.</span><span class="sxs-lookup"><span data-stu-id="79278-127">In more recent versions, C# has embraced other programming paradigms.</span></span> <span data-ttu-id="79278-128">C#에 람다 식과 같은 함수 프로그래밍 기법을 지원하는 기능이 포함되었습니다.</span><span class="sxs-lookup"><span data-stu-id="79278-128">C# has included features that support functional programming techniques like lambda expressions.</span></span> <span data-ttu-id="79278-129">다른 새로운 기능은 패턴 일치처럼 데이터와 알고리즘 분리를 지원합니다.</span><span class="sxs-lookup"><span data-stu-id="79278-129">Other new features support separating data and algorithms, like pattern matching.</span></span>
