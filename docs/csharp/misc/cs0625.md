# <a name="compiler-error-cs0625"></a><span data-ttu-id="c87c7-102">컴파일러 오류 CS0625</span><span class="sxs-lookup"><span data-stu-id="c87c7-102">Compiler Error CS0625</span></span>
<span data-ttu-id="c87c7-103">'필드': StructLayout(LayoutKind.Explicit)으로 표시된 인스턴스 필드 유형에는 FieldOffset 특성이 있어야 합니다.</span><span class="sxs-lookup"><span data-stu-id="c87c7-103">'field': instance field types marked with StructLayout(LayoutKind.Explicit) must have a FieldOffset attribute.</span></span>
  
<span data-ttu-id="c87c7-104">구조체가 명시적 **StructLayout** 특성으로 표시된 경우 구조체의 모든 필드에 [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) 특성이 있어야 합니다.</span><span class="sxs-lookup"><span data-stu-id="c87c7-104">When a struct is marked with an explicit **StructLayout** attribute, all fields in the struct must have the [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) attribute.</span></span> <span data-ttu-id="c87c7-105">자세한 내용은 [StructLayoutAttribute 클래스](xref:System.Runtime.InteropServices.StructLayoutAttribute)를 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="c87c7-105">For more information, see [StructLayoutAttribute Class](xref:System.Runtime.InteropServices.StructLayoutAttribute).</span></span>

<span data-ttu-id="c87c7-106">다음 샘플에서는 CS0625를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="c87c7-106">The following sample generates CS0625:</span></span>  
  
```csharp  
// CS0625.cs  
// compile with: /target:library  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Explicit)]  
struct A  
{  
   public int i;   // CS0625 not static; an instance field  
}  
  
// OK  
[StructLayout(LayoutKind.Explicit)]  
struct B  
{  
   [FieldOffset(5)]  
   public int i;  
}  
```
