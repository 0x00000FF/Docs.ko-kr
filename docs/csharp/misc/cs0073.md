 <span data-ttu-id="ae752-104">**이벤트** 정의에서 **add** 또는 [remove](../../csharp/language-reference/keywords/event.md) 키워드는 본문이 있어야 합니다.</span><span class="sxs-lookup"><span data-stu-id="ae752-104">An **add** or **remove** keyword in an [event](../../csharp/language-reference/keywords/event.md) definition must have a body.</span></span> <span data-ttu-id="ae752-105">자세한 내용은 [이벤트](../../csharp/programming-guide/events/index.md)를 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="ae752-105">For more information, see [Events](../../csharp/programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="ae752-106">다음 샘플에서는 CS0073을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="ae752-106">The following sample generates CS0073:</span></span>  
  
```csharp  
// CS0073.cs  
delegate void del();  
  
class Test  
{  
   public event del MyEvent  
   {  
      add;   // CS0073  
      // try the following lines instead  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      remove  
      {  
         MyEvent -= value;  
      }  
  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
