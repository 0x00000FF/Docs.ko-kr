 <span data-ttu-id="d264f-109">다음 코드에서는 클래스 메서드가 `void` 를 반환하고 동일한 이름의 인터페이스 멤버가 `int`를 반환하기 때문에 CS0738을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="d264f-109">The following code generates CS0738 because the class method returns `void` and the interface member of the same name returns `int`:</span></span>

    int TestMethod();
