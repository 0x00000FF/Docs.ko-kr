# <a name="compiler-error-cs0522"></a><span data-ttu-id="ae212-102">컴파일러 오류 CS0522</span><span class="sxs-lookup"><span data-stu-id="ae212-102">Compiler Error CS0522</span></span>
<span data-ttu-id="ae212-103">'constructor': 구조체는 기본 클래스 생성자를 호출할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="ae212-103">'constructor' : structs cannot call base class constructors</span></span>  
  
 <span data-ttu-id="ae212-104">[구조체](../../csharp/language-reference/keywords/struct.md) 는 기본 클래스 생성자를 호출할 수 없습니다. 기본 클래스 생성자에 대한 호출을 제거합니다.</span><span class="sxs-lookup"><span data-stu-id="ae212-104">A [struct](../../csharp/language-reference/keywords/struct.md) cannot call a base class constructor; remove the call to the base class constructor.</span></span>  
  
 <span data-ttu-id="ae212-105">다음 샘플에서는 CS0522를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="ae212-105">The following sample generates CS0522:</span></span>  
  
```csharp  
// CS0522.cs  
public class clx  
{  
   public clx(int i)  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
  
public struct cly  
{  
   public cly(int i):base(0)   // CS0522  
   // try the following line instead  
   // public cly(int i)  
   {  
   }  
}  
```
