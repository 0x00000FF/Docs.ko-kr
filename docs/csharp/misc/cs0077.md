# <a name="compiler-error-cs0077"></a><span data-ttu-id="71418-102">컴파일러 오류 CS0077</span><span class="sxs-lookup"><span data-stu-id="71418-102">Compiler Error CS0077</span></span>
<span data-ttu-id="71418-103">as 연산자는 참조 형식 또는 nullable 형식과 함께 사용해야 합니다. 'int'는 null을 허용하지 않는 값 형식입니다.</span><span class="sxs-lookup"><span data-stu-id="71418-103">The as operator must be used with a reference type or nullable type ('int' is a non-nullable value type).</span></span>  
  
 <span data-ttu-id="71418-104">[as](../../csharp/language-reference/keywords/as.md) 연산자가 [값 형식](../../csharp/language-reference/keywords/value-types.md)으로 전달되었습니다.</span><span class="sxs-lookup"><span data-stu-id="71418-104">The [as](../../csharp/language-reference/keywords/as.md) operator was passed a [value type](../../csharp/language-reference/keywords/value-types.md).</span></span> <span data-ttu-id="71418-105">`as` 는 [null](../../csharp/language-reference/keywords/null.md)을 반환할 수 있기 때문에 [참조 형식](../../csharp/language-reference/keywords/reference-types.md) 또는 nullable 형식만 전달할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="71418-105">Because `as` can return [null](../../csharp/language-reference/keywords/null.md), it can only be passed [reference types](../../csharp/language-reference/keywords/reference-types.md) or nullable type.</span></span> <span data-ttu-id="71418-106">Nullable 형식에 대 한 자세한 내용은 참조 [Nullable 형식](../../csharp/programming-guide/nullable-types/index.md)합니다.</span><span class="sxs-lookup"><span data-stu-id="71418-106">For more information about nullable types, see [Nullable Types](../../csharp/programming-guide/nullable-types/index.md).</span></span>  
  
 <span data-ttu-id="71418-107">다음 샘플에서는 CS0077을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="71418-107">The following sample generates CS0077:</span></span>  
  
```  
// CS0077.cs  
using System;  
  
class C  
{  
}  
  
struct S  
{  
}  
  
class M  
{  
   public static void Main()  
   {  
      object o1, o2;  
      C c;  
      S s;  
  
      o1 = new C();  
      o2 = new S();  
  
      s = o2 as S;  // CS0077, S is not a reference type.  
      // try the following line instead  
      // c = o1 as C;  
   }  
}  
```
