# <a name="compiler-error-cs0077"></a>컴파일러 오류 CS0077
as 연산자는 참조 형식 또는 nullable 형식과 함께 사용해야 합니다. 'int'는 null을 허용하지 않는 값 형식입니다.  
  
 [as](../../csharp/language-reference/operators/type-testing-and-conversion-operators.md#as-operator) 연산자가 [값 형식](../../csharp/language-reference/keywords/value-types.md)으로 전달되었습니다. `as` 는 [null](../../csharp/language-reference/keywords/null.md)을 반환할 수 있기 때문에 [참조 형식](../../csharp/language-reference/keywords/reference-types.md) 또는 nullable 형식만 전달할 수 있습니다. Nullable 형식에 대 한 자세한 내용은 참조 하세요. [Nullable 형식](../../csharp/programming-guide/nullable-types/index.md)합니다.  
  
 다음 샘플에서는 CS0077을 생성합니다.  
  
```csharp  
// CS0077.cs  
using System;  
  
class C  
{  
}  
  
struct S  
{  
}  
  
class M  
{  
   public static void Main()  
   {  
      object o1, o2;  
      C c;  
      S s;  
  
      o1 = new C();  
      o2 = new S();  
  
      s = o2 as S;  // CS0077, S is not a reference type.  
      // try the following line instead  
      // c = o1 as C;  
   }  
}  
```
