# <a name="compiler-error-cs0155"></a><span data-ttu-id="4df4a-102">컴파일러 오류 CS0155</span><span class="sxs-lookup"><span data-stu-id="4df4a-102">Compiler Error CS0155</span></span>
<span data-ttu-id="4df4a-103">Catch 또는 Throw된 형식은 System.Exception에서 파생되어야 합니다.</span><span class="sxs-lookup"><span data-stu-id="4df4a-103">The type caught or thrown must be derived from System.Exception</span></span>  
  
 <span data-ttu-id="4df4a-104">**System.Exception** 에서 파생되지 않는 데이터 형식을 [catch](../../csharp/language-reference/keywords/try-catch.md) 블록에 전달하려고 했습니다.</span><span class="sxs-lookup"><span data-stu-id="4df4a-104">An attempt was made to pass a data type that does not derive from **System.Exception** into a [catch](../../csharp/language-reference/keywords/try-catch.md) block.</span></span> <span data-ttu-id="4df4a-105">**System.Exception** 에서 파생된 데이터 형식만 **catch** 블록에 전달할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="4df4a-105">Only data types that derive from **System.Exception** can be passed into a **catch** block.</span></span> <span data-ttu-id="4df4a-106">자세한 내용은 [예외 및 예외 처리](../../csharp/programming-guide/exceptions/index.md)합니다.</span><span class="sxs-lookup"><span data-stu-id="4df4a-106">For more information, see [Exceptions and Exception Handling](../../csharp/programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="4df4a-107">다음 샘플에서는 CS0155를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="4df4a-107">The following sample generates CS0155:</span></span>  
  
```csharp  
// CS0155.cs  
using System;  
  
namespace MyNamespace  
{  
    public class MyClass2  
    // try the following line instead  
    // public class MyClass2 : Exception  
    {  
    }  
    public class MyClass  
    {  
        public static void Main()  
        {  
            try  
            {  
            }  
            catch (MyClass2)   // CS0155, resolves if you derive MyClass2 from Exception  
            {  
            }  
        }  
    }  
}  
```
