## <a name="example"></a><span data-ttu-id="ff781-107">예제</span><span class="sxs-lookup"><span data-stu-id="ff781-107">Example</span></span>

```  
// CS0425.cs

{  
}  
  
class C2  
{  
}

{  
    void F<ItemType>(ItemType item) where ItemType : C1;  
}

    public void F<ItemType>(ItemType item) where ItemType : C2  // CS0425

    {  
    }  
}  
```
