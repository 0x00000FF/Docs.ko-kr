# <a name="compiler-error-cs0243"></a><span data-ttu-id="a1a4d-102">컴파일러 오류 CS0243</span><span class="sxs-lookup"><span data-stu-id="a1a4d-102">Compiler Error CS0243</span></span>

<span data-ttu-id="a1a4d-103">'method'는 재정의 메서드이기 때문에 Conditional 특성을 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="a1a4d-103">The Conditional attribute is not valid on 'method' because it is an override method</span></span>  
  
 <span data-ttu-id="a1a4d-104">[Conditional](../../csharp/programming-guide/concepts/attributes/common-attributes.md#Conditional) 특성은 [override](../../csharp/language-reference/keywords/override.md) 키워드로 표시된 메서드에서 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="a1a4d-104">The [Conditional](../../csharp/programming-guide/concepts/attributes/common-attributes.md#Conditional) attribute is not allowed on a method that is marked with the [override](../../csharp/language-reference/keywords/override.md) keyword.</span></span> <span data-ttu-id="a1a4d-105">자세한 내용은 [하는 경우 사용 하 여 Override 및 New 키워드를](../../csharp/programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md)입니다.</span><span class="sxs-lookup"><span data-stu-id="a1a4d-105">For more information, see [Knowing When to Use Override and New Keywords](../../csharp/programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span></span>  
  
 <span data-ttu-id="a1a4d-106">컴파일러는 재정의 메서드에 바인딩되지 않습니다. 컴파일러는 기본 메서드에만 바인딩되며, 공용 언어 런타임에서 재정의를 적절하게 호출합니다.</span><span class="sxs-lookup"><span data-stu-id="a1a4d-106">The compiler never binds to override methods; it only binds to the base method and the common language runtime calls the override, as appropriate.</span></span>  
  
 <span data-ttu-id="a1a4d-107">다음 샘플에서는 CS0243을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="a1a4d-107">The following sample generates CS0243:</span></span>  

```csharp
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```
