# <a name="compiler-error-cs0243"></a><span data-ttu-id="ed3d7-102">컴파일러 오류 CS0243</span><span class="sxs-lookup"><span data-stu-id="ed3d7-102">Compiler error CS0243</span></span>

<span data-ttu-id="ed3d7-103">'method'는 재정의 메서드이기 때문에 Conditional 특성을 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="ed3d7-103">The Conditional attribute is not valid on 'method' because it is an override method</span></span>  
  
<span data-ttu-id="ed3d7-104"><xref:System.Diagnostics.ConditionalAttribute> [재정의](../language-reference/keywords/override.md) 키워드로 표시된 메서드에는 특성이 허용되지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="ed3d7-104">The <xref:System.Diagnostics.ConditionalAttribute> attribute is not allowed on a method that is marked with the [override](../language-reference/keywords/override.md) keyword.</span></span> <span data-ttu-id="ed3d7-105">자세한 내용은 [재정의 및 새 키워드를 사용할 시기 를 참조하십시오.](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md)</span><span class="sxs-lookup"><span data-stu-id="ed3d7-105">For more information, see [Knowing When to Use Override and New Keywords](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span></span>  
  
<span data-ttu-id="ed3d7-106">컴파일러는 메서드를 재정의하기 위해 바인딩되지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="ed3d7-106">The compiler never binds to override methods.</span></span> <span data-ttu-id="ed3d7-107">기본 메서드에만 바인딩되며 공통 언어 런타임은 적절한 경우 재정의를 호출합니다.</span><span class="sxs-lookup"><span data-stu-id="ed3d7-107">It only binds to the base method, and the common language runtime calls the override, as appropriate.</span></span>  
  
<span data-ttu-id="ed3d7-108">다음 코드는 CS0243을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="ed3d7-108">The following code generates CS0243:</span></span>  

```csharp
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```
