# <a name="compiler-error-cs0017"></a><span data-ttu-id="ab941-102">컴파일러 오류 CS0017</span><span class="sxs-lookup"><span data-stu-id="ab941-102">Compiler Error CS0017</span></span>

 <span data-ttu-id="ab941-105">프로그램 하나만 [Main](../../csharp/programming-guide/main-and-command-args/index.md) 메서드.</span><span class="sxs-lookup"><span data-stu-id="ab941-105">A program can only have one [Main](../../csharp/programming-guide/main-and-command-args/index.md) method.</span></span>  
  
 <span data-ttu-id="ab941-106">이 오류를 해결 하려면 하나를 제외한 코드의 Main 메서드를 삭제 하거나 하거나 사용할 수는 [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) 컴파일러 옵션을 사용 하려는 Main 메서드를 지정 합니다.</span><span class="sxs-lookup"><span data-stu-id="ab941-106">To resolve this error, you can either delete all Main methods in your code, except one, or you can use the [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) compiler option to specify which Main method you want to use.</span></span>  
  
 <span data-ttu-id="ab941-107">다음 샘플에서는 CS0017을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="ab941-107">The following sample generates CS0017:</span></span>  
  
```  
// CS0017.cs  
// compile with: /target:exe  
public class clx  
{  
   static public void Main()  
   {  
   }  
}  
  
public class cly  
{  
   public static void Main()   // CS0017, delete one Main or use /main  
   {  
   }  
}  
```
