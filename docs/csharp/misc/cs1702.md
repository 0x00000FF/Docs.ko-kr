# <a name="compiler-warning-level-3-cs1702"></a>컴파일러 경고(수준 3) CS1702
"Assembly Name #1" 어셈블리 참조가 "Assembly Name #2"와 일치하는 것으로 간주합니다. 런타임 정책을 지정해야 합니다.  
  
 두 개의 어셈블리 참조에 서로 다른 빌드 및/또는 수정 번호가 있으므로 자동으로 통합되지 않습니다. 응용 프로그램 .config 파일의 지시문을 사용하여 강제로 통합하는 런타임 정책을 제공해야 할 수도 있습니다.
