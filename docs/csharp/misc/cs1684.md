## <a name="example"></a><span data-ttu-id="1b792-108">예제</span><span class="sxs-lookup"><span data-stu-id="1b792-108">Example</span></span>  
  
```csharp  
// CS1684_a.cs  
// compile with: /target:library /keyfile:CS1684.key  
public class A {  
   public void Test() {}  
}  
  
public class C2 {}  
```  
  
## <a name="example"></a><span data-ttu-id="1b792-109">예제</span><span class="sxs-lookup"><span data-stu-id="1b792-109">Example</span></span>  
  
```csharp  
// CS1684_b.cs  
// compile with: /target:library /r:cs1684_a.dll  
// post-build command: del /f CS1684_a.dll  
using System;  
public class Ref   
{  
   public static A GetA() { return new A(); }  
   public static C2 GetC() { return new C2(); }  
}  
```
