# <a name="compiler-error-cs0308"></a><span data-ttu-id="3af93-102">컴파일러 오류 CS0308</span><span class="sxs-lookup"><span data-stu-id="3af93-102">Compiler Error CS0308</span></span>
<span data-ttu-id="3af93-103">제네릭이 아닌 형식 또는 메서드 'identifier'는 형식 인수와 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="3af93-103">The non-generic type-or-method 'identifier' cannot be used with type arguments.</span></span>  
  
 <span data-ttu-id="3af93-104">메서드 또는 형식이 제네릭이 아니지만 형식 인수와 사용되었습니다.</span><span class="sxs-lookup"><span data-stu-id="3af93-104">The method or type is not generic, but it was used with type arguments.</span></span> <span data-ttu-id="3af93-105">이 오류를 방지하려면 대괄호와 형식 인수를 제거하거나 메서드 또는 형식을 제네릭 메서드 또는 형식으로 다시 선언합니다.</span><span class="sxs-lookup"><span data-stu-id="3af93-105">To avoid this error, remove the angled brackets and type arguments, or redeclare the method or type as a generic method or type.</span></span>  
  
 <span data-ttu-id="3af93-106">다음 예제에서는 CS0308을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="3af93-106">The following example generates CS0308:</span></span>  
  
```  
// CS0308a.cs  
class MyClass  
{  
   public void F() {}  
   public static void Main()  
   {  
      F<int>();  // CS0308 – F is not generic.  
      // Try this instead:  
      // F();  
   }  
}  
```  
  
 <span data-ttu-id="3af93-107">또한 다음 예제에서는 CS0308을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="3af93-107">The following example also generates CS0308.</span></span> <span data-ttu-id="3af93-108">오류를 해결하려면 "System.Collections.Generic 사용" 지시문을 사용합니다.</span><span class="sxs-lookup"><span data-stu-id="3af93-108">To resolve the error, use the directive "using System.Collections.Generic."</span></span>  
  
```  
// CS0308b.cs  
// compile with: /t:library  
using System.Collections;  
// To resolve, uncomment the following line:  
// using System.Collections.Generic;  
public class MyStack<T>  
{  
    // Store the elements of the stack:  
    private T[] items = new T[100];  
    private int stack_counter = 0;  
  
    // Define the iterator block:  
    public IEnumerator<T> GetEnumerator()   // CS0308  
    {  
        for (int i = stack_counter - 1 ; i >= 0; i--)  
        yield return items[i];  
    }  
}  
```
