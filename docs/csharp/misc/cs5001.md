<span data-ttu-id="2a2e8-103">'program' 프로그램에는 진입점에 적합한 정적 'Main' 메서드가 포함되어 있지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="2a2e8-103">Program 'program' does not contain a static 'Main' method suitable for an entry point</span></span>  
  
 <span data-ttu-id="2a2e8-104">정적 없는 경우이 오류가 발생 [Main](../../csharp/programming-guide/main-and-command-args/index.md) 올바른 서명이 있는 메서드는 실행 파일을 생성 하는 코드에서 찾을 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="2a2e8-104">This error occurs when no static [Main](../../csharp/programming-guide/main-and-command-args/index.md) method with a correct signature is found in the code that produces an executable file.</span></span> <span data-ttu-id="2a2e8-105">이 오류는 진입점 함수인 `Main`이 소문자 `main`과 같은 잘못된 대/소문자로 정의된 경우에도 발생할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="2a2e8-105">This error also occurs if the entry point function, `Main`, is defined with the wrong case, such as lower-case `main`.</span></span>  
  
-   <span data-ttu-id="2a2e8-106">`Main`반환 해야 하며 정적으로 선언 해야 [void](../../csharp/language-reference/keywords/void.md) 또는 [int](../../csharp/language-reference/keywords/int.md), 하거나 매개 변수가 없습니다. 그렇지 않으면 매개 변수 형식이 있어야 `string[]`합니다.</span><span class="sxs-lookup"><span data-stu-id="2a2e8-106">`Main` must be declared as static and it must return [void](../../csharp/language-reference/keywords/void.md) or [int](../../csharp/language-reference/keywords/int.md), and it must have either no parameters or else one parameter of type `string[]`.</span></span>
