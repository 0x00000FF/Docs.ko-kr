# <a name="compiler-error-cs5001"></a><span data-ttu-id="8b74e-102">컴파일러 오류 CS5001</span><span class="sxs-lookup"><span data-stu-id="8b74e-102">Compiler Error CS5001</span></span>
<span data-ttu-id="8b74e-103">'program' 프로그램에는 진입점에 적합한 정적 'Main' 메서드가 포함되어 있지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="8b74e-103">Program 'program' does not contain a static 'Main' method suitable for an entry point</span></span>  
  
 <span data-ttu-id="8b74e-104">이 오류는 올바른 서명이 있는 고정 [Main](../../csharp/programming-guide/main-and-command-args/index.md) 메서드가 실행 파일을 생성하는 코드에서 발견된 경우 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="8b74e-104">This error occurs when no static [Main](../../csharp/programming-guide/main-and-command-args/index.md) method with a correct signature is found in the code that produces an executable file.</span></span> <span data-ttu-id="8b74e-105">이 오류는 진입점 함수인 `Main`이 소문자 `main`과 같은 잘못된 대/소문자로 정의된 경우에도 발생할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="8b74e-105">This error also occurs if the entry point function, `Main`, is defined with the wrong case, such as lower-case `main`.</span></span>  
  
-   <span data-ttu-id="8b74e-106">`Main` 은 고정으로 선언되고 [void](../../csharp/language-reference/keywords/void.md) 또는 [int](../../csharp/language-reference/keywords/int.md)를 반환해야 하며 매개 변수가 없거나 `string[]`형식의 매개 변수 중 하나가 있어야 합니다.</span><span class="sxs-lookup"><span data-stu-id="8b74e-106">`Main` must be declared as static and it must return [void](../../csharp/language-reference/keywords/void.md) or [int](../../csharp/language-reference/keywords/int.md), and it must have either no parameters or else one parameter of type `string[]`.</span></span>
