# <a name="compiler-error-cs0081"></a><span data-ttu-id="d2322-102">컴파일러 오류 CS0081</span><span class="sxs-lookup"><span data-stu-id="d2322-102">Compiler Error CS0081</span></span>
<span data-ttu-id="d2322-103">형식 매개 변수 선언은 형식이 아니라 식별자여야 합니다.</span><span class="sxs-lookup"><span data-stu-id="d2322-103">Type parameter declaration must be an identifier not a type</span></span>  
  
 <span data-ttu-id="d2322-104">제네릭 메서드 또는 형식을 선언할 때 형식 매개 변수를 "T" 또는 "inputType"과 같은 식별자로 지정합니다.</span><span class="sxs-lookup"><span data-stu-id="d2322-104">When you declare a generic method or type, specify the type parameter as an identifier, for example "T" or "inputType".</span></span> <span data-ttu-id="d2322-105">클라이언트 코드가 메서드를 호출하면 이 메서드가 형식을 제공하며 메서드 또는 클래스 본문에서 각 식별자 발생을 대체합니다.</span><span class="sxs-lookup"><span data-stu-id="d2322-105">When client code calls the method, it supplies the type, which replaces each occurrence of the identifier in the method or class body.</span></span> <span data-ttu-id="d2322-106">자세한 내용은 참조 [제네릭 형식 매개 변수](../../csharp/programming-guide/generics/generic-type-parameters.md)합니다.</span><span class="sxs-lookup"><span data-stu-id="d2322-106">For more information, see [Generic Type Parameters](../../csharp/programming-guide/generics/generic-type-parameters.md).</span></span>  
  
```  
// CS0081.cs  
class MyClass  
{  
   public void F<int>() {}   // CS0081  
   public void F<T>(T input) {}   // OK  
  
   public static void Main()  
   {  
      MyClass a = new MyClass();  
      a.F<int>(2);  
      a.F<double>(.05);  
   }  
}  
```
