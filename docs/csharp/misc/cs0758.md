<span data-ttu-id="d021f-103">두 부분 메서드(Partial Method) 선언 모두 params 매개 변수를 사용하거나 params 매개 변수를 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="d021f-103">Both partial method declarations must use a params parameter or neither may use a params parameter</span></span>  
  
 <span data-ttu-id="d021f-104">부분 메서드의 한 부분이 `params` 매개 변수를 지정하는 경우 다른 부분에서도 지정해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="d021f-104">If one part of a partial method specifies a `params` parameter, the other part must specify one also.</span></span>
