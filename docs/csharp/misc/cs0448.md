# <a name="compiler-error-cs0448"></a><span data-ttu-id="fae78-102">컴파일러 오류 CS0448</span><span class="sxs-lookup"><span data-stu-id="fae78-102">Compiler Error CS0448</span></span>
<span data-ttu-id="fae78-103">++ 또는 -- 연산자의 반환 형식은 포함하는 형식이거나 포함하는 형식에서 파생되어야 합니다.</span><span class="sxs-lookup"><span data-stu-id="fae78-103">The return type for ++ or -- operator must be the containing type or derived from the containing type</span></span>  
  
 <span data-ttu-id="fae78-104">`++` 또는 `--` 연산자를 재정의하는 경우, 포함하는 형식과 동일한 형식을 반환하거나 포함하는 형식에서 파생된 형식을 반환해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="fae78-104">When you override the `++` or `--` operators, they must return the same type as the containing type, or return a type that is derived from the containing type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="fae78-105">예제</span><span class="sxs-lookup"><span data-stu-id="fae78-105">Example</span></span>  
 <span data-ttu-id="fae78-106">다음 샘플에서는 CS0448을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="fae78-106">The following sample generates CS0448.</span></span>  
  
```csharp  
// CS0448.cs  
class C5  
{  
   public static int operator ++(C5 c) { return null; }   // CS0448  
   public static C5 operator --(C5 c) { return null; }   // OK  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="fae78-107">예제</span><span class="sxs-lookup"><span data-stu-id="fae78-107">Example</span></span>  
 <span data-ttu-id="fae78-108">다음 샘플에서는 CS0448을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="fae78-108">The following sample generates CS0448.</span></span>

```csharp  
   public static S? operator ++(S s) { return new S(); }   // CS0448

   public static T operator --(T t) { return new T(); }  
   public static T operator ++(T t) { return new T(); }  
  
   public static T? operator --(T? t) { return new T(); }  
   public static T? operator ++(T? t) { return new T(); }  
  
   public static void Main() {}  
}  
```
