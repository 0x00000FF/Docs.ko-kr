# <a name="compiler-error-cs1577"></a>컴파일러 오류 CS1577
어셈블리를 생성하지 못했습니다. reason  
  
 C# 컴파일러가 소스 코드를 성공적으로 컴파일한 후 컴파일의 어셈블리 생성 부분이 실패했습니다. ALink 유틸리티에 대 한 오류 설명서를 참조 하십시오 ([Al.exe 도구 오류 및 경고](http://msdn.microsoft.com/library/7f125d49-0a03-47a6-9ba9-d61a679a7d4b))에 대 한 자세한 내용은 합니다.
