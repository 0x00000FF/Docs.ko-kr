# <a name="compiler-error-cs1631"></a><span data-ttu-id="fe87e-102">컴파일러 오류 CS1631</span><span class="sxs-lookup"><span data-stu-id="fe87e-102">Compiler Error CS1631</span></span>
<span data-ttu-id="fe87e-103">catch 절 본문에서는 값을 생성할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="fe87e-103">Cannot yield a value in the body of a catch clause</span></span>  
  
 <span data-ttu-id="fe87e-104">yield 문은 catch 절의 본문 내에서 허용되지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="fe87e-104">The yield statement is not allowed from within the body of a catch clause.</span></span> <span data-ttu-id="fe87e-105">이 오류를 방지하려면 yield 문을 catch 절 본문 밖으로 이동합니다.</span><span class="sxs-lookup"><span data-stu-id="fe87e-105">To avoid this error, move the yield statement outside the body of the catch clause.</span></span>  
  
 <span data-ttu-id="fe87e-106">다음 샘플에서는 CS1631을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="fe87e-106">The following sample generates CS1631:</span></span>  
  
```  
// CS1631.cs  
using System;  
using System.Collections;  
  
public class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()   
   {  
      try  
      {  
      }  
      catch(Exception e)  
      {  
        yield return this;  // CS1631  
      }  
   }    
  
   public static void Main()   
   {  
   }  
}  
```
