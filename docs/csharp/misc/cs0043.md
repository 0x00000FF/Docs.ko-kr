# <a name="compiler-error-cs0043"></a>컴파일러 오류 CS0043
'file' PDB 파일에 잘못되었거나 사용되지 않는 형식이 있습니다. 삭제한 다음 다시 빌드하세요.  
  
 이 컴파일에 대한 .pdb 파일을 삭제한 후 다시 컴파일하세요.
