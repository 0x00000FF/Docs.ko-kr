# <a name="compiler-error-cs1630"></a>컴파일러 오류 CS1630
/errorreport의 'option' 옵션이 잘못되었습니다. prompt, send, queue 또는 none이어야 합니다.  
  
 명령줄 옵션 [-errorreport](../language-reference/compiler-options/errorreport-compiler-option.md) 뒤에는 **prompt**, **send**, **queue**또는 **none**을 입력 하 여 내부 컴파일러 오류가 발생할 때 수행할 작업을 지정 해야 합니다.
