<span data-ttu-id="af5a1-103">정적 읽기 전용 필드 'name'의 필드는 ref 또는 out으로 전달할 수 없습니다. 단 정적 생성자에서는 예외입니다.</span><span class="sxs-lookup"><span data-stu-id="af5a1-103">Fields of static readonly field 'name' cannot be passed ref or out (except in a static constructor)</span></span>  
  
 <span data-ttu-id="af5a1-104">[readonly](../../csharp/language-reference/keywords/readonly.md) 키워드로 표시된 필드(변수)를 생성자 내부를 제외하고 [ref](../../csharp/language-reference/keywords/ref.md) 또는 [out](../../csharp/language-reference/keywords/out.md) 매개 변수로 전달할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="af5a1-104">A field (variable) marked with the [readonly](../../csharp/language-reference/keywords/readonly.md) keyword cannot be passed either to a [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out.md) parameter except inside a constructor.</span></span> <span data-ttu-id="af5a1-105">자세한 내용은 [필드](../../csharp/programming-guide/classes-and-structs/fields.md)를 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="af5a1-105">For more information, see [Fields](../../csharp/programming-guide/classes-and-structs/fields.md).</span></span>  
  
 <span data-ttu-id="af5a1-106">CS0192는 `readonly` 필드가 [static](../../csharp/language-reference/keywords/static.md) 인데 생성자가 `static`으로 표시되지 않은 경우에도 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="af5a1-106">CS0192 also results if the `readonly` field is [static](../../csharp/language-reference/keywords/static.md) and the constructor is not marked `static`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="af5a1-107">예제</span><span class="sxs-lookup"><span data-stu-id="af5a1-107">Example</span></span>  
 <span data-ttu-id="af5a1-108">다음 샘플에서는 CS0192를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="af5a1-108">The following sample generates CS0192.</span></span>  
  
```  
// CS0192.cs  
class MyClass  
{  
    public readonly int TestInt = 6;  
    static void TestMethod(ref int testInt)  
    {  
        testInt = 0;  
    }  
  
    MyClass()  
    {  
        TestMethod(ref TestInt);   // OK  
    }  
  
    public void PassReadOnlyRef()  
    {  
        TestMethod(ref TestInt);   // CS0192  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
