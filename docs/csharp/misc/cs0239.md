# <a name="compiler-error-cs0239"></a>컴파일러 오류 CS0239
'member': 'inherited member' 상속된 멤버가 봉인되어 있으므로 재정의할 수 없습니다.  
  
 멤버가 [봉인된](../../csharp/language-reference/keywords/override.md) 상속된 멤버를 [재정의](../../csharp/language-reference/keywords/sealed.md) 할 수 없습니다. 자세한 내용은 [Checked 및 Unchecked](../../csharp/language-reference/keywords/checked-and-unchecked.md)합니다.  
  
 다음 샘플에서는 CS0239를 생성합니다.  
  
```csharp  
// CS0239.cs  
abstract class MyClass  
{  
   public abstract void f();  
}  
  
class MyClass2 : MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public override sealed void f()  
   {  
   }  
}  
  
class MyClass3 : MyClass2  
{  
   public override void f()   // CS0239  
   // Try the following definition instead:  
   // public new void f()  
   {  
   }  
}  
```
