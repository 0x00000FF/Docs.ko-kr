# <a name="compiler-error-cs0761"></a><span data-ttu-id="5ecb3-102">컴파일러 오류 CS0761</span><span class="sxs-lookup"><span data-stu-id="5ecb3-102">Compiler Error CS0761</span></span>
<span data-ttu-id="5ecb3-103">두 부분 메서드 선언 ' 메서드\<T >' 일관성이 없는 형식 매개 변수 제약 조건이 있어야 합니다.</span><span class="sxs-lookup"><span data-stu-id="5ecb3-103">Partial method declarations of 'method\<T>' have inconsistent type parameter constraints.</span></span>  
  
 <span data-ttu-id="5ecb3-104">부분 메서드(Partial method)가 구현인 경우 제네릭 형식 제약 조건은 메서드 서명에 정의된 제약 조건과 동일해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="5ecb3-104">If a partial method has an implementation, the generic type constraint must be identical to the constraint defined on the method signature.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="5ecb3-105">이 오류를 해결하려면</span><span class="sxs-lookup"><span data-stu-id="5ecb3-105">To correct this error</span></span>  
  
1.  <span data-ttu-id="5ecb3-106">제네릭 형식 제약 조건을 부분 메서드(Partial method)의 각 부분에서 동일하게 만듭니다.</span><span class="sxs-lookup"><span data-stu-id="5ecb3-106">Make the generic type constraints identical on each part of the partial method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="5ecb3-107">예제</span><span class="sxs-lookup"><span data-stu-id="5ecb3-107">Example</span></span>  
 <span data-ttu-id="5ecb3-108">다음 코드에서는 CS0761을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="5ecb3-108">The following code generates CS0761:</span></span>  
  
```csharp

{  
    partial void Part<T>() where T : class;  
    partial void Part<T>() where T : struct // CS0761

    {  
        return 1;  
    }  
}
```  
