## <a name="example"></a><span data-ttu-id="28f1b-107">예</span><span class="sxs-lookup"><span data-stu-id="28f1b-107">Example</span></span>  
 <span data-ttu-id="28f1b-108">다음 샘플에서는 CS0411을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="28f1b-108">The following sample generates CS0411:</span></span>  
  
```csharp  
// CS0411.cs  
class C  
{  
    void G<T>()  
    {  
    }  
  
    public static void Main()  
    {  
        G();  // CS0411  
        // Try this instead:  
        // G<int>();  
    }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="28f1b-109">예</span><span class="sxs-lookup"><span data-stu-id="28f1b-109">Example</span></span>  
 <span data-ttu-id="28f1b-110">기타 가능한 오류 사례에는 매개 변수가 `null`, 즉 형식 정보가 없는 경우가 포함됩니다.</span><span class="sxs-lookup"><span data-stu-id="28f1b-110">Other possible error cases include when the parameter is `null`, which has no type information:</span></span>

    public void F<T>(T t) where T : C
