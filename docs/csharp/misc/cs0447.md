<span data-ttu-id="a20c5-103">특성은 형식 인수에 사용할 수 없고 형식 매개 변수에만 사용할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="a20c5-103">Attributes cannot be used on type arguments, only on type parameters</span></span>

 <span data-ttu-id="a20c5-104">이 오류는 호출 문에서 사용된 형식 인수에 특성을 적용하는 경우에 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="a20c5-104">This error occurs when you apply an attribute to a type argument that occurs in an invocation statement.</span></span> <span data-ttu-id="a20c5-105">다음과 같은 클래스 또는 메서드 선언문의 형식 매개 변수에 특성을 적용할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="a20c5-105">It is acceptable to apply an attribute to a type parameter in a class or method declaration statement such as the following:</span></span>
