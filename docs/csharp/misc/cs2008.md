# <a name="compiler-error-cs2008"></a>컴파일러 오류 CS2008
지정한 입력이 없습니다.  
  
 컴파일러를 호출하고 컴파일러 옵션을 지정했지만 소스 코드 파일이 전달되지 않았습니다.
