# <a name="compiler-error-cs0745"></a><span data-ttu-id="95ab9-102">컴파일러 오류 CS0745</span><span class="sxs-lookup"><span data-stu-id="95ab9-102">Compiler Error CS0745</span></span>

 <span data-ttu-id="95ab9-104">`group` 절에 대한 패턴은 다음 예제와 같이 `group...by` 뒤에 선택적 `into`가 오는 것입니다.</span><span class="sxs-lookup"><span data-stu-id="95ab9-104">The pattern for the `group` clause is `group...by` followed by an optional `into`, as shown in the following example:</span></span>
