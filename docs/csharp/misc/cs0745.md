# <a name="compiler-error-cs0745"></a><span data-ttu-id="204e0-102">컴파일러 오류 CS0745</span><span class="sxs-lookup"><span data-stu-id="204e0-102">Compiler Error CS0745</span></span>
<span data-ttu-id="204e0-103">키워드 'by'가 필요합니다.</span><span class="sxs-lookup"><span data-stu-id="204e0-103">Expected contextual keyword 'by'</span></span>  
  
 <span data-ttu-id="204e0-104">`group` 절에 대한 패턴은 다음 예제와 같이 `group...by` 뒤에 선택적 `into`가 오는 것입니다.</span><span class="sxs-lookup"><span data-stu-id="204e0-104">The pattern for the `group` clause is `group...by` followed by an optional `into`, as shown in the following example:</span></span>

## <a name="example"></a><span data-ttu-id="204e0-108">예제</span><span class="sxs-lookup"><span data-stu-id="204e0-108">Example</span></span>  
 <span data-ttu-id="204e0-109">다음 코드에서는 CS0745를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="204e0-109">The following code generates CS0745:</span></span>  
  
```csharp  
// cs0745.cs

using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        string[] names = { "Bob", "Bill", "Jonetta", "Mary" };
```  
