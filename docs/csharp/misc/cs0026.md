## <a name="example"></a><span data-ttu-id="05386-107">예제</span><span class="sxs-lookup"><span data-stu-id="05386-107">Example</span></span>  
 <span data-ttu-id="05386-108">다음 예제에서는 CS0026을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="05386-108">The following example generates CS0026:</span></span>  
  
```csharp  
// CS0026.cs

{  
   public static int i = 0;  
  
   public static void Main()  
   {

      this.i = this.i + 1;     
      // Try the following line instead:  
      // i = i + 1;  
   }  
}  
```
