 <span data-ttu-id="39778-104">[is](../../csharp/language-reference/keywords/is.md) 및 [as](../../csharp/language-reference/keywords/as.md) 키워드는 포인터 형식에 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="39778-104">The [is](../../csharp/language-reference/keywords/is.md) and [as](../../csharp/language-reference/keywords/as.md) keywords are not valid for use on pointer types.</span></span> <span data-ttu-id="39778-105">자세한 내용은 [안전하지 않은 코드 및 포인터](../../csharp/programming-guide/unsafe-code-pointers/index.md)를 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="39778-105">For more information, see [Unsafe Code and Pointers](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span></span>

```  
// CS0244.cs  
// compile with: /unsafe  
  
class UnsafeTest  
{  
   unsafe static void SquarePtrParam (int* p)  
   {  
      bool b = p is object;   // CS0244 p is pointer  
   }  
  
   unsafe public static void Main()  
   {  
      int i = 5;  
      SquarePtrParam (&i);  
   }  
}  
```
