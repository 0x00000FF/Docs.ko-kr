 <span data-ttu-id="2aa04-104">[은](../../csharp/language-reference/operators/type-testing-and-conversion-operators.md#is-operator) 하 고 [으로](../../csharp/language-reference/operators/type-testing-and-conversion-operators.md#as-operator) 연산자는 포인터 형식에 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="2aa04-104">The [is](../../csharp/language-reference/operators/type-testing-and-conversion-operators.md#is-operator) and [as](../../csharp/language-reference/operators/type-testing-and-conversion-operators.md#as-operator) operators are not valid for use on pointer types.</span></span> <span data-ttu-id="2aa04-105">자세한 내용은 [안전하지 않은 코드 및 포인터](../../csharp/programming-guide/unsafe-code-pointers/index.md)를 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="2aa04-105">For more information, see [Unsafe Code and Pointers](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="2aa04-106">다음 샘플에서는 CS0244를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="2aa04-106">The following sample generates CS0244:</span></span>  
  
```csharp  
// CS0244.cs  
// compile with: /unsafe  
  
class UnsafeTest  
{  
   unsafe static void SquarePtrParam (int* p)  
   {  
      bool b = p is object;   // CS0244 p is pointer  
   }  
  
   unsafe public static void Main()  
   {  
      int i = 5;  
      SquarePtrParam (&i);  
   }  
}  
```
