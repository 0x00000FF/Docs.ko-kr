# <a name="compiler-error-cs1107"></a><span data-ttu-id="5a21c-102">컴파일러 오류 CS1107</span><span class="sxs-lookup"><span data-stu-id="5a21c-102">Compiler Error CS1107</span></span>
<span data-ttu-id="5a21c-103">매개 변수에는 'modifier name' 한정자 하나만 사용할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="5a21c-103">A parameter can only have one 'modifier name' modifier.</span></span>  
  
 <span data-ttu-id="5a21c-104">와 같은 매개 변수 한정자에 대 한 오류는 `this`, `ref`, `in`, 및 `out` 매개 변수 정의에 대 한 번 이상 표시 합니다.</span><span class="sxs-lookup"><span data-stu-id="5a21c-104">It is an error for parameter modifiers such as `this`, `ref`, `in`, and `out` to appear more than one time in a parameter definition.</span></span>  
  
## <a name="example"></a><span data-ttu-id="5a21c-105">예제</span><span class="sxs-lookup"><span data-stu-id="5a21c-105">Example</span></span>  
 <span data-ttu-id="5a21c-106">다음 예제에서는 CS1107을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="5a21c-106">The following example generates CS1107:</span></span>  
  
```csharp  
// cs1107.cs  
public static class Test
{
    // Extension methods.  
    public static void TestMethod(this this t) { } // CS1107  

}

public class TestTwo
{ 
    // Regular Instance Method  
    public void TestMethod(ref ref int i) { } // CS1107  

    // Regular Instance Method  
    public void TestMethod(in in double d) { } // CS1107  
}
```
