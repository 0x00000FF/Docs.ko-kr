## <a name="example"></a><span data-ttu-id="d6bea-105">예제</span><span class="sxs-lookup"><span data-stu-id="d6bea-105">Example</span></span>  
 <span data-ttu-id="d6bea-106">다음 예제에서는 CS1107을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="d6bea-106">The following example generates CS1107:</span></span>  
  
```csharp  
// cs1107.cs

{
    // Extension methods.  
    public static void TestMethod(this this t) { } // CS1107  

}

public class TestTwo
{
    // Regular Instance Method  
    public void TestMethod(ref ref int i) { } // CS1107  

    // Regular Instance Method  
    public void TestMethod(in in double d) { } // CS1107  
}
```
