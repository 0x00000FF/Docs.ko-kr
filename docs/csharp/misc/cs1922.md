# <a name="compiler-error-cs1922"></a><span data-ttu-id="d883c-102">컴파일러 오류 CS1922</span><span class="sxs-lookup"><span data-stu-id="d883c-102">Compiler Error CS1922</span></span>

<span data-ttu-id="d883c-103">컬렉션 이니셜라이저의 'type' 형식이 System.Collections.IEnumerable을 구현해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="d883c-103">Collection initializer requires its type 'type' to implement System.Collections.IEnumerable.</span></span>

<span data-ttu-id="d883c-104">형식과 함께 컬렉션 이니셜라이저를 사용하려면 형식이 `IEnumerable`을 구현해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="d883c-104">In order to use a collection initializer with a type, the type must implement `IEnumerable`.</span></span> <span data-ttu-id="d883c-105">이 오류는 개체 이니셜라이저를 사용하려다가 실수로 컬렉션 이니셜라이저 구문을 사용하는 경우에 발생할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="d883c-105">This error can occur if you accidentally use collection initializer syntax when you meant to use an object initializer.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="d883c-106">이 오류를 해결하려면</span><span class="sxs-lookup"><span data-stu-id="d883c-106">To correct this error</span></span>

- <span data-ttu-id="d883c-107">형식이 컬렉션을 나타내지 않는 경우 컬렉션 이니셜라이저 구문 대신 개체 이니셜라이저 구문을 사용합니다.</span><span class="sxs-lookup"><span data-stu-id="d883c-107">If the type does not represent a collection, use object initializer syntax instead of collection initializer syntax.</span></span>

- <span data-ttu-id="d883c-108">형식이 컬렉션을 나타내는 경우 컬렉션 이니셜라이저를 사용하여 해당 형식의 개체를 초기화하기 전에 `IEnumerable` 를 구현하도록 수정합니다.</span><span class="sxs-lookup"><span data-stu-id="d883c-108">If the type does represent a collection, modify it to implement `IEnumerable` before you can use collection initializers to initialize objects of that type.</span></span>

- <span data-ttu-id="d883c-109">형식이 컬렉션을 나타내고 소스 코드에 액세스할 수 없는 경우 클래스 생성자 또는 다른 초기화 메서드를 사용하여 해당 요소를 초기화합니다.</span><span class="sxs-lookup"><span data-stu-id="d883c-109">If the type represents a collection and you do not have access to the source code, just initialize its elements by using their class constructors or other initialization methods.</span></span>

## <a name="example"></a><span data-ttu-id="d883c-110">예제</span><span class="sxs-lookup"><span data-stu-id="d883c-110">Example</span></span>

<span data-ttu-id="d883c-111">다음 코드에서는 CS1922를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="d883c-111">The following code produces CS1922:</span></span>

```csharp
// cs1922.cs
public class Test
{
    public static void Main()
    {
        // Collection initializer.
        var tc = new TestClass  {1,"hello"} ; // CS1922

        // Object initializer.
        var tc2 = new TestClass { memberA = 1, memberB = "hello" }; // OK
    }
}

public class TestClass
{
    public int memberA { get; set; }
    public string memberB { get; set; }
}
```
