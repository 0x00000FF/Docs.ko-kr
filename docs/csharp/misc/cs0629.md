# <a name="compiler-error-cs0629"></a><span data-ttu-id="1b6e3-102">컴파일러 오류 CS0629</span><span class="sxs-lookup"><span data-stu-id="1b6e3-102">Compiler Error CS0629</span></span>
<span data-ttu-id="1b6e3-103">'member' 조건부 멤버는 'Type Name' 형식으로 'base class member' 인터페이스 멤버를 구현할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="1b6e3-103">Conditional member 'member' cannot implement interface member 'base class member' in type 'Type Name'</span></span>  
  
 <span data-ttu-id="1b6e3-104">[조건부](../../csharp/programming-guide/concepts/attributes/common-attributes.md#Conditional) 특성은 인터페이스의 구현에서 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="1b6e3-104">The [Conditional](../../csharp/programming-guide/concepts/attributes/common-attributes.md#Conditional) attribute cannot be used on the implementation of an interface.</span></span>  
  
 <span data-ttu-id="1b6e3-105">다음 샘플에서는 CS0629를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="1b6e3-105">The following sample generates CS0629:</span></span>  
  
```csharp  
// CS0629.cs  
interface MyInterface  
{  
   void MyMethod();  
}  
  
public class MyClass : MyInterface  
{  
   [System.Diagnostics.Conditional("debug")]  
   public void MyMethod()    // CS0629, remove the Conditional attribute  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
