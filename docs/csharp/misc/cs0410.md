## <a name="example"></a><span data-ttu-id="85df4-106">예제</span><span class="sxs-lookup"><span data-stu-id="85df4-106">Example</span></span>

```csharp
// CS0410.cs

{
    delegate void D(double d );
    static void F(int i) { }

    {
        D d = new D(F);  // CS0410
    }
}
```
