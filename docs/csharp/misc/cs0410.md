## <a name="example"></a><span data-ttu-id="ac44e-106">예제</span><span class="sxs-lookup"><span data-stu-id="ac44e-106">Example</span></span>

```csharp
// CS0410.cs

{
    delegate void D(double d );
    static void F(int i) { }

    static void Main()
    {
        D d = new D(F);  // CS0410
    }
}
```
