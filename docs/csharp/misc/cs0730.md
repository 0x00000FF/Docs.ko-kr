## <a name="example"></a><span data-ttu-id="2fc34-105">예</span><span class="sxs-lookup"><span data-stu-id="2fc34-105">Example</span></span>  
 <span data-ttu-id="2fc34-106">다음 샘플에서는 CS0730을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="2fc34-106">The following sample generates CS0730.</span></span> <span data-ttu-id="2fc34-107">이는 두 개의 원본 파일로 구성됩니다.</span><span class="sxs-lookup"><span data-stu-id="2fc34-107">It consists of two source files.</span></span> <span data-ttu-id="2fc34-108">먼저, `CS0730a.cs`라이브러리 파일을 컴파일하고, 해당 라이브러리 파일을 참조하는 `CS0730.cs` 파일을 컴파일합니다.</span><span class="sxs-lookup"><span data-stu-id="2fc34-108">First, compile the library file `CS0730a.cs`, and the compile the file `CS0730.cs` referencing the library file.</span></span>  
  
```csharp  
// CS0730a.cs  
// compile with: /t:library  
public class Outer  
{  
   public class Nested {}  
}  
```  
  
```csharp  
// CS0730.cs  
// compile with: /t:library /r:CS0730a.dll  
using System.Runtime.CompilerServices;  
  
[assembly:TypeForwardedToAttribute(typeof(Outer.Nested))]   // CS0730  
  
[assembly:TypeForwardedToAttribute(typeof(Outer))]   // OK  
```
