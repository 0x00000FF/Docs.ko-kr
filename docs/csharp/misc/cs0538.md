# <a name="compiler-error-cs0538"></a><span data-ttu-id="4bbba-102">컴파일러 오류 CS0538</span><span class="sxs-lookup"><span data-stu-id="4bbba-102">Compiler Error CS0538</span></span>
<span data-ttu-id="4bbba-103">명시적 인터페이스 선언에서 'name'은 인터페이스가 아닙니다.</span><span class="sxs-lookup"><span data-stu-id="4bbba-103">'name' in explicit interface declaration is not an interface</span></span>  
  
 <span data-ttu-id="4bbba-104">[인터페이스](../language-reference/keywords/interface.md)를 명시적으로 선언하려고 했지만 인터페이스가 지정되지 않았습니다.</span><span class="sxs-lookup"><span data-stu-id="4bbba-104">An attempt was made to explicitly declare an [interface](../language-reference/keywords/interface.md), but an interface was not specified.</span></span>  
  
 <span data-ttu-id="4bbba-105">다음 샘플에서는 CS0538을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="4bbba-105">The following sample generates CS0538:</span></span>  
  
```csharp  
// CS0538.cs  
interface MyIFace  
{  
   void F();  
}  
  
public class MyClass  
{  
   public void G()  
   {  
   }  
}  
  
class C: MyIFace  
{  
   void MyIFace.F()  
   {  
   }  
  
   void MyClass.G()   // CS0538, MyClass not an interface  
   {  
   }  
}  
```
