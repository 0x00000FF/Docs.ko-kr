# <a name="compiler-error-cs0585"></a>컴파일러 오류 CS0585
내부 컴파일러 오류: 'stage' 단계  
  
 컴파일러가 예기치 않은 구문을 분석할 수 없어 실패하는지 여부를 확인합니다. 시도 하지 않은 경우 [의견을 보내](https://docs.microsoft.com/visualstudio/ide/talk-to-us)합니다.
