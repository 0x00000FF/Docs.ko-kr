# <a name="compiler-error-cs1558"></a><span data-ttu-id="47550-102">컴파일러 오류 CS1558</span><span class="sxs-lookup"><span data-stu-id="47550-102">Compiler Error CS1558</span></span>
<span data-ttu-id="47550-103">'class'에 적합한 정적 Main 메서드가 없습니다.</span><span class="sxs-lookup"><span data-stu-id="47550-103">'class' does not have a suitable static Main method</span></span>  
  
 <span data-ttu-id="47550-104">[/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) 컴파일러 옵션에서 **Main** 메서드를 찾을 클래스를 지정했습니다.</span><span class="sxs-lookup"><span data-stu-id="47550-104">The [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) compiler option specified a class in which to look for a **Main** method.</span></span> <span data-ttu-id="47550-105">그러나 [Main](../../csharp/programming-guide/main-and-command-args/index.md) 메서드가 올바르게 정의되지 않았습니다.</span><span class="sxs-lookup"><span data-stu-id="47550-105">However, the [Main](../../csharp/programming-guide/main-and-command-args/index.md) method was not defined correctly.</span></span>  
  
 <span data-ttu-id="47550-106">다음 예제에서는 잘못된 반환 형식으로 인해 CS1558을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="47550-106">The following example generates CS1558 because of invalid return type.</span></span>  
  
```csharp  
// CS1558.cs  
// compile with: /main:MyNamespace.MyClass  
  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static float Main()   
      {  
         return 0.0; // CS1558 because the return type is a float.  
      }  
   }  
}  
```
