# <a name="compiler-error-cs1558"></a><span data-ttu-id="6a749-102">컴파일러 오류 CS1558</span><span class="sxs-lookup"><span data-stu-id="6a749-102">Compiler Error CS1558</span></span>
<span data-ttu-id="6a749-103">'class'에 적합한 정적 Main 메서드가 없습니다.</span><span class="sxs-lookup"><span data-stu-id="6a749-103">'class' does not have a suitable static Main method</span></span>  
  
 <span data-ttu-id="6a749-104">[주/](../../csharp/language-reference/compiler-options/main-compiler-option.md) 컴파일러 옵션이 지정에 대해 알아볼 수 있는 클래스는 **Main** 메서드.</span><span class="sxs-lookup"><span data-stu-id="6a749-104">The [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) compiler option specified a class in which to look for a **Main** method.</span></span> <span data-ttu-id="6a749-105">그러나는 [Main](../../csharp/programming-guide/main-and-command-args/index.md) 메서드가 올바르게 정의 되지 않았습니다.</span><span class="sxs-lookup"><span data-stu-id="6a749-105">However, the [Main](../../csharp/programming-guide/main-and-command-args/index.md) method was not defined correctly.</span></span>  
  
 <span data-ttu-id="6a749-106">다음 예제에서는 잘못된 반환 형식으로 인해 CS1558을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="6a749-106">The following example generates CS1558 because of invalid return type.</span></span>  
  
```  
// CS1558.cs  
// compile with: /main:MyNamespace.MyClass  
  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static float Main()   
      {  
         return 0.0; // CS1558 because the return type is a float.  
      }  
   }  
}  
```
