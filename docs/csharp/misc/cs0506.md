# <a name="compiler-error-cs0506"></a><span data-ttu-id="67301-102">컴파일러 오류 CS0506</span><span class="sxs-lookup"><span data-stu-id="67301-102">Compiler Error CS0506</span></span>
<span data-ttu-id="67301-103">'function1': 상속된 'function2' 멤버는 "virtual", "abstract" 또는 "override"로 표시되지 않았으므로 재정의할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="67301-103">'function1' : cannot override inherited member 'function2' because it is not marked "virtual", "abstract", or "override"</span></span>  
  
 <span data-ttu-id="67301-104">**virtual**, **abstract**또는 `override`로 명시적으로 표시되지 않은 메서드를 재정의했습니다.</span><span class="sxs-lookup"><span data-stu-id="67301-104">A method was overridden that was not explicitly marked as **virtual**, **abstract**, or `override`.</span></span>  
  
 <span data-ttu-id="67301-105">다음 샘플에서는 CS0506을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="67301-105">The following sample generates CS0506:</span></span>  
  
```  
// CS0506.cs  
namespace MyNameSpace  
{  
   abstract public class ClassX  
   {  
      public int i = 0;  
  
      public int f()  
      {  
         return 0;  
      }  
      // Try the following definition for f() instead:  
      // abstract public int f();  
   }  
  
   public class ClassY : ClassX  
   {  
      public override int f()   // CS0506  
      {  
         return 0;  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
