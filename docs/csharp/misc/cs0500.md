 <span data-ttu-id="f4bbb-104">[abstract](../../csharp/language-reference/keywords/abstract.md) 메서드는 구현을 포함할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="f4bbb-104">An [abstract](../../csharp/language-reference/keywords/abstract.md) method cannot contain its implementation.</span></span>

```csharp

   {  
      abstract public void f(){}   // CS0500

      {

      }  
   }  
}  
```
