# <a name="compiler-error-cs1100"></a><span data-ttu-id="7e035-102">컴파일러 오류 CS1100</span><span class="sxs-lookup"><span data-stu-id="7e035-102">Compiler Error CS1100</span></span>
<span data-ttu-id="7e035-103">'name' 메서드의 첫 번째 매개 변수가 아닌 매개 변수에 매개 변수 한정자 'this'가 있습니다.</span><span class="sxs-lookup"><span data-stu-id="7e035-103">Method 'name' has a parameter modifier 'this' which is not on the first parameter.</span></span>  
  
 <span data-ttu-id="7e035-104">`this` 한정자는 메서드의 첫 번째 매개 변수에서만 사용할 수 있습니다. 이는 컴파일러에 메서드가 확장 메서드라는 것을 나타냅니다.</span><span class="sxs-lookup"><span data-stu-id="7e035-104">The `this` modifier is allowed only on the first parameter of a method, which indicates to the compiler that the method is an extension method.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="7e035-105">이 오류를 해결하려면</span><span class="sxs-lookup"><span data-stu-id="7e035-105">To correct this error</span></span>  
  
1.  <span data-ttu-id="7e035-106">메서드의 첫 번째 매개 변수를 제외한 모든 매개 변수에서 `this` 한정자를 제거합니다.</span><span class="sxs-lookup"><span data-stu-id="7e035-106">Remove the `this` modifier from all except the first parameter of the method.</span></span>

 <span data-ttu-id="7e035-108">`this` 매개 변수가 두 번째 매개 변수를 수정하고 있기 때문에 다음 코드에서 CS1100을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="7e035-108">The following code generates CS1100 because a `this` parameter is modifying the second parameter:</span></span>
