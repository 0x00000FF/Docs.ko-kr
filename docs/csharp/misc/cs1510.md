## <a name="example"></a><span data-ttu-id="a9f60-106">예</span><span class="sxs-lookup"><span data-stu-id="a9f60-106">Example</span></span>  
 <span data-ttu-id="a9f60-107">다음 샘플에서는 CS1510을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="a9f60-107">The following sample generates CS1510:</span></span>  
  
```csharp  
// CS1510.cs  
public class C  
{  
   public static int j = 0;  
  
   public static void M(ref int j)  
   {  
      j++;  
   }  
  
   public static void Main ()  
   {  
      M (ref 2);   // CS1510, can't pass a number as a ref parameter  
      // try the following to resolve the error  
      // M (ref j);  
   }  
}  
```
