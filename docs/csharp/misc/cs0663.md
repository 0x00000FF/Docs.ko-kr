 <span data-ttu-id="2fe2a-104">해당 메일의 사용만 다른 메서드는 [에](../../csharp/language-reference/keywords/in-parameter-modifier.md), [ref](../../csharp/language-reference/keywords/ref.md) 및 [아웃](../../csharp/language-reference/keywords/out-parameter-modifier.md) 매개 변수에 허용 되지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="2fe2a-104">Methods that differ only on their use of [in](../../csharp/language-reference/keywords/in-parameter-modifier.md), [ref](../../csharp/language-reference/keywords/ref.md) and [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) on a parameter are not allowed.</span></span>  
  
 <span data-ttu-id="2fe2a-105">다음 샘플에서는 CS0663을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="2fe2a-105">The following sample generates CS0663:</span></span>  
  
```csharp

   public static void Main()  
   {  
   }

   {  
   }  
}  
```
