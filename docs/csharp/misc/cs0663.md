<span data-ttu-id="f8dbd-103">ref 및 out만 다른 오버로드된 메서드를 정의할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="f8dbd-103">Cannot define overloaded methods that differ only on ref and out.</span></span>  
  
 <span data-ttu-id="f8dbd-104">[In](../language-reference/keywords/in-parameter-modifier.md), [ref](../language-reference/keywords/ref.md) 및 [out](../language-reference/keywords/out-parameter-modifier.md) 매개 변수에만 다른 메서드는 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="f8dbd-104">Methods that differ only on their use of [in](../language-reference/keywords/in-parameter-modifier.md), [ref](../language-reference/keywords/ref.md) and [out](../language-reference/keywords/out-parameter-modifier.md) on a parameter are not allowed.</span></span>  
  
 <span data-ttu-id="f8dbd-105">다음 샘플에서는 CS0663을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="f8dbd-105">The following sample generates CS0663:</span></span>  
  
```csharp  
// CS0663.cs  
class TestClass  
{  
   public static void Main()  
   {  
   }  
  
   public void Test(ref int i)  
   {  
   }  
  
   public void Test(out int i)   // CS0663  
   {  
   }  
}  
```
