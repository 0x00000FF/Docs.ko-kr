# <a name="compiler-error-cs0663"></a><span data-ttu-id="51732-102">컴파일러 오류 CS0663</span><span class="sxs-lookup"><span data-stu-id="51732-102">Compiler Error CS0663</span></span>
<span data-ttu-id="51732-103">ref 및 out만 다른 오버로드된 메서드를 정의할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="51732-103">Cannot define overloaded methods that differ only on ref and out.</span></span>  
  
 <span data-ttu-id="51732-104">해당 메일의 사용만 다른 메서드는 [에](../../csharp/language-reference/keywords/in-parameter-modifier.md), [ref](../../csharp/language-reference/keywords/ref.md) 및 [아웃](../../csharp/language-reference/keywords/out-parameter-modifier.md) 매개 변수에 허용 되지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="51732-104">Methods that differ only on their use of [in](../../csharp/language-reference/keywords/in-parameter-modifier.md), [ref](../../csharp/language-reference/keywords/ref.md) and [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) on a parameter are not allowed.</span></span>  
  
 <span data-ttu-id="51732-105">다음 샘플에서는 CS0663을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="51732-105">The following sample generates CS0663:</span></span>  
  
```csharp  
// CS0663.cs  
class TestClass  
{  
   public static void Main()  
   {  
   }  
  
   public void Test(ref int i)  
   {  
   }  
  
   public void Test(out int i)   // CS0663  
   {  
   }  
}  
```
