<span data-ttu-id="efaea-103">무명 메서드, 람다 식 또는 쿼리 식 안에서는 ref 또는 out 매개 변수 'parameter'를 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="efaea-103">Cannot use ref or out parameter 'parameter' inside an anonymous method, lambda expression, or query expression</span></span>  
  
 <span data-ttu-id="efaea-104">이 오류는 무명 메서드 블록 내에서 ref 또는 out 매개 변수를 사용한 경우 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="efaea-104">This error occurs if you use a ref or out parameter inside an anonymous method block.</span></span> <span data-ttu-id="efaea-105">이 오류를 방지하려면 지역 변수 또는 일부 다른 생성자를 사용합니다.</span><span class="sxs-lookup"><span data-stu-id="efaea-105">To avoid this error, use a local variable or some other construct.</span></span>  
  
 <span data-ttu-id="efaea-106">다음 샘플에서는 CS1628을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="efaea-106">The following sample generates CS1628:</span></span>  
  
```  
// CS1628.cs

{  
  public static void F(ref int i)  
  {  
      MyDelegate d = delegate { return i; };  // CS1628  
      // Try this instead:  
      // int tmp = i;  
      // MyDelegate d = delegate { return tmp; };  
  }  
  
  public static void Main()  
  {  
  
  }  
}  
```
