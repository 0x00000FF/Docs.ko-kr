 <span data-ttu-id="7161d-105">이 오류는 정적 메서드가 인터페이스 멤버 구현으로 암시적 또는 명시적으로 선언된 경우에 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="7161d-105">This error is generated when a static method is either implicitly or explicitly declared as an implementation of an interface member.</span></span>

## <a name="example"></a><span data-ttu-id="7161d-110">예제</span><span class="sxs-lookup"><span data-stu-id="7161d-110">Example</span></span>  
 <span data-ttu-id="7161d-111">`Program.testMethod` 가 static으로 선언되었기 때문에 다음 코드에서는 CS0736을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="7161d-111">The following code generates CS0736 because `Program.testMethod` is declared as static:</span></span>  
  
```csharp  
// cs0736.cs  
namespace CS0736  
{     
  
    interface ITest  
    {  
        int testMethod(int x);  
    }  
  
    class Program : ITest // CS0736  
    {  
        public static int testMethod(int x) { return 0; }  
        // Try the following line instead.  
        // public int testMethod(int x) { return 0; }  
        public static void Main() { }  
    }      
}  
```
