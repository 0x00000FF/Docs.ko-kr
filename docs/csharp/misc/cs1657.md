# <a name="compiler-error-cs1657"></a><span data-ttu-id="203f4-102">컴파일러 오류 CS1657</span><span class="sxs-lookup"><span data-stu-id="203f4-102">Compiler Error CS1657</span></span>

 <span data-ttu-id="203f4-104">변수가로 전달 되는 경우이 오류가 발생 한 [ref](../../csharp/language-reference/keywords/ref.md) 또는 [아웃](../../csharp/language-reference/keywords/out-parameter-modifier.md) 는 변수가 읽기 전용인은 컨텍스트에 대 한 인수입니다.</span><span class="sxs-lookup"><span data-stu-id="203f4-104">This error occurs when a variable is passed as a [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) argument in a context in which that variable is readonly.</span></span> <span data-ttu-id="203f4-105">읽기 전용 컨텍스트에는 [foreach](../../csharp/language-reference/keywords/foreach-in.md) 반복 변수, [using](../../csharp/language-reference/keywords/using-statement.md) 변수 및 `fixed` 변수가 포함됩니다.</span><span class="sxs-lookup"><span data-stu-id="203f4-105">Readonly contexts include [foreach](../../csharp/language-reference/keywords/foreach-in.md) iteration variables, [using](../../csharp/language-reference/keywords/using-statement.md) variables, and `fixed` variables.</span></span> <span data-ttu-id="203f4-106">이 오류를 해결하려면 `foreach`블록, `using` 문 및 `fixed` 문에서 `ref` , `out` 또는 `using` 변수를 `foreach` 또는 `fixed` 매개 변수로 사용하는 함수를 호출하지 마세요.</span><span class="sxs-lookup"><span data-stu-id="203f4-106">To resolve this error, do not call functions that take the `foreach`, `using` or `fixed` variable as a `ref` or `out` parameter in `using` blocks, `foreach` statements, and `fixed` statements.</span></span>  
  
## <a name="example"></a><span data-ttu-id="203f4-107">예</span><span class="sxs-lookup"><span data-stu-id="203f4-107">Example</span></span>  
 <span data-ttu-id="203f4-108">다음 샘플에서는 CS1657을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="203f4-108">The following sample generates CS1657:</span></span>  
  
```csharp  
// CS1657.cs  
using System;  
class C : IDisposable  
{  
    public int i;  
    public void Dispose() {}  
}  
  
class CMain  
{  
    static void f(ref C c)  
    {  
    }  
    static void Main()  
    {  
        using (C c = new C())  
        {  
            f(ref c);  // CS1657  
        }  
    }  
}  
```
