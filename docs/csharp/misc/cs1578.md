# <a name="compiler-error-cs1578"></a><span data-ttu-id="a7e6b-102">컴파일러 오류 CS1578</span><span class="sxs-lookup"><span data-stu-id="a7e6b-102">Compiler Error CS1578</span></span>
<span data-ttu-id="a7e6b-103">파일 이름, 한 줄로 된 주석 또는 줄 끝(EOL)이 필요합니다.</span><span class="sxs-lookup"><span data-stu-id="a7e6b-103">Filename, single-line comment or end-of-line expected</span></span>  
  
 <span data-ttu-id="a7e6b-104">[#line](../../csharp/language-reference/preprocessor-directives/preprocessor-line.md) 지시문 뒤에는 큰따옴표로 묶은 파일 이름 또는 한 줄로 된 주석만 허용됩니다.</span><span class="sxs-lookup"><span data-stu-id="a7e6b-104">After a [#line](../../csharp/language-reference/preprocessor-directives/preprocessor-line.md) directive, only a file name (in double quotation marks) or a single-line comment is allowed.</span></span>  
  
 <span data-ttu-id="a7e6b-105">다음 샘플에서는 CS1578을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="a7e6b-105">The following sample generates CS1578:</span></span>  
  
```csharp  
// CS1578.cs  
class MyClass  
{  
   static void Main()  
   {  
      #line 101 abc.cs   // CS1578  
      // try the following line instead  
      //#line 101 "abc.cs"  
      intt i;          // error will be reported on line 101  
   }  
}  
```
