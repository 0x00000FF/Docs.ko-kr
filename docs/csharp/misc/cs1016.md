## <a name="example"></a><span data-ttu-id="86cb4-105">예</span><span class="sxs-lookup"><span data-stu-id="86cb4-105">Example</span></span>  
 <span data-ttu-id="86cb4-106">다음 샘플에서는 CS1016을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="86cb4-106">The following sample generates CS1016:</span></span>  
  
```csharp  
// CS1016.cs  
using System;  
  
[AttributeUsage(AttributeTargets.Class)]  
public class HelpAttribute : Attribute  
{  
    public HelpAttribute(string url)   // url is a positional parameter  
    {  
        m_url = url;  
    }  
  
    public string Topic = null;      // Topic is a named parameter  
    private string m_url = null;  
}  
  
[HelpAttribute(Topic="Samples", "http://intranet/inhouse")]   // CS1016  
// try the following line instead  
//[HelpAttribute("http://intranet/inhouse", Topic="Samples")]  
public class MainClass  
{  
    public static void Main ()  
    {  
    }  
}  
```
