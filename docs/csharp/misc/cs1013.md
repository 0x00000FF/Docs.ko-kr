# <a name="compiler-error-cs1013"></a>컴파일러 오류 CS1013
숫자가 잘못되었습니다.  
  
 컴파일러가 잘못된 형식의 숫자를 검색했습니다. 정수 형식에 대한 자세한 내용은 [Integral Types Table](../../csharp/language-reference/keywords/integral-types-table.md)을 참조하세요.  
  
## <a name="example"></a>예제  
 다음 샘플에서는 CS1013을 생성합니다.  
  
```csharp  
// CS1013.cs  
class Sample  
{  
    static void Main()  
    {  
        int i = 0x;   // CS1013  
    }  
}  
```
