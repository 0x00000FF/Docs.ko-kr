# <a name="compiler-error-cs0619"></a><span data-ttu-id="a21b1-102">컴파일러 오류 CS0619</span><span class="sxs-lookup"><span data-stu-id="a21b1-102">Compiler Error CS0619</span></span>
<span data-ttu-id="a21b1-103">'member'는 사용되지 않습니다. 'text'</span><span class="sxs-lookup"><span data-stu-id="a21b1-103">'member' is obsolete: 'text'</span></span>  
  
 <span data-ttu-id="a21b1-104">클래스 멤버가 [Obsolete] ((.)로 표시 되었습니다. /programming-guide/concepts/attributes/common-attributes.md # Obsolete) 특성을 사용 합니다 .이 특성을 사용 하면 클래스 멤버가 참조 될 때 오류가 발행 됩니다.</span><span class="sxs-lookup"><span data-stu-id="a21b1-104">A class member was marked with the [Obsolete]((../programming-guide/concepts/attributes/common-attributes.md#Obsolete) attribute, such that an error will be issued when the class member is referenced.</span></span>  
  
 <span data-ttu-id="a21b1-105">다음 샘플에서는 CS0619를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="a21b1-105">The following sample generates CS0619:</span></span>  
  
```csharp  
// CS0619.cs  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", true)]   // generates an error on use  
   public static void m()  
   {  
   }  
  
   // this is the method you should be using  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m();   // CS0619  
   }  
}  
```
