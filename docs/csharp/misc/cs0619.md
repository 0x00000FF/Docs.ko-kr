# <a name="compiler-error-cs0619"></a><span data-ttu-id="b9a50-102">컴파일러 오류 CS0619</span><span class="sxs-lookup"><span data-stu-id="b9a50-102">Compiler Error CS0619</span></span>
<span data-ttu-id="b9a50-103">'member'는 사용되지 않습니다. 'text'</span><span class="sxs-lookup"><span data-stu-id="b9a50-103">'member' is obsolete: 'text'</span></span>  
  
 <span data-ttu-id="b9a50-104">클래스 멤버를 [사용 되지 않음]으로 표시 되어 있으므로 ((... /.. / csharp/programming-guide/concepts/attributes/common-attributes.md#Obsolete) 특성을 클래스 멤버를 참조할 때 오류가 발생 합니다.</span><span class="sxs-lookup"><span data-stu-id="b9a50-104">A class member was marked with the [Obsolete]((../../csharp/programming-guide/concepts/attributes/common-attributes.md#Obsolete) attribute, such that an error will be issued when the class member is referenced.</span></span>  
  
 <span data-ttu-id="b9a50-105">다음 샘플에서는 CS0619를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="b9a50-105">The following sample generates CS0619:</span></span>  
  
```csharp  
// CS0619.cs  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", true)]   // generates an error on use  
   public static void m()  
   {  
   }  
  
   // this is the method you should be using  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m();   // CS0619  
   }  
}  
```
