# <a name="compiler-error-cs0216"></a><span data-ttu-id="6a3f3-102">컴파일러 오류 CS0216</span><span class="sxs-lookup"><span data-stu-id="6a3f3-102">Compiler Error CS0216</span></span>
<span data-ttu-id="6a3f3-103">'operator' 연산자를 사용하려면 짝이 되는 'missing_operator' 연산자도 정의해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="6a3f3-103">The operator 'operator' requires a matching operator 'missing_operator' to also be defined</span></span>  
  
 <span data-ttu-id="6a3f3-104">사용자 정의 [true](../language-reference/keywords/true-false-operators.md) 연산자를 사용하려면 사용자 정의 [false](../language-reference/keywords/true-false-operators.md) 연산자가 필요하며 그 반대도 마찬가지입니다.</span><span class="sxs-lookup"><span data-stu-id="6a3f3-104">A user-defined [true](../language-reference/keywords/true-false-operators.md) operator requires a user-defined [false](../language-reference/keywords/true-false-operators.md) operator, and vice versa.</span></span>
  
 <span data-ttu-id="6a3f3-105">다음 샘플에서는 CS0216을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="6a3f3-105">The following sample generates CS0216:</span></span>  
  
```csharp  
// CS0216.cs  
class MyClass  
{  
   public static bool operator true (MyClass MyInt)   // CS0216  
   {  
      return true;  
   }  
  
   // to resolve, uncomment the following operator definition  
   /*  
   public static bool operator false (MyClass MyInt)  
   {  
      return true;  
   }  
   */  
  
   public static void Main()  
   {  
   }  
}  
```
