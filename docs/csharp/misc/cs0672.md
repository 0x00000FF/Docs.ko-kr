# <a name="compiler-warning-level-1-cs0672"></a><span data-ttu-id="a2bda-102">컴파일러 경고(수준 1) CS0672</span><span class="sxs-lookup"><span data-stu-id="a2bda-102">Compiler Warning (level 1) CS0672</span></span>
<span data-ttu-id="a2bda-103">'member1' 멤버가 사용되지 않는 'member2' 멤버를 재정의합니다.</span><span class="sxs-lookup"><span data-stu-id="a2bda-103">Member 'member1' overrides obsolete member 'member2.</span></span> <span data-ttu-id="a2bda-104">'member1'에 사용되지 않음 특성을 추가합니다.</span><span class="sxs-lookup"><span data-stu-id="a2bda-104">Add the Obsolete attribute to 'member1'</span></span>  
  
 <span data-ttu-id="a2bda-105">컴파일러가 `override` 로 표시된 메서드로의 `obsolete`를 발견했습니다.</span><span class="sxs-lookup"><span data-stu-id="a2bda-105">The compiler found an `override` to a method marked as `obsolete`.</span></span> <span data-ttu-id="a2bda-106">그러나 재정의 메서드 자체는 사용되지 않음으로 표시되지 않았습니다.</span><span class="sxs-lookup"><span data-stu-id="a2bda-106">However, the overriding method was not itself marked as obsolete.</span></span> <span data-ttu-id="a2bda-107">재정의 메서드를 호출하면 [CS0612](./cs0612.md)가 생성됩니다.</span><span class="sxs-lookup"><span data-stu-id="a2bda-107">The overriding method will still generate [CS0612](./cs0612.md), if called.</span></span>  
  
 <span data-ttu-id="a2bda-108">메서드 선언을 검토하고 메서드(및 모든 재정의)가 `obsolete`로 표시되어야 하는지 여부를 명시적으로 나타냅니다.</span><span class="sxs-lookup"><span data-stu-id="a2bda-108">Review your method declarations and explicitly indicate whether a method (and all of its overrides) should be marked `obsolete`.</span></span>  
  
 <span data-ttu-id="a2bda-109">다음 샘플에서는 CS0672를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="a2bda-109">The following sample generates CS0672:</span></span>  
  
```csharp  
// CS0672.cs  
// compile with: /W:1  
class MyClass  
{  
   [System.Obsolete]  
   public virtual void ObsoleteMethod()  
   {  
   }  
}  
  
class MyClass2 : MyClass  
{  
   public override void ObsoleteMethod()   // CS0672  
   {  
   }  
}  
  
class MainClass  
{  
   static public void Main()  
   {  
   }  
}  
```
