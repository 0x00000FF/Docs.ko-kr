# <a name="compiler-error-cs0193"></a><span data-ttu-id="0a118-102">컴파일러 오류 CS0193</span><span class="sxs-lookup"><span data-stu-id="0a118-102">Compiler Error CS0193</span></span>

<span data-ttu-id="0a118-103">\* 또는-> 연산자는 포인터에 적용 되어야 합니다</span><span class="sxs-lookup"><span data-stu-id="0a118-103">The \* or -> operator must be applied to a pointer</span></span>  
  
<span data-ttu-id="0a118-104">\* 또는-> 연산자가 포인터가 아닌 형식으로 사용 되었습니다.</span><span class="sxs-lookup"><span data-stu-id="0a118-104">The \* or -> operator was used with a nonpointer type.</span></span> <span data-ttu-id="0a118-105">자세한 내용은 [포인터 형식](../../csharp/programming-guide/unsafe-code-pointers/pointer-types.md)을 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="0a118-105">For more information, see [Pointer types](../../csharp/programming-guide/unsafe-code-pointers/pointer-types.md).</span></span>  
  
 <span data-ttu-id="0a118-106">다음 샘플에서는 CS0193을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="0a118-106">The following sample generates CS0193:</span></span>  
  
```csharp  
// CS0193.cs  
using System;  
  
public struct Age  
{  
   public int AgeYears;  
   public int AgeMonths;  
   public int AgeDays;  
}  
  
public class MyClass  
{  
   public static void SetAge(ref Age anAge, int years, int months, int days)  
   {  
      anAge->Months = 3;   // CS0193, anAge is not a pointer  
      // try the following line instead  
      // anAge.AgeMonths = 3;  
   }  
  
   public static void Main()  
   {  
      Age MyAge = new Age();  
      Console.WriteLine(MyAge.AgeMonths);  
      SetAge(ref MyAge, 22, 4, 15);  
      Console.WriteLine(MyAge.AgeMonths);  
   }  
}  
```
