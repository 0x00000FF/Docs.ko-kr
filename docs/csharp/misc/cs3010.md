## <a name="example"></a><span data-ttu-id="42b7c-107">예제</span><span class="sxs-lookup"><span data-stu-id="42b7c-107">Example</span></span>  
 <span data-ttu-id="42b7c-108">다음 예제에서는 CS3010을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="42b7c-108">The following example generates CS3010:</span></span>  
  
```csharp  
// CS3010.cs  
  
using System;  
  
[assembly:CLSCompliant(true)]  
public interface I  
{  
    [CLSCompliant(false)]  
    int M();   // CS3010  
}  
  
public class C : I  
{  
    public int M()  
    {  
        return 1;  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
