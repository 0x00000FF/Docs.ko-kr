# 컴파일러 오류 CS1022
형식이나 네임스페이스 정의 또는 파일 끝\(EOF\)이 필요합니다.  
  
 소스 코드 파일에 일치하는 중괄호 집합이 없습니다.  
  
 다음 샘플에서는 CS1022를 생성합니다.  
  
```  
// CS1022.cs namespace x { } }   // CS1022  
```
