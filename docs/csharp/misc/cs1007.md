## <a name="example"></a><span data-ttu-id="37921-106">예제</span><span class="sxs-lookup"><span data-stu-id="37921-106">Example</span></span>

```csharp  
    {  
        get  
        {  
            return 0;  
        }  
        get   // CS1007, this is the second get method

            return 0;  
        }  
    }  
  
    public static void Main() {}  
}  
```
