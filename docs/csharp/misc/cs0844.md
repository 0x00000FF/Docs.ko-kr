## <a name="example"></a><span data-ttu-id="65240-111">예제</span><span class="sxs-lookup"><span data-stu-id="65240-111">Example</span></span>

    {  
        int num;  
        public void TestMethod()  
        {  
            num = 5; // CS0844  
            int num = 6;

        public static int Main()  
        {

        }

## <a name="correct-the-error-by-using-thisnum-to-refer-to-the-class-field"></a><span data-ttu-id="65240-113">사용 하 여 오류를 수정 `this.num` 클래스 필드를 참조할 때</span><span class="sxs-lookup"><span data-stu-id="65240-113">Correct the error by using `this.num` to refer to the class field</span></span>

```csharp  
class Test  
    {  
        int num;  
        public void TestMethod()  
        {  
            this.num = 5; // Error fixed.
            int num = 6;
        }  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```
