# 컴파일러 오류 CS1713
Typename1 형식의 메타데이터 이름을 만드는 동안 오류가 발생했습니다. 'Reason'  
  
 이 오류는 종종 내부 컴파일러 오류를 발생시킵니다. 이름 길이를 줄이는 등 코드를 약간 변경한 다음 다시 컴파일하면 이 오류가 해결될 수 있습니다.
