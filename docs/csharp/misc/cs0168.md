# <a name="compiler-warning-level-3-cs0168"></a><span data-ttu-id="d7e37-102">컴파일러 경고(수준 3) CS0168</span><span class="sxs-lookup"><span data-stu-id="d7e37-102">Compiler Warning (level 3) CS0168</span></span>
<span data-ttu-id="d7e37-103">'Var' 변수가 선언 되었지만 사용 되지 않았습니다.</span><span class="sxs-lookup"><span data-stu-id="d7e37-103">The variable 'var' is declared but never used</span></span>

<span data-ttu-id="d7e37-104">변수를 선언 하지만 사용 하지 않는 경우 컴파일러가 수준 3 경고 합니다.</span><span class="sxs-lookup"><span data-stu-id="d7e37-104">The compiler issues a level-three warning when you declare a variable, but do not use it.</span></span>
  
<span data-ttu-id="d7e37-105">다음 샘플에서는 CS0168 경고를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="d7e37-105">The following sample generates a CS0168 warning:</span></span>  
  
```csharp
// CS0168.cs  
// compile with: /W:3  
public class clx
{
    public int i;
}

public class clz
{
    public static void Main() {
        clx a; // CS0168, the variable 'a' is declared but never used
        // try the following line instead
        // clx a = new clx();  // this does not generate a warning because the clx constructor must execute.
    }
}
```
