# <a name="compiler-error-cs0708"></a><span data-ttu-id="e8a92-102">컴파일러 오류 CS0708</span><span class="sxs-lookup"><span data-stu-id="e8a92-102">Compiler Error CS0708</span></span>
<span data-ttu-id="e8a92-103">'field': 정적 클래스에 인스턴스 멤버를 선언할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="e8a92-103">'field': cannot declare instance members in a static class</span></span>  
  
 <span data-ttu-id="e8a92-104">이 오류는 정적으로 선언된 클래스에서 비정적 멤버를 선언하는 경우에 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="e8a92-104">This error occurs if you declare a non-static member in a class that is declared static.</span></span> <span data-ttu-id="e8a92-105">정적 클래스의 인스턴스를 생성할 수 없으므로 인스턴스 변수는 의미가 없습니다.</span><span class="sxs-lookup"><span data-stu-id="e8a92-105">It is not possible to create instances of static classes, so instance variables would not be meaningful.</span></span> <span data-ttu-id="e8a92-106">**static** 키워드는 정적 클래스의 모든 멤버에 적용해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="e8a92-106">The **static** keyword should be applied to all members of static classes.</span></span>  
  
 <span data-ttu-id="e8a92-107">다음 샘플에서는 CS0708을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="e8a92-107">The following sample generates CS0708:</span></span>  
  
```csharp  
// CS0708.cs

public static class C  
{  
   int i;  // CS0708  
   static int j;  // OK  
}  
```
