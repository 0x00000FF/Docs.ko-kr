## <a name="examples"></a><span data-ttu-id="4694f-107">예</span><span class="sxs-lookup"><span data-stu-id="4694f-107">Examples</span></span>
<span data-ttu-id="4694f-108">다음 샘플에서는 CS0200을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="4694f-108">The following sample generates CS0200:</span></span>

```csharp
// CS0200.cs
public class Example
{
    private int _mi;
    int I
    {
        get
        {
            return _mi;
        }
        // uncomment the set accessor and declaration for _mi
        /*
        set
        {
            _mi = value;
        }
        */
    }

    public static void Main()
    {  
        Example example = new Example();
        example.I = 9;   // CS0200
    }
}  
```

<span data-ttu-id="4694f-109">다음 샘플에서는 [자동 구현 속성,](../programming-guide/classes-and-structs/auto-implemented-properties.md) [개체 초기화자를](../programming-guide/classes-and-structs/object-and-collection-initializers.md)사용하고 CS0200을 계속 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="4694f-109">The following sample uses [auto-implemented properties](../programming-guide/classes-and-structs/auto-implemented-properties.md), [object initializers](../programming-guide/classes-and-structs/object-and-collection-initializers.md), and still generates CS0200:</span></span>

```csharp
// CS0200.cs
public class Example
{
    int I
    {
        get;
        // uncomment the set accessor and declaration
        //set;
    }

    public static void Main()
    {  
        var example = new Example
        {
            I = 9   // CS0200
        };
    }
}
```

<span data-ttu-id="4694f-110">만 읽는 속성 또는 인덱서 '속성'에 할당, 세트 접근자 추가 하거나 개체의 생성자의 속성에 할당 하 여 수행할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="4694f-110">Assignment to a property or indexer 'property' that is read only, can be achieved through adding a set accessor or by assigning to the property in the object's constructor.</span></span>

```csharp
public class Example
{
    int I { get; }

    public Example()
    {
        I = -7;
    }
}
```
