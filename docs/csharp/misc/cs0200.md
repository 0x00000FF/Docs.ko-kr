# <a name="compiler-error-cs0200"></a>컴파일러 오류 CS0200
속성 또는 인덱서 '속성'에 할당할 수 없습니다-- 그것은 단지 읽습니다

[속성에](../programming-guide/classes-and-structs/using-properties.md)값을 할당하려고 했지만 속성에 set 접근자가 없거나 할당이 생성자 외부에 있었습니다. set 접근자를 추가하여 오류를 해결합니다. 자세한 내용은 [읽기 쓰기 속성을 선언하고 사용하는 방법을](../programming-guide/classes-and-structs/how-to-declare-and-use-read-write-properties.md)참조하십시오.

## <a name="examples"></a>예
다음 샘플에서는 CS0200을 생성합니다.

```csharp
// CS0200.cs
public class Example
{
    private int _mi;
    int I
    {
        get
        {
            return _mi;
        }
        // uncomment the set accessor and declaration for _mi
        /*
        set
        {
            _mi = value;
        }
        */
    }

    public static void Main()
    {  
        Example example = new Example();
        example.I = 9;   // CS0200
    }
}  
```

다음 샘플에서는 [자동 구현 속성,](../programming-guide/classes-and-structs/auto-implemented-properties.md) [개체 초기화자를](../programming-guide/classes-and-structs/object-and-collection-initializers.md)사용하고 CS0200을 계속 생성합니다.

```csharp
// CS0200.cs
public class Example
{
    int I
    {
        get;
        // uncomment the set accessor and declaration
        //set;
    }

    public static void Main()
    {  
        var example = new Example
        {
            I = 9   // CS0200
        };
    }
}
```

만 읽는 속성 또는 인덱서 '속성'에 할당, 세트 접근자 추가 하거나 개체의 생성자의 속성에 할당 하 여 수행할 수 있습니다.

```csharp
public class Example
{
    int I { get; }

    public Example()
    {
        I = -7;
    }
}
```
