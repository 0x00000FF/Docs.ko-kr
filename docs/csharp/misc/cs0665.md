<span data-ttu-id="dfb80-103">조건식에 할당을 사용하면 항상 상수가 됩니다. = 대신 ==을 사용하세요.</span><span class="sxs-lookup"><span data-stu-id="dfb80-103">Assignment in conditional expression is always constant; did you mean to use == instead of = ?</span></span>  
  
 <span data-ttu-id="dfb80-104">조건식에서 [== 연산자](../../csharp/language-reference/operators/assignment-operator.md) 가 아닌 [= 연산자](../../csharp/language-reference/operators/equality-comparison-operator.md)를 사용했습니다.</span><span class="sxs-lookup"><span data-stu-id="dfb80-104">A conditional expression used the [= operator](../../csharp/language-reference/operators/assignment-operator.md) and not the [== operator](../../csharp/language-reference/operators/equality-comparison-operator.md).</span></span>  
  
 <span data-ttu-id="dfb80-105">다음 샘플에서는 CS0665를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="dfb80-105">The following sample generates CS0665:</span></span>

```csharp  
{  
   public static void Main()  
   {  
      bool i = false;  
  
      if (i = true)   // CS0665  
      // try the following line instead  
      // if (i == true)  
      {  
      }  
  
      System.Console.WriteLine(i);  
   }  
}  
```
