# <a name="compiler-error-cs1650"></a><span data-ttu-id="b06d6-102">컴파일러 오류 CS1650</span><span class="sxs-lookup"><span data-stu-id="b06d6-102">Compiler Error CS1650</span></span>
<span data-ttu-id="b06d6-103">정적 읽기 전용 필드 'identifier'의 필드에는 할당할 수 없습니다. 단 정적 생성자 또는 변수 이니셜라이저에서는 예외입니다.</span><span class="sxs-lookup"><span data-stu-id="b06d6-103">Fields of static readonly field 'identifier' cannot be assigned to (except in a static constructor or a variable initializer)</span></span>  
  
 <span data-ttu-id="b06d6-104">이 오류는 수정을 할 수 없는 읽기 전용 필드 및 정적 필드의 멤버를 수정하려고 시도할 때 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="b06d6-104">This error occurs when you attempt to modify a member of a field which is readonly and static where it is not allowed to be modified.</span></span> <span data-ttu-id="b06d6-105">이 오류를 해결하려면 읽기 전용 필드에 대한 할당을 생성자 또는 변수 이니셜라이저로 제한하거나 필드의 선언에서 `readonly` 키워드를 제거합니다.</span><span class="sxs-lookup"><span data-stu-id="b06d6-105">To resolve this error, limit assignments to readonly fields to the constructor or variable initializer, or remove the `readonly` keyword from the declaration of the field.</span></span>  
  
```csharp  
// CS1650.cs  
public struct Inner  
{  
    public int i;  
}  
  
class Outer  
{  
    public static readonly Inner inner = new Inner();  
}  
  
class D  
{  
    static void Main()  
    {  
        Outer.inner.i = 1;  // CS1650  
    }  
}  
```
