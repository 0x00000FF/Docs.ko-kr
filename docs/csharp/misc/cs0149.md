# <a name="compiler-error-cs0149"></a><span data-ttu-id="a5fb0-102">컴파일러 오류 CS0149</span><span class="sxs-lookup"><span data-stu-id="a5fb0-102">Compiler Error CS0149</span></span>
<span data-ttu-id="a5fb0-103">메서드 이름이 필요합니다.</span><span class="sxs-lookup"><span data-stu-id="a5fb0-103">Method name expected</span></span>  
  
 <span data-ttu-id="a5fb0-104">[대리자](../language-reference/builtin-types/reference-types.md)를 만드는 경우 메서드를 지정합니다.</span><span class="sxs-lookup"><span data-stu-id="a5fb0-104">When creating a [delegate](../language-reference/builtin-types/reference-types.md), specify a method.</span></span> <span data-ttu-id="a5fb0-105">자세한 내용은 [대리자](../programming-guide/delegates/index.md)를 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="a5fb0-105">For more information, see [Delegates](../programming-guide/delegates/index.md).</span></span>  
  
 <span data-ttu-id="a5fb0-106">다음 샘플에서는 CS0149를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="a5fb0-106">The following sample generates CS0149:</span></span>  
  
```csharp  
// CS0149.cs  
using System;  
  
delegate string MyDelegate(int i);  
  
class MyClass  
{  
   // class member-field of the declared delegate type  
   static MyDelegate dt;
  
   public static void Main()  
   {  
      dt = new MyDelegate(17.45);   // CS0149  
      // try the following line instead  
      // dt = new MyDelegate(Func2);  
      F(dt);  
   }  
  
   public static string Func2(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void F(MyDelegate myFunc)  
   {  
      myFunc(8);  
   }  
}  
```
