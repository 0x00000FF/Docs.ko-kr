# <a name="compiler-error-cs0041"></a>컴파일러 오류 CS0041
'type'의 정규화된 이름이 너무 길어서 디버그 정보에 사용할 수 없습니다. '/debug' 옵션 없이 컴파일합니다.  
  
 사용 하는 경우이 오류가 발생할 수는 [디버그](../../csharp/language-reference/compiler-options/debug-compiler-option.md) 컴파일러 옵션입니다. 이 오류가 발생하면 bin 디렉터리에서 PDB 파일을 삭제하고 다시 컴파일합니다. 복구 하거나 다시 설치 해야 할 수도이 오류가 계속 발생 하는 경우 [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)]합니다.
