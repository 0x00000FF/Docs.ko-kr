# 컴파일러 오류 CS0076
'value\_\_' 열거자 이름은 예약된 것이므로 사용할 수 없습니다.  
  
 [열거형](../../csharp/language-reference/keywords/enum.md) 항목에 **value\_\_**라는 식별자가 있어서는 안 됩니다.
