# <a name="compiler-error-cs1615"></a><span data-ttu-id="14b5b-102">컴파일러 오류 CS1615</span><span class="sxs-lookup"><span data-stu-id="14b5b-102">Compiler Error CS1615</span></span>
<span data-ttu-id="14b5b-103">'number' 인수는 'keyword' 키워드와 함께 전달할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="14b5b-103">Argument 'number' should not be passed with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="14b5b-104">함수에서 해당 인수에 대해 `ref` 또는 **out** 매개 변수를 사용하지 않는데 `ref` 또는 **out** 키워드 중 하나가 사용되었습니다.</span><span class="sxs-lookup"><span data-stu-id="14b5b-104">One of the keywords `ref` or **out** was used when the function did not take a `ref` or **out** parameter for that argument.</span></span> <span data-ttu-id="14b5b-105">이 오류를 해결하려면 잘못된 키워드를 제거하고 함수 선언에 맞는 적절한 키워드(있는 경우)를 사용합니다.</span><span class="sxs-lookup"><span data-stu-id="14b5b-105">To resolve this error, remove the incorrect keyword and use the appropriate keyword that matches the function declaration, if any.</span></span>  
  
 <span data-ttu-id="14b5b-106">다음 샘플에서는 CS1615를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="14b5b-106">The following sample generates CS1615:</span></span>  
  
```csharp  
// CS1615.cs  
class C  
{  
   public void f(int i) {}  
   public static void Main()  
   {  
      int i = 1;  
      f(ref i);  // CS1615  
   }  
}  
```
