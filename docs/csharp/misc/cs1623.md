# <a name="compiler-error-cs1623"></a><span data-ttu-id="a5a67-102">컴파일러 오류 CS1623</span><span class="sxs-lookup"><span data-stu-id="a5a67-102">Compiler Error CS1623</span></span>
<span data-ttu-id="a5a67-103">반복기는 ref 또는 out 매개 변수 여야 합니다.</span><span class="sxs-lookup"><span data-stu-id="a5a67-103">Iterators cannot have in ref or out parameters</span></span>  
  
 <span data-ttu-id="a5a67-104">반복기 메서드를 사용 하는 경우이 오류가 발생 한 `in`, `ref`, 또는 `out` 매개 변수입니다.</span><span class="sxs-lookup"><span data-stu-id="a5a67-104">This error occurs if an iterator method takes an `in`, `ref`, or `out` parameter.</span></span> <span data-ttu-id="a5a67-105">이 오류를 방지 하려면 제거의 `in`, `ref`, 또는 `out` 메서드 서명에서 키워드입니다.</span><span class="sxs-lookup"><span data-stu-id="a5a67-105">To avoid this error, remove the `in`, `ref`, or `out` keyword from the method signature.</span></span>  
  
## <a name="example"></a><span data-ttu-id="a5a67-106">예제</span><span class="sxs-lookup"><span data-stu-id="a5a67-106">Example</span></span>  
 <span data-ttu-id="a5a67-107">다음 샘플에서는 CS1623을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="a5a67-107">The following sample generates CS1623:</span></span>  
  
```csharp  
// CS1623.cs  
using System.Collections;

class C : IEnumerable
{
    public IEnumerator GetEnumerator()
    {
        yield return 0;
    }

    // To resolve the error, remove in  
    public IEnumerator GetEnumerator(in short i)  // CS1623  
    {
        yield return i;
    }
    // To resolve the error, remove ref  
    public IEnumerator GetEnumerator(ref int i)  // CS1623  
    {
        yield return i;
    }

    // To resolve the error, remove out  
    public IEnumerator GetEnumerator(out float f)  // CS1623  
    {
        f = 0.0F;
        yield return f;
    }
}
```
