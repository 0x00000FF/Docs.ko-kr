 <span data-ttu-id="8b491-104">[stackalloc](../../csharp/language-reference/keywords/stackalloc.md)를 사용하여 요청된 할당의 크기를 대괄호 안에 지정해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="8b491-104">The size of the requested allocation, with [stackalloc](../../csharp/language-reference/keywords/stackalloc.md), must be specified in square brackets.</span></span>  
  
 <span data-ttu-id="8b491-105">다음 샘플에서는 CS1575를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="8b491-105">The following sample generates CS1575:</span></span>  
  
```csharp  
// CS1575.cs  
// compile with: /unsafe

{  
   unsafe public static void Main()  
   {  
      int *p = stackalloc int (30);   // CS1575  
      // try the following line instead  
      // int *p = stackalloc int [30];  
   }  
}  
```
