 <span data-ttu-id="dbccd-104">[stackalloc](../../csharp/language-reference/keywords/stackalloc.md)를 사용하여 요청된 할당의 크기를 대괄호 안에 지정해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="dbccd-104">The size of the requested allocation, with [stackalloc](../../csharp/language-reference/keywords/stackalloc.md), must be specified in square brackets.</span></span>

```csharp

{  
   unsafe public static void Main()  
   {  
      int *p = stackalloc int (30);   // CS1575

      // int *p = stackalloc int [30];  
   }  
}  
```
