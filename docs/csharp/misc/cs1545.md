# <a name="compiler-error-cs1545"></a><span data-ttu-id="952cd-102">컴파일러 오류 CS1545</span><span class="sxs-lookup"><span data-stu-id="952cd-102">Compiler Error CS1545</span></span>
<span data-ttu-id="952cd-103">'property' 속성, 인덱서 또는 이벤트는 이 언어에서 지원되지 않습니다. 'set accessor' 또는 'get accessor' 접근자 메서드를 직접 호출해 보세요.</span><span class="sxs-lookup"><span data-stu-id="952cd-103">Property, indexer, or event 'property' is not supported by the language; try directly calling accessor methods 'set accessor' or 'get accessor'</span></span>  
  
 <span data-ttu-id="952cd-104">코드가 기본이 아닌 [인덱서](../../csharp/programming-guide/indexers/index.md) 가 있는 개체를 사용하여 인덱싱된 구문을 사용하려고 합니다.</span><span class="sxs-lookup"><span data-stu-id="952cd-104">The code is consuming an object that has a non-default [indexer](../../csharp/programming-guide/indexers/index.md) and tried to use the indexed syntax.</span></span> <span data-ttu-id="952cd-105">이 오류를 해결하려면 속성의 `get` 또는 `set` 접근자 메서드를 호출하세요.</span><span class="sxs-lookup"><span data-stu-id="952cd-105">To resolve this error, call the property's `get` or `set` accessor method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="952cd-106">예제</span><span class="sxs-lookup"><span data-stu-id="952cd-106">Example</span></span>  
  
```cpp  
// CPP1545.cpp  
// compile with: /clr /LD  
// a Visual C++ program  
using namespace System;  
public ref struct Employee {  
   Employee( String^ s, int d ) {}  
  
   property String^ name {  
      String^ get() {  
         return nullptr;  
      }  
   }  
};  
  
public ref struct Manager {  
   property Employee^ Report [String^] {  
      Employee^ get(String^ s) {  
         return nullptr;  
      }  
  
      void set(String^ s, Employee^ e) {}  
   }  
};  
```  
  
## <a name="example"></a><span data-ttu-id="952cd-107">예제</span><span class="sxs-lookup"><span data-stu-id="952cd-107">Example</span></span>  
 <span data-ttu-id="952cd-108">다음 샘플에서는 CS1545를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="952cd-108">The following sample generates CS1545.</span></span>  
  
```csharp  
// CS1545.cs  
// compile with: /r:CPP1545.dll  
  
class x {  
   public static void Main() {  
      Manager Ed = new Manager();  
      Employee Bob = new Employee("Bob Smith", 12);  
      Ed.Report[ Bob.name ] = Bob;   // CS1545  
      Ed.set_Report( Bob.name, Bob);   // OK  
   }  
}  
```
