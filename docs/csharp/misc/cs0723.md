# <a name="compiler-error-cs0723"></a>컴파일러 오류 CS0723
'type' 정적 형식의 변수를 선언할 수 없습니다.  
  
 정적 형식의 인스턴스를 만들 수 없습니다.  
  
 다음 샘플에서는 CS0723을 생성합니다.  
  
```csharp  
// CS0723.cs  
public static class SC  
{  
}  
  
public class CMain  
{  
   public static void Main()  
   {  
      SC sc = null;  // CS0723  
   }  
}  
```
