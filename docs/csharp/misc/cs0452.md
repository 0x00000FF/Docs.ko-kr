# <a name="compiler-error-cs0452"></a><span data-ttu-id="e9e96-102">컴파일러 오류 CS0452</span><span class="sxs-lookup"><span data-stu-id="e9e96-102">Compiler Error CS0452</span></span>
<span data-ttu-id="e9e96-103">제네릭 형식 또는 메서드 'identifier of generic'에서 'type name' 형식을 'parameter name' 매개 변수로 사용하려면 해당 형식이 참조 형식이어야 합니다.</span><span class="sxs-lookup"><span data-stu-id="e9e96-103">The type 'type name' must be a reference type in order to use it as parameter 'parameter name' in the generic type or method 'identifier of generic'</span></span>  
  
 <span data-ttu-id="e9e96-104">이 오류는 `struct` 또는 `int` 등의 값 형식을 참조 형식 제약 조건이 있는 제네릭 형식 또는 메서드에 매개 변수로 전달할 때 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="e9e96-104">This error occurs when you pass a value type such as a `struct` or `int` as a parameter to a generic type or method that has a reference type constraint.</span></span>  
  
## <a name="example"></a><span data-ttu-id="e9e96-105">예제</span><span class="sxs-lookup"><span data-stu-id="e9e96-105">Example</span></span>  
 <span data-ttu-id="e9e96-106">다음 코드에서는 CS0452 오류를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="e9e96-106">The following code generates error CS0452.</span></span>  
  
```csharp  
// CS0452.cs

public class Derived1 : BaseClass<int> { } // CS0452  
public class Derived2<S> : BaseClass<S> where S : struct { } // CS0452  
```
