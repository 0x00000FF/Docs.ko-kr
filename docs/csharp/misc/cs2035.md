# 컴파일러 오류 CS2035
명령줄 구문 오류: 'compiler\_option' 옵션에 대한 ':\<number\>'가 없습니다.  
  
 일부 컴파일러 옵션은 값이 필요합니다.  
  
## 예제  
 다음 샘플에서는 CS2035를 생성합니다.  
  
```  
// CS2035.cs // compile with: /baseaddress // CS2035 expected  
```
