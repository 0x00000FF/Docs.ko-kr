<span data-ttu-id="0178b-103">할당되지 않은 'field' 필드를 사용하고 있는 것 같습니다.</span><span class="sxs-lookup"><span data-stu-id="0178b-103">Use of possibly unassigned field 'field'</span></span>  
  
 <span data-ttu-id="0178b-104">구조체의 필드를 먼저 초기화하지 않고 사용했습니다.</span><span class="sxs-lookup"><span data-stu-id="0178b-104">A field in a structure was used without first being initialized.</span></span> <span data-ttu-id="0178b-105">이 문제를 해결하려면 먼저 초기화되지 않은 필드를 확인하고 이를 초기화한 다음 액세스를 시도합니다.</span><span class="sxs-lookup"><span data-stu-id="0178b-105">To solve this problem, first determine which field was uninitialized and then initialize it before you try to access it.</span></span> <span data-ttu-id="0178b-106">구조체를 초기화 하는 방법에 대 한 자세한 내용은 [구조체 형식](../language-reference/builtin-types/struct.md)을 참조 하세요.</span><span class="sxs-lookup"><span data-stu-id="0178b-106">For more information about initializing structs, see [Structure types](../language-reference/builtin-types/struct.md).</span></span>
  
 <span data-ttu-id="0178b-107">다음 샘플에서는 CS0170을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="0178b-107">The following sample generates CS0170:</span></span>  
  
```csharp  
// CS0170.cs  
public struct error  
{  
   public int i;  
}  
  
public class MyClass  
{  
   public static void Main()  
   {  
      error e;  
      // uncomment the next line to resolve this error  
      // e.i = 0;  
      System.Console.WriteLine( e.i );   // CS0170 because   
                                         //e.i was never assigned  
   }  
}  
```
