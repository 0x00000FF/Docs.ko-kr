# <a name="compiler-error-cs0744"></a><span data-ttu-id="93701-102">컴파일러 오류 CS0744</span><span class="sxs-lookup"><span data-stu-id="93701-102">Compiler Error CS0744</span></span>
<span data-ttu-id="93701-103">키워드 'equals'가 필요합니다.</span><span class="sxs-lookup"><span data-stu-id="93701-103">Expected contextual keyword 'equals'</span></span>  
  
 <span data-ttu-id="93701-104">다음 예제와 같이 `join` 절에 대한 패턴은 `join`...`in`...`on`...`equals`입니다.</span><span class="sxs-lookup"><span data-stu-id="93701-104">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>  
  
```csharp  
var query = from x in array1  
            join y in array2 on x equals y  
            select x;  
```  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="93701-105">이 오류를 해결하려면</span><span class="sxs-lookup"><span data-stu-id="93701-105">To correct this error</span></span>  
  
1. <span data-ttu-id="93701-106">`equals` 절에 `join` 키워드를 추가합니다.</span><span class="sxs-lookup"><span data-stu-id="93701-106">Add the `equals` keyword to the `join` clause.</span></span>  
  
## <a name="example"></a><span data-ttu-id="93701-107">예제</span><span class="sxs-lookup"><span data-stu-id="93701-107">Example</span></span>  
 <span data-ttu-id="93701-108">다음 코드에서는 CS0744를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="93701-108">The following code generates CS0744:</span></span>  
  
```csharp  
// cs0744.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3 ,4, 5, 6 };  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 on x y // CS0744  
                select x;  
        return 1;  
    }  
}  
```
