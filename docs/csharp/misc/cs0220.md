# <a name="compiler-error-cs0220"></a><span data-ttu-id="6e55a-102">컴파일러 오류 CS0220</span><span class="sxs-lookup"><span data-stu-id="6e55a-102">Compiler Error CS0220</span></span>
<span data-ttu-id="6e55a-103">checked 모드에서 컴파일하면 작업이 오버플로됩니다.</span><span class="sxs-lookup"><span data-stu-id="6e55a-103">The operation overflows at compile time in checked mode</span></span>  
  
 <span data-ttu-id="6e55a-104">작업이 기본값인 [checked](../../csharp/language-reference/keywords/checked.md)로 검색되어 데이터가 손실되었습니다.</span><span class="sxs-lookup"><span data-stu-id="6e55a-104">An operation was detected by [checked](../../csharp/language-reference/keywords/checked.md), which is the default, and a data loss resulted.</span></span> <span data-ttu-id="6e55a-105">이 오류를 해결하려면 할당에 대한 입력을 수정하거나 [unchecked](../../csharp/language-reference/keywords/unchecked.md) 를 사용합니다.</span><span class="sxs-lookup"><span data-stu-id="6e55a-105">Either correct the inputs to the assignment or use [unchecked](../../csharp/language-reference/keywords/unchecked.md) to resolve this error.</span></span> <span data-ttu-id="6e55a-106">자세한 내용은 참조 [선택 및 선택 취소](../../csharp/language-reference/keywords/checked-and-unchecked.md)합니다.</span><span class="sxs-lookup"><span data-stu-id="6e55a-106">For more information, see [Checked and Unchecked](../../csharp/language-reference/keywords/checked-and-unchecked.md).</span></span>  
  
 <span data-ttu-id="6e55a-107">다음 샘플에서는 CS0220을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="6e55a-107">The following sample generates CS0220:</span></span>  
  
```csharp  
// CS0220.cs  
using System;  
  
class TestClass  
{  
   const int x = 1000000;  
   const int y = 1000000;  
  
   public int MethodCh()  
   {  
      int z = (x * y);   // CS0220  
      return z;  
   }  
  
   public int MethodUnCh()  
   {  
      unchecked  
      {  
         int z = (x * y);  
         return z;  
      }  
   }  
  
   public static void Main()  
   {  
      TestClass myObject = new TestClass();  
      Console.WriteLine("Checked  : {0}", myObject.MethodCh());  
      Console.WriteLine("Unchecked: {0}", myObject.MethodUnCh());  
   }  
}  
```
