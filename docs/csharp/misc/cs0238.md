# <a name="compiler-error-cs0238"></a><span data-ttu-id="2ef3d-102">컴파일러 오류 CS0238</span><span class="sxs-lookup"><span data-stu-id="2ef3d-102">Compiler Error CS0238</span></span>
<span data-ttu-id="2ef3d-103">'member'는 override가 아니므로 sealed가 될 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="2ef3d-103">'member' cannot be sealed because it is not an override</span></span>  
  
 <span data-ttu-id="2ef3d-104">[override](../../csharp/language-reference/keywords/sealed.md) 로 표시되지 않은 멤버에 [sealed](../../csharp/language-reference/keywords/override.md)를 사용했습니다.</span><span class="sxs-lookup"><span data-stu-id="2ef3d-104">[sealed](../../csharp/language-reference/keywords/sealed.md) was used on a member that was not also marked with [override](../../csharp/language-reference/keywords/override.md).</span></span> <span data-ttu-id="2ef3d-105">자세한 내용은 [상속](../../csharp/programming-guide/classes-and-structs/inheritance.md)을 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="2ef3d-105">For more information, see [Inheritance](../../csharp/programming-guide/classes-and-structs/inheritance.md).</span></span>  
  
 <span data-ttu-id="2ef3d-106">다음 샘플에서는 CS0238을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="2ef3d-106">The following sample generates CS0238:</span></span>  
  
```csharp  
// CS0238.cs  
abstract class MyClass  
{  
   public abstract void f();  
}  
  
class MyClass2 : MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public sealed void f() // CS0238  
   // Try the following definition instead:  
   // public override sealed void f()  
   {  
   }  
}  
```
