<span data-ttu-id="166ed-103">이 switch 문에 이미 'label' 레이블이 있습니다.</span><span class="sxs-lookup"><span data-stu-id="166ed-103">The label 'label' already occurs in this switch statement</span></span>

 <span data-ttu-id="166ed-106">다음 샘플에서는 CS0152를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="166ed-106">The following sample generates CS0152:</span></span>  
  
```csharp  
// CS0152.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
  
         switch (i)  
         {  
            case 1:  
               i++;  
               return;  
  
            case 1:   // CS0152, two case 1 statements  
               i++;  
               return;  
         }  
      }  
   }  
}  
```
