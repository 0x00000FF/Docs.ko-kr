 <span data-ttu-id="614a3-104">할당을 완료할 수 없습니다. 접미사를 사용하여 지침을 수정합니다.</span><span class="sxs-lookup"><span data-stu-id="614a3-104">An assignment could not be completed; use a suffix to correct the instruction.</span></span> <span data-ttu-id="614a3-105">각 형식에 대한 설명서는 형식에 대한 해당 접미사를 식별합니다.</span><span class="sxs-lookup"><span data-stu-id="614a3-105">The documentation for each type identifies the corresponding suffix for the type.</span></span> <span data-ttu-id="614a3-106">변환에 대 한 자세한 내용은 참조 하십시오.  <span data-ttu-id="614a3-107">다음 샘플에서는 CS0664를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="614a3-107">The following sample generates CS0664:</span></span>  
  
```csharp  
// CS0664.cs  
class Example  
{  
    static void Main()  
    {  
        decimal d1 = 1.0;   // CS0664, because 1.0 is interpreted  
                            // as a double.  
  
        // Try the following line instead.  
        decimal d2 = 1.0M;  // The M tells the compiler that 1.0 is a  
                            // decimal.  
        Console.WriteLine(d2);  
    }  
}  
```
