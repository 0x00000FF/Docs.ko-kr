# <a name="compiler-error-cs0698"></a>컴파일러 오류 CS0698
'class'는 특성 클래스이므로 제네릭 형식을 파생시킬 수 없습니다.  
  
 특성 클래스에서 파생되는 모든 클래스는 특성입니다. 특성은 제네릭 형식이 될 수 없습니다.  
  
 다음 샘플에서는 CS0698을 생성합니다.  
  
```  
// CS0698.cs  
class C<T> : System.Attribute  // CS0698  
{  
}  
```
