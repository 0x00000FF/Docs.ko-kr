<span data-ttu-id="44ce9-103">생성자, 소멸자, 연산자 또는 명시적 인터페이스 구현이므로 'function'에서 Conditional 특성을 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="44ce9-103">The Conditional attribute is not valid on 'function' because it is a constructor, destructor, operator, or explicit interface implementation</span></span>  
  
 <span data-ttu-id="44ce9-104">지정된 메서드에`Conditional` 을 적용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="44ce9-104">`Conditional` cannot be applied to the specified methods.</span></span>  
  
 <span data-ttu-id="44ce9-105">예를 들어 명시적 인터페이스 정의에 일부 특성을 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="44ce9-105">For example, you cannot use some attributes on an explicit interface definition.</span></span> <span data-ttu-id="44ce9-106">다음 샘플에서는 CS0577을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="44ce9-106">The following sample generates CS0577:</span></span>  
  
```csharp  
// CS0577.cs  
// compile with: /target:library  
interface I  
{  
   void m();  
}  
  
public class MyClass : I  
{  
   [System.Diagnostics.Conditional("a")]   // CS0577  
   void I.m() {}  
}  
```
