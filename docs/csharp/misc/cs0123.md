# <a name="compiler-error-cs0123"></a><span data-ttu-id="5feac-102">컴파일러 오류 CS0123</span><span class="sxs-lookup"><span data-stu-id="5feac-102">Compiler Error CS0123</span></span>
<span data-ttu-id="5feac-103">'delegate' 대리자와 일치하는 'method'에 대한 오버로드가 없습니다.</span><span class="sxs-lookup"><span data-stu-id="5feac-103">No overload for 'method' matches delegate 'delegate'</span></span>  
  
 <span data-ttu-id="5feac-104">올바른 서명을 사용하지 않아 대리자를 만들려는 시도가 실패했습니다.</span><span class="sxs-lookup"><span data-stu-id="5feac-104">An attempt to create a delegate failed because the correct signature was not used.</span></span> <span data-ttu-id="5feac-105">대리자 선언과 동일한 서명을 사용하여 대리자의 인스턴스를 선언해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="5feac-105">Instances of a delegate must be declared with the same signature as the delegate declaration.</span></span>  
  
 <span data-ttu-id="5feac-106">이 오류를 해결하려면 메서드 또는 대리자 서명을 조정합니다.</span><span class="sxs-lookup"><span data-stu-id="5feac-106">You can resolve this error by adjusting either the method or delegate signature.</span></span> <span data-ttu-id="5feac-107">자세한 내용은 [대리자](../../csharp/programming-guide/delegates/index.md)를 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="5feac-107">For more information, see [Delegates](../../csharp/programming-guide/delegates/index.md).</span></span>  
  
 <span data-ttu-id="5feac-108">다음 샘플에서는 CS0123을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="5feac-108">The following sample generates CS0123.</span></span>  
  
```  
// CS0123.cs  
delegate void D();  
delegate void D2(int i);  
  
public class C  
{  
   public static void f(int i) {}  
  
   public static void Main()  
   {  
      D d = new D(f);   // CS0123  
      D2 d2 = new D2(f);   // OK  
   }  
}  
```
