# <a name="compiler-error-cs0160"></a><span data-ttu-id="06e0c-102">컴파일러 오류 CS0160</span><span class="sxs-lookup"><span data-stu-id="06e0c-102">Compiler Error CS0160</span></span>
<span data-ttu-id="06e0c-103">이전의 catch 절에서 이 형식이나 상위 형식('type')의 예외를 모두 catch합니다.</span><span class="sxs-lookup"><span data-stu-id="06e0c-103">A previous catch clause already catches all exceptions of this or of a super type ('type')</span></span>  
  
 <span data-ttu-id="06e0c-104">일련의 **catch** 문을 파생의 내림차순으로 정렬해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="06e0c-104">A series of **catch** statements needs to be in decreasing order of derivation.</span></span> <span data-ttu-id="06e0c-105">예를 들어 가장 많이 파생된 개체가 첫 번째로 나타나야 합니다.</span><span class="sxs-lookup"><span data-stu-id="06e0c-105">For example, the most derived objects must appear first.</span></span>  
  
 <span data-ttu-id="06e0c-106">자세한 내용은 참조 [예외 처리문](../../csharp/language-reference/keywords/exception-handling-statements.md) 및 [예외 및 예외 처리](../../csharp/programming-guide/exceptions/index.md)합니다.</span><span class="sxs-lookup"><span data-stu-id="06e0c-106">For more information, see [Exception Handling Statements](../../csharp/language-reference/keywords/exception-handling-statements.md) and [Exceptions and Exception Handling](../../csharp/programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="06e0c-107">다음 샘플에서는 CS0160을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="06e0c-107">The following sample generates CS0160:</span></span>  
  
```  
// CS0160.cs  
public class MyClass2 : System.Exception {}  
public class MyClass  
{  
   public static void Main()  
   {  
      try {}  
  
      catch(System.Exception) {}   // Second-most derived; should be second catch  
      catch(MyClass2) {}   // CS0160  Most derived; should be first catch  
   }  
}  
```
