<span data-ttu-id="1c4bc-103">제어를 호출자에게 반환하려면 자동으로 구현된 'name' 속성의 지원 필드가 완전히 할당되어야 합니다.</span><span class="sxs-lookup"><span data-stu-id="1c4bc-103">Backing field for automatically implemented property 'name' must be fully assigned before control is returned to the caller.</span></span> <span data-ttu-id="1c4bc-104">생성자 이니셜라이저에서 기본 생성자를 호출하세요.</span><span class="sxs-lookup"><span data-stu-id="1c4bc-104">Consider calling the default constructor from a constructor initializer.</span></span>  
  
 <span data-ttu-id="1c4bc-105">[구조체](../../csharp/language-reference/keywords/struct.md) 의 생성자는 구조체의 모든 필드를 초기화해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="1c4bc-105">A constructor in a [struct](../../csharp/language-reference/keywords/struct.md) must initialize all fields in the struct.</span></span> <span data-ttu-id="1c4bc-106">자세한 내용은 [생성자](../../csharp/programming-guide/classes-and-structs/constructors.md)를 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="1c4bc-106">For more information, see [Constructors](../../csharp/programming-guide/classes-and-structs/constructors.md).</span></span>  
  
 <span data-ttu-id="1c4bc-107">다음 샘플에서는 CS0171을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="1c4bc-107">The following sample generates CS0171:</span></span>  
  
```csharp  
// CS0171.cs  
struct MyStruct  
{  
   MyStruct(int initField)   // CS0171  
   {  
      // i = initField;      // uncomment this line to resolve this error  
   }  
   public int i;  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      MyStruct aStruct = new MyStruct();  
   }  
}  
```
