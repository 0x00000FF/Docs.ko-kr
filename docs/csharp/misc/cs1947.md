## <a name="example"></a><span data-ttu-id="d50ee-109">예제</span><span class="sxs-lookup"><span data-stu-id="d50ee-109">Example</span></span>  
 <span data-ttu-id="d50ee-110">다음 코드에서는 CS1947을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="d50ee-110">The following code generates CS1947:</span></span>  
  
```csharp  
// cs1947.cs  
using System.Linq;  
class Test  
{  
    static void Main()  
    {  
        int[] array = new int[] { 1, 2, 3, 4, 5 };  
        var x = from i in array  
                let k = i  
                select i = 5; // CS1947  
        x.ToList();  
    }  
}  
```
