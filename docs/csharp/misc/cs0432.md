# <a name="compiler-error-cs0432"></a><span data-ttu-id="06024-102">컴파일러 오류 CS0432</span><span class="sxs-lookup"><span data-stu-id="06024-102">Compiler Error CS0432</span></span>
<span data-ttu-id="06024-103">'identifier' 별칭을 찾을 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="06024-103">Alias 'identifier' not found</span></span>  
  
 <span data-ttu-id="06024-104">이 오류는 별칭이 아닌 식별자의 오른쪽에 "::"을 사용하는 경우에 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="06024-104">This error occurs when you use "::" to the right of an identifier that is not an alias.</span></span> <span data-ttu-id="06024-105">오류를 해결하려면 "."를 대신 사용합니다.</span><span class="sxs-lookup"><span data-stu-id="06024-105">To resolve the error, use "." instead.</span></span>  
  
 <span data-ttu-id="06024-106">다음 예제에서는 CS0432를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="06024-106">The following example generates CS0432:</span></span>  
  
```csharp  
// CS0432.cs  
namespace A {  
    public class B {  
        public static void Meth() { }  
    }  
}  
  
public class Test  
{  
    public static void Main()  
    {  
        A::B.Meth();   // CS0432  
       // To resolve, use the following line instead:  
       // A.B.Meth();  
    }  
}  
```
