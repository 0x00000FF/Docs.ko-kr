<span data-ttu-id="8b3d4-104">[상수](../language-reference/keywords/const.md) 선언에 지정 된 형식은 `byte`, `sbyte`, `ushort`, `short`, `uint`, `int`, `ulong`, `long`, `char`, `float`, `double`, `decimal`, `bool`, `string`, [열거형](../language-reference/builtin-types/enum.md) 형식 또는 `null`값이 할당 된 참조 형식 이어야 합니다.</span><span class="sxs-lookup"><span data-stu-id="8b3d4-104">The type specified in a [constant](../language-reference/keywords/const.md) declaration must be `byte`, `sbyte`, `ushort`, `short`, `uint`, `int`, `ulong`, `long`, `char`, `float`, `double`, `decimal`, `bool`, `string`, an [enum](../language-reference/builtin-types/enum.md) type, or a reference type that is assigned a value of `null`.</span></span> <span data-ttu-id="8b3d4-105">각 상수 식은 대상 형식 또는 대상 형식으로 암시적으로 변환할 수 있는 형식의 값을 생성 해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="8b3d4-105">Each constant expression must yield a value of the target type or of a type that is implicitly convertible to the target type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="8b3d4-106">예</span><span class="sxs-lookup"><span data-stu-id="8b3d4-106">Example</span></span>

```csharp  
}

{  
    // To resolve the error but retain the "const-ness",

    }  
}  
```
