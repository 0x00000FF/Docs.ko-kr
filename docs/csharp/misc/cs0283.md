## <a name="example"></a><span data-ttu-id="6c198-106">예제</span><span class="sxs-lookup"><span data-stu-id="6c198-106">Example</span></span>  
 <span data-ttu-id="6c198-107">다음 예제에서는 CS0283을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="6c198-107">The following example generates CS0283.</span></span>  
  
```csharp

}

{  
    // To resolve the error but retain the "const-ness",  
    // change const to readonly.

        return 1;  
    }  
}  
```
