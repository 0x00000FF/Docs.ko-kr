 <span data-ttu-id="87ca1-104">[정적](../../csharp/language-reference/keywords/static.md) 생성자를 하나 이상의 매개 변수와 함께 선언할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="87ca1-104">A [static](../../csharp/language-reference/keywords/static.md) constructor cannot be declared with one or more parameters.</span></span> <span data-ttu-id="87ca1-105">자세한 내용은 [생성자](../../csharp/programming-guide/classes-and-structs/constructors.md)를 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="87ca1-105">For more information, see [Constructors](../../csharp/programming-guide/classes-and-structs/constructors.md).</span></span>

```csharp

   public class MyClass  
   {  
      public MyClass(int i)  
      {  
      }  
   }

      static MyClass2(int i)   // CS0132  
      {  
      }  
   }  
}  
```
