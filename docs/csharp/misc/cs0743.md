<span data-ttu-id="ea451-103">키워드 'on'이 필요합니다.</span><span class="sxs-lookup"><span data-stu-id="ea451-103">Expected contextual keyword 'on'</span></span>  
  
 <span data-ttu-id="ea451-104">다음 예제와 같이 `join` 절에 대한 패턴은 `join`...`in`...`on`...`equals`입니다.</span><span class="sxs-lookup"><span data-stu-id="ea451-104">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>

## <a name="example"></a><span data-ttu-id="ea451-107">예제</span><span class="sxs-lookup"><span data-stu-id="ea451-107">Example</span></span>

```  
// cs0743.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3 ,4, 5, 6,};  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 x equals y // CS0743  
                select x;  
        return 1;  
    }  
}  
```
