# <a name="compiler-error-cs0743"></a><span data-ttu-id="20e90-102">컴파일러 오류 CS0743</span><span class="sxs-lookup"><span data-stu-id="20e90-102">Compiler Error CS0743</span></span>
<span data-ttu-id="20e90-103">키워드 'on'이 필요합니다.</span><span class="sxs-lookup"><span data-stu-id="20e90-103">Expected contextual keyword 'on'</span></span>  
  
 <span data-ttu-id="20e90-104">다음 예제와 같이 `join` 절에 대한 패턴은 `join`...`in`...`on`...`equals`입니다.</span><span class="sxs-lookup"><span data-stu-id="20e90-104">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>  
  
```csharp  
var query = from x in array1  
            join y in array2 on x equals y  
            select x;  
```  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="20e90-105">이 오류를 해결하려면</span><span class="sxs-lookup"><span data-stu-id="20e90-105">To correct this error</span></span>  
  
1.  <span data-ttu-id="20e90-106">`on` 절에 `join` 키워드를 추가합니다.</span><span class="sxs-lookup"><span data-stu-id="20e90-106">Add the `on` keyword to the `join` clause.</span></span>  
  
## <a name="example"></a><span data-ttu-id="20e90-107">예제</span><span class="sxs-lookup"><span data-stu-id="20e90-107">Example</span></span>  
 <span data-ttu-id="20e90-108">다음 코드에서는 CS0743을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="20e90-108">The following code generates CS0743:</span></span>  
  
```csharp  
// cs0743.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3 ,4, 5, 6,};  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 x equals y // CS0743  
                select x;  
        return 1;  
    }  
}  
```
