 <span data-ttu-id="5ecfa-105">반복기는 return 문 대신 yield 문을 통해 값을 반환하는 특수 함수입니다.</span><span class="sxs-lookup"><span data-stu-id="5ecfa-105">An iterator is a special function that returns a value via the yield statement rather than the return statement.</span></span> <span data-ttu-id="5ecfa-106">자세한 내용은 **반복기**를 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="5ecfa-106">For more information, see **iterators**.</span></span>  
  
 <span data-ttu-id="5ecfa-107">다음 샘플에서는 CS1622를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="5ecfa-107">The following sample generates CS1622:</span></span>  
  
```csharp  
// CS1622.cs  
// compile with: /target:library  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      return (IEnumerator) this;  // CS1622  
      yield return this;   // OK  
   }  
}  
```
