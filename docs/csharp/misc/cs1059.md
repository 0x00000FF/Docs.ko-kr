## <a name="example"></a><span data-ttu-id="d27e6-110">예제</span><span class="sxs-lookup"><span data-stu-id="d27e6-110">Example</span></span>  
 <span data-ttu-id="d27e6-111">다음 예제에서는 `i` 가 변수가 아니라 상수이고 `E` 가 해당 요소도 상수 값인 `Enum` 형식이기 때문에 CS1059를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="d27e6-111">The following example generates CS1059 because `i` is a constant, not a variable, and `E` is an `Enum` type, whose elements are also constant values.</span></span>  
  
```  
// CS1059.cs  
    class Program  
    {  
        public enum E : sbyte  
        {  
            a = 1,  
            b = 2  
        }  
  
        static void Main(string[] args)  
        {  
            const int i = 0;  
            i++;            // CS1059  
            E test = E.a++; // CS1059  
        }  
    }  
```
