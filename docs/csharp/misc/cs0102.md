## <a name="example"></a><span data-ttu-id="21330-106">예제</span><span class="sxs-lookup"><span data-stu-id="21330-106">Example</span></span>

```csharp

// compile with: /target:library

{  
   public class MyClass  
   {  
      string s = "Hello";  
      string s = "Goodbye";   // CS0102  
  
      public void GetString()  
      {  
         string s = "Hello again";   // method scope, no error  
      }  
   }  
}  
```
