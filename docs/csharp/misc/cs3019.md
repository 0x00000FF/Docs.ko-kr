## <a name="example"></a><span data-ttu-id="61f74-107">예제</span><span class="sxs-lookup"><span data-stu-id="61f74-107">Example</span></span>  
 <span data-ttu-id="61f74-108">다음 샘플에서는 CS3019를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="61f74-108">The following sample generates CS3019:</span></span>  
  
```csharp  
// CS3019.cs  
// compile with: /W:2  
  
using System;  
  
[assembly: CLSCompliant(true)]  
  
// To fix the error, remove the next line  
[CLSCompliant(true)]  // CS3019  
class C  
{  
    [CLSCompliant(false)]  // CS3019  
    void Foo()  
    {  
    }  
  
    static void Main()  
    {  
    }  
}  
```
