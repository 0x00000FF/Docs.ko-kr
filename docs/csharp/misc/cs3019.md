## <a name="example"></a><span data-ttu-id="a554e-107">예제</span><span class="sxs-lookup"><span data-stu-id="a554e-107">Example</span></span>

```

// compile with: /W:2

[assembly: CLSCompliant(true)]

    }

    {  
    }  
}  
```
