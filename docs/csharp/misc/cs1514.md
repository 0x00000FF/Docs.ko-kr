# <a name="compiler-error-cs1514"></a>컴파일러 오류 CS1514
{가 필요합니다.  
  
 컴파일러에서 열린 중괄호(`{`)를 찾았지만 없습니다.  
  
 다음 샘플에서는 CS1514를 생성합니다.  
  
```  
// CS1514.cs  
namespace x  
// CS1514, no open curly brace  
```
