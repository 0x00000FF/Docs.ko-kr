# <a name="compiler-error-cs1676"></a><span data-ttu-id="9b4b3-102">컴파일러 오류 CS1676</span><span class="sxs-lookup"><span data-stu-id="9b4b3-102">Compiler Error CS1676</span></span>
<span data-ttu-id="9b4b3-103">'number' 매개 변수는 'keyword' 키워드를 사용하여 선언해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="9b4b3-103">Parameter 'number' must be declared with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="9b4b3-104">이 오류는 무명 메서드의 매개 변수 형식 한정자가 메서드를 캐스팅하는 대리자의 선언에서 사용되는 것과 다른 경우 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="9b4b3-104">This error occurs when the parameter type modifier in an anonymous method is different from that used in the declaration of the delegate you are casting the method to.</span></span>  
  
 <span data-ttu-id="9b4b3-105">다음 샘플에서는 CS1676을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="9b4b3-105">The following sample generates CS1676:</span></span>  
  
```  
// CS1676.cs  
delegate void E(ref int i);  
class Errors   
{  
   static void Main()  
   {  
      E e = delegate(out int i) { };   // CS1676  
      // To resolve, use the following line instead:  
      // E e = delegate(ref int i) { };  
   }  
}  
```
