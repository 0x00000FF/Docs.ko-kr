 <span data-ttu-id="e2df6-104">[interface](../../csharp/language-reference/keywords/interface.md) 의 상속 목록에 자신에 대한 직접 또는 간접 참조가 포함되어 있습니다.</span><span class="sxs-lookup"><span data-stu-id="e2df6-104">The inheritance list for an [interface](../../csharp/language-reference/keywords/interface.md) includes a direct or indirect reference to itself.</span></span> <span data-ttu-id="e2df6-105">인터페이스는 다른 인터페이스에서만 상속할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="e2df6-105">An interface cannot inherit from itself.</span></span>

```csharp

   public interface a  
   {  
   }  
  
   public interface b : a, c

   }

   {  
   }  
}  
```
