 <span data-ttu-id="e7d07-104">[interface](../../csharp/language-reference/keywords/interface.md) 의 상속 목록에 자신에 대한 직접 또는 간접 참조가 포함되어 있습니다.</span><span class="sxs-lookup"><span data-stu-id="e7d07-104">The inheritance list for an [interface](../../csharp/language-reference/keywords/interface.md) includes a direct or indirect reference to itself.</span></span> <span data-ttu-id="e7d07-105">인터페이스는 다른 인터페이스에서만 상속할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="e7d07-105">An interface cannot inherit from itself.</span></span>  
  
 <span data-ttu-id="e7d07-106">다음 샘플에서는 CS0529를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="e7d07-106">The following sample generates CS0529:</span></span>  
  
```  
// CS0529.cs

{  
   public interface a  
   {  
   }  
  
   public interface b : a, c  
   {  
   }  
  
   public interface c : b   // CS0529, b inherits from c  
   {  
   }  
}  
```
