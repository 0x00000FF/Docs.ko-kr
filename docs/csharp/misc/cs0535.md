 <span data-ttu-id="5ef27-104">[클래스](../../csharp/language-reference/keywords/class.md) 가 [인터페이스](../../csharp/language-reference/keywords/interface.md)에서 파생되었지만 해당 클래스가 인터페이스 멤버를 하나 이상 구현하지 않았습니다.</span><span class="sxs-lookup"><span data-stu-id="5ef27-104">A [class](../../csharp/language-reference/keywords/class.md) derived from an [interface](../../csharp/language-reference/keywords/interface.md), but the class did not implement one or more of the interface's members.</span></span> <span data-ttu-id="5ef27-105">클래스는 클래스가 파생된 인터페이스의 모든 멤버를 구현하거나 `abstract`로 선언되어야 합니다.</span><span class="sxs-lookup"><span data-stu-id="5ef27-105">A class must implement all members of interfaces from which it derives or else be declared `abstract`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="5ef27-106">예제</span><span class="sxs-lookup"><span data-stu-id="5ef27-106">Example</span></span>  
 <span data-ttu-id="5ef27-107">다음 샘플에서는 CS0535를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="5ef27-107">The following sample generates CS0535.</span></span>  
  
```  
// CS0535.cs  
public interface A  
{  
   void F();  
}  
  
public class B : A {}   // CS0535 A::F is not implemented  
  
// OK  
public class C : A {  
   public void F() {}  
   public static void Main() {}  
}  
```
