## <a name="example"></a><span data-ttu-id="1ccef-106">예</span><span class="sxs-lookup"><span data-stu-id="1ccef-106">Example</span></span>  
 <span data-ttu-id="1ccef-107">다음 샘플에서는 CS0305를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="1ccef-107">The following sample generates CS0305.</span></span>  
  
```csharp  
// CS0305.cs  
public class MyList<T> {}  
public class MyClass<T> {}  
  
class MyClass  
{  
   public static void Main()  
   {  
      MyList<MyClass, MyClass> list1 = new MyList<MyClass>();   // CS0305  
      MyList<MyClass> list2 = new MyList<MyClass>();   // OK  
   }  
}  
```
