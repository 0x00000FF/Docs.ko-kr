## <a name="example"></a><span data-ttu-id="358fa-106">예제</span><span class="sxs-lookup"><span data-stu-id="358fa-106">Example</span></span>  
 <span data-ttu-id="358fa-107">다음 샘플에서는 CS0704를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="358fa-107">The following sample generates CS0704:</span></span>  
  
```  
// CS0704.cs  
class B  
{  
    public class I { }  
}  
  
class C<T> where T : B  
{  
    T.I f;  // CS0704 – member lookup on type parameter  
    // Try this instead:  
    // B.I f;  
}  
  
class CMain  
{  
    public static void Main() {}  
}  
```
