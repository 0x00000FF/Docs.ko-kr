## <a name="example"></a><span data-ttu-id="ea57b-106">예제</span><span class="sxs-lookup"><span data-stu-id="ea57b-106">Example</span></span>

```csharp

{  
    public class I { }  
}  
  
class C<T> where T : B  
{  
    T.I f;  // CS0704 – member lookup on type parameter  
    // Try this instead:  
    // B.I f;  
}

{  
    public static void Main() {}  
}  
```
