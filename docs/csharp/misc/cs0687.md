## <a name="example"></a><span data-ttu-id="b64db-108">예제</span><span class="sxs-lookup"><span data-stu-id="b64db-108">Example</span></span>

```csharp

using System;  
  
public class Test
{  
    public static int x = 77;  
  
    public static void Main()
    {  
        Console.WriteLine(M::x); // CS0687  
        // To resolve use the following line instead:  
        // Console.WriteLine(M.x);  
    }  
}  
```
