# <a name="compiler-error-cs1583"></a>컴파일러 오류 CS1583
'file'은 올바른 Win32 리소스 파일이 아닙니다.  
  
 이 오류와 파일 이름을 지정 하는 경우에 발생는 [/win32res](../../csharp/language-reference/compiler-options/win32res-compiler-option.md) 잘못 되었거나 올바른 형식의 Win32 리소스 파일 되지 않는 컴파일러 옵션입니다. Visual Studio에서는 프로젝트 디자이너의 응용 프로그램 창에서 파일 이름을 지정합니다.
