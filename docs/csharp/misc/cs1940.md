## <a name="example"></a><span data-ttu-id="b7c92-109">예제</span><span class="sxs-lookup"><span data-stu-id="b7c92-109">Example</span></span>  
 <span data-ttu-id="b7c92-110">다음 코드에서는 CS1940을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="b7c92-110">The following code generates CS1940:</span></span>  
  
```  
// cs1940.cs  
using System; //must include explicitly for types defined in 3.5  
class Test  
{  
    public delegate int Dele(int x);  
    int num = 0;  
    public int Select(Func<int, int> d)  
    {  
        return d(this.num);  
    }  
    public int Select(Dele d) // CS1940  
    {  
        return d(this.num) + 1;  
    }  
    public static void Main()  
    {  
        var q = from x in new Test()  
        select x;  
    }  
}  
```
