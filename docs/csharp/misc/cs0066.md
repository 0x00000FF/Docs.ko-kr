# <a name="compiler-error-cs0066"></a><span data-ttu-id="ab260-102">컴파일러 오류 CS0066</span><span class="sxs-lookup"><span data-stu-id="ab260-102">Compiler Error CS0066</span></span>
<span data-ttu-id="ab260-103">'event': 이벤트는 대리자 형식이어야 합니다.</span><span class="sxs-lookup"><span data-stu-id="ab260-103">'event': event must be of a delegate type</span></span>  
  
 <span data-ttu-id="ab260-104">이벤트 키워드에는 [delegate](../../csharp/language-reference/keywords/delegate.md) 형식이 필요합니다.</span><span class="sxs-lookup"><span data-stu-id="ab260-104">The event keyword requires a [delegate](../../csharp/language-reference/keywords/delegate.md) type.</span></span> <span data-ttu-id="ab260-105">자세한 내용은 참조 [이벤트](../../csharp/programming-guide/events/index.md) 및 [대리자](../../csharp/programming-guide/delegates/index.md)합니다.</span><span class="sxs-lookup"><span data-stu-id="ab260-105">For more information, see [Events](../../csharp/programming-guide/events/index.md) and [Delegates](../../csharp/programming-guide/delegates/index.md).</span></span>  
  
 <span data-ttu-id="ab260-106">다음 샘플에서는 CS0066을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="ab260-106">The following sample generates CS0066:</span></span>  
  
```csharp  
// CS0066.cs  
using System;  
  
public class EventHandler  
{  
}  
  
// to fix the error, remove the event declaration and the  
// EventHandler class declaration, and uncomment the following line  
// public delegate void EventHandler();  
  
public class a  
{  
   public event EventHandler Click;   // CS0066  
  
   private void TestMethod()  
   {  
      if (Click != null)  
         Click();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
