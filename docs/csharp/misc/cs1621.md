# <a name="compiler-error-cs1621"></a>컴파일러 오류 CS1621
yield 문은 무명 메서드 또는 람다 식 안에 사용할 수 없습니다.  
  
[Yield](../../csharp/language-reference/keywords/yield.md) 문은 [무명 메서드](../language-reference/operators/delegate-operator.md) 또는 [람다 식](../../csharp/programming-guide/statements-expressions-operators/lambda-expressions.md)에서 사용할 수 없습니다.
  
다음 샘플에서는 CS1621을 생성합니다.
  
```csharp  
// CS1621.cs  
  
using System.Collections;  
  
delegate object MyDelegate();  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        MyDelegate d = delegate  
        {  
            yield return this; // CS1621  
            return this;  
        };  
        d();  
        // Try this instead:  
        // MyDelegate d = delegate { return this; };  
        // yield return d();  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
