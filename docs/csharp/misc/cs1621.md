# <a name="compiler-error-cs1621"></a><span data-ttu-id="7b270-102">컴파일러 오류 CS1621</span><span class="sxs-lookup"><span data-stu-id="7b270-102">Compiler Error CS1621</span></span>
<span data-ttu-id="7b270-103">yield 문은 무명 메서드 또는 람다 식 안에 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="7b270-103">The yield statement cannot be used inside an anonymous method or lambda expression</span></span>  
  
<span data-ttu-id="7b270-104">[Yield](../language-reference/keywords/yield.md) 문은 [무명 메서드](../language-reference/operators/delegate-operator.md) 또는 [람다 식](../programming-guide/statements-expressions-operators/lambda-expressions.md)에서 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="7b270-104">You cannot use the [yield](../language-reference/keywords/yield.md) statement in an [anonymous method](../language-reference/operators/delegate-operator.md) or a [lambda expression](../programming-guide/statements-expressions-operators/lambda-expressions.md).</span></span>
  
<span data-ttu-id="7b270-105">다음 샘플에서는 CS1621을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="7b270-105">The following sample generates CS1621:</span></span>
  
```csharp  
// CS1621.cs  
  
using System.Collections;  
  
delegate object MyDelegate();  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        MyDelegate d = delegate  
        {  
            yield return this; // CS1621  
            return this;  
        };  
        d();  
        // Try this instead:  
        // MyDelegate d = delegate { return this; };  
        // yield return d();  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
