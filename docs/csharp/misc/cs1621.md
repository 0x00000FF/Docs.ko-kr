# <a name="compiler-error-cs1621"></a><span data-ttu-id="2bcb2-102">컴파일러 오류 CS1621</span><span class="sxs-lookup"><span data-stu-id="2bcb2-102">Compiler Error CS1621</span></span>
<span data-ttu-id="2bcb2-103">yield 문은 무명 메서드 또는 람다 식 안에 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="2bcb2-103">The yield statement cannot be used inside an anonymous method or lambda expression</span></span>  
  
 <span data-ttu-id="2bcb2-104">yield 문은 반복기의 무명 메서드 블록 안에 있을 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="2bcb2-104">The yield statement cannot be in an anonymous method block in an iterator.</span></span>  
  
## <a name="example"></a><span data-ttu-id="2bcb2-105">예제</span><span class="sxs-lookup"><span data-stu-id="2bcb2-105">Example</span></span>  
 <span data-ttu-id="2bcb2-106">다음 샘플에서는 CS1621을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="2bcb2-106">The following sample generates CS1621:</span></span>  
  
```  
// CS1621.cs  
  
using System.Collections;  
  
delegate object MyDelegate();  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        MyDelegate d = delegate  
        {  
            yield return this; // CS1621  
            return this;  
        };  
        d();  
        // Try this instead:  
        // MyDelegate d = delegate { return this; };  
        // yield return d();  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
