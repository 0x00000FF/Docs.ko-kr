## <a name="example"></a><span data-ttu-id="68923-105">예</span><span class="sxs-lookup"><span data-stu-id="68923-105">Example</span></span>  
 <span data-ttu-id="68923-106">다음 샘플에서는 CS1686을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="68923-106">The following sample generates CS1686.</span></span>  
  
```csharp  
// CS1686.cs  
// compile with: /unsafe /target:library  
class MyClass  
{  
   public unsafe delegate int * MyDelegate();  
  
   public unsafe int * Test()  
   {  
      int j = 0;  
      MyDelegate d = delegate { return &j; };   // CS1686  
      return &j;   // OK  
   }  
}  
```
