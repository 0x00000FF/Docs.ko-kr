## <a name="example"></a><span data-ttu-id="fd15b-106">예제</span><span class="sxs-lookup"><span data-stu-id="fd15b-106">Example</span></span>
 <span data-ttu-id="fd15b-107">다음 예제에서는 CS0471를 생성 합니다.</span><span class="sxs-lookup"><span data-stu-id="fd15b-107">The following example generates CS0471:</span></span>

```csharp
// CS0471.cs
// compile with: /t:library
class Test
{
    public void F(bool x, bool y) {}
    public void F1()
    {
        int a = 1, b = 2, c = 3;
        F(a<b, c>(3));    // CS0471
        // To resolve, try the following instead:
        // F((a<b), c>(3));
    }
}
```
