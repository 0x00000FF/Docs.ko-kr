<span data-ttu-id="7032b-103">stackalloc는 catch 또는 finally 블록에 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="7032b-103">stackalloc may not be used in a catch or finally block</span></span>  
  
 <span data-ttu-id="7032b-104">[stackalloc](../../csharp/language-reference/keywords/stackalloc.md) 키워드는 [catch](../../csharp/language-reference/keywords/try-catch.md) 또는 [finally](../../csharp/language-reference/keywords/try-catch-finally.md) 블록에 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="7032b-104">The [stackalloc](../../csharp/language-reference/keywords/stackalloc.md) keyword may not be used in a [catch](../../csharp/language-reference/keywords/try-catch.md) or [finally](../../csharp/language-reference/keywords/try-catch-finally.md) block.</span></span> <span data-ttu-id="7032b-105">자세한 내용은 참조 [예외 및 예외 처리](../../csharp/programming-guide/exceptions/index.md)합니다.</span><span class="sxs-lookup"><span data-stu-id="7032b-105">For more information, see [Exceptions and Exception Handling](../../csharp/programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="7032b-106">다음 샘플에서는 CS0255를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="7032b-106">The following sample generates CS0255:</span></span>  
  
```  
// CS0255.cs  
// compile with: /unsafe  
using System;  
  
public class TestTryFinally  
{  
   public static unsafe void Test()  
   {  
      int i = 123;  
      string s = "Some string";  
      object o = s;  
  
      try  
      {  
         // Conversion is not valid; o contains a string not an int  
         i = (int) o;  
      }  
  
      finally  
      {  
         Console.Write("i = {0}", i);  
         int* fib = stackalloc int[100];   // CS0255  
      }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
