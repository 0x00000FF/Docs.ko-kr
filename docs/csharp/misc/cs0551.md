<span data-ttu-id="51ebb-103">'implementation' 명시적 인터페이스 구현에 'accessor' 접근자가 누락되었습니다.</span><span class="sxs-lookup"><span data-stu-id="51ebb-103">Explicit interface implementation 'implementation' is missing accessor 'accessor'</span></span>  
  
 <span data-ttu-id="51ebb-104">명시적으로 인터페이스 속성을 구현하는 클래스는 인터페이스가 정의하는 모든 접근자를 구현해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="51ebb-104">A class that explicitly implements an interface's property must implement all the accessors that the interface defines.</span></span>

## <a name="example"></a><span data-ttu-id="51ebb-106">예제</span><span class="sxs-lookup"><span data-stu-id="51ebb-106">Example</span></span>  
 <span data-ttu-id="51ebb-107">다음 샘플에서는 CS0551을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="51ebb-107">The following sample generates CS0551.</span></span>  
  
```csharp  
// CS0551.cs  
// compile with: /target:library  
interface ii  
{  
   int i  
   {  
      get;  
      set;  
   }  
}  
  
public class a : ii  
{  
   int ii.i { set {} }   // CS0551  
  
   // OK  
   int ii.i      
   {  
      set {}  
      get { return 0; }  
   }  
}  
```
