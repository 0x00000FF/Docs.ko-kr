<span data-ttu-id="7f6c2-103">'class'는 Object.Equals(object o)를 재정의하지만 Object.GetHashCode()를 재정의하지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="7f6c2-103">'class' overrides Object.Equals(object o) but does not override Object.GetHashCode()</span></span>  

<span data-ttu-id="7f6c2-104">컴파일러의 재정의 검색 합니다 <xref:System.Object.Equals%2A?displayProperty=nameWithType> 했지만 메서드 재정의 <xref:System.Object.GetHashCode%2A?displayProperty=nameWithType> 메서드.</span><span class="sxs-lookup"><span data-stu-id="7f6c2-104">The compiler detected an override of the <xref:System.Object.Equals%2A?displayProperty=nameWithType> method but no override of the <xref:System.Object.GetHashCode%2A?displayProperty=nameWithType> method.</span></span> <span data-ttu-id="7f6c2-105">재정의 <xref:System.Object.Equals%2A> 도 재정의 하고자 함을 나타냅니다 <xref:System.Object.GetHashCode%2A>합니다.</span><span class="sxs-lookup"><span data-stu-id="7f6c2-105">An override of <xref:System.Object.Equals%2A> implies that you also want to override <xref:System.Object.GetHashCode%2A>.</span></span>  

<span data-ttu-id="7f6c2-106">다음 코드에서는 CS0659를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="7f6c2-106">The following code generates CS0659:</span></span>  

```csharp
// CS0659.cs

class Test
{  
   public override bool Equals(object o) { return true; }   // CS0659  
}  
  
// OK  
class Test2  
{  
   public override bool Equals(object o) { return true; }  
   public override int GetHashCode() { return 0; }  
}  
```
