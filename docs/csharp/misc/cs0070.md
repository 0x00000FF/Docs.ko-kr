# <a name="compiler-error-cs0070"></a><span data-ttu-id="0fc96-102">컴파일러 오류 CS0070</span><span class="sxs-lookup"><span data-stu-id="0fc96-102">Compiler Error CS0070</span></span>
<span data-ttu-id="0fc96-103">'event' 이벤트는 += 또는 -=의 왼쪽에만 사용할 수 있습니다. 단 이 이벤트가 'type' 형식에서 사용될 때는 예외입니다.</span><span class="sxs-lookup"><span data-stu-id="0fc96-103">The event 'event' can only appear on the left hand side of += or -= (except when used from within the type 'type')</span></span>  
  
 <span data-ttu-id="0fc96-104">정의된 클래스의 외부에서는 [이벤트](../../csharp/language-reference/keywords/event.md) 가 참조를 추가하거나 뺄 수만 있습니다.</span><span class="sxs-lookup"><span data-stu-id="0fc96-104">Outside of the class it is defined in, an [event](../../csharp/language-reference/keywords/event.md) can only add or subtract references.</span></span> <span data-ttu-id="0fc96-105">자세한 내용은 [이벤트](../../csharp/programming-guide/events/index.md)를 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="0fc96-105">For more information, see [Events](../../csharp/programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="0fc96-106">다음 샘플에서는 CS0070을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="0fc96-106">The following sample generates CS0070:</span></span>  
  
```csharp  
// CS0070.cs  
using System;  
public delegate void EventHandler();  
  
public class A  
{  
   public event EventHandler Click;  
  
   public static void OnClick()  
   {  
      EventHandler eh;  
      A a = new A();  
      eh = a.Click;  
   }  
  
   public static void Main()  
   {  
   }  
}  
  
public class B  
{  
   public int Foo ()  
   {  
      EventHandler eh = new EventHandler(A.OnClick);  
      A a = new A();  
      eh = a.Click;   // CS0070  
      // try the following line instead  
      // a.Click += eh;  
      return 1;  
   }  
}  
```
