<span data-ttu-id="0c8e9-103">관리되는 형식('type')의 주소 또는 크기를 가져오거나 해당 형식에 대한 포인터를 선언할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="0c8e9-103">Cannot take the address of, get the size of, or declare a pointer to a managed type ('type')</span></span>  
  
 <span data-ttu-id="0c8e9-104">[unsafe](../../csharp/language-reference/keywords/unsafe.md) 키워드와 함께 사용하는 경우에도 관리 개체의 주소를 사용하거나 관리 개체의 크기를 가져올 수 없으며 관리되는 형식에 대한 포인터를 선언할 수도 없습니다.</span><span class="sxs-lookup"><span data-stu-id="0c8e9-104">Even when used with the [unsafe](../../csharp/language-reference/keywords/unsafe.md) keyword, taking the address of a managed object, getting the size of a managed object, or declaring a pointer to a managed type is not allowed.</span></span> <span data-ttu-id="0c8e9-105">관리되는 형식은 다음과 같습니다.</span><span class="sxs-lookup"><span data-stu-id="0c8e9-105">A managed type is:</span></span>  
  
-   <span data-ttu-id="0c8e9-106">임의 참조 형식</span><span class="sxs-lookup"><span data-stu-id="0c8e9-106">any reference type</span></span>  
  
-   <span data-ttu-id="0c8e9-107">필드 또는 속성으로 참조 형식을 포함하는 임의 구조체</span><span class="sxs-lookup"><span data-stu-id="0c8e9-107">any struct that contains a reference type as a field or property</span></span>  
  
 <span data-ttu-id="0c8e9-108">자세한 내용은 참조 [안전 하지 않은 코드 및 포인터](../../csharp/programming-guide/unsafe-code-pointers/index.md) 및 [sizeof](../../csharp/language-reference/keywords/sizeof.md)합니다.</span><span class="sxs-lookup"><span data-stu-id="0c8e9-108">For more information, see [Unsafe Code and Pointers](../../csharp/programming-guide/unsafe-code-pointers/index.md) and [sizeof](../../csharp/language-reference/keywords/sizeof.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="0c8e9-109">예제</span><span class="sxs-lookup"><span data-stu-id="0c8e9-109">Example</span></span>  
 <span data-ttu-id="0c8e9-110">다음 샘플에서는 CS0208을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="0c8e9-110">The following sample generates CS0208:</span></span>  
  
```csharp  
// CS0208.cs  
// compile with: /unsafe  
  
class myClass  
{  
    public int a = 98;  
}  
  
struct myProblemStruct  
{  
    string s;  
    float f;  
}  
  
struct myGoodStruct  
{  
    int i;  
    float f;  
}  
  
public class MyClass  
{  
    unsafe public static void Main()  
    {  
        // myClass is a class, a managed type.  
        myClass s = new myClass();    
        myClass* s2 = &s;    // CS0208  
  
        // The struct contains a string, a managed type.  
        int i = sizeof(myProblemStruct); //CS0208  
  
        // The struct contains only value types.  
        i = sizeof(myGoodStruct); //OK  
  
    }  
}  
```
