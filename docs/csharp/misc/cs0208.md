-   <span data-ttu-id="4f981-107">필드 또는 속성으로 참조 형식을 포함하는 임의 구조체</span><span class="sxs-lookup"><span data-stu-id="4f981-107">any struct that contains a reference type as a field or property</span></span>  
  
 <span data-ttu-id="4f981-108">자세한 내용은 참조 [안전 하지 않은 코드 및 포인터](../../csharp/programming-guide/unsafe-code-pointers/index.md) 및 [sizeof](../../csharp/language-reference/keywords/sizeof.md)합니다.</span><span class="sxs-lookup"><span data-stu-id="4f981-108">For more information, see [Unsafe Code and Pointers](../../csharp/programming-guide/unsafe-code-pointers/index.md) and [sizeof](../../csharp/language-reference/keywords/sizeof.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="4f981-109">예</span><span class="sxs-lookup"><span data-stu-id="4f981-109">Example</span></span>  
 <span data-ttu-id="4f981-110">다음 샘플에서는 CS0208을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="4f981-110">The following sample generates CS0208:</span></span>  
  
```csharp  
// CS0208.cs  
// compile with: /unsafe  
  
class myClass  
{  
    public int a = 98;  
}  
  
struct myProblemStruct  
{  
    string s;  
    float f;  
}  
  
struct myGoodStruct  
{  
    int i;  
    float f;  
}  
  
public class MyClass  
{  
    unsafe public static void Main()  
    {  
        // myClass is a class, a managed type.  
        myClass s = new myClass();    
        myClass* s2 = &s;    // CS0208  
  
        // The struct contains a string, a managed type.  
        int i = sizeof(myProblemStruct); //CS0208  
  
        // The struct contains only value types.  
        i = sizeof(myGoodStruct); //OK  
  
    }  
}  
```
