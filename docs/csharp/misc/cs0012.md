 <span data-ttu-id="c8103-108">다음 컴파일 시퀀스는 CS0012를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="c8103-108">The following sequence of compilations will result in CS0012:</span></span>

 <span data-ttu-id="c8103-109">다음을 수행합니다.</span><span class="sxs-lookup"><span data-stu-id="c8103-109">Then:</span></span>

```csharp  
   public static A f()

   }
```  

 <span data-ttu-id="c8103-110">다음을 수행합니다.</span><span class="sxs-lookup"><span data-stu-id="c8103-110">Then:</span></span>

```csharp  
   }

```
