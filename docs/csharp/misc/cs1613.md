# <a name="compiler-error-cs1613"></a><span data-ttu-id="2b3da-102">컴파일러 오류 CS1613</span><span class="sxs-lookup"><span data-stu-id="2b3da-102">Compiler Error CS1613</span></span>
<span data-ttu-id="2b3da-103">'interface' 인터페이스에 대해 관리되는 coclass 래퍼 클래스 'class'를 찾을 수 없습니다. 어셈블리 참조가 있는지 확인하세요.</span><span class="sxs-lookup"><span data-stu-id="2b3da-103">The managed coclass wrapper class 'class' for interface 'interface' cannot be found (are you missing an assembly reference?)</span></span>  
  
 <span data-ttu-id="2b3da-104">인터페이스에서 COM 개체를 인스턴스화하려고 했습니다.</span><span class="sxs-lookup"><span data-stu-id="2b3da-104">An attempt was made to instantiate a COM object from an interface.</span></span> <span data-ttu-id="2b3da-105">인터페이스에 **ComImport** 및 `CoClass` 특성이 있지만 컴파일러가 `CoClass` 특성에 대해 지정된 형식을 찾을 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="2b3da-105">The interface has the **ComImport** and `CoClass` attributes, but the compiler cannot find the type given for the `CoClass` attribute.</span></span>  
  
 <span data-ttu-id="2b3da-106">이 오류를 해결하려면 다음 중 하나를 시도할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="2b3da-106">To resolve this error, you can try one of the following:</span></span>  
  
-   <span data-ttu-id="2b3da-107">coclass가 있는 어셈블리에 대한 참조를 추가합니다(대체로 인터페이스와 coclass는 동일한 어셈블리에 있어야 함).</span><span class="sxs-lookup"><span data-stu-id="2b3da-107">Add a reference to the assembly that has the coclass (most of the time the interface and coclass should be in the same assembly).</span></span> <span data-ttu-id="2b3da-108">참조 [/참조](../../csharp/language-reference/compiler-options/reference-compiler-option.md) 또는 [참조 추가 대화 상자](http://msdn.microsoft.com/library/2feb0fe2-0805-4cc9-8cba-b0315849dfb7) 정보에 대 한 합니다.</span><span class="sxs-lookup"><span data-stu-id="2b3da-108">See [/reference](../../csharp/language-reference/compiler-options/reference-compiler-option.md) or [Add Reference Dialog Box](http://msdn.microsoft.com/library/2feb0fe2-0805-4cc9-8cba-b0315849dfb7) for information.</span></span>  
  
-   <span data-ttu-id="2b3da-109">인터페이스의 `CoClass` 특성을 수정합니다.</span><span class="sxs-lookup"><span data-stu-id="2b3da-109">Fix the `CoClass` attribute on the interface.</span></span>  
  
 <span data-ttu-id="2b3da-110">다음 샘플에서는 **CoClassAttribute**의 올바른 사용법을 보여 줍니다.</span><span class="sxs-lookup"><span data-stu-id="2b3da-110">The following sample demonstrates correct usage of **CoClassAttribute**:</span></span>  
  
```csharp  
// CS1613.cs  
using System;  
using System.Runtime.InteropServices;  
  
[Guid("1FFD7840-E82D-4268-875C-80A160C23296")]  
[ComImport()]  
[CoClass(typeof(A))]  
public interface IA{}  
public class A : IA {}  
  
public class AA  
{  
   public static void Main()  
   {  
      IA i;  
      i = new IA(); // This is equivalent to new A().  
                    // because of the CoClass attribute on IA  
   }  
}  
```
