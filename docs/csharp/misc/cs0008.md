# 컴파일러 오류 CS0008
'file' 파일에서 메타데이터를 읽는 동안 예기치 않은 오류가 발생했습니다. 'description'  
  
 메타데이터를 가져오기 위해 DLL을 열었지만 손상되었습니다. 따라서 데이터를 읽을 수 없습니다. 자세한 내용은 [\/reference \(Import Metadata\)](../../csharp/language-reference/compiler-options/reference-compiler-option.md)을 참조하세요.
