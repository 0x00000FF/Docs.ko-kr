## <a name="example"></a><span data-ttu-id="ac23f-109">예</span><span class="sxs-lookup"><span data-stu-id="ac23f-109">Example</span></span>  
 <span data-ttu-id="ac23f-110">다음 예제에서는 CS0279를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="ac23f-110">The following example generates CS0279:</span></span>  
  
```csharp  
// CS0279.cs  
  
using System;  
using System.Collections;  
  
public class myTest : IEnumerable  
{  
    IEnumerator IEnumerable.GetEnumerator()  
    {  
        yield return 0;  
    }  
  
    internal IEnumerator GetEnumerator()  
    {  
        yield return 0;  
    }  
  
    public static void Main()  
    {  
        foreach (int i in new myTest()) {}  // CS0279  
    }  
}  
```
