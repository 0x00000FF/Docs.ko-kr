 <span data-ttu-id="42193-104">[params](../../csharp/language-reference/keywords/params.md) 매개 변수는 가변적인 인수 개수를 지원하고 다른 모든 매개 변수 뒤에 있어야 합니다.</span><span class="sxs-lookup"><span data-stu-id="42193-104">The [params](../../csharp/language-reference/keywords/params.md) parameter supports a variable numbers of arguments and must be after all other parameters.</span></span> <span data-ttu-id="42193-105">자세한 내용은 [메서드](../../csharp/programming-guide/classes-and-structs/methods.md)를 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="42193-105">For more information, see [Methods](../../csharp/programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="42193-106">다음 샘플에서는 CS0231을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="42193-106">The following sample generates CS0231:</span></span>  
  
```csharp  
// CS0231.cs

{  
   public void TestMethod(params int[] p, int i) {} // CS0231  
   // To resolve the error, use the following line instead:  
   // public void TestMethod(int i, params int[] p) {}

   {  
   }  
}  
```
