## <a name="example"></a><span data-ttu-id="386b9-106">예제</span><span class="sxs-lookup"><span data-stu-id="386b9-106">Example</span></span>  
 <span data-ttu-id="386b9-107">다음 샘플에서는 CS0508을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="386b9-107">The following sample generates CS0508.</span></span>  
  
```csharp  
// CS0508.cs  
// compile with: /target:library  
abstract public class Clx  
{  
   public int i = 0;  
   // Return type is int.  
   abstract public int F();  
}  
  
public class Cly : Clx  
{  
   public override double F()  
   {  
      return 0.0;   // CS0508  
   }  
}  
```
