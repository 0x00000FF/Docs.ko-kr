<span data-ttu-id="36a2b-103">일관성 없는 접근성: 'interface 1' 기본 인터페이스가 'interface 2' 인터페이스보다 액세스하기 어렵습니다.</span><span class="sxs-lookup"><span data-stu-id="36a2b-103">Inconsistent accessibility: base interface 'interface 1' is less accessible than interface 'interface 2'</span></span>  
  
 <span data-ttu-id="36a2b-104">[공용](../../csharp/language-reference/keywords/public.md) 구문은 공개적으로 액세스 가능한 개체를 반환해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="36a2b-104">A [public](../../csharp/language-reference/keywords/public.md) construct must return a publicly accessible object.</span></span>  
  
 <span data-ttu-id="36a2b-105">파생된 인터페이스에서는 인터페이스 접근성을 축소할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="36a2b-105">Interface accessibility cannot be narrowed in a derived interface.</span></span> <span data-ttu-id="36a2b-106">자세한 내용은 참조 [인터페이스](../../csharp/programming-guide/interfaces/index.md) 및 [액세스 한정자](../../csharp/programming-guide/classes-and-structs/access-modifiers.md)합니다.</span><span class="sxs-lookup"><span data-stu-id="36a2b-106">For more information, see [Interfaces](../../csharp/programming-guide/interfaces/index.md) and [Access Modifiers](../../csharp/programming-guide/classes-and-structs/access-modifiers.md).</span></span>  
  
 <span data-ttu-id="36a2b-107">다음 샘플에서는 CS0061을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="36a2b-107">The following sample generates CS0061.</span></span>  
  
```  
// CS0061.cs  
// compile with: /target:library  
internal interface A {}  
public interface AA : A {}  // CS0061  
  
// OK  
public interface B {}  
internal interface BB : B {}  
  
internal interface C {}  
internal interface CC : C {}  
```
