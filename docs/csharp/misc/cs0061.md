# <a name="compiler-error-cs0061"></a><span data-ttu-id="a90e2-102">컴파일러 오류 CS0061</span><span class="sxs-lookup"><span data-stu-id="a90e2-102">Compiler Error CS0061</span></span>
<span data-ttu-id="a90e2-103">일관성 없는 접근성: 'interface 1' 기본 인터페이스가 'interface 2' 인터페이스보다 액세스하기 어렵습니다.</span><span class="sxs-lookup"><span data-stu-id="a90e2-103">Inconsistent accessibility: base interface 'interface 1' is less accessible than interface 'interface 2'</span></span>  
  
 <span data-ttu-id="a90e2-104">[공용](../language-reference/keywords/public.md) 구문은 공개적으로 액세스 가능한 개체를 반환해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="a90e2-104">A [public](../language-reference/keywords/public.md) construct must return a publicly accessible object.</span></span>  
  
 <span data-ttu-id="a90e2-105">파생된 인터페이스에서는 인터페이스 접근성을 축소할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="a90e2-105">Interface accessibility cannot be narrowed in a derived interface.</span></span> <span data-ttu-id="a90e2-106">자세한 내용은 [인터페이스](../programming-guide/interfaces/index.md) 및 [액세스 한정자](../programming-guide/classes-and-structs/access-modifiers.md)를 참조 하세요.</span><span class="sxs-lookup"><span data-stu-id="a90e2-106">For more information, see [Interfaces](../programming-guide/interfaces/index.md) and [Access Modifiers](../programming-guide/classes-and-structs/access-modifiers.md).</span></span>  
  
 <span data-ttu-id="a90e2-107">다음 샘플에서는 CS0061을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="a90e2-107">The following sample generates CS0061.</span></span>  
  
```csharp  
// CS0061.cs  
// compile with: /target:library  
internal interface A {}  
public interface AA : A {}  // CS0061  
  
// OK  
public interface B {}  
internal interface BB : B {}  
  
internal interface C {}  
internal interface CC : C {}  
```
