 <span data-ttu-id="3c2f2-104">[공용](../../csharp/language-reference/keywords/public.md) 구문은 공개적으로 액세스 가능한 개체를 반환해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="3c2f2-104">A [public](../../csharp/language-reference/keywords/public.md) construct must return a publicly accessible object.</span></span>

 <span data-ttu-id="3c2f2-107">다음 샘플에서는 CS0061을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="3c2f2-107">The following sample generates CS0061.</span></span>  
  
```csharp  
// CS0061.cs

internal interface A {}  
public interface AA : A {}  // CS0061  
  
// OK

internal interface BB : B {}  
  
internal interface C {}  
internal interface CC : C {}  
```
