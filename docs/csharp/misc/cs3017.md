## <a name="example"></a><span data-ttu-id="72622-108">예제</span><span class="sxs-lookup"><span data-stu-id="72622-108">Example</span></span>  
 <span data-ttu-id="72622-109">다음 예제는 CS3017을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="72622-109">The following example generates CS3017:</span></span>  
  
```csharp

[module: CLSCompliant(true)]

// Try this line instead:  
// [assembly: CLSCompliant(true)]

{

}  
```
