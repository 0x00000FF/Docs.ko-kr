## <a name="example"></a><span data-ttu-id="2a387-105">예</span><span class="sxs-lookup"><span data-stu-id="2a387-105">Example</span></span>  
 <span data-ttu-id="2a387-106">다음 샘플에서는 CS1529를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="2a387-106">The following sample generates CS1529:</span></span>  
  
```csharp  
// CS1529.cs

{  
    namespace Subspace  
    {  
        using Microsoft;  
  
        class SomeClass  
        {  
        };  
  
        using Microsoft;      // CS1529, place before class definition  
    }  
  
    using System.Reflection;  // CS1529, place before namespace 'Subspace'  
}

```
