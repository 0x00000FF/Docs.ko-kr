# <a name="compiler-error-cs0584"></a>컴파일러 오류 CS0584
내부 컴파일러 오류: 'stage' 단계 'symbol' 기호  
  
 컴파일러가 예기치 않은 구문을 분석할 수 없어 실패하는지 여부를 확인합니다. 시도 하지 않은 경우 [의견 보내기](/visualstudio/ide/talk-to-us)합니다.
