# 컴파일러 오류 CS2022
'\/out' 및 '\/target' 옵션은 소스 파일 이름 앞에 사용해야 합니다.  
  
 [\/out\(출력 파일 이름 설정\)](../../csharp/language-reference/compiler-options/out-compiler-option.md) 및 [\/target\(출력 파일 형식 지정\)](../../csharp/language-reference/compiler-options/target-compiler-option.md) 컴파일러 옵션이 명령줄에서 지정된 경우 소스 코드 파일 앞에 와야 합니다.
