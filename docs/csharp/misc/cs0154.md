## <a name="example"></a><span data-ttu-id="17f2e-106">예제</span><span class="sxs-lookup"><span data-stu-id="17f2e-106">Example</span></span>  
 <span data-ttu-id="17f2e-107">다음 샘플에서는 CS0154를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="17f2e-107">The following sample generates CS0154:</span></span>  
  
```csharp  
// CS0154.cs  
public class MyClass2  
{  
    public int i  
    {  
        set  
        {  
        }  
        // uncomment the get method to resolve this error  
        /*  
        get  
        {  
            return 0;  
        }  
        */  
    }  
}  
  
public class MyClass  
{  
    public static void Main()  
    {  
        MyClass2 myClass2 = new MyClass2();  
        int j = myClass2.i;   // CS0154, no get method  
    }  
}  
```
