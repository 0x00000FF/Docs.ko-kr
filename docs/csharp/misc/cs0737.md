 <span data-ttu-id="02373-105">인터페이스 멤버를 구현하는 메서드에는 public 접근성이 있어야 합니다.</span><span class="sxs-lookup"><span data-stu-id="02373-105">A method that implements an interface member must have public accessibility.</span></span> <span data-ttu-id="02373-106">모든 인터페이스 멤버는 `public`입니다.</span><span class="sxs-lookup"><span data-stu-id="02373-106">All interface members are `public`.</span></span>

## <a name="example"></a><span data-ttu-id="02373-109">예제</span><span class="sxs-lookup"><span data-stu-id="02373-109">Example</span></span>  
 <span data-ttu-id="02373-110">다음 코드에서는 CS0737을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="02373-110">The following code generates CS0737:</span></span>  
  
```csharp  
// cs0737.cs  
interface ITest  
{  
    // Default access of private with no modifier.  
    int Return42();  
    // Try the following line instead.  
    // public int Return42();  
}  
  
struct Struct1 : ITest // CS0737  
{  
    int Return42() { return (42); }  
}  
  
public class Test  
{  
    public static int Main(string[] args)  
    {  
        Struct1 s1 = new Struct1();  
  
        return (1);  
    }  
  
}  
```
