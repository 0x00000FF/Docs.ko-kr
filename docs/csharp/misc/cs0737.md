## <a name="example"></a><span data-ttu-id="048ac-109">예제</span><span class="sxs-lookup"><span data-stu-id="048ac-109">Example</span></span>  
 <span data-ttu-id="048ac-110">다음 코드에서는 CS0737을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="048ac-110">The following code generates CS0737:</span></span>  
  
```  
// cs0737.cs  
interface ITest  
{  
    // Default access of private with no modifier.  
    int Return42();  
    // Try the following line instead.  
    // public int Return42();  
}  
  
struct Struct1 : ITest // CS0737  
{  
    int Return42() { return (42); }  
}  
  
public class Test  
{  
    public static int Main(string[] args)  
    {  
        Struct1 s1 = new Struct1();  
  
        return (1);  
    }  
  
}  
```
