# <a name="compiler-error-cs0572"></a><span data-ttu-id="9c9d9-102">컴파일러 오류 CS0572</span><span class="sxs-lookup"><span data-stu-id="9c9d9-102">Compiler Error CS0572</span></span>
<span data-ttu-id="9c9d9-103">'type': 식을 통해 형식을 참조할 수 없습니다. 대신 'path_to_type'을 시도하세요.</span><span class="sxs-lookup"><span data-stu-id="9c9d9-103">'type' : cannot reference a type through an expression; try 'path_to_type' instead</span></span>  
  
 <span data-ttu-id="9c9d9-104">식별자를 통해 클래스의 멤버에 액세스하려고 했습니다. 이는 이 상황에서는 허용되지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="9c9d9-104">An attempt was made to access a member of a class through an identifier, which is not permitted in this situation.</span></span>  
  
 <span data-ttu-id="9c9d9-105">다음 샘플에서는 CS0572를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="9c9d9-105">The following sample generates CS0572:</span></span>  
  
```csharp  
// CS0572.cs  
using System;  
class C  
{  
   public class Inner  
   {  
      public static int v = 9;  
   }  
}  
  
class D : C  
{  
   public static void Main()  
   {  
      C cValue = new C();  
      Console.WriteLine(cValue.Inner.v);   // CS0572  
      // try the following line instead  
      // Console.WriteLine(C.Inner.v);  
   }  
}  
```
