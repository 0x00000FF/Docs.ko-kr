# <a name="compiler-warning-level-1-cs3004"></a>컴파일러 경고(수준 1) CS3004
혼합/분해된 유니코드 문자가 CLS 규격이 아닙니다.  
  
 CLS(공용 언어 사양)를 따르기 위해 [public](../../csharp/language-reference/keywords/public.md), [protected](../../csharp/language-reference/keywords/protected.md)또는 `protected internal` 식별자에는 구성된 유니코드 문자만 허용됩니다. CLS 규격에 대 한 자세한 내용은 참조 하세요. [언어 독립성 및 언어 독립적 구성 요소](../../standard/language-independence.md)합니다.
