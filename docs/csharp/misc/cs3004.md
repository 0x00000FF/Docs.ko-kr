# <a name="compiler-warning-level-1-cs3004"></a>컴파일러 경고(수준 1) CS3004
혼합/분해된 유니코드 문자가 CLS 규격이 아닙니다.  
  
 에 구성 된 유니코드 문자는 사용할 수 [공용](../../csharp/language-reference/keywords/public.md), [보호](../../csharp/language-reference/keywords/protected.md), 또는 `protected internal` 공용 언어 사양 (CLS 규격) 되려면 식별자입니다. CLS 규격에 대한 자세한 내용은 [CLS 규격 코드 작성](http://msdn.microsoft.com/library/4c705105-69a2-4e5e-b24e-0633bc32c7f3) 및 [언어 독립성 및 언어 독립적 구성 요소](../../../docs/standard/language-independence-and-language-independent-components.md)를 참조하세요.
