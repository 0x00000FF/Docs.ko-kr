 <span data-ttu-id="ad9c2-109">두 번째 파일은 이전 예제에서 만든 `C` 클래스에서 파생된 `B` 클래스를 정의하는 DLL을 만듭니다.</span><span class="sxs-lookup"><span data-stu-id="ad9c2-109">The second file creates a DLL that defines a class `C` that is derived from the class `B` that was created in the previous example.</span></span>
