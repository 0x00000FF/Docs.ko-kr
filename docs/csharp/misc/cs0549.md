# <a name="compiler-error-cs0549"></a><span data-ttu-id="41450-102">컴파일러 오류 CS0549</span><span class="sxs-lookup"><span data-stu-id="41450-102">Compiler Error CS0549</span></span>
<span data-ttu-id="41450-103">'function'은 봉인 클래스 'class'의 새 가상 멤버입니다.</span><span class="sxs-lookup"><span data-stu-id="41450-103">'function' is a new virtual member in sealed class 'class'</span></span>  
  
 <span data-ttu-id="41450-104">[Sealed](../language-reference/keywords/sealed.md)[클래스](../language-reference/keywords/class.md) 는 기본 클래스로 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="41450-104">A [sealed](../language-reference/keywords/sealed.md)[class](../language-reference/keywords/class.md) cannot be used as a base class.</span></span>  <span data-ttu-id="41450-105">그러므로 봉인된 클래스에는 가상 메서드가 있어도 소용이 없습니다.</span><span class="sxs-lookup"><span data-stu-id="41450-105">Therefore, it is useless to have a virtual method in sealed class.</span></span>  
  
 <span data-ttu-id="41450-106">다음 샘플에서는 CS0549를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="41450-106">The following sample generates CS0549:</span></span>  
  
```csharp  
// CS0549.cs  
// compile with: /target:library  
sealed public class MyClass  
{  
   virtual public void TestMethod() {}   // CS0549  
   public void TestMethod2() {}   // OK  
}  
```
