# <a name="compiler-error-cs1604"></a>컴파일러 오류 CS1604
읽기 전용인 'variable'에는 할당할 수 없습니다.  
  
 읽기 전용 변수에 할당했습니다. 이 오류를 방지하려면 이 컨텍스트에서 이 변수에 할당하지 않거나 수정하세요.
