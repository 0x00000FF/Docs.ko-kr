 <span data-ttu-id="e9624-105">해당 형식이 값을 저장할 수 없는 변수에 값을 할당하려고 시도했습니다.</span><span class="sxs-lookup"><span data-stu-id="e9624-105">An attempt was made to assign a value to a variable whose type cannot store the value.</span></span> <span data-ttu-id="e9624-106">자세한 내용은 [형식](../../csharp/programming-guide/types/index.md)을 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="e9624-106">For more information, see [Types](../../csharp/programming-guide/types/index.md).</span></span>  
  
 <span data-ttu-id="e9624-107">다음 샘플에서는 Checked와 Unchecked 컨텍스트 둘 다에서 CS0031을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="e9624-107">The following sample generates CS0031 in both checked and unchecked contexts:</span></span>

```csharp  
{  
   public class a  
   {  
      public static void Main()  
      {  
         int num = (int)2147483648M; //CS0031  
         // Try using a larger numeric type instead:  
         // long num = (long)2147483648M; //CS0031

            unchecked  
            {  
                const byte b = (byte)d; // CS0031  
                // For small values try using a signed byte instead:  
                // const sbyte b = (sbyte)d;  
            }  
      }  
   }  
}  
```
