# <a name="compiler-warning-level-2-cs0252"></a><span data-ttu-id="332a0-102">컴파일러 경고(수준 2) CS0252</span><span class="sxs-lookup"><span data-stu-id="332a0-102">Compiler Warning (level 2) CS0252</span></span>
<span data-ttu-id="332a0-103">의도하지 않은 참조 비교가 있을 수 있습니다. 값 비교를 가져오려면 왼쪽을 'type' 형식으로 캐스팅하세요.</span><span class="sxs-lookup"><span data-stu-id="332a0-103">Possible unintended reference comparison; to get a value comparison, cast the left hand side to type 'type'</span></span>  
  
 <span data-ttu-id="332a0-104">컴파일러가 참조 비교를 수행합니다.</span><span class="sxs-lookup"><span data-stu-id="332a0-104">The compiler is doing a reference comparison.</span></span> <span data-ttu-id="332a0-105">문자열의 값을 비교하려는 경우 식의 왼쪽을 `type`로 캐스트합니다.</span><span class="sxs-lookup"><span data-stu-id="332a0-105">If you want to compare the value of strings, cast the left side of the expression to `type`.</span></span>  
  
 <span data-ttu-id="332a0-106">다음 샘플에서는 CS0252를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="332a0-106">The following sample generates CS0252:</span></span>  
  
```csharp  
// CS0252.cs  
// compile with: /W:2  
using System;  
  
class MyClass  
{  
   public static void Main()  
   {  
      string s = "11";  
      object o = s + s;  
  
      bool b = o == s;   // CS0252  
      // try the following line instead  
      // bool b = (string)o == s;  
   }  
}  
```
