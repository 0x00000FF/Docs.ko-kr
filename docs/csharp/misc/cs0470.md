## <a name="example"></a><span data-ttu-id="19905-107">예제</span><span class="sxs-lookup"><span data-stu-id="19905-107">Example</span></span>

```csharp

interface I  
{  
   int P { get; }  
}

{  
   public int get_P() { return 0; }   // CS0470

}  
```
