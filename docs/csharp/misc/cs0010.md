# 컴파일러 오류 CS0010
예기치 않은 오류 \-\- 'error'  
  
 이 오류는 완전히 예기치 않은 문제가 발생하여 컴파일을 중지시킬 때 생성됩니다.  
  
### 이 오류를 해결하려면  
  
1.  다른 프로젝트를 다시 컴파일합니다. 동일한 오류를 확인한 경우 Visual Studio를 다시 설치합니다. 이 오류를 확인한 경우 오류 보고서를 Microsoft로 보냅니다.
