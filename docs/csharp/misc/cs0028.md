 <span data-ttu-id="26ea7-104">`Main` 에 대한 메서드 선언이 잘못되었습니다. 잘못된 서명으로 선언되었습니다.</span><span class="sxs-lookup"><span data-stu-id="26ea7-104">The method declaration for `Main` was invalid: it was declared with an invalid signature.</span></span> <span data-ttu-id="26ea7-105">`Main` 을 정적으로 선언해야 하며, [int](../../csharp/language-reference/keywords/int.md) 또는 [void](../../csharp/language-reference/keywords/void.md)를 반환해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="26ea7-105">`Main` must be declared as static and it must return either [int](../../csharp/language-reference/keywords/int.md) or [void](../../csharp/language-reference/keywords/void.md).</span></span> <span data-ttu-id="26ea7-106">자세한 내용은 참조 [main ()과 명령줄 인수](../../csharp/programming-guide/main-and-command-args/index.md)합니다.</span><span class="sxs-lookup"><span data-stu-id="26ea7-106">For more information, see [Main() and Command-Line Arguments](../../csharp/programming-guide/main-and-command-args/index.md).</span></span>  
  
 <span data-ttu-id="26ea7-107">다음 샘플에서는 CS0028을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="26ea7-107">The following sample generates CS0028:</span></span>

    }
