<span data-ttu-id="1f7b6-103">'base' 키워드는 정적 메서드에서 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="1f7b6-103">Keyword 'base' is not available in a static method</span></span>  
  
 <span data-ttu-id="1f7b6-104">[base](../../csharp/language-reference/keywords/base.md) 키워드가 [정적](../../csharp/language-reference/keywords/static.md) 메서드에서 사용되었습니다.</span><span class="sxs-lookup"><span data-stu-id="1f7b6-104">The [base](../../csharp/language-reference/keywords/base.md) keyword was used in a [static](../../csharp/language-reference/keywords/static.md) method.</span></span> <span data-ttu-id="1f7b6-105">`base` 는 인스턴스 생성자, 인스턴스 메서드 또는 인스턴스 접근자에서만 호출할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="1f7b6-105">`base` can only be called in an instance constructor, instance method, or instance accessor.</span></span>  
  
## <a name="example"></a><span data-ttu-id="1f7b6-106">예제</span><span class="sxs-lookup"><span data-stu-id="1f7b6-106">Example</span></span>  
 <span data-ttu-id="1f7b6-107">다음 샘플에서는 CS1511을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="1f7b6-107">The following sample generates CS1511.</span></span>  
  
```csharp  
// CS1511.cs  
// compile with: /target:library  
public class A  
{  
   public int j = 0;  
}  
  
class C : A  
{  
   public void Method()  
   {  
      base.j = 3;   // base allowed here  
   }  
  
   public static int StaticMethod()  
   {  
      base.j = 3;   // CS1511  
      return 1;  
   }  
}  
```
