# <a name="compiler-error-cs1101"></a><span data-ttu-id="3f5eb-102">컴파일러 오류 CS1101</span><span class="sxs-lookup"><span data-stu-id="3f5eb-102">Compiler Error CS1101</span></span>
<span data-ttu-id="3f5eb-103">매개 변수 한정자 'ref'는 'this'와 함께 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="3f5eb-103">The parameter modifier 'ref' cannot be used with 'this'.</span></span>  
  
 <span data-ttu-id="3f5eb-104">`this` 키워드가 정적 메서드의 첫 번째 매개 변수를 수정하는 경우 메서드가 확장 메서드임을 컴파일러에 알립니다.</span><span class="sxs-lookup"><span data-stu-id="3f5eb-104">When the `this` keyword modifies the first parameter of a static method, it signals to the compiler that the method is an extension method.</span></span> <span data-ttu-id="3f5eb-105">확장 메서드의 첫 번째 매개 변수에는 다른 한정자가 필요하지 않으며 허용되지도 않습니다.</span><span class="sxs-lookup"><span data-stu-id="3f5eb-105">No other modifiers are needed or allowed on the first parameter of an extension method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="3f5eb-106">예제</span><span class="sxs-lookup"><span data-stu-id="3f5eb-106">Example</span></span>  
 <span data-ttu-id="3f5eb-107">다음 예제에서는 CS1101을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="3f5eb-107">The following example generates CS1101:</span></span>  
  
```csharp  
// cs1101.cs  
// Compile with: /target:library  
public static class Extensions  
{  
    // No type parameters.  
        public static void Test(ref this int i) {} // CS1101  
  
    // Single type parameter.  
        public static void Test<T>(ref this T t) {}// CS1101  
  
    // Multiple type parameters.  
        public static void Test<T,U,V>(ref this U u) {}// CS1101  
}  
```
