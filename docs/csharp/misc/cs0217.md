# <a name="compiler-error-cs0217"></a>컴파일러 오류 CS0217
사용자 정의 논리 연산자('operator')를 단락(short circuit) 연산자로 사용하려면 연산자의 두 매개 변수와 같은 형식을 반환해야 합니다.  
  
 사용자 정의 형식에 대한 연산자를 정의한 다음 연산자를 단락 연산자로 사용하려면 사용자 정의 연산자에 동일한 형식의 반환 값 및 매개 변수가 있어야 합니다. 연산자를 단락 (short-circuit)에 대 한 자세한 내용은 [ `&&` 연산자](../../csharp/language-reference/operators/boolean-logical-operators.md#conditional-logical-and-operator-) 하 고 [ `||` 연산자](../../csharp/language-reference/operators/boolean-logical-operators.md#conditional-logical-or-operator-)합니다. 사용자 정의 하는 방법에 대 한 자세한 내용은 단락 (short-circuit), 조건 연산자 참조를 [사용자 정의 조건부 논리 연산자](~/_csharplang/spec/expressions.md#user-defined-conditional-logical-operators) 섹션을 [ C# 언어 사양](~/_csharplang/spec/introduction.md).
  
 다음 샘플에서는 CS0217을 생성합니다.  
  
```csharp  
// CS0217.cs  
using System;  
  
public class MyClass  
{  
   public static bool operator true (MyClass f)  
   {  
      return false;  
   }  
  
   public static bool operator false (MyClass f)  
   {  
      return false;  
   }  
  
   public static implicit operator int(MyClass x)  
   {  
      return 0;  
   }  
  
   public static int operator & (MyClass f1, MyClass f2)   // CS0217  
   // try the following line instead  
   // public static MyClass operator & (MyClass f1, MyClass f2)  
   {  
      return new MyClass();  
   }  
  
   public static void Main()  
   {  
      MyClass f = new MyClass();  
      int i = f && f;  
   }  
}  
```
