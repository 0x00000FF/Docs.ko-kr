# <a name="compiler-error-cs0601"></a><span data-ttu-id="92a6e-102">컴파일러 오류 CS0601</span><span class="sxs-lookup"><span data-stu-id="92a6e-102">Compiler Error CS0601</span></span>
<span data-ttu-id="92a6e-103">DllImport 특성은 'static' 및 'extern'으로 표시된 메서드에만 지정할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="92a6e-103">The DllImport attribute must be specified on a method marked 'static' and 'extern'</span></span>  
  
 <span data-ttu-id="92a6e-104">올바른 액세스 키워드가 없는 메서드에서 `DllImport` 특성이 사용되었습니다.</span><span class="sxs-lookup"><span data-stu-id="92a6e-104">The `DllImport` attribute was used on a method that did not have the correct access keywords.</span></span>  
  
 <span data-ttu-id="92a6e-105">다음 샘플에서는 CS0601을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="92a6e-105">The following sample generates CS0601:</span></span>  
  
```csharp  
// CS0601.cs  
using System.Runtime.InteropServices;  
using System.Text;  
  
public class C  
{  
   [DllImport("KERNEL32.DLL")]  
   extern int GetCurDirectory(int bufSize, StringBuilder buf);   // CS0601  
   // Try the following line instead:  
   // static extern int GetCurDirectory(int bufSize, StringBuilder buf);  
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
   }  
}  
```
