# <a name="compiler-error-cs0037"></a>컴파일러 오류 CS0037
null을 허용하지 않는 값 형식이므로 null을 'type'으로 변환할 수 없습니다.  
  
 컴파일러는 값 형식에 null을 할당할 수 없습니다. null은 [참조 형식](../language-reference/keywords/reference-types.md) 또는 Nullable 형식에만 할당할 수 있습니다. [struct](../language-reference/keywords/struct.md) 가 값 형식입니다. 자세한 내용은 [Null 허용 형식](../programming-guide/nullable-types/index.md)을 참조하세요.  
  
 다음 샘플에서는 CS0037을 생성합니다.  
  
```csharp  
// CS0037.cs  
public struct s  
{  
}  
  
class a  
{  
   public static void Main()  
   {  
      int i = null;   // CS0037  
      s ss = null;    // CS0037  
   }  
}  
```
