<span data-ttu-id="5ba73-103">null을 허용하지 않는 값 형식이므로 null을 'type'으로 변환할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="5ba73-103">Cannot convert null to 'type' because it is a non-nullable value type</span></span>  
  
 <span data-ttu-id="5ba73-104">컴파일러는 값 형식에 `null`을 할당할 수 없습니다. `null`은 [참조 형식](../language-reference/keywords/reference-types.md) 또는 [nullable 값 형식](../programming-guide/nullable-types/index.md)에만 할당할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="5ba73-104">The compiler cannot assign `null` to a value type; `null` can only be assigned to a [reference type](../language-reference/keywords/reference-types.md) or to a [nullable value type](../programming-guide/nullable-types/index.md).</span></span> <span data-ttu-id="5ba73-105">[struct](../language-reference/keywords/struct.md) 가 값 형식입니다.</span><span class="sxs-lookup"><span data-stu-id="5ba73-105">[struct](../language-reference/keywords/struct.md) is a value type.</span></span>
  
 <span data-ttu-id="5ba73-106">다음 샘플에서는 CS0037을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="5ba73-106">The following sample generates CS0037:</span></span>  
  
```csharp

{  
}

{  
   public static void Main()  
   {  
      int i = null;   // CS0037

   }  
}  
```
