# <a name="compiler-warning-level-1-cs0684"></a><span data-ttu-id="23d92-102">컴파일러 경고(수준 1) CS0684</span><span class="sxs-lookup"><span data-stu-id="23d92-102">Compiler Warning (level 1) CS0684</span></span>
<span data-ttu-id="23d92-103">'interface' 인터페이스는 'CoClassAttribute'로 표시되어 있고 'ComImportAttribute'로 표시되어 있지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="23d92-103">'interface' interface marked with 'CoClassAttribute' not marked with 'ComImportAttribute'</span></span>  
  
 <span data-ttu-id="23d92-104">인터페이스에서 **CoClassAttribute** 를 지정한 경우 **ComImportAttribute**도 지정해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="23d92-104">If you specify **CoClassAttribute** on an interface, you must also specify **ComImportAttribute**.</span></span>  
  
 <span data-ttu-id="23d92-105">다음 샘플에서는 CS0684를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="23d92-105">The following sample generates CS0684:</span></span>  
  
```csharp  
// CS0684.cs  
// compile with: /W:1  
using System;  
using System.Runtime.InteropServices;  
  
[CoClass(typeof(C))] // CS0684  
// try the following line instead  
// [CoClass(typeof(C)), ComImport]  
interface I  
{  
}  
  
class C  
{  
   static void Main() {}  
}  
```
