# <a name="compiler-error-cs0531"></a><span data-ttu-id="4245f-102">컴파일러 오류 CS0531</span><span class="sxs-lookup"><span data-stu-id="4245f-102">Compiler Error CS0531</span></span>
<span data-ttu-id="4245f-103">'member': 인터페이스 멤버에는 정의를 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="4245f-103">'member' : interface members cannot have a definition</span></span>  
  
 <span data-ttu-id="4245f-104">[인터페이스](../../csharp/language-reference/keywords/interface.md) 에서 선언된 메서드는 인터페이스 자체가 아니라 이 인터페이스에서 상속된 클래스에서 구현되어야 합니다.</span><span class="sxs-lookup"><span data-stu-id="4245f-104">Methods that are declared in an [interface](../../csharp/language-reference/keywords/interface.md) must be implemented in a class that inherits from it and not in the interface itself.</span></span>  
  
 <span data-ttu-id="4245f-105">다음 샘플에서는 CS0531을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="4245f-105">The following sample generates CS0531:</span></span>  
  
```csharp  
// CS0531.cs  
namespace x  
{  
   public interface clx  
   {  
      int xclx()   // CS0531, cannot define xclx  
      // Try the following declaration instead:  
      // int xclx();  
      {  
         return 0;  
      }  
   }  
  
   public class cly  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
