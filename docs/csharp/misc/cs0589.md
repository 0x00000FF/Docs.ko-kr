# 컴파일러 오류 CS0589
내부 컴파일러 오류: 'PARSE' 단계  
  
 컴파일러가 예기치 않은 구문을 분석할 수 없어 실패하는지 여부를 확인합니다. 컴파일러에 오류가 없는 경우 [의견 보내기](/visual-studio/ide/talk-to-us)를 시도합니다.
