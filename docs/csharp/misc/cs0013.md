# <a name="compiler-error-cs0013"></a>컴파일러 오류 CS0013
메타데이터를 'ModelStore.dll' 파일에 쓰는 동안 예기치 않은 오류가 발생했습니다. '남아 있는 논리 공간이 없으므로 사용자 문자열을 더 이상 만들 수 없습니다.  
  
 .NET Framework 공용 언어 런타임이 메타데이터를 내보내지 않았습니다. 경로가 올바르고 디스크가 꽉 차지 않았는지 확인합니다. 문제가 지속되면 [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)] 및/또는 .NET Framework를 복구하거나 다시 설치해야 할 수 있습니다.
