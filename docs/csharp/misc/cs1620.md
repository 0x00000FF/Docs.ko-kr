# <a name="compiler-error-cs1620"></a><span data-ttu-id="b59f1-102">컴파일러 오류 CS1620</span><span class="sxs-lookup"><span data-stu-id="b59f1-102">Compiler Error CS1620</span></span>
<span data-ttu-id="b59f1-103">'number' 인수는 'keyword' 키워드와 함께 전달해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="b59f1-103">Argument 'number' must be passed with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="b59f1-104">사용 하는 함수 인수를 전달 하는 경우이 오류가 발생 한 [ref](../../csharp/language-reference/keywords/ref.md) 또는 [아웃](../../csharp/language-reference/keywords/out-parameter-modifier.md) 매개 변수를 포함 하지 않습니다는 `ref` 또는 `out` 키워드 이거나, 호출 시 포함 잘못 된 키워드입니다.</span><span class="sxs-lookup"><span data-stu-id="b59f1-104">This error occurs if you are passing an argument to a function that takes a [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) parameter and you don't include the `ref` or `out` keyword at the point of call, or you include the wrong keyword.</span></span> <span data-ttu-id="b59f1-105">오류 텍스트는 사용할 적절한 키워드와 오류를 발생시킨 인수를 나타냅니다.</span><span class="sxs-lookup"><span data-stu-id="b59f1-105">The error text indicates the appropriate keyword to use and which argument caused the failure.</span></span>  
  
 <span data-ttu-id="b59f1-106">다음 샘플에서는 CS1620을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="b59f1-106">The following sample generates CS1620:</span></span>  
  
```csharp  
// CS1620.cs  
class C  
{  
    void f(ref int i) {}  
    public static void Main()  
    {  
        int x = 1;  
        f(out x);  // CS1620 – f takes a ref parameter, not an out parameter  
        // Try this line instead:  
        // f(ref x);  
    }  
}  
```
