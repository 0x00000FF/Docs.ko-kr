## <a name="example"></a><span data-ttu-id="69845-106">예제</span><span class="sxs-lookup"><span data-stu-id="69845-106">Example</span></span>  
 <span data-ttu-id="69845-107">다음 샘플에서는 C3015를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="69845-107">The following sample generates C3015.</span></span>  
  
```  
// CS3015.cs  
// compile with: /target:library

[assembly:CLSCompliant(true)]  
public class MyAttribute : Attribute  
{  
   public MyAttribute(int[] ai) {}   // CS3015  
   // try the following line instead  
   // public MyAttribute(int ai) {}  
}  
```
