 <span data-ttu-id="91823-104">정적 생성자는 클래스 인스턴스를 만들기 전에 자동으로 호출되기 때문에 정적 생성자에서 `this` 를 호출할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="91823-104">Calling `this` in the static constructor is not allowed because the static constructor is called automatically before creating any instance of the class.</span></span> <span data-ttu-id="91823-105">또한 정적 생성자는 상속되지 않고 직접 호출할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="91823-105">Also, static constructors are not inherited, and cannot be called directly.</span></span>

## <a name="example"></a><span data-ttu-id="91823-107">예</span><span class="sxs-lookup"><span data-stu-id="91823-107">Example</span></span>  
 <span data-ttu-id="91823-108">다음 예제에서는 CS0514를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="91823-108">The following example generates CS0514:</span></span>  
  
```csharp  
// CS0514.cs

    static A() : base(0) // CS0514  
    {  
    }  
  
    public A(object o)  
    {  
    }  
}

{  
    static B() : this(null) // CS0514  
    {  
    }  
  
    public B(object o)  
    {  
    }  
}  
```
