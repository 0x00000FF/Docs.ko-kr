 <span data-ttu-id="ead9d-104">정적 생성자는 클래스 인스턴스를 만들기 전에 자동으로 호출되기 때문에 정적 생성자에서 `this` 를 호출할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="ead9d-104">Calling `this` in the static constructor is not allowed because the static constructor is called automatically before creating any instance of the class.</span></span> <span data-ttu-id="ead9d-105">또한 정적 생성자는 상속되지 않고 직접 호출할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="ead9d-105">Also, static constructors are not inherited, and cannot be called directly.</span></span>

## <a name="example"></a><span data-ttu-id="ead9d-107">예제</span><span class="sxs-lookup"><span data-stu-id="ead9d-107">Example</span></span>

```csharp  
    static B() : this(null) // CS0514

    }  
  
    public B(object o)  
    {  
    }  
}  
```
