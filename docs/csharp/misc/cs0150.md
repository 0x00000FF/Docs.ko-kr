# <a name="compiler-error-cs0150"></a><span data-ttu-id="9fe52-102">컴파일러 오류 CS0150</span><span class="sxs-lookup"><span data-stu-id="9fe52-102">Compiler Error CS0150</span></span>
<span data-ttu-id="9fe52-103">상수 값이 필요합니다.</span><span class="sxs-lookup"><span data-stu-id="9fe52-103">A constant value is expected</span></span>  
  
 <span data-ttu-id="9fe52-104">상수가 필요한 변수가 있습니다.</span><span class="sxs-lookup"><span data-stu-id="9fe52-104">A variable was found where a constant was expected.</span></span> <span data-ttu-id="9fe52-105">자세한 내용은 [switch](../../csharp/language-reference/keywords/switch.md)을 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="9fe52-105">For more information, see [switch](../../csharp/language-reference/keywords/switch.md).</span></span>  
  
 <span data-ttu-id="9fe52-106">다음 샘플에서는 CS0150을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="9fe52-106">The following sample generates CS0150:</span></span>  
  
```csharp  
// CS0150.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
         int j = 0;  
  
         switch(i)  
         {  
            case j:   // CS0150, j is a variable int, not a constant int  
            // try the following line instead  
            // case 1:  
         }  
      }  
   }  
}  
```  
  
 <span data-ttu-id="9fe52-107">이 오류는 배열 크기가 변수 값으로 지정되고 배열 이니셜라이저로 초기화되는 경우에도 생성됩니다.</span><span class="sxs-lookup"><span data-stu-id="9fe52-107">This error is also produced when an array size is specified with a variable value and initialized with an array initializer.</span></span> <span data-ttu-id="9fe52-108">오류를 제거하려면 별도 문에서 배열을 초기화합니다.</span><span class="sxs-lookup"><span data-stu-id="9fe52-108">To remove the error, initialize the array in a separate statement or statements.</span></span>  
  
```csharp  
// CS0150.cs  
    namespace MyNamespace  
    {  
        public class MyClass  
        {  
            public static void Main()  
            {  
                int size = 2;  
                double[] nums = new double[size] { 46.9, 89.4 }; //CS0150  
                // Try the following lines instead  
                // double[] nums = new double[size];  
                // nums[0] = 46.9;   
                // nums[1] = 89.4;  
            }  
        }  
  
    }  
```
