<span data-ttu-id="0ad1a-103">기본 인터페이스 'type_2'가 CLS 규격이 아니므로 'type_1'은 CLS 규격이 아닙니다.</span><span class="sxs-lookup"><span data-stu-id="0ad1a-103">'type_1' is not CLS-compliant because base interface 'type_2' is not CLS-compliant</span></span>

## <a name="example"></a><span data-ttu-id="0ad1a-105">예</span><span class="sxs-lookup"><span data-stu-id="0ad1a-105">Example</span></span>  
 <span data-ttu-id="0ad1a-106">다음 샘플은 CLS 규격이 아닌 형식을 서명에 사용하여 형식을 CLS 규격으로 만드는 메서드가 있는 인터페이스를 포함합니다.</span><span class="sxs-lookup"><span data-stu-id="0ad1a-106">The following sample contains an interface with a method that uses a non-CLS compliant type in its signature, making the type non-CLS compliant.</span></span>

```csharp  
// compile with: /target:library

{  
   void IMethod(uint i);  
}  
```  
  
## <a name="example"></a><span data-ttu-id="0ad1a-107">예</span><span class="sxs-lookup"><span data-stu-id="0ad1a-107">Example</span></span>
