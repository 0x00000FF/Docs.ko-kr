## <a name="example"></a><span data-ttu-id="8109a-108">예제</span><span class="sxs-lookup"><span data-stu-id="8109a-108">Example</span></span>

```  
// cs0314.cs  
// Compile with: /target:library

public class A<T> where T : ClassConstraint  
{ }

{ }  
  
// Try using this instead.

{ }  
```
