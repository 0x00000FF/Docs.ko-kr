## <a name="example"></a><span data-ttu-id="21723-108">예제</span><span class="sxs-lookup"><span data-stu-id="21723-108">Example</span></span>

```csharp  
// cs0314.cs  
// Compile with: /target:library

public class A<T> where T : ClassConstraint  
{ }  
  
public class B<T> : A<T> //CS0314  
{ }  
  
// Try using this instead.

{ }  
```
