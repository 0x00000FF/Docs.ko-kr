<span data-ttu-id="8b49e-103">제네릭 형식 또는 메서드 'name'에서 'type1' 형식을 형식 매개 변수 'name'으로 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="8b49e-103">The type 'type1' cannot be used as type parameter 'name' in the generic type or method 'name'.</span></span> <span data-ttu-id="8b49e-104">'type1'에서 'type2'로의 형식 매개 변수 변환 또는 boxing 변환이 없습니다.</span><span class="sxs-lookup"><span data-stu-id="8b49e-104">There is no boxing conversion or type parameter conversion from 'type1' to 'type2'.</span></span>

## <a name="example"></a><span data-ttu-id="8b49e-108">예제</span><span class="sxs-lookup"><span data-stu-id="8b49e-108">Example</span></span>  
 <span data-ttu-id="8b49e-109">다음 코드에서는 CS0314를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="8b49e-109">The following code generates CS0314:</span></span>  
  
```csharp  
// cs0314.cs  
// Compile with: /target:library  
public class ClassConstraint { }  
  
public class A<T> where T : ClassConstraint  
{ }  
  
public class B<T> : A<T> //CS0314  
{ }  
  
// Try using this instead.  
public class C<T> : A<T> where T : ClassConstraint  
{ }  
```
