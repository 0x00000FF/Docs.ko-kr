# <a name="compiler-error-cs2006"></a>컴파일러 오류 CS2006
명령줄 구문 오류: 'option' 옵션에 대한 ':text'가 없습니다.  
  
 *옵션* 구문은 추가 텍스트가 필요합니다. 자세한 내용은 [컴파일러 옵션](../../csharp/language-reference/compiler-options/index.md)을 참조하세요.
