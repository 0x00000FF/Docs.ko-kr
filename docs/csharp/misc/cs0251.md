# <a name="compiler-warning-level-2-cs0251"></a><span data-ttu-id="b8dc8-102">컴파일러 경고(수준 2) CS0251</span><span class="sxs-lookup"><span data-stu-id="b8dc8-102">Compiler Warning (level 2) CS0251</span></span>
<span data-ttu-id="b8dc8-103">음수 인덱스를 사용하여 배열을 인덱싱했습니다. 배열 인덱스는 항상 0부터 시작합니다.</span><span class="sxs-lookup"><span data-stu-id="b8dc8-103">Indexing an array with a negative index (array indices always start at zero)</span></span>  
  
 <span data-ttu-id="b8dc8-104">음수를 사용하여 배열로 인덱싱하지 마세요.</span><span class="sxs-lookup"><span data-stu-id="b8dc8-104">Do not use a negative number to index into an array.</span></span>  
  
 <span data-ttu-id="b8dc8-105">다음 샘플에서는 CS0251을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="b8dc8-105">The following sample generates CS0251:</span></span>  
  
```csharp  
// CS0251.cs  
// compile with: /W:2  
class MyClass  
{  
   public static void Main()  
   {  
      int[] myarray = new int[] {1,2,3};
      try  
      {  
         myarray[-1]++;   // CS0251  
         // try the following line instead  
         // myarray[1]++;  
      }  
      catch (System.IndexOutOfRangeException e)  
      {  
         System.Console.WriteLine("{0}", e);  
      }  
   }  
}  
```
