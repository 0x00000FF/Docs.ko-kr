 <span data-ttu-id="a09ed-104">비속성 데이터 형식을 [property](../../csharp/programming-guide/classes-and-structs/properties.md)로 재정의하는 작업은 허용되지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="a09ed-104">An attempt was made to override a nonproperty data type as a [property](../../csharp/programming-guide/classes-and-structs/properties.md), which is not allowed.</span></span>
