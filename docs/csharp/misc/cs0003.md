# <a name="compiler-error-cs0003"></a>컴파일러 오류 CS0003
메모리 부족  
  
 컴파일러에서 컴파일을 완료하는 데 충분한 가상 메모리를 할당할 수 없습니다. 불필요한 응용 프로그램을 모두 닫고 다시 컴파일합니다.  
  
 또한 페이지 파일 크기를 늘리고 사용 가능한 디스크 공간이 있는지 확인하는 것이 좋습니다.  
  
 이 오류는 버전의 일치 하지 않는 것으로 나타날 수도 있습니다는 [!INCLUDE[winsdklong](../../csharp/language-reference/compiler-options/includes/winsdklong_md.md)] C# 컴파일러 및 C# 컴파일러를 지원 합니다; Visual Studio를 다시 설치 하는 하나 이상의 손상 된 파일입니다.
