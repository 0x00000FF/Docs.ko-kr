 <span data-ttu-id="8c938-104">[finally](../../csharp/language-reference/keywords/try-catch-finally.md) 절의 모든 문을 실행해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="8c938-104">All of the statements in a [finally](../../csharp/language-reference/keywords/try-catch-finally.md) clause must execute.</span></span> <span data-ttu-id="8c938-105">자세한 내용은 참조 [예외 처리문](../../csharp/language-reference/keywords/exception-handling-statements.md) 및 [예외 및 예외 처리](../../csharp/programming-guide/exceptions/index.md)합니다.</span><span class="sxs-lookup"><span data-stu-id="8c938-105">For more information, see [Exception Handling Statements](../../csharp/language-reference/keywords/exception-handling-statements.md) and [Exceptions and Exception Handling](../../csharp/programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="8c938-106">다음 샘플에서는 CS0157을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="8c938-106">The following sample generates CS0157:</span></span>  
  
```  
// CS0157.cs

{  
   public class MyClass2 : Exception  
   {  
   }  
  
   public class MyClass  
   {  
      public static void Main()  
      {  
         try  
         {  
         }  
  
         finally  
         {  
            return;   // CS0157, cannot leave finally clause  
         }  
      }  
   }  
}  
```
