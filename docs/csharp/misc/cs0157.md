# <a name="compiler-error-cs0157"></a><span data-ttu-id="6e474-102">컴파일러 오류 CS0157</span><span class="sxs-lookup"><span data-stu-id="6e474-102">Compiler Error CS0157</span></span>
<span data-ttu-id="6e474-103">제어가 finally 절의 본문을 벗어날 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="6e474-103">Control cannot leave the body of a finally clause</span></span>  
  
 <span data-ttu-id="6e474-104">모든의 문에 [마지막으로](../../csharp/language-reference/keywords/try-catch-finally.md) 절을 실행 해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="6e474-104">All of the statements in a [finally](../../csharp/language-reference/keywords/try-catch-finally.md) clause must execute.</span></span> <span data-ttu-id="6e474-105">자세한 내용은 참조 [예외 처리문](../../csharp/language-reference/keywords/exception-handling-statements.md) 및 [예외 및 예외 처리](../../csharp/programming-guide/exceptions/index.md)합니다.</span><span class="sxs-lookup"><span data-stu-id="6e474-105">For more information, see [Exception Handling Statements](../../csharp/language-reference/keywords/exception-handling-statements.md) and [Exceptions and Exception Handling](../../csharp/programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="6e474-106">다음 샘플에서는 CS0157을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="6e474-106">The following sample generates CS0157:</span></span>  
  
```  
// CS0157.cs  
using System;  
namespace MyNamespace  
{  
   public class MyClass2 : Exception  
   {  
   }  
  
   public class MyClass  
   {  
      public static void Main()  
      {  
         try  
         {  
         }  
  
         finally  
         {  
            return;   // CS0157, cannot leave finally clause  
         }  
      }  
   }  
}  
```
