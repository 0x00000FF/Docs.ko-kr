## <a name="example"></a><span data-ttu-id="5fb00-107">예제</span><span class="sxs-lookup"><span data-stu-id="5fb00-107">Example</span></span>

```csharp  
// CS1667.cs  
using System;  
  
public class C  
{  
    private int i;  
  
    //Try this instead:  
    //[Obsolete]  
    public int ObsoleteProperty  
    {  
        [Obsolete]  // CS1667  
        get { return i; }  
        set { i = value; }  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
