# <a name="compiler-error-cs0588"></a>컴파일러 오류 CS0588
내부 컴파일러 오류: 'LEX' 단계  
  
 컴파일러가 예기치 않은 구문을 분석할 수 없어 실패하는지 여부를 확인합니다. 없는 경우 시도 [의견 보내기](/visualstudio/ide/talk-to-us)합니다.
