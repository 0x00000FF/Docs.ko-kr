<span data-ttu-id="47bf4-103">클래스 형식 제약 조건 'constraint'는 다른 모든 제약 조건보다 앞에 와야 합니다.</span><span class="sxs-lookup"><span data-stu-id="47bf4-103">The class type constraint 'constraint' must come before any other constraints</span></span>

## <a name="example"></a><span data-ttu-id="47bf4-106">예</span><span class="sxs-lookup"><span data-stu-id="47bf4-106">Example</span></span>  
 <span data-ttu-id="47bf4-107">다음 샘플에서는 CS0406을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="47bf4-107">The following sample generates CS0406.</span></span>
