<span data-ttu-id="32953-103">'class': ComImport 특성이 있는 클래스는 기본 클래스를 지정할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="32953-103">'class': a class with the ComImport attribute cannot specify a base class</span></span>  
  
 <span data-ttu-id="32953-104"><xref:System.Runtime.InteropServices.ComImportAttribute> 특성을 지정하는 것은 COM 모듈에서 클래스에 대한 구현을 가져올 것을 의미합니다.</span><span class="sxs-lookup"><span data-stu-id="32953-104">Specifying the <xref:System.Runtime.InteropServices.ComImportAttribute> attribute implies that the implementation for the class is to be imported from a COM module.</span></span> <span data-ttu-id="32953-105">기본 클래스에서 상속된 추가 메서드 또는 필드는 COM 모듈에 정의된 구현에 추가할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="32953-105">Additional methods or fields inherited from the base class are not allowed to be added to the implementation defined in the COM module.</span></span>

```

[ ComImport, Guid("7ab770c7-0e23-4d7a-8aa2-19bfad479829") ]

```
