## <a name="creating-the-data-set"></a><span data-ttu-id="5380b-135">데이터 집합 만들기</span><span class="sxs-lookup"><span data-stu-id="5380b-135">Creating the Data Set</span></span>

}

<span data-ttu-id="5380b-141">여러 `from` 절이 `SelectMany`를 생성합니다. 그러면 첫 번째 시퀀스의 각 요소를 두 번째 시퀀스의 각 요소와 조합하는 단일 시퀀스가 만들어집니다.</span><span class="sxs-lookup"><span data-stu-id="5380b-141">The multiple `from` clauses produce a `SelectMany`, which creates a single sequence from combining each element in the first sequence with each element in the second sequence.</span></span> <span data-ttu-id="5380b-142">이 순서는 현재 목적에 따라 매우 중요합니다.</span><span class="sxs-lookup"><span data-stu-id="5380b-142">The order is important for our purposes.</span></span> <span data-ttu-id="5380b-143">첫 번째 소스 시퀀스(Suites)의 첫 번째 요소는 두 번째 시퀀스(Values)의 모든 요소와 조합됩니다.</span><span class="sxs-lookup"><span data-stu-id="5380b-143">The first element in the first source sequence (Suits) is combined with every element in the second sequence (Values).</span></span> <span data-ttu-id="5380b-144">그 결과 첫 번째 세트의 13개 카드가 생성됩니다.</span><span class="sxs-lookup"><span data-stu-id="5380b-144">This produces all thirteen cards of first suit.</span></span> <span data-ttu-id="5380b-145">이 프로세스는 첫 번째 시퀀스(Suites)의 각 요소에 대해 반복됩니다.</span><span class="sxs-lookup"><span data-stu-id="5380b-145">That process is repeated with each element in the first sequence (Suits).</span></span> <span data-ttu-id="5380b-146">최종 결과는 카드 데크를 세트별로 정렬한 후 다시 값별로 정렬한 상태입니다.</span><span class="sxs-lookup"><span data-stu-id="5380b-146">The end result is a deck of cards ordered by suits, followed by values.</span></span>

<span data-ttu-id="5380b-147">다음으로 Suits() 및 Ranks() 메서드를 빌드해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="5380b-147">Next, you'll need to build the Suits() and Ranks() methods.</span></span> <span data-ttu-id="5380b-148">시퀀스를 문자열 열거형으로 생성하는 정말 간단한 *반복기 메서드*부터 시작해 보겠습니다.</span><span class="sxs-lookup"><span data-stu-id="5380b-148">Let's start with a really simple set of *iterator methods* that generate the sequence as an enumerable of strings:</span></span>

    yield return "six";
    yield return "seven";
    yield return "eight";
    yield return "nine";
    yield return "ten";
    yield return "jack";
    yield return "queen";
    yield return "king";
    yield return "ace";

<span data-ttu-id="5380b-149">이러한 두 메서드는 `yield return` 구문을 활용하여 실행 시 시퀀스를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="5380b-149">These two methods both utilize the `yield return` syntax to produce a sequence as they run.</span></span> <span data-ttu-id="5380b-150">컴파일러는 `IEnumerable<T>`을 구현하는 개체를 빌드하고 요청 시 문자열 시퀀스를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="5380b-150">The compiler builds an object that implements `IEnumerable<T>` and generates the sequence of strings as they are requested.</span></span>
