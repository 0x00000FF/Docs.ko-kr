### <a name="wpf-aero2-and-aerolite-themes-toggle-button-doesnt-show-any-state"></a>WPF Aero2 및 AeroLite 테마 토글 단추에서 상태가 표시되지 않습니다.

|   |   |
|---|---|
|설명|.NET Framework 4.5에서 WPF의 Aero2 및 AeroLite 테마는 토글 단추의 &#39;on&#39; 및 &#39;off&#39; 상태 간의 차이를 표시하지 않습니다.|
|제안 해결 방법|이 문제는 서비스 업데이트에서 수정되었습니다. 이 문제를 해결하려면 .NET Framework 4.5를 업데이트하거나 .NET Framework 4.5.1 이상으로 업그레이드하십시오. 또는 이 문제를 해결하기 위해 토글 단추에 명시적으로 스타일을 지정할 수 있습니다.|
|범위|주요함|
|버전|4.5|
|형식|런타임|
|영향을 받는 API|<ul><li><xref:System.Windows.Controls.Primitives.ToggleButton?displayProperty=nameWithType></li><li><xref:System.Windows.Controls.Primitives.ToggleButton.%23ctor?displayProperty=nameWithType></li></ul>|
