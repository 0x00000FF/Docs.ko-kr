|   |   |
|---|---|
|설명|.NET Framework 4.5에서 <xref:System.Xml.Serialization.XmlSerializer?displayProperty=name>이 변경됨에 따라 직렬화된 XML에서 필드의 서식이 다르게 지정됩니다.|
|제안 해결 방법|이 동작은 .NET Framework 4.5 서비스 업데이트에서 수정되었습니다. 이 문제를 해결하려면 .NET Framework 4.5를 업데이트하거나 .NET Framework 4.5.1 이상으로 업그레이드하십시오. 또는 다음 구성 설정을 4.0 <xref:System.Xml.Serialization.XmlSerializer?displayProperty=name> 동작으로 되돌립니다.<pre><code>&lt;system.xml.serialization&gt;&#13;&#10;&lt;xmlSerializer useLegacySerializerGeneration=&quot;true&quot; /&gt;&#13;&#10;&lt;/system.xml.serialization&gt;&#13;&#10;</code></pre>|
